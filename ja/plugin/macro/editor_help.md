# Editor\_Help

指定するヘルプのページを表示します。このインライン関数を使うか、または
[EE\_\_HELP メッセージ](../message/ee_help) を直接送ることができます。

Editor\_Help( HWND hwnd, LPCTSTR szPageURL );

## パラメータ

_hwnd_

> EmEditor ビューまたはフレームのウィンドウ ハンドルを指定します。

_szString_

> 表示するヘルプのページのURLを指定します。

## 戻り値

> 戻り値は利用されません。

## バージョン

> EmEditor Version 7.00 以上で利用できます。
