# Editor\_UpdateToolbar

指定するコマンドIDのツールバーの状態を更新します。このインライン関数を使うか、または
[EE\_UPDATE\_TOOLBAR メッセージ](../message/ee_update_toolbar) を直接送ることができます。

Editor\_UpdateToolbar( HWND hwnd, UINT nCmdID );

## パラメータ

_hwnd_

EmEditor ビューまたはフレームのウィンドウ ハンドルを指定します。

_nCmdID_

実行するIDのコマンドIDを指定します。

## 戻り値

戻り値は利用されません。
