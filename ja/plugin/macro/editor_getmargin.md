# Editor\_GetMargin

折り返し桁数を返します。このインライン関数を使うか、または [EE\_GET\_MARGIN](../message/ee_get_margin) メッセージを直接送ることができます。

Editor\_Convert( HWND hwnd );

## パラメータ

_hwnd_

EmEditor ビューまたはフレームのウィンドウ ハンドルを指定します。

## 戻り値

現在の折り返し桁数を返します。普通行と引用行とで桁数が違う場合は、どちらか大きいほうを返します。ウィンドウの右端で折り返している場合は、ウィンドウの大きさによって値が決まります。
