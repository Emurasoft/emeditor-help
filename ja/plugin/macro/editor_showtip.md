# Editor\_ShowTip

ツール チップの表示/非表示を切り替えます。このインライン関数を使うか、または [EE\_SHOW\_TIP](../message/ee_show_tip) メッセージを直接送ることができます。

Editor\_ShowTip( HWND hwnd, TIP\_INFO\* pTipInfo );

## パラメータ

_hwnd_

> EmEditor ビューまたはフレームのウィンドウ ハンドルを指定します。

_pFilterInfo_

> [TIP\_INFO 構造体](../structure/tip_info) へのポインタを指定します。

## 戻り値

> 戻り値は利用されません。

## バージョン

> EmEditor Professional Version 16.9 以上で利用できます。
