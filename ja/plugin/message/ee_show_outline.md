# EE\_SHOW\_OUTLINE

アウトラインを表示または非表示に設定します。このメッセージを直接送るか、または
[Editor\_ShowOutline インライン関数](../macro/editor_showoutline) を使うことができます。

EE\_SHOW\_OUTLINE

wParam = (WPARAM) nFlags;

lParam = 0;

## パラメータ

_nFlags_

次の値のいずれかを指定します。

| 値 | 説明 |
| --- | --- |
| SHOW\_OUTLINE\_SHOW | アウトラインを表示します。 |
| SHOW\_OUTLINE\_HIDE | アウトラインを非表示に設定します。 |

## 戻り値

戻り値は利用されません。

## バージョン

EmEditor Professional Version 6.00 以上で利用できます。
