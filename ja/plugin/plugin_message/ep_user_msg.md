# EP\_USER\_MSG

プラグインへユーザー定義メッセージを送信します。

EP\_USER\_MSG

hwnd = hwndView;

wParam = wParam;

lParam = lParam;

## パラメータ

_hwndView_

> EmEditor ビューのウィンドウ ハンドルが格納されています。

_wParam_

> 第1パラメータを指定します。このパラメータの意味は各プラグインに依存します。

_lParam_

> 第2パラメータを指定します。このパラメータの意味は各プラグインに依存します。

## 戻り値

> 戻り値の意味は各プラグインに依存します。

## バージョン

> Version 15.5 以上で利用できます。
