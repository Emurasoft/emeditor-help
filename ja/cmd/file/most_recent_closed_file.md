# \[最も最近閉じたファイル\] コマンド

## 概要

最も最近閉じたファイルを開きます。

## 説明

最も最近閉じたファイルを開きます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[開く\] \- \[最も最近閉じたファイル\]
- ツール バー:
なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+T

## プラグイン コマンド ID

```
EEID_MOST_RECENT_CLOSED_FILE (4583)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4583);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4583
```
