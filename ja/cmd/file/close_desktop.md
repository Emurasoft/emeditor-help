# \[すべて閉じる(現在のデスクトップのみ)\] コマンド

### 概要

> 現在の仮想デスクトップのすべての文書を閉じます。

### 説明

> 現在の仮想デスクトップのすべての文書を閉じます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[閉じる\] \- \[すべて閉じる(現在のデスクトップのみ)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SAVE\_CLOSE\_DESKTOP (3935)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3935);

#### \[VBScript\]

> editor.ExecuteCommandByID 3935
