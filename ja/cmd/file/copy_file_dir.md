# \[ファイル ディレクトリ名をコピー\] コマンド

## 概要

ファイルのディレクトリ名をコピーしてクリップボードに保存します。

## 説明

ファイルのディレクトリ名をコピーしてクリップボードに保存します。無題の場合はこのコマンドを選択することができません。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[コピー\] \- \[ファイル ディレクトリ名\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_COPY_FILE_DIR (4327)```

## マクロ

### \[JavaScript\]

```
document.CopyPath();
```

### \[VBScript\]

```
document.CopyPath
```
