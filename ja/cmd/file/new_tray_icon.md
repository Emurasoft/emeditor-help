# \[トレイ アイコンを表示\] コマンド

### 概要

> 新しいトレイ アイコンを表示します。

### 説明

> このコマンドを実行すると、Windows タスク バーの通知領域に EmEditor のトレイ
> アイコンを表示します。既に表示されている場合は、このコマンドを実行しても変化はありません。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[新規作成\] \- \[トレイ アイコンを表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_NEW\_TRAY\_ICON (4121)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4121);

#### \[VBScript\]

> editor.ExecuteCommandByID 4121
