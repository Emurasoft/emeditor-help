# \[ファイル パス名をコピー\] コマンド

## 概要

ファイルの完全パスと名前をコピーしてクリップボードに保存します。

## 説明

ファイルの完全パスと名前をコピーしてクリップボードに保存します。無題の場合はこのコマンドを選択することができません。

## 実行方法

- 既定のメニュー: \[ファイル\] \- \[ファイル パス名をコピー\]
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[コピー\] \- \[ファイル パス名\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_COPY_FILE_PATH (4326)```

## マクロ

### \[JavaScript\]

```
document.CopyFullName();
```

### \[VBScript\]

```
document.CopyFullName
```
