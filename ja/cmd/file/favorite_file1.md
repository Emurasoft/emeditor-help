# \[お気に入りファイル\] コマンド

## 概要

指定したお気に入りファイルを開きます (複数項目)。

## 説明

指定したお気に入りファイルを開きます (複数項目)。

## 実行方法

- 既定のメニュー: \[ファイル\] \- \[お気に入り\] \- \[お気に入りファイル\]
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[お気に入り\] \- \[お気に入りファイル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
