# \[お気に入りに追加\] コマンド

## 概要

現在のファイルをお気に入りの一覧に追加します。

## 説明

現在のファイルをお気に入りの一覧に追加します。

## 実行方法

- 既定のメニュー: \[ファイル\] \- \[お気に入り\] \- \[お気に入りに追加\]
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[お気に入り\] \- \[お気に入りに追加\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

EEID_ADD_TO_FAVORITES (23250)
