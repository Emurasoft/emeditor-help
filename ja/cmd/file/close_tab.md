# \[タブを閉じる\] コマンド

## 概要

選択タブを閉じます。

## 説明

このコマンドを実行すると、選択タブを閉じます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ファイル\] \- \[閉じる\] \- \[タブを閉じる\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID__CLOSE_TAB (23246)
```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23246);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23246
```
