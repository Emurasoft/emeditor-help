# \[基本のプロパティ\] コマンド

### 概要

> 現在の設定の基本のプロパティを変更します。

### 説明

> このコマンドを実行すると、現在選択されている設定のプロパティの
> [\[基本\] ページ](../../dlg/properties/general/index) を表示します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[基本\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_PROPERTY\_MARGIN (8960)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(8960);

#### \[VBScript\]

> editor.ExecuteCommandByID 8960
