# \[古い設定を削除\] コマンド

## 概要

使われていない古い設定を削除します。

## 説明

使われていない古い設定を削除します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[古い設定を削除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_DELETE_OLD_SETTINGS (3984)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3984);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3984
```
