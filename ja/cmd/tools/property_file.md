# \[ファイルのプロパティ\] コマンド

### 概要

> 現在の設定のファイルのプロパティを変更します。

### 説明

> このコマンドを実行すると、現在選択されている設定のプロパティの
> [\[ファイル\] ページ](../../dlg/properties/file/index) を表示します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[ファイル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_PROPERTY\_FILE (8962)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(8962);

#### \[VBScript\]

> editor.ExecuteCommandByID 8962
