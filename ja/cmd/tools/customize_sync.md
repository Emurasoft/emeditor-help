# \[同期のカスタマイズ\] コマンド

### 概要

> \[カスタマイズ\] ダイアログ ボックスの \[同期\] ページを開きます。

### 説明

> [\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[同期\] ページを表示して、アプリケーション共通の設定をできるようにします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[同期\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CUSTOMIZE\_SYNC (9060)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(9060);

#### \[VBScript\]

> editor.ExecuteCommandByID 9060
