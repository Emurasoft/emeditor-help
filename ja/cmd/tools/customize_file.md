# \[ファイルのカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[ファイル\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[ファイル\] ページを表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[ファイル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_FILE (9040)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(9040);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 9040
```
