# \[ツール ジョブの中止\] コマンド

### 概要

> 現在実行されているツール ジョブを中止します。

### 説明

> 現在実行されているツール ジョブを中止します。

### 実行方法

- 既定のメニュー: \[ツール\] \- \[外部ツール\] \- \[ツール ジョブの中止\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[ツール ジョブの中止\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Break

### プラグイン コマンド ID

- EEID\_TERMINATE\_TOOL (4451)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4451);

#### \[VBScript\]

> editor.ExecuteCommandByID 4451
