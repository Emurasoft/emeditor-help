# \[ファイルを結合\] コマンド

## 概要

開いている複数文書または既存のファイルを1個のファイルに結合します。

## 説明

開いている複数文書または既存のファイルを1個のファイルに結合します。

## 実行方法

- 既定のメニュー: \[ツール\] \- \[ファイルを結合\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[分割/結合\] \-
\[ファイルを結合\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_COMBINE_FILES (4607)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4607);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4607
