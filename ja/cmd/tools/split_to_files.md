# \[ファイルを分割\] コマンド

## 概要

現在の文書または既存のファイルを分割して複数のファイルに保存します。

## 説明

現在の文書または既存のファイルを分割して複数のファイルに保存します。

## 実行方法

- 既定のメニュー: \[ツール\] \- \[ファイルを分割\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[分割/結合\] \-
\[ファイルを分割\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
