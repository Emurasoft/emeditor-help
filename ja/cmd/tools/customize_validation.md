# \[構文チェックのカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[構文チェック\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[構文チェック\] ページを表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[構文チェック\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_VALIDATION (9062)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(9062);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9062
```
