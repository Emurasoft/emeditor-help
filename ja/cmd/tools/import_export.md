# \[インポートとエクスポート\] コマンド

## 概要

ウィザードを表示してファイルから設定をインポートしたりファイルに現在の設定をエクスポートします。

## 説明

このコマンドを実行すると、 [インポートとエクスポートのウィザード](../../dlg/import_export/index) を表示します。そこで、ファイルから設定をインポートするか、またはファイルに現在の設定をエクスポートするかを選択することができます。

## 実行方法

- 既定のメニュー: \[ツール\] \- \[インポートとエクスポート\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[インポートとエクスポート\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_IMPORT_EXPORT (4380)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4380);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4380
```
