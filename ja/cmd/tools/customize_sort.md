# \[並べ替えのカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[並べ替え\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[並べ替え\] ページを表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: \[並べ替え\] \- \[並べ替えのオプション\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_SORT (9053)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(9053);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9053
```
