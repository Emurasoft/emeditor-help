# \[トレイ アイコンの設定\] コマンド

## 概要

トレイ アイコンの設定を行います。

## 説明

[\[トレイ アイコンの設定\] ダイアログ ボックス](../../dlg/tray/index) を表示して、トレイ
アイコンに関する設定を行います。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[トレイ アイコンの設定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_TRAY (4273)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4273);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4273
```
