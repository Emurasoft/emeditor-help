# \[言語のカスタマイズ\] コマンド

### 概要

> \[カスタマイズ\] ダイアログ ボックスの \[言語\] ページを開きます。

### 説明

> [\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[言語\] ページを表示して、アプリケーション共通の設定をできるようにします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[言語\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CUSTOMIZE\_LANG (9047)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(9047);

#### \[VBScript\]

> editor.ExecuteCommandByID 9047
