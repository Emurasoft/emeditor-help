# \[スペル チェックのプロパティ\] コマンド

### 概要

> 現在の設定のスペル チェックのプロパティを変更します。

### 説明

> このコマンドを実行すると、現在選択されている設定のプロパティの
> [\[スペル チェック\] ページ](../../dlg/properties/spell/index) を表示します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[スペル チェック\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_PROPERTY\_SPELL (8975)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(8975);

#### \[VBScript\]

> editor.ExecuteCommandByID 8975
