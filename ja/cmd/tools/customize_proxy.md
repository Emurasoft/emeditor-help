# \[プロキシのカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[プロキシ\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[プロキシ\] ページを表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[プロキシ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_PROXY (9076)
```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(9076);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9076
```
