# \[エンコードの定義\] コマンド

### 概要

> ファイルを開いたり保存するときに使用するエンコードの定義を行います。

### 説明

> [\[エンコードの定義\] ダイアログ ボックス](../../dlg/encodings/index) を表示して、ファイルを開いたり保存するときに使用するエンコードの定義を行います。

### 実行方法

- 既定のメニュー: \[ツール\] \- \[エンコードの定義\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[エンコードの定義\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_DEFINE\_CODE\_PAGE (4264)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4264);

#### \[VBScript\]

> editor.ExecuteCommandByID 4264
