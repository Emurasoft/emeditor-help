# \[自動保存のプロパティ\] コマンド

## 概要

現在の設定の自動保存のプロパティを変更します。

## 説明

このコマンドを実行すると、現在選択されている設定のプロパティの
[\[自動保存\] ページ](../../dlg/properties/autosave/index) を表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[自動保存\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_PROPERTY_AUTO_SAVE (8964)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(8964);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8964
```
