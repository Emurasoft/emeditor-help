# \[通知のカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[通知\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の [\[通知\] ページ](../../dlg/customize/notifications/index) を表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[通知\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_NOTIFICATIONS (9063)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(9063);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9063
```
