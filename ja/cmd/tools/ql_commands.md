# \[コマンドを検索\] コマンド

### 概要

> クイック起動ウィンドウを表示して、コマンドを選択します。

### 説明

> クイック起動ウィンドウを表示して、コマンドを選択します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[コマンドを検索\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[コマンドを検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: CTRL+SHIFT+C

### プラグイン コマンド ID

- EEID\_QL\_OPTIONS (3951)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3951);

#### \[VBScript\]

> editor.ExecuteCommandByID 3951
