# \[関連付け\] コマンド

### 概要

> 指定するファイルの拡張子を開くように関連付けます。

### 説明

> [\[EmEditorと関連付け\] ダイアログ ボックス](../../dlg/file_associate/index) を表示して、EmEditor
> に関連付ける拡張子を設定できるようにします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[関連付け\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FILE\_ASSOCIATE (4223)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4223);

#### \[VBScript\]

> editor.ExecuteCommandByID 4223
