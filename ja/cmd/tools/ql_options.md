# \[オプションを検索\] コマンド

### 概要

> クイック起動ウィンドウを表示して、オプションを選択します。

### 説明

> クイック起動ウィンドウを表示して、オプションを選択します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[オプションを検索\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[オプションを検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: CTRL+SHIFT+O

### プラグイン コマンド ID

- EEID\_QL\_OPTIONS (3952)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3952);

#### \[VBScript\]

> editor.ExecuteCommandByID 3952
