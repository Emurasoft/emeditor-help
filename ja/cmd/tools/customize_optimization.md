# \[最適化のカスタマイズ\] コマンド

## 概要

\[カスタマイズ\] ダイアログ ボックスの \[最適化\] ページを開きます。

## 説明

[\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[最適化\] ページを表示して、アプリケーション共通の設定をできるようにします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \- \[最適化\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CUSTOMIZE_OPTIMIZATION (9061)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(9061);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9061
```
