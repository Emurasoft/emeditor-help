# \[設定の関連付け\] コマンド

### 概要

> ファイルの種類を設定に関連付けます。

### 説明

> ファイルの種類を設定に関連付けます。

### 実行方法

- 既定のメニュー: \[ツール\] \- \[設定の関連付け\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[設定の関連付け\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CONFIG\_ASSOCIATE (3847)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3847);

#### \[VBScript\]

> editor.ExecuteCommandByID 3847
