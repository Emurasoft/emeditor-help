# \[ファイル保存時の詳細のプロパティ\] コマンド

## 概要

現在の設定のファイル保存時の詳細のプロパティを変更します。

## 説明

このコマンドを実行すると、現在選択されている設定のプロパティの
\[ファイル\] ページの
[\[保存時の詳細\] サブダイアログ ボックス](../../dlg/properties/spell/index) を表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[ファイル保存時の詳細\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_PROPERTY_FILE_SAVE (8979)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(8979);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8979
```
