# \[クイック起動\] コマンド

## 概要

すべてのコマンドから指定するコマンドを検索して素早く起動します。

## 説明

すべてのコマンドから指定するコマンドを検索して素早く起動します。

## 実行方法

- 既定のメニュー: \[ツール\] \- \[クイック起動\]
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[クイック起動\] \- \[クイック起動\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: CTRL+Q

## プラグイン コマンド ID

```
EEID_SEARCH_ALL_COMMANDS (4608)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4608);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4608
```
