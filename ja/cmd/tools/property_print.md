# \[印刷のプロパティ\] コマンド

### 概要

> 現在の設定の印刷のプロパティを変更します。

### 説明

> このコマンドを実行すると、現在選択されている設定のプロパティの
> [\[印刷\] ページ](../../dlg/properties/print/index) を表示します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[現在の設定のプロパティ\] \- \[印刷\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_PROPERTY\_PRINT (8972)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(8972);

#### \[VBScript\]

> editor.ExecuteCommandByID 8972
