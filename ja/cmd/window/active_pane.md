# \[アクティブなペイン\] コマンド

## 概要

アクティブなペインに切り替えます。

## 説明

カスタム バーが表示されている場合、アクティブなペインに切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[ペイン\] \- \[アクティブなペイン\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_ACTIVE_PANE (4409)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4409);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4409
```
