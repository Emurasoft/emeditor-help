# \[左へスクロール\] コマンド

### 概要

> 文書を一文字ずつ左へスクロールします。

### 説明

> 文書を一文字ずつ左へスクロールします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[左\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_SCROLL\_LEFT (4474)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4474);

#### \[VBScript\]

> editor.ExecuteCommandByID 4474
