# \[上下に並べて表示\] コマンド

## 概要

ウィンドウを上下に並べて表示します。

## 説明

ウィンドウを上下に並べて表示します。

## 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[上下に並べて表示\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[上下に並べて表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

EEID_WINDOW_TILE_HORZ (4242)```

## マクロ
