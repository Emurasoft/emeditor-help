# \[上下に分割して位置を固定\] コマンド

### 概要

> ウィンドウを上下に分割して位置を固定します。

### 説明

> ウィンドウを上下に分割して位置を固定します。このコマンドを実行すると、ウィンドウを中央で水平に 2 分割にして、すぐに分割位置を固定します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[分割\] \- \[上下に分割して位置を固定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_SPLIT\_HORZ\_FIX (4335)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4335);

#### \[VBScript\]

> editor.ExecuteCommandByID 4335
