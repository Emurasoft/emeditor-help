# \[左右に分割して位置を固定\] コマンド

## 概要

ウィンドウを左右に分割して位置を固定します。

## 説明

ウィンドウを左右に分割して位置を固定します。このコマンドを実行すると、ウィンドウを中央で垂直に 2 分割にして、すぐに分割位置を固定します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[分割\] \- \[左右に分割して位置を固定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_SPLIT_VERT_FIX (4336)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4336);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4336
```
