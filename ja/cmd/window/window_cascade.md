# \[重ねて表示\] コマンド

### 概要

> ウィンドウを重ねて表示します。

### 説明

> ウィンドウを重ねて表示します。

### 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[重ねて表示\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[重ねて表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_CASCADE (4241)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4241);

#### \[VBScript\]

> editor.ExecuteCommandByID 4241
