# \[分割ウィンドウの垂直同期スクロール\] コマンド

### 概要

> 分割ウィンドウの垂直スクロールを同期します。

### 説明

> 分割ウィンドウの垂直スクロールを同期します。

### 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[分割ウィンドウの垂直同期スクロール\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[分割\] \- \[分割ウィンドウの垂直同期スクロール\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SYNC\_VERT\_SCROLL\_SPLIT (4524)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4524);

#### \[VBScript\]

> editor.ExecuteCommandByID 4524
