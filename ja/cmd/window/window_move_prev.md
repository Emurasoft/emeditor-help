# \[前へ移動\] コマンド

## 概要

現在の文書をタブの前の位置に移動します。

## 説明

現在の文書をタブの前の位置に移動します。このコマンドを使って、タブ上に表示されるボタンの順番を変更することができます。また、タブ上のボタンをドラッグして、移動先でドロップすることによっても、ボタンの順番を変更することができます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[文書ナビゲーション\] \- \[前へ移動\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_MOVE_PREV (4383)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4383);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4383
```
