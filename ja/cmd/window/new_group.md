# \[新しいグループ\] コマンド

### 概要

> 現在の文書を結合されたウィンドウから切り離して新しいグループにします。

### 説明

> 現在の文書を結合されたウィンドウから切り離して新しいグループにします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[新しいグループ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_NEW\_GROUP (4391)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4391);

#### \[VBScript\]

> editor.ExecuteCommandByID 4391
