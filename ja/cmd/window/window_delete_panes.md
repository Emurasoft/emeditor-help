# \[分割の解除\] コマンド

### 概要

> 分割を解除します。

### 説明

> ウィンドウの分割を解除します。

### 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[分割の解除\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[分割\] \- \[分割の解除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Shift+F12

### プラグイン コマンド ID

- EEID\_DELETE\_PANES (4284)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4284);

#### \[VBScript\]

> editor.ExecuteCommandByID 4284
