# \[右のページにスクロール\] コマンド

> 右のページにスクロールします。

### 説明

> 右のページにスクロールします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[右のページ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SCROLL\_PAGE\_RIGHT (3869)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3869);

#### \[VBScript\]
