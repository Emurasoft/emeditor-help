# \[1行上へスクロール\] コマンド

## 概要

1行上へスクロールします。

## 説明

1 行上へスクロールします。ただし、プロパティの [\[スクロール\] タブ](../../dlg/properties/scroll/index) で\[2行スクロール\] チェック ボックスがチェックされている場合、2 行上へスクロールします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[1行上\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Up

## プラグイン コマンド ID

```
EEID_SCROLL_UP (4170)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4170);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4170
```
