# \[ポイントした時ミニマップを表示\] コマンド

### 概要

> マウスで垂直スクロール バー上にポイントしている間、ミニマップを表示します。

### 説明

> マウスで垂直スクロール バー上にポイントしている間、ミニマップを表示します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[ポイントした時ミニマップを表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MINIMAP\_HOVER (4044)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4044);

#### \[VBScript\]

> editor.ExecuteCommandByID 4044
