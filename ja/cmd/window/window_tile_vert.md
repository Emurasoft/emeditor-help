# \[左右に並べて表示\] コマンド

## 概要

ウィンドウを左右に並べて表示します。

## 説明

ウィンドウを左右に並べて表示します。

## 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[左右に並べて表示\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[左右に並べて表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_TILE_VERT (4243)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4243);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4243
```
