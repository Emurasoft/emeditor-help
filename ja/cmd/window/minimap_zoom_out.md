# \[ミニマップを縮小\] コマンド

### 概要

> ミニマップを拡大します。

### 説明

> ミニマップを拡大します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[縮小\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MINIMAP\_ZOOM\_OUT (3977)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3977);

#### \[VBScript\]

> editor.ExecuteCommandByID 3977
