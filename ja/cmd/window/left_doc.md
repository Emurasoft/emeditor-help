# \[左の文書\] コマンド

## 概要

左の文書を表示します。

## 説明

左の文書を表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[文書ナビゲーション\] \- \[左の文書\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+F6

## プラグイン コマンド ID

```
EEID_LEFT_DOC (4548)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4548);
```

### \[VBScript\]

```
