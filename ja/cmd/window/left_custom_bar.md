# \[左カスタム バー\] コマンド

## 概要

左のカスタム バーに切り替えます。

## 説明

利用可能なら左のカスタム バーに切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[カスタム バー\] \- \[左\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_FOCUS_LEFT_BAR (4425)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4425);
```

### \[VBScript\]

```
editor.ExecuteCommandByID(4425)
```
