# \[元の位置に戻す\] コマンド

### 概要

> このウィンドウを以前保存した位置に戻します。

### 説明

> このウィンドウを以前保存した位置に戻します。あらかじめ、 [**\[カスタマイズ\]** ダイアログ ボックス](../../dlg/customize/index) - [**\[ウィンドウ\]** タブ](../../dlg/customize/window/index) の **\[現在のウィンドウ位置を保存\]** ボタンをクリックして、ウィンドウ位置が保存されている場合に、このコマンドが有効になります。タブの有無に関係なく、このコマンドは利用できます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[元に戻す\] \- \[ウィンドウを元の位置に戻す\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl + 9

### プラグイン コマンド ID

- EEID\_RESTORE\_POS (4406)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4406);

#### \[VBScript\]

> editor.ExecuteCommandByID 4406
