# \[ミニマップを拡大\] コマンド

### 概要

> ミニマップを拡大します。

### 説明

> ミニマップを拡大します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[拡大\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MINIMAP\_ZOOM\_IN (3976)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3976);

#### \[VBScript\]

> editor.ExecuteCommandByID 3976
