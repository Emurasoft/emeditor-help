# \[次のペインまたはカスタム バー\] コマンド

### 概要

> 次の分割されたペインまたはカスタム バーに切り替えます。

### 説明

> ウィンドウが分割されている場合、次の分割されたペインに切り替えます。または、次のカスタム バーに切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[ペイン\] \- \[次のペインまたはカスタム バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_NEXT\_PANE\_OR\_BAR (4552)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4552);

#### \[VBScript\]

> editor.ExecuteCommandByID 4552
