# \[最後にアクセスした文書\] コマンド

### 概要

> 最後にアクセスした文書に切り替えます。

### 説明

> 最後にアクセスした文書に切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[文書ナビゲーション\] \- \[最後にアクセスした文書\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_NEXT\_ZORDER\_DOC (4544)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4544);

#### \[VBScript\]

> editor.ExecuteCommandByID 4544
