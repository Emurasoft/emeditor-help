# \[すべて最小化\] コマンド

### 概要

> ウィンドウをすべて最小化します。

### 説明

> ウィンドウをすべて最小化します。

### 実行方法

- 既定のメニュー: \[ウィンドウ\] \- \[すべて最小化\]
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[すべて最小化\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_MINIMIZE\_ALL (4244)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4244);

#### \[VBScript\]

> editor.ExecuteCommandByID 4244
