# \[右の文書\] コマンド

## 概要

右の文書を表示します。

## 説明

右の文書を表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[文書ナビゲーション\] \- \[右の文書\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F6

## プラグイン コマンド ID

```
EEID_RIGHT_DOC (4547)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4547);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4547
```
