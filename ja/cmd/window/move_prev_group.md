# \[前のグループに移動\] コマンド

### 概要

> 現在の文書を前のグループに移動して結合します。

### 説明

> 現在の文書を前のグループに移動して結合します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[前のグループに移動\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MOVE\_PREV\_GROUP (4396)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4396);

#### \[VBScript\]

> editor.ExecuteCommandByID 4396
