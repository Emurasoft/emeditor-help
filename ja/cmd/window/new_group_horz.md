# \[新しい上下グループ\] コマンド

## 概要

現在の文書を結合されたウィンドウから切り離して新しいグループにし、ウィンドウを左右に並べて表示します。

## 説明

現在の文書を結合されたウィンドウから切り離して新しいグループにし、ウィンドウを左右に並べて表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[新しい上下グループ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
