# \[下カスタム バー\] コマンド

### 概要

> 下のカスタム バーに切り替えます。

### 説明

> 利用可能なら下のカスタム バーに切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[カスタム バー\] \- \[下\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FOCUS\_RIGHT\_BAR (4428)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4428);

#### \[VBScript\]

> EEID\_FOCUS\_BOTTOM\_BAR (4428)
