# \[下カスタム バー\] コマンド

## 概要

下のカスタム バーに切り替えます。

## 説明

利用可能なら下のカスタム バーに切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[カスタム バー\] \- \[下\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_FOCUS_RIGHT_BAR (4428)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4428);
```

### \[VBScript\]

```
EEID\FOCUS\BOTTOM\BAR (4428)
```
