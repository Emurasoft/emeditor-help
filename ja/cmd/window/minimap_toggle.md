# \[ミニマップの表示/非表示を切り替え\] コマンド

### 概要

> ミニマップの表示/非表示を切り替えます。

### 説明

> ミニマップの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[ミニマップを表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MINIMAP\_TOGGLE (3975)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3975);

#### \[VBScript\]

> editor.ExecuteCommandByID 3975
