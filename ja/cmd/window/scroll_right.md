# \[右へスクロール\] コマンド

### 概要

> 文書を一文字ずつ右へスクロールします。

### 説明

> 文書を一文字ずつ右へスクロールします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[右\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_SCROLL\_RIGHT (4475)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4475);

#### \[VBScript\]

> editor.ExecuteCommandByID 4475
