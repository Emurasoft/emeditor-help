# \[垂直スクロール バー上のマーカーの表示/非表示を切り替え\] コマンド

### 概要

> 垂直スクロール バー上のマーカーの表示/非表示を切り替えます。

### 説明

> 垂直スクロール バー上のマーカーの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[マーカーを表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SCROLL\_MARKS\_TOGGLE (3989)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3989);

#### \[VBScript\]

> editor.ExecuteCommandByID 3989
