# \[左右分割の切り替え\] コマンド

### 概要

> ウィンドウの左右分割の表示/非表示を切り替えます。

### 説明

> ウィンドウが上下に分割されているか、まったく分割されていない場合、このコマンドを実行すると、ウィンドウを中央で左右に分割にして、すぐに分割位置を固定します。
> ウィンドウが既に左右に分割されている場合は、左右の分割を解除します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[分割\] \- \[左右分割の切り替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_SPLIT\_VERT\_TOGGLE (4386)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4386);

#### \[VBScript\]

> editor.ExecuteCommandByID 4386
