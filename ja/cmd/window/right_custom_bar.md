# \[右カスタム バー\] コマンド

## 概要

右のカスタム バーに切り替えます。

## 説明

利用可能なら右のカスタム バーに切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[カスタム バー\] \- \[右\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_FOCUS_RIGHT_BAR (4427)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4427);
```

### \[VBScript\]

```
editor.ExecuteCommandByID(4427)
```
