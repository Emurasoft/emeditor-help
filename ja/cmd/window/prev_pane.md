# \[前のペイン\] コマンド

### 概要

> 前の分割されたペインに切り替えます。

### 説明

> ウィンドウが分割されている場合、前の分割されたペインに切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[ペイン\] \- \[前のペイン\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Shift+F6

### プラグイン コマンド ID

- EEID\_PREV\_PANE (4215)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4215);

#### \[VBScript\]

> editor.ExecuteCommandByID 4215
