# \[タブを有効にする \- オフ\] コマンド

## 概要

タブを無効にしてすべてのウィンドウの結合を解除します。

## 説明

タブを無効にしてすべてのウィンドウの結合を解除します。すべてのウィンドウの結合が解除されると、文書毎にウィンドウが表示され、Windows のタスク バーには文書の数だけアイコンが表示されます。タブは表示されなくなります。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[タブを有効にする \- オフ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_COMBINE (4408)```

## マクロ

## \[JavaScript\]

```
editor.EnableTab = false;
```

## \[VBScript\]

```
editor.EnableTab = False
```
