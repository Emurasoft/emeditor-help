# \[文書の一覧\] コマンド

## 概要

指定する文書をアクティブにして表示します (複数項目)。

## 説明

指定する文書をアクティブにして表示します。もし指定する文書が最小化されている場合は、通常のサイズに戻します。

## 実行方法

- 既定のメニュー: \[ウィンドウ\] \- (ファイル名)
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- (ファイル名)
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_MENU から EEID_WINDOW_MENU + 255 まで (5376 から 5376 + 255 )```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(5376 + i);  // i は 0 から 255 までの整数
```

### \[VBScript\]

```
editor.ExecuteCommandByID 5376 + i   ' i は 0 から 255 までの整数
```
