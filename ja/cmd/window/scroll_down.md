# \[1行下へスクロール\] コマンド

## 概要

1行下へスクロールします。

## 説明

1 行下へスクロールします。ただし、プロパティの [**\[スクロール\]** ページ](../../dlg/properties/scroll/index) で **\[2行スクロール\]** チェック ボックスがチェックされている場合、2 行下へスクロールします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[1行下\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Down

## プラグイン コマンド ID

```
EEID_SCROLL_DOWN (4171)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4171);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4171
```
