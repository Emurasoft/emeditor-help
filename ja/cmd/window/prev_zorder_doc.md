# \[最もアクセスしていない文書\] コマンド

### 概要

> 最もアクセスしていない文書に切り替えます。

### 説明

> 最もアクセスしていない文書に切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[文書ナビゲーション\] \- \[最もアクセスしていない文書\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_PREV\_ZORDER\_DOC (4545)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4545);

#### \[VBScript\]

> editor.ExecuteCommandByID 4545
