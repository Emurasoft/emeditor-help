# \[新しい左右グループ\] コマンド

## 概要

現在の文書を結合されたウィンドウから切り離して新しいグループにし、ウィンドウを上下に並べて表示します。

## 説明

現在の文書を結合されたウィンドウから切り離して新しいグループにし、ウィンドウを上下に並べて表示します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[新しい左右グループ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_NEW_GROUP_VERT (4395)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4395);
