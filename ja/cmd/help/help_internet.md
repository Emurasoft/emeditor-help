# \[Web ヘルプ\] コマンド

### 概要

> Web ヘルプを表示します。

### 説明

> Web ヘルプを表示します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[Web ヘルプ\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[Web ヘルプ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HELP\_INTERNET (3932)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3932);

#### \[VBScript\]

> editor.ExecuteCommandByID 3932
