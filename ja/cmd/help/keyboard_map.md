# \[キーボード マップ\] コマンド

### 概要

> キーボード コマンド一覧を表示します。

### 説明

> キーボード コマンド一覧を表示します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[キーボード マップ\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[キーボード マップ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_KEYBOARD\_MAP (4298)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4298);

#### \[VBScript\]

> editor.ExecuteCommandByID 4298
