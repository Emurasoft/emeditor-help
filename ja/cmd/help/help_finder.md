# \[トピックの検索\] コマンド

## 概要

ヘルプ トピックを検索します。

## 説明

EmEditor のヘルプ トピックを表示します。

## 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[トピックの検索\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[トピックの検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F1

## プラグイン コマンド ID

```
EEID_HELP_FINDER (4247)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4247
```
