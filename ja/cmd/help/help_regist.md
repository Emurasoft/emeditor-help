# \[購入方法\] コマンド

### 概要

> 購入方法を表示します。

### 説明

> EmEditor の購入方法を表示します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[購入方法\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[購入方法\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HELP\_REGIST (4248)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4248);

#### \[VBScript\]

> editor.ExecuteCommandByID 4248
