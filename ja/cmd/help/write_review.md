# \[レビューを書く\] コマンド

### 概要

> EmEditorについてのレビューを書きます。

### 説明

> EmEditorについてのレビューを書きます。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[レビューを書く\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[レビューを書く\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WRITE\_REVIEW (4059)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4059);

#### \[VBScript\]

> editor.ExecuteCommandByID 4059
