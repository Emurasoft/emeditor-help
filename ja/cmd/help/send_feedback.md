# \[フィードバックを送信\] コマンド

### 概要

> EmEditor開発チームにフィードバックを送信します。

### 説明

> EmEditor開発チームにフィードバックを送信します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[フィードバックを送信\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[フィードバックを送信\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SEND\_FEEDBACK (4058)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4058);

#### \[VBScript\]

> editor.ExecuteCommandByID 4058
