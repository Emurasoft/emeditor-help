# \[商標を挿入\] コマンド

### 概要

> 商標を挿入します。

### 説明

> カーソル位置に商標文字 ™ を挿入します。

### 実行方法

- 既定のメニュー: \[挿入\] \- \[記号\] \- \[商標\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[記号\] \- \[商標\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Ctrl+T

### プラグイン コマンド ID

- EEID\_INSERT\_TRADEMARK (4316)

### マクロ

#### \[JavaScript\]

> document.selection.Text="™";

#### \[VBScript\]

> document.selection.Text="™"
