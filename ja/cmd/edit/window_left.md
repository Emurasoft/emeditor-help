# \[ウィンドウの左へ\] コマンド

## 概要

カーソルをウィンドウの左端へ移動します。

## 説明

カーソル位置をウィンドウの左端へ移動します。水平スクロールの位置は変更しません。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[ウィンドウ左\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_WINDOW_LEFT (4295)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4295);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4295
```
