# \[ステップの繰り返し\] コマンド

## 概要

指定した回数だけ指定した最後のステップを繰り返します。

## 説明

[\[繰り返し\] ダイアログ ボックス](../../dlg/repeat_count/index) を表示して、指定した回数だけ指定した最後のステップを繰り返します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[ステップの繰り返し\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[やり直し\] \- \[ステップの繰り返し\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_REPEAT_STEPS (4075)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4075);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4075
```
