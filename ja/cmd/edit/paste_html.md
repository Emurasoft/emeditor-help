# \[HTMLで貼り付け\] コマンド

## 概要

HTMLフォーマットで貼り付けます。

## 説明

カーソル位置にクリップボード内のテキストをHTMLで貼り付けます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[貼り付け\] \- \[HTMLで貼り付け\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_EDIT_PASTE_HTML (4071)```

## マクロ

## \[JavaScript\]

```
document.selection.Paste(eeCopyHTML);
```

## \[VBScript\]

```
document.selection.Paste eeCopyHTML
```
