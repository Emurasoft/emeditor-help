# \[日時が新しいから古いへ並べ替え\] コマンド

### 概要

> 現在の列の日時を降順で並べ替えます。

### 説明

> 現在の列の日時を降順で並べ替えます。

### 実行方法

- 既定のメニュー: \[並べ替え\] \- \[日時が新しいから古いへ並べ替え\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[日時が新しいから古いへ並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_DATE\_D (3973)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3973);

#### \[VBScript\]

> editor.ExecuteCommandByID 3973
