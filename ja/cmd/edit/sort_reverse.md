# \[逆順に並べ替え\] コマンド

### 概要

> 現在の列を逆順に並べ替えます。

### 説明

> 現在の列を逆順に並べ替えます。

### 実行方法

- 既定のメニュー: \[並べ替え\] \- \[逆順に並べ替え\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[逆順に並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_REVERSE (4039)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4039);

#### \[VBScript\]

> editor.ExecuteCommandByID 4039
