# \[列のコピー\] コマンド

## 概要

選択列または現在列をクリップボードにコピーします。

## 説明

選択列または現在列をクリップボードにコピーします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[コピー\] \- \[列のコピー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
