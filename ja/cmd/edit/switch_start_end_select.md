# \[選択開始位置と終了位置を切り替える\] コマンド

### 概要

> 選択開始位置と終了位置を切り替えます。

### 説明

> 選択開始位置と終了位置を切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択モード\] \-
\[選択開始位置と終了位置を切り替える\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Shift + F8

### プラグイン コマンド ID

- EEID\_SWITCH\_START\_END\_SELECT (3850)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3850);

#### \[VBScript\]

> editor.ExecuteCommandByID 3850
