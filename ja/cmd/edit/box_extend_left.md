# \[箱型選択範囲を1文字左へ\] コマンド

## 概要

箱型選択範囲を1文字左へ移動します。

## 説明

箱型選択範囲を1文字左へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[箱型選択範囲を1文字左へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: ALT+CTRL+LEFT

## プラグイン コマンド ID

```
EEID_BOX_EXTEND_LEFT (4026)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4026);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4026
```
