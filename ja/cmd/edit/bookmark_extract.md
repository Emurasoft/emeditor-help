# \[この文書のブックマークされた行を抽出して新規作成\] コマンド

## 概要

この文書のすべてのブックマークされた行を抽出して新規ファイルにコピーします。

## 説明

この文書のすべてのブックマークされた行を抽出して新規ファイルにコピーします。

- ステータス バー: なし
- 既定のショートカット: なし
