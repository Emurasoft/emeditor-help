# \[下のセル (セル選択モードのみ)\] コマンド

### 概要

> CSV文書で下のセルにジャンプします。

### 説明

> CSV文書で下のセルにジャンプします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[下のセル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CSV\_DOWN (4080)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4080);

#### \[VBScript\]

> editor.ExecuteCommandByID 4080
