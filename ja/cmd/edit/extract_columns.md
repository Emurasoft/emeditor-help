# \[列を抽出\] コマンド

### 概要

> 現在のCSV文書から指定する列を抽出します。

### 説明

> 現在のCSV文書から指定する列を抽出します。

### 実行方法

- 既定のメニュー: \[CSV\] - \[高度\] - \[抽出\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[高度\] - \[抽出\]
- ツール バー: ![](../../images/extract_columns.png)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_EXTRACT\_COLUMNS (3930)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3930);

#### \[VBScript\]

> editor.ExecuteCommandByID 3930
