# \[文末へ\] コマンド

### 概要

> 文末へ移動します。

### 説明

> カーソル位置を文末へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[文末へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+End

### プラグイン コマンド ID

- EEID\_BOTTOM (4169)

### マクロ

#### \[JavaScript\]

> document.selection.EndOfDocument(false);

#### \[VBScript\]

> document.selection.EndOfDocument false
