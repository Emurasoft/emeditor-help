# \[行選択を開始\] コマンド

## 概要

行選択を開始します。

## 説明

行選択を開始します。このコマンドを実行すると、行選択モードになり、キーボードによってカーソルを移動すると、選択範囲が行単位で変更されます。選択範囲を決定後、 [\[コピー\] コマンド](edit_copy)、または [\[切り取り\] コマンド](edit_cut) を実行すると、行選択モードが解除されます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[行選択を開始\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F8

## プラグイン コマンド ID

```
EEID_SELECT_LINE (4154)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4154);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4154
```
