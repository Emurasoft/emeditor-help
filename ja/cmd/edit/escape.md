# \[選択のキャンセル\] コマンド

### 概要

> 選択状態をキャンセルします。

### 説明

> 選択状態を解除し、選択されていない状態にします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[選択のキャンセル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Esc

### プラグイン コマンド ID

- EEID\_ESCAPE (4187)

### マクロ

#### \[JavaScript\]

> document.selection.Collapse();

#### \[VBScript\]

> document.selection.Collapse
