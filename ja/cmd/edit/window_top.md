# \[ウィンドウの上へ\] コマンド

### 概要

> カーソルをウィンドウの一番上へ移動します。

### 説明

> カーソルをウィンドウの一番上へ移動します。垂直スクロール バーの位置は移動しません。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[ウィンドウ上\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Page Up

### プラグイン コマンド ID

- EEID\_WINDOW\_TOP (4292)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4292);

#### \[VBScript\]

> editor.ExecuteCommandByID 4292
