# \[セル編集 (セル選択モードのみ)\] コマンド

### 概要

> CSV文書の現在のセルを編集します。

### 説明

> CSV文書の現在のセルを編集します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[セル編集\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CSV\_EDIT\_CELL (3960)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3960);

#### \[VBScript\]

> editor.ExecuteCommandByID 3960
