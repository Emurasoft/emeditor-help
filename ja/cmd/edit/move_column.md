# \[列を移動/コピー\] コマンド

## 概要

CSV文書の選択列または現在の列を指定する場所に移動またはコピーします。

## 説明

このコマンドを実行すると、 [\[列を移動/コピー\] ダイアログ ボックス](../../dlg/move_column/index) が表示され、CSV文書の選択列または現在の列を指定する場所に移動またはコピーします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列を移動/コピー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MOVE_COLUMN (4046)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4046);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4046
```
