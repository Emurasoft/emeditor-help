# \[戻る\] コマンド

## 概要

カーソルを以前の位置に移動します。

## 説明

カーソルを以前の位置に移動します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[高度な操作\] \- \[戻る\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[戻る\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+←

## プラグイン コマンド ID

```
EEID_CARET_BACK (4597)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4597);
```

```
