# \[列幅をリセット\] コマンド

## 概要

CSV文書の選択列の幅をリセットします。

## 説明

CSV文書の選択列の幅をリセットします。

## 実行方法

- 既定のメニュー: \[CSV\] - \[列\] - \[列幅をリセット\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列幅をリセット\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_RESET_COLUMN_WIDTH (4052)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4052);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4052
```
