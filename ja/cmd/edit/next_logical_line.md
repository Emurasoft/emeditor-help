# \[次の論理行\] コマンド

## 概要

次の論理行へ移動します。

## 説明

次の論理行の行頭へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[次の論理行\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Page Down

## プラグイン コマンド ID

```
EEID_NEXT_LOGICAL_LINE (4268)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4268);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4268
```
