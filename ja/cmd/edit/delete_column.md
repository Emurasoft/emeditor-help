# \[列の削除\] コマンド

### 概要

> 選択列または現在列を削除します。

### 説明

> 選択列または現在列を削除します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[削除\] \- \[列の削除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_DELETE\_COLULMN (3962)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3962);

#### \[VBScript\]

> editor.ExecuteCommandByID 3962
