# \[列の切り取り\] コマンド

### 概要

> 選択列または現在列を切り取ってクリップボードに保存します。

### 説明

> 選択列または現在列を切り取ってクリップボードに保存します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[切り取り\] \- \[列の切り取り\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CUT\_COLULMN (3963)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3963);

#### \[VBScript\]

> editor.ExecuteCommandByID 3963
