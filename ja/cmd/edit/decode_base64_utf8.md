# \[Base64をプレーン テキストに変換(UTF-8)\] コマンド

### 概要

> 選択テキストをUTF-8を用いて Base64 エンコーディングからプレーン テキストにデコードします。

### 説明

> 選択テキストをUTF-8を用いて Base64 エンコーディングからプレーン テキストにデコードします。

### 実行方法

- 既定のメニュー: \[変換\] \- \[選択範囲のエンコード/デコード\] \- \[Base64をプレーン テキストに変換(UTF-8)\]
- [すべてのコマンド](../../glossary/allcommands): \[変換\] \- \[選択範囲のエンコード/デコード\] \- \[Base64をプレーン テキストに変換(UTF-8)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_DECODE\_BASE64\_UTF8 (3941)
