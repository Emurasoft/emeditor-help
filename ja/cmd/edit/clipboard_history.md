# \[クリップボードの履歴を表示\] コマンド

## 概要

クリップボードの履歴を表示します。

## 説明

クリップボードの履歴を表示します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[クリップボードの履歴を表示\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[貼り付け\] \- \[クリップボードの履歴を表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+V, Ctrl+Shift+Insert

## プラグイン コマンド ID

```
EEID_CLIPBOARD_HISTORY (4045)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4045);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4045
```
