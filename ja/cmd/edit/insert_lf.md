# \[LFのみ挿入\] コマンド

## 概要

カーソル位置に LF のみを挿入します。

## 説明

カーソル位置に LF (改行) のみを挿入します。EmEditor では、CR と LF の混じった文書が編集できます。Enter
を入力すると、その行で使用されている改行コードと同じ改行コードを CR、LF、または CR+LF
のいずれかから自動的に選択されて挿入されます。このコマンドを実行すると、その行で使用されている改行コードによらずに、常に LF のみを挿入します。

## 実行方法

- 既定のメニュー: \[挿入\] \- \[LFのみ\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[LFのみ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_INSERT_LF (4146)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4146);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4146
```
