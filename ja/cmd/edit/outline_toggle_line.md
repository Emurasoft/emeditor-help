# \[アウトラインを展開/縮小\] コマンド

## 概要

アウトライン表示時、現在行の展開/縮小を切り替えます。

## 説明

アウトライン表示時、現在行が縮小状態なら展開し、展開された状態なら縮小状態にします。アウトライン表示中に、行の左端に表示される \[+\] または \[-\] のボタンをクリックするのと同じ意味です。アウトラインを表示するには、アウトライン用のプラグインを実行する必要があります。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[アウトライン\] \- \[アウトラインを展開/縮小\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_OUTLINE_TOGGLE_LINE (4412)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4412);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4412
```
