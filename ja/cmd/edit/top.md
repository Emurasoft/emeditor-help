# \[文頭へ\] コマンド

## 概要

文頭へ移動します。

## 説明

カーソル位置を文頭へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[文頭へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Home

## プラグイン コマンド ID

```
EEID_TOP (4168)```

## マクロ

### \[JavaScript\]

```
document.selection.StartOfDocument(false);
```

### \[VBScript\]

```
document.selection.StartOfDocument false
```
