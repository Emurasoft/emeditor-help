# \[選択範囲を1行上へ\] コマンド

## 概要

選択範囲を 1 行上へ移動します。

## 説明

選択範囲を 1 行上へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[選択範囲を1行上へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Shift+Up

## プラグイン コマンド ID

```
EEID_SHIFT_UP (4176)```

## マクロ

### \[JavaScript\]

```
document.selection.LineUp(true,1);
```

### \[VBScript\]

```
document.selection.LineUp true,1
```
