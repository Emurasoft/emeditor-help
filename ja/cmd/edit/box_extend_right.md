# \[箱型選択範囲を1文字右へ\] コマンド

### 概要

> 箱型選択範囲を1文字右へ移動します。

### 説明

> 箱型選択範囲を1文字右へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[箱型選択範囲を1文字右へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: ALT+CTRL+RIGHT

### プラグイン コマンド ID

- EEID\_BOX\_EXTEND\_RIGHT (4025)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4025);

#### \[VBScript\]

> editor.ExecuteCommandByID 4025
