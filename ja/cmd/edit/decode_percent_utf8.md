# \[パーセント エンコーディングをUnicodeに変換(UTF-8)\] コマンド

### 概要

> 選択テキストを現在のエンコードを用いてパーセント エンコーディングからUnicodeにデコードします。

### 説明

> 選択テキストを現在のエンコードを用いてパーセント エンコーディングからUnicodeにデコードします。

### 実行方法

- 既定のメニュー: \[変換\] \- \[選択範囲のエンコード/デコード\] \- \[パーセント エンコーディングをUnicodeに変換(UTF-8)\]
- [すべてのコマンド](../../glossary/allcommands): \[変換\] \- \[選択範囲のエンコード/デコード\] \- \[パーセント エンコーディングをUnicodeに変換(UTF-8)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
