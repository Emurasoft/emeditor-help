# \[マーカーに追加/削除\] コマンド

## 概要

マーカー一覧に選択テキストを追加するか、マーカー一覧から選択テキストを削除します。

## 説明

マーカー一覧に選択テキストを追加するか、マーカー一覧から選択テキストを削除します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[マーカーに追加/削除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

EEID_ADD_REMOVE_MARKERS (4591)```
