# \[マクロンを挿入\] コマンド

## 概要

文字を続けてマクロン付文字を挿入します。

## 説明

カーソル位置に a, e, i, o, u, A, E, I, O, U を続けてマクロン付文字 ā, ē, ī, ō, ū, Ā, Ē, Ī, Ō, Ū
を挿入します。

## 実行方法

- 既定のメニュー: \[挿入\] \- \[発音区別符号\] \- \[マクロン\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[発音区別符号\] \- \[マクロン\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt + -

## プラグイン コマンド ID

```
EEID_INSERT_MACRON (3861)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3861);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3861
```
