# \[箱型選択範囲を1文字下へ\] コマンド

### 概要

> 箱型選択範囲を1文字下へ移動します。

### 説明

> 箱型選択範囲を1文字下へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[箱型選択範囲を1文字下へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: ALT+CTRL+DOWN

### プラグイン コマンド ID

- EEID\_BOX\_EXTEND\_DOWN (4028)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4028);

#### \[VBScript\]

> editor.ExecuteCommandByID 4028
