# \[コピーして選択解除\] コマンド

### 概要

> 選択範囲をコピーしてクリップボードに保存して選択を解除します。

### 説明

> 選択されたテキストをコピーして、その内容をクリップボードに保存し、さらに選択範囲を解除します。このコマンドは、 [\[コピー\] \
> コマンド](edit_copy) の後、 [\[選択のキャンセル\] コマンド](escape) を実行することと等価です。この操作の後、別の場所にカーソルを移動して
> [\[貼り付け\] コマンド](edit_paste) を実行することにより、コピーしたテキストを他の場所にコピーすることができます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[コピー\] \- \[コピーして選択解除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_EDIT\_COPY\_DESELECT (4128)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4128);

#### \[VBScript\]

> editor.ExecuteCommandByID 4128
