# \[右端へ (セル選択モードのみ)\] コマンド

### 概要

> 現在のデータ領域の右端へ移動します。

### 説明

> 現在のデータ領域の右端へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[右端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Right (セル選択モードのみ)

### プラグイン コマンド ID

- EEID\_RIGHT\_EDGE (4017)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4017);

#### \[VBScript\]

> editor.ExecuteCommandByID 4017
