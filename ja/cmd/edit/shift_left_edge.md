# \[選択範囲を左端へ (セル選択モードのみ)\] コマンド

### 概要

> 選択範囲を現在のデータ領域の左端へ移動します。

### 説明

> 選択範囲を現在のデータ領域の左端へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[選択範囲を左端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Left (セル選択モードのみ)

### プラグイン コマンド ID

- EEID\_SHIFT\_LEFT\_EDGE (4022)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4022);

#### \[VBScript\]

> editor.ExecuteCommandByID 4022
