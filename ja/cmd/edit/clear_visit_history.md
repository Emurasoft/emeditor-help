# \[表示済みリンクの履歴をクリア\] コマンド

### 概要

> 表示済みURL、メール アドレス、タグの履歴をクリアします。

### 説明

> 表示済みURL、メール アドレス、タグの履歴をクリアします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[表示済みリンクの履歴をクリア\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CLEAR\_VISIT\_HISTORY (3947)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3947);

#### \[VBScript\]

> editor.ExecuteCommandByID 3947
