# \[行頭へ\] コマンド

### 概要

> 行頭へ移動します。

### 説明

> カーソル位置を表示行の行頭へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[行頭へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Ctrl+Home

### プラグイン コマンド ID

- EEID\_HOME (4164)

### マクロ

#### \[JavaScript\]

> document.selection.StartOfLine(false,eeLineView);

#### \[VBScript\]

> document.selection.StartOfLine false,eeLineView
