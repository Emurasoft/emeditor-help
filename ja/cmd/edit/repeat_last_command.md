# \[最後のコマンドを繰り返す\] コマンド

## 概要

最後に実行したコマンドを繰り返します。

## 説明

最後に実行したコマンドを繰り返します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[やり直し\] \- \[最後のコマンドを繰り返す\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: CTRL+Shift+G

## プラグイン コマンド ID

```
EEID_REPEAT_LAST_COMMAND (3846)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID (3846);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3846
```
