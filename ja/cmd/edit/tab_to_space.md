# \[文書全体のタブを空白に変換\] コマンド

## 概要

文書全体のタブを空白に変換します。

## 説明

文書全体のタブを空白に変換します。タブの桁数については、プロパティの [\[基本\] ページ](../../dlg/properties/general/index) の

\[タブ/インデント\] ボタンをクリックし、 [\[タブ/インデント\] ダイアログ ボックス](../../dlg/properties/general/indent/index) で設定できます。このコマンドは、文書全体を選択してから、 [\[選択範囲のタブを空白に変換\] コマンド](../convert/untabify) を実行することと等価ですが、選択範囲は変更されません。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[文書全体のタブを空白に変換\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_TAB_TO_SPACE (4253)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4253);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4253
```
