# \[オートフィル/オート リピート\] コマンド

### 概要

> 連番やパターンを隣のセルに続けます。

### 説明

> 連番やパターンを隣のセルに続けます。

### 実行方法

- 既定のメニュー: \[挿入\] \- \[オートフィル/オート リピート\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[オートフィル/オート リピート\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_AUTO\_FILL (3992)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3992);

#### \[VBScript\]

> editor.ExecuteCommandByID 3992
