# \[逆立ち疑問符を挿入\] コマンド

## 概要

逆立ち疑問符を挿入します。

## 説明

カーソル位置に逆立ち疑問符 ¿ を挿入します。

## 実行方法

- 既定のメニュー: \[挿入\] \- \[記号\] \- \[逆立ち疑問符\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[記号\] \- \[逆立ち疑問符\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Ctrl+Shift+/

## プラグイン コマンド ID

```
EEID_INSERT_INVERTED_QUESTION (4312)```

## マクロ

## \[JavaScript\]

```
document.selection.Text="¿";
```

## \[VBScript\]

```
document.selection.Text="¿"
```
