# \[タブの左移動/逆インデント\] コマンド

### 概要

> 選択されていなければタブの左移動を行い、選択されていれば行インデントを少なくします。

### 説明

> 複数行が選択されていなければカーソル位置をタブの文字数分だけ左に移動します。複数行が選択されていれば、その行の最初の 1
> 文字のタブ文字または相当数の空白を削除して行インデントを少なくします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[タブの左移動/逆インデント\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SHIFT\_TAB (4189)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4189);

#### \[VBScript\]

> editor.ExecuteCommandByID 4189
