# \[1行上へ\] コマンド

## 概要

1 行上へ移動します。

## 説明

カーソル位置を 1 行上へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[1行上へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Up

## プラグイン コマンド ID

```
EEID_UP (4160)```

## マクロ

### \[JavaScript\]

```
document.selection.LineUp(false,1);
```

### \[VBScript\]

```
document.selection.LineUp false,1
```
