# \[前のノードへ\] コマンド

## 概要

アウトライン表示時、前のノードへ移動します。

## 説明

アウトライン表示時、前のノードへ移動します。アウトラインを表示するには、アウトライン用のプラグインを実行する必要があります。

## 実行方法

- 既定のメニュー: \[編集\] \- \[アウトライン\] \- \[前のノードへ\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[アウトライン\] \- \[前のノードへ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Up

## プラグイン コマンド ID

```
EEID_OUTLINE_PREV_NODE (4416)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4416);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4416
```
