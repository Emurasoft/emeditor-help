# \[選択範囲を右端へ (セル選択モードのみ)\] コマンド

### 概要

> 選択範囲を現在のデータ領域の右端へ移動します。

### 説明

> 選択範囲を現在のデータ領域の右端へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[選択範囲を右端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Right (セル選択モードのみ)

### プラグイン コマンド ID

- EEID\_SHIFT\_RIGHT\_EDGE (4021)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4021);

#### \[VBScript\]

> editor.ExecuteCommandByID 4021
