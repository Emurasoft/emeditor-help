# \[行末の空白を削除\] コマンド

### 概要

> 全テキストの行末の空白を削除します。

### 説明

> 開いている文書全体の論理行末の空白を削除します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[削除\] \- \[行末の空白を削除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_TRIM\_RIGHT (4278)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4278);

#### \[VBScript\]

> editor.ExecuteCommandByID 4278
