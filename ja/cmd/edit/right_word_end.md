# \[単語の右端へ\] コマンド

### 概要

> 単語の右端へ移動します。

### 説明

> 単語の右端へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[単語の右端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Right

### プラグイン コマンド ID

- EEID\_RIGHT\_WORD\_END (4030)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4030);

#### \[VBScript\]

> editor.ExecuteCommandByID 4030
