# \[行の最初を選択\] コマンド

## 概要

現在選択されている範囲の行の最初のみを選択します。

## 説明

現在選択されている範囲の行の最初のみを選択します。

## 実行方法

- 既定のメニュー: \[編集\] \- \[選択モード\] \- \[行の最初を選択\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択モード\] \-
\[行の最初を選択\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SELECT_BEGIN_LINES```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3848);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3848
```
