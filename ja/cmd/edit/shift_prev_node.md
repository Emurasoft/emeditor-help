# \[選択範囲を前のノードへ\] コマンド

### 概要

> アウトライン表示時、選択範囲を前のノードへ移動します。

### 説明

> アウトライン表示時、選択範囲を前のノードへ移動します。アウトラインを表示するには、アウトライン用のプラグインを実行する必要があります。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[アウトライン\] \- \[選択範囲を前のノードへ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Shift+Up

### プラグイン コマンド ID

- EEID\_SHIFT\_PREV\_NODE (4418)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4418);

#### \[VBScript\]

> editor.ExecuteCommandByID 4418
