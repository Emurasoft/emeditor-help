# \[左端へ (セル選択モードのみ)\] コマンド

## 概要

現在のデータ領域の左端へ移動します。

## 説明

現在のデータ領域の左端へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[左端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Left (セル選択モードのみ)

## プラグイン コマンド ID

```
EEID_LEFT_EDGE (4018)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4018);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4018
```
