# \[文書全体の空白をタブに変換\] コマンド

## 概要

文書全体の空白をタブに変換します。

## 説明

文書全体の行頭の空白をタブに変換します。タブの桁数については、プロパティの [\[基本\] ページ](../../dlg/properties/general/index) の

\[タブ/インデント\] ボタンをクリックし、 [\[タブ/インデント\] ダイアログ ボックス](../../dlg/properties/general/indent/index) で設定できます。このコマンドは、文書全体を選択してから、 [\[選択範囲の空白をタブに変換\] コマンド](../convert/tabify) を実行することと等価ですが、選択範囲は変更されません。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[文書の空白をTabに変換\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SPACE_TO_TAB (4355)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4355);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4355
```
