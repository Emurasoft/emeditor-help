# \[複数選択から文頭の選択を解除します\] コマンド

## 概要

複数選択から文頭の選択をキャンセルします。

## 説明

複数選択から文頭の選択をキャンセルします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[複数選択から文頭の選択を解除します\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Ctrl+Shift+上

## プラグイン コマンド ID

```
EEID_CANCEL_TOP_SEL (4030)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4030);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4030
```
