# \[自動コピー\] コマンド

### 概要

> 自動コピー モードのオン/オフを切り替えます。

### 説明

> 自動コピー モードのオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: \[編集\] \- \[自動コピー\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[コピー\] \- \[自動コピー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_AUTO\_COPY (3979)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID (3979);

#### \[VBScript\]

> editor.ExecuteCommandByID 3979
