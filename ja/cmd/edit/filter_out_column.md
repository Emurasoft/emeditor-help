# \[列でフィルター除外\] コマンド

## 概要

現在の文書を現在の列の選択テキストまたはカーソル位置の単語でフィルターを否定で設定します。

## 説明

現在の文書を現在の列の選択テキストまたはカーソル位置の単語でフィルターを否定で設定します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[列でフィルター除外\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
