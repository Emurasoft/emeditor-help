# \[選択範囲を上端へ (セル選択モードのみ)\] コマンド

## 概要

選択範囲を現在のデータ領域の上端へ移動します。

## 説明

選択範囲を現在のデータ領域の上端へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[選択範囲を上端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Up (セル選択モードのみ)

## プラグイン コマンド ID

```
EEID_SHIFT_TOP_EDGE (4023)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4023);
```
