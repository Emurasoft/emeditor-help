# \[上端へ (セル選択モードのみ)\] コマンド

## 概要

現在のデータ領域の上端へ移動します。

## 説明

現在のデータ領域の上端へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[上端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Up (セル選択モードのみ)

## プラグイン コマンド ID

```
EEID_TOP_EDGE (4019)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4019);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4019
```
