# \[1行下へ\] コマンド

## 概要

1 行下へ移動します。

## 説明

カーソル位置を 1 行下へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[1行下へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Down

## プラグイン コマンド ID

```
EEID_DOWN (4161)```

## マクロ

### \[JavaScript\]

```
document.selection.LineDown(false,1);
```

### \[VBScript\]

```
document.selection.LineDown false,1
```
