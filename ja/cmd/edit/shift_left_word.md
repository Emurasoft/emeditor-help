# \[選択範囲を1単語左へ\] コマンド

## 概要

選択範囲を 1 単語左へ移動します。

## 説明

選択範囲を 1
単語左へ移動します。現在のカーソル位置が単語の先頭の場合、左の単語の先頭に移動します。現在のカーソル位置が単語の途中の場合、その単語の先頭に移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[選択範囲を1単語左へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Left

## プラグイン コマンド ID

```
EEID_SHIFT_LEFT_WORD (4175)```

## マクロ

### \[JavaScript\]

```
document.selection.WordLeft(true,1);
```

### \[VBScript\]

```
document.selection.WordLeft true,1
```
