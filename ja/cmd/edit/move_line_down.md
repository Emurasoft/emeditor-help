# \[1行下へ移動\] コマンド

## 概要

選択行または現在の行を下へ1行移動します。

## 説明

選択行または現在の行を下へ1行移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[移動\] \- \[1行下へ移動\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Down

## プラグイン コマンド ID

```
ID_MOVE_LINE_DOWN (3859)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3859);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3859
```
