# \[選択範囲を単語の右端へ\] コマンド

## 概要

選択範囲を単語の右端へ移動します。

## 説明

選択範囲を単語の右端へ移動します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[選択範囲を単語の右端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Right

EEID_SHIFT_RIGHT_WORD_END (4031)```
