# \[日時が古いから新しいへ並べ替え\] コマンド

### 概要

> 現在の列の日時を昇順で並べ替えます。

### 説明

> 現在の列の日時を昇順で並べ替えます。

### 実行方法

- 既定のメニュー: \[並べ替え\] \- \[日時が古いから新しいへ並べ替え\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[日時が古いから新しいへ並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_DATE\_A (3972)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3972);

#### \[VBScript\]

> editor.ExecuteCommandByID 3972
