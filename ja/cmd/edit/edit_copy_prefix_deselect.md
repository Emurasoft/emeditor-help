# \[引用付きコピーして選択解除\] コマンド

## 概要

選択範囲を引用マークを付けてクリップボードに保存して選択を解除します。

## 説明

選択されたテキストを行の最初に引用マークを付けてコピーして、その内容をクリップボードに保存し、さらに選択範囲を解除します。このコマンドは、 [\[引用付きコピー\] \
コマンド](edit_copy_prefix) の後、 [\[選択のキャンセル\] コマンド](escape) を実行することと等価です。この操作の後、別の場所にカーソルを移動して
[\[貼り付け\] コマンド](edit_paste) を実行することにより、コピーしたテキストを他の場所にコピーすることができます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[コピー\] \- \[引用付きコピーして選択解除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Q

## プラグイン コマンド ID

```
EEID_EDIT_COPY_PREFIX_DESELECT (4131)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID (4131);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4131
```
