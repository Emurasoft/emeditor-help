# \[選択範囲を次のノードへ\] コマンド

## 概要

アウトライン表示時、選択範囲を次のノードへ移動します。

## 説明

アウトライン表示時、選択範囲を次のノードへ移動します。アウトラインを表示するには、アウトライン用のプラグインを実行する必要があります。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[アウトライン\] \- \[選択範囲を次のノードへ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Shift+Down

## プラグイン コマンド ID

```
EEID_SHIFT_NEXT_NODE (4417)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4417);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4417
```
