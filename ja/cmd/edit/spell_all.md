# \[すべてのスペル チェック\] コマンド

### 概要

> 文書全体のスペル チェックを行います。

### 説明

> 文書全体のスペル チェックを行います。

### 実行方法

- 既定のメニュー: \[編集\] \- \[スペル チェック\] \- \[スペル チェックを行う場所\] \- \[すべて\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[スペル チェック\] \- \[スペル チェックを行う場所\] \- \[すべて\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SPELL\_ALL (4522)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4522);

#### \[VBScript\]

> editor.ExecuteCommandByID 4522
