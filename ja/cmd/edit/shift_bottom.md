# \[選択範囲を文末へ\] コマンド

### 概要

> 選択範囲を文末へ移動します。

### 説明

> 選択範囲を文末へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[選択範囲の変更\] \- \[選択範囲を文末へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+End

### プラグイン コマンド ID

- EEID\_SHIFT\_BOTTOM (4185)

### マクロ

#### \[JavaScript\]

> document.selection.EndOfDocument(true);

#### \[VBScript\]

> document.selection.EndOfDocument true
