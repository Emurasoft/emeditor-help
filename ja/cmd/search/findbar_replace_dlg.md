# \[置換 >\> (検索ツール バー)\] コマンド

### 概要

> 検索ツール バーの \[置換 >>\] ボタンのオン/オフを切り替えます。

### 説明

> 検索ツール バーの \[置換 >>\] ボタンのオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バー\] \- \[置換 >>\]
- ツール バー: ![](../../images/find_replace_dlg.png) (検索ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FINDBAR\_REPLACE\_DLG (4579)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4579);

#### \[VBScript\]

> editor.ExecuteCommandByID 4579
