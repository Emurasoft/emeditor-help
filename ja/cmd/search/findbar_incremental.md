# \[インクリメンタル サーチ (検索ツール バー)\] コマンド

### 概要

> 検索ツール バーの \[インクリメンタル サーチ\] ボタンのオン/オフを切り替えます。

### 説明

> 検索ツール バーの \[インクリメンタル サーチ\] ボタンのオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バー\] \- \[インクリメンタル サーチ\]
- ツール バー: ![](../../images/find_incremental.png) (検索ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FINDBAR\_INCREMENTAL (4571)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4571);

#### \[VBScript\]

> editor.ExecuteCommandByID 4571
