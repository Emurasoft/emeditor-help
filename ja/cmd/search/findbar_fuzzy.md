# \[あいまい一致 (検索ツール バー)\] コマンド

### 概要

> 検索ツール バーの \[あいまい一致\] ボタンのオン/オフを切り替えます。

### 説明

> 検索ツール バーの \[あいまい一致\] ボタンのオン/オフを切り替えます。
