# \[次を置換\] コマンド

### 概要

> 次を置換します。

### 説明

> 前回検索した文字列を再び同一条件で下の方向に置換します。

### 実行方法

- 既定のメニュー: \[検索\] \- \[次を置換\]
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[次を置換\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_REPLACE\_NEXT (4447)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4447);

#### \[VBScript\]

> editor.ExecuteCommandByID 4447
