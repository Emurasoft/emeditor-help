# \[検索ツール バーにフォーカスを設定する\] コマンド

### 概要

> 検索ツール バーにフォーカスを設定します。

### 説明

> 検索ツール バーにフォーカスを設定します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バーにフォーカスを設定する\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F

### プラグイン コマンド ID

- EEID\_FOCUS\_FIND\_BAR (4565)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4565);

#### \[VBScript\]

> editor.ExecuteCommandByID 4565
