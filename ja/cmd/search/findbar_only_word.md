# \[単語のみ検索する (検索ツール バー)\] コマンド

## 概要

検索ツール バーの \[単語のみ検索する\] ボタンのオン/オフを切り替えます。

## 説明

検索ツール バーの \[単語のみ検索する\] ボタンのオン/オフを切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バー\] \- \[単語のみ検索する\]
- ツール バー: ![](../../images/find_only_word.png) (検索ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

EEID_FINDBAR_ONLY_WORD (4576)```
