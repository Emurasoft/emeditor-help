# \[置換文字列を設定\] コマンド

## 概要

カーソル位置の単語を次に置換する文字列として設定します。

## 説明

カーソル位置の単語を次に置換する文字列として設定します。このコマンドを実行後に、 [\[次を置換\] コマンド](replace_next) を実行すると、ここで設定された文字列ですぐに置換します。また、このコマンドを実行後に、 [\[置換\] コマンド](../../cmd/search/edit_replace) を実行すると、 [\[置換\] ダイアログ ボックス](../../dlg/replace/index) には、このコマンドで設定した文字列が既定の置換文字列として表示されます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[置換文字列を設定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_RETRIEVE_REPLACE_TEXT (4446)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4446);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4446
```
