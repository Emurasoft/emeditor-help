# \[フィルター ツール バーを閉じる\] コマンド

### 概要

> フィルター ツール バーを閉じます。

### 説明

> フィルター ツール バーを閉じます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[フィルター ツール バー\] \- \[フィルター ツール バーを閉じます\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CLOSE\_FILTER\_BAR (3913)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3913);

#### \[VBScript\]

> editor.ExecuteCommandByID 3913
