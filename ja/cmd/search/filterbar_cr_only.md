# \[CRのみ (フィルター ツール バー)\] コマンド

## 概要

フィルター ツール バーの \[CRのみ\] コマンドのオン/オフを切り替えます。

## 説明

フィルター ツール バーの \[CRのみ\] コマンドのオン/オフを切り替えます。

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[フィルター ツール バー\] \- \[改行コードが一致する\] \- \[CRのみ\]
- ツール バー: ![](../../images/match_newline_characters.png) \+ \[CRのみ\] (フィルター ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし
