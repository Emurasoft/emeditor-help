# \[単語のみ検索する (フィルター ツール バー)\] コマンド

### 概要

> フィルター ツール バーの \[単語のみ検索する\] ボタンのオン/オフを切り替えます。

### 説明

> フィルター ツール バーの \[単語のみ検索する\] ボタンのオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[フィルター ツール バー\] \- \[単語のみ検索する\]
- ツール バー: ![](../../images/find_only_word.png) (フィルター ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FILTERBAR\_REG\_EXP (3910)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3910);

#### \[VBScript\]

> editor.ExecuteCommandByID 3910
