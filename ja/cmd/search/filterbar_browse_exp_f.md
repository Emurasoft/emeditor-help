# \[フィルター用の表現の参照 (フィルター ツール バー)\] コマンド

### 概要

> フィルター ツール バーで使用するフィルター用の正規表現またはエスケープ シーケンスを参照します。

### 説明

> フィルター ツール バーで使用するフィルター用の正規表現またはエスケープ シーケンスを参照します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[フィルター ツール バー\] \- \[フィルター用の表現の参照\]
- ツール バー: ![](../../images/find_browse_exp.png) (フィルター ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FILTERBAR\_BROWSE\_EXP\_F (3904)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3904);

#### \[VBScript\]

> editor.ExecuteCommandByID 3904
