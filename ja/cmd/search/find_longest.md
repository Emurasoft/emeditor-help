# \[最長行/セルを検索\] コマンド

### 概要

> 非CSV文書の最長行またはCSV文書の現在の列の最長セルを検索します。

### 説明

> 非CSV文書の最長行またはCSV文書の現在の列の最長セルを検索します。

### 実行方法

- 既定のメニュー: \[検索\] \- \[最長行/セルを検索\]
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[最長行/セルを検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FIND\_LONGEST (4049)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4049);

#### \[VBScript\]

> editor.ExecuteCommandByID 4049
