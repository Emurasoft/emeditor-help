# \[すべての文字列を選択\] コマンド

## 概要

指定するテキストをすべて選択します。

## 説明

現在の文書から、指定するテキストをすべて選択します。

## 実行方法

- 既定のメニュー: \[検索\] \- \[すべての文字列を選択\]
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[すべての文字列を選択\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: CTRL+Shift+A

## プラグイン コマンド ID

```
EEID_SELECT_ALL_OCCURRENCES (4603)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4603);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4603
```
