# \[検索文字列を設定\] コマンド

## 概要

カーソル位置の単語を次に検索する文字列として設定します。

## 説明

カーソル位置の単語を次に検索する文字列として設定します。このコマンドを実行後に、 [\[次を検索\] コマンド](edit_repeat) を実行すると、ここで設定された文字列をすぐに検索します。また、 [\[検索\] ダイアログ ボックス](../../dlg/find/index) の
\[検索する文字列\] の右側にある \[>\] ボタンをクリックして表示されるメニューで、\[カーソル位置の単語\]
が選択されていない場合、 このコマンドを実行後に、 [\[検索\] コマンド](edit_find) を実行すると、 [\[検索\] ダイアログ ボックス](../../dlg/find/index) には、このコマンドで設定した文字列が既定の検索文字列として表示されます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索文字列を設定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+F

## プラグイン コマンド ID

```
EEID_RETRIEVE_FIND_TEXT (4325)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4325);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4325
```
