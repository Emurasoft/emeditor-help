# \[検索ツール バーを閉じる\] コマンド

## 概要

検索ツール バーを閉じます。

## 説明

検索ツール バーを閉じます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バーを閉じる\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_CLOSE_FIND_BAR (4584)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4584);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4584
