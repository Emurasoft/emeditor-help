# \[空でない最短行/セルを検索\] コマンド

## 概要

非CSV文書の空でない最短行またはCSV文書の現在の列の空でない最短セルを検索します。

## 説明

非CSV文書の空でない最短行またはCSV文書の現在の列の空でない最短セルを検索します。

## 実行方法

- 既定のメニュー: \[検索\] \- \[空でない最短行/セルを検索\]
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[空でない最短行/セルを検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_FIND_SHORTEST (4050)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4050);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4050
```
