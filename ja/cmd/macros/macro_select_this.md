# \[このマクロを選択\] コマンド

### 概要

> 現在開いているファイルを現在のマクロとして選択します。

### 説明

> 現在の EmEditor で開いているファイルをマクロ ファイルとして選択し、その後 [\[マクロの実行\] コマンド](quick_macro_run) を実行する際、このマクロ ファイルを実行するようにします。このコマンドを実行すると、 [\[マクロの実行\] \
> コマンド](quick_macro_run) のメニュー項目に、現在開いているファイル名が表示され、このマクロ ファイルが選択されたことを示します。

### 実行方法

- 既定のメニュー: \[マクロ\] \- \[これを選択\]
- [すべてのコマンド](../../glossary/allcommands): \[マクロ\] \- \[これを選択\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MACRO\_SELECT\_THIS (4349)

### マクロ

> 実行できません。
