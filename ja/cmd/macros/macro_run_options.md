# \[一時オプションを設定してマクロを実行\] コマンド

## 概要

一時オプションを設定してマクロを実行します。

## 説明

このコマンドを実行すると、 [\[マクロ一時オプション\] \
ダイアログ ボックス](../../dlg/macro_temp_options/index) が表示され、マクロの繰り返し回数、および検索に失敗したら中止するかどうかを指定してから、マクロを実行することができます。実行するマクロは、 [\[マクロの実行\] \
コマンド](quick_macro_run) の場合と同じです。

## 実行方法

- 既定のメニュー: \[マクロ\] \- \[一時オプションを設定して実行\]
- [すべてのコマンド](../../glossary/allcommands): \[マクロ\] \- \[一時オプションを設定して実行\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MACRO_RUN_OPTIONS (4364)```

## マクロ

実行できません。
