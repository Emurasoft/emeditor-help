# \[マクロ キーワードを検索\] コマンド

### 概要

> カーソル位置のキーワードをマクロ リファレンスから検索します。

### 説明

> カーソル位置のキーワードをマクロ リファレンスから検索します。

### 実行方法

- 既定のメニュー: \[マクロ\] \- \[キーワードを検索\]
- [すべてのコマンド](../../glossary/allcommands): \[マクロ\] \- \[キーワードを検索\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: F1

### プラグイン コマンド ID

- EEID\_MACRO\_HELP\_WORD (4361)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4361);

#### \[VBScript\]

> editor.ExecuteCommandByID 4361
