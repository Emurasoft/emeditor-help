# \[マイ マクロの一覧\] コマンド

## 概要

マイ マクロの指定するマクロを実行します (複数項目)。

## 説明

このコマンドは通常、複数のメニュー項目から成り立っています。マイ
マクロに登録されているマクロに対して実行できます。このコマンドを実行すると、指定したマクロ ファイルを実行します。マイ マクロに含めるマクロ
ファイルについては、 [\[マクロのカスタマイズ\] コマンド](customize_macro) で設定できます。

## 実行方法

- 既定のメニュー: \[マクロ\] \- (マイ マクロのマクロ)
- [すべてのコマンド](../../glossary/allcommands): \[マクロ\] \- (マイ マクロのマクロ)
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MACRO1 から EEID_MACRO1 + 1023 まで (9216 から 9216 + 1023)```

## マクロ

実行できません。
