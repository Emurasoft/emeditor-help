# \[IPv6アドレス降順に並べ替え\] コマンド

## 概要

現在の列のIPv6アドレスを降順で並べ替えます。

## 説明

現在の列のIPv6アドレスを降順で並べ替えます。IPv6アドレスは、RFC 5952で推奨されている正規のテキスト表現でなければなりません。設定のプロパティの [\[リンク\]](../../dlg/properties/link/index) ページの \[アクティブな文字列\] リスト ボックスで、IPv6 アドレスを有効にしておくと、IPv6 アドレスとして認識されるテキストを区別することができます。

## 実行方法

- 既定のメニュー: \[並べ替え\] \- \[IPv6アドレス降順に並べ替え\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[IPv6アドレス降順に並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
