# \[列の再表示\] コマンド

### 概要

> 選択列または現在列を再表示します。

### 説明

> 選択列または現在列を再表示します。

### 実行方法

- 既定のメニュー: \[CSV\] - \[列\] - \[列の再表示\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列の再表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_UNHIDE\_COLUMN (4088)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4088);

#### \[VBScript\]

> editor.ExecuteCommandByID 4088
