# \[列の非表示\] コマンド

### 概要

> 選択列または現在列を非表示にします。

### 説明

> 選択列または現在列を非表示にします。

### 実行方法

- 既定のメニュー: \[CSV\] - \[列\] - \[列の非表示\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列の非表示\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HIDE\_COLUMN (4087)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4087);

#### \[VBScript\]

> editor.ExecuteCommandByID 4087
