# \[比較\] コマンド

### 概要

> 最近表示した 2 個の文書をオプションを指定せずに比較します。

### 説明

> 最近表示した 2 個の文書をオプションを指定せずに比較します。

### 実行方法

- 既定のメニュー: \[比較\] \- \[... と ... を比較\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[... と ... を比較\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_COMPARE\_DIRECT (4492)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4492);

#### \[VBScript\]

> editor.ExecuteCommandByID 4492
