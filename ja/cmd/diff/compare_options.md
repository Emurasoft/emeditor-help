# \[オプションを指定して比較\] コマンド

### 概要

> 比較のウィザードを開始して最近表示した 2 個の文書を比較します。

### 説明

> 比較のウィザードを開始して最近表示した 2 個の文書を比較します。

### 実行方法

- 既定のメニュー: \[比較\] \- \[オプションを指定して比較\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[オプションを指定して比較\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_COMPARE\_OPTIONS (4493)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4493);

#### \[VBScript\]

> editor.ExecuteCommandByID 4493
