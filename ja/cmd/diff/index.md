# \[比較\] カテゴリ

|     |     |
| --- | --- |
| コマンド名 | 説明 |
| [比較](compare_direct) | 最近表示した 2 個の文書をオプションを指定せずに比較します。 |
| [オプションを指定して比較](compare_options) | 比較のウィザードを開始して最近表示した 2 個の文書を比較します。 |
| [再スキャン](compare_rescan) | 比較した文書を再スキャンして結果を最新情報に更新します。 |
| [同期スクロール](sync_scroll_direct) | 最近表示した 2 個の文書をオプションを指定せずに同期スクロールします。 |
| [オプションを指定して同期スクロール](sync_scroll_options) | 比較のウィザードを開始して最近表示した 2 個の文書を同期スクロールします。 |
| [分割表示](split_view_direct) | 最近表示した 2 個の文書を分割表示します。 |
| [リセット](compare_reset) | 比較や同期スクロールをリセットして比較結果をクリアします。 |
| [行頭の空白を無視](ignore_lead_space) | 文書比較時、行頭の空白を無視します。 |
| [行末の空白を無視](ignore_trail_space) | 文書比較時、行末の空白を無視します。 |
| [埋め込まれた空白を無視](ignore_embedded_space) | 文書比較時、行の中間に埋め込まれた空白を無視します。 |
| [大文字小文字の区別を無視](ignore_cases) | 文書比較時、大文字小文字の区別を無視します。 |
| [改行コードを無視](ignore_returns) | 文書比較時、改行コードの違いを無視します。 |
| [コメントを無視](ignore_comment) | 文書比較時、コメントを無視します。 |
| [エンコードを無視](ignore_encoding) | 文書比較時、エンコードを無視します。 |
| [垂直同期スクロール](sync_vert) | 垂直スクロールを同期します。 |
| [水平同期スクロール](sync_horz) | 水平スクロールを同期します。 |
| [カーソル位置の同期](sync_caret) | カーソル位置を同期します。 |
| [終了時ウィンドウ位置を復元](sync_restore_pos) | 比較または同期スクロール終了時、ウィンドウ位置を復元します。 |
| [オプションの保存](compare_save_options) | 比較または同期スクロールのオプションを保存します。 |
| [前の変更](compare_prev) | 前の変更にジャンプします。 |
| [次の変更](compare_next) | 次の変更にジャンプします。 |
| [前の変更行](compare_prev_line) | 前の変更行にジャンプします。 |
| [次の変更行](compare_next_line) | 次の変更行にジャンプします。 |
| [前の変更文字](compare_prev_char) | 前の変更文字にジャンプします。 |
| [次の変更文字](compare_next_char) | 次の変更文字にジャンプします。 |
| [変更をブックマーク](compare_bookmark) | 変更行すべてにブックマークを設定します。 |
| [レポートを作成](compare_generate_report) | 現在の比較の結果に基づいてレポート ファイルを作成します。 |
| [他方にジャンプ](compare_jump_to_other) | 現在の文書が比較またはスクロールしている他方の文書にジャンプします。 |
| [他へコピー](copy_to_other) | すべての変更を左へコピーします。 |
| [すべて他へコピー](copy_all_to_other) | すべての変更を右へコピーします。 |
