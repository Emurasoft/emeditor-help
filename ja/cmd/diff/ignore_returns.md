# \[改行コードを無視\] コマンド

## 概要

文書比較時、改行コードの違いを無視します。

## 説明

文書比較時、改行コードの違いを無視します。

## 実行方法

- 既定のメニュー: \[比較\] \- \[改行コードを無視\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[改行コードを無視\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_IGNORE_RETURNS (4502)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4502);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4502
```
