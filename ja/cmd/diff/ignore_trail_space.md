# \[行末の空白を無視\] コマンド

## 概要

文書比較時、行末の空白を無視します。

## 説明

文書比較時、行末の空白を無視します。

## 実行方法

- 既定のメニュー: \[比較\] \- \[行末の空白を無視\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[行末の空白を無視\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_IGNORE_TRAIL_SPACE (4499)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4499);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4499
```
