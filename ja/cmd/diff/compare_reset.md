# \[リセット\] コマンド

## 概要

比較や同期スクロールをリセットして比較結果をクリアします。

## 説明

比較や同期スクロールをリセットして比較結果をクリアします。

## 実行方法

- 既定のメニュー: \[比較\] \- \[リセット\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[リセット\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_COMPARE_RESET (4497)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4497);
```

```
editor.ExecuteCommandByID 4497
