# \[埋め込まれた空白を無視\] コマンド

## 概要

文書比較時、行の中間に埋め込まれた空白を無視します。

## 説明

文書比較時、行の中間に埋め込まれた空白を無視します。

## 実行方法

- 既定のメニュー: \[比較\] \- \[埋め込まれた空白を無視\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[埋め込まれた空白を無視\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_IGNORE_EMBEDDED_SPACE (4500)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4500);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4500
```
