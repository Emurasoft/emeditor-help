# \[同期スクロール\] コマンド

### 概要

> 最近表示した 2 個の文書をオプションを指定せずに同期スクロールします。

### 説明

> 最近表示した 2 個の文書をオプションを指定せずに同期スクロールします。

### 実行方法

- 既定のメニュー: \[比較\] \- \[... と ... を同期スクロール\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[... と ... を同期スクロール\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SYNC\_SCROLL\_DIRECT (4495)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4495);

#### \[VBScript\]

> editor.ExecuteCommandByID 4495
