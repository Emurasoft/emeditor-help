# \[変更をブックマーク\] コマンド

### 概要

> 変更行すべてにブックマークを設定します。

### 説明

> 変更行すべてにブックマークを設定します。

### 実行方法

- 既定のメニュー: \[比較\] \- \[変更をブックマーク\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[変更をブックマーク\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_COMPARE\_BOOKMARK (4593)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4593);

#### \[VBScript\]

> editor.ExecuteCommandByID 4593
