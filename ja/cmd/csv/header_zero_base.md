# \[列ヘッダーに0から始まる数字を使用\] コマンド

## 概要

列ヘッダーに0から始まる数字を使用します。

## 説明

列ヘッダーに0から始まる数字を使用します。

## 実行方法

- 既定のメニュー: \[CSV\] - \[列ヘッダーのタイプ\] - \[0, 1, 2, ...\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列ヘッダーのタイプ\] - \[0, 1, 2, ...\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

EEID_HEADER_ZERO_BASE (3988)```
