# \[行ヘディング 4\] コマンド

## 概要

最初の4列を非スクロール領域の行ヘディングとして設定します。

## 説明

最初の4列を非スクロール領域の行ヘディングとして設定します。

## 実行方法

- 既定のメニュー: \[CSV\] - \[ヘディング (ウィンドウ枠の固定)\] - \[行ヘディング 4\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[ヘディング (ウィンドウ枠の固定)\] - \[行ヘディング 4\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

```
EEID_ROW_HEADING_4 (4095)```
