# \[列を結合\] コマンド

## 概要

CSV文書の選択列を結合します。

## 説明

このコマンドを実行すると、 [\[列を結合\] ダイアログ ボックス](../../dlg/combine_columns/index) が表示され、CSV文書の選択列を結合します。

## 実行方法

- 既定のメニュー: \[CSV\] - \[列\] - \[列を結合\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列\] - \[列を結合\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_COMBINE_COLUMNS (4047)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4047);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4047
```
