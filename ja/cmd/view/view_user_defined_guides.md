# \[ユーザー定義ガイド\] コマンド

### 概要

> ユーザー定義ガイドの表示/非表示を切り替えます。

### 説明

> ユーザー定義ガイドの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[表示\] \- \[記号\] \- \[ユーザー定義ガイド\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[記号\] \- \[ユーザー定義ガイド\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_VIEW\_USER\_DEFINED\_GUIDES (3863)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3863);

#### \[VBScript\]

> editor.ExecuteCommandByID 3863
