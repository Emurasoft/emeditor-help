# \[改行コード\] コマンド

## 概要

改行コードの表示/非表示を切り替えます。

## 説明

改行コードの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[記号\] \- \[改行コード\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[記号\] \- \[改行コード\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_VIEW_RETURN (4532)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4532);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4532
```
