# \[メイン メニュー\] コマンド

## 概要

メイン メニューの表示/非表示を切り替えます。

## 説明

メイン メニューの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[メイン メニュー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[メイン メニュー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_MAIN_MENU (4072)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4072);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4072
```
