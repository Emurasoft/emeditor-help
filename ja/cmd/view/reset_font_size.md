# \[フォント サイズをリセット\] コマンド

### 概要

> 表示フォント サイズをリセットします。

### 説明

> 表示フォント サイズをリセットします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[フォント\]  \- \[フォント サイズをリセット\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+0

### プラグイン コマンド ID

- EEID\_RESET\_FONT\_SIZE (3876)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3876);

#### \[VBScript\]

> editor.ExecuteCommandByID 3876
