# \[ファンクション バー\] コマンド

## 概要

ファンクション バーの表示/非表示を切り替えます。

## 説明

ファンクション バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ファンクション バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ファンクション バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_VIEW_STATUS_BAR (4448)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4448);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4448
```
