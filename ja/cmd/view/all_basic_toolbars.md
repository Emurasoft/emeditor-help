# \[すべての基本ツール バーの表示/非表示を切り替え\] コマンド

### 概要

> すべての基本ツール バーの表示/非表示を切り替えます。

### 説明

> すべての基本ツール バーの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[すべての基本ツール バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[すべての基本ツール バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_ALL\_BASIC\_TOOLBARS (4068)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4068);

#### \[VBScript\]

> editor.ExecuteCommandByID 4068
