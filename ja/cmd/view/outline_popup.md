# \[アウトライン (ポップアップ メニュー)\] コマンド

### 概要

> ポップアップ メニューを表示してアウトラインの設定を行います。

### 説明

> ポップアップ メニューを表示してアウトラインの設定を行います。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[アウトライン\] \- \[アウトライン (ポップアップ メニュー)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_OUTLINE\_POPUP (3875)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3875);

#### \[VBScript\]

> editor.ExecuteCommandByID 3875
