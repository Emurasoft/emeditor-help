# \[ツール バーを固定する\] コマンド

### 概要

> ツール バーを固定します。

### 説明

> ツール バーが固定されていなければ固定し、固定いていれば固定を解除します。ツール バーが固定されていない場合、ツール バーの左端のグリップをカーソルでドラッグしてツール バーの位置を変更できます。固定されていると、ツール バーの左端にグリップが表示されず、ドラッグすることができなくなります。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[ツール バーを固定する\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_LOCK\_TOOLBARS (4341)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4341);

#### \[VBScript\]

> editor.ExecuteCommandByID 4341
