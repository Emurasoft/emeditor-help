# \[列ヘッダーにアルファベットを使用\] コマンド

### 概要

> 列ヘッダーにアルファベットを使用します。

### 説明

> 列ヘッダーにアルファベットを使用します。

### 実行方法

- 既定のメニュー: \[CSV\] - \[列ヘッダーのタイプ\] - \[A, B, C, ...\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列ヘッダーのタイプ\] - \[A, B, C, ...\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HEADER\_ALPHA (3987)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3987);

#### \[VBScript\]

> editor.ExecuteCommandByID 3987
