# \[列ヘッダー\] コマンド

### 概要

> CSV用の列ヘッダーの表示/非表示を切り替えます。

### 説明

> CSV用の列ヘッダーの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[CSV\] - \[列ヘッダー\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列ヘッダー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HEADER\_TOGGLE (3985)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3985);

#### \[VBScript\]

> editor.ExecuteCommandByID 3985
