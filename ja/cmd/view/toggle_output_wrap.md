# \[ウィンドウの右端で折り返し (アウトプット バー)\] コマンド

## 概要

アウトプット バーで行をウィンドウの右端で折り返します。

## 説明

アウトプット バーで行をウィンドウの右端で折り返します。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[アウトプット バー\] \- \[ウィンドウの右端で折り返し\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし
