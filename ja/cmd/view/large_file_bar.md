# \[巨大ファイル コントローラー\] コマンド

### 概要

> 巨大ファイル コントローラーの表示/非表示を切り替えます。

### 説明

> 巨大ファイル コントローラーの表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[表示\] \- \[巨大ファイル コントローラー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[巨大ファイル コントローラー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_LARGE\_FILE\_BAR (4449)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4449);

#### \[VBScript\]

> editor.ExecuteCommandByID 4449
