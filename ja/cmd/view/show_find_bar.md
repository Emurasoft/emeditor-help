# \[検索ツール バー\] コマンド

## 概要

検索ツール バーの表示/非表示を切り替えます。

## 説明

検索ツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[検索ツール バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[検索ツール バー\]
- ツール バー:
![](../../images/emeditor12_toggle_find_bar_button.png)
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_FIND_BAR (4559)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4559);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4559
