# \[タブの自動整列\] コマンド

### 概要

> タブを自動的に整列します。

### 説明

> このコマンドがチェックされていると、タブを自動的に整列します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[タブの整列\] \- \[タブの自動整列\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_AUTO\_SORT (4404)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4404);

#### \[VBScript\]

> editor.ExecuteCommandByID 4404
