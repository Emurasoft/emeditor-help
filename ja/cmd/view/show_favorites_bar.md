# \[お気に入りツール バー\] コマンド

## 概要

お気に入りツール バーの表示/非表示を切り替えます。

## 説明

お気に入りツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[お気に入り\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[お気に入り\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_FAVORITES_BAR (23251)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(23251);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23251
```
