# \[すべてのマーカーを有効にする\] コマンド

### 概要

> すべてのマーカーを有効にします。

### 説明

> すべてのマーカーを有効にします。

### 方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[マーカー\] \- \[すべて有効\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MARKER\_ENABLE\_ALL (4585)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4585);

#### \[VBScript\]

> editor.ExecuteCommandByID 4585
