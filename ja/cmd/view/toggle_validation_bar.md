# \[構文チェック バー\] コマンド

## 概要

構文チェック バーの表示/非表示を切り替えます。

## 説明

構文チェック バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[構文チェック バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[構文チェック バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_TOGGLE_VALIDATION_BAR (4038)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4038);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4038
```
