# \[現在のフォント サイズを既定に設定\] コマンド

## 概要

表示フォント サイズをリセットします。

## 説明

表示フォント サイズをリセットします。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[フォント\] \- \[現在のフォント サイズを既定に設定\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SET_DEFAULT_FONT_SIZE (3877)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3877);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3877
```
