# \[タブ\] コマンド

## 概要

タブの表示/非表示を切り替えます。

## 説明

タブの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[記号\] \- \[タブ\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[記号\] \- \[タブ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_VIEW_TAB (4534)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4534);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4534
```
