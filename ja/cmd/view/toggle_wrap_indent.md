# \[折り返しインデント\] コマンド

### 概要

> 折り返しインデントを有効にします。

### 説明

> 折り返しインデントを有効にします。

### 実行方法

- 既定のメニュー: \[表示\] \- \[タブ/インデント\] \- \[折り返しインデント\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[タブ/インデント\] \- \[折り返しインデント\]
- ツール バー: ![](../../images/wrap_indent24x16.gif)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_TOGGLE\_WRAP\_INDENT (4542)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4542);

#### \[VBScript\]

> editor.ExecuteCommandByID 4542
