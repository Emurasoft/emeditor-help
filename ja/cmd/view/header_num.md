# \[列ヘッダーに1から始まる数字を使用\] コマンド

### 概要

> 列ヘッダーに1から始まる数字を使用します。

### 説明

> 列ヘッダーに1から始まる数字を使用します。

### 実行方法

- 既定のメニュー: \[CSV\] - \[列ヘッダーのタイプ\] - \[1, 2, 3, ...\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[列ヘッダーのタイプ\] - \[1, 2, 3, ...\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HEADER\_NUM (3986)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3986);

#### \[VBScript\]

> editor.ExecuteCommandByID 3986
