# \[更新日時で整列\] コマンド

## 概要

タブを更新日時によって整列します。

## 説明

タブを更新日時によって整列します。 [\[タブの自動整列\] コマンド](auto_sort) がチェックされている場合は、ファイルを開いたり名前を付けて保存する際にこのコマンドが毎回実行され、自動的に整列されます。 [\[タブの自動整列\] コマンド](auto_sort) がチェックされていない場合は、このコマンドを実行した時に 1 回だけ実行されます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[タブの整列\] \- \[更新日時\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SORT_MODIFIED (4400)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4400);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4400
```
