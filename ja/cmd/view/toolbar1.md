# \[ツール バーの表示/非表示を切り替え\] コマンド

## 概要

指定するツール バーの表示/非表示を切り替えます (複数項目)。

## 説明

指定するツール バーの表示/非表示を切り替えます (複数項目)。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_TOOLBAR1 から EEID_TOOLBAR1 + 255 まで (22976 から 22976 + 255 まで)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(22976 + i);  // i は 0 から 255 までの整数
```

### \[VBScript\]

```
editor.ExecuteCommandByID 22976 + i  ' i は 0 から 255 までの整数
```
