# \[アクティブで整列\] コマンド

### 概要

> タブを最近アクティブにした順で整列します。

### 説明

> タブを最近アクティブにした順で整列します。 [\[タブの自動整列\] コマンド](auto_sort) がチェックされている場合は、文書をアクティブにしたり、ファイルを開いた際にこのコマンドが毎回実行され、自動的に整列されます。 [\[タブの自動整列\] コマンド](auto_sort) がチェックされていない場合は、このコマンドを実行した時に 1 回だけ実行されます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[タブの整列\] \- \[アクティブ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_ZORDER (4401)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4401);

#### \[VBScript\]

> editor.ExecuteCommandByID 4401
