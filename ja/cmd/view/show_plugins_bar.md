# \[プラグイン ツール バー\] コマンド

### 概要

> プラグイン ツール バーの表示/非表示を切り替えます。

### 説明

> プラグイン ツール バーの表示/非表示を切り替えます。プラグイン ツール バーには、ユーザーが設定したプラグインが表示されます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[プラグイン ツール バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SHOW\_PLUGINS\_BAR (4285)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4285);

#### \[VBScript\]

> editor.ExecuteCommandByID 4285
