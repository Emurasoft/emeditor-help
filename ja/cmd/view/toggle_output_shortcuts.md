# \[ショートカット キーを有効にする (アウトプット バー)\] コマンド

## 概要

アウトプット バーでショートカット キーの有効と無効を切り替えます。

## 説明

アウトプット バーでショートカット キーの有効と無効を切り替えます。

## 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[アウトプット バー\] \- \[ショートカット キーを有効にする\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_TOGGLE_OUTPUT_SHORTCUTS (4549)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4549);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4549
```
