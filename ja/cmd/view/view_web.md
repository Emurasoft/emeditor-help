# \[Webブラウザ\] コマンド

## 概要

Webブラウザの表示/非表示を切り替えます。

## 説明

Webブラウザの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[Webブラウザ\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[Webブラウザ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+2

## プラグイン コマンド ID

```
EEID_VIEW_WEB (23243)
```

## マクロ

### \[JavaScript\]

```
WebBar.Visible = !WebBar.Visible;
```

### \[VBScript\]

```
WebBar.Visible = Not WebBar.Visible
```
