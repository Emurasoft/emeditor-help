# \[半角空白\] コマンド

## 概要

半角空白の表示/非表示を切り替えます。

## 説明

半角空白の表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[記号\] \- \[半角空白\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[記号\] \- \[半角空白\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_VIEW_SPACE (4535)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(4535);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4535
```
