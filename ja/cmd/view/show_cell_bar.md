# \[セル ツール バー\] コマンド

## 概要

セル ツール バーの表示/非表示を切り替えます。

## 説明

セル ツール バーの表示/非表示を切り替えます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[セル\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[セル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_CELL_BAR (3967)```

## マクロ

### \[JavaScript\]

```
editor.ExecuteCommandByID(3967);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3967
```
