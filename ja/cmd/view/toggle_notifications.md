# \[通知\] コマンド

### 概要

> 通知の表示/非表示を切り替えます。

### 説明

> 通知の表示/非表示を切り替えます。

### 実行方法

- 既定のメニュー: \[表示\] \- \[通知\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[通知\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_TOGGLE\_NOTIFICATIONS (4061)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4061);

#### \[VBScript\]

> editor.ExecuteCommandByID 4061
