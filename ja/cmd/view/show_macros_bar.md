# \[マクロ ツール バー\] コマンド

## 概要

マクロ ツール バーの表示/非表示を切り替えます。

## 説明

マクロ ツール バーの表示/非表示を切り替えます。マクロ ツール バーには、ユーザーが設定したマイ マクロが表示されます。

## 実行方法

- 既定のメニュー: \[表示\] \- \[ツール バー\] \- \[マクロ ツール バー\]
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[ツール バー\] \- \[マクロ ツール バー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

## プラグイン コマンド ID

```
EEID_SHOW_MACROS_BAR (4421)```

## マクロ

## \[JavaScript\]

```
editor.ExecuteCommandByID(4421);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4421
```
