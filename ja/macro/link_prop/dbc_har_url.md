# DBCharURL プロパティ (LinkProp オブジェクト)

設定のプロパティの [**\[リンク\]** タブ](../../dlg/properties/link/index) の **\[全角文字をURLとして認識する\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.DBCharURL;
object.DBCharURL = b;
```

### \[VBScript\]

```
b = object.DBCharURL
object.DBCharURL = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
