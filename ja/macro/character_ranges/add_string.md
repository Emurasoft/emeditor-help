# AddString メソッド ()

文字列を追加します。

## 

### \[JavaScript\]

```
list.AddString( sFind, sTreatAs );
```

### \[VBScript\]

```
list.AddString sFind, sTreatAs
```

## パラメータ

_sFind_

検索する文字列を指定します。

_sTreatAs_

代替文字を指定します。空の文字列を指定すると、指定した範囲は無視されます。

## バージョン

EmEditor Professional Version 22.0 以上で利用できます。
