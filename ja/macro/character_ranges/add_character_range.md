# AddCharacterRange メソッド ()

文字範囲を追加します。

## 

### \[JavaScript\]

```
list.AddCharacterRange( nMin, nMax, sProperties, sTreatAs );
```

### \[VBScript\]

```
list.AddCharacterRange nMin, nMax, sProperties, sTreatAs
```

## パラメータ

_nMin_

範囲の最小の文字を指定します。

_nMax_

範囲の最大の文字を指定します。

_sProperties_

Unicode 用字、Unicode 一般カテゴリを表現する文字列を指定します。

_sTreatAs_

代替文字を指定します。空の文字列を指定すると、指定した範囲は無視されます。

## バージョン

EmEditor Professional Version 22.0 以上で利用できます。
