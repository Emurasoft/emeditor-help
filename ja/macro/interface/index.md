# Interface オブジェクト

## プロパティ

|     |     |
| --- | --- |
| **[x](x)** | サポートされたプラグインから渡された水平カーソル位置を返します。 |
| **[y](y)** | サポートされたプラグインから渡された垂直カーソル位置を返します。 |

## メソッド

|     |     |
| --- | --- |
| **[write](write)** | 文字列をサポートされたプラグインに渡します。 |
| **[writeln](writeln)** | 文字列と改行コードをサポートされたプラグインに渡します。 |

## バージョン

Version 9.00 以上で利用できます。


```{toctree}
:maxdepth: 1
write
writeln
x
y
```
