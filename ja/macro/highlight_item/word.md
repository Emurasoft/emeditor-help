# Word プロパティ

設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の
**\[単語のみ\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
item. **Word**;

item. **Word** = _b_;

#### \[VBScript\]

_b_ =
item. **Word**

item. **Word** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
