# Word プロパティ (HighlightItem オブジェクト)

設定のプロパティの [\[強調(1)\] ページ](../../dlg/properties/highlight1/index) の
\[単語のみ\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = item.Word;
item.Word = b;
```

### \[VBScript\]

```
b = item.Word
item.Word = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
