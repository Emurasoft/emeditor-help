# HighlightItem オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [Case](case) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の <br> **\[大文字小文字を区別\]** チェック ボックスに相当します。 |
| [Color](color) | オブジェクトの色を指定します。 |
| [InsideTag](inside_tag) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の **\[タグの内側のみ\]** チェック ボックスに相当します。 |
| [Name](name) | オブジェクトの名前を指定します。 |
| [RegExp](reg_exp) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の <br> **\[正規表現\]** チェック ボックスに相当します。 |
| [RightAll](right_all) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の **\[行の右全部を強調\]** チェック ボックスに相当します。 |
| [RightSide](right_side) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の **\[行の右を強調\]** チェック ボックスに相当します。 |
| [Word](word) | 設定のプロパティの [**\[強調(1)\]** タブ](../../dlg/properties/highlight1/index) の **\[単語のみ\]** チェック ボックスに相当します。 |

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
