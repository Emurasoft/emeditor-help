# NoSpaceEdge プロパティ (GeneralProp オブジェクト)

設定のプロパティの [\[基本\] タブ](../../dlg/properties/general/index) の\[ウィンドウ左端に空白を入れない\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.NoSpaceEdge;
object.NoSpaceEdge = b;
```

### \[VBScript\]

```
b = object.NoSpaceEdge
object.NoSpaceEdge = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
