# ShowRuler プロパティ (GeneralProp オブジェクト)

設定のプロパティの [**\[基本\]** ページ](../../dlg/properties/general/index) の
**\[ルーラを表示\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.ShowRuler;
object.ShowRuler = b;
```

### \[VBScript\]

```
b = object.ShowRuler
object.ShowRuler = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
