# PasteAnsi プロパティ (GeneralProp オブジェクト)

設定のプロパティの [\[基本\] タブ](../../dlg/properties/general/index) の
\[常にANSIで貼り付け\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.PasteAnsi;
object.PasteAnsi = b;
```

### \[VBScript\]

```
b = object.PasteAnsi
object.PasteAnsi = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
