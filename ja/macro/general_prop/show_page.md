# ShowPage プロパティ (GeneralProp オブジェクト)

設定のプロパティの [\[基本\] タブ](../../dlg/properties/general/index) の\[ページ番号を表示\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.ShowPage;
object.ShowPage = b;
```

### \[VBScript\]

```
b = object.ShowPage
object.ShowPage = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
