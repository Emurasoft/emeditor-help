# MarginQuote プロパティ (GeneralProp オブジェクト)

設定のプロパティの [**\[基本\]** ページ](../../dlg/properties/general/index) の
**\[引用行の文字数\]** テキスト ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.MarginQuote;
object.MarginQuote = b;
```

### \[VBScript\]

```
b = object.MarginQuote
object.MarginQuote = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
