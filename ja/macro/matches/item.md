# Item プロパティ (Matches コレクション)

指定したインデックスの [Match オブジェクト](../match/index) を取得します。

## 

### \[JavaScript\]

```
item = list.Item( Index );
```

### \[VBScript\]

```
item = list.Item( Index )
```

## パラメータ

_Index_

アイテムのインデックスを １ から始まる整数として指定します。

## バージョン

EmEditor Professional Version 15.9 以上で利用できます。
