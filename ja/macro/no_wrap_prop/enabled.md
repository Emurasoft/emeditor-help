# Enabled プロパティ (NoWrapProp オブジェクト)

設定のプロパティの [**\[分割禁止単語\]** タブ](../../dlg/properties/no_wrap/index) の
**\[分割禁止単語の処理を有効にする\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Enabled;
object.Enabled = b;
```

### \[VBScript\]

```
b = object.Enabled
object.Enabled = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
