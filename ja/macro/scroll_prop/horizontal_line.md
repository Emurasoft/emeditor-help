# HorizontalLine プロパティ (ScrollProp オブジェクト)

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の **\[水平罫線\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.HorizontalLine;
object.HorizontalLine = b;
```

### \[VBScript\]

```
b = object.HorizontalLine
object.HorizontalLine = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
