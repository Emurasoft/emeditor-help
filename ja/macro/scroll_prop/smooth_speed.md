# SmoothSpeed プロパティ (ScrollProp オブジェクト)

設定のプロパティの [\[スクロール\] タブ](../../dlg/properties/scroll/index) の\[スムーズ スクロール\] スライダに相当します。

## 

### \[JavaScript\]

```
n = object.SmoothSpeed;
object.SmoothSpeed = n;
```

### \[VBScript\]

```
n = object.SmoothSpeed
object.SmoothSpeed = n
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
