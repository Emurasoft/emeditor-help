# HalfPage プロパティ (ScrollProp オブジェクト)

設定のプロパティの [\[スクロール\] タブ](../../dlg/properties/scroll/index) の
\[半ページ スクロール\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.HalfPage;
object.HalfPage = b;
```

### \[VBScript\]

```
b = object.HalfPage
object.HalfPage = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
