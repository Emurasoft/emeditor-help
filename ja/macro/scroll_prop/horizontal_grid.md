# HorizontalGrid プロパティ (ScrollProp オブジェクト)

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の
**\[水平グリッド\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.HorizontalGrid;
object.HorizontalGrid = b;
```

### \[VBScript\]

```
b = object.HorizontalGrid
object.HorizontalGrid = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
