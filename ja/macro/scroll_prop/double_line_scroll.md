# DoubleLineScroll プロパティ

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の
**\[2行スクロール\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **DoubleLineScroll**;

object. **DoubleLineScroll** = _b_;

#### \[VBScript\]

_b_ =
object. **DoubleLineScroll**

object. **DoubleLineScroll** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
