# FullName プロパティ ()

EmEditor の実行ファイル emeditor.exe の完全パスとファイル名を取得します。

#### \[JavaScript\]

_strName_ = editor. **FullName**;

#### \[VBScript\]

_strName_ = editor. **FullName**

## 例

#### \[JavaScript\]

strName = editor. **FullName**;

alert( "EmEditor の完全パスは" + strName + "です。" );

#### \[VBScript\]

strName = editor. **FullName**

alert "EmEditor の完全パスは" & strName & "です。"

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
