# DetectDSV プロパティ (FileProp オブジェクト)

設定のプロパティの [**\[ファイル\]** タブ](../../dlg/properties/file/index) の **\[DSV (ユーザー定義区切り) を検出\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.DetectDSV;
object.DetectDSV = b;
```

### \[VBScript\]

```
b = object.DetectDSV
object.DetectDSV = b
```

## バージョン

EmEditor Professional Version 9.00 以上で利用できます。
