# ShowFullPath プロパティ (FileProp オブジェクト)

設定のプロパティの [**\[ファイル\]** ページ](../../dlg/properties/file/index) の **\[ファイル名をフルパスで表示\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.ShowFullPath;
object.ShowFullPath = b;
```

### \[VBScript\]

```
b = object.ShowFullPath
object.ShowFullPath = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
