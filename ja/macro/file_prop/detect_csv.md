# DetectCSV プロパティ (FileProp オブジェクト)

設定のプロパティの [\[ファイル\] タブ](../../dlg/properties/file/index) の\[CSV（カンマ区切り）を検出\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.DetectCSV;
object.DetectCSV = b;
```

### \[VBScript\]

```
b = object.DetectCSV
object.DetectCSV = b
```

## バージョン

EmEditor Professional Version ９.00 以上で利用できます。
