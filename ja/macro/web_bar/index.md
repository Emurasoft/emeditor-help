# WebBar オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [Visible](visible) | Webブラウザの表示/非表示を切り替えます。 |

## メソッド

|     |     |
| --- | --- |
| [Open](open) | 指定する URL のWebサイトを開きます。 |
| [SetFocus](set_focus) | Webブラウザにキーボード フォーカスを設定します。 |

## 例

### \[JavaScript\]

```
WebBar.Open( "https://www.emeditor.com/" );
WebBar.SetFocus();
```

### \[VBScript\]

```
WebBar.Open "https://www.emeditor.com/"
WebBar.SetFocus
```

## バージョン

EmEditor Professional Version 23.0 以上で利用できます。
