# InTag プロパティ (SpellProp オブジェクト)

設定のプロパティの [**\[スペル チェック\]** ページ](../../dlg/properties/spell/index) の **\[タグ\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.InTag;
object.InTag = b;
```

### \[VBScript\]

```
b = object.InTag
object.InTag = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
