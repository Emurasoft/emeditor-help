# Quote プロパティ ()

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の **\[引用マークで始まる行\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **Quote**;

object. **Quote** = _b_;

#### \[VBScript\]

_b_ =
object. **Quote**

object. **Quote** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
