# Quote プロパティ (SpellProp オブジェクト)

設定のプロパティの [**\[スペル チェック\]** ページ](../../dlg/properties/spell/index) の **\[引用マークで始まる行\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Quote;
object.Quote = b;
```

### \[VBScript\]

```
b = object.Quote
object.Quote = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
