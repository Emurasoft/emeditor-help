# Highlight プロパティ (SpellProp オブジェクト)

設定のプロパティの [\[スペル チェック\] タブ](../../dlg/properties/spell/index) の
\[強調文字列\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Highlight;
object.Highlight = b;
```

### \[VBScript\]

```
b = object.Highlight
object.Highlight = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
