# Enabled プロパティ ()

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の
**\[スペル チェックを行う\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **Enabled**;

object. **Enabled** = _b_;

#### \[VBScript\]

_b_ =
object. **Enabled**

object. **Enabled** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
