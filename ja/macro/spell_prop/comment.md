# Comment プロパティ

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の **\[コメント\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **Comment**;

object. **Comment** = _b_;

#### \[VBScript\]

_b_ =
object. **Comment**

object. **Comment** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
