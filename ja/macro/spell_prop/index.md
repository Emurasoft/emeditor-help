# SpellProp オブジェクト

|     |     |
| --- | --- |

| [**Quote**](quote) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[引用マークで始まる行\] チェック ボックスに相当します。 |
| [**SingleQuotes**](singlequotes) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の \[1重引用符で囲まれた文字列\] チェック ボックスに相当します。 |
| [**DoubleQuotes**](doublequotes) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[2重引用符で囲まれた文字列\] チェック ボックスに相当します。 |
| [**Comment**](comment) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[コメント\] チェック ボックスに相当します。 |

| [**InTag**](intag) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[タグ\] チェック ボックスに相当します。 |
| [**Highlight**](highlight) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[強調文字列\] チェック ボックスに相当します。 |
| [**Hyperlink**](hyperlink) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の \[ハイパーリンク\] チェック ボックスに相当します。 |
| [**Rest**](rest) | 設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の <br> \[上記以外\] チェック ボックスに相当します。 |
