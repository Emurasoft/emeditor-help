# Hyperlink プロパティ ()

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の **\[ハイパーリンク\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **Hyperlink**;

object. **Hyperlink** = _b_;

#### \[VBScript\]

_b_ =
object. **Hyperlink**

object. **Hyperlink** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
