# CamelCase プロパティ (SpellProp オブジェクト)

設定のプロパティの [\[スペル チェック\] ページ](../../dlg/properties/spell/index) の \[キャメル ケースに対応\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.CamelCase;
object.CamelCase = b;
```

### \[VBScript\]

```
b = object.CamelCase
object.CamelCase = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
