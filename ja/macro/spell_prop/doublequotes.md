# DoubleQuotes プロパティ ()

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の
**\[2重引用符で囲まれた文字列\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **DoubleQuotes**;

object. **DoubleQuotes** = _b_;

#### \[VBScript\]

_b_ =
object. **DoubleQuotes**

object. **DoubleQuotes** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
