# Rest プロパティ (SpellProp オブジェクト)

設定のプロパティの [\[スペル チェック\] タブ](../../dlg/properties/spell/index) の
\[上記以外\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Rest;
object.Rest = b;
```

### \[VBScript\]

```
b = object.Rest
object.Rest = b
```

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
