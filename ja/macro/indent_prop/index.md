# IndentProp オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [AutoIndent](auto_indent) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[自動インデントを有効にする\]** チェック ボックスに相当します。 |
| [Begin](begin) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の **\[インデント開始\]** テキスト ボックスに相当します。 |
| [End](end) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の **\[インデント終了\]** テキスト ボックスに相当します。 |
| [EndOfStatement](end_of_statement) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[ステートメント終了\]** テキスト ボックスに相当します。 |
| [IgnoreAfter](ignore_after) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の **\[次の行の後は無視 (正規表現)\]** テキスト ボックスに相当します。 |
| [IndentColumns](indent_columns) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[インデントの桁数\]** テキスト ボックスに相当します。 |
| [InsertSpaces](insert_spaces) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[タブの代わりに空白を挿入\]** チェック ボックスに相当します。 |
| [RegExp](reg_exp) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[正規表現を使用する\]** チェック ボックスに相当します。 |
| [TabColumns](tab_columns) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の <br> **\[タブの桁数\]** テキスト ボックスに相当します。 |
| [WrapIndent](wrap_indent) | [**\[タブ/インデント\]** ダイアログ ボックス](../../dlg/properties/general/indent/index) の **\[折り返しインデント\]** チェック ボックスに相当します。 |
