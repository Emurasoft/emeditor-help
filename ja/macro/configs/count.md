# Count プロパティ ()

設定の数を取得します。

#### \[JavaScript\]

_nCount_ = editor.Configs. **Count**;

#### \[VBScript\]

_nCount_ = editor.Configs. **Count**

## 例

#### \[JavaScript\]

alert( "設定の数: " + editor.Configs.Count );

#### \[VBScript\]

alert "設定の数: " & editor.Configs.Count

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
