# Count プロパティ (Configs コレクション)

設定の数を取得します。

## 

### \[JavaScript\]

```
nCount = editor.Configs.Count;
```

### \[VBScript\]

```
nCount = editor.Configs.Count
```

## 例

### \[JavaScript\]

```
alert( "設定の数: " + editor.Configs.Count );
```

### \[VBScript\]

```
alert "設定の数: " & editor.Configs.Count
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
