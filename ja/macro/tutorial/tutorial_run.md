# マクロの実行 ()

マクロを保存すると、保存されたマクロが既定のマクロ ファイルとなっています。つまり、前回保存したマクロが、 [**\[マクロの実行\]** \
コマンド](../../cmd/macros/quick_macro_run) を選択した時に実行されるマクロです。\[マクロ\] メニューを開くと、\[実行\]
コマンドのメニュー項目の右側に、「tutorial.jsee」または「tutorial.vbee」と表示されているはずです。また、ツール バーの
\[マクロの実行\] ボタン上にマウス カーソルを置くと、「マクロの実行 tutorial.jsee」または「マクロの実行
tutorial.vbee」と表示されるはずです。つまり、これらのマクロの実行コマンドを選択する時に実行されるマクロのファイル名が表示され、これが既定のマクロ
ファイルということになります。この既定のマクロ ファイルは、次に別のマクロを記録するまで、あるいは別のマクロ
ファイルを選択もしくは保存するまでの間、変わりません。一度、EmEditor を終了してから、再び起動しても、既定のマクロ ファイルは変わることはありません。

それでは、EmEditor の [**\[Textを新規作成\]** コマンド](../../cmd/file/file_new) を選択して、新しい
EmEditor ウィンドウを表示します。そして、 [**\[マクロの実行\]** \
コマンド](../../cmd/macros/quick_macro_run) を選択します。すると、さきほど記録した

EmEditor supports macros.

の 1 文が表示されます。このマクロの実行の操作は何度でも繰り返すことができます。

他のマクロを記録したり選択した場合、別のマクロが既定のマクロになってしまうため、上記の 1
回の操作では以前に保存したマクロを実行できません。その場合、次のいずれかの操作によって、以前に保存したマクロを再実行することができます。

1. [**\[マクロの選択\]** コマンド](../../cmd/macros/macro_select) (\[マクロ\] メニューの
\[選択\]) を選択すると、\[ファイルを開く\] ダイアログ ボックスが表示されるので、ここで、実行したいマクロ ファイル
(このチュートリアルの場合、tutorial.jsee または tutorial.vbee) を選択します。そして、上記の操作 （マクロの実行)
を実行します。
2. \[マクロ\] メニューには、以前に保存したことがあるマクロの一覧が表示されています。これを「マイ
マクロ」と呼ぶことにします。この一覧から実行したいマクロ ファイル名 (このチュートリアルの場合 tutorial.jsee または
tutorial.vbee) を選択して、実行することができます。

EmEditor は既定では、以前に保存、または選択されたマクロは、自動的に「マイ
マクロ」の一覧に追加されます。この機能は便利ですが、非常に多くのマクロの保存、選択を繰り返したい場合は、かえって不便になることも考えられます。その場合
[**\[マクロのカスタマイズ\]** コマンド](../../cmd/macros/customize_macro) (\[マクロ\] メニューの
\[カスタマイズ\]) を選択し、 [**\[マクロのカスタマイズ\]** ダイアログ \
ボックス](../../dlg/macro_customize/index) の [**\[オプション\]** タブ](../../dlg/macro_customize/options/index) で、 **\[新しいマクロを保存または選択時、マイ**
**マクロに追加する\]** チェック ボックスをオフにすると、この機能が無効になり、マクロの保存、選択時にマイ マクロに自動的に追加されません。

また、必要のなくなったマクロ ファイルをマイ マクロより削除したい場合は、 [**\[マクロのカスタマイズ\]** ダイアログ ボックス](../../dlg/macro_customize/index) の [**\[マイ マクロ\]** \
タブ](../../dlg/macro_customize/my_macros/index) で、必要のないマクロ ファイル名を選択してから、\[削除\] ボタンをクリックします。さらに、このタブで、\[マクロ\] メニューに表示されるマクロ
ファイルの順番を入れ替えることもできます。

実行したいマクロをキーボードのショートカット キーから実行したい場合は、 [**\[すべての設定のプロパティ\]** \
コマンド](../../cmd/tools/all_prop) または [**\[現在の設定のプロパティ\]** コマンド](../../cmd/tools/customize) で、 [**\[キーボード\]** \
タブ](../../dlg/properties/keyboard/index) を選択し、ショートカット キーを割り当てたいマクロ ファイル名を選択して、好きなショートカット キーを割り当てることができます。

次に、保存されたマクロを編集してみましょう。
