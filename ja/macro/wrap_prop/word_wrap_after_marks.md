# WordWrapAfterMarks プロパティ (WrapProp オブジェクト)

設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の
\[記号の後の折り返しを有効にする\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.WordWrapAfterMarks;
object.WordWrapAfterMarks = b;
```

### \[VBScript\]

```
b = object.WordWrapAfterMarks
object.WordWrapAfterMarks = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
