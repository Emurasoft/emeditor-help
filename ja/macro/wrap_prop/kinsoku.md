# Kinsoku プロパティ (WrapProp オブジェクト)

設定のプロパティの [\[禁則文字\] タブ](../../dlg/properties/wrap/index) の
\[禁則処理を行う\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Kinsoku;
object.Kinsoku = b;
```

### \[VBScript\]

```
str = object.Kinsoku
object.Kinsoku = str
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
