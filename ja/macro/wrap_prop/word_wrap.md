# WordWrap プロパティ (WrapProp オブジェクト)

設定のプロパティの [**\[禁則文字\]** タブ](../../dlg/properties/wrap/index) の
**\[欧文ワード ラップを行う\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.WordWrap;
object.WordWrap = b;
```

### \[VBScript\]

```
b = object.WordWrap
object.WordWrap = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
