# WrapProp オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [**Kinsoku**](kinsoku) | 設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の \[禁則処理を行う\] チェック ボックスに相当します。 |
| [**KinsokuBegin**](kinsoku_begin) | 設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の \[行頭禁則文字\] テキスト ボックスに相当します。 |
| [**KinsokuEnd**](kinsoku_end) | 設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の \[行末禁則文字\] テキスト ボックスに相当します。 |
| [**WordWrap**](word_wrap) | 設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の \[欧文ワード ラップを行う\] チェック ボックスに相当します。 |
| [**WordWrapAfterMarks**](word_wrap_after_marks) | 設定のプロパティの [\[禁則文字\] ページ](../../dlg/properties/wrap/index) の \[記号の後の折り返しを有効にする\] チェック ボックスに相当します。 |

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
