# InitialFolder プロパティ

[**\[新規作成時の詳細\]** のダイアログ ボックス](../../dlg/properties/file/new_details/index) の **\[初期フォルダ\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_s_ = object. **InitialFolder**;

object. **InitialFolder** = _s_;

#### \[VBScript\]

_s_ = object. **InitialFolder**

object. **InitialFolder** = _s_

## バージョン

EmEditor Professional Version 14.1 以上で利用できます。
