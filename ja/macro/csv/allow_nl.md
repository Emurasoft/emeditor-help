# AllowNL プロパティ (Csv オブジェクト)

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[引用符内の改行コードを許可\] チェック ボックス に相当します。

## 

### \[JavaScript\]

```
b = object.AllowNL;
object.AllowNL = b;
```

### \[VBScript\]

```
b = object.AllowNL
object.AllowNL = b
```

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
