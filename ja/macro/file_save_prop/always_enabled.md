# AlwaysEnabled プロパティ

[**\[保存時の詳細\]** ダイアログ ボックス](../../dlg/properties/file/save_details/index) の **\[変更がなくても保存を有効\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **AlwaysEnabled**;

object. **AlwaysEnabled** = _b_;

#### \[VBScript\]

_b_ =
object. **AlwaysEnabled**

object. **AlwaysEnabled** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
