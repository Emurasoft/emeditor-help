# ForegroundWindow プロパティ ()

現在のフォアグラウンド ウィンドウを取得します。

#### \[JavaScript\]

wnd = shell. **ForegroundWindow**;

#### \[VBScript\]

wnd = shell. **ForegroundWindow**

## 例

#### \[JavaScript\]

wnd = shell.FouregroundWindow;

str = wnd.Caption;

alert( str );

#### \[VBScript\]

wnd = shell.FouregroundWindow

str = wnd.Caption

alert str

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
