# CreateFolder メソッド ()

フォルダを作成します。

#### \[JavaScript\]

shell. **CreateFolder**( _strFolder_ );

#### \[VBScript\]

shell. **CreateFolder** _strFolder_

## パラメータ

_strFolder_

作成するフォルダを指定します。

## 例

#### \[JavaScript\]

shell.CreateFolder( "C:\\\Test" );

#### \[VBScript\]

shell.CreateFolder "C:\\Test"

## バージョン

EmEditor Professional Version 22.1 以上で利用できます。
