# CreateFolder メソッド (Shell オブジェクト)

フォルダを作成します。

## 

### \[JavaScript\]

```
shell.CreateFolder( strFolder );
```

### \[VBScript\]

```
shell.CreateFolder strFolder
```

## パラメータ

_strFolder_

作成するフォルダを指定します。

## 例

### \[JavaScript\]

```
shell.CreateFolder( "C:\\\Test" );
```

### \[VBScript\]

```
shell.CreateFolder "C:\\Test"
```

## バージョン

EmEditor Professional Version 22.1 以上で利用できます。
