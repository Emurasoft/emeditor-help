# write メソッド ()

現在のカーソル位置に文字列を挿入、または上書きします。

#### \[JavaScript\]

document. **write**( _str_ );

#### \[VBScript\]

document. **write** _str_

## 引数

_str_

挿入、または上書きする文字列を指定します。

## 例

#### \[JavaScript\]

document. **write**( "こんにちは" );

#### \[VBScript\]

document. **write** "こんにちは"

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
