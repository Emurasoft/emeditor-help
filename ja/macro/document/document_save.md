# Save メソッド (Document オブジェクト)

文書を保存します。

## 

### \[JavaScript\]

```
document.Save( [ strName ] );
```

### \[VBScript\]

```
document.Save [ strName ]
```

## パラメータ

_strName_

省略できます。保存するファイル名を完全パスと名前で指定します。省略すると、名前を付けて保存するダイアログ ボックスが表示されます。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
