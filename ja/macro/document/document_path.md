# Path プロパティ

文書を含むディレクトリのパスをファイル名を付けないで取得します。

#### \[JavaScript\]

_strPath_ = document. **Path**;

#### \[VBScript\]

_strPath_ = document. **Path**

## 例

#### \[JavaScript\]

alert( document.Path );

#### \[VBScript\]

alert document.Path

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
