# HighlightTag プロパティ

タグを強調表示するかどうかを取得、または設定します。

#### \[JavaScript\]

_bHighlightTag_ = document. **HighlightTag**;

document. **HighlightTag** = _bHighlightTag_;

#### \[VBScript\]

_bHighlightTag_ = document. **HighlightTag**

document. **HighlightTag** = _bHighlightTag_

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
