# HighlightTag プロパティ (Document オブジェクト)

タグを強調表示するかどうかを取得、または設定します。

## 

### \[JavaScript\]

```
bHighlightTag = document.HighlightTag;
document.HighlightTag = bHighlightTag;
```

### \[VBScript\]

```
bHighlightTag = document.HighlightTag
document.HighlightTag = bHighlightTag
```

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
