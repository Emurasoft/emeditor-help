# HighlightFind プロパティ ()

検索した文字列を強調表示するかどうかを取得、または設定します。

#### \[JavaScript\]

_bHighlightFind_ = document. **HighlightFind**;

document. **HighlightFind** = _bHighlightFind_;

#### \[VBScript\]

_bHighlightFind_ = document. **HighlightFind**

document. **HighlightFind** = _bHighlightFind_

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
