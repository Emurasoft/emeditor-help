# HighlightFind プロパティ ()

検索した文字列を強調表示するかどうかを取得、または設定します。

## 

### \[JavaScript\]

```
bHighlightFind = document.HighlightFind;
document.HighlightFind = bHighlightFind;
```

### \[VBScript\]

```
bHighlightFind = document.HighlightFind
document.HighlightFind = bHighlightFind
```

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
