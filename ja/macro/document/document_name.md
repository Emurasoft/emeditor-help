# Name プロパティ (Document オブジェクト)

文書のファイル名をパスを付けないで取得するか、または文書のファイル名をリネームします。文書が無題の場合、ファイルを保存しないで文書のタイトルを変更します。

## 

### \[JavaScript\]

```
strName = document.Name;document.Name = strName;
```

### \[VBScript\]

```
strName = document.Name document.Name = strName
```

## 例

### \[JavaScript\]

```
alert( document.Name );
```

### \[VBScript\]

```
alert document.Name
```

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
