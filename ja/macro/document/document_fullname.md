# FullName プロパティ ()

文書ファイルの完全パスと名前を取得します。

#### \[JavaScript\]

_strName_ = document. **FullName;**

#### \[VBScript\]

_strName_ = document. **FullName**

## 例

#### \[JavaScript\]

alert( document.FullName );

#### \[VBScript\]

alert document.FullName

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
