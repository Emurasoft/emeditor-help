# NewlineCode プロパティ ()

文書の改行コードを取得します。

#### \[JavaScript\]

_nNewlineCode_ = document. **NewlineCode**;

#### \[VBScript\]

_nNewlineCode_ = document. **NewlineCode**

## Parameters

_nNewlineCode_

次のいずれかの値を取得します。

|     |     |
| --- | --- |
| eeCrAndLf | CR+LF |
| eeCrOnly | CR only |
| eeLfOnly | LF only |
| eeNewlineMixed | Mixed |

#### バージョン

EmEditor Professional Version 14.6 以上で利用できます。
