# NewlineCode プロパティ (Document オブジェクト)

文書の改行コードを取得します。

## 

### \[JavaScript\]

```
nNewlineCode = document.NewlineCode;
```

### \[VBScript\]

```
nNewlineCode = document.NewlineCode
```

## Parameters

_nNewlineCode_

次のいずれかの値を取得します。

|     |     |
| --- | --- |
| eeCrAndLf | CR+LF |
| eeCrOnly | CR only |
| eeLfOnly | LF only |
| eeNewlineMixed | Mixed |

### バージョン

EmEditor Professional Version 14.6 以上で利用できます。
