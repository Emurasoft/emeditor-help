# NarrowingBottom プロパティ ()

部分編集の最下部の位置 (Y 座標) を取得、または設定します。-1 は、部分編集が設定されていないことを意味します。

#### \[JavaScript\]

_n_ = document. **NarrowingBottom**;

document. **NarrowingBottom** = _n_;

#### \[VBScript\]

_n_ = document. **NarrowingBottom**

document. **NarrowingBottom** = _n_

## バージョン

EmEditor Professional Version 14.6 以上で利用できます。
