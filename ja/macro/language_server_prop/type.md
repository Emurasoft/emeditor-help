# Type プロパティ (LanguageServerProp オブジェクト)

設定のプロパティの [\[言語サーバー\] ページ](../../dlg/properties/language_server/index) の \[文書タイプ\] ドロップダウン リスト ボックスに相当します。

## 

### \[JavaScript\]

```
type = object.Type;
object.Type = type;
```

### \[VBScript\]

```
type = object.Type
object.Type = type
```

## バージョン

EmEditor Professional Version 23.0 以上で利用できます。
