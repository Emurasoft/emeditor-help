# Enabled プロパティ (LanguageServerProp オブジェクト)

設定のプロパティの [\[言語サーバー\] ページ](../../dlg/properties/language_server/index) の \[言語サーバー プロトコルを有効にする\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.Enabled;
object.Enabled = b;
```

### \[VBScript\]

```
b = object.Enabled
object.Enabled = b
```

## バージョン

EmEditor Professional Version 23.0 以上で利用できます。
