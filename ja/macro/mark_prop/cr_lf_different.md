# CrLfDifferent プロパティ (MarkProp オブジェクト)

設定のプロパティの [\[記号\] ページ](../../dlg/properties/marks/index) の \[CRとLFを区別して表示\] チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.CrLfDifferent;
object.CrLfDifferent = b;
```

### \[VBScript\]

```
b = object.CrLfDifferent
object.CrLfDifferent = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
