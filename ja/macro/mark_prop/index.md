# MarkProp オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [**CrLfDifferent**](cr_lf_different) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の **\[CRとLFを区別して表示\]** チェック ボックスに相当します。 |
| [**ShowControlCharacters**](show_control_characters) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の **\[制御文字を表示\]** チェック ボックスに相当します。 |
| [**ShowDBSpaces**](show_db_spaces) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の <br> **\[全角空白表示\]** チェック ボックスに相当します。 |
| [**ShowEOF**](show_eof) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の **\[EOF表示\]** チェック ボックスに相当します。 |
| [**ShowIndentGuides**](show_indent_guides) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の **\[インデント ガイドを表示\]** チェック ボックスに相当します。 |
| [**ShowReturns**](show_returns) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の <br> **\[改行コード表示\]** チェック ボックスに相当します。 |
| [**ShowSpaces**](show_spaces) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の <br> **\[半角空白表示\]** チェック ボックスに相当します。 |
| [**ShowTabs**](show_tabs) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の <br> **\[タブ表示\]** チェック ボックスに相当します。 |
| [**ShowUserDefinedGuides**](show_user_defined_guides) | 設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の <br> **\[ユーザー定義ガイドを表示\]** チェック ボックスに相当します。 |
