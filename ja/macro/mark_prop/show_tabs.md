# ShowTabs プロパティ (MarkProp オブジェクト)

設定のプロパティの [**\[記号\]** ページ](../../dlg/properties/marks/index) の **\[タブ表示\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.ShowTabs;
object.ShowTabs = b;
```

### \[VBScript\]

```
b = object.ShowTabs
object.ShowTabs = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
