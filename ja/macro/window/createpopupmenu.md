# CreatePopupMenu メソッド (Window オブジェクト)

ポップアップ メニューを作成します。

## 

### \[JavaScript\]

```
popupmenu =CreatePopupMenu();
```

### \[VBScript\]

```
Set popupmenu =CreatePopupMenu
```

## 戻り値

新しく作成した [PopupMenu オブジェクト](../popupmenu/index) を返します。

## バージョン

EmEditor Professional Version 5.00 以上で利用できます。
