# confirm メソッド ()

\[OK\] ボタンと \[キャンセル\] ボタンが付いたダイアログ ボックス内にメッセージを表示します。

#### \[JavaScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_);

#### \[VBScript\]

_bAnswer_ = **confirm**( _strQuestion, strMessage2_ )

## パラメータ

_strQuestion_

表示する文字列を指定します。

_strMessage2_

省略可能。最初のメッセージの下に表示する追加の文字列を指定します。

## 戻り値

\[OK\] ボタンを選択した場合は true を、\[キャンセル\] ボタンを選択した場合は false を返します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
