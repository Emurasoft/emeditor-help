# SetPosition メソッド (Window オブジェクト)

ウィンドウのサイズと位置を設定します。

## 

### \[JavaScript\]

```
wnd.SetPosition( x, y, width, height );
```

### \[VBScript\]

```
wnd.SetPosition x, y, width, height
```

## パラメータ

_x_

ウィンドウの水平方向の位置をピクセル単位で指定します。

_y_

ウィンドウの垂直方向の位置をピクセル単位で指定します。

_width_

ウィンドウの幅をピクセル単位で指定します。

_height_

ウィンドウの高さをピクセル単位で指定します。

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
