# alert メソッド (Window オブジェクト)

\[OK\] ボタンが付いたダイアログ ボックス内にメッセージを表示します。

## 

### \[JavaScript\]

```
alert( strMessage, strMessage2 );
```

### \[VBScript\]

```
alert strMessage, strMessage2
```

## パラメータ

_strMessage_

表示する文字列を指定します。

_strMessage2_

省略可能。最初のメッセージの下に表示する追加の文字列を指定します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
