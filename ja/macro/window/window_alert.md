# alert メソッド

\[OK\] ボタンが付いたダイアログ ボックス内にメッセージを表示します。

#### \[JavaScript\]

**alert**( _strMessage_, _strMessage2_ );

#### \[VBScript\]

**alert** _strMessage, strMessage2_

## パラメータ

_strMessage_

表示する文字列を指定します。

_strMessage2_

省略可能。最初のメッセージの下に表示する追加の文字列を指定します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
