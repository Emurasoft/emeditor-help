# clipboardData オブジェクト

## メソッド

|     |     |
| --- | --- |
| [clearData](cleardata) | クリップボードを空にします。 |
| [getData](getdata) | クリップボードからデータを取得します。 |
| [setData](setdata) | クリップボードに指定するデータを設定します。 |

## バージョン

EmEditor Professional Version 5.00 以上で利用できます。
