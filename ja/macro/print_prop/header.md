# Header プロパティ (PrintProp オブジェクト)

設定のプロパティの [**\[印刷\]** ページ](../../dlg/properties/print/index) の
**\[ヘッダ\]** テキスト ボックスに相当します。

## 

### \[JavaScript\]

```
s = object.Header;
object.Header = s;
```

### \[VBScript\]

```
s = object.Header
object.Header = s
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
