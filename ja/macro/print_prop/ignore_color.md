# IgnoreColor プロパティ (PrintProp オブジェクト)

設定のプロパティの [**\[印刷\]** タブ](../../dlg/properties/print/index) の
**\[色と下線を無効にする\]** チェック ボックスに相当します。

## 

### \[JavaScript\]

```
b = object.IgnoreColor;
object.IgnoreColor = b;
```

### \[VBScript\]

```
b = object.IgnoreColor
object.IgnoreColor = b
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
