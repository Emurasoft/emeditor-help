# LineSpace プロパティ (PrintProp オブジェクト)

設定のプロパティの [**\[印刷\]** タブ](../../dlg/properties/print/index) の **\[行間隔\]** テキスト ボックスに相当します。

## 

### \[JavaScript\]

```
n = object.LineSpace;
object.LineSpace = n;
```

### \[VBScript\]

```
n = object.LineSpace
object.LineSpace = n
```

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
