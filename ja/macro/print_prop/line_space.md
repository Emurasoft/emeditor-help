# LineSpace プロパティ

設定のプロパティの [**\[印刷\]** タブ](../../dlg/properties/print/index) の **\[行間隔\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_n_ = object. **LineSpace**;

object. **LineSpace** = _n_;

#### \[VBScript\]

_n_ = object. **LineSpace**

object. **LineSpace** = _n_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
