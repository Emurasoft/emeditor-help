# GoToBrace メソッド (Selection オブジェクト)

対応するかっこへ移動します。

## 

### \[JavaScript\]

```
document.selection.GoToBrace( [ bExtend ] );
```

### \[VBScript\]

```
document.selection.GoToBrace [ bExtend ]
```

## 引数

_bExtend_

省略可能。選択範囲を解除するかどうかを指定します。省略すると、選択範囲を解除して移動します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
