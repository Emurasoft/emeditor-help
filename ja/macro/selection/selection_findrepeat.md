# FindRepeat メソッド

前回検索した文字列を再度検索します。

#### \[JavaScript\]

document.selection. **FindRepeat**( _nFlags_ );

#### \[VBScript\]

document.selection. **FindRepeat** _nFlags_

## 引数

_nFlags_

次の値の組み合わせを指定します。

|     |     |
| --- | --- |
| eeFindRepeatNext | カーソル位置から下方向に検索します。 |
| eeFindRepeatPrevious | カーソル位置から上方向に検索します。 |
| eeFindRepeatWord | 選択範囲が空の場合、カーソル位置の単語を検索します。 |

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
