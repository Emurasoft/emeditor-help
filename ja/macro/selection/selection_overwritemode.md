# OverwriteMode プロパティ ()

上書き状態、または挿入状態かどうかを取得、または設定します。

#### \[JavaScript\]

_bOverwrite_ = document.selection. **OverwriteMode**;

document.selection. **OverwriteMode** = _bOverwrite_;

#### \[VBScript\]

_bOverwrite_ = document.selection. **OverwriteMode**

document.selection. **OverwriteMode** = _bOverwrite_

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
