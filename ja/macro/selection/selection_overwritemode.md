# OverwriteMode プロパティ (Selection オブジェクト)

上書き状態、または挿入状態かどうかを取得、または設定します。

## 

### \[JavaScript\]

```
bOverwrite = document.selection.OverwriteMode;
document.selection.OverwriteMode = bOverwrite;
```

### \[VBScript\]

```
bOverwrite = document.selection.OverwriteMode
document.selection.OverwriteMode = bOverwrite
```

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
