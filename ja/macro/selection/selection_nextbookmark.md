# NextBookmark メソッド

開いている文書の次のブックマークに移動します。

#### \[JavaScript\]

_bFound_ = document.selection. **NextBookmark**();

#### \[VBScript\]

_bFound_ = document.selection. **NextBookmark**

## 戻り値

次のブックマークが見つかると true を返します。見つからない場合は false を返します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
