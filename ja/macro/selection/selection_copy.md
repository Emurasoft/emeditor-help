# Copy メソッド ()

選択範囲のテキストをクリップボードにコピーします。

## 

### \[JavaScript\]

```
document.selection.Copy( nFlags );
```

### \[VBScript\]

```
document.selection.Copy nFlags
```

## 引数

_nFlags_

次の値の組み合わせになります。

|     |     |
| --- | --- |
| eeCopyUnicode | Unicode でコピーします (既定)。 |
| eeCopyQuotes | 引用マークを付けてコピーします。 |

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
