# InsertDate メソッド ()

時刻と日付を挿入します。

#### \[JavaScript\]

document.selection. **InsertDate**( _nFlags_ );

#### \[VBScript\]

document.selection. **InsertDate** \[ _nFlags_ \]

## 引数

_nFlags_

次の値のいずれかを指定します。

|     |     |
| --- | --- |
| eeDateTimeDate | 時刻、日付の順番に挿入します。時刻、日付の形式は、コントロール パネルの短い形式で指定します。 |
| eeDateDateTime | 日付、時刻の順番に挿入します。時刻、日付の形式は、コントロール パネルの短い形式で指定します。 |

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
