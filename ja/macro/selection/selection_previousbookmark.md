# PreviousBookmark メソッド ()

開いている文書の前のブックマークへ移動します。

#### \[JavaScript\]

_bFound_ = document.selection. **PreviousBookmark**();

#### \[VBScript\]

_bFound_ = document.selection. **PreviousBookmark**

## 戻り値

前のブックマークが見つかると true を返します。見つからない場合は false を返します。

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
