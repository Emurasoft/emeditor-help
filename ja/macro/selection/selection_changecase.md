# ChangeCase メソッド

選択されたテキストの大文字と小文字を変換します。

#### \[JavaScript\]

document.selection. **ChangeCase**( _nFlags_ );

#### \[VBScript\]

document.selection. **ChangeCase** _nFlags_

## パラメータ

_nFlags_

次の中から選択して指定ます。

|     |     |
| --- | --- |
| eeCaseLowerCase | 大文字を小文字に変換します。 |
| eeCaseUpperCase | 小文字を大文字に変換します。 |
| eeCaseCapitalize | 単語の最初の文字を大文字に変換します (EmEditor Professional Version 4.10 以上のみ)。 |

## バージョン

EmEditor Professional Version 4.00 以上で利用できます。
