# Count プロパティ ()

文書の数を取得します。

## 

### \[JavaScript\]

```
nCount = editor.Documents.Count;
```

### \[VBScript\]

```
nCount = editor.Documents.Count
```

## 例

### \[JavaScript\]

```
alert( "文書の数: " + editor.Documents.Count );
```

### \[VBScript\]

```
alert "文書の数: " & editor.Documents.Count
```

## バージョン

EmEditor Professional Version 5.00 以上で利用できます。
