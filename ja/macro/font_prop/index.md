# FontProp オブジェクト

## プロパティ

|     |     |
| --- | --- |
| [**DisplayList**](display_list) | 表示用フォントの [FontList コレクション](../font_list/index) を取得します。 |
| [**PrintList**](print_list) | 印刷用フォントの [FontList コレクション](../font_list/index) を取得します。 |
| [**SameFontPrint**](same_font_print) | [\[カスタマイズ フォント\] ダイアログ ボックス](../../dlg/properties/font/index) の <br> \[印刷フォントに表示フォントを使用\] チェック ボックスに相当します。 |
| [**UsePrinterFont**](use_printer_font) | [\[カスタマイズ フォント\] ダイアログ ボックス](../../dlg/properties/font/index) の <br> \[通常のプリンタ用フォントも含める\] チェック ボックスに相当します。 |

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
