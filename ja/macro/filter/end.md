# End プロパティ (Filter オブジェクト)

検索したいテキストの終了位置のインデックスを論理文字単位で指定します。最後まで全部を検索する場合には 0 を指定します。このフィールドを有効にするには Column パラメータに -1 を指定する必要があります。

## 

### \[JavaScript\]

```
n = item.End;
item.End = n;
```

### \[VBScript\]

```
n = item.End
item.End = n
```

## バージョン

EmEditor Professional Version 16.0 以上で利用できます。
