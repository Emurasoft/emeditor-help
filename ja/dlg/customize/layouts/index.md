# \[レイアウト\] ページ

**\[レイアウト\]** ページでは、レイアウトを定義したり、各レイアウトのオプションの設定を行います。

## \[レイアウト\] リスト ボックス

定義されたレイアウトが表示されます。

## \[追加\] ボタン

一覧に項目を追加します。

## \[削除\] ボタン

一覧から選択した項目を削除します。

## \[オプション\] リスト ボックス

各レイアウトのオプションの設定を行います。

## \[すべて有効/無効にする\] チェック ボックス

一覧のすべての項目の有効と無効を切り替えます。

## \[現在のレイアウトを保存\] ボタン

ウィンドウ位置、ツール バーのレイアウトを含む現在のレイアウトを、レイアウト一覧の選択した項目に削除します。

## \[リセット\] ボタン

設定を既定にします。
