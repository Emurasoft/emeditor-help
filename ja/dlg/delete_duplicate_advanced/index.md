# \[重複行の削除/ブックマーク\] ダイアログ ボックス

このダイアログ ボックスは、 [\[重複行の削除/ブックマーク(高度)\] コマンド](../../cmd/edit/delete_duplicate_advanced) を実行するか、ツール バーで \[重複行の削除/ブックマーク(高度)\] ボタンをクリックすると表示されます。

### \[行全体を調べる\]  ラジオ ボタン

このボタンがチェックされていると、EmEditor は行全体に対して重複を調べます。

### \[選択文字列のみを調べる\]  ラジオ ボタン

このボタンがチェックされていると、箱型選択または複数選択が存在する時、選択された文字列のみに対して重複を調べます。

### \[指定する行のみを調べる\]  ラジオ ボタン

このボタンがチェックされていると、指定された列のみに対して重複を調べます。

### \[列\]  リスト ボタン

列の一覧を表示します。

### \[すべて有効/無効にする\] チェック ボックス

一覧のすべての項目の有効と無効を切り替えます。

### \[大文字と小文字を区別しない\] チェック ボックス

重複行を削除する際、大文字小文字の区別を無視します。

### \[空行を無視する\] チェック ボックス

重複行を削除する際、空行を無視します。

### \[空セルを無視する\] チェック ボックス

CSV 文書で重複行を削除する際、空セルを無視します。言い換えれば、すべての列が選択されている場合、区切り文字のみを含む行は無視されます。複数の列が指定されている場合、指定した列のすべてのセルが空である必要があります。

### \[隣の行のみ比較する(既に並べ替え済と仮定)\] チェック ボックス

隣接する行のみを調べます。このオプションは文書が既に並べ替えられている場合に使用すると便利です。

### \[グループのすべての文書に適用\] チェック ボックス

このグループのすべての文書に対して適用します。

### \[選択した範囲のみ\] チェック ボックス

選択したテキストに対してのみ適用します。

### \[各重複のすべての行を含める\] チェック ボックス

各重複のすべての行を削除、または各重複のすべての行にブックマークを設定します。

### \[削除\] ボタン

指定した設定で重複行の削除を開始します。

### \[ブックマーク\] ボタン

指定した設定で重複行にブックマークの設定を開始します。
