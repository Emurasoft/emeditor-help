# \[自動マーカー\] タブ

**\[自動マーカー\]** タブでは、マーカー一覧に関する設定を行います。

### \[自動マーカーを有効にする\] チェック ボックス

自動マーカーを有効にします。

### \[カーソル位置の単語を取得する\] チェック ボックス

カーソル位置の単語を取得します。

### \[単語のみ\] チェック ボックス

単語全体が文字列の場合のみ選択テキストを強調します。

### \[大文字と小文字を区別\] チェック ボックス

大文字と小文字が一致した場合のみ選択テキストを強調します。

### \[半角英数字のみ\] チェック ボックス

選択テキストが半角英数字のみを含んでいる場合のみ選択テキストを強調します。

### \[リセット\] ボタン

設定を既定にします。
