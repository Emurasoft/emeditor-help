# 設定のプロパティ

設定のプロパティは、 [\[設定の定義\] ダイアログ ボックス](../configurations/index) で
\[プロパティ\] ボタンをクリックした時、 [\[現在の設定のプロパティ\] コマンド](../../cmd/tools/customize)、あるいは
[\[すべての設定のプロパティ\] コマンド](../../cmd/tools/all_prop) を選択した時に表示されます。設定のプロパティには、次のページが存在します。
