# 「CSV 格式」頁面

**CSV 格式** 頁面讓您能自訂與 CSV 格式相關的設定。

### CSV 格式清單方塊

顯示定義的 CSV 格式清單。為了在打開一個檔案時偵測該檔案是否含有 CSV 格式，必須到組態屬性中 [**檔案** 頁面](../../properties/file/index) 上的 **要偵測的 CSV 檔案** 清單方塊中勾選要偵測的 CSV 格式。

### 「添加」按鈕

添加一個項目到清單上。

### 「刪除」按鈕

點擊該按鈕從清單中刪除被選取的項目。

### 「上移」按鈕

把選取的項目在清單上往上移。

### 「下移」按鈕

把選取的項目在清單上往下移。

### 「全選」按鈕

選擇整個清單。

### 「工具列按鈕標題」核取方塊和文字方塊

指定與 CSV 格式名不同的標題。

### 「分隔符號」文字方塊

指定一個分隔符號來分隔不同類別的數據。分隔符號可以包括以下的逸出序列。分隔符號也可以是一個最長 39 個字元的字串。

|     |     |
| --- | --- |
| \\a | 警告 (響鈴) |
| \\b | 退格 |
| \\t | 水平 Tab |
| \\v | 垂直 Tab |
| \\\ | 反斜杠 |
| \\xhhhh | 用十六進位表示的 Unicode 字元 |

### 「引號標記」文字方塊

指定引號標記來引用數據項。引號可以包括以上逸出序列。引號必須必須是一個字元。

### 「允許分隔符號在引號內」核取方塊

指定分隔符號是否可以在引號內。

### 「允許在引號內換行」核取方塊

指定是否允許在引號內的換行。

### 「使用逸出序列」核取方塊

指定是否應使用一個反斜線來轉義分隔符號和反斜線。此選項允許非標準的 CSV 格式，即引號內不允許分隔符號。

### 「欄標題」文字方塊

指定在一個 CSV 文檔中的欄標題所占的資料行數。

### 「列標題」文字方塊

指定在一個 CSV 文檔中的列標題所占的資料欄數。

### 「要偵測的最小行數」文字方塊

指定作為一個 CSV 文檔所要偵測的最小行數。當打開一個檔案時，EmEditor 會在指定的行數中計算分隔符號的數目，如果所有行都有相同數目的分隔符號，檔案會被偵測為 CSV 文檔。不管您如何選擇，CSV 文檔至少必須要有兩行。而且，在組態屬性中的 [**檔案** 頁面](../../properties/file/index) 上，CSV 格式必須被勾選。

### 「要偵測的最大行數」文字方塊

指定作為一個 CSV 文檔所要偵測的最大行數。當打開一個檔案時，EmEditor 會在指定的行數中計算分隔符號的數目，如果所有行都有相同數目的分隔符號，檔案會被偵測為 CSV 文檔。不管您如何選擇，CSV 文檔至少必須要有兩行。而且，在組態屬性中的 [**檔案** 頁面](../../properties/file/index) 上，CSV 格式必須被勾選。

### 「要偵測的最小分隔符號數」文字方塊

這個數字指定要偵測為一個 CSV 文檔檔案所要包含的最小分隔符號數。例如，如果在一個 CSV 文檔中僅包含 3 個分隔符號，并且在這個文字方塊選項中的數字是 3，那么這個檔案會被偵測為 CSV 檔案。然而，如果在這個文字方塊選項中的數字是 4，那么檔案就不會被偵測為 CSV 檔案。而且，必須在組態屬性 [**檔案** 頁面](../../properties/file/index) 上的 **要偵測的 CSV 檔案** 清單方塊中勾選 CSV 格式。

### 「要偵測的第一行」文字方塊

這個數字指定了作為開始偵測 CSV 文檔的第一行的行號。例如，一個文檔的前 3 行不包含分隔符號，并且這個文字方塊中的數字是 3，那么 EmEditor 會忽略文檔的前 3 行，把其余的文檔作為 CSV 檔案。

### 「一個儲存格中的最大換行數」文字方塊

指定可以嵌入儲存格中的最大新行數。此選項只有在勾選了 **允許換行符號在引號內** 時才可用。

### 「重設」按鈕

重設為預設設定。
