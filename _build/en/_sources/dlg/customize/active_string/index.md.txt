# 「主動字串」頁面

**主動字串** 頁面讓你能為每個事件一個行為。

### 「類型」清單方塊

顯示可以自訂為 **主動字串** 的項目清單。這些項目包括URL，電子郵件地址，IPv4 和 IPv6 地址，用引號括起來的字串，HTML 字元參考，通用字元名稱/百分比編碼，十六進位顏色，RGB 顏色，標記和搜索字串。另外，可以在組態屬性的 [**連結** 頁面](../../properties/link/index) 中為每個組態啟用或禁用某個主動字串。顯示一列可以自訂條目會出現在主動字串中項目。

### 「事件」清單方塊

顯示可以選擇作為觸發器來自動運行操作的事件的清單。

### 「操作」下拉清單方塊

為某個主動字串類型指定特定事件觸發時的動作。

### 「在操作之前選擇字串」核取方塊

如果選中此選項，則在運行指定的操作之前將選擇主動字串。 某些操作可能需要勾選此核取方塊。如果在 **操作** 下拉清單方塊中選擇了 **無**，則不選中此選項也會選擇主動字串。

### 「命令」文字方塊

當在 **操作** 下拉清單方塊中選擇 **運行程式** 時，你可以指定要打開的程式，資料夾，文檔或 Internet 資源的名稱。你可以用 **「>」** 按鈕來選擇預定義的巨集，例如，用 _$(Path)_，或 **「...」** 按鈕從電腦中選擇一個檔案。

### 「巨集的路徑」文字方塊

當在 **操作** 下拉清單方塊中選擇 **運行巨集** 時，你可以指定要運行的巨集檔案。你可以用 **「>」** 按鈕來選擇預定義的巨集，例如，用 _$(Path)_，或 **「...」** 按鈕從電腦中選擇一個檔案。

### 「工具」下拉清單方塊

當在 **操作** 下拉清單方塊中選擇 **外部工具** 時，你可以指定要運行的外部工具。外部工具可以在 [**自訂工具** 對話方塊](../../tools/index) 中定義。

### 「命令」下拉清單方塊

當在 **操作** 下拉清單方塊中選擇 **EmEditor 命令** 時，你可以從這個下拉清單方塊中選取要執行的命令。一些命令需要在執行之前選中主動字串，如果是這種情況，要勾選 **在操作之前選擇字串** 核取方塊。

### 「重置」按鈕

重置為預設設定。
