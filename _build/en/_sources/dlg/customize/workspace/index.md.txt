# 「工作區」頁面

**工作區** 頁面讓您能自訂與 EmEditor 工作區相關的設定。

### 「自動工作區」下拉清單方塊

指定儲存和加載行為。您可以從下列選項中選擇:

|     |     |
| --- | --- |
| **無** | 不儲存或還原工作區。 |
| **僅儲存** | 當即將關閉最後一個分組時，工作區將自動儲存，但在啟動 EmEditor 時不會自動恢復工作區。 |
| **僅加載** | 當啟動 EmEditor 時，將自動恢復工作區，但當要關閉最後一個分組時不會自動儲存工作區。 |
| **儲存並還原** | 當即將關閉最後一個分組時，將自動儲存工作區，並在 EmEditor 啟動時自動恢復。 |
| **儲存還原並保留復原信息** | 當即將關閉最後一個分組時，將自動儲存工作區，並在 EmEditor 啟動時自動恢復。同時還會儲存和恢復復原和重做信息。 |
| **儲存還原並保留復原信息，且不提示儲存檔案** | 當即將關閉最後一個分組時，將自動儲存工作區，並在 EmEditor 啟動時自動恢復。同時還會儲存和恢復復原和重做信息，並在關閉最後一個分組時不顯示儲存檔案的提示。 |

### 「退出前不再詢問」核取方塊

如果勾選了該核取方塊，則在退出時不會顯示選擇工作區參數的對話方塊。

### 「不包含使用暫時資料夾的檔案」核取方塊

指定工作區不應包含使用暫時資料夾的檔案。

### 「不包含超過指定行數的檔案」核取方塊

指定工作區不應包含行數超過指定數量的檔案。

### 「崩潰時儲存工作區」核取方塊

如果勾選了該核取方塊，EmEditor 會在崩潰時儲存工作區，並允許你在下次啟動 EmEditor 時恢復工作區。

### 「定期儲存工作區:」核取方塊

如果勾選了該核取方塊，EmEditor 將按指定的時間間隔定期儲存工作區，並允許你在啟動 EmEditor 時恢復工作區。

### 「不要在「儲存更改？」對話方塊中顯示「儲存工作區」按鈕」核取方塊

如果勾選了該核取方塊，EmEditor 將不在「儲存更改？」對話方塊中顯示「儲存工作區」按鈕。

### 「重置」按鈕

重置為預設設置。
