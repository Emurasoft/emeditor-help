# 安裝外掛程式

1. 從網上下載您想要安裝的外掛程式。Emurasoft 網站有很多可用的外掛程式。
2. 按照附帶的下載外掛程式的說明正確地安裝它們。通常，把一個帶 **.DLL** 副檔名的檔案複製到 EmEditor 目錄的 **外掛程式** 資料夾中 (一般而言是 C:\\Program Files\\EmEditor3\\PlugIns)， 就完成了安裝。
3. 當您運行 EmEditor 時，您安裝的外掛程式功能表項目會被添加到 **工具** 功能表下，工具列上也會出現一個按鈕。

### 提示

- 用最新版的 EmEditor 以及外掛程式可確保它們一起正常工作。
- 您可以自訂工具列按鈕，功能表和鍵盤快速鍵。
