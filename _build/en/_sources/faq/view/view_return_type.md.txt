# Q. 怎么知道每行的換行方式 (例如，CR,LF,CR+LF) ？

請在「 **自訂**」對話方塊中的「 **狀態列**」頁面上勾選「 **游標處字元的字元代碼**」。當游標在任一行的行末處時，您會看到 EmEditor 視窗中的狀態列上分別顯示 **0D+0A** (CR+LF) ，或 **0D** (僅僅CR) 或 **0A** (僅僅LF)。
另外，您可以到組態屬性中的「 **標記**」頁面上勾選 **「使用不同的標記來顯示 CR 與 LF」** 來標示不同的換行方式。由于在 EmEditor 中每一行都可以有不同的換行方式，如果您勾選了這個選項，EmEditor 會顯示每一行的換行方式。
