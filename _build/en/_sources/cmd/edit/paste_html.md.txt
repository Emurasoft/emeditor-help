# 貼上為 HTML 命令

### 摘要

> 將剪貼簿內容貼上為 HTML 格式。

### 說明

> 用 HTML 格式貼上剪貼簿中的內容。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **貼上**
\> **貼上為 HTML**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_EDIT\_PASTE\_HTML (4071)

### 巨集

#### \[JavaScript\]

> document.selection.Paste(eeCopyHTML);

#### \[VBScript\]

> document.selection.Paste eeCopyHTML
