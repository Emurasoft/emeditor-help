# Undo All command

### Summary

> Undo all the previous actions.

### Description

> Undo all the previous actions.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Undo** \> **Undo All**
- Toolbar: ![](../../images/editundo.gif) (on the arrow)
- Status Bar: None
- Default Keyboard Shortcut: None

### Plug-in Command ID

- EEID\_UNDO\_ALL (3980)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(3980);

#### \[VBScript\]

> editor.ExecuteCommandByID 3980
