# Redo All command

### Summary

> Redo all the previous actions.

### Description

> Redo all the previous actions.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Redo** \> **Redo All**
- Toolbar: ![](../../images/editredo.gif) (on the arrow)
- Status Bar: None
- Default Keyboard Shortcut: None

### Plug-in Command ID

- EEID\_REDO\_ALL (3981)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(3981);

#### \[VBScript\]

> editor.ExecuteCommandByID 3981
