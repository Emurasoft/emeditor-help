# Customize Outline command

### Summary

> Opens the Outline page of the Customize dialog box.

### Description

> Displays the Outline page of the [**Customize** dialog box](../../dlg/customize/index),
> where you can define common application settings.

### How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Outline**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_CUSTOMIZE\_OUTLINE (9054)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(9054);

#### \[VBScript\]

> editor.ExecuteCommandByID 9054
