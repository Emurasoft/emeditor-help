# Enable All Markers command

### Summary

> Enables all markers.

### Description

> Enables all markers.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **View** \> **Markers**
\> **Enable All**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_MARKER\_ENABLE\_ALL (4585)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(4585);

#### \[VBScript\]

> editor.ExecuteCommandByID 4585
