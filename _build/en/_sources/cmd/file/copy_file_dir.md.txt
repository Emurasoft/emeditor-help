# 複製檔案所在資料夾名命令

### 摘要

> 複製目前的檔案所在資料夾的完整路徑名到剪貼簿。

### 說明

> 複製目前的檔案所在資料夾的完整路徑名到剪貼簿。如果文檔沒有標題，這個命令則不能被選取。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **複製**
\> **檔案類別名**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_COPY\_FILE\_DIR (4327)

### 巨集

#### \[JavaScript\]

> document.CopyPath();

#### \[VBScript\]

> document.CopyPath
