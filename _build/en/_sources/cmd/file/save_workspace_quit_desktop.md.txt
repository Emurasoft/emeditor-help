# 儲存工作區并全部關閉 (僅目前的桌面) 命令

### 摘要

> 在目前的虛擬桌面上，儲存工作區并關閉所有打開的檔案。

### 說明

> 在目前的虛擬桌面上，儲存工作區并關閉所有打開的檔案。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **關閉**
\> **儲存工作區并全部關閉 (僅目前的桌面)**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_SAVE\_WORKSPACE\_QUIT\_DESKTOP (3938)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3938);

#### \[VBScript\]

> editor.ExecuteCommandByID 3938
