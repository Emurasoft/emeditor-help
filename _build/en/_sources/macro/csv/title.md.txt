# Title Property

Corresponds to the **Toolbar button title** text box in the [**CSV** page](../../dlg/customize/csv/index) of the Customize dialog box.

#### \[JavaScript\]

_s_ = object. **Title**;

object. **Title** = _s_;

#### \[VBScript\]

_s_ = object. **Title**

object. **Title** = _s_

## Version

Supported on EmEditor Professional Version 19.4 or later.
