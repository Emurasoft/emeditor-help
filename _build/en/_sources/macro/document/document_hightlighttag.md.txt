# HighlightTag Property

Determines whether to highlight tags.

#### \[JavaScript\]

_bHighlightTag_ = document. **HighlightTag**;

document. **HighlightTag** = _bHighlightTag_;

#### \[VBScript\]

_bHighlightTag_ = document. **HighlightTag**

document. **HighlightTag** = _bHighlightTag_

## Version

Supported on EmEditor Professional Version 4.00 or later.
