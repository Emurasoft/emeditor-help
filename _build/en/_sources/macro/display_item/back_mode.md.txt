# BackMode Property

Specifies the background color mode.

#### \[JavaScript\]

_nMode_ = item. **BackMode**;

item. **BackMode** = _nMode_;

#### \[VBScript\]

_nMode_ = item. **BackMode**

item. **BackMode** = _nMode_

## Notes

The nMode parameter should be one of the following values.

|     |     |
| --- | --- |
| 0 | Custom |
| 1 | Default |
| 2 | Transparent |

## Version

Supported on EmEditor Professional Version 7.00 or later.
