# UndoBufferSize 屬性

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **最大復原次數** 文字方塊相對應。

#### \[JavaScript\]

_b_ =
object. **UndoBufferSize**;

object. **UndoBufferSize** = _b_;

#### \[VBScript\]

_b_ =
object. **UndoBufferSize**

object. **UndoBufferSize** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
