# Add Method

Adds an item.

#### \[JavaScript\]

list. **Add**( _strName_, _strDelimiter_ );

#### \[VBScript\]

list. **Add** _strName_, _strDelimiter_

## Parameters

_strName_

Specifies a name.

_strDelimiter_

Specifies a delimiter.

## Version

Supported on EmEditor Professional Version 19.4 or later.
