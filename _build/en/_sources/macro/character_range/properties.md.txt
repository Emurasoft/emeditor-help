# Properties Property

Specifies a string representing Unicode Script and/or Unicode General Category properties of the object.

#### \[JavaScript\]

_s_ =
item. **Properties**;

item. **Properties** = _s_;

#### \[VBScript\]

_s_ =
item. **Properties**

item. **Properties** = _s_

## Version

Supported on EmEditor Professional Version 22.0 or later.
