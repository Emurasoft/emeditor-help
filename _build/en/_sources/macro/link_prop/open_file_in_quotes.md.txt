# OpenFileInQuotes 屬性

與組態屬性中 [**連結** 頁面](../../dlg/properties/link/index) 上的 **點擊引號內的檔案名打開檔案** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **OpenFileInQuotes**;

object. **OpenFileInQuotes** = _b_;

#### \[VBScript\]

_b_ =
object. **OpenFileInQuotes**

object. **OpenFileInQuotes** = _b_

## 版本

支持 EmEditor 13 或之後的版本。
