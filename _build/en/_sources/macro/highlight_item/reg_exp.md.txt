# RegExp Property

Corresponds to the **Regular**
**Expression** check box in the [**Highlight (1)** page](../../dlg/properties/highlight1/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
item. **RegExp**;

item. **RegExp** = _b_;

#### \[VBScript\]

_b_ =
item. **RegExp**

item. **RegExp** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
