# RightSide Property

Corresponds to the **Highlight**
**Right All** check box in the [**Highlight (1)** page](../../dlg/properties/highlight1/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
item. **RightSide**;

item. **RightSide** = _b_;

#### \[VBScript\]

_b_ =
item. **RightSide**

item. **RightSide** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
