# MarkProp 對象

## 屬性

|     |     |
| --- | --- |
| [CrLfDifferent](cr_lf_different) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **使用不同的標記來顯示 CR 與 LF** 核取方塊相對應。 |
| [ShowControlCharacters](show_control_characters) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示控制字元** 核取方塊相對應。 |
| [ShowEOF](show_eof) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示檔案結束標記** 核取方塊相對應。 |
| [ShowIndentGuides](show_indent_guides) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示縮進線** 核取方塊相對應。 |
| [ShowReturns](show_returns) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示換行標記** 核取方塊相對應。 |
| [ShowSpaces](show_spaces) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示空白** 核取方塊相對應。 |
| [ShowTabs](show_tabs) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示 Tab** 核取方塊相對應。 |
| [ShowUserDefinedGuides](show_user_defined_guides) | 與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示使用者自訂參考線** 核取方塊相對應。 |

## 版本

支持 EmEditor 7.00 或之後的版本。

```{toctree}
:maxdepth: 1
cr_lf_different
show_control_characters
show_db_spaces
show_eof
show_indent_guides
show_returns
show_spaces
show_tabs
show_user_defined_guides
```
