# ShowEOF Property

Corresponds to the **Show**
**End of File** check box in the
[**Marks** page](../../dlg/properties/marks/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
object. **ShowEOF**;

object. **ShowEOF** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowEOF**

object. **ShowEOF** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
