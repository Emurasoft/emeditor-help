# ShowUserDefinedGuides 屬性

與組態屬性中 [**標記** 頁面](../../dlg/properties/marks/index) 上的 **顯示使用者自訂參考線** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **ShowUserDefinedGuides**;

object. **ShowUserDefinedGuides** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowUserDefinedGuides**

object. **ShowUserDefinedGuides** = _b_

## 版本

支持 EmEditor 14.2.0 或之後的版本。
