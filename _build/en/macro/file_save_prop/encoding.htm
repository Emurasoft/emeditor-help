Encoding Property¶

Corresponds to the Encoding drop-down list box in the Save Details dialog box.

[JavaScript]¶

n = object. Encoding;

object. Encoding = n;

[VBScript]¶

n = object. Encoding

object. Encoding = n

Version¶

Supported on EmEditor Professional Version 7.00 or later.
