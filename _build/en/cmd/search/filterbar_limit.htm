Block Multiline Changes (Filter Toolbar) command¶

Summary¶

Toggles the Block Multiline Changes button on the Filter toolbar.

Description¶

Toggles the Block Multiline Changes button on the Filter toolbar.

How to Run¶

  • Default Menu: None

  • All Commands: Search > Filter Toolbar > Block Multiline Changes

  • Toolbar: (Filter toolbar)

  • Status Bar: None

  • Default Shortcut Key: None

Plug-in Command ID¶

  • EEID_FILTERBAR_LIMIT(3915)

Macros¶

[JavaScript]¶

editor.ExecuteCommandByID(3915);

[VBScript]¶

editor.ExecuteCommandByID 3915
