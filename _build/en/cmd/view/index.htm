No Wrap

Displays each line without wrapping.

Wrap by Characters

Wraps lines by a specified number of characters.

Wrap by Window

Wraps lines according to window size.

Wrap by Page

Wraps lines according to page width.

Main Menu

Shows or hides the main menu.

Toggle Toolbar

Shows or hides a specified toolbar (multiple items).

Toggle All Basic Toolbars

Shows or hides all basic toolbars.

Toggle All Special Toolbars

Shows or hides all special toolbars.

Toggle Function Bar

Shows or hides the function bar.

Toggle Status Bar

Shows or hides the status bar.

Toggle Output Bar

Shows or hides the output bar.

Clear Output Bar

Clears the contents of the Output Bar.

Enable Shortcut Keys (Output Bar)

Enables or disables shortcut keys in the Output Bar.

Wrap by Window (Output Bar)

Wraps lines according to window size in the Output Bar.

Syntax Checker

Shows the syntax checker bar and check syntax, or hides the syntax checker bar.

Toggle Large File Controller

Shows or hides the large file controller.

Toggle Outline Bar and Guide

Shows or hides the outline bar and the outline guide.

Toggle Outline Bar

Shows or hides the outline bar

Toggle Outline Guide

Shows or hides the outline guide.

Outlining (Popup Menu)

Shows a popup menu to specify outline settings.

Notifications

Show or hides Notifications.

Layout 1 (Focus Mode)

Toggles Layout 1 (Focus Mode by default).

Layout

Shows or hides the macros toolbar.

Shows or hides the markers toolbar.

Shows or hides the CSV toolbar.

Shows or hides the filter toolbar.

Shows or hides the find toolbar.

Cell Toolbar

Shows or hides the cell toolbar.

Cell Toolbar Options (Popup)

Shows a popup menu to set options for the cell toolbar.

Automatically Show/Hide Cell Toolbar

Automatically shows or hides the cell toolbar.

Automatically Set Focus (Cell Toolbar)

Automatically sets focus to the cell toolbar.

Automatically Adjust Height (Cell Toolbar)

Automatically adjusts the height of the cell toolbar.

Show Toolbar Titles

Shows toolbar titles.

Lock Toolbars

Locks all toolbars.

Refresh Toolbars

Refresh all toolbars.

Fonts List

Selects a recently used font (multiple items).

Font

Shows a popup menu to select a font.

Increase Font Size

Increases the display font size.

Decrease Font Size

Decreases the display font size.

Reset Font Size

Resets the display font size.

Set Current Font Size as Default

Sets the current font size as the default size.

Display in Normal font

Displays text in a Normal font.

Display in Arabic font

Displays text in a Arabic font.

Display in Baltic font

Displays text in a Baltic font.

Display in Central European font

Displays text in a Central European font.

Display in Simplified Chinese font

Displays text in a Simplified Chinese font.

Display in Traditional Chinese font

Displays text in a Traditional Chinese font

Display in Cyrillic font

Displays text in a Cyrillic font.

Display in Greek font

Displays text in a Greek font.

Display in Hebrew font

Displays text in a Hebrew font.

Display in Japanese font

Displays text in a Japanese font.

Display in Korean font

Displays text in a Korean font.

Display in Thai font

Displays text in a Thai font.

Display in Turkish font

Displays text in a Turkish font.

Display in Vietnamese font

Displays text in a Vietnamese font.

Display in Western European font

Displays text in a Western European font.

Display in OEM/DOS font

Displays text in a OEM/DOS font.

Customize Font

Change font settings.

Use DirectWrite

Specifies whether to use the DirectWrite for text rendering.

Character Code Value

Displays the Unicode character value.

Line Numbers

Shows or hides line numbers.

Ruler

Shows or hides the ruler.

Newline Characters

Shows or hides newline characters.

EOF

Shows or hides the end of file mark.

Tabs

Shows or hides tab marks.

Spaces

Shows or hides space marks.

Wide Spaces

Shows or hides wide space marks.

CR and LF with Different Marks

Shows CR and LF with different marks.

Control Characters

Shows or hides control character marks.

Marks

Toggles newline characters, EOF and tab marks.

All Marks

Shows or hides all marks.

Indent Guides

Shows or hides indent guides.

User-Defined Guides

Shows or hides user-defined guides.

Add/Remove User-Defined Guide

Add or remove user-defined guide at the cursor position (or at the pointed position on the ruler).

Tab Columns

Displays a popup menu to select tab columns.

Indent Columns

Displays a popup menu to select indent columns.

Auto Indent

Enables or disables the auto indent.

Insert Spaces for Tabs

Inserts spaces for tabs.

Wrap Indent

Enables or disables the wrap indent.

Tag Links

Shows hyperlinks to tags.

Arrange by Name

Arranges tabs by name.

Arrange by Type

Arranges tabs by type.

Arrange by Modified

Arranges tabs by modified time.

Arrange by Active

Arranges tabs by active order.

Arrange in Ascending Order

Arranges in ascending order.

Arrange in Descending Order

Arranges in descending order.

Auto Arrange

Arranges tabs automatically.

Enable All Markers

Enables all markers.

Disable All Markers

Disables all markers.

Delete All Markers

Deletes all markers.

Count Marker Matches

Counts matches of all markers in the current document.

Clear Marker Match Count

Clears match counts of all markers in the current document.

Clear Markers for Saved Lines

Clears all markers indicating saved lines.

Show Last Message

Shows the last received message (available only in the tray icon).
