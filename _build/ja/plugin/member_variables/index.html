CETLFrame 内部で必要なメンバー変数¶

CETLFrame で定義される必要のある変数があります。_ID で始まる変数はリソース用の ID が必要です。

ID

説明

_IDS_NAME

[プラグインの設定] ダイアログ ボックスに名前として表示されます。

_IDS_MENU

[ツール] メニューの [プラグイン] で表示されます。

_IDS_STATUS

プラグイン ツール バーのツールチップとしてとステータス バーに表示されます。

_IDS_VER

[プラグインの設定] ダイアログ ボックスにバージョンとして表示されます。

_USE_LOC_DLL

1 は多言語リソース用の独立した DLL が使われることを示します。そうでなければ、0 になります。これらの DLL はプラグインの mui サブ フォルダに置かれます。

_IDB_BITMAP

16色、16x16 ピクセル。他の色数、サイズが定義されていない場合は、このビットマップが使用されることもある。万一 EmEditor がアイコンをロードしない場合は、ペイントでエクスポートしてみてください。次のアイコン イメージは異なるツール バーの設定で最適に表示するために使用されます。

_IDB_16C_24

16色、24x24 ピクセル。

_IDB_TRUE_16_DEFAULT

最高色 (24 または 32 ビット)、16x16 ピクセル。

_IDB_TRUE_16_HOT

最高色、16x16 ピクセル。このイメージはカーソルがアイコンの上に置かれた時に表示されます。

_IDB_TRUE_16_BW

最高色、16x16 ピクセル、モノクロ。プラグインが無効の時に表示されます。

_IDB_TRUE_24_DEFAULT

最高色、24x24 ピクセル。

_IDB_TRUE_24_HOT

最高色、24x24 ピクセル。このイメージはカーソルがアイコンの上に置かれた時に表示されます。

_IDB_TRUE_24_BW

最高色、24x24 ピクセル、モノクロ。プラグインが無効の時に表示されます。

_MASK_TRUE_COLOR

最高色のアイコンで透明になる色。RGB(r,g,b) の形式でこの値を指定してください。

_ALLOW_OPEN_SAME_GROUP

ファイルを同じグループ内で開くことを許す場合に TRUE。

_ALLOW_MULTIPLE_INSTANCES

プラグインが複数インスタンスをサポートする場合に TRUE。プラグインが 2 個以上のフレームで同時に動作することが許される場合には、このメッセージは TRUE を返す必要があります。複数インスタンスが実行している間、グローバル変数は共有されることに注意してください。

_MAX_EE_VERSION

対応するもっとも新しい EmEditor のバージョン番号 * 1000。

_MIN_EE_VERSION

対応するもっとも古い EmEditor のバージョン番号 * 1000。

_SUPPORT_EE_PRO

EmEditor Professional をサポートする場合に TRUE。

_SUPPORT_EE_STD

EmEditor Standard をサポートする場合に TRUE。
