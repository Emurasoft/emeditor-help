Regex オブジェクト¶

プロパティ¶

Engine

正規表現エンジンの種類を設定または取得します。

Fuzzy

あいまい一致のフラグを設定または取得します。

Global

グローバル フラグを設定または取得します。

IgnoreCase

正規表現が大文字小文字の区別を無視するかどうかを設定または取得します。

OnlyWord

正規表現が単語全体に一致するかどうかを設定または取得します。

Pattern

正規表現を設定または取得します。

SeparateCrLf

正規表現が CR と LF を区別するかどうかを設定または取得します。

メソッド¶

Find

指定する文字列から正規表現を検索し、見つかったら Matches コレクションを返します。

Replace

指定する文字列から正規表現を検索し、指定する文字列で置換します。

Test

正規表現が指定する文字列にマッチするかどうかをテストします。

バージョン¶

EmEditor Professional Version 15.9 以上で利用できます。
