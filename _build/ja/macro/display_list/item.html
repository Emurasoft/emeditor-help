Item プロパティ¶

指定のインデックスの DisplayItem オブジェクト を取得します。

[JavaScript]¶

item = list. Item( Index );

[VBScript]¶

item = list. Item( Index )

パラメータ¶

Index

アイテムのインデックスを １ から始まる整数として指定します。

オブジェクトが色の一覧の場合、次の値のいずれかになります。
