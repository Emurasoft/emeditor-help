# \[次のグループに移動\] コマンド

### 概要

> 現在の文書を次のグループに移動して結合します。

### 説明

> 現在の文書を次のグループに移動して結合します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[次のグループに移動\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MOVE\_NEXT\_GROUP (4397)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4397);

#### \[VBScript\]

> editor.ExecuteCommandByID 4397
