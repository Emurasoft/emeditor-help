# \[前のページにスクロール\] コマンド

> 前のページにスクロールします。

### 説明

> 前のページにスクロールします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[スクロール\] \- \[前のページ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SCROLL\_PAGE\_UP (3866)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3866);

#### \[VBScript\]
