# \[グループのすべての右を閉じる\] コマンド

### 概要

> 現在のグループのすべての右側の文書を閉じます。

### 説明

> 現在のグループのすべての右側の文書を閉じます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[グループ\] \- \[グループのすべての右を閉じる\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_GROUP\_CLOSE\_RIGHT (4390)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4390);

#### \[VBScript\]

> editor.ExecuteCommandByID 4390
