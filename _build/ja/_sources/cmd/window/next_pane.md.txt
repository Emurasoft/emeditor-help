# \[次のペイン\] コマンド

### 概要

> 次の分割されたペインに切り替えます。

### 説明

> ウィンドウが分割されている場合、次の分割されたペインに切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ウィンドウ\] \- \[ペイン\] \- \[次のペイン\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: F6

### プラグイン コマンド ID

- EEID\_NEXT\_PANE (4214)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4214);

#### \[VBScript\]

> editor.ExecuteCommandByID 4214
