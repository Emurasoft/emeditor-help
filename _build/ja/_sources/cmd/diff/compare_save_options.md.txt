# \[オプションの保存\] コマンド

### 概要

> 比較または同期スクロールのオプションを保存します。

### 説明

> 比較または同期スクロールのオプションを保存します。

### 実行方法

- 既定のメニュー: \[比較\] \- \[オプションの保存\]
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[オプションの保存\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_COMPARE\_SAVE\_OPTIONS (4508)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4508);

#### \[VBScript\]

> editor.ExecuteCommandByID 4508
