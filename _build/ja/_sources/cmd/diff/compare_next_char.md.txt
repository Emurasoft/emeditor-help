# \[次の変更文字\] コマンド

### 概要

> 次の変更文字にジャンプします。

### 説明

> 次の変更文字にジャンプします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[比較\] \- \[次の変更文字\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F7

### プラグイン コマンド ID

- EEID\_COMPARE\_NEXT\_CHAR (4090)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4090);

#### \[VBScript\]

> editor.ExecuteCommandByID 4090
