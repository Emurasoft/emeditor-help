# \[ローカル ヘルプ\] コマンド

### 概要

> ローカル ヘルプを表示します。

### 説明

> ローカル ヘルプを表示します。

### 実行方法

- 既定のメニュー: \[ヘルプ\] \- \[ローカル ヘルプ\]
- [すべてのコマンド](../../glossary/allcommands): \[ヘルプ\] \- \[ローカル ヘルプ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HELP\_LOCAL (3933)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3933);

#### \[VBScript\]

> editor.ExecuteCommandByID 3933
