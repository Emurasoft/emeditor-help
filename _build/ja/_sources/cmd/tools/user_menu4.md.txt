# \[ユーザー メニュー (4)\] コマンド

### 概要

> ユーザー メニュー (4) を表示します。2 ストローク キーの最初のショートカット キーとして使用されます。

### 説明

> ユーザー メニュー (4) を表示します。2 ストローク キーの最初のショートカット キーとして使用されます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[ユーザー メニュー\] \- \[ユーザー メニュー (4)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_USER\_MENU1 (4432)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4432);

#### \[VBScript\]

> editor.ExecuteCommandByID 4432
