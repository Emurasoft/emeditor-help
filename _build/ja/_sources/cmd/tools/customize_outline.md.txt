# \[アウトラインのカスタマイズ\] コマンド

### 概要

> \[カスタマイズ\] ダイアログ ボックスの \[アウトライン\] ページを開きます。

### 説明

> [\[カスタマイズ\] ダイアログ ボックス](../../dlg/customize/index) の \[アウトライン\] ページを表示して、アプリケーション共通の設定をできるようにします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[ツール\] \- \[カスタマイズ\] \-
\[カスタマイズ\] \- \[アウトライン\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CUSTOMIZE\_OUTLINE (9054)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(9054);

#### \[VBScript\]

> editor.ExecuteCommandByID 9054
