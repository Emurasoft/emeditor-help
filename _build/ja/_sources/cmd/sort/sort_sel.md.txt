# \[選択範囲の分割文字列の並べ替え/重複の削除\] コマンド

### 概要

> 選択範囲の分割文字列を並べ替え、または重複した分割文字列を削除します。

### 説明

> 選択範囲の分割文字列を並べ替え、または重複した分割文字列を削除します。このコマンドを選択すると、 [\[選択範囲の並べ替え\] ダイアログ ボックス](../../dlg/sort_sel/index) が表示されます。

### 実行方法

- 既定のメニュー: \[並べ替え\] \- \[選択範囲の分割文字列の並べ替え/重複の削除\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[選択範囲の分割文字列の並べ替え/重複の削除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_SEL (23237)

### マクロ

#### \[JavaScript\]

#### \[JavaScript\]

> editor.ExecuteCommandByID(23237);

#### \[VBScript\]

> editor.ExecuteCommandByID 23237
