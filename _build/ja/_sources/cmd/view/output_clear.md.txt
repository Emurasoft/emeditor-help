# \[アウトプットのクリア\] コマンド

### 概要

> アウトプット バーの中身を消去します。

### 説明

> アウトプット バーの中身を消去します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\]\- \[アウトプットバー\]  \- \[アウトプットのクリア\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_OUTPUT\_BAR (4452)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4452);

#### \[VBScript\]

> editor.ExecuteCommandByID 4452
