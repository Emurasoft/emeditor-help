# \[すべてのマーカーを無効にする\] コマンド

### 概要

> すべてのマーカーを無効にします。

### 説明

> すべてのマーカーを無効にします。

### 方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[マーカー\] \- \[すべて無効\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MARKER\_DISABLE\_ALL (4586)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4586);

#### \[VBScript\]

> editor.ExecuteCommandByID 4586
