# \[マーカーに一致する文字列の数をクリアする\] コマンド

### 概要

> 現在の文書でマーカーに一致する文字列の数をクリアします。

### 説明

> 現在の文書でマーカーに一致する文字列の数をクリアします。

### 方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[表示\] \- \[マーカー\] \- \[一致する文字列の数をクリア\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MARKER\_CLEAR\_COUNT (4043)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4043);

#### \[VBScript\]

> editor.ExecuteCommandByID 4043
