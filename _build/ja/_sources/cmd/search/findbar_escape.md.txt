# \[エスケープ シーケンスを使用する (検索ツール バー)\] コマンド

### 概要

> 検索ツール バーの \[エスケープ シーケンスを使用する\] ボタンのオン/オフを切り替えます。

### 説明

> 検索ツール バーの \[エスケープ シーケンスを使用する\] ボタンのオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[検索\] \- \[検索ツール バー\] \- \[エスケープ シーケンスを使用する\]
- ツール バー: ![](../../images/find_escape.png) (検索ツール バー)
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_FINDBAR\_ESCAPE (4575)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4575);

#### \[VBScript\]

> editor.ExecuteCommandByID 4575
