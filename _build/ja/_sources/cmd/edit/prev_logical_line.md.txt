# \[前の論理行\] コマンド

### 概要

> 前の論理行へ移動します。

### 説明

> 前の論理行の行頭へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[垂直にカーソル移動\] \- \[前の論理行\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Page Up

### プラグイン コマンド ID

- EEID\_PREV\_LOGICAL\_LINE (4269)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4269);

#### \[VBScript\]

> editor.ExecuteCommandByID 4269
