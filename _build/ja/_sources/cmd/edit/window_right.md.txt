# \[ウィンドウの右へ\] コマンド

### 概要

> カーソルをウィンドウの右端へ移動します。

### 説明

> カーソル位置をウィンドウの右端へ移動します。水平スクロールの位置は変更しません。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[ウィンドウ右\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_WINDOW\_RIGHT (4294)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4294);

#### \[VBScript\]

> editor.ExecuteCommandByID 4294
