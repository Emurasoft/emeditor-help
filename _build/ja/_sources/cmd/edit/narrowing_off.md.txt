# \[部分編集の解除\] コマンド

### 概要

> 文書全体を編集可能にします。

### 説明

> 文書全体を編集可能にします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[部分編集\] \- \[部分編集の解除\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_NARROWING\_OFF (4458)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4458);

#### \[VBScript\]

> editor.ExecuteCommandByID 4458
