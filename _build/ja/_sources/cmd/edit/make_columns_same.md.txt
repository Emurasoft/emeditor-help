# \[列数をそろえる\] コマンド

### 概要

> CSV文書の列数をそろえます。

### 説明

> CSV文書の列数をそろえます。

### 実行方法

- 既定のメニュー: \[CSV\] - \[高度\] - \[列数をそろえる\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[高度\] - \[列数をそろえる\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_MAKE\_COLUMNS\_SAME (3993)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3993);

#### \[VBScript\]

> editor.ExecuteCommandByID 3993
