# \[Tabを挿入\] コマンド

### 概要

> カーソル位置に Tab を挿入します。

### 説明

> カーソル位置に Tab を挿入します。

### 実行方法

- 既定のメニュー: \[挿入\] \- \[Tab\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[Tab\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_INSERT\_TAB (3878)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3878);

#### \[VBScript\]

> editor.ExecuteCommandByID 3878
