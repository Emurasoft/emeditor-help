# \[ヘディングの設定/解除 (ウィンドウ枠の固定/解除)\] コマンド

### 概要

> カーソル位置の上の行と左の列を非スクロール領域のヘディングとして設定または解除します。

### 説明

> カーソル位置の上の行と左の列を非スクロール領域のヘディングとして設定または解除します。

### 実行方法

- 既定のメニュー: \[CSV\] - \[ヘディング (ウィンドウ枠の固定)\] - \[ヘディングの設定/解除 (ウィンドウ枠の固定/解除)\]
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[ヘディング (ウィンドウ枠の固定)\] - \[ヘディングの設定/解除 (ウィンドウ枠の固定/解除)\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_HEADING\_TOGGLE (3880)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(3880);

#### \[VBScript\]

> editor.ExecuteCommandByID 3880
