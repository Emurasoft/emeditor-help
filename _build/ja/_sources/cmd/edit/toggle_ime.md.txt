# \[IMEを開く\] コマンド

### 概要

> IME オン/オフを切り替えます。

### 説明

> IME のオン/オフを切り替えます。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[IMEを開く\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+F11

### プラグイン コマンド ID

- EEID\_TOGGLE\_IME (4198)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4198);

#### \[VBScript\]

> editor.ExecuteCommandByID 4198
