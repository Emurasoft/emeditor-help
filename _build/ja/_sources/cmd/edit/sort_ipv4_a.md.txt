# \[IPv4アドレス昇順に並べ替え\] コマンド

### 概要

> 現在の列のIPv4アドレスを昇順で並べ替えます。

### 説明

> 現在の列のIPv4アドレスを昇順で並べ替えます。

### 実行方法

- 既定のメニュー: \[並べ替え\] \- \[IPv4アドレス昇順に並べ替え\]
- [すべてのコマンド](../../glossary/allcommands): \[並べ替え\] \- \[IPv4アドレス昇順に並べ替え\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_SORT\_IPV4\_A (4034)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4034);

#### \[VBScript\]

> editor.ExecuteCommandByID 4034
