# \[逆立ち感嘆符を挿入\] コマンド

### 概要

> 逆立ち感嘆符を挿入します。

### 説明

> カーソル位置に逆立ち感嘆符 ¡ を挿入します。

### 実行方法

- 既定のメニュー: \[挿入\] \- \[記号\] \- \[逆立ち感嘆符\]
- [すべてのコマンド](../../glossary/allcommands): \[挿入\] \- \[記号\] \- \[逆立ち感嘆符\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Alt+Ctrl+Shift+1

### プラグイン コマンド ID

- EEID\_INSERT\_INVERTED\_EXCLAMATION (4313)

### マクロ

#### \[JavaScript\]

> document.selection.Text="¡";

#### \[VBScript\]

> document.selection.Text="¡"
