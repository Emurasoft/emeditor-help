# \[選択範囲を下端へ (セル選択モードのみ)\] コマンド

### 概要

> 選択範囲を現在のデータ領域の下端へ移動します。

### 説明

> 選択範囲を現在のデータ領域の下端へ移動します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[選択範囲を下端へ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Ctrl+Shift+Down (セル選択モードのみ)

### プラグイン コマンド ID

- EEID\_SHIFT\_BOTTOM\_EDGE (4024)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4024);

#### \[VBScript\]

> editor.ExecuteCommandByID 4024
