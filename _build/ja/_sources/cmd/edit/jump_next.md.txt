# \[次へジャンプ\] コマンド

### 概要

> アウトプット バーの次の場所にジャンプします。

### 説明

> アウトプット バーの次の場所にジャンプします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[高度な操作\] \- \[次へジャンプ\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: F4

### プラグイン コマンド ID

- EEID\_JUMP\_NEXT (4442)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4442);

#### \[VBScript\]

> editor.ExecuteCommandByID 4442
