# \[タブ挿入/インデント\] コマンド

### 概要

> 選択されていなければタブを挿入し、選択されていれば行インデントを増やします。

### 説明

> 改行コードを含む複数行が選択されていなければカーソル位置にタブ文字を挿入します。改行コードを含む複数行が選択されていれば、その行の最初にタブ文字を挿入して行インデントを増やします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[水平にカーソル移動\] \- \[タブ挿入/インデント\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: Tab

### プラグイン コマンド ID

- EEID\_TAB (4188)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4188);

#### \[VBScript\]

> editor.ExecuteCommandByID 4188
