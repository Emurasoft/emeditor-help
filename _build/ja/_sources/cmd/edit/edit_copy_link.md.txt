# \[リンクをコピー\] コマンド

### 概要

> カーソル位置のハイパーリンクをクリップボードにコピーします。

### 説明

> カーソル位置に存在するハイパーリンクを、クリップボードにコピーします。この操作の後、別の場所にカーソルを移動して
> [\[貼り付け\] コマンド](edit_paste) を実行することにより、ハイパーリンクを他の場所にコピーすることができます。

### 実行方法

- 既定のメニュー: \[編集\] \- \[リンクをコピー\]
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[コピー\] \- \[リンクをコピー\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_EDIT\_COPY\_LINK (4140)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID (4140);

#### \[VBScript\]

> editor.ExecuteCommandByID 4140
