# \[上のセル (セル選択モードのみ)\] コマンド

### 概要

> CSV文書で上のセルにジャンプします。

### 説明

> CSV文書で上のセルにジャンプします。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[CSV\] - \[セル選択モードのみ\] - \[上のセル\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_CSV\_UP (4079)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID(4079);

#### \[VBScript\]

> editor.ExecuteCommandByID 4079
