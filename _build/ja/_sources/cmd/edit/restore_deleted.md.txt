# \[削除した文字列を復元\] コマンド

### 概要

> 以前削除した文字列をカーソル位置に復元します。

### 説明

> 以前削除した文字列をカーソル位置に復元します。

### 実行方法

- 既定のメニュー: なし
- [すべてのコマンド](../../glossary/allcommands): \[編集\] \- \[貼り付け\] \- \[削除した文字列を復元\]
- ツール バー: なし
- ステータス バー: なし
- 既定のショートカット: なし

### プラグイン コマンド ID

- EEID\_RESTORE\_DELETED (4419)

### マクロ

#### \[JavaScript\]

> editor.ExecuteCommandByID (4419);

#### \[VBScript\]

> editor.ExecuteCommandByID 4419
