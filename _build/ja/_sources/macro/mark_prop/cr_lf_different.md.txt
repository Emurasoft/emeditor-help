# CrLfDifferent プロパティ

設定のプロパティの [**\[記号\]** タブ](../../dlg/properties/marks/index) の **\[CRとLFを区別して表示\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ = object. **CrLfDifferent**;

object. **CrLfDifferent** = _b_;

#### \[VBScript\]

_b_ = object. **CrLfDifferent**

object. **CrLfDifferent** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
