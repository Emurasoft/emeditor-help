# ScriptEnd プロパティ

設定のプロパティの [**\[強調(2)\]** タブ](../../dlg/properties/highlight2/index) の \[スクリプト\] グループ \- \[終了\] テキスト ボックスに相当します。

#### \[JavaScript\]

_str_ = object. **ScriptEnd**;

object. **ScriptEnd** = _str_;

#### \[VBScript\]

_str_ = object. **ScriptEnd**

object. **ScriptEnd** = _str_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
