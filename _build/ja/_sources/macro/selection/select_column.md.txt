# SelectColumn メソッド

CSV モードで指定された列を選択します。

#### \[JavaScript\]

document.selection. **SelectColumn**( _iColumn_, \[ \[ _iColumn2_ \], _nFlags_ \] );

#### \[VBScript\]

document.selection. **SelectColumn** _iColumn_, \[ \[ _iColumn2_ \], _nFlags_ \]

## 引数

_nFlags_

0 または次の値を指定します。

|     |     |
| --- | --- |
| eeNoHeadings | 選択範囲からヘディングを除外します。 |

## 戻り値

戻り値はありません。

## バージョン

EmEditor Professional Version 21.0 以上で利用できます。
