# FastCursorMove プロパティ

設定のプロパティの [**\[スクロール\]** タブ](../../dlg/properties/scroll/index) の
**\[カーソル移動の高速化\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **FastCursorMove**;

object. **FastCursorMove** = _b_;

#### \[VBScript\]

_b_ =
object. **FastCursorMove**

object. **FastCursorMove** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
