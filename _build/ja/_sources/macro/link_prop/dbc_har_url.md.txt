# DBCharURL プロパティ

設定のプロパティの [**\[リンク\]** タブ](../../dlg/properties/link/index) の **\[全角文字をURLとして認識する\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **DBCharURL**;

object. **DBCharURL** = _b_;

#### \[VBScript\]

_b_ =
object. **DBCharURL**

object. **DBCharURL** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
