# QuoteMarks プロパティ

設定のプロパティの [**\[表示\]** タブ](../../dlg/properties/display/index) の **\[引用マーク\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_s_ = object. **QuoteMarks**;

object. **QuoteMarks** = _s_;

#### \[VBScript\]

_s_ = object. **QuoteMarks**

object. **QuoteMarks** = _s_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
