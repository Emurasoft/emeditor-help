# Escape プロパティ

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[エスケープ文字を使用\] チェック ボックス に相当します。

#### \[JavaScript\]

_b_ = object. **Escape**;

object. **Escape** = _b_;

#### \[VBScript\]

_b_ = object. **Escape**

object. **Escape** = _b_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
