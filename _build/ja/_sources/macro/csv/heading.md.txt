# Heading プロパティ

\[カスタマイズ\] ダイアログ ボックスの [\[CSV\] ページ](../../dlg/customize/csv/index) の \[列ヘディング\] テキスト ボックス に相当します。

#### \[JavaScript\]

_n_ = object. **Heading**;

object. **Heading** = _n_;

#### \[VBScript\]

_n_ = object. **Heading**

object. **Heading** = _n_

## バージョン

EmEditor Professional Version 19.4 以上で利用できます。
