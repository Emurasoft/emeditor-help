# LineColumnMode プロパティ

設定のプロパティの [**\[基本\]** タブ](../../dlg/properties/general/index) の
**\[行、桁の表示方法\]** ドロップダウン リスト ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **LineColumnMode**;

object. **LineColumnMode** = _b_;

#### \[VBScript\]

_b_ =
object. **LineColumnMode**

object. **LineColumnMode** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
