# UndoBufferSize プロパティ

設定のプロパティの [**\[基本\]** タブ](../../dlg/properties/general/index) の **\[\[元に戻す\] 最大回数\]** テキスト ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **UndoBufferSize**;

object. **UndoBufferSize** = _b_;

#### \[VBScript\]

_b_ =
object. **UndoBufferSize**

object. **UndoBufferSize** = _b_

## バージョン

EmEditor Professional Version 7.00 以上で利用できます。
