# CamelCase プロパティ

設定のプロパティの [**\[スペル チェック\]** タブ](../../dlg/properties/spell/index) の **\[キャメル ケースに対応\]** チェック ボックスに相当します。

#### \[JavaScript\]

_b_ =
object. **CamelCase**;

object. **CamelCase** = _b_;

#### \[VBScript\]

_b_ =
object. **CamelCase**

object. **CamelCase** = _b_

## バージョン

EmEditor Professional Version 10.00 以上で利用できます。
