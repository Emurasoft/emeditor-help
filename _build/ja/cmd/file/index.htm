[ファイル] カテゴリ¶

コマンド名

説明

Textを新規作成

新規に Text ファイルを作成します。

新規作成 (ポップアップ メニュー)

ポップアップ メニューを表示して指定する設定で新規にファイルを作成します。

新規作成して貼り付け

新規にファイルを作成して貼り付けます。

新規作成して引用付き貼り付け

新規にファイルを作成して引用マークを付けて貼り付けます。

新規作成して引用と改行コード付き貼り付け

新規にファイルを作成して引用マークと改行コードを付けて貼り付けます。

トレイ アイコンを表示

新しいトレイ アイコンを表示します。

指定した設定で新規作成の一覧

指定した設定で新規にファイルを作成します (複数項目)。

開く

既存のファイルを開きます。

閉じて開く

ファイルを閉じてから既存のファイルを開きます。

最も最近閉じたファイル

最も最近閉じたファイルを開きます。

上書き保存

ファイルを保存します。

名前を付けて保存

ファイルに名前を付けて保存します。

名前を変更

ファイル名を変更します。

すべて保存

開いているファイルをすべて保存します。

すべてエンコードを指定して保存

無題の文書を除き、すべての開いている文書を指定するフォルダ、エンコード、改行コード方法で保存します。

高度に開く

既存のファイルを高度なオプションで開きます。

システム既定エンコードで保存

システム既定エンコードで保存します。

UTF-16LEで保存

ファイルを Unicode (UTF-16LE) で保存します。

UTF-16LE(BOM付き)で保存

ファイルをUTF-16LE(BOM付き)で保存します。

UTF-16LE(BOM無し)で保存

ファイルをUTF-16LE(BOM無し)で保存します。

UTF-16BEで保存

ファイルを Unicode (UTF-16BE) で保存します。

UTF-16BE(BOM付き)で保存

ファイルをUTF-16BE(BOM付き)で保存します。

UTF-16BE(BOM無し)で保存

ファイルをUTF-16BE(BOM無し)で保存します。

UTF-8で保存

ファイルを Unicode (UTF-8) で保存します。

UTF-8(BOM付き)で保存

ファイルをUTF-8(BOM付き)で保存します。

UTF-8(BOM無し)で保存

ファイルをUTF-8(BOM無し)で保存します。

UTF-7で保存

ファイルを Unicode (UTF-7) で保存します。

バイナリ(アスキー表示)で保存

バイナリ(アスキー表示) で保存します。

バイナリ(16進数表示)で保存

ファイルをバイナリ (16進数表示) で保存します。

日本語Shift JISで保存

日本語 Shift JIS で保存します。

日本語JISで保存

日本語 JIS で保存します。

日本語EUCで保存

日本語 EUC で保存します。

保存エンコードの一覧

指定したエンコードで保存します (複数項目)。

CR+LFで保存

CR+LF (Windows 形式) で保存します。

CRのみで保存

CR のみ (Macintosh 形式) で保存します。

LFのみで保存

LF のみ (Unix 形式) で保存します。

ファイル パス名をコピー

ファイルの完全パス名をコピーしてクリップボードに保存します。

ファイル ディレクトリ名をコピー

ファイルのディレクトリ名をコピーしてクリップボードに保存します。

同じエンコードで読み直し

ファイルを同じエンコードで読み直します。

読み直し (ポップアップ メニュー)

ポップアップ メニューを表示して指定するエンコードで読み直します。

自動検出で読み直し

すべてのエンコードを自動検出して読み直します。

システム既定エンコードで読み直し

システム既定エンコードで読み直します。

UTF-16LEで読み直し

ファイルを Unicode (UTF-16LE) で読み直します。

UTF-16BEで読み直し

ファイルを Unicode (UTF-16BE) で読み直します。

UTF-8で読み直し

ファイルを Unicode (UTF-8) で読み直します。

UTF-7で読み直し

ファイルを Unicode (UTF-7) で読み直します。

バイナリ(アスキー表示)で読み直し

ファイルをバイナリ (アスキー表示) で読み直します。

バイナリ(16進数表示)で読み直し

ファイルをバイナリ (16進数表示) で読み直します。

日本語Shift JISで読み直し

ファイルを日本語 Shift JIS で読み直します。

日本語JISで読み直し

ファイルを日本語 JIS で読み直します。

日本語EUCで読み直し

ファイルを日本語 EUC で読み直します。

読み直しエンコードの一覧

ファイルを指定したエンコードで読み直します (複数項目)。

印刷

ファイルを印刷します。

すぐに印刷

通常使うプリンタでファイルをすぐに印刷します。

印刷プレビュー

この文書の印刷方法を表示します。

既定のワークスペースを復元します。

既定のワークスペースを保存します。

既定のワークスペースを消去します。

ワークスペースを開く

ワークスペースを上書き保存

ワークスペースを名前を付けて保存

ワークスペースを名前を付けて保存します。

保存して閉じる

ファイルを保存して閉じます。

閉じる

すべて保存して閉じる(現在のデスクトップのみ)

すべて閉じる(現在のデスクトップのみ)

すべての文書を閉じます。

すべて閉じる

すべて保存しないで閉じる

すべての文書を保存しないで閉じます。

ワークスペースを保存してすべて保存して閉じる(現在のデスクトップのみ)

ワークスペースを保存してすべてのファイルを保存してすべて閉じます。

ワークスペースを保存してすべて閉じる(現在のデスクトップのみ)

ワークスペースを保存してすべて閉じます。

ワークスペースを保存してすべて保存して閉じる

ワークスペースを保存してすべて閉じる

最近使ったファイルの一覧

最近使ったフォルダの一覧
