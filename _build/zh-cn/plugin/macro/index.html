插件参考 < 内联函数

内联函数¶

Editor_ActivateTemp

激活临时文本。

Editor_AddRef

递增插件的引用号。

Editor_AutoFill

对 CSV 文档执行自动填充或快速填充操作。

Editor_BatchFindInFiles

从指定路径中的多个文件中搜索多个字符串。

Editor_BatchFindReplace

搜索或替换多个字符串。

Editor_BatchReplaceInFiles

在指定位置替换多个文件中的多个字符串。

Editor_CloseTemp

关闭临时文本。

Editor_Compare

比较两个文件。

Editor_Convert

转换字符。

Editor_ConvertCsv

转换当前文档的 CSV 格式。

Editor_CustomBarOpen

打开自定义分栏。

Editor_CustomBarClose

关闭自定义分栏。

Editor_DevToView

转换指定位置的设备（客户端）坐标来显示坐标。

Editor_DocGetConfigA

用一个 ANSI 字符串检索选取的配置名称。

Editor_DocGetConfigW

为指定的文档检索选取的配置名称并把它作为一个 Unicode 字符串。

Editor_DocGetLines

检索指定文档的行数。

Editor_DocGetModified

检索指定文档的文本的修改状态。

Editor_DocInfo

检索或设置用于 EmEditor 的信息参数之一的值。

Editor_DocInfoEx

检索或设置用于 EmEditor 的信息参数之一的值。

Editor_DocSaveFileA

保存指定文档中的文本到另一个指定的文件中 (ANSI)。

Editor_DocSaveFileW

保存指定文档中的文本到另一个指定的文件中 (Unicode)。

Editor_DocSetConfigA

把指定文档变更为一个用 ANSI 字符串指定的配置。

Editor_DocSetConfigW

把指定文档变更为一个用 Unicode 字符串指定的配置。

Editor_DoIdle

刷新工具栏，窗口标题，标签页及其他。

Editor_EditColumn

移动，复制，删除，或合并当前 CSV 文档中的指定列。

Editor_EditTemp

把临时文本作为一个新文档打开。

Editor_EnumConfig

列举可用的配置。

Editor_EnumHighlight

列举高亮的字符串。

Editor_EmptyUndoBuffer

为撤消和重做命令清空缓冲区。

Editor_ExecCommand

执行一个指定的命令。

Editor_ExecPlugin

执行一个指定的插件。

Editor_ExtractFrequent

将常用字符串提取到新文档中。

Editor_Filter

用指定的字符串以及设定来筛选文档。

Editor_FindA

搜索一个 ANSI 字符串。

Editor_FindInFiles

在指定路径的多个文件中搜索一个字符串。

Editor_FindInFilesA

在指定路径的多个文件中搜索一个 ANSI 字符串。

Editor_FindInFilesW

在指定路径的多个文件中搜索一个 Unicode 字符串。

Editor_FindRegex

用正则表达式搜索一个字符串。

Editor_FindReplace

搜索或替换一个字符串。

Editor_FindW

搜索一个 Unicode 字符串。

Editor_Free

释放一个指定的插件。

Editor_GetAccelArray

检索快捷键的数组。

Editor_GetActiveString

检索活动字符串。

Editor_GetAnchorPos

检索选区的原点。

Editor_GetAttr

在剪贴板记录中的指定位置处删除文本。

Editor_GetCaretPos

检索当前光标位置。

Editor_GetCell

在指定的单元格内检索 Unicode 文本。

Editor_GetClip

在剪贴板记录中的指定位置处检索文本。

Editor_GetClipPos

检索剪贴板记录中的当前位置。

Editor_GetCmdID

获得插件命令 ID。

Editor_GetColor

检索指定部分的文本、背景颜色以及样式。

Editor_GetColumn

在 CSV 模式中设置一列文本。

Editor_GetConfigA

将所选配置名称检索为 ANSI 字符串。

Editor_GetConfigW

将所选配置名称检索为 Unicode 字符串。

Editor_GetDroppedFile

检索最近拖放的文件。

Editor_GetFilter

检索当前文档的筛选字符串及设定。

Editor_GetLineA

在指定行检索 ANSI 文本。

Editor_GetLines

检索当前文档的行数。

Editor_GetLineW

检索指定行的 Unicode 文本。

Editor_GetMargin

检索边距大小。

Editor_GetModified

检索文本的修改状态。

Editor_GetMultiSel

当多个选区都可用时，检索一个指定选区的信息。

Editor_GetOutlineLevel

检索指定逻辑行的大纲级别。

Editor_GetOutputString

检索输出栏中的文本。

Editor_GetPageSize

检索页面大小。

Editor_GetRef

检索一个指定插件的引用号。

Editor_GetRedraw

检索在 EmEditor 中允许或禁止重绘变更的标志。

Editor_GetScrollPos

检索滚动条的当前光标位置。

Editor_GetSelEnd

检索选区的结尾字符位置。

Editor_GetSelStart

检索选区的开始字符位置。

Editor_GetSelTextA

检索被选取的 ANSI 文本。

Editor_GetSelTextW

检索被选取的 Unicode 文本。

Editor_GetSelType

获得选区状态的类型。

Editor_GetSelTypeEx

获得选区状态的类型。

Editor_GetStatusA

检索显示在状态栏上的 ANSI 文本。

Editor_GetStatusW

检索显示在状态栏上的 Unicode 文本。

Editor_GetUnicodeName

检索指定字符或字符串的 Unicode 名。

Editor_GetVersion

返回版本号。

Editor_GetWord

检索光标位置处的一个单词。

Editor_Help

显示帮助中的指定页面。

Editor_Info

检索或设置用于 EmEditor 的信息参数之一的值。

Editor_InsertA

把一个 ANSI 字符串插入到当前光标位置处。

Editor_InsertClip

在剪贴板记录上的指定位置处插入文本。

Editor_InsertFileA

在光标处插入指定文件内容　(ANSI)。

Editor_InsertFileW

在光标处插入指定文件内容 (Unicode)。

Editor_InsertStringA

把一个 ANSI 字符串插入到当前光标位置处。取决于当前属性，这可能会改写已存在的字符串。

Editor_InsertStringW

把一个 Unicode 字符串插入到当前光标位置处。取决于当前属性，这可能会改写已存在的字符串。

Editor_InsertW

把一个 Unicode 字符串插入到当前光标位置处。

Editor_IsCharHalfOrFull

决定一个指定的字符是全角还是半角。它也可以计算指定字符串的总宽度。

Editor_Join

按指定键列,用一个与 JOIN 操作类似的方法合并两个 CSV 文档,并创建一个新文档。

Editor_KeyboardProp

显示指定命令 ID 和配置的键盘属性。

Editor_LineFromChar

检索包含指定字符索引（序列位置）的行的索引。

Editor_LineIndex

检索在 EmEditor 中一个指定行的第一个字符的字符索引。

Editor_LoadConfigA

重新载入由一个 ANSI 字符串指定名称的配置。

Editor_LoadConfigW

重新载入由一个 Unicode 字符串指定名称的配置。

Editor_LoadFileA

把一个指定文件载入到 EmEditor 中　(ANSI)。

Editor_LoadFileW

把一个指定文件载入到 EmEditor 中　(Unicode)。

Editor_LogicalToSerial

把逻辑坐标转换为序列位置。

Editor_LogicalToView

把逻辑坐标转换为显示坐标。

Editor_ManageDuplicates

删除或把重复行设为书签。

Editor_MatchRegex

决定一个字符串是否与一个指定的正则表达式相匹配。

Editor_Numbering

在鼠标位置或垂直选择处，插入编号。

Editor_OutputDir

为输出栏设置当前目录。

Editor_OutputString

追加一个字符串到输出栏中。

Editor_OverwriteA

通过在当前光标位置处改写已存在的字符串插入一个 ANSI 字符串。

Editor_OverwriteW

通过在当前光标位置处改写已存在的字符串插入一个 Unicode 字符串。

Editor_PivotTable

在 CSV 文档中创建数据透视表。

Editor_QueryStatus

查询命令的状态，无论命令是否被启用，也不管命令是否处于被选中的状态。

Editor_QueryString

查询与指定的命令相关联的字符串。

Editor_QueryStringEx

查询与指定的命令相关联的字符串。此内联函数支持超过 MAX_PATH 的长文件路径。

Editor_RearrangeColumns

重排 CSV 列。

Editor_Redraw

在 EmEditor 中允许或禁止重绘变更。

Editor_RegQueryValue

根据 EmEditor 的设定，从注册表或一个 INI 文件中查询一个值。

Editor_RegSetValue

根据 EmEditor 的设定，设一个值到注册表或一个 INI 文件中。

Editor_Release

递减插件的引用号。

Editor_RemoveClip

在剪贴板记录上的指定位置处删除文本。

Editor_ReplaceA

替换一个 ANSI 字符串。

Editor_ReplaceW

替换一个 Unicode 字符串。

Editor_ReplaceInFiles

在指定位置的多个文件中替换一个字符串。

Editor_ReplaceInFilesA

在指定位置的多个文件中搜索一个 ANSI 字符串。

Editor_ReplaceInFilesW

在指定位置的多个文件中替换一个 Unicode 字符串。

Editor_RotateClip

在剪贴板记录上旋转当前位置。

Editor_RunMacro

运行一个宏。

Editor_SaveFileA

保存文本到一个指定的文件中 (ANSI)。

Editor_SaveFileW

保存文本到一个指定的文件中 (Unicode)。

Editor_SaveTemp

保存临时文本。

Editor_SerialToLogical

把序列位置转换为逻辑坐标。

Editor_SetAnchorPos

设置选区的原点。

Editor_SetCaretPos

指定光标位置。

Editor_SetCaretPosEx

移动光标位置并且选择性地扩展选区。

Editor_SetCell

在指定的单元格中设置文本。

Editor_SetColumn

设置一列文本。

Editor_SetClipPos

在剪贴板记录上设置当前位置。

Editor_SetConfigA

变更为一个用 ANSI 字符串指定的配置。

Editor_SetConfigW

变更为一个用 Unicode 字符串指定的配置。

Editor_SetModified

变更文本的修改状态。

Editor_SetMultiSel

当多个选取内容可用时，设置指定的选取内容的信息。

Editor_SetOutlineArray

为指定的多行设置大纲级别。

Editor_SetOutlineLevel

为指定的逻辑行设置大纲级别。

Editor_SetScrollPos

指定滚动栏位置。

Editor_SetScrollPosEx

指定滚动栏位置。

Editor_SetSelLength

变更选区的字符长度。

Editor_SetSelType

设置选区状态的类型。

Editor_SetSelTypeEx

设置选区状态的类型。

Editor_SetSelView

变更选区的开始和结束位置。

Editor_SetStatusA

在状态栏上显示一条 ANSI 消息。

Editor_SetStatusW

在状态栏上显示一条 Unicode 消息。

Editor_ShowOutline

显示或隐藏大纲。

Editor_ShowTip

显示或隐藏工具提示。

Editor_Sort

排序文档。

Editor_SplitColumn

分割当前 CSV 文档的指定列。

Editor_ToolbarClose

关闭一个自定义工具栏。

Editor_ToolbarOpen

打开自定义工具栏。

Editor_ToolbarShow

显示或隐藏一个自定义工具栏。

Editor_Unpivot

Coverts columns into rows by flatterning the CSV data.

Editor_UpdateToolbar

在工具栏中更新一个按钮的状态。

Editor_ViewToDev

把显示指定位置的显示坐标转换为设备（客户端）坐标。

Editor_ViewToLogical

把一个指定位置的显示坐标转换为逻辑坐标。
