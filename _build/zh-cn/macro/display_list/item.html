Item 属性¶

为指定索引检索 DisplayItem 对象。

[JavaScript]¶

item = list. Item( Index );

[VBScript]¶

item = list. Item( Index )

参数¶

Index

指定以 1 为基准的整数为项目的索引。

如果对象是一个颜色列表，项目会是下列值之一。

eeColorNormal

一般

eeColorSelection

eeColorCurrentLine

当前行

eeColorQuote

引用行

eeColorURL

URL

eeColorMailTo

eeColorTag

包括“在文件中查找”结果的超链接的标签

eeColorSingleQuotes

带单引号 ‘…’ 的字符串

eeColorDoubleQuotes

eeColorComment
