历史记录 < Version 14.2

Version 14.2¶

2014年1月7日

一般新增功能¶

  • 你可以在 EmEidtor 中设置 用户自定义参考线。用户自定义参考线是一条条橙色的垂直虚线，让你能更容易地查看垂直排列的源代码或文本文件。

  • 右击标尺便会显示的上下文菜单上现在有了 添加/删除用户自定义参考线 命令，让你能更方便地在鼠标指定位置添加或删除参考线。

  • 快捷方式，CTRL + SHIFT + I， 被设置为 插入特殊字符 命令的默认快捷键。

  • 如果上一次在 插入特殊字符 对话框中使用的是 字符代码 文本框，新版本的 EmEditor 让你再次打开该对话框时，键盘焦点停留在 字符代码 文本框上。

  • 在默认的 HTML 和 CSS 配置中，新的 HTML 5 元素 和 CSS 3 属性 已被添加高亮显示。

  • 新的 插入长音符 命令让你能方便地插入长音符。而且，已存在的 插入重音字符 命令被扩展到输入下面 额外的 字符 (要查看完整列表，请参考 插入特殊字符):

新增选项¶

  • 在 自定义标记 对话框中的 选项 页面上，新增了 当标记工具栏隐藏时，自动禁用标记 的复选框。

  • 替换 对话框中新增了 「<< 查找」 按钮。

  • 配置属性中的 标记 页面上新增了 显示用户自定义参考线 复选框和 用户自定义参考线 文本框。

  • 配置属性 中 显示 页面上的 指定部分 新增了 用户自定义参考线 项目。
