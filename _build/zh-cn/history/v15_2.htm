历史记录 < Version 15.2

Version 15.2¶

2015年7月22日

一般新增功能¶

  • 大大提高了在 CSV 文档中对已选择的列进行 “全部替换” 命令，删除垂直选取以及在 全部替换 后 撤消 的执行速度。

  • 你能选取 CSV 文档中的列然后拖放这些列来重新排列或复制 CSV 列。

  • EmEditor 能检测到 CSV 文档中的无效双引号。

  • 帮助文档可以在网上浏览查看。你还能下载查看本地帮助文件（帮助菜单下的“首选项”子菜单）。帮助文件不再包括在安装程序内。它需要分开安装。

  • Consolas (11 point) 成为西欧字体类型的默认字体，以前的默认字体是 Courier New (10 point)。

  • 在自定义对话框中的标签页面上， 在新窗口中打开新的文件，否则在新标签页中打开 设定变为默认。

  • 以下这些对话框现在可以调整大小： 外部工具， 工具栏， 在文件中查找， 在文件中替换， 提取列， 合并 CSV， 删除/把重复行设为标签， 插入特殊字符， 自定义菜单 等等。

  • EmEditor v15.2 能与 Windows 10 更好地兼容。当你用 任务视图 功能时， 层叠， 水平平铺， 垂直平铺， 比较 以及 同步 命令现在可以在虚拟桌面上正常运行。例如，如果你使用的是之前版本的 EmEditor，当你执行 垂直平铺 命令时，就会出现两个 EmEditor 群组窗口显示在不同的虚拟桌面上。一个虚拟桌面上显示左半边的群组窗口，另一个虚拟桌面上显示右半边的群组窗口。EmEditor v15.2 则可以在当前虚拟桌面上并排显示窗口。

  • 在安装 EmEditor 时，如果你选择的是按每个用户安装，那么 记录鼠标活动及键盘输入到其他应用程序中 的复选框会自动变为无效。如果你选择的是按每台电脑安装，则会出现以下这个消息：“ 这个功能(录制鼠标活动和键盘输入到其他应用程序中)可能无法在一些特定的 Windows 版本和安全设定下正常运行。在以后的版本中，EmEditor 将不支持该功能。”。

  • 在 Windows 10 系统下，当你启用 用 EmEditor 在 Internet Explorer 查看源文件 设定时，会显示以下消息，“ 在 Internet Explorer 中用 EmEditor 查看源文件的功能无法与 Microsoft Edge 很好地兼容。如果您选择在 Microsoft Edge 上继续用 EmEditor 查看源文件，可能不会显示任何窗口。请选择“取消”来关闭该功能。”。

新增选项¶

  • 首行作为页眉重复 复选框被添加到 把文档或指定文件分割为多个文件 向导 中的 页眉和页脚 页面 上。

  • 自动调节分隔符位置 复选框被添加到 自定义 对话框中的 CSV 页面 上。
