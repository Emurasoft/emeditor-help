# 命令 ID

命令 ID 通过 [EE\_EXEC\_COMMAND 消息](../message/ee_exec_command)， [Editor\_ExecCommand 内联函数](../macro/editor_execcommand)， [EE\_QUERY\_STATUS 消息](../message/ee_query_status)，以及 [Editor\_QueryStatus 内联函数](../macro/editor_querystatus) 被使用。

命令 ID 的说明请访问 [命令参考](../../cmd/index)。

这些常数在头文件(plugin.h)中被定义。
