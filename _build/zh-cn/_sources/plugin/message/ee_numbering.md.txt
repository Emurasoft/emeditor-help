# EE\_NUMBERING

在鼠标位置或垂直选择处，插入编号。你能明确地发送该消息或用 [Editor\_Numbering 内联函数](../macro/editor_numbering)。

EE\_NUMBERING

wParam = (WPARAM)(NUMBERING\_INFO\*)pNI;

lParam = 0;

## 参数

_pUNI_

> 指定指针指向 [NUMBERING\_INFO 结构](../structure/numbering_info)。

## 返回值

> 如果成功的话，返回值为 0。

## 版本

> 支持 EmEditor Professional 19.1 或之后的版本。
