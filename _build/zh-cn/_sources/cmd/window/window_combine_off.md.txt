# 启用标签页 \- 关命令

### 摘要

> 禁用标签页并分隔所有窗口。

### 说明

> 禁用标签页并分隔所有窗口。当标签页被禁用，每一个打开的文档会显示在单独的窗口中。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **启用标签页** \> **启用标签页 \- 关**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_WINDOW\_COMBINE\_OFF (4408)

### 宏

#### \[JavaScript\]

> editor.EnableTab = false;

#### \[VBScript\]

> editor.EnableTab = False
