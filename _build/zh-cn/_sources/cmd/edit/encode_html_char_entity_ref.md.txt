# 将 Unicode 转换为 HTML 字符实体引用命令

### 摘要

> 把选中的文本编码为 HTML/XML 数字字符引用。

### 说明

> 把选中的文本编码为 HTML/XML 数字字符引用。

### 运行方法

- 默认菜单: **转换** \> **编码/解码所选内容** \> **将 Unicode 转换为 HTML 字符实体引用**
- [所有命令](../tools/all_commands): **转换** \> **编码/解码所选内容** \> **将 Unicode 转换为 HTML 字符实体引用**
- 工具栏:
无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_ENCODE\_HTML\_CHAR\_ENTITY\_REF (4563)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4563);

#### \[VBScript\]

> editor.ExecuteCommandByID 4563
