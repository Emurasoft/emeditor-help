# 编辑单元格 (仅限单元格选择模式) 命令

### 摘要

> 在 CSV 文档中编辑当前单元格。

### 说明

> 在 CSV 文档中编辑当前单元格。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **CSV** \> **仅限单元格选择模式** \> **编辑单元格**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CSV\_EDIT\_CELL (3960)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3960);

#### \[VBScript\]

> editor.ExecuteCommandByID 3960
