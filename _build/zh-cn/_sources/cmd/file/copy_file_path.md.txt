# 复制文件完整路径命令

### 摘要

> 复制当前文件的完整路径名到剪贴板。

### 说明

> 复制当前文件的完整路径名，并把它置放到剪贴板上。如果文档没有标题，这个命令则不能被选取。

### 运行方法

- 默认菜单: **文件** \> **复制文件完整路径**
- [所有命令](../tools/all_commands): **文件** \> **复制**
\> **文件完整路径**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_COPY\_FILE\_PATH (4326)

### 宏

#### \[JavaScript\]

> document.CopyFullName();

#### \[VBScript\]

> document.CopyFullName
