# 插入命令

### 摘要

> 插入一个指定的文件到当前光标位置。

### 说明

> 这个命令会显示 **插入** 对话框，让你能选取一个你想要插入到当前文件的文件。在选取一个文件后，这个命令会读取被选取的文件并把它的内容插入到光标所在位置。

### 运行方法

- 默认菜单: **插入** \> **插入**
- [所有命令](../tools/all_commands): **插入** \> **插入**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_FILE\_INSERT (4108)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4108);

#### \[VBScript\]

> editor.ExecuteCommandByID 4108
