# PrintProp 对象

## 属性

|     |     |
| --- | --- |
| [DrawSeparatingLines](draw_separating_lines) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **插入分隔线** 复选框相对应。 |
| [Footer](footer) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **页脚** 文本框相对应。 |
| [Header](header) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **页眉** 文本框相对应。 |
| [IgnoreColor](ignore_color) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **忽略颜色与下划线** 复选框相对应。 |
| [LineSpace](line_space) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **行间距** 复选框相对应。 |
| [ShowLineNumbers](show_line_numbers) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **打印行号** 复选框相对应。 |
| [WrapPage](wrap_page) | 与配置属性中 [**打印** 页面](../../dlg/properties/print/index) 上的 **打印时按页面换行** 复选框相对应。 |

## 版本

支持 EmEditor Professional Version 7.00 或之后的版本。

```{toctree}
:maxdepth: 1
draw_separating_lines
footer
header
ignore_color
line_space
show_line_numbers
wrap_page
```
