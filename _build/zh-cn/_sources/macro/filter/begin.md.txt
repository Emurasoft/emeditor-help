# Begin 属性

指定你要搜索的文本的起始列的索引 (逻辑字符数)。指定你要搜索的文本的起始列的索引 (逻辑字符数)。指定 0 如果你想要把文本的最后一部分指定为 _End_。要启用这个字段， _Column_ 属性必须是 -1 。

#### \[JavaScript\]

_n_ =
item. **Begin**;

item. **Begin** = _n_;

#### \[VBScript\]

_n_ =
item. **Begin**

item. **Begin** = _n_

## 版本

支持 EmEditor Professional Version 16.0 或之后的版本。
