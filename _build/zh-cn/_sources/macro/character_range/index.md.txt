# CharacterRange 对象

## 属性

|     |     |
| --- | --- |
| [Enabled](enabled) | 指定是否启用对象的标志。 |
| [Min](min) | 指定对象的最小字符。 |
| [Max](max) | 指定对象的最大字符。 |
| [Properties](properties) | 指定表示对象的 Unicode 脚本和/或 Unicode 常规类别属性的字符串。 |
| [TreatAs](treat_as) | 指定要替换的字符。 如果指定-1，则在用于模糊匹配时将忽略指定的范围。 |

## 版本

支持 EmEditor Professional Version 22.0 或之后的版本。
