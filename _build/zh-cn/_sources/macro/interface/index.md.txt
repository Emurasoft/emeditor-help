# Interface 对象

## 属性

|     |     |
| --- | --- |
| **[x](x)** | 返回被一个支持的插件所传递的光标的水平位置。 |
| **[y](y)** | 返回被一个支持的插件所传递的光标的垂直位置。 |

## 方法

|     |     |
| --- | --- |
| **[write](write)** | 把一个字符串传递到一个支持的插件中。 |
| **[writeln](writeln)** | 把一个字符串和新行传递到一个支持的插件中。 |

## 版本

支持 EmEditor 9.00 或之后的版本。

```{toctree}
:maxdepth: 1
write
writeln
x
y
```
