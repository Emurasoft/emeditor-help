# DefaultExtension 属性

与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **默认扩展名** 文本框相对应。

#### \[JavaScript\]

_s_ = object. **DefaultExtension**;

object. **DefaultExtension** = _s_;

#### \[VBScript\]

_s_ = object. **DefaultExtension**

object. **DefaultExtension** = _s_

## 版本

支持 EmEditor 7.00 或之后的版本。
