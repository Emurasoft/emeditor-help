# UnicodeSignature 属性

与 [**添加一个 Unicode 签名 (BOM)** 复选框](../../dlg/properties/file/save_details/index) 相对应。

#### \[JavaScript\]

_b_ =
object. **UnicodeSignature**;

object. **UnicodeSignature** = _b_;

#### \[VBScript\]

_b_ =
object. **UnicodeSignature**

object. **UnicodeSignature** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
