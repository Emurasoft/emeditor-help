# “输入数字范围”对话框

此对话框让你能创建一个 [数字范围表达式](../../howto/search/number_range_syntax)。 此对话框会出现当你在 **查找**， **替换**， **在文件中查找**， **在文件中替换**，或 **高级筛选** 对话框中点选了 **数字范围** 单选按钮。

### “最小值”文本框

指定一个数字范围的最小值。如果勾选了 **仅整数值** 复选框，此数值将不包括小数。

### “包括最小值”复选框

指定最小值是否包括在数字范围内。

### “最大值”文本框

指定一个数字范围的最大值。如果勾选了 **仅整数值** 复选框，此数值将不包括小数。

### “包括最大值”复选框

指定最大值是否包括在数字范围内。

### “递增值+最小数目”复选框

指定仅匹配最小整数加上的指定整数的倍数。

### “递增值”复选框

如果勾选了 **递增值+最小数目** 复选框，为之指定一个数值。

### “整数”单选按钮

指定是否仅匹配整数。

### “十进制数”单选按钮

指定应匹配十进制数字。

### “使用默认的小数点”复选框

指定应使用从当前语言环境派生的小数点。如果未设置，则应在下面的下拉列表框中指定小数点字符。

### “IP 地址”单选按钮

指定应匹配 IP 地址。

### “IP 地址/CIDR 标记”复选框

选择将 IP 地址范围指定为 IP 地址/CIDR 标记。如果勾选了此复选框，则应在下面的文本框中指定正确的 IPv4/IPv6 地址/CIDR 标记。例如，如果要搜索从 192.168.0.0 到 192.168.255.255 的 IPv4 地址，就要指定 "192.168.0.0/16" 。

### “日期/时间”单选按钮

指定应匹配日期和/或时间。

### “使用默认的日期格式”复选框

指定应使用从当前语言环境派生的日期格式。如果未勾选，则要在复选框下面的下拉列表框中指定日期和/或时间格式。

### 「>」按钮

单击此按钮可显示可以使用的日期/时间格式。

### “前缀”文本框

指定在匹配值之前匹配的字符串。该字段可以为空。

### “后缀”文本框

指定在匹配值之后匹配的字符串。该字段可以为空。

### “下次跳过此对话框”复选框

指定下次是否应不显示此对话框直接跳到 **查找** 或 **筛选** 下拉列表框中。
