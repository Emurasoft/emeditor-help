# Q. 当我用 EmEditor 写完一个 Java 类别并编译它时，我得到一个错误提示说在我创建的类别开头有 3 个无效字符。这个情况每次发生，但在我的类别关键字前，也就是错误提示指出的地方并没有任何字符。我该怎么处理?

在文件顶端的 3 个字符是 UTF-8 文件的字节顺序标记 ("Byte Order Mark")。在默认情况下，Java 文件是作为含有 BOM 的 UTF-8 文件被创建的，但这个默认设定可以更改。请参考下列步骤: **Java 属性** \> **文件** 页面 \> **「新建文件」** 按钮 \> 取消勾选 **添加一个 Unicode 签名 (BOM)** 复选框。
