# Q. 怎么用 Western European 而不是 UTF-8 打开一个 XML 文件?

如果一个 XML 文件是作为 Unicode 被打开的，您可以到 **“当前配置属性”** 中，选择 [**文件** 页面](../../dlg/properties/file/index)，然后查看 **“文件打开时的编码”** 旁的下拉列表框。如果被选择的是 **“Unicode”**，请您把它改为 **“系统默认”**。如果 **“检测 HTML/XML 字符”** 复选框被勾选了，一个“encoding=” 的指令会在一个 XML 文件中被搜寻。如果这个指令被找到了，EmEditor 使用这个指定的编码。要注意的是，为目前打开的文件所指定的编码与新建文件或正在储存的文件所指定的编码是不同的。
