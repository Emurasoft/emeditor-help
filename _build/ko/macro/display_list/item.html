Item 속성¶

지정된 인덱스를 위한 DisplayItem 개체 를 검색합니다.

[JavaScript]¶

item = list. Item( Index );

[VBScript]¶

item = list. Item( Index )

매개 변수¶

Index

1부터 시작하는 정수로된 항목의 인덱스를 지정합니다.
