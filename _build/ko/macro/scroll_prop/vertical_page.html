VerticalPage 속성¶

구성 속성 스크롤 탭 의 항상 1 페이지 세로 스크롤 활성화 체크 박스에 해당합니다.

[JavaScript]¶

b = object. VerticalPage ;

object. VerticalPage  = b;

[VBScript]¶

b = object. VerticalPage

object. VerticalPage  = b

버전¶

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
