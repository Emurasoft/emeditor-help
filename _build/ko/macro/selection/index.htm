매크로 참조 < Selection 개체

Selection 개체¶

속성¶

Count

현재 문서에서 선택된 영역의 수를 검색합니다.

IsActiveEndGreater

활성화 지점이 하단의 지점과 동일한 지 여부를 나타냅니다.

IsEmpty

활성화 지점이 기준 위치와 동일한 지 여부를 나타냅니다.

Mode

선택 모드를 나타내는 플래그를 설정하거나 검색합니다.

OverwriteMode

덮어쓰기 또는 삽입 모드를 나타내는 플래그를 설정하거나 검색합니다.

Text

선택된 텍스트를 검색하거나 현재 위치에 문자열을 삽입합니다.

메서드¶

ChangeCase

선택된 텍스트의 대,소문자를 변경합니다.

ChangeWidth

선택된 텍스트의 너비를 변경합니다.

CharLeft

커서를 지정된 숫자의 문자만큼 왼쪽으로 이동합니다.

CharRight

커서를 지정된 숫자의 문자만큼 오른쪽으로 이동합니다.

ClearBookmark

현재 줄에 책갈피를 지웁니다.

Collapse

활설화 지점에 선택 영역을 축소합니다.

Copy

클립보드에 선택 영역을 복사합니다.

CopyLink

클립보드에 커서 위치의 하이퍼링크를 복사합니다.

Cut

선택된 텍스트를 클립보드로 이동합니다.

Delete

선택 내용을 삭제합니다.

DeleteLeft

선택 내용을 삭제하거나 커서에서 왼쪽으로 지정된 숫자 만큼 삭제합니다.

DestructiveInsert

기존의 텍스트에 덮어쓰기하여 텍스트를 삽입합니다.

DuplicateLine

현재의 줄을 중복되게 합니다.

EndOfDocument

문서의 끝 지점으로 커서를 이동합니다.

EndOfLine

현재 줄의 끝 지점으로 커서를 이동합니다.

Find

지정된 문자열을 검색합니다.

FindRepeat

같은 문자열로 이전 검색을 반복합니다.

Format

선택 영역에 새로운 줄을 삽입하거나 삭제합니다.

GetActivePointX

커서 위치에 열 번호를 반환합니다.

GetActivePointY

커서 위치에 줄 번호를 반환합니다.

GetAnchorPointX

선택 영역의 원점의 열 번호를 반환합니다.

GetAnchorPointY

선택 영역의 원점의 줄 번호를 반환합니다.

GetBottomPointX

선택 영역의 하단의 열 번호를 반환합니다.

GetBottomPointY

선택 영역의 하단의 줄 번호를 반환합니다.

GetTopPointX

선택 영역의 상단의 열 번호를 반환합니다.

GetTopPointY

선택 영역의 상단의 줄 번호를 반환합니다.

GoToBrace

해당하는 대괄호/중괄호로 커서를 이동합니다.

Indent

들여쓰기 레벨의 지정된 수 만큼 선택된 줄을 들여쓰기 합니다.

InsertDate

현재 시간과 날짜를 삽입합니다.

InsertFromFile

커서 위치에 지정된 파일의 내용을 삽입합니다.

LineDown

지정된 줄의 수 만큼 커서를 아래로 이동합니다.

LineOpen

두 줄 사이에 빈 줄을 삽입합니다.

LineUp

지정된 줄의 수 만큼 커서를 위로 이동합니다.

NewLine

커서에 지정된 수만큼의 새로운 줄을 삽입합니다.

NextBookmark

문서 내에서 다음 책갈피로 이동합니다.

OpenLink

커서 위치에 하이퍼링크를 엽니다.

PageDown

문서 내에서 지정한 페이지 수만큼 아래로 커서를 이동합니다.

PageUp

문서 내에서 지정한 페이지 수만큼 위로 커서를 이동합니다.

Paste

커서에 클립보드 내용을 삽입합니다.

PreviousBookmark

문서 내에서 이전 책갈피로 이동합니다.

Replace

문서 내 문자열을 대체합니다.

SelectAll

문서 전체를 선택합니다.

SelectLine

커서 위치에 줄을 선택합니다.

SetActivePoint

커서 위치를 이동하고 필요에 따라 선택 영역을 확장합니다.

SetAnchorPoint

선택 영역의 원점을 설정합니다.

SetBookmark

커서 위치에 책갈피를 설정합니다.

SelectWord

커서 위치에 단어 전체를 선택합니다.

StartOfDocument

문서의 시작점으로 커서를 이동합니다.

StartOfLine

줄의 시작점으로 커서를 이동합니다.

Tabify

선택 영역의 공백을 탭으로 변환합니다.

TagJump

커서 위치에 태크로 점프합니다.

UnIndent

들여쓰기 레벨의 지정된 수 만큼 선택된 텍스트로부터 들여쓰기를 제거합니다.

Untabify

선택 영역에서 탭을 공백으로 변환합니다.

WordLeft

지정된 단어의 수만큼 왼쪽으로 커서를 이동합니다.

WordRight

지정된 단어의 수만큼 오른쪽으로 커서를 이동합니다.
