버전 6.00¶

2006년 6월 12일

  • 사용자 지정 대화 상자 의 바로가기 탭 이 추가되었으며, 이것으로 트레이 아이콘, 보내기와 같은 바로가기나 다른 바로가기들을 표시하거나 숨길 수 있습니다. 더 나아가, EmEditor를 빠르게 실행하기 위해 빠른 시작 체크 박스가 추가되었습니다.

  • 이제 사용자 지정 표시줄을 플러그 인을 사용해 표시될 수 있습니다.

  • 플러그 인 인터페이스에 EE_CUSTOM_BAR_OPEN, EE_CUSTOM_BAR_CLOSE , EE_MATCH_REGEX, EE_FIND_REGEX, EE_GET_OUTLINE_LEVEL, EE_SET_OUTLINE_LEVEL, EE_SHOW_OUTLINE, 및 EE_ENUM_CONFIG 메시지가 추가되었습니다.

  • 플러그 인 인터페이스에 Editor_CustomBarOpen, Editor_CustomBarClose, Editor_MatchRegex, Editor_FindRegex, Editor_GetOutlineLevel, Editor_SetOutlineLevel, Editor_ShowOutline, Editor_EnumConfig 인라인 함수가 추가되었습니다.

  • 새로운 flags가 EE_SET_CARET_POS 메시지, 및 Editor_SetCaretPos 와 Editor_SetCaretPosEx 인라인 함수에 추가되었습니다.

  • 플러그 인 인터페이스에 EVENT_IDLE, EVENT_CUSTOM_BAR_CLOSED, 및 EVENT_CUSTOM_BAR_CLOSING 이벤트가 추가되었습니다.

  • 플러그 인 메시지 에 EP_PRE_TRANSLATE_MSG 메시지가 추가되었습니다.

  • 플러그 인 인터페이스에 CUSTOM_BAR_INFO, CUSTOM_BAR_CLOSE_INFO, FIND_REGEX_INFO, MATCH_REGEX_INFO 구조가 추가되었습니다.

  • 설치기가 윈도우용 설치기가 되어 대화 상자 표시 없이 설치 가 가능합니다.

  • 사용자 지정 대화 상자 에 언어 탭 이 추가되었습니다. 추가적인 언어 구성 요소를 설치하여 사용자 인터페이스 언어를 바꿀수 있습니다.

  • 사용자 지정 대화 상자 의 창 탭 에 닫기 명령을 선택하여 마지막 문서를 닫지 않음 체크 박스가 추가되었습니다.

  • 외부 도구 속성 대화 상자 에 출력 모음 사용 체크 박스가 추가되었습니다.

  • 세부 정보 저장 대화 상자 에 제목없음 문서 자동 이름 지정 체크 박스가 추가되었습니다.

  • 현재 폴더를 위한 관리 방법이 바뀌었습니다. 이것은 이전에 사용된 폴더들이 삭제되는 문제를 해결했습니다.

  • 사용자 지정 대화 상자 의 탭 탭 에 탭의 외부 - 가운데 마우스 버튼 클릭 드롭 다운 리스트 박스와 탭의 외부 - 왼쪽 마우스 버튼 더블 클릭 드롭 다운 리스트 박스가 추가되었습니다.

  • 새로운 명령 줄 옵션 (/ar, /ca, /car, /di, /hide, /ipi, /sca)이 추가되었습니다.

  • 활성화 창, 선 확대/축소, 선 축소, 선 확대, 모두 축소, 모두 확대, 다음 노드, 이전 노드, 다음 노드 확장, 이전 노드 확장, 삭제된 문자열 복원 명령이 추가되었습니다.

  • 이제부터 찾기 와 바꾸기 대화 상자의 위치가 기억됩니다.

  • 찾기 대화 상자 에 모든 열린 문서 검색 체크 박스가 추가되었습니다.

  • 이제부터 매크로의 첫 줄에 #language 지침서 를 추가하여 JavaScript 와 VBScript가 아닌 ActiveScript 언어를 사용 할 수 있습니다.

  • 사용자 지정 대화 상자 의 파일 탭 에 프린터 엑세스 피하기 체크 박스가 추가되었습니다.

  • 사용자 지정 대화 상자 의 상태 탭에 현재 폴더 체크 박스가 추가되었습니다.

  • 사용자 계정 컨트롤이라 불리우는 새로운 보안 기능을 추가한 윈도우 비스타를 지원합니다. EmEditor는 권리자 권한이 필요한 옵션 ( EmEditor와 연결 대화 상자, 더 많은 사용자 지정 바로가기 대화 상자, 등등.) 을 분명하게 구별하고, 일반적으로 최소의 권리 환경으로 실행하도록 합니다.

  • 이제 EmEditor는 사용자 뿐만 아니라 컴퓨터당 설치할 수 있도록 합니다.

  • 이제 모든 설치자 패키지 (msi), 실행가능한 파일 (exe), 및 응용 프로그램 확장은 디지철 서명이 되어있습니다.

  • 제거하는 동안, 응용 프로그램 설정을 유지하도록 하는 새로운 옵션을 추가하였습니다.

  • EP_SET_UNINSTALL 플러그 인 메시지 에 새로운 매개 변수를 추가하였습니다.

  • 보기 메뉴에 출력 바 설정/해제 명령 이 추가되었습니다.
