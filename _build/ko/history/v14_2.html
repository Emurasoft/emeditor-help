버전 14.2¶

2014년 1월 7일

새로운 일반 기능¶

  • 이제 사용자 지정 가이드 를 설정 할 수 있습니다. 사용자 지정 가이드는 각 구성을 위해 편집기에 지정된 열에 위치한 수직 선입니다. 사용자 지정 가이드는 수직으로 정렬된 소스 코드 또는 텍스트 파일을 쉽게 볼 수 있도록 합니다.

  • 줄자에서 마우스 오른쪽을 클릭하였을 때 보이는 상황에 맞는 메뉴는 이제 마우스가 포인트한 위치에 편리하게 새로운 가이드를 추가하거나 존재하는 가이드를 삭제할 수 있는 사용자 지정 가이드 추가/삭제 명령을 포함합니다.

  • CTRL + SHIFT + I는 특수 문자 삽입 명령을 위한 기본 키보드 바로가기로 지정되었습니다.

  • 코드 값 텍스트 박스가 특수 문자 삽입 대화 상자에서 마지막으로 사용된 경우, 새 버전은 이제 대화 상자를 다시 열었을 때 처음부터 키보드 포커스가 코드 값 에 가도록 설정됩니다.

  • 새로운 장음 삽입 명령으로 장음 문자를 삽입할 수 있습니다. 더 나아가, 기존의 엑센트 문자 삽입 명령이 다음의 추가적인 문자를 입력하기 위해 확장되었습니다. (전체 목록을 보려면 특수 문자 삽입하기 를 참고하십시오):

새로운 옵션¶

  • 마커 사용자 지정 대화 상자의 옵션 탭에 마커 도구 모음이 숨겨졌을 때 자동적으로 마커 비활성화 체크 박스가 추가되었습니다.

  • 바꾸기 대화 상자에 << 찾기 버튼이 추가되었습니다.

  • 구성 속성의 표시 탭에 사용자 지정 가이드 보이기 체크 박스와 사용자 지정 가이드 위치 텍스트 박스가 추가되었습니다.

  • 사용자 지정 가이드 항목이 구성 속성 의 디스플레이 탭의 부분 지정 리스트 박스에 추가되었습니다.
