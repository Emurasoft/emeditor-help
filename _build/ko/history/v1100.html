버전 11¶

2011년 9월 21일

새로운 일반 기능¶

  • 마커

  • HTML/XML 문자 참조를 보여주는 도구 설명

  • 찾기 도구 모음

  • 연속 바꾸기

  • 새로운 중복된 줄 삭제 명령으로 선택된 영역내 혹은 문서 전체에서 중복된 줄을 삭제할 수 있습니다.

  • 텍스트가 선택되어졌을 때, 정렬 명령으로 지정된 텍스트 영역을 정렬할 수 있습니다.

  • 새로운 최근 닫은 파일 명령으로 최근에 닫은 파일을 열어볼 수 있습니다.

  • 한국어 IME의 한자를 지원합니다.

  • 상태 표시줄로 선택된 영역내의 문자의 개수, 단어 개수, 줄의 개수를 보도록 지정할 수 있습니다. 또한, 디스플레이 순서는 사용자 지정될 수 있습니다.

  • 유니코드 표의문자 변형 시퀀스(IVS)는 이제 지원됩니다. (이때, 지원되는 글꼴이 필요합니다.)

새로운 옵션¶

  • 마커 사용자 지정 대화 상자 가 추가되었습니다.

  • 연속 교체 버튼, 연속 항목에 추가 버튼, 수정 버튼, 목록 박스, 모두 활성/비활성 체크 박스, 위로 버튼, 아래로 버튼, 삭제 버튼 모두 선택 버튼, 가져오기 버튼, 내보내기 버튼, 그리고 모두 연속 교체 버튼은 바꾸기 대화 상자 에 추가되었습니다.

  • HTML/XML 문자 참조와 UCN 체크 박스 구성 속성 에 있는 강조 (2) 탭 에 추가되었습니다.

  • 찾기/바꾸기 대화 상자에 있는 바로가기 비활성 체크 박스, 찾은 후에 편집기로의 포커스 변환 체크 박스, 그리고 ESC 입력시 닫기 체크 박스가 사용자 지정 대화 상자 의 검색 탭 에 추가되었습니다. .

  • 상태 목록 상자, 위로 버튼, 그리고 아래로 버튼이 사용자 지정 대화 상자 의 상태 탭 에 추가되었습니다.

새로운 명령¶

  • 찾기 도구 모음

  • 마커에 추가

  • 마커 도구 모음

  • 마커 사용자 지정

  • 유니코드를 HTML 문자 항목 참조로

  • 중복된 줄 삭제

  • 찾기 도구 모음으로 포커스 지정

  • 찾기 정규식 검색 (찾기 도구 모음)

  • 바꾸기 정규식 검색 (찾기 도구 모음)

  • 이전 찾기 (찾기 도구 모음)

  • 다음 찾기 (찾기 도구 모음)

  • 증분 검색 (찾기 도구 모음)

  • 열린 모든 문서 검색 (찾기 도구 모음)

  • 대/소문자 구분 (찾기 도구 모음)

  • 정규식 사용 (찾기 도구 모음)

  • 이스케이프 시퀀스 사용 (찾기 도구 모음)

  • 단어만 찾기 (찾기 도구 모음)

  • 문서의 처음/끝으로 이동 (찾기 도구 모음)

  • 개수 일치 (찾기 도구 모음)

  • 바꾸기 >> (찾기 도구 모음)

  • 선택 내에서만 (찾기 도구 모음)

  • 다음 바꾸기 (찾기 도구 모음)

  • 모두 바꾸기 (찾기 도구 모음)

  • 가장 최근 닫은 문서

  • 찾기 도구 모음 닫기

  • 모든 마커 활성화

  • 모든 마커 비활성화

  • 모든 마커 삭제

  • 최근 닫은 파일 (여러 항목)

매크로의 새로운 기능¶

  • HighlightCharRef 속성 이 HighlightProp 개체 에 추가되었습니다.

  • eeEncodingBinary 과 eeEncodingHex 상수가 인코딩 상수 에 추가되었습니다.

플러그인 API 새로운 기능¶

  • m_bHighlightCharRef 구성원이 CCustomizeInfo 클래스에 포함되었습니다.

  • 새로운 명령 (EEID_SHOW_FIND_BAR 부터 EEID_CLOSE_FIND_BAR and EEID_RECENT_CLOSED_FILE1) 가 추가되었습니다.
