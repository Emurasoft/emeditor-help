버전 7.00¶

2007년 12월 18일

  • 새로운 다중 스레드 디자인은 대용량 파일을 여는 동안 문서를 보거나 편집할 수 있도록 합니다.

  • 사용자 지정 대화 상자의 고급 탭 에 원본 파일로부터 수정되지 않은 줄을 읽기 체크 박스, 원본 파일 잠금 체크 박스, 및 비동기적으로 열기를 위하여 파일 사이즈를 최소화 텍스트 박스가 추가되고, 대용량 파일을 열기 위해 속도를 최적화하였습니다.

  • 이제 플러그 인을 사용하여 사용자 지정 도구 모음을 표시할 수 있습니다.

  • 플러그 인 인터페이스에 EE_TOOLBAR_OPEN, EE_TOOLBAR_CLOSE, EE_TOOLBAR_SHOW, EE_HELP, EE_REG_SET_VALUE, EE_REG_QUERY_VALUE, EE_QUERY_STRING, EE_KEYBOARD_PROP, EE_GET_ACCEL_ARRAY, EE_OUTPUT_DIR, EE_OUTPUT_STRING, EE_ENUM_HIGHLIGHT 메시지가 추가되었습니다.

  • 플러그 인 인터페이스에 Editor_ToolbarOpen, Editor_ToolbarClose, Editor_ToolbarShow, Editor_Help, Editor_RegSetValue, Editor_RegQueryValue, Editor_QueryString, Editor_KeyboardProp, Editor_GetAccelArray Editor_DocGetLines, Editor_OutputDir, Editor_OutputString, Editor_EnumHighlight 인라인 함수가 추가되었습니다.

  • 플러그 인 인터페이스에 TOOLBAR_INFO, REG_SET_VALUE_INFO, REG_QUERY_VALUE_INFO 구조가 추가되었습니다.

  • 플러그 인 인터페이스에 EVENT_TOOLBAR_CLOSED, EVENT_TOOLBAR_SHOW 이벤트 가 추가되었습니다.

  • 이제 EmEditor는 이전 버전보다 더 큰 용량의 파일을 (사용 가능한 메모리에 따라)편집 할 수 있도록 합니다.

  • 내 매크로에 쉽게 액세스 할 수 있도록 매크로 도구 모음이 추가되었습니다.

  • 이제 매크로 버튼의 아이콘, 제목, 및 툴팁을 변경할 수 있습니다. ( #icon 지시문, #title 지시문, #tooltip 지시문).

  • 도구 모음 대화 상자 에 매크로 도구 모음 디스플레이 체크 박스가 추가되었습니다.

  • 이제 찾기 대화 상자 와 바꾸기 대화 상자 의 크기 재조정이 가능합니다.

  • EmEditor는 이제 EmEditor가 작동을 멈출 경우 혹은 자동 업데이트로 인해 윈도우가 강제적으로 다시 시작되려는 경우, 자동적으로 수정된 파일들을 대화 상자 표시 없이 “AutoSave-“의 이름으로 시작되는 파일명으로 저장합니다. 또한, 엠에디터는 이제 재시작 시 자동적으로 복구된 파일들을 보여주는 메시지 박스를 내보냅니다.

  • 파일이 열렸을 때 상태 표시줄에 표시되었던 파일 크기는 이제 필요에 따라 KB, MB, 또는 GB로 표시됩니다.

  • 세 번 클릭은 이제 전체 줄을 선택합니다.

  • QueryStringByID 메서드 가 추가되었습니다.

  • EmEditor는 이제 다른 어플리케이션에 대한 키입력과 마우스 활동을 녹화하고 실행할 수 있습니다.

  • Shell 개체 와 Windows 컬렉션 이 추가되었습니다.

  • Window 개체 에 Children, ClassName, Enabled, Height, hWnd, LeftX, OutputBar, Parent, ProcessID, ThreadID, Top, Valid, Visible, 및 Width 속성과 FindWindow, FindWindowByID, SetFocus, SetForeground, 및 SetPosition 메서드가 추가되었습니다.

  • Document 개체 에 Config 속성 이 추가되었습니다.

  • 구성 속성을 위한 개체 ( AssociationItem 개체, AssociationList 컬렉션, AssociationProp 개체, AutoSaveProp 개체, BackupProp 개체, Config 개체, Configs Collection, DisplayItem 개체, DisplayList 컬렉션, DisplayProp 개체, FileNewProp 개체, FileSaveProp 개체, FileProp 개체, FontItem 개체, FontList 컬렉션, FontProp 개체, GeneralProp 개체, HighlightItem 개체, HighlightList 컬렉션, HighlightProp 개체, IndentProp 개체, KeyboardItem 개체, KeyboardList 컬렉션, KeyboardProp 개체, LinkProp 개체, MarkProp 개체, NoWrapItem 개체, NoWrapList 컬렉션, NoWrapProp 개체, PrintProp 개체, ScrollProp 개체, 및 WrapProp 개체)가 추가되었습니다.

  • Document 개체 에 GetLine 메서드, GetLines 메서드 가 추가되었습니다.

  • Editor 개체 에 Configs 메서드 가 추가되었습니다.

  • OutputBar 개체 가 추가되었습니다.

  • 이제부터 가상 공간 모드가 지원됩니다 (구성 속성의 일반 탭 에 가상 공간 활성화 체크 박스).

  • 이제 열기 대화 상자로 여러 개의 파일을 동시에 열수 있습니다.

  • 휴대용 옵션 (USB 드라이브 설치를 위한 준비)

  • 매크로 사용자 지정 대화 상자 의 옵션 탭 에 마우스 활성 및 기타 응용 프로그램의 키보드 출력 기록 체크 박스, 모든 마우스 움직임 기록 체크 박스, 및 취침 모드 삽입 전 유휴 시간 텍스트 박스가 추가되었습니다.

  • 새로운 플러그 인: 프로젝트 플러그 인, Find Bar 플러그 인, HTML Bar 플러그 인, 및 단어 완성 플러그 인 이 추가되었습니다.

  • 매크로 도구 모음, 새로 고침 도구 모음, 모든 활동 녹화, 마우스/키보드를 제외한 모든 활동 녹화, 왼쪽 사용자 지정 바, 상위 사용자 지정 바, 오른쪽 사용자 지정 바, 하위 사용자 지정 바 명령이 추가되었습니다.

  • 바꾸기 명령, 대문자 명령, 소문자 명령, 첫 글자를 대문자로 명령, 반자 명령, 및 전자 명령 은 이제 수직 선택 영역을 변환할 수 있습니다.

  • 인쇄 시, Form Feed 코드 (U+000C)의 문서는 이제 코드에 새로운 페이지를 삽입합니다.

  • 텍스트와 배경 색상에 투명 옵션이 추가되었습니다.

  • 탭/들여쓰기 대화 상자 에 줄 바꿔서 들여쓰기 체크 박스가 추가되었습니다.

  • 사용자 지정 대화 상자 의 창 탭 에 자동으로 작업 영역 저장 체크 박스와 자동으로 작업 영역 복원 체크 박스가 추가되었습니다.

  • 구성 속성 의 강조 표시 (1) 탭 에 오른쪽 모두 강조 표시 체크 박스가 추가되었습니다.

  • 구성 속성 의 스크롤 탭 에 부드럽게 스크롤 체크 박스와 부드럽게 스크롤 속도 슬라이더가 추가되었습니다.

  • 이제부터 매크로로부터 라이브러리 파일을 포함할 수 있습니다 ( #include 지시문).

  • 새로운 명령 줄 옵션( /ne) 이 추가되었습니다.

  • 점프 대화 상자 에 열 텍스트 박스가 추가되었습니다.

  • 하나 이상의 파일이 수정되고 모두 닫기 명령이 선택되었을 때, 변경 내용을 …에 저장 메시지 박스가 이제 모두에 적용 체크 박스를 포함합니다.

  • EE_INFO 메시지 ( Editor_Info, Editor_DocInfo 인라인 함수)에 EI_GET_CURRENT_FOLDER 와 EI_IS_LARGE_DOC 명령이 추가되었습니다.

  • 외부 도구 속성 대화 상자 에 출력 인코딩 드롭 다운 리스트 박스가 추가되었습니다.

  • 닫기 버튼이 이제 활성화 택 또는 각 택에 표시됩니다 ( 닫기 버튼 드롭 다운 리스트 박스).

  • 이제부터 탭에 읽기 전용 아이콘이 표시됩니다. ( 읽기 전용 아이콘 보기 체크 박스).

  • 찾기 드롭 다운 리스트 박스에서 검색 기록으로부터 항목을 삭제하려면 ALT + DELETE를 누릅니다.

  • 사용자 정의 메뉴는 두개의 스트로크 키 바로가기를 허용합니다 ( 지연 시간 슬라이더).

  • 구성 속성의 디스플레이 탭 에 테마 드롭 다운 리스트 박스, > 버튼, 및 문자 공백e 텍스트 박스가 추가되었습니다.

  • 이제부터 이벤트 구동 매크로를 정의할 수 있습니다. ( 이벤트 버튼, 이벤트에 실행 체크 박스, 이벤트 텍스트 박스, 이벤트 선택 대화 상자).

  • 윈도우 비스타에서 열기/ 다른으로 저장 대화 상자는 이제 윈도우 비스타 디자인이고, 이전 버전의 파일에 더 쉽게 접근할 수 있도록 해 줍니다. 이 기능은 윈도우 비스타의 기능 중 한가지 입니다.

  • 설치기는 이전 버전이 이미 설치되어 있는 경우 설치 폴더를 변경할 수 없도록 합니다.
