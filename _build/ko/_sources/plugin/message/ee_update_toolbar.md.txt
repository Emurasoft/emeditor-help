# EE\_UPDATE\_TOOLBAR

도구 모음에 버튼 상태를 업데이트합니다. 이 메시지를 명시적으로 또는
[Editor\_UpdateToolbar](../macro/editor_updatetoolbar) 인라인 함수를 사용하여 보낼 수 있습니다.

EE\_UPDATE\_TOOLBAR

wParam = (WPARAM) (UINT) nCmdID;

lParam = 0;

## 매개 변수

_nCmdID_

> 상태가 검색된 명령의 식별자 입니다. [Command IDs](../cmdid/index) 를 참고하십시오.

## 반환 값

> 반환 값이 사용되지 않습니다.
