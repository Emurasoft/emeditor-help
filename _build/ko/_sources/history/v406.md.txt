# 버전 4.06

2004년 8월 26일

- v4.05의 설치 프로그램이 템플릿 (서식) 파일 (template.\*)을 설치할 수 없던 오류를 수정하였습니다.
- 인코딩이 UTF-8 또는 Unicode인 경우, 이제 상태 표시줄은 서명 (BOM) 있음 또는 없음을 나타냅니다.
- 파일이 외부에서 변경되었을 때. EmEditor에 포커스가 있는 경우에만 확인 메시지가 나타납니다.
- 설치 폴더가 변경 된 후, 설치 도중에 Internet Explorer v5 이상 HTML 편집기 목록에 EmEditor 추가하기가 체크 되어 있음에도 불구하고 인터넷 익스플로러 HTML 편집기 목록에 EmEditor가 나타나지 않았던 문제를 해결하였습니다.
- Windows XP SP2에 계단식/바둑판식 창 명령이 활동 창을 이동하지 않던 문제를 해결했습니다.
- 모두 바꾸기 명령을 최적화하였습니다.
