# 마지막 메시지 표시 명령

### 요약

> 마지막으로 받은 메시지를 표시합니다 (트레이 아이콘에서만 사용가능합니다.)

### 설명

> 엠에디터가 다른 PC로부터 **선택한 텍스트 보내기** 명령을 사용해 메시지를 받았을때,
> 풍선 팁이나 알림창을 사용해 메시지를 나타냅니다.
> 풍선 팀 또는 알림창이 사라진 후, 이 명령을 통해 마지막으로 받은 메시지를 다시 검색할수 있습니다.
