# 기능 표시줄 명령

### 요약

> 기능 표시줄을 표시하거나 숨깁니다.

### 설명

> 엠에디터 창 아래쪽에 기능 표시줄을 표시하거나 숨깁니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **기능 표시줄**
- [모든 명령](../tools/all_commands): **보기** >
**기능 표시줄**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_FUNCTION\_BAR (4448)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4448);

#### \[VBScript\]

> editor.ExecuteCommandByID 4448
