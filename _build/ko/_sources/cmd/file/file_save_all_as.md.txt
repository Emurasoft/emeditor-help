# 인코딩하여 모두 저장 명령

### 요약

> 제목이 없는 문서들을 제외하고 열려있는 모든 문서들을 지정된 대상 폴더, 인코딩 및 반환 방법으로 모두 저장합니다.

### 설명

> 이 명령은 (제목이 지정되지 않은 문서를 제외한) 열려있는 모든 파일을 저장할 옵션을 지정할수 있는 **인코딩하여 모두 저장** 대화 상자를 나타냅니다.
> 이 대화 상자는 대상 폴더, 저장 할 인코딩 및 반환 방법을 선택할수 있게 합니다.
>
> 이 명령을 사용하기 위해서는 이 명령을 통해 저장하고자 하는 열려져 있는 문서가 이전에 저장되어 있어야만 합니다.
> 즉, **저장** 또는 **다른 이름으로 저장** 명령을 통해서 아직 생성 되지 않은 무제의 문서들은 **인코딩하여 모두 저장** 명령을 사용할수 없습니다.

### 실행하는 방법

- 기본 메뉴: **파일** \> **인코딩하여 모두 저장**
- [모든 명령](../tools/all_commands): **파일** \> **인코딩하여 모두 저장**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FILE\_SAVE\_ALL\_AS (3843)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3843);

#### \[VBScript\]

> editor.ExecuteCommandByID 3843
