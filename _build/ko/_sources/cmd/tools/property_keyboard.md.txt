# 키보드 속성 명령

### 요약

> 현재 구성의 키보드 속성을 선택합니다.

### 설명

> **[구성 속성](../../dlg/properties/index)** 대화 상자의
> [**키보드**\
> 탭](../../dlg/properties/keyboard/index) 을 표시합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**현재 구성 속성** \> **키보드**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_PROPERTY\_KEYBOARD (8974)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(8974);

#### \[VBScript\]

> editor.ExecuteCommandByID 8974
