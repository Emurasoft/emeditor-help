# 도구 작업 종료 명령

### 요약

> 현재 실행중인 도구 작업을 종료합니다.

### 설명

> 현재 실행중인 도구 작업을 종료합니다.
> 이 명령은 출력 바를 사용한 외부 도구에 의해서 생성된 모든 프로세스를 종료합니다.

### 실행하는 방법

- 기본 메뉴: **도구** \> **외부 도구** \> **도구 작업 종료**
- [모든 명령](all_commands): **도구** >
**도구 작업 종료**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+BREAK

### 플러그인 명령 ID

- EEID\_TERMINATE\_TOOL (4451)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4451);

#### \[VBScript\]

> editor.ExecuteCommandByID 4451
