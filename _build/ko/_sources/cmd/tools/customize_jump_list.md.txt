# 점프 목록 사용자 지정 명령

### 요약

> 점프 목록을 사용자 지정합니다.

### 설명

> 점프 목록을 사용자 지정합니다.
> 점프 목록은 엠에디터 아이콘을 마우스 오른쪽 버튼으로 클릭 하였을 때 표시 줄에 나타납니다.
> 이 기능은 Windows 7 이상에서 사용 가능 합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**점프 목록 사용자 지정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CUSTOMIZE\_JUMP\_LIST (4455)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4455);

#### \[VBScript\]

> editor.ExecuteCommandByID 4455
