# 이전 창 명령

### 요약

> 이전 창으로 전환합니다.

### 설명

> 창이 여러 내부 창으로 분할되어 있는 경우, 커서를 이전 창으로 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **창** \> **이전 창**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: SHIFT+F6

### 플러그인 명령 ID

- EEID\_PREV\_PANE (4215)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4215);

#### \[VBScript\]

> editor.ExecuteCommandByID 4215
