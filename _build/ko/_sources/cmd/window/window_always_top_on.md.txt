# 항상 위 \- 켜짐 명령

### 요약

> 항상 이 창을 상단에 표시합니다.

### 설명

> 항상 현재 창을 상단에 표시합니다.
> 다른 문서들은 **항상 위** 에 설정되어 있는 문서 아래에 표시됩니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \>
**항상 위** \>
**항상 위 \- 켜짐**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WINDOW\_ALWAYS\_TOP\_ON (4343)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4343);

#### \[VBScript\]

> editor.ExecuteCommandByID 4343
