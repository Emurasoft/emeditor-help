# 매크로 참조 명령

### 요약

> 엠에디터 매크로 참조를 표시합니다.

### 설명

> 엠에디터 도움말의 매크로 참조 탭을 표시합니다.

### 실행하는 방법

- 기본 메뉴: **매크로** \> **참조**
- [모든 명령](../tools/all_commands): **매크로**
\> **참조**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_MACRO\_HELP (4360)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4360);

#### \[VBScript\]

> editor.ExecuteCommandByID 4360
