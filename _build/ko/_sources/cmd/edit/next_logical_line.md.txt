# 다음 논리 줄 명령

### 요약

> 다음 논리 줄의 시작 부분으로 커서를 이동합니다.

### 설명

> 다음 논리 줄의 시작 부분으로 커서를 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **세로로 커서 이동** \> **다음 논리 줄**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+PAGE DOWN

### 플러그인 명령 ID

- EEID\_NEXT\_LOGICAL\_LINE (4268)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4268);

#### \[VBScript\]

> editor.ExecuteCommandByID 4268
