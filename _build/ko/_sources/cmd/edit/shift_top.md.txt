# 맨 위까지 선택 명령

### 요약

> 문서의 맨 위까지 선택을 확장합니다.

### 설명

> 문서의 맨 위까지 선택을 확장합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **선택 확장** > **맨 위까지 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+SHIFT+HOME

### 플러그인 명령 ID

- EEID\_SHIFT\_TOP (4184)

### 매크로

#### \[JavaScript\]

> document.selection.StartOfDocument(true);

#### \[VBScript\]

> document.selection.StartOfDocument true
