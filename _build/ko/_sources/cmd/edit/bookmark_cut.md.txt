# 이 문서의 책갈피된 줄 잘라내기 명령

### 요약

> 현재 문서의 책갈피된 모든 줄을 잘라내어 클립보드로 이동합니다.

### 설명

> 현재 문서의 책갈피된 모든 줄을 잘라내어 클립보드로 이동합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **책갈피** \> **책갈피된 줄 잘라내기**
- [모든 명령](../tools/all_commands): **편집** \> **책갈피** \> **책갈피된 줄 잘라내기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_BOOKMARK\_CUT (3852)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3852);

#### \[VBScript\]

> editor.ExecuteCommandByID 3852
