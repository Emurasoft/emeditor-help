# 바뀐 내용 책갈피 명령

### 요약

> 변경된 모든 줄에 책갈피를 표시합니다.

### 설명

> 비교 모드에서 변경된 모든 줄에 책갈피를 표시합니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **바뀐 내용 책갈피**
- [모든 명령](../tools/all_commands): **비교** \> **바뀐 내용 책갈피**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_COMPARE\_BOOKMARK (4593)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4593);

#### \[VBScript\]

> editor.ExecuteCommandByID 4593
