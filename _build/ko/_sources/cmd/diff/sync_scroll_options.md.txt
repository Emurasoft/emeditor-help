# 옵션과 스크롤링 동기화 명령

### 요약

> 스크롤링 동기화 마법사를 실행하여 최근에 본 두 문서의 스크롤을 동기화합니다.

### 설명

> 옵션을 지정하는 기능을 사용하는 스크롤링 동기화 마법사를 실행하여 최근에 본 두 문서의 스크롤을 동기화합니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **옵션과 스크롤링 동기화**
- [모든 명령](../tools/all_commands): **비교** \> **옵션과 스크롤링 동기화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SYNC\_SCROLL\_옵션 (4496)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4496);

#### \[VBScript\]

> editor.ExecuteCommandByID 4496
