# 대/소문자 무시 명령

### 요약

> 문서를 비교할 때 대/소문자를 무시합니다.

### 설명

> 문서를 비교할 때 대/소문자를 무시합니다. (대/소문자 구분하지 않는 비교 실행).

### 실행하는 방법

- 기본 메뉴: **비교** \> **대/소문자를 무시**
- [모든 명령](../tools/all_commands): **비교** \> **대/소문자를 무시**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_IGNORE\_CASES (4501)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4501);

#### \[VBScript\]

> editor.ExecuteCommandByID 4501
