# 대／소문자 구분（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음에서 대/소문자 구분 버튼을 설정/해제 합니다.

### 설명

> 찾기 도구 모음에서 대/소문자 구분 버튼을 설정/해제 합니다.
> 이 명령이 설정되면 대/소문자를 구분하여 검색합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **대／소문자 구분**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_CASE (4573)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4573);

#### \[VBScript\]

> editor.ExecuteCommandByID 4573
