# PrintProp 개체

## 속성

|     |     |
| --- | --- |
| [DrawSeparatingLines](draw_separating_lines) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **구분 선 그리기** 체크 박스에 해당합니다. |
| [Footer](footer) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **바닥글** 텍스트 박스에 해당합니다. |
| [Header](header) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **머리글** 텍스트 박스에 해당합니다. |
| [IgnoreColor](ignore_color) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **색과 밑줄 무시하기** 체크 박스에 해당합니다. |
| [LineSpace](line_space) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **줄 사이 공백** 텍스트 박스에 해당합니다. |
| [ShowLineNumbers](show_line_numbers) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **줄 번호 인쇄** 체크 박스에 해당합니다. |
| [WrapPage](wrap_page) | 구성 속성 [**인쇄** 탭](../../dlg/properties/print/index) 의<br> **인쇄 시 페이지에 따라 줄 바꾸기** 체크 박스에 해당합니다. |

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.

```{toctree}
:maxdepth: 1
draw_separating_lines
footer
header
ignore_color
line_space
show_line_numbers
wrap_page
```
