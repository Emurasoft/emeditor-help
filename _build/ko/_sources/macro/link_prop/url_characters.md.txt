# URLCharacters 속성

구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의 **이 문자들을 인식** 텍스트 박스에 해당합니다.

#### \[JavaScript\]

_s_ = object. **URLCharacters**;

object. **URLCharacters** = _s_;

#### \[VBScript\]

_s_ = object. **URLCharacters**

object. **URLCharacters** = _s_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
