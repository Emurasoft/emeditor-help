# HighlightCharRef 속성

구성 속성 [**강조 표시 (2)** 탭](../../dlg/properties/highlight2/index) 의 **HTML/XML 문자 참조와 UCN에 강조 표시** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **HighlightCharRef**;

object. **HighlightCharRef** = _b_;

#### \[VBScript\]

_b_ =
object. **HighlightCharRef**

object. **HighlightCharRef** = _b_

## 버전

엠에디터 프로페셔널 버전 11.00 이상에서만 지원됩니다.
