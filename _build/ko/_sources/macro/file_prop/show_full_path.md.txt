# ShowFullPath 속성

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **전체 경로와 함께 파일 이름 보이기** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **ShowFullPath**;

object. **ShowFullPath** = _b_;

#### \[VBScript\]

_b_ =
object. **ShowFullPath**

object. **ShowFullPath** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
