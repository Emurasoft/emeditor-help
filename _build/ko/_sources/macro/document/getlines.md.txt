# GetLines 메서드

문서의 줄 수를 검색합니다.

#### \[JavaScript\]

_nLines_ = document. **GetLines**( \[ _nFlags_ \] );

#### \[VBScript\]

_nLines_ = document. **GetLines**( \[ _nFlags_ \] )

## 매개 변수

_nFlags_

선택 사항입니다. 다음의 값을 지정할 수 있습니다.
아무런 값이 지정되지 않은 경우, 논리적 좌표가 지정됩니다.

|     |     |
| --- | --- |
| eeGetLineView | 좌표가 보이게 지정됩니다. |

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
