# 상태 탭

**상태 탭** 은 엠에디터 상태 표시줄에 관련된 설정을 사용자 지정하도록 합니다.

### 상태 리스트 박스

상태 표시줄에 표시되는 리스트의 항목을 표시합니다. 이 목록은 다음의 항목들을 포함하고 있습니다:

|     |     |
| --- | --- |
| **현재 구성** | 현재 선택된 구성을 표시합니다. |
| **커서 위치** | 커서가 위치한 현재 행과 열을 표시합니다. |
| **커서 문자 코드** | 커서가 위치한 문자 코드를 표시합니다. |
| **현재 인코딩** | 현재 인코딩을 표시합니다. |
| **덮어쓰기 모드** | 덮어쓰기 모드를 표시합니다. |
| **읽기 전용 모드** | 읽기 전용 모드를 표시합니다. |
| **현재 폴더** | 현재 폴더를 표시합니다. |
| **반환 방법** | 전체 파일의 반환 방법을 표시합니다 (대용량 파일에는 사용 불가능 합니다). |
| **문자 수** | 선택된 문자의 수를 표시합니다. |
| **단어 개수** | 선택된 단어의 수를 표시합니다 (대용량 파일에는 사용 불가능 합니다). |
| **라인 수** | 선택된 라인의 수를 표시합니다. |
| **메모리 사이즈** | 메모리 사이즈를 표시합니다. |
| **L1 캐시 사이즈** | L1 캐시 사이즈를 표시합니다. |
| **L2 캐시 사이즈** | L2 캐시 사이즈를 표시합니다. |
| **사용중인 실제 메모리의 백분율** | 사용중인 실제 메모리의 백분율을 표시합니다. |

### 위로 버튼

선택된 항목을 목록의 위로 이동합니다.

### 아래로 버튼

선택된 항목을 목록의 아래로 이동합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
