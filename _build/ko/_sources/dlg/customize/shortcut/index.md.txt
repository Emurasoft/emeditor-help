# 바로가기 탭

**바로가기 탭** 은 바로가기에 관련된 설정을 사용자 지정하도록 합니다.

### "보내기" 메뉴에 바로가기 추가 체크 박스

엠에디터에서 탐색기 "보내기" 메뉴에 바로가기를 추가합니다.

### 작업 표시줄에 트레이 아이콘 나타내기 체크 박스

윈도우 작업 표시줄 알림 영역에 엠에디터 트레이 아이콘을 표시합니다.

### 다음 Windows 시작시 트레이 아이콘 나타내기 체크 박스

다음 Windows 시작시 엠에디터 트레이 아이콘을 표시합니다.

### 엠에디터 빠른 시작 체크 박스

엠에디터 트레이 아이콘이 표시되는 동안 엠에디터를 빠르게 시작합니다.
이 항목이 선택된 경우, 숨겨져 있는 엠에디터 창이 엠에디터 트레이 아이콘으로 동시에 실행됩니다.
엠에디터를 시작할 때 숨겨진 엠에디터 창이 표시 상태로 전환되고, 엠에디터가 빠르게 실행됩니다.

### 더 많은 바로가기 버튼

더 많은 바로가기를 사용자 지정할 수 있는 [**더 많은 사용자 지정 바로가기** 대화 상자](more_shortcuts/index)
를 표시합니다.

### 트레이 아이콘 사용자 지정 버튼

시스템 트레이 아이콘 설정을 사용자 지정할 수 있는 [**트레이 아이콘 사용자 지정** 대화 상자](../../tray/index)
를 표시합니다.
