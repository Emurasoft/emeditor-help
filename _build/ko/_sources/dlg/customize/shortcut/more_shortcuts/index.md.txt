# 더 많은 사용자 지정 바로가기 대화 상자

이 대화 상자는 [**사용자 지정 대화 상자**](../../index) 의 [**바로가기 탭**](../index)
에서 더 많은 바로가기 버튼을 선택하면 나타납니다. 이 대화 상자는 관리자 권한으로 로그온 되어있는 경우에만 사용 가능합니다.

### 익스플로러의 상황에 맞는 메뉴에 바로가기 추가 체크 박스

엠에디터에서 Explorer 컨텍스트 메뉴에 바로가기를 추가합니다.
이 옵션은 관리자 권한으로 로그온 되어있는 경우에만 사용 가능합니다.

### Internet Explorer에서 EmEditor로 원본 보기 체크 박스

엠에디터가 Internet Explorer의 소스보기 명령에 사용되도록 설정합니다.
이 옵션은 관리자 권한으로 로그온 되어있는 경우에만 사용 가능합니다.

### EmEditor를 Internet Explorer HTML 편집기 목록에 추가 체크 박스

Internet Explorer에서 사용되는 HTML 편집기 목록에 엠에디터를 추가합니다.
이 옵션은 관리자 권한으로 로그온 되어있는 경우에만 사용 가능합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
