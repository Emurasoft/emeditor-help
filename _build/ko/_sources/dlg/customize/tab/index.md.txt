# 탭 탭

**탭 탭** 은 탭에 관련된 설정을 사용자 지정하도록 합니다

### 스타일 드롭다운 리스트 박스

문서 탐색 창의 표시 방법을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **탭** | 탭으로 표시합니다. |
| **버튼** | 도구 모음 버튼으로 표시합니다. |

### 아이콘 드롭다운 리스트 박스

각 탭의 아이콘의 표시 방법을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **없음** | 아이콘을 표시하지 않습니다. |
| **문서 유형 아이콘** | 열린 문서 유형과 관련된 아이콘을 표시합니다. |
| **엠에디터 아이콘** | 엠에디터 아이콘을 표시합니다. |

### 삽입 위치 드롭다운 리스트 박스

새 탭이 삽입될 곳을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **맨 왼쪽** | 새로운 탭이 가장 왼쪽에 삽입됩니다. |
| **맨 오른쪽** | 새로운 탭이 가장 오른쪽에 삽입됩니다. |
| **동작 탭의 왼쪽** | 새로운 탭이 활성화된 문서 왼쪽에 삽입됩니다. |
| **동작 탭의 오른쪽** | 새로운 탭이 활성화된 문서 오른쪽에 삽입됩니다. |

### 종료 시 드롭다운 리스트 박스

탭을 닫을 때 활성화 될 탭을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **가장 최근 탭 활성화** | 가장 최근에 사용된 탭을 활성화합니다. |
| **왼쪽 탭 활성화** | 닫히는 탭의 왼쪽 탭을 활성화합니다. |
| **오른쪽 탭 활성화** | 닫히는 탭의 오른쪽 탭을 활성화합니다. |

### 열기 드롭다운 리스트 박스

어느 창에 새로운 파일을 열지 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **새 창** | 새로운 파일과 기존의 파일을 모두 새 창에 엽니다. |
| **가장 최근 창에 새로운 탭** | 새로운 파일과 기존의 파일을 모두 가장 최근 창에 엽니다. 하지만, 탭이 사용되지 않거나 플러그인이 실행되고 있으면<br> 파일은 항상 새로운 창에 열리게 됩니다. |
| **새 파일을 새창으로 열기, 기타 새 탭** | 새 파일은 새창에, 새 탭으로 기존의 파일을 엽니다. 하지만, 탭이 사용되지 않거나 플러그인이 실행되고 있으면<br> 파일은 항상 새로운 창에 열리게 됩니다. |

### 너비 드롭다운 리스트 박스

각 탭의 너비를 조정하는 법을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **파일 이름 변경** | 각 탭이 해당 파일 이름 길이에 조정됩니다. |
| **지정된 너비보다 긴 경우에만 자르기** | 각 탭이 해당 파일 이름 길이에 조정되지만, **지정된 너비** 텍스트 박스에서 지정된 너비보다<br> 긴 경우에만 자릅니다. |
| **지정된 너비로 수정** | 모든 탭이 **지정된 너비** 텍스트 박스에서 지정된 너비와 길이가 같게됩니다. |

### 맞지 않을 경우 드롭다운 리스트 박스

창 너비에 탭이 맞지 않을 경우 탭을 표시할 방법을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **없음** | 아무 작업도 수행하지 않습니다. |
| **너비 조정** | 창 너비에 맞게 각 탭의 너비를 조정합니다. |
| **행 추가** | 모든 탭을 표시하기 위해 행을 추가합니다. |

### 닫기 버튼 드롭다운 리스트 박스

닫기 버튼을 표시할 방법을 지정합니다. 아래의 옵션중에 하나를 선택합니다.

|     |     |
| --- | --- |
| **없음** | 닫기 버튼을 표시하지 않습니다. |
| **창의 오른쪽** | 창의 오른쪽에 닫기 버튼을 표시합니다. |
| **활성 탭** | 활성화된 탭에만 닫기 버튼을 표시합니다. |
| **각 탭** | 각 탭에 닫기 버튼을 표시합니다. |

### 지정된 너비 텍스트 박스

**너비** 드롭다운 리스트 박스에서 **지정된 너비보다 긴 경우에만 자르기** 또는
**지정된 너비로 수정** 이 선택되었을 때 각 탭의 너비를 지정합니다.

### 읽기 전용 아이콘 보이기 체크 박스

읽기 전용 모드로 열린 파일이 위치한 탭에 읽기 전용 아이콘을 보여줍니다.

### 사용자 지정 메뉴 버튼

메뉴를 사용자 지정하기 위해 [**사용자 지정 메뉴** 대화 상자](../../menus/index) 를
표시합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
