# 트레이 아이콘 사용자 지정 대화 상자

이 대화 상자는 [**트레이 아이콘 사용자 지정** 명령](../../cmd/tools/customize_tray) 이
선택되면 나타납니다.

### 왼쪽 마우스 버튼 클릭 드롭다운 리스트 박스

트레이 아이콘에서 왼쪽 마우스 버튼을 클릭하였을 때 실행되는 작업을 지정합니다.

### 오른쪽 마우스 버튼 클릭 드롭다운 리스트 박스

트레이 아이콘에서 오른쪽 마우스 버튼을 클릭하였을 때 실행되는 작업을 지정합니다.

### 중간 마우스 버튼 클릭 드롭다운 리스트 박스

트레이 아이콘에서 중간 마우스 버튼을 클릭하였을 때 실행되는 작업을 지정합니다.

### 왼쪽 마우스 버튼 더블 클릭 드롭다운 리스트 박스

트레이 아이콘에서 왼쪽 마우스 버튼을 더블 클릭하였을 때 실행되는 작업을 지정합니다.

### 왼쪽 마우스 버튼 바로가기 키 버튼 텍스트 박스

왼쪽 마우스 버튼의 바로가기 키를 지정합니다. 예를 들어, N을 입력하면 ALT + CTRL + N이
단축키로 지정될수 있습니다.

### 엠에디터로 텍스트 가져오기 바로가기 키 텍스트 박스

초점이 맞춰진 편집 상자의 내용으로 엠에디터 창을 시작하는 바로가기 키를 지정하고,
엠에디터 창을 닫을 때는 기존의 편집 상자가 엠에디터의 내용으로 채워집니다. 예를 들어, X를 입력하면
ALT + CTRL + X가 단축키로 지정될수 있습니다.

### 사용자 지정 메뉴 버튼

메뉴를 사용자 지정하는 [**사용자 지정 메뉴** 대화 상자](../menus/index) 를
표시합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
