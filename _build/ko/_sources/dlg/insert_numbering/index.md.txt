# 번호 매기기 삽입 대화 상자

이 대화 상자는 [**번호 매기기** 명령](../../cmd/edit/numbering) 이 선택되었을 때 나타납니다.
커서 위치 또는 수직 선택에 일련 번호를 삽입할 수 있도록 합니다.

### 첫번째 줄 텍스트 박스

첫번째 줄에 삽입할 초기 값 또는 문자를 지정합니다.

### 증분 텍스트 박스

십진수 표기법으로 증분을 지정합니다. 첫번째 줄과 두번째 줄 사이 값의 변화입니다.

### 줄의 개수 텍스트 박스

십진수 표기법으로 줄의 개수를 지정합니다.

### 16진수 라디오 버튼

16진수로 숫자를 삽입합니다.

### 10진수 라디오 버튼

10진수로 숫자를 삽입합니다.

### 8진수 라디오 버튼

8진수로 숫자를 삽입합니다.

### 2진수 라디오 버튼

2진수로 숫자를 삽입합니다.

### 문자 라디오 버튼

숫자 대신에 문자를 삽입합니다. 첫번째 줄 텍스트 박스에서 지정된 숫자로 시작하면,
이 옵션은 증분 텍스트 박스에서 지정된 유니코드 값의 증분을 이용하여 일련 번호를 삽입합니다.
각 줄에 한 문자씩만 삽입할 수 있습니다.

### 대문자 체크 박스

대문자로 16진수 값을 삽입합니다.

### 빈 줄 건너뛰기

빈 줄에 번호를 새기지 않고, 빈 줄 다음부터 번호 새기기를 계속합니다.

### 빈 줄 다음부터 다시 번호 매기기

빈 줄 다음부터 첫 번째 값으로 다시 번호 새기기를 시작합니다.
