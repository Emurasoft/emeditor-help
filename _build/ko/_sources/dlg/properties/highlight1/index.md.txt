# 강조 표시 (1) 탭

**강조 표시 (1)** 탭은 강조 표시 (1)에 관련된 속성을 설정하도록 합니다.

### 이 단어들을 강조 표시 체크 박스

강조 표시된 문자열을 사용합니다.

### 강조 표시 된 단어 리스트 박스

강조 표시된 문자열의 목록입니다. 체크 박스를 클릭하면 색상을 변경합니다.
숫자들은 [**디스플레이** 탭](../display/index) 에 지정된 강조 표시 색상에 해당합니다.

### 추가 버튼

리스트에 새로운 항목을 추가하려면 이 버튼을 선택합니다.

### 삭제 버튼

리스트에 선택된 항목을 삭제하려면 이 버튼을 선택합니다.

### 전체 단어 만 체크 박스

전체 단어가 문자열인 경우에만 선택된 문자열을 강조 표시합니다.

### 오른쪽 강조 표시 체크 박스

선택된 단어와 단어의 오른쪽을 줄 바꿈한 위치의 단어까지만 강조 표시합니다.

### 오른쪽 모두 강조 표시 체크 박스

줄 위의 선택된 단어와 단어의 오른쪽에 있는 모든 부분을 창의 테두리 (줄 바꾸기 없음이나 창에 맞춰 줄바꿈이 선택된 경우)
또는 지정된 너비나 페이지 너비까지 강조 표시합니다.

### 대/소문자 구분 체크 박스

대/소문자가 일치하는 경우에만 선택된 단어를 강조 표시합니다.

### 내부 태그 만 체크 박스

문자열이 태그안에 있는 경우에만 문자열을 강조 표시합니다.

### 정규식 체크 박스

정규식 표현을 사용한 선택된 문자열을 강조 표시합니다.

### 태그 시작 텍스트 박스

태그의 시작 문자입니다. 태그안에 존재하는 경우에만 강조 표시될 각 문자열을 설정할 수 있습니다.

### 태그 종료 텍스트 박스

태그의 종료 문자입니다. 태그안에 존재하는 경우에만 강조 표시될 각 문자열을 설정할 수 있습니다.

### 가져오기 버튼

CSV 또는 ESY 유형 파일에서 강조 표시된 문자열을 가져옵니다.
CSV 파일 내용은 문자열, **색상**, **전체 단어 만**, **오른쪽 강조 표시**,
**대/소문자 구분**, 및 **내부 태그 만** 으로 구성됩니다.
ESY 파일 구문은 다음 페이지에서 찾아볼 수 있습니다.

**참조**

- [새로운 구문 파일 만들기](../../../howto/customize/syntax_file)

### 보내기 버튼

CSV 또는 ESY 유형 파일에서 강조 표시된 문자열을 내보냅니다.
CSV 파일 내용은 문자열, **색상**, **전체 단어 만**, **오른쪽 강조 표시**,
**대/소문자 구분**, 및 **내부 태그 만** 으로 구성됩니다.
ESY 파일 구문은 다음 페이지에서 찾아볼 수 있습니다.

**참조**

- [새로운 구문 파일 만들기](../../../howto/customize/syntax_file)

### 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
