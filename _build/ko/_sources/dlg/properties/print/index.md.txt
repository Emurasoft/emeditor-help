# 인쇄 탭

**인쇄** 탭은 인쇄 작업과 관련된 속성을 설정하도록 합니다.

### 인쇄 줄 번호 체크 박스

페이지 왼쪽 아래에 줄 번호를 인쇄합니다.

### 색과 밑줄 무시하기 체크 박스

인쇄 할 때 색상과 밑줄을 무시합니다.

### 구분 선 그리기 체크 박스

텍스트와 머리글사이와 텍스트와 바닥글 사이에 수평한 선을 삽입합니다.

### 인쇄 시 페이지에 따라 줄 바꾸기 체크 박스

인쇄 할 때 페이지의 너비에 따라 줄 바꾸기를 합니다.

### 줄 사이 공백 텍스트 박스

인쇄 할 때 줄 사이 공백의 간격을 지정합니다.
이 값은 한 줄 높이의 1/10에 해당합니다.

### 머리글 체크 박스

지정된 머리글을 활성화 합니다.

### 머리글 텍스트 박스

다음의 표시들을 사용하여 페이지 상단에 문서 정보를 인쇄합니다.

### \> 버튼

미리 정의된 표시 목록에서 선택하려면 이 버튼을 사용합니다.

|     |     |
| --- | --- |
| &f | 파일 이름 |
| &n | 경로 이름 |
| &p | 페이지 |
| &a | 총 페이지 |
| &d | 날짜 |
| &t | 시간 |
| &c | 가운데 맞춤 |
| &r | 오른쪽 맞춤 |

### 바닥글 체크 박스

지정된 바닥글을 활성화 합니다.

### 바닥글 텍스트 박스

다음의 표시들을 사용하여 페이지 하단에 문서 정보를 인쇄합니다.

### \> 버튼

미리 정의된 표시 목록에서 선택하려면 이 버튼을 사용합니다.

|     |     |
| --- | --- |
| &f | 파일 이름 |
| &n | 경로 이름 |
| &p | 페이지 |
| &a | 총 페이지 |
| &d | 날짜 |
| &t | 시간 |
| &c | 가운데 맞춤 |
| &r | 오른쪽 맞춤 |

### 페이지 설정 버튼

페이지 설정을 지정하려면 이 버튼을 사용합니다.

### 글꼴 버튼

인쇄에 사용될 글꼴을 지정합니다.

### 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
