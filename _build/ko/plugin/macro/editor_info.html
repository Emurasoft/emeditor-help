Editor_Info¶

EmEditor에서 사용되는 정보 매개 변수 중 하나의 값을 설정하거나 검색합니다. 이 인라인 함수를 사용하거나 EE_INFO 메시지를 명시적으로 보낼 수 있습니다.

Editor_Info( HWND hwnd, int nCmd, LPARAM lParam );

매개 변수¶

nCmd

검색하거나 설정할 매개 변수를 지정합니다. 이 매개 변수는 다음 테이블 값들 중 하나가 될 수 있습니다.

nCmd

의미

lParam

Return 값

사용되지 않습니다.

(int)nCP
