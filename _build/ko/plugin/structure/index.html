플러그 인 참조 < 구조

구조¶

ATTR_INFO

EE_GET_ATTR 메시지에 의해 사용됩니다.

CLIP_INFO

EE_CLIP_HISTORY 메시지에 의해 사용됩니다.

CUSTOM_BAR_INFO

Editor_CustomBarOpen 인라인 함수
( EE_CUSTOM_BAR_OPEN 메시지)에 의해 사용됩니다.

CUSTOM_BAR_CLOSE_INFO

EVENT_CUSTOM_BAR_CLOSED 이벤트 에 의해 사용됩니다.

FIND_REGEX_INFO

Editor_FindRegex 인라인 함수
( EE_FIND_REGEX 메시지)에 의해 사용됩니다.

GET_LINE_INFO

Editor_GetLineA 와 Editor_GetLineW 인라인 함수 (EE_GET_LINEA,
EE_GET_LINEW 메시지)에 의해 사용됩니다.

GREP_INFOA

Editor_FindInFilesA 인라인 함수,
Editor_ReplaceInFilesA 인라인 함수
( EE_FIND_IN_FILESA 메시지, EE_REPLACE_IN_FILESA 메시지)에 의해 사용됩니다.

GREP_INFOW

Editor_FindInFilesW 인라인 함수,
Editor_ReplaceInFilesW 인라인 함수
( EE_FIND_IN_FILESW 메시지, EE_REPLACE_IN_FILESW 메시지)에 의해 사용됩니다.

HISTORY_INFO

EVENT_HISTORY 이벤트 에 의해 사용됩니다.

LOAD_FILE_INFO_EX

Editor_LoadFileA 와
Editor_LoadFileW
인라인 함수 ( EE_LOAD_FILEA 와
EE_LOAD_FILEW 메시지)에 의해 사용됩니다.

MATCH_REGEX_INFO

Editor_MatchRegex 인라인 함수
( EE_MATCH_REGEX 메시지)에 의해 사용됩니다.

OUTLINE_ARRAY_INFO

Editor_SetOutlineArray
인라인 함수( EE_SET_OUTLINE_ARRAY 메시지)에 의해 사용됩니다.

POINT_PTR

포인트 위치를 지정하는데 사용됩니다. 32 비트 플러그 인에서 POINT_PTR은 POINT 구조와 동일합니다.
64 비트 플러그 인에서 각 필드는 32비트 정수부터 64비트 정수로 확장됩니다.

REG_QUERY_VALUE_INFO

EE_REG_QUERY_VALUE 메시지 에 의해 사용됩니다.

REG_SET_VALUE_INFO

EE_REG_SET_VALUE 메시지 에 의해 사용됩니다.

RUN_MACRO_INFO

EE_RUN_MACRO 메시지에 의해 사용됩니다.

SEL_INFO

Editor_GetMultiSel 인라인 함수
( EE_GET_MULTI_SEL 메시지)에 의해 사용됩니다.

SIZE_PTR

크기를 지정하는데 사용됩니다. 32 비트 플러그 인에서 SIZE_PTR은 SIZE 구조와 동일합니다.
64 비트 플러그 인에서 각 필드는 32비트 정수부터 64비트 정수로 확장됩니다.

TEMP_INFO

EE_EDIT_TEMP 메시지와
EVENT_TEMP_SAVING 이벤트 에 의해 사용됩니다.

TOOLBAR_INFO

Used by Editor_ToolbarOpen 인라인 함수
( EE_TOOLBAR_OPEN 메시지)와 사용자 지정 도구 모음에 관련된
이벤트에 의해 사용됩니다.
