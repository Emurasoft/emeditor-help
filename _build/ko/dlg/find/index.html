찾기 대화 상자¶

이 명령은 찾기 명령 이 선택되었을 때 나타납니다. 찾을 문자열과 다른 옵션들 또한 지정할 수 있습니다.

찾기 드롭다운 리스트 박스¶

검색할 문자열을 입력합니다. 다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다. 드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

> 버튼¶

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

선택된 텍스트 또는 커서에 단어

이 항목이 선택된 경우, 대화 상자는 선택된 텍스트 또는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다.

선택된 텍스트

이 항목이 선택된 경우, 대화 상자는 선택된 텍스트로 찾기 드롭다운 리스트 박스를 초기화 합니다.

커서에 단어

이 항목이 선택된 경우, 대화 상자는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다.

사용자 지정

이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열로 찾기 드롭다운 리스트 박스를 초기화 합니다.

기본값으로 저장

(선택된 텍스트, 커서에 단어 및 사용자 지정)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다.

다중 선

드롭다운 리스트 박스의 한 선과 다중선 사이에서 선택/취소 합니다.
(이 명령은 찾기 대화 상자와 바꾸기 대화 상자에서만 나타납니다.)

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다. 리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

대/소문자 구분 체크 박스¶

문자열을 검색할 때 대/소문자를 일치시킵니다.

정규식 사용 체크 박스¶

정규식 표현을 가능하게 합니다. 정규식 표현식 구문에 대한 더 자세한 내용은 정규식 사용하기 를 참조하십시기 바랍니다.

이스케이프 시퀀스 사용 체크 박스¶

이스케이프 시퀀스를 가능하게 합니다. 이스케이프 시퀀스로 다음의 문자들을 사용할 수 있습니다.

\a

경고 (벨)

\b

백 스페이스

\f

용지 공급

\n

새로운 줄

\t

수평 탭

\v

수직 탭

\

백 슬래시

\oooooo

8진수 유니코드 문자

\xhhhh

16진수 유니코드 문자

Null 문자 (\0)는 사용되지 않을 수 있습니다. \r대신 \n이 사용되어야만 합니다.

전체 단어 일치 체크 박스¶

단어만을 검색합니다. 단어는 이와 같은 문자( A – Z, a – z, 0 – 9, 또는 밑줄)로 시작하고 끝나는 문자열로 정의됩니다. 전자 문자로 둘러쌓인 문자열을 단어로 간주합니다. 정규식 표현이 사용되는 경우, 이 체크 박스는 제대로 작동하지 않을 수도 있습니다. 정규식 표현을 사용하는 경우, 대신 단어의 경계식 (\<, \> 및 \b)을 사용하시기 바랍니다.

증분 검색 체크 박스¶

이 항목이 선택된 경우, 찾기 드롭다운 리스트 박스에 입력하면 즉시 검색을 시작합니다.

모든 열린 문서 검색 체크 박스¶

동일한 프레임 창에 열려져 있는 모든 문서를 검색합니다.

문서의 시작/끝으로 이동 체크 박스¶

현재 위치에서 아래쪽으로 검색을 하기위해 아래 찾기 를 선택합니다. 파일의 끝까지 단어를 발견하지 못한 경우, 파일의 맨 위에서부터 검색을 이어갑니다. 위 찾기 는 현재 위치에서 파일의 시작 부분까지 검색합니다. 단어가 발견되지 않은 경우, 파일의 맨 아래에서부터 검색을 이어갑니다.

일치 횟수 체크 박스¶

이 항목이 선택된 경우, 엠에디터는 문서에서 일치되는 문자열의 발생 횟수를 계산합니다. 결과는 상태 표시 줄에 표시됩니다.

완료 시 닫기 체크 박스¶

검색의 완료되면 대화 상자를 닫습니다.

이전 찾기 버튼¶

현재 위치로 부터 이전 항목을 검색합니다.

다음 찾기 버튼¶

현재 위치로부터 다음 항목을 검색합니다.

모두 찾기 버튼¶

검색어의 모든 항목을 검색합니다.

모두 책갈피 버튼¶

지정된 문자열과 일치하는 모든 라인에 책갈피를 지정합니다.

바꾸기 >> 버튼¶

선택한 옵션으로 문자열을 지정할 바꾸기 대화 상자 를 나타내기 위해 이 버튼을 클릭합니다.

닫기 버튼¶

대화 상자를 닫기 위해 이 버튼을 클릭합니다.

사용자 지정 버튼¶

사용자 지정 대화 상자 를 표시하기 위해 이 버튼을 클릭합니다.

이 대화 상자는 대화 상자의 오른쪽 하단 모서리를 드래그하여 크기를 조정할 수 있습니다. 대화 상자가 커질 때, 여러줄의 문자열이 검색어로 입력될 수 있습니다. 다중선 텍스트 박스가 활성화 되는 동안, CTRL + ENTER키를 사용하여 새로운 줄을 삽입할 수 있습니다.
