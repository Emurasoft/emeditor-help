일반 탭¶

일반 탭은 기본 작업과 관련된 속성을 설정하도록 합니다.

줄 바꿈 드롭다운 리스트 박스¶

줄 바꿈하는 방법을 지정합니다. 다음 옵션 중 하나를 선택합니다.

텍스트 배치 없음

줄 바꿈을 하지 않습니다.

지정된 문자

지정된 문자의 길이(바이트)로 줄 바꿈을 합니다.
문자의 길이는 일반 줄과 여백 텍스트 박스 또는 따옴표 붙은 줄 여백 텍스트 박스에서 지정할 수 있습니다.

창 너비

창 너비에 따라 줄 바꿈을 합니다.
창의 크기 조정을 한 경우, 창을 줄 바꿈된 위치가 조정됩니다.

페이지 너비

인쇄될 페이지 너비에 따라 줄 바꿈을 합니다.

선과 열을 다음으로 나타냅니다 드롭다운 리스트 박스¶

행 번호와 열의 위치가 표시되는 방법을 지정합니다. 다음 옵션 중 하나를 선택합니다.

좌표 표시

행 번호와 열 위치가 표시된 것과 같이 계산됩니다.
줄 바꿈이 된 경우, 줄 바뀐 위치가 계산됩니다.
줄 바뀐 위치에서 열 위치가 1로 복원됩니다.
전자는 2로 계산됩니다.
워드 프로세서와 같은 표현이라 할 수 있습니다.

논리 좌표. (전자. 2 Col로.)

행 번호와 열 위치가 실제 논리적 줄로 계산됩니다.
행 번호와 열 위치는 행이 줄 바꿈하는 방법에 관여되지 않습니다.
전자는 2 열로 계산됩니다. 탭 문자는 한 문자로 계산됩니다.

논리 좌표.(전자. 1 Col로.)

행 번호와 열 위치가 실제 논리적 줄로 계산됩니다.
행 번호와 열 위치는 행이 줄 바꿈하는 방법에 관여되지 않습니다.
전자는 1 열로 계산됩니다. 탭 문자는 한 문자로 계산됩니다.

탭 논리(전자. 2 Col로.)

행 번호와 열 위치가 실제 논리적 줄로 계산됩니다.
행 번호와 열 위치는 행이 줄 바꿈하는 방법에 관여되지 않습니다.
전자는 2 열로 계산됩니다. 탭 문자는 공백으로 대체된 것처럼 계산됩니다.

일반 줄과 여백 텍스트 박스¶

지정된 문자로 줄 바꿈 이 선택되었을 때 따옴표 문자로 시작하지 않는 바이트 줄 길이입니다.

따옴표 붙은 줄여백 텍스트 박스¶

지정된 문자로 줄 바꿈 이 선택되었을 때 따옴표 문자로 시작하는 바이트 줄 길이입니다.

따옴표 문자 텍스트 박스¶

엠에디터가 따옴표 붙은 줄여백 텍스트 박스에 사용하는 따옴표 문자입니다. 따옴표 내 복사 명령 으로도 사용됩니다.

[실행 취소] 최대 수 텍스트 박스¶

실행 취소 명령 으로 실행 취소할 최대 수를 지정합니다. 일부 명령 작업들은 여러 버퍼 실행 취소가 요구됩니다, 따라서 지정한 수만큼의 실행 취소가 불가능할 수도 있습니다.

줄 번호 보이기 체크 박스¶

창의 왼쪽에 줄 번호를 표시합니다. 인쇄 시 줄 번호를 인쇄하려면 인쇄 페이지의 줄 번호 인쇄 체크 박스를 선택해야 합니다.

눈금자 보이기 체크 박스¶

창의 상단에 눈금자를 표시합니다.

입력 방법 편집기 실행 체크 박스¶

엠에디터를 실행할 때 입력 방법 편집기(IME)를 실행합니다.

창 왼쪽 가장자리에 공백 없음 체크 박스¶

줄 번호가 표시되지 않은 때 엠에디터 창의 왼쪽 가장자리에 공백을 표시하지 않습니다.

페이지 번호 보이기 체크 박스¶

페이지 번호를 표시합니다. 페이지를 나누는 선이 그려지고, 커서가 위치한 페이지를 표시하기 위해 상태 표시줄에 페이지 번호가 표시됩니다. 줄 번호가 표시된 경우, 줄 번호는 각 페이지 상단에 기반을 둡니다. 프린터를 사용할 수 없는 경우, 페이지 번호는 표시되지 않습니다.

컨트롤 문자 삽입 허용 체크 박스¶

이 항목이 선택된 경우 다른 곳에서 바로가기 키로 구성되지 않는 한, CTRL + 한 알페벳 키나 CTRL + SHIFT + 한 알페벳키를 입력하여 컨트롤 문자를 삽입할 수 있습니다. 이 항목이 선택되지 않은 경우, 이 방법으로 컨트롤 문자를 삽입할 수 없습니다.

(CR/LF)클립보드 반환 유지 체크 박스¶

이 항목이 선택된 경우, 복사 및 붙여넣기에 반환 방법이 유지됩니다. 예를 들어, LF만 반환 은 다른 위치에 LF만 반환 으로 복사하고 붙여넣습니다.

항상 ANSI로 붙여 넣기 체크 박스¶

이 항목이 선택된 경우, 유니코드가 아닌 클립보드로부터 텍스트를 붙여넣지만, 항상 시스템 기본 인코딩 을 사용합니다. 이 옵션은 시스템 기본 인코딩 에서 유니코드로 텍스트를 전환하는데 문제가 있는 다른 응용 프로그램에서 텍스트를 복사하기 원할 때 유용하게 사용될 수 있습니다.

가상 공간 활성화 체크 박스¶

이 항목이 선택되었을 때, 커서는 각 행의 마지막을 넘어 이동할 수 있고 각 행의 마지막 넘어에 문자를 삽입할 때 공백이 삽입됩니다.

탭/들여쓰기 버튼¶

자동 들여쓰기 설정을 지정하는 탭/들여쓰기 대화 상자 를 나타내려면 이 버튼을 누릅니다.

초기화 버튼¶

기본값으로 초기화합니다. 초기화 대화 상자 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.

이 페이지에서 다음의 대화 상자로 이동할 수 있습니다.

( 탭/들여쓰기 버튼 선택)
