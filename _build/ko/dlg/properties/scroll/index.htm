스크롤 탭¶

스크롤 탭은 스크롤 작업과 관련된 속성을 설정하도록 합니다.

가로 스크롤 바 드롭다운 리스트 박스¶

가로 스크롤 바 표시 방법을 지정합니다.

없음

가로 스크롤 바를 표시하지 않습니다.

필요할 경우에만 보이기

창 너비보다 줄의 길이가 길때 가로 스크롤 바를 표시합니다.

항상 보이기

항상 가로 스크롤 바를 표시합니다.

세로 스크롤 바 드롭다운 리스트 박스¶

세로 스크롤 바 표시 방법을 지정합니다.

없음

세로 스크롤 바를 표시하지 않습니다.

필요할 경우에만 보이기

현재 창에서 나타낼수 있는 것보다 전체 줄의 수가 많을 때 세로 스크롤 바를 표시합니다.

항상 보이기

세로 스크롤 바를 표시합니다.

부드럽게 스크롤 체크 박스¶

이 항목이 선택된 경우, 엠에디터 창은 부드럽게 스크롤합니다.

속도 슬라이더¶

이 슬라이더를 사용하여 부드러운 스크롤 속도를 조정합니다.

수직 스크롤 막대에 마커 표시 체크 박스¶

이 항목이 선택된 경우, 엠에디터 창은 수직 스크롤 막대에 마커를 표시합니다.

수직 스크롤 막대에 마커 표시 그룹 박스 - 변경 체크 박스¶

이 항목이 선택된 경우, 변경/저장된 줄은 수직 스크롤 막대에 강조 표시됩니다.

수직 스크롤 막대에 마커 표시 그룹 박스 - 표시 체크 박스¶

이 항목이 선택된 경우, 책갈피와 비교 결과는 수직 스크롤 막대에 강조 표시됩니다.

수직 스크롤 막대에 마커 표시 그룹 박스 - 문자열 검색 체크 박스¶

이 항목이 선택된 경우, 모든 검색 결과는 수직 스크롤 막대에 강조 표시됩니다.

수직 스크롤 막대에 마커 표시 그룹 박스 - 커서 위치 체크 박스¶

이 항목이 선택된 경우, 커서 위치는 수직 스크롤 막대에 강조 표시됩니다.

부드럽게 스크롤 체크 박스¶

이 항목이 선택된 경우, 엠에디터 창은 부드럽게 스크롤합니다.

항상 1 페이지 가로 스크롤 활성화 체크 박스¶

항상 1 페이지 가로 스크롤을 활성화 합니다.

항상 1 페이지 세로 스크롤 활성화 체크 박스¶

항상 1 페이지 세로 스크롤을 활성화 합니다.

가로 줄 체크 박스¶

커서에 가로 줄을 표시합니다.

세로 줄 체크 박스¶

커서에 세로 줄을 표시합니다.

활성 창에서만 스크롤 바 보이기 체크 박스¶

분할 창에서 현재 창이 활성화되었을 때만 스크롤 바를 표시합니다.

이중선 스크롤 체크 박스¶

한 번에 2줄씩 스크롤합니다. 창을 빠르게 스크롤하기 원하는 경우 이 항목을 선택합니다.

빠른 커서 움직임 체크 박스¶

커서의 움직임을 빠르게 합니다. 이 항목이 선택된 경우, 화살표 키가 누른 상태로 반복되어 질때 커서가 더블로 움직입니다.

팁: 제어판키보드 속성의속도탭에서 키보드 반복 속도를 조정할 수 있습니다.¶

스크롤링으로 커서 이동 체크 박스¶

이 항목이 선택된 경우, 가로 또는 세로 스크롤 바 위치가 변경되었을 때 커서를 재위치 합니다.

1/2 페이지 스크롤 체크 박스¶

일반적으로 전체 페이지를 스크롤 할 때 1/2 페이지만 스크롤합니다.

가로 눈금 체크 박스¶

가로 눈금을 표시합니다.

초기화 버튼¶

기본값으로 초기화합니다. 초기화 대화 상자 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
