파일 내 바꾸기 대화 상자¶

이 대화 상자는 파일 내 바꾸기 명령 을 선택하였을 때 나타납니다. 찾을 문자열과 파일 내에서 찾기 위한 다른 옵션들 또한 지정할 수 있습니다.

찾기 드롭다운 리스트 박스¶

검색할 문자열을 입력합니다. 다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다. 드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

> 버튼¶

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

선택된 텍스트 또는 커서에 단어

이 항목이 선택된 경우, 대화 상자는 선택된 텍스트 또는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다.

선택된 텍스트

이 항목이 선택된 경우, 대화 상자는 선택된 텍스트로 찾기 드롭다운 리스트 박스를 초기화 합니다.

커서에 단어

이 항목이 선택된 경우, 대화 상자는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다.

사용자 지정

이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열로 찾기 드롭다운 리스트 박스를 초기화 합니다.

기본값으로 저장

(선택된 텍스트, 커서에 단어 및 사용자 지정)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다.

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다. 리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

바꾸기 드롭다운 리스트 박스¶

찾은 문자열을 대체할 문자열을 입력합니다. 다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다. 드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

파일 유형 드롭다운 리스트 박스¶

검색하려는 파일 유형입니다. 와일드카드(*와 ?)가 사용 가능합니다. 또한, 여러 파일 형식은 세미콜론(;)으로 구분 가능합니다.

> 버튼¶

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

현재 파일 유형

이 항목이 선택된 경우, 대화 상자는 현재 파일 유형을 사용하여 파일 유형 드롭다운 리스트를 초기화 합니다.

현 파일 확장명

이 항목이 선택된 경우, 대화 상자는 현 파일 확장명을 사용하여 파일 유형 드롭다운 리스트를 초기화 합니다.

현재 파일

이 항목이 선택된 경우, 대화 상자는 현재 파일을 사용하여 파일 유형 드롭다운 리스트를 초기화 합니다.

사용자 지정

이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열을 사용하여 파일 유형 드롭다운 리스트를 초기화 합니다.

기본값으로 저장

(현재 파일 유형, 현 파일 확장명, 현재 파일 및 사용자 지정)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다.

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다. 리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

폴더 내 드롭다운 리스트 박스¶

검색할 폴더를 지정합니다.

> 버튼¶

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

현재 폴더

이 항목이 선택된 경우, 대화 상자는 현재 폴더로 폴더 내 드롭다운 리스트 박스를 초기화 합니다.

상위 폴더

이 항목이 선택된 경우, 대화 상자는 현재 상위 폴더로 폴더 내 드롭다운 리스트 박스를 초기화 합니다.

최상위 폴더

이 항목이 선택된 경우, 대화 상자는 현재 최상위 폴더로 폴더 내 드롭다운 리스트 박스를 초기화 합니다.

루트 폴더

이 항목이 선택된 경우, 대화 상자는 현재 루트 폴더로 폴더 내 드롭다운 리스트 박스를 초기화 합니다.

사용자 지정

이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열로 폴더 내 드롭다운 리스트 박스를 초기화 합니다.

찾아보기

원하는 폴더를 검색하거나 찾아보도록 합니다.

기본값으로 저장

(현재 폴더, 상위 폴더, 최상위 폴더, 루트 폴더, 사용자 지정 및 찾아보기)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다.

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다. 리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

대/소문자 구분 체크 박스¶

문자열을 검색할 때 대/소문자를 일치시킵니다.

하위 폴더 보기 체크 박스¶

폴더 내 드롭다운 리스트 박스의 하위 폴더를 찾아봅니다.

정규식 사용 체크 박스¶

정규식 표현을 가능하게 합니다. 정규식 표현식 구문에 대한 더 자세한 내용은 정규식 사용하기 를 참조하십시기 바랍니다.

이스케이프 시퀀스 사용 체크 박스¶

이스케이프 시퀀스를 가능하게 합니다. 이스케이프 시퀀스로 다음의 문자들을 사용할 수 있습니다.

\a

경고 (벨)

\b

백 스페이스

\f

용지 공급

\n

새로운 줄

\t

수평 탭

\v

수직 탭

\

백 슬래시

\oooooo

8진수 유니코드 문자

\xhhhh

16진수 유니코드 문자

Null 문자 (\0)는 사용되지 않을 수 있습니다. \r대신 \n이 사용되어야만 합니다.

전체 단어 일치 체크 박스¶

단어만을 검색합니다. 단어는 이와 같은 문자( A – Z, a – z, 0 – 9, 또는 밑줄)로 시작하고 끝나는 문자열로 정의됩니다. 전자 문자로 둘러쌓인 문자열을 단어로 간주합니다. 정규식 표현이 사용되는 경우, 이 체크 박스는 제대로 작동하지 않을 수도 있습니다. 정규식 표현을 사용하는 경우, 대신 단어의 경계식 (\<, \> 및 \b)을 사용하시기 바랍니다.

완료 시 닫기 체크 박스¶

검색이 완료되면 대화 상자를 닫습니다.

수정된 파일 열린 상태 유지 체크 박스¶

이 항목이 선택된 경우, 파일에 대한 변경 사항이 즉시 적용되지 않습니다; 변경 사항이 생긴 모든 파일들을 열려진 채로 남겨질 것입니다. 그러므로 변경된 파일들을 확인 후에 저장할 수 있습니다. 이 옵션이 체크된 상태로 파일 내 바꾸기를 한 후에, 변경 사항이 만족스럽다면 저장 후 모두 닫기 명령 을 선택할 수 있고, 변경 사항을 취소하기 원한다면 저장 없이 모두 닫기 명령 을 선택할 수 있습니다. 이 항목이 선택되어 있지 않은 경우, 최대 대체 가능한 파일의 수는 32로 제한됩니다.

이 항목이 선택되어 있지 않은 경우, 변경 사항을 취소할 수 없습니다. 파일 내 바꾸기를 하기 전에 백업을 만들어 놓을 것을 권장합니다.

정규식 표현을 사용하여 대체할 문자열을 지정할 때 이 체크 박스에 따라 캐리지 리턴(\r)과 라인 피드(\n)를 지정하는 법이 정해지게 됩니다. 선택된 경우, 캐리지 리턴과 라인 피드를 구분하지 않고 \n을 지정합니다. 선택되지 않은 경우, 지정할 때 캐리지 리턴(\r)과 라인 피드(\n)를 반드시 구분합니다.

백업 저장 체크 박스¶

이 항목이 선택된 경우, 백업 폴더 텍스트 박스에 지정된 폴더에 기존 파일의 백업이 생성될 것입니다. 하위 폴더 내에서 찾기 체크 박스가 체크된 경우, 기존 폴더 트리는 백업 폴더에 보존됩니다.

백업 폴더 텍스트 박스¶

백업 저장 체크 박스가 선택된 경우, 원본을 복사할 파일을 지정합니다.

인코딩 드롭다운 리스트 박스¶

인코딩을 선택합니다. 구성된 인코딩 을 선택한 경우, 열린 파일 확장명과 연계된 인코딩이 선택됩니다.

모두 검색 체크 박스¶

사용 가능한 모든 코드 페이지로부터 통계적으로 코드 페이지를 검색합니다. 특히 파일이 매우 작은 경우, 검출 실수를 할수 있습니다. 이 옵션은 HTML/XML 문자 집합 검색을 포함하지 않습니다.

HTML/XML 문자 집합 검색 체크 박스¶

HTML 파일로부터 “charset=…”, XML 파일로부터 “encoding=…”, 또는 Python과 Ruby 파일로부터 “#coding=…”을 찾고, 해당하는 인코딩을 사용합니다. HTML, XML, Python, 또는 Ruby 파일을 연 경우,이 옵션만을 선택하고 다른 옵션들의 선택은 제거합니다.

유니코드 서명 검색 체크 박스¶

파일이 바이트 순서 표시 (BOM) 서명을 포함하는지 여부를 감지하고, BOM 서명을 감지한 경우,

UTF-8 검색 체크 박스¶

통계적으로 UTF-8을 검색합니다.

찾기 버튼¶

지정된 조건으로 검색을 시작합니다.

모두 바꾸기 버튼¶

파일 내 바꾸기를 시작하기 위해 이 버튼을 클릭합니다.

고급 버튼¶

고급 옵션을 설정할 고급 대화 상자 를 나타내기 위해 이 버튼을 클릭합니다.
