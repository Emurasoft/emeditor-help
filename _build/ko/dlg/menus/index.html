사용자 지정 메뉴 대화 상자¶

이 대화 상자는 메뉴 사용자 지정 명령 이 선택되면 나타납니다.

사용자 지정 메뉴 리스트 박스¶

사용자 지정 하려는 메뉴를 지정합니다.

정의된 메뉴 리스트 박스¶

정의된 메뉴의 구성 요소입니다.

지연 시간 슬라이더¶

사용자 메뉴가 선택되었을 때 메뉴를 표시하는 지연 시간을 지정합니다. 메뉴를 표시하기 전에 바로가기 키를 입력할수 있습니다.

위에 삽입 버튼¶

선택된 항목을 위에 삽입합니다.

아래에 삽입 버튼¶

선택된 항목을 아래에 삽입합니다.

오른쪽 삽입 버튼¶

선택된 팝업 메뉴의 오른쪽에 새로운 메뉴를 삽입합니다.

속성 버튼¶

목록에서 선택된 구성 속성을 표시하거나 편집합니다.

삭제 버튼¶

목록에서 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

위로 버튼¶

목록에서 선택된 항목을 위로 이동합니다.

아래로 버튼¶

목록에서 선택된 항목을 아래로 이동합니다.

초기화 버튼¶

기본 설정으로 초기화 합니다.

이 대화 상자를 통해 다음의 대화 상자들을 사용 가능합니다.
