개수 일치（찾기 도구 모음）명령¶

요약¶

찾기 도구 모음에서 개수 일치 버튼을 설정/해제 합니다.

설명¶

찾기 도구 모음에서 개수 일치 버튼을 설정/해제 합니다. 이 명령이 실행될 때, 엠에디터는 지정된 문자열과 일치되는 항목의 수를 표시합니다.

실행하는 방법¶

  • 기본 메뉴: 없음

  • 모든 명령: 검색 > 찾기 도구 모음 > 개수 일치

  • 도구 모음: 없음

  • 상태 표시줄: 없음

  • 기본 바로 가기 키: 없음

플러그인 명령 ID¶

  • EEID_FINDBAR_COUNT (4578)

매크로¶

[JavaScript]¶

editor.ExecuteCommandByID(4578);

[VBScript]¶

editor.ExecuteCommandByID 4578
