비교 범주¶

직접 비교 명령

옵션을 지정하지 않고 최근에 본 두 문서를 직접 비교합니다.

옵션과 비교 명령

비교 마법사를 실행하여 최근에 본 두 문서를 비교합니다.

다시 검사 명령

비교 문서를 다시 검사하여 결과를 새롭게 나타냅니다.

직접 스크롤링 동기화 명령

옵션을 지정하지 않고 최근에 본 두 문서의 스크롤링을 동기화 합니다.

옵션과 스크롤링 동기화 명령

스크롤링 동기화 마법사를 실행하여 최근에 본 두 문서의 스크롤을 동기화합니다.

일반 모드로 초기화 명령

비교 또는 스크롤 동기화 모드를 초기화하고 비교 결과를 지웁니다.

선행 공백 무시 명령

문서를 비교할 때 각 줄의 선행 공백을 무시합니다.

후행 공백 무시 명령

문서를 비교할 때 각 줄의 후행 공백을 무시합니다.

포함 공백 무시 명령

문서를 비교할 때 각 줄의 포함 공백을 무시합니다.

대/소문자 무시 명령

문서를 비교할 때 대/소문자를 무시합니다.

반환 무시 명령

문서를 비교할 때 각 줄의 반환을 무시합니다.

주석 무시 명령

문서를 비교할 때 주석을 무시합니다.

인코딩 무시 명령

문서를 비교할 때 인코딩을 무시합니다.

세로 스크롤링 동기화 명령

세로 스크롤링을 동기화 합니다.

가로 스크롤링 동기화 명령

가로 스크롤링을 동기화 합니다.

커서 위치 동기화 명령

커서 위치를 동기화 합니다.

완료 후 창 위치 복원 명령

비교 또는 동기화가 완료되면 창 위치를 복원시킵니다.

옵션 저장 명령

비교 또는 스크롤을 동기화하는 옵션을 저장합니다.

이전 변경 내용 명령

이전 변경 사항으로 이동합니다.

다음 변경 내용 명령

다음 변경 사항으로 이동합니다.

이전 변경된 줄 명령

다음 변경된 줄 명령

다음으로 변경된 줄로 이동합니다.
