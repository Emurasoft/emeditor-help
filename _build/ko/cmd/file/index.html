파일 범주¶

새 텍스트

새 텍스트 파일을 생성합니다.

새로 만들기(팝업 메뉴)

지정된 구성을 사용하여 새로운 파일을 생성하는 팝업 메뉴를 표시합니다.

새로 만들기 후 붙여넣기

새 파일을 생성하고 클립보드의 내용을 붙여넣습니다.

새로 만들기 후 따옴표 내 붙여넣기

새 파일을 생성하고 클립보드의 내용을 따옴표 내 붙여넣습니다.

새 파일, 따옴표 내 붙여넣기와 반환

새 파일을 생성하고 현재 선택 영역을 따옴표 내 붙여넣고 반환합니다.

트레이 아이콘 보이기

새로운 시스템 트레이 아이콘을 표시합니다.

새 구성 리스트

지정된 구성으로 새 파일을 생성합니다 (여러 항목).

열기

닫은 후 열기
