「多檔取代」對話方塊¶

這個對話方塊會出現，當你選擇 多檔取代 命令 時。你可以指定一個要尋找的字串以及其他多檔尋找的參數。

「尋找」下拉清單方塊¶

輸入要搜尋的字串。如果這是一個多行的文字方塊，你可以通過按 CTRL + ENTER 來輸入新的行。當下拉清單打開時，你可以通過按 ALT + DELETE 來清除被選取的歷史條目。

「>」按鈕¶

點擊該按鈕來顯示可用命令清單。

選取內容或游標位置處的單字

當該選項被勾選時，對話方塊會把選取的內容或游標處的單字作為尋找文字方塊中的初始字串。

選取內容

當該選項被勾選時，對話方塊會把選取的內容作為尋找文字方塊中的初始字串。

游標處的單字

當該選項被勾選時，對話方塊會把選取的內容作為尋找文字方塊中的初始字串。

上次使用的值

當該選項被勾選時，對話方塊會把上次尋找的字串作尋找文字方塊中的初始字串。

固定值

當該選項被勾選時，對話方塊會把設為固定值的字串作為尋找文字方塊中的初始字串。

自動儲存

把選項 (選取內容或游標處的單字，選取內容，游標處的單字，上次使用的值，或固定值) 儲存為預設值；當下一次對話方塊顯示時，就會是預設的設定。

多行

讓文字方塊在單一行與多行之間切換。

使用編輯器字型

當該選項被勾選時， 尋找 下拉清單方塊會使用與編輯器相同的字型。

從批次清單中選擇

從批次清單中選擇要搜尋的字串。

交換尋找與取代字串

交換尋找與取代字串。

該命令還包括可用的逸出數列或規則運算式。另外，從清單中選擇一個條目會把選定的條目插入到按鈕旁的文字方塊中。

「取代為」下拉清單方塊¶

輸入一個你想要用來取代搜尋字串的字串。如果這是一個多行的文字方塊，你能通過輸入 CTRL + ENTER 來換行。當下拉清單開啟時，你可以通過按 ALT + DELETE 鍵來清除選取的歷史條目。

「檔案類型」下拉清單方塊¶

要搜尋的檔案類型。可以使用通配符（\* 以及 ?）。同樣，多個檔案類型可以用分號（;）來區分。

「>」按鈕¶

點擊該按鈕來顯示可用命令清單。

目前的檔案類型

當該選項被勾選時，對話方塊會用目前的檔案類型初始檔案類型下拉清單方塊。

目前的檔案副檔名

當該選項被勾選時，對話方塊會用目前的檔案副檔名初始檔案類型下拉清單方塊。

目前的檔案

當該選項被勾選時，對話方塊會用目前的檔案初始檔案類型下拉清單方塊。

上次使用的值

當該選項被勾選時，對話方塊會把上次用過的字串作尋找文字方塊中的初始字串。

固定值

當該選項被勾選時，對話方塊會把設為固定值的字串作為尋找文字方塊中的初始字串。

自動儲存

把選項 (目前的檔案類型，目前的檔案副檔名，目前的檔案，或上次使用的值) 儲存為預設值；當下一次對話方塊顯示時，就會是預設的設定。

該命令還包括可用的逸出數列或規則運算式清單。從清單中選擇一個條目會把選定的條目插入到按鈕旁的文字方塊中。

「在資料夾中」下拉清單方塊¶

指定要搜尋的資料夾。

「>」按鈕¶

點擊該按鈕來顯示可用命令清單。

目前的資料夾

當該選項被勾選時，對話方塊會把目前的資料夾作為「搜尋資料夾」的初始資料夾。

父資料夾

當該選項被勾選時，對話方塊會把父資料夾作為「搜尋資料夾」的初始資料夾。

祖父資料夾

當該選項被勾選時，對話方塊會把目前的資料夾的祖父資料夾作為「搜尋資料夾」的初始資料夾。

根資料夾

當該選項被勾選時，對話方塊會把目前的資料夾的根資料夾作為「搜尋資料夾」的初始資料夾。

上次使用的值

當該選項被勾選時，對話方塊會把上次用過的資料夾作為「搜尋資料夾」的初始資料夾。

固定值

當該選項被勾選時，對話方塊會把把設為固定值的資料夾作為「搜尋資料夾」的初始資料夾。

瀏覽

讓你能瀏覽或搜尋所需的資料夾。

瀏覽並添加

讓你能瀏覽或搜尋所需的資料夾，並把該資料夾添加到已存在的指定資料夾中。

自動儲存

把（目前的資料夾，父資料夾，祖父資料夾，根資料夾，上次使用的值或瀏覽）儲存為預設值；當下一次對話方塊顯示時，就會是預設的設定。

該命令還包括可用的逸出數列或規則運算式清單。從清單中選擇一個條目會把選定的條目插入到按鈕旁的文字方塊中。

「大小寫須相符」核取方塊¶

當搜尋一個字串時，區分大小寫。

「尋找子資料夾」核取方塊¶

檢視 所在資料夾 下拉清單方塊中資料夾的子資料夾。

「全字拼字須相符」核取方塊¶

僅搜尋符合的單字。一個單字被定義為一個以這些字元開頭並結尾的字串: A – Z，a – z，0 – 9，或一條底線。由全形字元包圍的字串被認作是一個單字。當使用規則運算式時，該核取方塊可能不能正常運作。所以,當使用規則運算式時，請用運算式來標明單字界限（\<，\>，以及 \b）。

「搜尋完成後關閉該對話方塊」核取方塊¶

當完成搜尋或取代後，關閉該對話方塊。

「保持修改的檔案為打開狀態」核取方塊¶

如果勾選了改核取方塊，檔案不會馬上變更；所有發生變更的檔案都會被打開。這樣，你能在確認這些變更之後儲存檔案。當你用這個選項來執行多檔取代命令時，你能選擇 儲存並全部關閉 命令 如果你滿意所有的變更，或選擇 全部關閉不儲存 命令 如果你想要復原變更。如果沒有勾選該核取方塊，最多能被取代的檔案數目是 32。

如果該核取方塊沒有被勾選，你將不能復原所做的變更。因此，建議你在執行多檔取代命令前先創建原始檔案的備份。

如何指定歸位 (\r) 與換行 (\n) 取決於是否勾選這個核取方塊當你用規則運算式來指定要取代的字串時。如果勾選了該核取方塊，搜尋時，\n 不區分歸位與換行。如果沒有勾選，當指定搜尋字串時一定要區分歸位 (\r) 和換行 (\n)。

「儲存備份」核取方塊¶

如果該核取方塊被勾選了，EmEditor 將會在 備份資料夾 文字方塊中指定的資料夾中創建原始檔案的備份。如果 在子資料夾中尋找 核取方塊被勾選了，原始的資料夾樹狀目錄將會在備份資料夾中保留。

「（無）」選項按鈕¶

指定字串應按字面意思符合。

「規則運算式」選項按鈕¶

啟用 規則運算式。

「逸出數列」選項按鈕¶

啟用逸出數列。你能把下列字元作為逸出數列。

\a

警告（響鈴）

\b

退格

\f

換頁

\n

換行

\t

水平 Tab

\v

垂直 Tab

\

反斜杠

\oooooo

Unicode 字元在八進位運算式中

\xhhhh

Unicode 字元在十六進位運算式中

空字元（\0）可能不能使用。一定要用 \n 而不是 \r。

「數字範圍」選項按鈕¶

啟用 數字範圍運算式。

「備份資料夾」文字方塊¶

如果 儲存備份 核取方塊被勾選了，會在備份資料夾中複製原始檔案的備份。

「編碼」下拉清單方塊¶

選擇編碼。如果你選擇 已組態的編碼，與打開的檔案副檔名相關聯的編碼會被選取。

「全部偵測」核取方塊¶

從所有可用的程式碼頁中統計偵測編碼頁。這個偵測可能會出現失誤，特別當檔案很小時。這個參數並不包括 HTML/XML 字元集偵測。

「偵測 HTML/XML 字元集」核取方塊¶

在 HTML 檔案中尋找」charset=…“，在 XML 檔案中尋找」encoding=…”，或在 Phyton 以及 Ruby 檔案中尋找」#coding=…」，然後使用向對應的編碼。如果你打開 HTML，XML，Python，或 Ruby 檔案，勾選這個參數並清除其他參數。

「偵測 UTF-8」核取方塊¶

統計偵測 UTF-8。

「儲存設定」核取方塊¶

將編碼相關的選項（ 編碼 下拉清單方塊，以及 全部偵測， 偵測 HTML/XML 字元集，和 偵測 UTF-8 核取方塊）儲存為預設值，以便下次打開此對話方塊時還原這些選項。

「停止如果符合數達到」核取方塊/文字方塊¶

如果在設定此核取方塊後符合數達到指定值，EmEditor 將停止搜索檔案。

「尋找」按鈕¶

用指定的條件開始搜尋。

「全部取代」按鈕¶

點擊該按鈕開始多檔取代。

「<< 尋找」按鈕¶

點擊該按鈕儲存目前的要尋找的字串以及參數，關閉目前的對話方塊，並顯示 多檔尋找 對話方塊。

「關閉」按鈕¶

點擊該按鈕來關閉該對話方塊。

「進階」按鈕¶

點擊該按鈕顯示 進階 對話方塊 來設定進階的選項。

「加入批次」按鈕¶

添加目前的設定到批次清單中。

「批次處理 >>」按鈕¶

在 多檔取代 對話方塊與 批次多檔取代 對話方塊之間切換。

下列控件僅在該對話方塊延伸為 批次多檔取代 對話方塊時顯示。

「儲存至批次」按鈕¶

儲存目前的設定到批次清單中。

清單方塊¶

顯示用來批次的尋找和取代組合清單。下列字母縮寫用在清單方塊的條件列中，代表指定的尋找條件。

C

大小寫需符合

R

使用規則運算式

W

只搜索單字

E

僅符合 CSV 中的嵌入式換行

S

區別 CR 和 LF

D

規則運算式 “.” 符合換行符

B

把 Boost.Regex 作為規則運算式引擎

O

把 Onigmo 作為規則運算式引擎

「全部啟用/停用」核取方塊¶

啟用或停用清單上所有的條目。

「匯入」按鈕¶

匯入之前匯出的包含取代條件的 TSV 檔案。

「匯出」按鈕¶

把已定義的取代條件匯出到一個檔案中。檔案格式可以是 TSV，JavaScript 或 VBScript 巨集格式。

「多項尋找」按鈕¶

單擊此按鈕可同時搜尋所選項目。

「批次尋找」按鈕¶

單擊此按鈕在多個文檔中尋找與批次清單符合的所選條目。單擊右邊的「▼」按鈕會顯示一個內容功能表，讓你能訪問 批次選項 對話方塊。

「多項取代全部」按鈕¶

使用批次清單取代指定文檔中的所有字元串。詳細信息請參閱 批次和多項之間的區別。

「批次取代全部」按鈕¶

取代多個文檔中的與批次清單所選條目符合的所有字串。

「<< 批次處理」按鈕¶

在 批次多檔取代 對話方塊與 多檔取代 對話方塊之間切換。

這個對話方塊能通過拖動對話方塊的右下角來調整對話方塊大小。當啟用多行的文字方塊時，按 CTRL + ENTER 能插入新行。
