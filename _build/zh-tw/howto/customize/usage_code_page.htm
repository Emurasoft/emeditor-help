定義編碼¶

  1. 選擇 工具 功能表下的 定義編碼 命令。

  2. 點擊 「新增」 按鈕如果您想要定義一個新編碼。

  3. 選擇 編碼 以及 字型類別 并未該編碼輸入 名稱。

提示¶

要用一個被定義的編碼來打開一個檔案，在 打開 對話方塊中選擇指定編碼。
