歷史記錄 < Version 7.00

Version 7.00¶

2007年12月18日

  • 新的多線程設計讓您能檢視或編輯文檔當打開一個大檔案時。

  • 「從原始檔案中讀取未修改的行」復選框，「鎖定原始檔案」復選框以及「使用異步打開的最小檔案大小」文本框被添加到自訂對話方塊中的 進階選項卡 上，更優化打開大檔案的速度。

  • 自訂分欄可以通過外掛程式顯示。

  • EE_TOOLBAR_OPEN， EE_TOOLBAR_CLOSE， EE_TOOLBAR_SHOW, EE_HELP， EE_REG_SET_VALUE， EE_REG_QUERY_VALUE， EE_QUERY_STRING， EE_KEYBOARD_PROP， EE_GET_ACCEL_ARRAY， EE_OUTPUT_DIR， EE_OUTPUT_STRING， EE_ENUM_HIGHLIGHT 消息被添加到外掛程式界面。

  • Editor_ToolbarOpen， Editor_ToolbarClose， Editor_ToolbarShow, Editor_Help， Editor_RegSetValue， Editor_RegQueryValue， Editor_QueryString， Editor_KeyboardProp， Editor_GetAccelArray Editor_DocGetLines， Editor_OutputDir， Editor_OutputString， Editor_EnumHighlight 內嵌函式被添加到外掛程式界面。

  • TOOLBAR_INFO， REG_SET_VALUE_INFO， REG_QUERY_VALUE_INFO 結構被添加到外掛程式界面。

  • EVENT_TOOLBAR_CLOSED，EVENT_TOOLBAR_SHOW 事件 被添加到外掛程式界面。

  • EmEditor 現在讓您能編輯比之前版本更大的檔案 (取決于可用的內存)。

  • 巨集工具列被添加讓您能更方便地訪問「我的巨集」。

  • 您現在能變更一個巨集的圖示，標題，以及工具提示 ( #icon 指令， #title 指令， #tooltip 指令)。

  • 顯示巨集工具列 復選框被添加到 工具列 對話方塊 中。

  • 查找 對話方塊 和 替換 對話方塊 現在可以調整大小。

  • EmEditor 現在不顯示對話方塊就能自動把修改過的檔案保存為前綴為 “AutoSave-” 的檔案名，當 EmEditor 出現故障或當 Windows 在自動更新下被強制重新啟動時。同樣，EmEditor 現在會出現一個提示消息，顯示自動恢復的檔案當重啟時。

  • 當打開一個檔案時，在狀態列上顯示的檔案大小現在根據需要以 KB，MB，或 GB 顯示。

  • 用鼠標點擊三次能選擇一整行。

  • 添加了 QueryStringByID 方式。

  • EmEditor 能錄制并播放在其他應用程序中的鍵盤以及鼠標操作。

  • 添加了 Shell 對象 以及 Windows 集合。

  • Children， ClassName， Enabled， Height, hWnd， LeftX， OutputBar, Parent， ProcessID， ThreadID, Top， Valid， Visible，和 Width 屬性還有 FindWindow， FindWindowByID， SetFocus， SetForeground，和 SetPosition 方式被添加到了 Window 對象 中。

  • Config 屬性 被添加到 Document 對象 中。

  • 添加了配置屬性的對象 ( AssociationItem 對象， AssociationList 集合， AssociationProp 對象， AutoSaveProp 對象， BackupProp 對象， Config 對象， Configs 集合, DisplayItem 對象， DisplayList 集合， DisplayProp 對象， FileNewProp 對象， FileSaveProp 對象， FileProp 對象， FontItem 對象， FontList 集合， FontProp 對象， GeneralProp 對象， HighlightItem 對象， HighlightList 集合， HighlightProp 對象， IndentProp 對象， KeyboardItem 對象， KeyboardList 集合， KeyboardProp 對象， LinkProp 對象， MarkProp 對象， NoWrapItem 對象， NoWrapList 集合， NoWrapProp 對象， PrintProp 對象， ScrollProp 對象，和 WrapProp 對象)。

  • The GetLine 方式， GetLines 方式 被添加到 Document 對象 中。

  • The Configs 方式 被添加到 Editor 對象 中。

  • 添加了 OutputBar 對象。

  • 支持虛擬空間模式 (在配置屬性中的 一般 選項卡 上添加了 啟用虛擬空間 復選框)。

  • 「打開」對話方塊現在能允許同時打開多個檔案。

  • 便攜性選項 (為 USB 驅動器安裝準備)

  • 記錄鼠標活動及鍵盤輸入到其他應用程序中 復選框， 錄制所有鼠標動作 復選框，以及 在插入休眠模式前的空閑時間 文本框被添加到了 自訂巨集 對話方塊 中的 選項 選項卡 上。

  • 添加了新的外掛程式: 專案外掛程式，查找欄外掛程式 HTML 工具列外掛程式，以及 字詞自動完成外掛程式。

  • 巨集工具列， 刷新工具列， 錄制所有動作, 錄制除鼠標/鍵盤外的所有動作， 左自訂分欄， 頂部自訂分欄， 右自訂分欄， 底部自訂分欄 命令被推薦了。

  • 替換 命令， 大寫 命令， 小寫 命令， Capitalize 命令, 半角 命令，以及 全角 命令 現在能轉換一個垂直選取的區域。

  • 當打印時，一個有 換頁 碼 (U+000C) 的文檔現在可以在代碼處插入一個新的分頁。

  • 文本以及背景顏色添加了 透明 選項。

  • 制表符/縮進 對話方塊 中添加了 換行縮進 復選框。

  • 自訂對話方塊 中 窗口 選項卡 添加了 自動保存工作區 復選框以及 自動還原工作區 復選框。

  • 全部高亮 復選框被添加到 配置屬性 中的 高亮 (1) 選項卡 上。

  • 配置屬性 中的 滾動 選項卡 添加了 平滑滾動 復選框以及 平滑滾動 滑塊。

  • 您現在能把一個庫檔案包括進一個巨集中 ( #include 指令)。

  • 添加了 新命令行選項 ( /ne)。

  • 添加了 列 文本框到 跳轉 對話方塊 中。

  • 當多個檔案被修改并且 全部關閉 命令被選取， 保存更改到… 消息框現在包括 全部應用 復選框。

EI_GET_CURRENT_FOLDER and EI_IS_LARGE_DOC 命令被添加到 EE_INFO 消息 ( Editor_Info， Editor_DocInfo 內嵌函式)中。

  • 輸出編碼 下拉列表框被添加到 外部工具屬性 對話方塊 中。

  • 「關閉」按鈕可以在激活的選項卡或每個選項卡上顯示 (在「自訂」對話方塊框中「標簽」選項卡上的 關閉按鈕 下拉列表框中設定)。

  • 只讀圖示能被顯示在標簽上。( 顯示只讀圖示 復選框)。

  • 在「查找」下拉列表框中用 ALT + DELETE 從搜尋記錄中清除一個項目。

  • 用戶定義的菜單允許兩次按鍵快捷方式 ( 延時 滑塊)。

  • 主題 下拉列表框 drop-down list box，the 「>」 按鈕， 字符間距 文本框被添加到配置屬性中的 顯示選項卡 上。

  • 您可以定義事件驅動的巨集。( 「事件」 按鈕， 在發生事件時運行 復選框， 事件 文本框， 選擇事件 對話方塊)。

  • 在 Windows Vista 系統上， 打開/ 另存為 對話方塊現在與Windows Vista設計一致，讓您能輕松訪問檔案的歷史版本，這也是 Windows Vista 的功能之一。

  • 安裝程序現在讓您能更改安裝程序資料夾如果前一個版本已經被安裝的話。
