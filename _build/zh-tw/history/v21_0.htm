歷史記錄 < Version 21.0

Version 21.0¶

2021年8月10日

一般新增功能¶

  • 新增了 版面配置 命令（多項），可以選擇顯示或隱藏哪些 UI 元素，是否切換到全屏模式，以及下次啟動 EmEditor 時是否保留版面配置。預設情況下，定義了 全螢幕 (F11) 和 焦點模式 (Alt+F11) 螢幕 版面配置。

  • 添加了 /layout 切換到 命令列選項。

  • 當檔案中存在很長的行時，提高了選取，捲動和規則運算式搜索的速度。

  • 新版本在儲存/編輯錄制的巨集時，將命令名稱作為註釋添加到每個 editor.ExecuteCommandByID 方法中。

新增選項¶

  • 添加了 版面配置 頁面 到 自訂 對話方塊中。

  • 添加了 插入為 HTML 命令到 剪貼簿歷史記錄 功能表中。

  • 如果 HTML 格式在 剪貼簿 中可用，新版本將 貼上為 HTML 按鈕添加到貼上字串時顯示的 顯示剪貼簿歷史記錄 按鈕中。

  • 添加了 巨集 文字方塊， … 按鈕， 設定重複次數 核取方塊， 對每個打開的文檔運行巨集 核取方塊， 名稱 清單方塊， 添加 按鈕， 刪除 按鈕， > 按鈕， 運行巨集後儲存並關閉每個文檔 核取方塊，以及 儲存選項 核取方塊到 巨集臨時選項 對話方塊 中。

外掛程式 API 的新增功能¶

新增巨集功能¶

  • 添加了 DeleteColumn 方法 到 Document 對象中。

  • 添加了 SelectColumn 方法 到 Selection 對象中。
