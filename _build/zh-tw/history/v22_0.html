歷史記錄 < Version 22.0

Version 22.0¶

2022年10月5日

一般新增功能¶

  • 添加了 模糊比對 功能，即使用符合近似字串來搜索、篩選和合併 CSV 的能力。 該功能在 EmEditor Professional 中包含多個選項，而在 EmEditor Free 中只有 符合相似字串 選項可用。

  • 添加了 模糊比對 按鈕到 尋找 和 篩選 工具列上。

  • 能夠亮顯顯示用於電子郵件報文頭部的 多用途互聯網郵件延伸（MIME）編碼的字詞（Base64），並在將滑鼠停留在它們上方時顯示帶有解碼字串的工具提示。添加了使用現有的 將 Base 64 轉換為純文字（目前的編碼或 UTF-8）命令對其進行解碼的能力。

  • 添加了 作為字元插入， 以列插入， 垂直插入 以及 作為儲存格插入 命令到右鍵單擊 剪貼簿記錄 所顯示的功能表上。

  • 預設的 主 功能表經過重新設計，包括了 插入、 轉換、 書籤、 排序，以及 外掛程式 功能表項。這些變更使您無需多次點擊功能表即可訪問常用命令。 插入 功能表中的 變音符號/ 標記 命令讓您能無需記住相應的快速鍵就可以鍵入這些特殊字元。

  • 支持 Unicode 15.0 的 Unicode 名稱、 東亞寬度、 Unicode 腳本 和一 般類別屬性。

新增選項¶

  • 添加了 模糊比對 核取方塊到 尋找， 取代， 多檔尋找 以及 多檔取代 對話方塊中。

  • 添加了 …（ 模糊比對選項）到 尋找， 取代， 多檔尋找 以及 多檔取代 對話方塊中。

  • 添加了 模糊比對 核取方塊和 …（ 模糊比對選項）到 批次尋找， 批次取代， 批次多檔尋找， 批次多檔取代，以及 進階篩選 對話方塊。

  • 添加了 模糊比對 核取方塊和 …（ 模糊比對選項）按鈕到 合併 CSV 對話方塊中。

  • 添加了 模糊比對選項 對話方塊 和 字串/字元範圍 對話方塊。

  • 添加了 列標題 文字方塊到 自訂 對話方塊中的 CSV 格式 頁面上。

新增命令¶

  • 模糊比對 (篩選工具列)

  • 模糊比對 (尋找工具列)

外掛程式的新增功能 ¶

  • 通過添加比較分支的功能改進了 CommitList 外掛程式。

  • 添加了 FLAG_FIND_FUZZY 標志到 FILTER_INFO_EX， FIND_REGEX_INFO_EX， FIND_REPLACE_INFO， GREP_INFO_EX，和 MATCH_REGEX_INFO_EX 結構中。

  • 添加了 FLAG_FIND_FUZZY 標志到 Editor_Filter， Editor_FindReplace，和 Editor_BatchFindReplace 內嵌函式中。

  • 添加了 JOIN_FLAG_FUZZY 標志到 JOIN_INFO 結構和 Editor_Join 內嵌函式中。

新增巨集功能 ¶

  • 添加了 eeExFindFuzzy 標志到 Selection 對象的 Find 和 Replace 方法中， Editor 對象的 FindInFiles 和 ReplaceInFiles 中， Document 對象的 Filter 方法中，以及 Filters 集合的 Add， AddFind，和 AddReplace 方法中。

  • 添加了 eeJoinFuzzy 標志到 Editor 對象的 Join 方法 中。

  • 添加了 Fuzzy 屬性 到 Regex 對象中。

  • 添加了 FuzzyOptions object， CharacterRange object，和 CharacterRanges 集合。

  • 添加了 FuzzyOptions 屬性 到 Editor 對象中。

  • 添加了 RowHeading 屬性到 Csv 對象中。

  • 添加了將 合併 CSV 命令記錄到巨集的功能。
