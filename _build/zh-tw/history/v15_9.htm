歷史記錄 < Version 15.9

Version 15.9¶

2016年4月19日

一般新增功能¶

  • 添加了新的 CSV 轉換器。

  • CSV 轉換命令優化。

  • 添加另一個分隔符到 CSV 文檔的右邊。

  • 剪下欄， 復制欄， 貼上， 刪除欄， 在左邊插入一欄，以及 在右邊插入一欄 命令被添加到了右擊尺規顯示的快顯功能表上。

  • 添加了 在上方插入一行 和 在下方插入一行 命令到右擊編輯器左邊緣會顯示的功能表中。

  • 廢棄了復原緩衝區的設定。改用實體記憶體大小來決定復原的信息是否應該保留。實體記憶體閾值可以在 自訂 對話方塊的 進階 頁面上設置。

  • 新版本讓你能取消目前的命令當復原數據不可用時。

新增選項¶

  • 添加了 預設欄寬 文字方塊到 自訂 對話方塊中的 CSV 頁面上。

  • 添加了 總是丟棄過長的復原信息來加速 核取方塊到 自訂 對話方塊中的 編輯 頁面上。

  • 添加了 最大記憶體大小 文字方塊到 自訂 對話方塊中的 排序 頁面上。

新增命令¶

  • CSV 轉換器

  • 刪除欄

  • 剪下欄

  • 復制欄

  • 在右邊插入一欄

外掛程式新增功能¶

  • 復制 以及 全選 命令被添加到 字數統計 外掛程式中的內容功能表上。

  • 能夠用路徑調用程式碼片段。

外掛程式 API 新增功能¶

  • 延伸了 FIND_REGEX_INFO_EX 結構。添加了 BACK_REF 結構。

巨集的新增功能¶

  • 添加了 Regex 對象， Matches 集合，and Match 對象。

  • 添加了 regex 屬性 到 Editor 對象中。

  • 在 JSON 對象中支持 JScript Version 5.8。

修復 Bug¶

  • 修復了一個與 CSV 儲存格模式相關的 bug。

  • 優化了用 Onigmo 進行的 Lookbehind 規則運算式搜索。

  • 修復了在選取多個區域時與全部取代相關的一個 bug。

  • 修復了一個與巨集的 Replace 方法相關的 bug。
