# Version 15.2

2015年7月30日

## 一般新增功能

- 大大提高了在 CSV 文檔中對已選擇的欄進行 **「全部取代」** 命令，刪除垂直選取以及在 **全部取代** 后 **復原** 的執行速度。
- 您能選取 CSV 文檔中的欄然后拖放這些欄來重新排序或複製 CSV 欄。
- EmEditor 能偵測到 CSV 文檔中的無效雙引號。
- 說明文檔可以在網頁上瀏覽。您還能下載檢視本機說明檔案 (說明功能表下的「首選項」子功能表) 。說明檔案不再包括在安裝程式內。它需要分開安裝。
- **Consolas** (11 point) 成為西歐字型類型的預設字型，以前的預設字型是 **Courier New** (10 point)。
- 在自訂對話方塊中的標籤頁面上， **在新視窗中打開新的檔案，在新標籤頁中打開已存在的檔案** 設定變為預設。
- 以下這些對話方塊現在可以調整大小： **外部工具**， **工具列**， **多檔尋找**， **多檔取代**， **擷取欄**， **合併 CSV**， **刪除/把重複行設為標籤**， **插入特殊字元**， **自訂功能表** 等等。
- EmEditor v15.2 能與 **Windows 10** 更好地兼容。當您用 **任務檢視** 功能時， **重疊顯示**， **水平並排顯示**， **垂直並排顯示**， **比較** 以及 **同步** 指令現在可以在虛擬桌面上正常運行。例如，如果您使用的是之前版本的 EmEditor，當您執行 **垂直並排顯示** 指令時，就會出現兩個 EmEditor 群組視窗顯示在不同的虛擬桌面上。一個虛擬桌面上顯示左半邊的群組視窗，另一個虛擬桌面上顯示右半邊的群組視窗。EmEditor v15.2 則可以在目前的虛擬桌面上並排顯示視窗。
- 在安裝 EmEditor 時，如果您選擇的是按每個使用者安裝，那么 **記錄游標活動及鍵盤輸入到其他應用程式中** 的核取方塊會自動變為無效。如果您選擇的是按每臺電腦安裝，則會出現以下這個訊息：「 _這個功能(錄制游標活動和鍵盤輸入到其他應用程式中)可能無法在一些特定的 Windows 版本和安全設定下正常運行。在以后的版本中，EmEditor 將不支持該功能。_」。
- 在 **Windows 10** 系統下，當您啟用 **用 EmEditor 在 Internet Explorer 檢視原始檔** 設定時，會顯示以下訊息，「 _在 Internet Explorer 中用 EmEditor 檢視原始檔的功能無法與 Microsoft Edge 很好地兼容。如果您選擇在 Microsoft Edge 上繼續用 EmEditor 檢視原始檔，可能不會顯示任何視窗。請選擇「取消」來關閉該功能。_」。

## 新增選項

- **把第一行作為頁首重複** 核取方塊被添加到 [**把目前文件或指定檔案分割成多個檔案** 精靈](../dlg/split_to_files/index) 中的 [**頁首和頁尾** 頁面](../dlg/split_to_files/split_header) 上。
- **自動調節分隔符位置** 核取方塊被添加到 **自訂** 對話方塊中的 [**CSV** 頁面](../dlg/customize/csv/index) 上。
