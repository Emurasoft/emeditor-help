# 搜尋類別

|     |     |
| --- | --- |
| **[尋找](edit_find)** | 尋找一個字串。 |
| **[取代](edit_replace)** | 取代一個字串。 |
| **[尋找下一個](edit_repeat)** | 尋找下一個符合。 |
| **[尋找上一個](edit_repeat_back)** | 尋找上一個符合。 |
| **[尋找下一單字](find_next_word)** | 尋找目前的單字的下一個符合結果。 |
| **[尋找前一單字](find_prev_word)** | 尋找目前的單字的前一個符合結果。 |
| **[添加下一個符合結果](add_next_occurrence)** | 添加選取指定文字的下一個符合結果。 |
| **[添加下下個符合結果](add_next2_occurrence)** | 添加選取指定文字的下下個符合結果。 |
| **[選擇所有符合結果](select_all_occurrences)** | 選擇指定文字的所有符合結果。 |
| [**取代下一個**](replace_next) | 取代下一個符合併再次尋找符合。 |
| **[設置單字為尋找字串](retrieve_find_text)** | 設置目前的游標所在的單字為查詢字串。 |
| [**設置單字為取代字串**](retrieve_replace_text) | 設置目前的游標所在的單字為取代字串。 |
| **[取消尋找亮顯](erase_find_hilite)** | 清除搜尋字串的亮顯。 |
| **[尋找下一個警告/Unicode 字元](find_next_unicode)** | 尋找下一個無法轉換為要儲存的編碼的警告或 Unicode 字元。 |
| **[尋找上一個警告/Unicode 字元](find_prev_unicode)** | 尋找上一個無法轉換為要儲存的編碼的警告或 Unicode 字元。 |
| **[取消 Unicode 高亮顯示](erase_unicode_hilite)** | 取消亮顯顯示無法轉換為要儲存的編碼的 Unicode 字元。 |
| **[多檔尋找](grep)** | 在多個檔案中搜尋符合的字串。 |
| **[多檔取代](replace_in_files)** | 取代多個檔案中的文字。 |
| **[抽出常用字串](extract_frequent)** | 將常用字串抽出到新文檔中。 |
| **[尋找最長行/儲存格](find_longest)** | 尋找非 CSV 文檔中最長的行或 CSV 文檔的目前的欄中最長的儲存格。 |
| **[尋找非空最短行/儲存格](find_shortest)** | 尋找非 CSV 文檔中的非空最短行或 CSV 文檔的目前的欄中的非空最短儲存格。 |
| **[尋找最短行/儲存格](find_empty_or_shortest)** | 尋找非 CSV 文檔中的最短行或 CSV 文檔目前的欄中的最短儲存格。 |
| **[將焦點設在篩選工具列](focus_filter_bar)** | 把焦點設到篩選工具列上。 |
| **[關閉篩選工具列](close_filter_bar)** | 關閉篩選工具列。 |
| **[瀏覽篩選表達式 (篩選工具列)](filterbar_browse_exp_f)** | 為在篩選工具列中使用的篩選字串瀏覽規則運算式或逸出序列。 |
| **[重新整理 (篩選工具列)](filterbar_refresh)** | 用目前的篩選設定重新整理文檔。 |
| **[中止 (篩選工具列)](filterbar_abort)** | 中止篩選并清空篩選字串。 |
| **[漸進式搜尋 (篩選工具列)](filterbar_incremental)** | 切換「篩選」工具列上的「漸進式搜尋」按鈕。 |
| **[使用規則運算式 (篩選工具列)](filterbar_reg_exp)** | 切換「篩選」工具列上的「使用規則運算式」按鈕。 |
| **[使用逸出序列 (篩選工具列)](filterbar_escape)** | 切換「篩選」工具列上的「使用逸出序列」按鈕。 |
| **[使用數字範圍 (篩選工具列)](filterbar_number_range)** | 切換「篩選」工具列上的「使用數字範圍」按鈕。 |
| **[模糊比對 (篩選工具列)](filterbar_fuzzy)** | 切換「篩選」工具列上的「模糊比對」按鈕。 |
| **[大小寫需符合 (篩選工具列)](filterbar_case)** | 切換「篩選」工具列上的「大小寫需符合」按鈕。 |
| **[整個字需符合 (篩選工具列)](filterbar_only_word)** | 切換「篩選」工具列上的「整個字需符合」按鈕。 |
| **[整個字串需符合 (篩選工具列)](filterbar_whole_string)** | 切換「篩選」工具列上的「整個字串需符合 」按鈕。 |
| **[負向 (篩選工具列)](filterbar_negative)** | 切換「篩選」工具列上的「負向」按鈕。 |
| **[僅書籤行 (篩選工具列)](filterbar_bookmarked_only)** | 切換「篩選」工具列上的「僅書籤行」按鈕。 |
| **[僅未標示書籤的行 (篩選工具列)](filterbar_unbookmarked_only)** | 切換「篩選」工具列上的「僅未標示書籤的行」按鈕。 |
| **[符合換行符號 (篩選工具列)](filterbar_match_nl)** | 切換「篩選」工具列上的「符合換行符號」按鈕。 |
| **[僅 CR (篩選工具列)](filterbar_cr_only)** | 切換「篩選」工具列上的「僅 CR」命令。 |
| **[僅 LF (篩選工具列)](filterbar_lf_only)** | 切換「篩選」工具列上的「僅 LF」命令。 |
| **[CR+LF (篩選工具列)](filterbar_cr_lf)** | 切換「篩選」工具列上的「CR+LF」命令。 |
| **[其他 (篩選工具列)](filterbar_nl_others)** | 切換「篩選」工具列上的「其他」命令。 |
| **[進階篩選 (篩選工具列)](filterbar_advanced)** | 顯示「進階篩選」對話方塊。 |
| **[阻止對多行進行修改 (篩選工具列)](filterbar_limit)** | 切換「篩選」工具列上的「阻止對多行進行修改」按鈕。 |
| **[全部設為書籤 (篩選工具列)](filterbar_bookmark)** | 為所有篩選出來的行設置書籤。 |
| **[抽出所有行 (篩選工具列)](filterbar_extract)** | 將所有篩選出來的行抽出到一個新文檔中。 |
| **[抽出符合字串 (篩選工具列)](filterbar_extract_matches)** | 將符合的字串抽出到新文檔中。 |
| **[抽出選項 (篩選工具列)](filterbar_extract_options)** | 指定抽出符合字串命令的選項。 |
| **[將焦點移至搜尋工具列](focus_find_bar)** | 將焦點移至搜尋工具列。 |
| **[關閉搜尋工具列](close_find_bar)** | 關閉搜尋工具列。 |
| **[瀏覽尋找表達式 (搜尋工具列)](findbar_browse_exp_f)** | 瀏覽用於搜尋工具列中「尋找」字串的規則運算式或逸出序列。 |
| **[瀏覽取代表達式 (搜尋工具列)](findbar_browse_exp_r)** | 瀏覽用於搜尋工具列中「取代」字串的規則運算式或逸出序列。 |
| **[尋找上一個 (搜尋工具列)](findbar_find_prev)** | 用搜尋工具列尋找上一個符合。 |
| **[尋找下一個 (搜尋工具列)](findbar_find_next)** | 用搜尋工具列尋找下一個符合。 |
| **[漸進式搜尋 (搜尋工具列)](findbar_incremental)** | 切換搜尋工具列上「漸進式搜尋」按鈕的狀態。 |
| **[搜尋所有打開的文檔 (搜尋工具列)](findbar_open_doc)** | 切換搜尋工具列上「搜尋所有打開的文檔」按鈕的狀態。 |
| **[使用規則運算式 (搜尋工具列)](findbar_reg_exp)** | 切換工具列上使用「規則運算式」按鈕的狀態。 |
| **[使用逸出序列 (搜尋工具列)](findbar_escape)** | 切換搜尋工具列上「使用逸出序列」按鈕的狀態。 |
| **[使用數字範圍 (搜尋工具列)](findbar_number_range)** | 切換搜尋工具列上「使用數字範圍」按鈕的狀態。 |
| **[模糊比對 (搜尋工具列)](findbar_fuzzy)** | 切換搜尋工具列上「模糊比對」按鈕的狀態。 |
| **[區分大小寫 (搜尋工具列)](findbar_case)** | 切換搜尋工具列上「區分大小寫」按鈕的狀態。 |
| **[符合整個單字 (搜尋工具列)](findbar_only_word)** | 切換搜尋工具列上「符合整個單字」按鈕的狀態。 |
| **[進階 (搜尋工具列)](findbar_advanced)** | 顯示尋找工具列的進階對話方塊。 |
| **[循環 (搜尋工具列)](findbar_around)** | 切換工具列上「循環」按鈕的狀態。 |
| **[符合次數 (搜尋工具列)](findbar_count)** | 切換工具列上「符合次數」按鈕狀態。 |

| **[<< 尋找(尋找工具列)](findbar_find_dlg)** | 切換尋找工具列上的「<< 尋找」按鈕。 |
| **[取代下一個 (搜尋工具列)](findbar_replace_next)** | 用搜尋工具列取代下一個符合的字串。 |
