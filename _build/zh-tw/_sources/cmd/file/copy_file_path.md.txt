# 複製檔案完整路徑命令

### 摘要

> 複製目前的檔案的完整路徑名到剪貼簿。

### 說明

> 複製目前的檔案的完整路徑名，并把它置放到剪貼簿上。如果文檔沒有標題，這個命令則不能被選取。

### 運行方法

- 預設功能表: **檔案** \> **複製檔案完整路徑**
- [全部命令](../tools/all_commands): **檔案** \> **複製**
\> **檔案完整路徑**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_COPY\_FILE\_PATH (4326)

### 巨集

#### \[JavaScript\]

> document.CopyFullName();

#### \[VBScript\]

> document.CopyFullName
