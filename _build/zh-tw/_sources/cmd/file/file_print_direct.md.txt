# 立即列印命令

### 摘要

> 使用預設的印表機而不指定任何選項列印目前的檔案。

### 說明

> 這個命令會立即執行列印文檔文檔命令，而不再顯示 **列印** 對話方塊。印表機會使用預設設置的。這個預設設置可以在控制面板中的 **設備和印表機 (Windows 8，7，還有 Vista) ，印表機和傳真 (WinXP)** 或者 **印表機 (Win98 / ME)** 中設定或更改。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **列印** \> **立即列印**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_FILE\_PRINT\_DIRECT (4115)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4115);

#### \[VBScript\]

> editor.ExecuteCommandByID 4115
