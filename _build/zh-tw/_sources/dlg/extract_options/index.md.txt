# 「抽出選項」對話方塊

這個對話方塊會顯示當在 [**尋找** 對話方塊](../find/index) 中選擇 **抽出選項** 時。

### 「以輸出列顯示」核取方塊

如果勾選了這個核取方塊，EmEditor 會在輸出欄中顯示搜索結果。

### 「抽出選項」下列清單方塊

選擇如何顯示搜索結果。

|     |     |
| --- | --- |
| **顯示檔案名和行** | 檔案名稱以及包含搜索字串的行將會作為結果被顯示。 |
| **僅顯示檔案名** | 只有檔案名會被顯示為尋找結果，包含搜索字串的行將不會被顯示。 |
| **僅顯示符合的行** | 只有包含搜索字串的行將作為結果被顯示。 |
| **僅顯示符合的字串** | 只有符合的字串會作為結果被顯示。 |

### 「符合的行以上的額外的行數」下拉清單方塊

當 **抽出選項** 下拉功能表中選擇 **顯示檔案名和行** 或 **僅顯示符合的行** 時，指定符合行上方的額外的行數。

### 「符合的行以下的額外的行數」下拉清單方塊

當 **抽出選項** 下拉功能表中選擇 **顯示檔案名和行** 或 **僅顯示符合的行** 時，指定符合行下方的額外的行數。

### 「分隔符號」文字方塊

當為 **符合的行以上的額外的行數** 或 **符合的行以下的額外的行數** 下拉清單方塊指定大於 0 的數字時，指定分隔符號。

### 「計算常用字串」核取方塊

如果勾選了該核取方塊，EmEditor 會計算常用字串並將其抽出到新文檔中。

### 「行數」文字方塊

指定要抽出的最大字串數。

### 「CSV 格式」下拉清單方塊

指定要顯示為的 CSV 格式。

### 「添加」按鈕

添加一個條目到清單上。

### 「刪除」按鈕

點擊該按鈕從清單中刪除被選取的條目。
