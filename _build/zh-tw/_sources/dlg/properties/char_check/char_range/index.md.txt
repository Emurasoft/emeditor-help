# 「字元範圍」對話方塊

這個對話方塊會出現當點擊 [**字元檢查** 頁面](../index) 或 [**模糊比對選項** 對話方塊](../../../fuzzy_options/index) 上的 [**「添加」** 按鈕](../index)。

### 「Unicode 指令碼」清單方塊

選擇 Unicode 指令碼。

### 「Unicode 的一般類別」清單方塊

選擇 Unicode 的一般類別。

### 「最小字元」文字方塊

輸入最小字元範圍。您可以在十六進位值中輸入至少 4 位字元程式碼值，帶或不帶「U+」前綴，或者實際的一個字元。例如，您可以輸入：「U+0061」、「0061」或「a」。

### 「最大字元」文字方塊

輸入與 **最小字元** 文字方塊格式相同的最大字元範圍。
