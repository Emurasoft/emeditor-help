# 「新增檔案詳細信息」對話方塊

這個對話方塊會出現當點擊組態屬性中 [**檔案** 頁面](../index) 上的 [**「新增檔案」** 按鈕](../index)。

### 「編碼」下拉清單方塊

選擇您儲存一個新檔案的編碼。

### 「添加一個 Unicode 簽名 (BOM)」核取方塊

添加一個簽名當您用 Unicode 或 UTF-8 儲存一個新檔案時。

### 「換行方式」下拉清單方塊

當您創建一個新檔案時，選擇一個換行方式。

### 「添加序列號到未命名的文檔名稱中」核取方塊

當你新增一個未命名檔案時，EmEditor 會添加序號到未命名檔案名稱中，文檔名稱會變成 "untitled-1"，"untitled\_2"，...。

### 「使用模板」核取方塊

用一個模板檔案來開啟一個新檔案。

### 「模板檔案」文字方塊

指定一個要加載新檔案的模板檔案名稱。

### 「...」按鈕

點擊該按鈕來尋找指定的檔案。

### 「設置初始資料夾」核取方塊

設置一個預設的初始資料夾。當一個 **無標題** 文檔被儲存時， **另存新檔** 對話方塊就會被用指定的初始資料夾初始。注意如果一個
dialog box will be initialized
with the specified initial folder. Notes that if a **已知的資料夾** (例如 **文檔**， **圖片** 等) 被指定，Windows 系統可能會重新定位檔案位置。如果 **Sky Drive** 整合在 Windows 8.1 上被啟動，那么 **文檔** 資料夾可能會被重新定位到 **Sky Drive 文檔** 資料夾中。

### 「初始資料夾」文字方塊

為預設初始資料夾指定資料夾名稱。

### 「...」按鈕

點擊該按鈕來尋找指定的初始資料夾。

### 「重設」按鈕

重設為預設設定。 [**重設** 對話方塊](../../reset/index) 會被顯示并讓您能從另一個組態複製設定。
