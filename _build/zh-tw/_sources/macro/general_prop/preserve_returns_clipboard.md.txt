# PreserveReturnsClipboard 屬性

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **在剪貼簿保留換行方式 (CR/LF)** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **PreserveReturnsClipboard**;

object. **PreserveReturnsClipboard** = _b_;

#### \[VBScript\]

_b_ =
object. **PreserveReturnsClipboard**

object. **PreserveReturnsClipboard** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
