# FileSaveProp 對象

## 屬性

|     |     |
| --- | --- |
| [AlwaysEnabled](always_enabled) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **總是啟用儲存** 核取方塊相對應。 |
| [DefaultExtension](default_extension) | 與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **預設副檔名** 文字方塊相對應。 |
| [DeleteEmpty](delete_empty) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **儲存時刪除空檔案** 核取方塊相對應。 |
| [DeleteSpacesAtEnd](delete_spaces_at_end) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **刪除行末空白** 核取方塊相對應。 |
| [Encoding](encoding) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **編碼** 下拉清單方塊相對應。 |
| [InsertReturns](insert_returns) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **儲存時自動換行** 核取方塊相對應。 |
| [NamedEntity](named_entity) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **使用命名實體引用** 核取方塊相對應。 |
| [NameUntitled](name_untitled) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **自動為無標題文檔命名** 核取方塊相對應。 |
| [NumericEntity](numeric_entity) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **把 Unicode 儲存為 HTML/XML 字元引用** 核取方塊相對應。 |
| [PromptRename](prompt_rename) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **更名時在刪除舊檔案前提示** 核取方塊相對應。 |
| [PromptUnicode](prompt_unicode) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **當儲存 Unicode 為 ANSI 時進行提示** 核取方塊相對應。 |
| [ReturnMethod](return_method) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **換行方式** 下拉清單方塊相對應。 |
| [TabsAsSpaces](tabs_as_spaces) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **把 Tab 儲存為空白** 核取方塊相對應。 |
| [UnicodeSignature](unicode_signature) | 與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **添加一個 Unicode 簽名 (BOM)** 核取方塊相對應。 |
