# SaveToFolderIfRecycleFailed 屬性

與組態屬性中 [**備份** 頁面](../../dlg/properties/backup/index) 上的 **無法使用回收站，儲存到備份資料夾** 核取方塊對應。

#### \[JavaScript\]

_b_ = object. **SaveToFolderIfRecycleFailed**;

object. **SaveToFolderIfRecycleFailed** = _b_;

#### \[VBScript\]

_b_ = object. **SaveToFolderIfRecycleFailed**

object. **SaveToFolderIfRecycleFailed** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
