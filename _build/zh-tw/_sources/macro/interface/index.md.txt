# Interface 對象

## 屬性

|     |     |
| --- | --- |
| **[x](x)** | 返回被一個支持的外掛程式所傳遞的游標的水平位置。 |
| **[y](y)** | 返回被一個支持的外掛程式所傳遞的游標的垂直位置。 |

## 方法

|     |     |
| --- | --- |
| **[write](write)** | 把一個字串傳遞到一個支持的外掛程式中。 |
| **[writeln](writeln)** | 把一個字串和新行傳遞到一個支持的外掛程式中。 |

## 版本

支持 EmEditor 9.00 或之後的版本。

```{toctree}
:maxdepth: 1
write
writeln
x
y
```
