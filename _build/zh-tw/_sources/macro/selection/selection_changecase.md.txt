# ChangeCase 方法

變更所選取的文字的大小寫。

#### \[JavaScript\]

document.selection. **ChangeCase**( _nFlags_ );

#### \[VBScript\]

document.selection. **ChangeCase** _nFlags_

## 參數

_nFlags_

指定下列值之一:

|     |     |
| --- | --- |
| eeCaseLowerCase | 把大寫字母轉換成小寫字母。 |
| eeCaseUpperCase | 把小寫字母轉換成大寫字母。 |
| eeCaseCapitalize | 大寫每一個單字的第一個字母 (EmEditor 4.10 或之後的版本) 。 |

## 版本

支持 EmEditor 4.00 或之後的版本。
