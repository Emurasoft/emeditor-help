# URLCharacters 屬性

與組態屬性中 [**連結** 頁面](../../dlg/properties/link/index) 上的 **識別這些字元** 核取方塊相對應。

#### \[JavaScript\]

_s_ = object. **URLCharacters**;

object. **URLCharacters** = _s_;

#### \[VBScript\]

_s_ = object. **URLCharacters**

object. **URLCharacters** = _s_

## 版本

支持 EmEditor 7.00 或之後的版本。
