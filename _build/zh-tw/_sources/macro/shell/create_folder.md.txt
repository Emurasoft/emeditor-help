# CreateFolder 方法

創建一個資料夾。

#### \[JavaScript\]

shell. **CreateFolder**( _strFolder_ );

#### \[VBScript\]

shell. **CreateFolder** _strFolder_

## 參數

_strFolder_

指定要創建的資料夾。

## 範例

#### \[JavaScript\]

shell.CreateFolder( "C:\\\Test" );

#### \[VBScript\]

shell.CreateFolder "C:\\Test"

## 版本

支持 EmEditor Professional 22.1 或之後的版本。
