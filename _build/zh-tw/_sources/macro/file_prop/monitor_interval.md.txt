# MonitorInterval 屬性

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **監控時間間隔** 文字方塊相對應。

#### \[JavaScript\]

_n_ = object. **MonitorInterval**;

object. **MonitorInterval** = _n_;

#### \[VBScript\]

_n_ = object. **MonitorInterval**

object. **MonitorInterval** = _n_

## 版本

支持 EmEditor 7.00 或之後的版本。
