# NewlineCode 屬性

檢索文檔的目前的新行字元碼。

#### \[JavaScript\]

_nNewlineCode_ = document. **NewlineCode**;

#### \[VBScript\]

_nNewlineCode_ = document. **NewlineCode**

## 參數

_nNewlineCode_

指定下列其中一個值。

|     |     |
| --- | --- |
| eeCrAndLf | CR+LF |
| eeCrOnly | 僅 CR |
| eeLfOnly | 僅 LF |
| eeNewlineMixed | 混合 |

## 版本

支持 EmEditor Professional 17.6 或之後的版本。
