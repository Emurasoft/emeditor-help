這些常數在頭檔案 (plugin.h) 中被定義。

同樣支持下列 Windows API 消息。其中，有一些支持可能不完整，因為消息的視窗控制代碼參數指定的是視圖的視窗控制代碼，而不是 EmEditor 的框架。有關這些消息的更多信息請檢視微軟 MSDN 資源庫。

|     |     |
| --- | --- |
| EM\_GETSEL | 檢索目前的選區的開始和結束字元位置。 |
| EM\_SCROLLCARET | 捲動游標到視圖。 |
| EM\_SETSEL | 選擇一個字元範圍。 |
| EM\_REPLACESEL | 用指定的文字取代目前的選區。 |
| WM\_CLEAR | 刪除目前的選區。 |
| WM\_COPY | 把目前的選區複製到剪貼簿上。 |
| WM\_CUT | 刪除目前的選區并複製被刪除的文字到剪貼簿上。 |
| WM\_GETTEXT | 複製整個文檔到一個緩衝區。 |
| WM\_GETTEXTLENGTH | 檢索用來檢索整個文檔，除了終止空字元外的緩衝區大小。 |
| WM\_PASTE | 在目前的游標位置複製目前的在剪貼簿上的內容。 |
| WM\_SETTEXT | 設置整個文檔。 |
| WM\_UNDO | 撤回最后的操作。 |
