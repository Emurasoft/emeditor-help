啟用標籤頁 - 開命令¶

摘要¶

啟用標籤頁并合併所有視窗。

說明¶

啟用標籤頁并合併所有視窗。如果不同的群組中已用標籤頁顯示不同的文檔，那么這個命令會合併所有群組。當標籤頁被啟用，所有目前的打開的文檔會被顯示在視窗上方的的標籤功能表上。只有一個 EmEditor 圖示會被顯示在 Windows 任務欄中。

運行方法¶

  • 預設功能表: 無

  • 全部命令: 視窗 > 啟用標籤頁 > 啟用標籤頁 - 開

  • 工具列: 無

  • 狀態列: 無

  • 預設捷徑: 無

外掛程式命令 ID¶

  • EEID_WINDOW_COMBINE_ON (4407)

巨集¶

[JavaScript]¶

editor.EnableTab = true;

[VBScript]¶

editor.EnableTab = True
