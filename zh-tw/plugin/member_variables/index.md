# 所需的成員變數

有幾個變數必須在 CETLFrame 中定義才能編譯。以 \_ID 開頭的變數需要該資源的 ID。

|     |     |
| --- | --- |
| **識別碼** | **說明** |
| \_IDS\_NAME | 在「自訂外掛程式」視窗中顯示為「名稱」。 |
| \_IDS\_MENU | 顯示在工具 \> 外掛程式功能表中。 |
| \_IDS\_STATUS | 顯示在外掛程式工具列的工具提示中。 |
| \_IDS\_VER | 在「自訂外掛程式」視窗中顯示為「版本」。 |
| \_USE\_LOC\_DLL | 值為 1 指定要使用包含多語言使用者界面資源的獨立 DLL。 否則，該值為 0。這些 DLL 將被放置在 EmEditor > mui 中。 |
| \_IDB\_BITMAP | 點陣圖圖像用作所有工具列工具盤和大小的預設圖示。 如果 EmEditor 未加載圖示，請嘗試將其匯出到 Paint 中。 以下圖示圖像用於使用不同的工具列設置時進行最佳顯示。 |
| \_IDB\_16C\_24 | 16 色，24x24 px。 |
| \_IDB\_TRUE\_16\_DEFAULT | 全彩 (24 或 32 位元)，16x16 px。 |
| \_IDB\_TRUE\_16\_HOT | 全彩，16x16 px。游標懸停在圖示上時顯示此圖像。 |
| \_IDB\_TRUE\_16\_BW | 全彩，16x16 px，黑白。當外掛程式停用時顯示。 |
| \_IDB\_TRUE\_24\_DEFAULT | 全彩，24x24 px。 |
| \_IDB\_TRUE\_24\_HOT | 全彩，24x24 px。游標懸停在圖示上時顯示此圖像。 |
| \_IDB\_TRUE\_24\_BW | 全彩，24x24 px，黑白。 |
| \_MASK\_TRUE\_COLOR | 在全彩圖示中變透明的顏色。用 RGB(r,g,b) 設置值。 |
| \_ALLOW\_OPEN\_SAME\_GROUP | 允許在外掛程式執行期間在同一個視窗群組中打開檔案。 |
| \_ALLOW\_MULTIPLE\_INSTANCES | 如果外掛程式支持多個例項，則為 TRUE。 如果允許外掛程式同時在多個畫面格中運行，則此消息應該返回 TRUE。 請注意，全域變數將在多個實例運行時共享。 |
| \_MAX\_EE\_VERSION | 不使用。 |
| \_MIN\_EE\_VERSION | 不使用。 |
| \_SUPPORT\_EE\_PRO | 不使用。 |
| \_SUPPORT\_EE\_STD | 不使用。 |
