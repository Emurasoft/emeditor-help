# EE\_NUMBERING

在游標位置或垂直選擇處，插入編號。你能明確地發送該消息或用 [Editor\_Numbering 內嵌函式](../macro/editor_numbering)。

EE\_NUMBERING

wParam = (WPARAM)(NUMBERING\_INFO\*)pNI;

lParam = 0;

## 參數

_pUNI_

> 指定指針指向 [NUMBERING\_INFO 結構](../structure/numbering_info)。

## 返回值

> 如果成功的話，返回值為 0。

## 版本

> 支持 EmEditor Professional 19.1 或之後的版本。
