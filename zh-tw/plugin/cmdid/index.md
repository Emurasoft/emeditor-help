# 命令 ID

命令 ID 通過 [EE\_EXEC\_COMMAND 消息](../message/ee_exec_command)， [Editor\_ExecCommand 內嵌函式](../macro/editor_execcommand)， [EE\_QUERY\_STATUS 消息](../message/ee_query_status)，以及 [Editor\_QueryStatus 內嵌函式](../macro/editor_querystatus) 被使用。

命令 ID 的說明請訪問 [命令參考](../../cmd/index)。

這些常數在頭檔案(plugin.h)中被定義。
