# ShowDBSpaces 扽俶 (MarkProp 對象)

Corresponds to theShow
Double-Byte Spaces check box in the
[Marks tab](../../dlg/properties/marks/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.ShowDBSpaces;
object.ShowDBSpaces = b;
```

### \[VBScript\]

```
b = object.ShowDBSpaces
object.ShowDBSpaces = b
```
