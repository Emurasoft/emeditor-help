# CrLfDifferent 屬性 (MarkProp 對象)

與組態屬性中 [標記 頁面](../../dlg/properties/marks/index) 上的使用不同的標記來顯示 CR 與 LF 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.CrLfDifferent;
object.CrLfDifferent = b;
```

### \[VBScript\]

```
b = object.CrLfDifferent
object.CrLfDifferent = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
