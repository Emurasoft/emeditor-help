# ShowControlCharacters 屬性 (MarkProp 對象)

與組態屬性中 [標記 頁面](../../dlg/properties/marks/index) 上的顯示控制字元 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.ShowControlCharacters;
object.ShowControlCharacters = b;
```

### \[VBScript\]

```
b = object.ShowControlCharacters
object.ShowControlCharacters = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
