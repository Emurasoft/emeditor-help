# Count 屬性 (Configs 集合)

檢索組態的數目。

## 

### \[JavaScript\]

```
nCount = editor.Configs.Count;
```

### \[VBScript\]

```
nCount = editor.Configs.Count
```

## 示例

### \[JavaScript\]

```
alert( "Number of Configurations: " + editor.Configs.Count );
```

### \[VBScript\]

```
alert "Number of Configurations: " & editor.Configs.Count
```

## 版本

支持 EmEditor 7.00 或之後的版本。
