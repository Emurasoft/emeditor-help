# ScriptBegin 屬性 (HighlightProp 對象)

與組態屬性中 [**亮顯 (2)** 頁面](../../dlg/properties/highlight2/index) 上的 **「腳本」群組方塊 \- 「開始」** 文字方塊相對應。

## 

### \[JavaScript\]

```
str = object.ScriptBegin;
object.ScriptBegin = str;
```

### \[VBScript\]

```
str = object.ScriptBegin
object.ScriptBegin = str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
