# QuoteContinue 屬性 (HighlightProp 對象)

與組態屬性中 [亮顯 (2) 頁面](../../dlg/properties/highlight2/index) 上的「引號內的字串」群組方塊 \- 「轉到下一行」 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.QuoteContinue;
object.QuoteContinue = b;
```

### \[VBScript\]

```
b = object.QuoteContinue
object.QuoteContinue = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
