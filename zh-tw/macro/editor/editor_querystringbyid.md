# QueryStringByID 方法 (Editor 對象)

檢索與指定命令相關聯的字串。

## 

### \[JavaScript\]

```
str = editor.QueryStringByID( nID );
```

### \[VBScript\]

```
str = editor.QueryStringByID( nID )
```

## 返回值

返回字串，該字串取決于 nID 參數的含義。如果 nID 參數無效，該方法會返回一個空字串。

## 版本

支持 EmEditor 7.00 或之後的版本。
