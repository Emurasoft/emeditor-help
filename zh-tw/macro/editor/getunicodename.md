# GetUnicodeName 方法 (Editor 對象)

檢索指定字元或字串的 Unicode 名。

## 

### \[JavaScript\]

```
strName = editor.GetUnicodeName( strChar );
```

### \[VBScript\]

```
strName = editor.GetUnicodeName( strChar )
```

## 參數

_strChar_

指定源字元或字串。

## 返回值

返回值時指定的字元或字串的 Unicode 名。

## 版本

支持 EmEditor Professional v19.1 或之後的版本。
