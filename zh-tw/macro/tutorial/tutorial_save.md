# 儲存巨集 (教程)

要儲存一個巨集，請選擇 [**儲存巨集** 命令](../../cmd/macros/macro_save)，EmEditor 會顯示一個 **另存新檔** 對話方塊。EmEditor 的預設資料夾是 **我的文檔** 資料夾下的 **我的巨集** 資料夾。要更改預設資料夾，請選擇 [**自訂巨集** 命令](../../cmd/macros/customize_macro)，然后在 **自訂巨集** 對話方塊上的 [**選項** 頁面](../../dlg/macro_customize/options/index) 上變更資料夾。

在儲存時，您能在快顯的對話方塊上的 **儲存類型** 下拉清單方塊中選擇 JavaScript 或 VBScript。

現在給巨集檔案命名。讓我們把這個教程檔案儲存為:

"tutorial"

注意: 當您儲存檔案時，您不需要指定一個副檔名。適當的檔案副檔名會根據檔案被儲存為的指令碼語言而自動追加。

點擊 **「儲存」** 用我們指定的名字儲存該巨集。
