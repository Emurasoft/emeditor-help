# 錄制巨集 (教程)

要錄制一個巨集，選擇 [**開始/停止巨集錄制** 命令](../../cmd/macros/quick_macro_record)。
游標指針會變為一個攝像機圖示，表示巨集錄制開啟。在這個狀態下，您能在 EmEditor 中錄制大部分的操作。讓我們通過輸入下列文字開始教程:

"EmEditor supports macros."

在輸入文字后，再次選擇 [**開始/停止巨集錄制** 命令](../../cmd/macros/quick_macro_record)。
攝像機圖示會變回原來的 Windows 游標指針。

我們已經錄制了插入 _"EmEditor supports macros."_ 文字到 EmEditor 中。

我們把這個稱為一個 "臨時巨集" 因為還沒有儲存這個巨集 (它只是被臨時地儲存在 Windows 注冊表中) 。您能執行該臨時巨集通過選擇 [**運行巨集** 命令](../../cmd/macros/quick_macro_run)。臨時巨集會一直被程式儲存直到錄制一個新的巨集操作為止，并且您想執行幾次都可以。當您退出 EmEditor 后再重新打開它，臨時巨集仍然可用。

一個臨時巨集可提供一種快捷、方便的自動化任務的方式。如果您想要錄制并執行多個巨集，您需要儲存巨集
(請參考 **[儲存巨集](tutorial_save)**)。

## 提示

EmEditor 能記錄包括鍵盤和游標指針移動的大多數操作。
