# Count 屬性 (CsvList 集合)

檢索 Csv 對象的數目。

## 

### \[JavaScript\]

```
nCount = editor.CsvList.Count;
```

### \[VBScript\]

```
nCount = editor.CsvList.Count
```

## 範例

### \[JavaScript\]

```
alert( "Csv 對象數： " + editor.CsvList.Count );
```

### \[VBScript\]

```
alert "Csv 對象數： " & editor.CsvList.Count
```

## 版本

支持 EmEditor 19.4 或之後的版本。
