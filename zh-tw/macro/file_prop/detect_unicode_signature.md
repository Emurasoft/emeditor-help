# DetectUnicodeSignature 屬性 (FileProp 對象)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **偵測 Unicode 簽名 (BOM)** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DetectUnicodeSignature;
object.DetectUnicodeSignature = b;
```

### \[VBScript\]

```
b = object.DetectUnicodeSignature
object.DetectUnicodeSignature = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
