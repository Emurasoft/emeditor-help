# PromptInconsistentReturns 屬性 (FileProp 對象)

與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的換行方式不一致時進行提示 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.PromptInconsistentReturns;
object.PromptInconsistentReturns = b;
```

### \[VBScript\]

```
b = object.PromptInconsistentReturns
object.PromptInconsistentReturns = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
