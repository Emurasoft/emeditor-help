# NotepadDiary 屬性 (FileProp 對象)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **創建與記事本兼容的日志檔案** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.NotepadDiary;
object.NotepadDiary = b;
```

### \[VBScript\]

```
b = object.NotepadDiary
object.NotepadDiary = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
