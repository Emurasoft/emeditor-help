# PromptInvalid 屬性 (FileProp 對象)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **如果發現無效字元時進行提示** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.PromptInvalid;
object.PromptInvalid = b;
```

### \[VBScript\]

```
b = object.PromptInvalid
object.PromptInvalid = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
