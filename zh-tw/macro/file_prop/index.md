# FileProp 對象

## 屬性

|     |     |
| --- | --- |
| [CheckFileChanged](check_file_changed) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的被其他程式更改時 下拉清單方塊相對應。 |
| [DetectAll](detect_all) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的全部偵測 核取方塊相對應。 |
| [DetectCharset](detect_charset) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 HTML/XML 字元集 核取方塊相對應。 |
| [DetectCSV](detect_csv) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 CSV (逗號分隔) 核取方塊相對應。 |
| [DetectDSV](detect_dsv) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 DSV (使用者定義的分隔符分隔) 核取方塊相對應。 |
| [DetectTSV](detect_tsv) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 TSV (Tab 鍵分隔) 核取方塊相對應。 |
| [DetectUnicodeSignature](detect_unicode_signature) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 Unicode 簽名 (BOM) 核取方塊相對應。 |
| [DetectUTF8](detect_utf8) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的偵測 UTF-8 核取方塊相對應。 |
| [Encoding](encoding) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的檔案打開時的編碼 下拉清單方塊相對應。 |
| [MonitorInterval](monitor_interval) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的監控時間間隔 文字方塊相對應。 |
| [NoFullPathIfNotActive](no_full_path_if_not_active) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的失去焦點時不顯示完整路徑 核取方塊相對應。 |
| [NotepadDiary](notepad_diary) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的創建與記事本兼容的日志檔案 核取方塊相對應。 |
| [PromptInconsistentReturns](prompt_inconsistent_returns) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的換行方式不一致時進行提示 核取方塊相對應。 |
| [PromptInvalid](prompt_invalid) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的如果發現無效字元時進行提示 核取方塊相對應。 |
| [PromptNull](prompt_null) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的如果發現空字元時進行提示 核取方塊相對應。 |
| [ShowFullPath](show_full_path) | 與組態屬性中 [檔案 頁面](../../dlg/properties/file/index) 上的顯示包含完整路徑的檔案名 核取方塊相對應。 |
