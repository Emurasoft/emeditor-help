# DetectCharset 屬性 (FileProp 對象)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **偵測 HTML/XML 字元集** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DetectCharset;
object.DetectCharset = b;
```

### \[VBScript\]

```
b = object.DetectCharset
object.DetectCharset = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
