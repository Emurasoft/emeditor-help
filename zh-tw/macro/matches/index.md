# Matches 集合

Matches 集合提供一集合的規則運算式搜尋結果的 [Match 對象](../match/index)。該集合由作為完全符合項目的第一項 (反向參考 0)，以及子對應項目 (反向參考 1, 2, 3, ...) 構成。

## 屬性

|     |     |
| --- | --- |
| **[Count](count)** | 檢索 Match 對象的數目。 |
| **[Item](item)** | 按指定索引檢索 Match 對象。Item(0) 返回完全符合的項目 (反向參考 0), Item(1) 返回反向參考 1。 |

## 版本

支持 EmEditor Professional Version 15.9 或之後的版本。
