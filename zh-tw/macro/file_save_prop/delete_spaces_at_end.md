# DeleteSpacesAtEnd 屬性 (FileSaveProp 對象)

與 [**儲存詳細信息** 對話方塊](../../dlg/properties/file/save_details/index) 中的 **刪除行末空格** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DeleteSpacesAtEnd;
object.DeleteSpacesAtEnd = b;
```

### \[VBScript\]

```
b = object.DeleteSpacesAtEnd
object.DeleteSpacesAtEnd = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
