# TabsAsSpaces 屬性 (FileSaveProp 對象)

與 [把 Tab 儲存為空格 核取方塊](../../dlg/properties/file/save_details/index) 相對應。

## 

### \[JavaScript\]

```
b = object.TabsAsSpaces;
object.TabsAsSpaces = b;
```

### \[VBScript\]

```
b = object.TabsAsSpaces
object.TabsAsSpaces = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
