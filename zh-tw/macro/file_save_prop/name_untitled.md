# NameUntitled 屬性 (FileSaveProp 對象)

與 [儲存詳細信息 對話方塊](../../dlg/properties/file/save_details/index) 中的自動為無標題文檔命名 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.NameUntitled;
object.NameUntitled = b;
```

### \[VBScript\]

```
b = object.NameUntitled
object.NameUntitled = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
