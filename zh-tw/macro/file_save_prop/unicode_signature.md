# UnicodeSignature 屬性 (FileSaveProp 對象)

與 [**添加一個 Unicode 簽名 (BOM)** 核取方塊](../../dlg/properties/file/save_details/index) 相對應。

## 

### \[JavaScript\]

```
b = object.UnicodeSignature;
object.UnicodeSignature = b;
```

### \[VBScript\]

```
b = object.UnicodeSignature
object.UnicodeSignature = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
