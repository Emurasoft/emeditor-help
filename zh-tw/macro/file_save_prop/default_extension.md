# DefaultExtension 屬性 (FileSaveProp 對象)

與組態屬性中 [**檔案** 頁面](../../dlg/properties/file/index) 上的 **預設副檔名** 文字方塊相對應。

## 

### \[JavaScript\]

```
s = object.DefaultExtension;
object.DefaultExtension = s;
```

### \[VBScript\]

```
s = object.DefaultExtension
object.DefaultExtension = s
```

## 版本

支持 EmEditor 7.00 或之後的版本。
