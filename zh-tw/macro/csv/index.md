# Csv 對象

## 屬性

|     |     |
| --- | --- |
| [AllowDelimiter](allow_delimiter) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的允許分隔符號在引號內 核取方塊對應。 |
| [AllowNL](allow_nl) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的允許在引號內換行 核取方塊對應。 |
| [Delimiter](delimiter) | 檢索或設定目前的 Csv 對象的分隔符號。 |
| [EnableTitle](enable_title) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的工具列按鈕標題 核取方塊對應。 |
| [Escape](escape) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的使用逸出字元 核取方塊對應。 |
| [FirstDetectLine](first_detect_line) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的要偵測的第一行 文字方塊對應。 |
| [Heading](heading) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的欄標題 文字方塊對應。 |
| [LinesCell](lines_cell) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的一個儲存格中的最大換行數 文字方塊對應。 |
| [MaxDetectLines](max_detect_lines) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的要偵測的最大行數 文字方塊對應。 |
| [MinDetectDelimiter](min_detect_delimiter) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的要偵測的最小分隔符號數 文字方塊對應。 |
| [Name](name) | 檢索或設定目前的 Csv 對象的名稱。 |
| [QuotationMark](quotation_mark) | 檢索或設定目前的 Csv 對象的引號標記。 |
| [RowHeading](row_heading) | 與在自訂對話方塊中 [CSV 格式 頁面](../../dlg/customize/csv/index) 上的列標題 文字方塊對應。 |
| [Title](title) | 與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的工具列按鈕標題 文字方塊對應。 |
