# MinDetectDelimiter 屬性 (Csv 對象)

與在自訂對話方塊中 [**CSV** 頁面](../../dlg/customize/csv/index) 上的 **要偵測的最小分隔符號數** 文字方塊對應。

## 

### \[JavaScript\]

```
n = object.MinDetectDelimiter;
object.MinDetectDelimiter = n;
```

### \[VBScript\]

```
n = object.MinDetectDelimiter
object.MinDetectDelimiter = n
```

## 版本

支持 EmEditor 19.4 或之後的版本。
