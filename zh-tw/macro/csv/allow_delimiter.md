# AllowDelimiter 屬性 (Csv 對象)

與在自訂對話方塊中 [CSV 頁面](../../dlg/customize/csv/index) 上的允許分隔符號在引號內 核取方塊對應。

## 

### \[JavaScript\]

```
b = object.AllowDelimiter;
object.AllowDelimiter = b;
```

### \[VBScript\]

```
b = object.AllowDelimiter
object.AllowDelimiter = b
```

## 版本

支持 EmEditor 19.4 或之後的版本。
