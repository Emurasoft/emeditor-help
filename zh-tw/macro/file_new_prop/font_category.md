# FontCategory 扽俶 (FileNewProp 對象)

Corresponds to theFont
Category drop-down list box in the
[New File Details dialog box.](../../dlg/properties/file/new_details/index)

## 

### \[JavaScript\]

```
n = object.FontCategory;
object.FontCategory = n;
```

### \[VBScript\]

```
n = object.FontCategory
object.FontCategory = n
```
