# FileNewProp 對象

## 屬性

|     |     |
| --- | --- |
| [Encoding](encoding) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **編碼** 下拉清單方塊相對應。 |
| [InitialFolder](initial_folder) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **初始資料夾** 文字方塊相對應。 |
| [ReturnMethod](return_method) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **換行方式** 下拉清單方塊相對應。 |
| [SetInitialFolder](set_initial_folder) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **設置初始資料夾** 核取方塊相對應。 |
| [TemplateFile](template_file) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **模板檔案** 文字方塊相對應。 |
| [UnicodeSignature](unicode_signature) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **添加一個 Unicode 簽名 (BOM)** 核取方塊相對應。 |
| [UseTemplate](use_template) | 與 [**新增檔案詳細信息** 對話方塊](../../dlg/properties/file/new_details/index) 中的 **使用模板** 文字方塊相對應。 |
