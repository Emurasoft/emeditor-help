# UnicodeSignature 屬性 (FileNewProp 對象)

與 [新增檔案詳細信息 對話方塊](../../dlg/properties/file/new_details/index) 中的添加一個 Unicode 簽名 (BOM) 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.UnicodeSignature;
object.UnicodeSignature = b;
```

### \[VBScript\]

```
b = object.UnicodeSignature
object.UnicodeSignature = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
