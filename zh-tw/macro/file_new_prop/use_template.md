# UseTemplate 屬性 (FileNewProp 對象)

與 [新增檔案詳細信息 對話方塊](../../dlg/properties/file/new_details/index) 中的使用模板 文字方塊相對應。

## 

### \[JavaScript\]

```
b = object.UseTemplate;
object.UseTemplate = b;
```

### \[VBScript\]

```
b = object.UseTemplate
object.UseTemplate = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
