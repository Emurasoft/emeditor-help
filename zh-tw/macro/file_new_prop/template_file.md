# TemplateFile 屬性 (FileNewProp 對象)

與 [新增檔案詳細信息 對話方塊](../../dlg/properties/file/new_details/index) 中的模板檔案 文字方塊相對應。

## 

### \[JavaScript\]

```
s = object.TemplateFile;
object.TemplateFile = s;
```

### \[VBScript\]

```
s = object.TemplateFile
object.TemplateFile = s
```

## 版本

支持 EmEditor 7.00 或之後的版本。
