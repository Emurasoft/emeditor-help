# IgnoreAfter 屬性 (IndentProp 對象)

與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的忽略規則運算式之後的內容 文字方塊相對應。

## 

### \[JavaScript\]

```
s = object.IgnoreAfter;
object.IgnoreAfter = s;
```

### \[VBScript\]

```
s = object.IgnoreAfter
object.IgnoreAfter = s
```

## 版本

支持 EmEditor 9.00 或之後的版本。
