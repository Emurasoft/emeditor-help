# EndOfStatement 屬性 (IndentProp 對象)

與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的語句結束 文字方塊相對應。

## 

### \[JavaScript\]

```
s = object.EndOfStatement;
object.EndOfStatement = s;
```

### \[VBScript\]

```
s = object.EndOfStatement
object.EndOfStatement = s
```

## 版本

支持 EmEditor 7.00 或之後的版本。
