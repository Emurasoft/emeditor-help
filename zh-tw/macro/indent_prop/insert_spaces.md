# InsertSpaces 屬性 (IndentProp 對象)

與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的將 Tab轉換為空格 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.InsertSpaces;
object.InsertSpaces = b;
```

### \[VBScript\]

```
b = object.InsertSpaces
object.InsertSpaces = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
