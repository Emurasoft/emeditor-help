# IndentColumns 屬性 (IndentProp 對象)

與 [**Tab/縮排** 對話方塊](../../dlg/properties/general/indent/index) 上的 **縮排列** 文字方塊相對應。

## 

### \[JavaScript\]

```
n = object.IndentColumns;
object.IndentColumns = n;
```

### \[VBScript\]

```
n = object.IndentColumns
object.IndentColumns = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
