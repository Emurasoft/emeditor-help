# IndentProp 對象

## 屬性

|     |     |
| --- | --- |
| [AutoIndent](auto_indent) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的啟用自動縮排 核取方塊相對應。 |
| [Begin](begin) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的開始縮排 文字方塊相對應。 |
| [End](end) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的結束縮排 文字方塊相對應。 |
| [EndOfStatement](end_of_statement) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的語句結束 文字方塊相對應。 |
| [IgnoreAfter](ignore_after) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的忽略規則運算式之後的內容 文字方塊相對應。 |
| [IndentColumns](indent_columns) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的縮排列 文字方塊相對應。 |
| [InsertSpaces](insert_spaces) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的將 Tab轉換為空白 核取方塊相對應。 |
| [RegExp](reg_exp) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的使用規則運算式 核取方塊相對應。 |
| [TabColumns](tab_columns) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的Tab 列 文字方塊相對應。 |
| [WrapIndent](wrap_indent) | 與 [Tab/縮排 對話方塊](../../dlg/properties/general/indent/index) 上的換行縮排 核取方塊相對應。 |

## 版本

支持 EmEditor 7.00 或之後的版本。


```{toctree}
:maxdepth: 1
auto_indent
begin
end
end_of_statement
ignore_after
indent_columns
insert_spaces
reg_exp
tab_columns
wrap_indent
```
