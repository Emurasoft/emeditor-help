# SpellProp 對象

## 屬性

|     |     |
| --- | --- |
| [Enabled](enabled) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的拼字檢查 核取方塊相對應。 |
| [CamelCase](camelcase) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的支持駝峰式大小寫 核取方塊相對應。 |
| [Quote](quote) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的引用行 核取方塊相對應。 |
| [SingleQuotes](singlequotes) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的單引號內的字串 核取方塊相對應。 |
| [DoubleQuotes](doublequotes) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的雙引號內的字串 核取方塊相對應。 |
| [Comment](comment) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的注釋 核取方塊相對應。 |
| [Script](script) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的指令碼 核取方塊相對應。 |
| [InTag](intag) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的標記 核取方塊相對應。 |
| [Highlight](highlight) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的已亮顯的文字 核取方塊相對應。 |
| [Hyperlink](hyperlink) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的超連結 核取方塊相對應。 |
| [Rest](rest) | 與組態屬性中 [拼字檢查 頁面](../../dlg/properties/spell/index) 上的未指定的文字 核取方塊相對應。 |

## 版本

支持 EmEditor 10.00 或之後的版本。
