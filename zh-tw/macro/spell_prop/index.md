# SpellProp 對象

## 屬性

|     |     |
| --- | --- |
| [Enabled](enabled) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **拼字檢查** 核取方塊相對應。 |
| [CamelCase](camelcase) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **支持駝峰式大小寫** 核取方塊相對應。 |
| [Quote](quote) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **引用行** 核取方塊相對應。 |
| [SingleQuotes](singlequotes) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **單引號內的字串** 核取方塊相對應。 |
| [DoubleQuotes](doublequotes) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **雙引號內的字串** 核取方塊相對應。 |
| [Comment](comment) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **注釋** 核取方塊相對應。 |
| [Script](script) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **指令碼** 核取方塊相對應。 |
| [InTag](intag) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **標記** 核取方塊相對應。 |
| [Highlight](highlight) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **已亮顯的文字** 核取方塊相對應。 |
| [Hyperlink](hyperlink) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **超連結** 核取方塊相對應。 |
| [Rest](rest) | 與組態屬性中 [**拼字檢查** 頁面](../../dlg/properties/spell/index) 上的 **未指定的文字** 核取方塊相對應。 |

## 版本

支持 EmEditor 10.00 或之後的版本。
