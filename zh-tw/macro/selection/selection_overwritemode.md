# OverwriteMode 屬性

設置或檢索表示覆寫或插入模式的標志。

#### \[JavaScript\]

_bOverwrite_ = document.selection. **OverwriteMode**;

document.selection. **OverwriteMode** = _bOverwrite_;

#### \[VBScript\]

_bOverwrite_ = document.selection. **OverwriteMode**

document.selection. **OverwriteMode** = _bOverwrite_

## 版本

支持 EmEditor 4.00 或之後的版本。
