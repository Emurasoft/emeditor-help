# VirtualSpace 屬性 (GeneralProp 對象)

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **啟用虛空格** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.VirtualSpace;
object.VirtualSpace = b;
```

### \[VBScript\]

```
b = object.VirtualSpace
object.VirtualSpace = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
