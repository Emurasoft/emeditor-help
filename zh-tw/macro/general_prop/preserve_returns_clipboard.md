# PreserveReturnsClipboard 屬性 (GeneralProp 對象)

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **在剪貼簿保留換行方式 (CR/LF)** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.PreserveReturnsClipboard;
object.PreserveReturnsClipboard = b;
```

### \[VBScript\]

```
b = object.PreserveReturnsClipboard
object.PreserveReturnsClipboard = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
