# AllowControlCharacter 屬性 (GeneralProp 對象)

與組態屬性中 [一般 頁面](../../dlg/properties/general/index) 上的允許插入控制字元 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.AllowControlCharacter;
object.AllowControlCharacter = b;
```

### \[VBScript\]

```
b = object.AllowControlCharacter
object.AllowControlCharacter = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
