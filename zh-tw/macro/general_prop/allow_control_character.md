# AllowControlCharacter 屬性

與組態屬性中 [**一般** 頁面](../../dlg/properties/general/index) 上的 **允許插入控制字元** 核取方塊相對應。

#### \[JavaScript\]

_b_ =
object. **AllowControlCharacter**;

object. **AllowControlCharacter** = _b_;

#### \[VBScript\]

_b_ =
object. **AllowControlCharacter**

object. **AllowControlCharacter** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
