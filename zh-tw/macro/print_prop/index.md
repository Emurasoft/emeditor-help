# PrintProp 對象

## 屬性

|     |     |
| --- | --- |
| [DrawSeparatingLines](draw_separating_lines) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **插入分隔線** 核取方塊相對應。 |
| [Footer](footer) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **頁腳** 文字方塊相對應。 |
| [Header](header) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **頁眉** 文字方塊相對應。 |
| [IgnoreColor](ignore_color) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **忽略顏色與下劃線** 核取方塊相對應。 |
| [LineSpace](line_space) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **行間距** 核取方塊相對應。 |
| [ShowLineNumbers](show_line_numbers) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **列印行號** 核取方塊相對應。 |
| [WrapPage](wrap_page) | 與組態屬性中 [**列印** 頁面](../../dlg/properties/print/index) 上的 **列印時按頁面換行** 核取方塊相對應。 |

## 版本

支持 EmEditor Professional Version 7.00 或之後的版本。
