# DrawSeparatingLines 屬性 (PrintProp 對象)

與組態屬性中 [列印 頁面](../../dlg/properties/print/index) 上的插入分隔線 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DrawSeparatingLines;
object.DrawSeparatingLines = b;
```

### \[VBScript\]

```
b = object.DrawSeparatingLines
object.DrawSeparatingLines = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
