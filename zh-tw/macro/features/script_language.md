# 支持 JavaScript 或 VBScript (功能)

EmEditor 把 JavaScript 或 VBScript 作為它的巨集語言，所以對于熟悉 HTML 或 Windows 指令碼編寫的使用者，編寫巨集并不困難。對于不熟悉這些指令碼語言的使用者，EmEditor 也能讓您輕松地編輯巨集。EmEditor 能錄制鍵盤動作，并把它儲存到一個能隨時加載的巨集檔案中。通過 JavaScript 或 VBScript，您還能輕松地檢查代碼出錯的部分。例如，在 JavaScript 中，您能用下列語句來排查錯誤:

try { ... } catch(e) { ... }

當存在一個錯誤，例如無法打開檔案，try-catch 語句會允許指令碼繼續執行，而不是強制終止它。
