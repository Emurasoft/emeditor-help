# CharacterRanges 集合

CharacterRanges 集合提供 [**CharacterRange** 對象](../character_range/index) 的集合。

## 屬性

|     |     |
| --- | --- |
| [Count](count) | 檢索項目的數目。 |
| [Item](item) | 為指定索引檢索 [**CharacterRange** 對象](../character_range/index)。 |

## 方法

|     |     |
| --- | --- |
| [AddCharacterRange](add_character_range) | 添加字元範圍。 |
| [AddString](add_string) | 添加一個字串。 |
| [Clear](clear) | 刪除集合中的所有項目。 |

## 版本

支持 EmEditor Professional Version 22.0 或之後的版本。
