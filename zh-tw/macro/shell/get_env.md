# GetEnv 方法 (Shell 對象)

檢索環境變數。

## 

### \[JavaScript\]

```
str = shell.GetEnv( strName );
```

### \[VBScript\]

```
str = shell.GetEnv( strName )
```

## 參數

_strFile_

環境變數的名稱。

## 範例

### \[JavaScript\]

```
str = shell.GetEnv( "OPENAI_API_KEY" );
```

### \[VBScript\]

```
str = shell.GetEnv( "OPENAI_API_KEY" )
```

## 返回值

返回指定的環境變數的內容。

## 版本

支持 EmEditor Professional 24.0 或之後的版本。
