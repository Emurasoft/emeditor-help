# FontProp 對象

## 屬性

|     |     |
| --- | --- |
| [DisplayList](display_list) | 檢索 FontList 對象。 |
| [PrintList](print_list) | 檢索 PrintList 對象。 |
| [SameFontPrint](same_font_print) | 與自訂字型對話方塊中的 [**將顯示字型作為印表機字型** 核取方塊](../../dlg/properties/font/index) 相對應。 |
| [UsePrinterFont](use_printer_font) | 與自訂字型對話方塊中的 [**在下方的清單中包含預設的印表機字型** 核取方塊](../../dlg/properties/font/index) 相對應。 |

## 版本

支持 EmEditor 7.00 或之後的版本。


```{toctree}
:maxdepth: 1
display_list
print_list
same_font_print
use_printer_font
```
