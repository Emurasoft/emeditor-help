# Enabled 屬性 (LanguageServerProp 對象)

與**啟用語言伺服器協議**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.Enabled;
object.Enabled = b;
```

### \[VBScript\]

```
b = object.Enabled
object.Enabled = b
```

## 版本

支持 EmEditor Professional 版 23.00 或之後的版本。
