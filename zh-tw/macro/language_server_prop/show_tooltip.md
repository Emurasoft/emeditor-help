# ShowTooltip 屬性 (LanguageServerProp 對象)

與**顯示停留工具提示**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.ShowTooltip;
object.ShowTooltip = b;
```

### \[VBScript\]

```
b = object.ShowTooltip
object.ShowTooltip = b
```

## 版本

支持 EmEditor Professional 版 23.0 或之後的版本。
