# Type 屬性 (LanguageServerProp 對象)

與**文檔類型**下拉清單方塊相對應，該清單方塊在在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.Type;
object.Type = b;
```

### \[VBScript\]

```
b = object.Type
object.Type = b
```

## 版本

支持 EmEditor Professional 版 23.00 或之後的版本。
