# Type 屬性 (LanguageServerProp 對象)

與**文檔類型**下拉清單方塊相對應，該清單方塊在在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
type = object.Type;
object.Type = type;
```

### \[VBScript\]

```
type = object.Type
object.Type = type
```

## 備註

Type 屬性應該是以下值之一。

|     |     |
| --- | --- |
| eeTypeCSS | CSS |
| eeTypeCpp | C++ |
| eeTypeHTML | HTML |
| eeTypeJS | JavaScript |
| eeTypeJSON | JSON |
| eeTypePerl | Perl |
| eeTypePython | Python |


## 版本

支持 EmEditor Professional 版 23.0 或之後的版本。
