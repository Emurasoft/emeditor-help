# ShowCompletionList 屬性 (LanguageServerProp 對象)

與**顯示完成清單**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言服務器**頁面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.ShowCompletionList;
object.ShowCompletionList = b;
```

### \[VBScript\]
```
b = object.ShowCompletionList
object.ShowCompletionList = b
```

## 版本

支持 EmEditor Professional 版 23.1 或之後的版本。
