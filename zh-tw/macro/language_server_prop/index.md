# LanguageServerProp 對象

## 屬性

|     |     |
| --- | --- |
| **[Enabled](enabled)** | 與**啟用語言伺服器協議**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。 |
| **[ShowTooltip](show_tooltip)** | 與**顯示停留工具提示**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。 |
| **[ShowCompletionList](show_completion_list)** | 與**顯示完成清單**核取方塊相對應，該核取方塊在組態屬性對話方塊中的[**語言服務器**頁面](../../dlg/properties/language_server/index)上。 |
| **[Type](type)** | 與**文檔類型**下拉清單方塊相對應，該清單方塊在在組態屬性對話方塊中的[**語言伺服器**頁面](../../dlg/properties/language_server/index)上。 |

## 版本

支持 EmEditor Professional 版 23.0 或之後的版本。
