# CharacterRange 對象

## 屬性

|     |     |
| --- | --- |
| [Enabled](enabled) | 指定是否啟用對象的標志。 |
| [Min](min) | 指定對象的最小字元。 |
| [Max](max) | 指定對象的最大字元。 |
| [Properties](properties) | 指定表示對象的 Unicode 腳本和/或 Unicode 常規類別屬性的字串。 |
| [TreatAs](treat_as) | 指定要取代的字元。 如果指定-1，則在用於模糊比對時將忽略指定的範圍。 |

## 版本

支持 EmEditor Professional Version 22.0 或之後的版本。
