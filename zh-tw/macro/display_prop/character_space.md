# CharacterSpace 屬性 (DisplayProp 對象)

與組態屬性中的 [顯示 頁面](../../dlg/properties/display/index) 上的字元間距 文字方塊相對應。

## 

### \[JavaScript\]

```
n = object.CharacterSpace;
object.CharacterSpace = n;
```

### \[VBScript\]

```
n = object.CharacterSpace
object.CharacterSpace = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
