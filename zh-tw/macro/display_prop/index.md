# DisplayProp 對象

## 屬性

|     |     |
| --- | --- |
| [CharacterSpace](character_space) | 與在 **組態屬性** 對話方塊中 [**顯示** 頁面](../../dlg/properties/display/index) 上的 **字元間距** 文字方塊相對應。 |
| [ColorList](color_list) | 為色彩清單檢索 [**DisplayList** 對象](../display_item/index)。 |
| [LineSpace](line_space) | 與在 **組態屬性** 對話方塊中 [**顯示** 頁面](../../dlg/properties/display/index) 上的 **行間距** 文字方塊相對應。 |
| [QuoteMarks](quote_marks) | 與在 **組態屬性** 對話方塊中 [**顯示** 頁面](../../dlg/properties/display/index) 上的 **引用標記** 文字方塊相對應。 |
| [SearchColors](search_colors) | 與在 **組態屬性** 對話方塊中 [**顯示** 頁面](../../dlg/properties/display/index) 上的 **搜尋亮顯** 文字方塊相對應。 |
| [SearchList](search_list) | 為搜尋亮顯顏色清單檢索 [**HighlightList** 對象](../highlight_item/index)。 |

## 版本

支持 EmEditor 7.00 或之後的版本。
