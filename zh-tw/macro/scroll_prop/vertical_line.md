# VerticalLine 屬性 (ScrollProp 對象)

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **垂直線** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.VerticalLine ;
object.VerticalLine  = b;
```

### \[VBScript\]

```
b = object.VerticalLine
object.VerticalLine  = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
