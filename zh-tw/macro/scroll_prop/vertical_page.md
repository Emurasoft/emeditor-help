# VerticalPage 屬性 (ScrollProp 對象)

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **總是啟用 1 頁垂直捲動** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.VerticalPage ;
object.VerticalPage  = b;
```

### \[VBScript\]

```
b = object.VerticalPage
object.VerticalPage  = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
