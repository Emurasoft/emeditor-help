# DoubleLineScroll 屬性 (ScrollProp 對象)

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **每次捲動兩行** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.DoubleLineScroll;
object.DoubleLineScroll = b;
```

### \[VBScript\]

```
b = object.DoubleLineScroll
object.DoubleLineScroll = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
