# HorizontalBar 屬性 (ScrollProp 對象)

與組態屬性中 [捲動 頁面](../../dlg/properties/scroll/index) 上的水平捲軸 下拉清單方塊相對應。

## 

### \[JavaScript\]

```
n = object.HorizontalBar;
object.HorizontalBar = n;
```

### \[VBScript\]

```
n = object.HorizontalBar
object.HorizontalBar = n
```

## 版本

支持 EmEditor 7.00 或之後的版本。
