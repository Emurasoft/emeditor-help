# FastCursorMove 屬性 (ScrollProp 對象)

與組態屬性中 [捲動 頁面](../../dlg/properties/scroll/index) 上的快速移動游標 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.FastCursorMove;
object.FastCursorMove = b;
```

### \[VBScript\]

```
b = object.FastCursorMove
object.FastCursorMove = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
