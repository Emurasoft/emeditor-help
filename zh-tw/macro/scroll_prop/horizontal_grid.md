# HorizontalGrid 屬性 (ScrollProp 對象)

與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **水平格線** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.HorizontalGrid;
object.HorizontalGrid = b;
```

### \[VBScript\]

```
b = object.HorizontalGrid
object.HorizontalGrid = b
```

## 版本

支持 EmEditor 10.00 或之後的版本。
