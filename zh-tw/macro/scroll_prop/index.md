# ScrollProp 對象

## 屬性

|     |     |
| --- | --- |
| [DoubleLineScroll](double_line_scroll) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **每次捲動兩行** 核取方塊相對應。 |
| [FastCursorMove](fast_cursor_move) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **快速移動游標** 核取方塊相對應。 |
| [HalfPage](half_page) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **捲動半頁** 核取方塊相對應。 |
| [HorizontalBar](horizontal_bar) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **水平捲軸** 下拉清單方塊相對應。 |
| [HorizontalGrid](horizontal_grid) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **水平格線** 核取方塊相對應。 |
| [HorizontalLine](horizontal_line) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **水平線** 核取方塊相對應。 |
| [HorizontalPage](horizontal_page) | Corresponds to the **總是啟用 1 頁水平捲動** 核取方塊相對應。 |
| [MoveCursorScroll](move_cursor_scroll) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **通過捲動位置移動游標** 核取方塊相對應。 |
| [ShowOnlyActive](show_only_active) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **只在活動窗格中顯示捲軸** 核取方塊相對應。 |
| [Smooth](smooth) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **平滑捲動** 核取方塊相對應。 |
| [SmoothSpeed](smooth_speed) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **平滑捲動滑塊** 相對應。 |
| [VerticalBar](vertical_bar) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **垂直捲軸** 下拉清單方塊相對應。 |
| [VerticalLine](vertical_line) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **垂直線** 核取方塊相對應。 |
| [VerticalPage](vertical_page) | 與組態屬性中 [**捲動** 頁面](../../dlg/properties/scroll/index) 上的 **總是啟用 1 頁垂直捲動** 核取方塊相對應。 |
