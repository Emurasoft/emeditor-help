# DisplayItem 對象

## 屬性

|     |     |
| --- | --- |
| [BackColor](back_color) | 指定對象的背景顏色。 |
| [BackMode](back_mode) | 指定背景顏色模式。 |
| [LineColor](line_color) | 指定對象的下劃線顏色。 |
| [Style](style) | 指定對象的樣式。 |
| [TextColor](text_color) | 指定對象的文字顏色。 |
| [TextMode](text_mode) | 指定文字顏色模式。 |

## 版本

支持 EmEditor 7.00 或之後的版本。
