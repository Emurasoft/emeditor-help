# KinsokuBegin 屬性 (WrapProp 對象)

與組態屬性中 [換行 頁面](../../dlg/properties/wrap/index) 上的行首不允許換行字元 文字方塊相對應。

## 

### \[JavaScript\]

```
str = object.KinsokuBegin;
object.KinsokuBegin = str;
```

### \[VBScript\]

```
str = object.KinsokuBegin
object.KinsokuBegin = str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
