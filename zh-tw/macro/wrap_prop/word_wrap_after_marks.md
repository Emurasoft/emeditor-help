# WordWrapAfterMarks 屬性 (WrapProp 對象)

與組態屬性中 [**換行** 頁面](../../dlg/properties/wrap/index) 上的 **啟用標記后自動換行** 核取方塊相對應。

## 

### \[JavaScript\]

```
str = object.WordWrapAfterMarks;
object.WordWrapAfterMarks = str;
```

### \[VBScript\]

```
str = object.WordWrapAfterMarks
object.WordWrapAfterMarks = str
```

## 版本

支持 EmEditor 7.00 或之後的版本。
