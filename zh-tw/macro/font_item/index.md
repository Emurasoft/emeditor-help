# FontItem 對象

## 屬性

|     |     |
| --- | --- |
| [Bold](bold) | True 如果字型的格式被設置為粗體。 |
| [Italic](italic) | True 如果字型的格式被設置為斜體。 |
| [Name](name) | 指定對象的名稱。 |
| [Script](script) | 指定字型的指令碼。 |
| [Size](size) | 用點數(pt)指定字型大小。 |

## 版本

支持 EmEditor 7.00 或之後的版本。

```{toctree}
:hidden:
:maxdepth: 1

```
