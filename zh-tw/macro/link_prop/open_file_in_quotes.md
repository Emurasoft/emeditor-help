# OpenFileInQuotes 屬性 (LinkProp 對象)

與組態屬性中 [**連結** 頁面](../../dlg/properties/link/index) 上的 **點擊引號內的檔案名打開檔案** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.OpenFileInQuotes;
object.OpenFileInQuotes = b;
```

### \[VBScript\]

```
b = object.OpenFileInQuotes
object.OpenFileInQuotes = b
```

## 版本

支持 EmEditor 13 或之後的版本。
