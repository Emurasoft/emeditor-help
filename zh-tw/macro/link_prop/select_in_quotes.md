# SelectInQuotes 屬性 (LinkProp 對象)

與組態屬性中 [連結 頁面](../../dlg/properties/link/index) 上的點擊引號內的文字選擇整個字串 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.SelectInQuotes;
object.SelectInQuotes = b;
```

### \[VBScript\]

```
b = object.SelectInQuotes
object.SelectInQuotes = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
