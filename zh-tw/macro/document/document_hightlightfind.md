# HighlightFind 屬性 (Document 對象)

決定是否要高亮搜尋字符串。

## 

### \[JavaScript\]

```
bHighlightFind = document.HighlightFind;
document.HighlightFind = bHighlightFind;
```

### \[VBScript\]

```
bHighlightFind = document.HighlightFind
document.HighlightFind = bHighlightFind
```

## 版本

支持 EmEditor 4.00 或之後的版本。
