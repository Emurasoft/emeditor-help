# RearrangeColumns 方法 (Document 對象)

重排 CSV 欄。

## 

### \[JavaScript\]

```
document.RearrangeColumns( strColumns );
```

### \[VBScript\]

```
document.RearrangeColumns strColumns
```

## 參數

_strColumns_

指定一個字串來指示要重新排列的欄的順序。例如，"1, 3, 5" 表示結果將包括原始 CSV 文檔的第一列，第三列，和第五列。

## 版本

支持 EmEditor Professional v22.1 或之後的版本。
