# FullName 屬性 (Document 對象)

檢索文檔的完整路徑以及檔案名稱。

## 

### \[JavaScript\]

```
strName = document.FullName;
```

### \[VBScript\]

```
strName = document.FullName
```

## 示例

### \[JavaScript\]

```
alert( document.FullName );
```

### \[VBScript\]

```
alert document.FullName
```

## 版本

支持 EmEditor 4.00 或之後的版本。
