# Encoding 屬性

檢索或設置打開的檔案的當前編碼。

#### \[JavaScript\]

_nEncoding_ = document. **Encoding**;

document. **Encoding** = _nEncoding_;

#### \[VBScript\]

_nEncoding_ = document. **Encoding**

document. **Encoding** = _nEncoding_

## 參數

_nEncoding_

從 **[編碼常數](../const/const_encoding)** 中選擇或指定任何用於 Windows 操作系統的代碼頁。

## 版本

支持 EmEditor 4.00 或之後的版本。
