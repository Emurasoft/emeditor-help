# Encoding 屬性 (Document 對象)

檢索或設置打開的檔案的當前編碼。

## 

### \[JavaScript\]

```
nEncoding = document.Encoding;
document.Encoding = nEncoding;
```

### \[VBScript\]

```
nEncoding = document.Encoding
document.Encoding = nEncoding
```

## 參數

_nEncoding_

從 **[編碼常數](../const/const_encoding)** 中選擇或指定任何用於 Windows 操作系統的代碼頁。

## 版本

支持 EmEditor 4.00 或之後的版本。
