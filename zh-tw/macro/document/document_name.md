# Name 屬性

檢索不帶路徑的文檔的檔案名，或重新命名文檔的檔案名。如果文檔沒有標題，則重新命名文檔標題而不儲存檔案。

#### \[JavaScript\]

_strName_ = document. **Name**;document. **Name** = _strName_;

#### \[VBScript\]

_strName_ = document. **Name** document. **Name** = _strName_

## 示例

#### \[JavaScript\]

alert( document.Name );

#### \[VBScript\]

alert document.Name

## 版本

支持 EmEditor 4.00 或之後的版本。
