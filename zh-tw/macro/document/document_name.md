# Name 屬性 (Document 對象)

檢索不帶路徑的文檔的檔案名，或重新命名文檔的檔案名。如果文檔沒有標題，則重新命名文檔標題而不儲存檔案。

## 

### \[JavaScript\]

```
strName = document.Name;
document.Name = strName;
```

### \[VBScript\]

```
strName = document.Name 
document.Name = strName
```

## 示例

### \[JavaScript\]

```
alert( document.Name );
```

### \[VBScript\]

```
alert document.Name
```

## 版本

支持 EmEditor 4.00 或之後的版本。
