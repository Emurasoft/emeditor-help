# RenameIfSameNameExists 屬性 (BackupProp 對象)

與組態屬性中 [備份 頁面](../../dlg/properties/backup/index) 上的重新命名檔案，檔案名已存在 核取方塊對應。

## 

### \[JavaScript\]

```
b = object.RenameIfSameNameExists;
object.RenameIfSameNameExists = b;
```

### \[VBScript\]

```
b = object.RenameIfSameNameExists
object.RenameIfSameNameExists = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
