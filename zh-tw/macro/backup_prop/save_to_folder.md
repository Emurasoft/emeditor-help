# SaveToFolder 屬性

與組態屬性中 [**備份** 頁面](../../dlg/properties/backup/index) 上的 **儲存備份檔案到備份資料夾** 核取方塊對應。

#### \[JavaScript\]

_b_ = object. **SaveToFolder**;

object. **SaveToFolder** = _b_;

#### \[VBScript\]

_b_ = object. **SaveToFolder**

object. **SaveToFolder** = _b_

## 版本

支持 EmEditor 7.00 或之後的版本。
