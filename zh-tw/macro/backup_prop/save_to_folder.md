# SaveToFolder 屬性 (BackupProp 對象)

與組態屬性中 [備份 頁面](../../dlg/properties/backup/index) 上的儲存備份檔案到備份資料夾 核取方塊對應。

## 

### \[JavaScript\]

```
b = object.SaveToFolder;
object.SaveToFolder = b;
```

### \[VBScript\]

```
b = object.SaveToFolder
object.SaveToFolder = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
