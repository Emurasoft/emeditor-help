# SaveToFolder 屬性 (AutoSaveProp 對象)

與組態屬性中 [**自動儲存** 頁面](../../dlg/properties/autosave/index) 上的 **儲存到自動儲存資料夾** 核取方塊相對應。

## 

### \[JavaScript\]

```
b = object.SaveToFolder;
object.SaveToFolder = b;
```

### \[VBScript\]

```
b = object.SaveToFolder
object.SaveToFolder = b
```

## 版本

支持 EmEditor 7.00 或之後的版本。
