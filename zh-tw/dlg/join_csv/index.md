# 「合併 CSV」對話方塊

當您選擇工具列上的「合併 CSV」按鈕時，該對話方塊會出現。

### 「CSV 文檔 1」下拉清單方塊

選擇第一個文檔。

### 「CSV 文檔 2」下拉清單方塊

選擇第二個文檔。兩個 CSV 文檔會按索引鍵資料欄中指定的共同的索引鍵資料欄值來合併。

### 「Key1/Key2 資料欄」下拉清單方塊

選擇索引鍵資料欄。

### 「唯一索引鍵」核取方塊

如果鍵列的每一行都包含一個唯一鍵，那么這個核取方塊能被用來加快執行合併命令的速度。

### 「包含所有不符合的資料列」核取方塊

當合併兩個文檔時，包含所有不符合的資料列。這等同于 SQL 中的 OUTER JOIN 如果兩個核取方塊都勾選的話；如果只勾選左邊的核取方塊，它的作用等同于 LEFT JOIN；如果只勾選右邊的核取方塊，那么就是 RIGHT JOIN；兩個核取方塊都不勾選的話，就是 INNER JOIN。

### 「忽略標題」核取方塊

如果勾選了該核取方塊，會忽略在索引鍵資料欄中的標題行，這樣原本的標題行能在合併後的文檔中保留。

### 「條件」下拉清單方塊

指定一個條件。

### 「大小須相符」核取方塊

如果勾選了該核取方塊，EmEditor 會在比較索引鍵資料欄中的值時，區分大小寫。

### 「分隔符號」文字方塊

如果選擇了 _兩個分割的字串都符合_， _Key1 符合分割的 Key2_，或 _Key2 符合分割的 Key1_ 為條件，則輸入分割字串的分隔符。

### 「限制分割」核取方塊和文字方塊

指定最多分割次數。

### 「CSV 文檔/列」清單方塊

這個清單會顯示兩個文檔中所有的列。選取的列會按此順序顯示在輸出中。

### 「全部啟用/禁用」核取方塊

這個核取方塊讓你能啟用或禁用清單中的所有條目。

### 「上移」按鈕

把選取的條目往清單的上方移動。

### 「下移」按鈕

把選取的條目往清單的下方移動。

### 「頂部」按鈕

把所選的條目移動到清單的頂部。

### 「底部」按鈕

把所選的條目移動到清單的底部。

### 「與上一欄合併」核取方塊

這個核取方塊讓你能與上一欄合併。

### 「串連」按鈕

如果勾選了此核取方塊，EmEditor 會將目前的欄與上一欄串連。

### 「使用第一個非空白值」按鈕

如果勾選了此核取方塊，EmEditor 會用第一個非空白值將目前的欄與上一欄合併。

### 「匯出」按鈕

把目前的設定匯出到一個 JavaScript 或 VBScript 巨集檔案中。一個被匯出的巨集檔案能被用來運行與設定同樣的操作。

### 「立即合併」按鈕

合併文檔。
