# 「輸入數字範圍」對話方塊

此對話方塊讓你能創建一個 [數字範圍運算式](../../howto/search/number_range_syntax)。 此對話方塊會出現當你在 **尋找**， **取代**， **多檔尋找**， **多檔取代**，或 **進階篩選** 對話方塊中點選了 **數字範圍** 選項按鈕。

## 

### 「最小值」文字方塊

指定一個數字範圍的最小值。如果勾選了 **僅整數值** 核取方塊，此數值將不包括小數。

### 「包括最小值」核取方塊

指定最小值是否包括在數字範圍內。

### 「最大值」文字方塊

指定一個數字範圍的最大值。如果勾選了 **僅整數值** 核取方塊，此數值將不包括小數。

### 「包括最大值」核取方塊

指定最大值是否包括在數字範圍內。

### 「遞增值+最小值」核取方塊

指定僅與最小值加上指定整數的倍數符合的數字。

### 「遞增值」核取方塊

如果勾選了 **遞增值+最小值** 核取方塊，為之指定一個數值。

### 「整數」核取方塊

指定是否僅與整數相符。

### 「小數位數」選項按鈕

指定與小數位數相符。

### 「使用預設的小數點」核取方塊

指定使用從目前的語言環境派生的小數點。如果未設定，則應在下面的下拉清單方塊中指定小數點字元。

### 「IP 地址」選項按鈕

指定與 IP 地址相符。

### 「IP 地址/CIDR 標記」核取方塊

選擇將 IP 地址範圍指定為 IP 地址/CIDR 標記。如果勾選了此核取方塊，則應在下面的文字方塊中指定正確的 IPv4/IPv6 地址/CIDR 標記。例如，如果要搜索從 192.168.0.0 到 192.168.255.255 的 IPv4 地址，就要指定 "192.168.0.0/16" 。

### 「日期/時間」選項按鈕

指定與日期和/或時間相符。

### 「使用預設的日期格式」核取方塊

指定使用從目前的語言環境派生的日期格式。如果未勾選，則要在核取方塊下面的下拉清單方塊中指定日期和/或時間格式。

### 「>」按鈕

單擊此按鈕可顯示可以使用的日期/時間格式。

### 「前置詞」文字方塊

指定在比對值之前相符的字串。該欄位可以為空。

### 「後置詞」文字方塊

指定在比對值之後相符的字串。該欄位可以為空。

### 「下次跳過此對話方塊」核取方塊

指定下次是否應不顯示此對話方塊直接跳到 **尋找** 或 **篩選** 下拉清單方塊中。
