# 「檔案」頁面

**檔案** 頁面讓您能設置與檔案操作相關的屬性。

## 「如果發現空字元時進行提示」核取方塊

顯示警示消息當打開的檔案包含 [空字元](../../../glossary/nullcharacter) 時。當一個檔案打開后，空字元會轉換為空白。您必須小心因為包含空字元的檔案可能損壞原始的檔案。

## 「如果發現無效字元時進行提示」核取方塊

當打開包含無法轉換為指定編碼的 Unicode 字元的檔案時，會顯示一條警示消息「一些字元無法用指定的編碼被轉換。檔案將被損壞如果您繼續編輯并儲存檔案。」例如，如果您試著用日文 Shift-JIS 編碼打開一個日文 EUC 檔案，并且檔案包含的字元代碼值不在日文 Shift-JIS 編碼範圍內，就會出現這個警示消息。

**提示**

即使該核取方塊被勾選了，一些編碼不會顯示該警示消息。例如，當用日文 EUC 編碼試著打開一個日文 Shift-JIS 檔案時，將不會顯示該消息。

## 「換行方式不一致時進行提示」核取方塊

顯示警示消息如果要打開的檔案的換行方式不一致。

## 「除了二進位檔案」核取方塊

當編碼是二進位 (ASCII 視圖) 時，停用 **換行方式不一致時進行提示** 選項。

## 「顯示包含完整路徑的檔案名」核取方塊

在 EmEditor 視窗標題上用完整路徑顯示檔案名。

## 「失去焦點時不顯示完整路徑」核取方塊

將不會在 EmEditor 視窗標題上顯示有完整路徑的檔案名稱，除非焦點回到視窗上。

## 「創建與記事本兼容的日志檔案」核取方塊

啟用與記事本相似的日志功能。如果您在第一行用 .LOG 儲存，每次您打開檔案時，它都會自動在文檔末尾插入時間與日期。

## 「偵測 HTML/XML 字元集」核取方塊

在 HTML 檔案中，尋找 "charset=..."，在 XML 檔案中，"encoding=..."或者在 Python 和 Ruby 檔案中，尋找"#coding=..."并使用相對應的編碼。如果您打開 HTML，XML，Python，或者 Ruby 檔案，請只勾選這個選項并清除其他的選項。

## 「偵測全部」核取方塊

從所有可用的代碼頁中統計偵測編碼頁。這個偵測可能會出現失誤，特別當檔案很小時。這個參數并不包括 HTML/XML 字元集偵測。

## 「偵測 Unicode 簽名 (BOM)」核取方塊

偵測檔案是否包含一個位元順序標記 (BOM) 簽名；如果偵測到 BOM 簽名，EmEditor 會把檔案作為 UTF-16LE，UTF-16BE，或 UTF-8 檔案打開。

## 「偵測 UTF-8」核取方塊

統計偵測 UTF-8。

## 「優先選擇 UTF-8」核取方塊

當勾選「偵測 UTF-8」核取方塊以及這個核取方塊時，如果一個檔案能同時被 UTF-8 和選擇的編碼解讀，UTF-8 會被優先選擇。例如，一個只包含 ANSI 字元 (字母以及數字) 的檔案，它可以用 UTF-8 以及西歐文編碼方式打開。在這個情況下，EmEditor 會把檔案作為 UTF-8 打開。EmEditor 還會用 UTF-8 編碼打開一個空檔案如果該選項被勾選的話。

## 「要偵測的 CSV 檔案」清單方塊

當 EmEditor 打開一個檔案時，偵測 CSV 格式檢查器。CSV 格式能在 **自訂** 對話方塊中的 [**CSV** 頁面](../../customize/csv/index) 上被定義。

為了被偵測到，CSV 檔案必須至少有兩行，并且檔案中的每行有相同數目的分隔符。

## 「檔案打開時的編碼」下拉清單方塊

選擇用來打開的編碼。

## 「被其他程式更改時」下拉清單方塊

選擇動作當您正在編輯的檔案被其他程式修改時。

|     |     |
| --- | --- |
| 忽略 | 不重新載入檔案如果檔案被另一個程式修改。這個選項不會改變 EmEditor 的唯讀狀態即使檔案的唯讀屬性被改變。 |
| 提示 | 顯示一個消息方塊來詢問是否可以重新載入檔案。這個選項允許改變 EmEditor 的唯讀狀態當檔案的唯讀屬性被改變。 |
| 自動重新載入 | 總是重新載入檔案。這個選項允許改變 EmEditor 的唯讀狀態當檔案的唯讀屬性被改變。 |
| 保持鎖定 | 僅用 EmEditor 打開檔案并禁止另一個程式編寫該檔案當檔案打開時。這個選項允許改變 EmEditor 的唯讀狀態當檔案的唯讀屬性被改變。 |

## 「預設副檔名」文字方塊

如果您用一個新的名稱儲存檔案，并在 **檔案類型** 組合方塊中選擇了 **所有檔案 (\*.\*)**，但沒有指定檔案副檔名，那么在該文字方塊中指定的副檔名將會成為該檔案的副檔名。

## 「新增檔案」按鈕

點擊該按鈕來顯示 [**新增檔案詳細信息** 對話方塊](new_details/index)，讓您能組態編碼以及其他設定當您創建一個新檔案時。

## 「儲存中」按鈕

點擊該按鈕來顯示 [**儲存詳細信息** 對話方塊](save_details/index)，讓您能組態編碼以及其他設定當您儲存檔案時。

## 「重設」按鈕

重設為預設設定。 [**重設** 對話方塊](../reset/index) 會被顯示并讓您能從另一個組態複製設定。
