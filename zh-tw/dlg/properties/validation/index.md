# 「語法檢查」頁面

**語法檢查** 頁面讓你能設置與語法檢查相關聯的屬性。

### 「啟用語法檢查」核取方塊

如果勾選了該核取方塊，語法檢查錯誤會以紅色波浪下劃線（可自訂）顯示，並且游標指針懸停在錯誤部分會顯示有關錯誤信息的小提示。

### 「文檔類型」下拉清單方塊

指定要檢查語法的文檔類型。請註意，如果選擇了 **CSV**，則僅在 CSV 模式下時才能啟用語法檢查。如果選擇了 **語言伺服器協議**，將會使用在 [**語言伺服器** 頁面](../language_server/index) 上指定的 **文檔類型**。

### 「顯示清單」下拉清單方塊

|     |     |
| --- | --- |
| （無） | 語法檢查結果不會顯示為清單。即使結果不顯示為清單，如果勾選了 **啟用語法檢查** 核取方塊，語法錯誤也會以紅色波浪下劃線表示。 |
| 在打開一個文檔時 | 打開此組態的文檔時，語法檢查結果將顯示為清單。 |
| 僅在檢測到錯誤時 | 僅當打開此組態的文檔時檢測到錯誤時，語法檢查結果才會顯示為清單。 |

### 「重置」按鈕

重置為預設設定。會顯示 [**重置** 對話方塊](../reset/index) 並讓你能從另一個組態復制設定。
