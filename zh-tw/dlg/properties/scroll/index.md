# 「捲動」頁面

**捲動** 頁面讓您能設置與捲動操作相關的屬性。

### 「水平捲軸」下拉清單方塊

指定如何顯示水平捲軸。

|     |     |
| --- | --- |
| **不顯示** | 不顯示水平捲軸。 |
| **僅在需要時顯示** | 顯示水平捲軸僅當一行的長度大于視窗寬度時。 |
| **總是顯示** | 總是顯示水平捲軸。 |

### 「垂直捲軸」下拉清單方塊

指定如何顯示垂直捲軸。

|     |     |
| --- | --- |
| **不顯示** | 不顯示垂直捲軸。 |
| **僅在需要時顯示** | 顯示垂直捲軸僅當行的總數大于目前的視窗可以顯示的總數時。 |
| **總是顯示** | 總是顯示垂直捲軸。 |

### 「平滑捲動」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會平穩地捲動。

### 「速度」滑桿

使用該滑桿來調整平滑捲動的速度。

### 「總是啟用 1 頁水平捲動」核取方塊

總是啟用 1 頁水平捲動。

### 「總是啟用 1 頁垂直捲動」核取方塊

總是啟用 1 頁垂直捲動。

### 「水平線」核取方塊

在游標處顯示水平線。

### 「垂直線」核取方塊

在游標處顯示垂直線。

### 「只在活動窗格中顯示捲軸」核取方塊

只有當在一個分割的視窗中的目前的窗格是活動的時候，才顯示捲軸。

### 「每次捲動兩行」核取方塊

一次捲動 2 行。勾選該核取方塊如果您想要更快地捲動視窗。

### 「快速移動游標」核取方塊

讓游標移動更快。如果該核取方塊被勾選了，游標移動速度加快兩被當按住鍵盤上的箭頭鍵來移動游標時。

#### **提示:** 您也可以到 **控制面板** 上的 **鍵盤** 屬性中的 **速度** 頁面上調整鍵盤重複速率。

### 「通過捲動位置移動游標」核取方塊

如果該核取方塊被勾選了，游標會被重新定位當水平或垂直捲軸位置變更時。

### 「捲動半頁」核取方塊

通常是捲動一整頁，但勾選該核取方塊后，僅會捲動半頁。

### 「水平格線」核取方塊

顯示水平格線。

### 「總是顯示游標上/下1行」核取方塊

總是顯示游標所在行的上一行以及下一行當你上下移動游標時。

### 「3D 外觀」核取方塊

如果勾選了該核取方塊，捲軸會有 3D 框線。

### 「在垂直捲軸上顯示標記」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸上顯示標記。

### 「在垂直捲軸上顯示標記」群組方塊 \- 「更改」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸上標記出更改過的或更改后儲存過的行。

### 「在垂直捲軸上顯示標記」群組方塊 \- 「標記」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸上標記出書籤位置以及當您比較兩個檔案時出現的差異。

### 「在垂直捲軸上顯示標記」群組方塊 \- 「找到的字串」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸上標記出與搜尋字串相符合的行。

### 「在垂直捲軸上顯示標記」群組方塊 \- 「游標位置」核取方塊

如果該核取方塊被勾選了， EmEditor 視窗會在垂直捲軸上標記出游標位置。

### 「迷你地圖」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸旁顯示文檔的迷你地圖。

### 「停留時顯示迷你地圖」核取方塊

如果勾選了該核取方塊，EmEditor 視窗會在滑鼠停留在垂直捲軸上時顯示迷你地圖。

### 「顯示整個地圖」核取方塊

如果該核取方塊被勾選了，EmEditor 視窗會在垂直捲軸旁顯示整個地圖。

### 「縮放」滑桿

使用該滑桿來調整縮放迷你地圖。

### 「比例」滑桿

使用該滑桿來調整迷你地圖的比例。

### 「顯示箭頭」核取方塊

如果勾選了該核取方塊，在迷你地圖頂部和底部，會顯示向上以及向下箭頭。

### 「重設」按鈕

重設為預設設定。 [**重設** 對話方塊](../reset/index) 會被顯示并讓您能從另一個組態複製設定。
