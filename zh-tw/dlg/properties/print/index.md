# 「列印」頁面

列印 頁面讓您能設置與列印操作相關的屬性。

## 「列印行號」核取方塊

在頁面的左邊列印行號。

## 「忽略顏色與下劃線」核取方塊

當列印時，忽略顏色與下劃線。

## 「插入分隔線」核取方塊

在頁眉與文字之間以及文字與頁腳之間插入水平線。

## 「列印時按頁面換行」核取方塊

列印時按頁面寬度換行。

## 「行距」文字方塊

指定列印時行與行之間的距離。這個值對應于一個單一的行高度的 1/10。

## 「頁眉」核取方塊

啟用指定的頁眉。

## 「頁眉」文字方塊

用下列標記在頁面頂部列印文檔信息。

## 「>」按鈕

用該按鈕從預設的標記清單中選取。

|     |     |
| --- | --- |
| &f | 檔案名稱 |
| &n | 路徑名稱 |
| &p | 頁號 |
| &a | 總頁數 |
| &d | 今天的日期 |
| &t | 目前的時間 |
| &c | 居中對齊 |
| &r | 右對齊 |
| &e | 上次修改日期 |
| &u | 上次修改時間 |

## 「頁腳」核取方塊

啟用指定的頁腳。

## 「頁腳」文字方塊

用下列標記在頁面底部列印文檔信息。

## \> button

Use this button to select from a list of predefined marks.

|     |     |
| --- | --- |
| &f | 檔案名稱 |
| &n | 路徑名稱 |
| &p | 頁號 |
| &a | 總頁數 |
| &d | 日期 |
| &t | 時間 |
| &c | 居中對齊 |
| &r | 右對齊 |

## 「頁面設置」按鈕

用該按鈕來指定頁面設定。

## 「字型」按鈕

指定用來列印的字型。

## 「重設」按鈕

重設為預設設定。
[重設 對話方塊](../reset/index) 讓您能從另一個組態中複製設定。
