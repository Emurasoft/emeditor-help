# 「鍵盤」頁面

**鍵盤** 頁面讓您能設置與基本操作相關的屬性。

### 「類別」下拉清單方塊

命令的可用類別。

### 「命令」下拉清單方塊

在選取的類別中可用的命令。Commands available for the selected category.

### 「<」按鈕

跳回到之前選取的命令處。

### 「按下新的快速鍵」文字方塊

為選取的命令按下新的快速鍵。

### 「目前的已被分配到」文字

顯示剛按下的快速鍵目前被分配的命令。

### 「>」按鈕

跳轉到與目前的已被分配的鍵相關的命令上。

### 「目前的鍵」清單方塊

被分配給選取命令的目前的鍵。

### 「描述」文字方塊

選取命令的描述。

### 「分配」按鈕

分配快速鍵給目前的選取的命令。

### 「刪除」按鈕

刪除目前的被選取的快速鍵。

### 「重設」按鈕

重設為預設設定。會顯示 [**重設** 對話方塊](../reset/index) 并讓您能從另一個組態複製設定。
