# 匯入匯出精靈

這個精靈會出現當您執行 [**匯入匯出** 命令](../../cmd/tools/import_export) 時。您能選擇從一個檔案中匯入設定或匯入目前的設定到一個檔案中。

有關精靈中每一個控制項 (例如，核取方塊或文字方塊) 的更多信息，請參考下面的控制項清單，或按 **F1** 鍵當選取一個控制項時。

### 「請選擇要執行的操作」清單方塊

您可以從清單中選擇一個要執行的操作。

|     |     |
| --- | --- |
| **匯出所有設置到一個注冊表檔案中。** | 匯出所有自訂的設置，包括組態，功能表，工具列，外部工具以及最近使用的檔案清單到一個注冊表檔案中。以后，您用這個檔案匯入來還原目前的的設定，或複製該設定到另一臺電腦上。 |
| **匯出所有設置到 INI 檔案中。** | 匯出自訂的設置以及全局設置到 INI 檔案中。這個選項會被禁用如果 INI 檔案已經存在在 EmEditor 安裝資料夾中。 |
| **從一個注冊表檔案中匯入所有設置。** | 讀取通過選擇 **匯出所有設置到一個檔案中** 來儲存設置的檔案。您能用這個選項來還原之前儲存的設定。 |
| **設置一個使用預設值的可移動驅動器 (例如: USB 驅動器)。** | 複製 EmEditor 檔案并用預設設定創建 INI 檔案到一個可移動驅動器中。這個選項會被禁用如果 INI 檔案已經存在在 EmEditor 安裝資料夾中。 |
| **設置一個可移動驅動器 (例如: USB 驅動器) 并將所有設置匯出為 INI 檔案。** | 複製 EmEditor 檔案并用目前的設定創建 INI 檔案到一個可移動驅動器中。這個選項會被禁用如果 INI 檔案已經存在在 EmEditor 安裝資料夾中。 |
| **從 INI 檔案匯入至注冊表。** | 從 INI 檔案匯入至注冊表。 |

**注意**

下列 INI 檔案會被創建當您選擇創建 INI 檔案的選項之一:

- eeCommon.ini
- eeConfig.ini
- eeLM.ini

當 EmEditor 開始并在它的安裝資料夾中尋找 eeUseIni.ini 時，EmEditor 會用這些 INI 檔案，而不是注冊表，并且不會儲存任何設定到注冊表中。
