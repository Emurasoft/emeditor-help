# 「註冊信息」對話方塊

**註冊信息**對話方塊會顯示有關您的裝置以及訂閱或註冊金鑰的信息。您還可以檢查您的訂閱或註冊金鑰是否有效。

您可以通過 **說明 > 註冊信息** 訪問此對話方塊。

## 信息文字

文字方塊中可能顯示以下信息。

- 第一行包含 「EmEditor Professional」 或 「EmEditor Free」 以及版本字串。
- 如果您已登錄，您的電子郵件地址將被顯示。
- 如果您尚未註冊，將顯示「未註冊」。
- 裝置 ID
- 如果您使用 Stripe 訂閱，則顯示您的訂閱 ID
- 列印您的裝置驗證結果。如果成功，將顯示「驗證成功」。如果不成功，將顯示診斷信息以說明解決問題。
    - 如果您需要支持，請[聯繫我們](https://zh-tw.emeditor.com/support/#contact)。

## 「登入/輸入註冊金鑰」按鈕

此按鈕會打開[註冊產品對話方塊](../regist/index)。

## 編輯裝置信息

此按鈕會打開 Emurasoft 客戶中心的編輯裝置頁面，您可以在那裡檢視有關您的裝置的更多信息。
