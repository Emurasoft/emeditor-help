# 「與 EmEditor 關聯」對話方塊

這個對話方塊會出現當您選擇 [**與 EmEditor 關聯** 命令](../../cmd/tools/file_associate)，或者在 [**自訂** 對話方塊](../customize/index) 中 [**檔案** 頁面](../customize/file/index) 上的 **\[與 EmEditor 關聯\]** 按鈕。您可以把檔案副檔名關聯到 EmEditor。這個對話方塊僅在以管理者權限登錄時可用。

## 「副檔名」清單方塊

目前的相關聯的檔案副檔名清單。

## 「關閉」按鈕

點擊該按鈕關閉對話方塊。

## 「添加」按鈕

點擊該按鈕添加一個新的項目到清單中。

## 「刪除」按鈕

點擊該按鈕從清單上刪除被選取的項目。

## 「屬性」按鈕

顯示或編輯清單中所選取的組態屬性。
