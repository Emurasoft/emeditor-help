# 「合併行」對話方塊

當你執行 [**合併行** 命令](../../cmd/csv/combine_lines) 時，這個對話方塊會出現。此命令合併垂直相鄰的重複儲存格。

## 「要檢查重複項的欄」清單方塊

顯示可以指定檢查重複項的欄的清單。

## 「忽略大小寫」核取方塊

如果勾選了此核取方塊，EmEditor 會在比較儲存格時忽略大小寫。

## 「要合併的欄」清單方塊

顯示要合併的欄的清單。

## 「要插入的字串」清單方塊

您可以指定合併行時要插入的字串。

## 「刪除空字串」核取方塊

如果勾選了該核取方塊，EmEditor 會在合併行時移除空字串。

## 「刪除重複字串」核取方塊

如果勾選了該核取方塊，EmEditor 會在合併行時移除重複的字串。

## 「排序字串」核取方塊

如果勾選了該核取方塊，EmEditor 會使用指定的排序條件對字串進行排序。

## 「如何排序」清單方塊

顯示已定義排序條件的清單。

|     |     |
| --- | --- |
| 按字母升序排序 | 按文字升序排序。 |
| 按字母降序排序 | 按文字降序排序。 |
| 按數字升序排序 | 按數字升序排序。 |
| 按數字降序排序 | 按數字降序排序。 |
| 按文字長度從短到長排序 | 按文字長度升序排序。 |
| 按文字長度從長到短排序 | 按文字長度降序排序。 |
| 按字數從少到多排序 | 按字數升序排序文字。 |
| 按字數從多到少排序 | 按字數降序排序文字。 |
| 按日期和時間從舊到新排序 | 把日期和時間按升序排序。 |
| 按日期和時間從新到舊排序 | 把日期和時間按降序排序。 |
| 按出現次數升序排序 | 把出現次數按升序排序。 |
| 按出現次數降序排序 | 把出現次數按降序排序。 |
| IPv4 地址升序排序 | 把 IPv4 地址按升序排序。 |
| IPv4 地址降序排序 | 把 IPv4 地址按降序排序。 |
| IPv6 地址升序排序 | 把 IPv6 地址按升序排序。 |
| IPv6 地址降序排序 | 把 IPv6 地址按降序排序。 |
| 隨機排序 | 隨機排序。 |
| 反向排序 | 以相反的順序對字串進行排序。 |

## 「要插入的字串」文字方塊

在合併行時，指定一個要插入的字串。
