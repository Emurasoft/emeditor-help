# 「進階篩選」對話方塊

這個對話方塊會出現當您選擇了
**進階篩選** 命令或按了 **篩選** 工具列上的
**「進階篩選」** 按鈕。

### 「尋找 \| 列 \| 條件」清單方塊

用清單來顯示定義過的篩選。

### 「列」下拉清單方塊

更改清單中「列」的選項。

### 「大小寫需符合」核取方塊

大小寫需符合當搜索一個字串時。

### 「整個單字需符合」核取方塊

僅搜索單字。一個單字被定義為一個以這些字元開頭并結尾的字串: A – Z，a – z，0 – 9，或下劃線。由全形字元包圍的字串被認為是單字。當使用規則運算式時，該核取方塊可能不能正常運作。當使用規則運算式時，請用單字邊界表達式 (\\<，\\>，以及 \\b) 。

### 「符合整個字串」核取方塊

符合所選列，字元位置或整行中的整個項目。建議盡可能設置此選項以優化篩選的速度。當使用邏輯或運算定義多個篩選條件時，請為所有條件設置此選項，以便更快地提高篩選速度。

### 「負向」核取方塊

勾選該核取方塊將會排除所有符合的搜索結果。文檔會隱藏所有與指定字串相符合的結果。

### 「僅書籤行」核取方塊

僅符合書籤行。

### 「僅未標示書籤的行」核取方塊

僅符合僅未標示書籤的行。

### 「（無）」選項按鈕

指定字串應按字面意思符合。

### 「規則運算式」選項按鈕

啟用 [規則運算式](../../howto/search/search_regexp)。

### 「逸出序列」選項按鈕

啟用逸出序列。你能把下列字元作為逸出序列。

|     |     |
| --- | --- |
| \\a | 警告（響鈴） |
| \\b | 退格 |
| \\f | 換頁 |
| \\n | 換行 |
| \\t | 水平 Tab |
| \\v | 垂直 Tab |
| \\\ | 反斜杠 |
| \\oooooo | Unicode 字元在八進位運算式中 |
| \\xhhhh | Unicode 字元在十六進位運算式中 |

空字元（\\0）可能不能使用。一定要用 \\n 而不是 \\r。

### 「數字範圍」選項按鈕

啟用 [數字範圍運算式](../../howto/search/number_range_syntax)。

### 「符合換行符號」核取方塊

符合指定換行符號。此選項應與 **僅 CR**， **僅 LF**， **僅 CR+LF**，以及/或 **其他** 核取方塊合用。

### 「僅 CR」核取方塊

只符合換行符號是 CR 的行。此選項應與 **符合換行符號** 選項合用。

### 「僅 LF」核取方塊

只符合換行符號是 LF 的行。此選項應與 **符合換行符號** 選項合用。

### 「CR+LF」核取方塊

符合換行符號是 CR+LF 的行。此選項應與 **符合換行符號** 選項合用。

### 「其他」核取方塊

符合沒有換行符號的行。這些行包括檔案的最後一行以及非常長的行，例如繼續到下一行而沒有換行符號的行。此選項應與 **符合換行符號** 選項合用。

### 「開始篩選」核取方塊

用字元數指定開始篩選的位置。如果同時存在「開始篩選」的項目和另一個「結束篩選」的項目，則 EmEditor 僅顯示與「開始篩選」符合的行和與「結束篩選」符合的行之間的行。

### 「結束篩選」核取方塊

用字元數指定結束篩選的位置。

### 「對前一個條件用運算邏輯分離 (OR)」核取方塊

在目前的條件與前一個條件上執行運算邏輯分離 (OR)。如果不勾選該核取方塊，會預設使用運算邏輯結合 (AND)。

### 「篩選」按鈕

應用指定的篩選條件并關閉該對話方塊。

### 「設為書籤」按鈕

把滿足指定條件的行設為書籤。

### 「中止」按鈕

中止目前的的篩選。

### 「匯入」按鈕

匯入之前匯出的包含篩選條件的 TSV 檔案。

### 「匯出」按鈕

把已定義的篩選條件匯出到一個檔案中。檔案格式可以是 TSV，JavaScript 或 VBScript 巨集格式。
