# Markdown 頁面

**Markdown** 頁面讓你能自訂與 Markdown 設定相關的選項。

## 「自動顯示 Markdown 工具列」核取方塊

如果勾選了該核取方塊，EmEditor 會在打開 Markdown 檔案時自動顯示 Markdown 工具列。

## 「自動切換到 Markdown 設計視圖」核取方塊

如果勾選了該核取方塊，EmEditor 會在打開 Markdown 檔案時自動切換到 Markdown 設計視圖。

## 「超連結可以省略 .md 檔案副檔名」核取方塊

如果勾選了該核取方塊，你可以在超連結中省略 .md 檔案副檔名。

## 「為 AI 響應建立新文檔時切換到 Markdown 檢視」核取方塊

如果勾選了該核取方塊，EmEditor 會在為 AI 響應建立新文檔時切換到 Markdown 檢視。

## 「轉義這些字元」核取方塊

指定是否在 Markdown 檔案中轉義指定的字元。如果你選擇轉義某些字元，可以在核取方塊下方指定字元。

## 「重設」按鈕

重設為預設設定。
