# 「語法檢查」頁面

**語法檢查** 頁面讓你能自訂語法檢查器。

### 「語法檢查器引擎」下拉清單方塊

指定要給 HTML/CSS/JSON 文檔使用的語法檢查器引擎。

### 「停用語法檢查如果行數超過」文字方塊

如果文檔中的行數超過此數量，將停用 HTML/CSS/JSON 文檔的語法檢查。

### 「停用語法檢查如果行的長度超過」文字方塊

如果文檔中每行的任何行長度超過此數字，將停用 HTML/CSS/JSON 文檔的語法檢查。

### 「翻譯 URL」文字方塊

指定用於 HTML/CSS/JSON 語法檢查結果的「線上翻譯」命令的 URL。"%1" 是給目的語言識別項的一個預留位置（例如，"ja" 代表日文），還有 "%2" 是給 UTF-8 百分比編碼原始程式文字的一個預留位置。請注意，只有在「自訂」對話方塊的 **語言** 頁面中選擇了非英語時，才能使用翻譯功能。

### 「驗證 XML 文檔（XML）」核取方塊

如果勾選該核取方塊，EmEditor 將驗證 XML 文檔。如果不勾選，EmEditor將僅檢查XML文檔是否格式正確。

### 「解析外部定義（XML）」核取方塊

如果勾選該核取方塊，則 EmEditor 將嘗試解析外部定義，可解析的命名空間，文檔類型定義（DTD）外部子集和外部實體引用。

### 「自動重新整理」核取方塊

如果勾選該核取方塊，EmEditor 會自動重新整理語法檢查亮顯。

### 「停用自動重新整理如果行數超過」文字方塊

如果文檔中的行數超過此數量，將停用自動重新整理。

### 「搜索 URL」文字方塊

指定用於語法檢查結果的「線上搜索」命令的 URL。"%1" 將取代為原始程式文字（UTF-8 百分比編碼）。

### 「重置」按鈕

重置為預設設定。
