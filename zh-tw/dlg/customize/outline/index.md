# 「大綱」頁面

**大綱** 頁面讓您能自訂與大綱有關的設定。

## 大綱顯示條位置

選擇自訂顯示條在 EmEditor 視窗中所處的位置。

## 為每一個組態切換大綱顯示條

指定是否要為每一個組態切換大綱顯示條。

## 為每一個組態切換大綱導引

指定是否為每一個組態切換大綱導引。

## 第一次運行時將焦點移至自訂顯示條

指定是在顯示大綱顯示條時否要將焦點移至自訂顯示條。

## 使用 ESC 鍵關閉自訂顯示條

指定是否要通過按 ESC 鍵來關閉自訂顯示條。

## 使用歸位鍵或 GO 命令來關閉自訂顯示條

指定是否要通過按歸位鍵或選擇 GO 命令來關閉自訂顯示條。

## 「重設」按鈕

重設為預設設定。
