# 「檢視」頁面

**檢視** 頁面讓您能自訂與檢視有關的設定。

## 「自訂顯示條顏色 \- 使用系統顏色」核取方塊

如果該核取方塊被勾選了，每一個自訂顯示條的文字和背景顏色都會用系統顏色。

## 「自訂顯示條顏色 \- 文字」按鈕

點擊該按鈕更改自訂顯示條的文字顏色。

## 「自訂顯示條顏色 \- 背景」按鈕

點擊該按鈕變更自訂顯示條的背景顏色。

## 「自訂顯示條顏色 \- (示例) 」文字

用目前的選取的顏色顯示一個文字示例。

## 「游標大小與顏色 \- 設定覆寫游標大小和顏色」核取方塊

指定在與「插入」模式相對的「覆寫」模式期間，適用於游標大小和顏色的下列選項。

## 「游標大小與顏色 \- 寬度」下拉清單方塊

指定游標的寬度。

## 「游標大小與顏色 \- 高度」下拉清單方塊

指定游標的高度。

## 「游標大小與顏色 \- 使用系統顏色」核取方塊

如果該核取方塊被勾選了，游標顏色會變為與背景色相反的顏色。

## 「游標大小與顏色 \- 顏色」按鈕

點擊該按鈕更改游標顏色。

## 「使用雙重緩存」核取方塊

使用雙重緩存在屏幕上繪制文字。雙重緩存會讓閃爍消失，但繪制會變慢。如果使用的是運行比較慢的電腦，文字繪制地也比較緩慢，您可以清除勾選該核取方塊。

## 「同步換行模式」核取方塊

在所有有著相同組態的文檔中同步換行模式。如果該核取方塊沒有被勾選，那么換行模式就會由每一個文檔單獨設置，而不僅僅是每一個組態。

## 「App 主題」下列清單方塊

如果選擇了 **系統**，EmEditor 會用 Windows 10 中指定的模式，比如如果您在 Windows 10 中選擇了夜間模式時，EmEditor 也會沿用夜間模式。如果選擇了日間模式，EmEditor 會沿用日間模式。編輯器顏色可以在組態屬性的 [**顯示** 頁面](../../properties/display/index) 上設定。

## 「很暗 (beta)」按鈕

如果設定了此選項，僅在 Windows 10 Build 17763 或更高版本中使用「深色」模式時，EmEditor 才會在所有元素（包括功能表列和對話方塊）上使用深色。 由於 Windows API 尚未正式支持針對桌面應用程式的深色模式，因此該功能仍處於測試階段。Windows 的未來版本可能不支持它。

## 「重設」按鈕

重設為預設設定。
