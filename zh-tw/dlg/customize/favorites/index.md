# 「我的最愛」頁面

**我的最愛** 頁面讓您能自訂與我的最愛相關的設定。

## 「圖示」下拉清單方塊

指定每個檔案的圖示的顯示方式。選擇以下選項之一。

|     |     |
| --- | --- |
| **無** | 不顯示任何圖示。 |
| **文件類型圖示** | 顯示與文件類型關聯的圖示。 |
| **EmEditor 圖示** | 顯示 EmEditor 圖示。 |

## 「寬度」下拉清單方塊

指定如何調整每個按鈕的寬度。選擇以下選項之一。

|     |     |
| --- | --- |
| **按檔案名稱調整** | 每個按鈕都根據其檔案名長度進行調整。 |
| **只在超出指定寬度時截短** | 每個按鈕都會根據其檔案名長度進行調整，但僅當其長度大於 **指定寬度** 文字方塊中指定的寬度時才會被截短。 |
| **固定為指定的寬度** | 所有按鈕的長度與 **指定寬度** 文字方塊中指定的長度相同。 |

## 「當不適合時」下拉清單方塊

指定當按鈕不適合視窗寬度時按鈕的顯示方式。指定以下選項之一。

|     |     |
| --- | --- |
| **無** | 什么也不做。 |
| **調整寬度** | 調整每個按鈕的寬度以適合視窗寬度。 |
| **添加行** | 添加行以顯示所有按鈕。 |

## 「指定寬度」文字方塊

在「寬度」下拉清單方塊中選擇「只在超出指定寬度時截短」或「固定為指定寬度」時，指定每個按鈕的寬度。

## 自動顯示或隱藏我的最愛工具列

勾選此核取方塊後，當工具列上沒有按鈕時，EmEditor 會自動隱藏我的最愛工具列；當使用者向工具列添加按鈕時，EmEditor 會自動顯示我的最愛工具列。

## 「添加」按鈕

單擊此按鈕可將新項目添加到清單中。

## 「刪除」按鈕

單擊此按鈕可從清單中刪除所選項目。

## 「列」文字方塊

指定打開該檔案時初始游標位置的列號。

## 「欄」文字方塊

指定打開該檔案時初始游標位置的欄號。

## 「重設」按鈕

重設為預設設定。
