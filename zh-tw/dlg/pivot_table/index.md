# 「樞紐分析表」對話方塊

當你執行 [**樞紐分析表** 命令](../../cmd/csv/pivot_table) 時，這個對話方塊會出現。

## 「類型」下拉清單方塊

選擇要建立的樞紐分析表類型。

|     |     |
| --- | --- |
| 計數 | 計算出現次數。 |
| 總和 | 對值求和。 |
| 平均值 | 計算所有值的平均值。 |
| 最大值 | 最大值。 |
| 最小值 | 最小值。 |

## 「小數位數」文字方塊

如果在 **類型** 下拉清單方塊中選擇了 **總和**、 **平均值**、 **最大值** 或 **最小值**，請指定小數位數。如果指定了 0，則所有值都被視為整數，並且忽略小數點右邊的任何數字。

## 「資料列」下拉清單方塊

選擇要延伸成新樞紐分析表中的資料列的 CSV 文檔的欄。

## 「排序」下拉清單方塊

選擇新資料列（每列的第一欄）的排序方法。

| 按數字降序排序 | 按數字降序排序。 |
| 按文字長度從短到長排序 | 按文字長度升序排序。 |
| 按文字長度從長到短排序 | 按文字長度降序排序。 |
| 按字數從少到多排序 | 按字數升序排序文字。 |
| 按字數從多到少排序 | 按字數降序排序文字。 |
| 按日期和時間從舊到新排序 | 把日期和時間按升序排序。 |
| 按日期和時間從新到舊排序 | 把日期和時間按降序排序。 |

| IPv4 地址升序排序 | 把 IPv4 地址按升序排序。 |

選擇要延伸成新樞紐分析表中的資料欄的 CSV 文檔的欄。

## 「值」下拉清單方塊

選擇要延伸到新樞紐分析表中的值的 CSV 文檔的欄。

## 「總計欄」核取方塊

在最後一列顯示每欄的總值。如果已設定，請輸入要用於該列第一欄的標籤。

## 「總計列」核取方塊

在最右側的欄中顯示每列的總值。如果已設定，請輸入要用於欄第一列的標籤。

## 「標題」下拉清單方塊

指定用作標題的列數。
