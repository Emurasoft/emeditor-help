# Q. 當我用預設設置打開一個檔案時，檔案變為不可讀；如果我從「檢視」下的「字型分類」選項中選擇另一個字元集，檔案看起來更糟糕。但是，如果我在打開檔案時選擇一個合適的編碼，檔案則能正常顯示。這是怎么一回事？

EmEditor 的內部處理編碼是 Unicode。當您打開一個 ANSI 檔案時，EmEditor 可以把 ANSI 文字轉換為 Unicode 文字。當您要打開一個檔案時，應該在 **「打開」** 對話方塊中選擇一個編碼。如果您不選擇編碼，EmEditor 會用 [系統預設編碼](../../glossary/systemdefaultencoding) 來打開檔案。
所以，要用一個不同的編碼來打開檔案，您需要指定打開該檔案的編碼。您可以在打開檔案之後更改編碼，到 **檔案** 功能表下選擇 [**重新載入** 命令](../../cmd/file/file_reload_defined)，然后指定一個不同的編碼。
