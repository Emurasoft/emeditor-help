# Q. 當我嘗試打開一個非常大的檔案時，會出現類似於「分頁檔太小，無法完成操作。」的錯誤消息。如何解決此錯誤？

您將需要增加實體或虛擬記憶體才能打開一個非常大的檔案。我們建議先增加 PC 中的實體記憶體（RAM）。如果錯誤仍然存在，或者您無法增加實體記憶體大小，則可以增加虛擬記憶體。要增加虛擬記憶體，請按照以下步驟操作：

1. 按 **Win**(⊞)+ **R** 鍵來打開 **執行** 對話方塊，輸入 **SystemPropertiesPerformance.exe** 來顯示 **效能選項** 對話方塊。
2. 選擇 **效能選項** 對話方塊的 **進階** 索引標籤。
3. 單擊 **虛擬記憶體** 方塊中的 **變更** 按鈕。
4. 清除 **自動管理所有磁碟的分頁檔大小** 核取方塊。
5. 選擇 **自訂大小**，然後在 **起始大小（MB）** 和 **最大容量（MB）** 方塊中輸入建議的數字。我們建議在 **初始大小（MB）** 和 **最大容量（MB）** 方塊中輸入相同的大數字。例如，在兩個方塊中都輸入 80000。
6. 單擊確定，然後重新啟動 Windows。
7. 嘗試再次使用 EmEditor 打開大檔案。如果錯誤仍然存在，請重複上述步驟以增加虛擬記憶體大小，直到錯誤消失。
