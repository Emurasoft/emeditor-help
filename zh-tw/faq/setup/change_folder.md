# Q. 怎么變更安裝資料夾？

預設設定下，安裝新版 EmEditor 的資料夾與舊版的資料夾相同，如果是第一次安裝的話，預設的安裝資料夾是 %LocalAppData%\\Programs\\EmEditor（按每個使用者安裝 - 「只為我」）或 Program Files\\EmEditor（按每臺計算機安裝 - 」每個人（所有使用者）「）。如果要變更安裝資料夾，請使用系統管理權限打開命令提示字元，然後用 NOSKIP 選項運行安裝程式。

例如，如果 EmEditor 安裝檔案是 emed64\_18.9.3.msi，您想要把 EmEditor 安裝到另一資料夾中，請運行以下命令：

msiexec /i emed64\_18.9.3.msi NOSKIP=1

這將使您有機會在安裝過程中選擇 "Custom"（自訂）按鈕，允許您變更安裝資料夾。

注意: 您不能變更安裝資料夾如果 EmEditor 是按每個使用者安裝。要變更安裝資料夾，您必須選擇為所有使用者安裝 EmEditor 選項。

不同版本的 EmEditor 無法同時存在在一臺電腦上。如果舊的版本已經安裝了，您必須先卸載舊的版本，或安裝新的版本到與舊版本相同的資料夾中。
