# Q. 我現在使用的是舊版的 EmEditor。我能不用卸載舊版的 EmEditor 就安裝新版本的 EmEditor 嗎？

如果您使用的是 EmEditor v3.00或v3以后的版本，并且想要保持以前使用的 EmEditor 配置設定的話，您不需要卸載就可以覆蓋舊版。步驟如下:

1. 安裝最新的版本。
2. 當安裝完成時，重新啟動Windows。

如果您用的 EmEditor 版本在v3.00以前，也不想要保持以前使用的 EmEditor 配置設定的話，我們推薦您可以先完全卸載以前的版本，重新啟動計算機，然后再安裝新版本。具體步驟如下:

1. 點擊 **「開始」**， 在「開始」菜單上選擇「控制面板。從 **「卸載或更改程序」** 中選擇 EmEditor (或 EmEditor v3)。然后點擊 **「刪除」** 按鈕。
2. 當卸載完成后，重新啟動 Windows。
3. 按屏幕指示，安裝新版本。
