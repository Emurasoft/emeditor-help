# Q. 如何使用 EmEditor 打開 Excel 檔案？

Excel 檔案是二進位檔案，您不能直接使用 EmEditor 打開 Excel 檔案。但是，有幾種方法可以完成此任務。

1. 使用 Excel 打開 Excel 檔案，並將其另存新檔文字檔案（ Tab分隔或逗號分隔）。然後使用 EmEditor 打開文字檔案。
2. 使用 Excel 增益集 [CSV Import+Export](https://appsource.microsoft.com/zh-tw/product/office/wa200000025) 來把 Excel 檔案匯出為 CSV 檔案，然後使用 EmEditor 打開 CSV 檔案。
3. 用 Excel 打開一個 Excel 檔案，按 CTRL + A，然後按 CTRL + C 選擇所有數據並將其複製到剪貼簿。啟動 EmEditor，從 CSV 工具列中選擇 Tab分隔模式，然後按 CTRL + V 進行貼上。
