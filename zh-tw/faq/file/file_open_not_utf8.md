# Q. 怎么用西歐 (Western European) 而不是 UTF-8 打開一個 XML 檔案?

如果一個 XML 檔案是作為 Unicode 被打開的，您可以到 **「當前組態屬性」** 中，選擇 [**檔案** 頁面](../../dlg/properties/file/index)，然后檢視 **「檔案打開時的編碼」** 旁的下拉清單方塊。如果被選擇的是 **「Unicode」**，請您把它改為 **「系統預設」**。如果 **「偵測 HTML/XML 字元」** 核取方塊被勾選了，一個「encoding=」的指令會在一個 XML 檔案中被搜尋。如果這個指令被找到了，EmEditor使用這個指定的編碼。要注意的是，為目前打開的檔案所指定的編碼與新增檔案或正在儲存的檔案所指定的編碼是不同的。
