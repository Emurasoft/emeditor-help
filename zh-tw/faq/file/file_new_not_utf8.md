# Q. 怎么用西歐 (Western European) 而不是 UTF-8 來新增 Java 檔案或 XML 檔案?

在預設情況下，Java檔案和 XML 檔案會作為 UTF-8 打開當您從檔案功能表中選擇
[新增 Java 或 XML 檔案 命令](../../cmd/file/file_new_config) 時。您可以更改這一預設設定。首先，打開 Java (或 XML) 檔案的組態屬性，選擇檔案 頁面，點擊「新增檔案…」 按鈕，從
[編碼 下拉清單方塊](../../dlg/properties/file/new_details/index) 中選擇「系統預設」。要注意的是新增檔案所設的編碼可能會與正在打開的或正在儲存的檔案不同。
