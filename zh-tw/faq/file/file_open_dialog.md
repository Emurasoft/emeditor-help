# Q. 怎么能設定一個預設資料夾，讓「打開」對話方塊一直從這個資料夾中打開檔案？

選擇工具功能表下的「 **自訂**」，然后在「 **檔案**」頁面上勾選「 **記住最后打開的資料夾**」。

另外，您也可以右擊 EmEditor 捷徑，打開 **屬性**。在「 **目標**」(Target) 文字方塊中的路徑后加 /cd。例如，如果 EmEditor 的安裝資料夾是 C:\\Program Files\\EmEditor，那么請輸入 "C:\\Program Files\\EmEditor\\emeditor.exe" /cd，并且在「 **起始位置**」(Start in) 文字方塊中輸入您要打開的資料夾路徑。
