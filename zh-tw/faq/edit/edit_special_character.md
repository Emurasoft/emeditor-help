# Q. 怎么插入特殊字元？

在 **編輯** 功能表下選擇 **進階**，然后點選 **插入特殊字元**。要檢查您要插入的一個 Unicode 字元代碼，請參考這個連結:
[拉丁延伸字元參考](../../howto/edit/special_char_sample)。您也能夠通過 **字元映射表** 來檢查一個字元，如果您的 Windows 系統有組態「字元映射表」
(「開始」功能表 > 所有程式 >
附件 > 系統工具 \> 字元映射表，或在搜尋方塊中，鍵入「字元映射表」，然后單擊尋找結果)。您可能需要適當調整字型來顯示特殊字元。

同樣，請參考 [「插入特殊字元」](../../howto/edit/edit_special_char) 來檢視特殊字元的鍵盤捷徑。
