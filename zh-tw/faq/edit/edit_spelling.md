# Q. 怎么檢查拼字？

在 EmEditor Professional v10 后的版本中，您可以不使用任何外掛程式就能檢查拼字。在 EmEditor 的內容功能表中 (右鍵功能表)，選擇進行「 **拼字檢查**」。如果無法看到這一選項，請到「工具」下的「 **自訂功能表**」中選擇「 **內容功能表**」，然后按「 **重設**」按鈕。EmEditor 預設安裝了英語 (美國) 字典。其他語言的字典請到 extensions.OpenOffice.org ( [http://extensions.openoffice.org/](http://extensions.openoffice.org/)) 中下載。在下載一個字典檔案后，請把副檔名從 .oxt 改為 .zip，然后解壓縮這個 zip 檔案，并且把 \*.dic 和 \*.aff 檔案複製到 EmEditor 安裝資料夾中的 Dictionaries (字典) 子資料夾中 (通常在 %LOCALAPPDATA%\\Programs\\EmEditor\\Dictionaries 或 C:\\Program Files\\EmEditor\\Dictionaries)。
