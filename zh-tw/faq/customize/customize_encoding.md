# Q. 在「檔案」功能表下「重新載入」的子功能表中，除了相同編碼，系統預設，UTF-16LE，UTF-16BE，UTF-8，和UTF-7 之外，沒有其他編碼可用。我怎么添加更多的編碼頁？

如果您沒有適當地安裝 EmEditor，只是把 emeditor.exe 檔案從一臺電腦複製到另一臺電腦并且嘗試運行程式，那么一些系統內設置的編碼可能不會在 EmEditor 複製版中顯示。

如果您使用的是 Windows NT/2000/XP，要安裝另外的編碼，您可以作為進階使用者(Power User)或管理員身份登入，并在 **「工具」** 功能表下，選擇
[**「定義編碼」** 命令](../../cmd/tools/define_code_page) 然后點擊 **「重設」** 按鈕。

編碼的可用性取決于安裝在 Windows 中的語言系統檔案。如果您無法在 EmEditor 中找到您要的那種編碼，您可以到「控制台」中的「區域和語言選項」安裝額外的語言檔案。
當語言系統檔案安裝到您的 Windows 系統中后，再次到 EmEditor 中到 **「工具」** 功能表下選擇 [**「定義編碼」** 命令](../../cmd/tools/define_code_page)，然后點擊 **「重設」** 按鈕。
