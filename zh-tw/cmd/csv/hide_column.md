# 隱藏欄命令

## 摘要

隱藏選取的欄或目前欄。

## 說明

隱藏選取的欄或目前欄。

## 運行方法

- 預設功能表: **編輯** \> **CSV** \> **欄** \> **隱藏欄**
- [全部命令](../tools/all_commands): **編輯** \> **CSV** \> **欄** \> **隱藏欄**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_HIDE_COLUMN (4087)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4087);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4087
```
