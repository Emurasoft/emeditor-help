# 在 CSV 中刪除內嵌新行命令

## 摘要

在 CSV 文檔中刪除內嵌新行。

## 說明

在 CSV 文檔中刪除內嵌新行。

## 運行方法

- 預設功能表: **CSV** \> **進階** \> **刪除內嵌新行**
- [全部命令](../tools/all_commands): **CSV** \> **進階** \> **刪除內嵌新行**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_REMOVE\_EMBEDDED\_NL (3922)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3922);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3922
```
