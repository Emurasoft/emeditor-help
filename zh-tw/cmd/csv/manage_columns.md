# 管理欄命令

## 摘要

移動、複製、刪除或排序 CSV 文檔中的欄。

## 說明

移動、複製、刪除或排序 CSV 文檔中的欄。當選擇該命令時會顯示 [管理欄 對話方塊](../../dlg/manage_columns/index)。

## 運行方法

- 預設功能表:CSV \>欄 \>管理欄
- [所有命令](../tools/all_commands):CSV \>欄 \>管理欄
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_MANAGE_COLUMNS (23239)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23239);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23239
```
