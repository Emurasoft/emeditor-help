# 列標題 4 命令

## 摘要

把前四欄作為標題並且固定不捲動。

## 說明

把前四欄作為標題並且固定不捲動。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>標題（凍結窗格） \>列標題 4
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ROW\_HEADING\_4 (4095)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4095);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4095
```
