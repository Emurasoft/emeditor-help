# 取消隱藏欄命令

## 摘要

顯示選取欄之間的隱藏欄。

## 說明

顯示選取欄之間的隱藏欄。

## 運行方法

- 預設功能表:編輯 \>CSV \>欄 \>取消隱藏欄
- [全部命令](../tools/all_commands):編輯 \>CSV \>欄 \>取消隱藏欄
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_UNHIDE_COLUMN (4088)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4088);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4088
```
