# 列標題 3 命令

## 摘要

把前三欄作為標題並且固定不捲動。

## 說明

把前三欄作為標題並且固定不捲動。

## 運行方法

- 預設功能表: **CSV** \> **標題（凍結窗格）** \> **列標題 3**
- [全部命令](../tools/all_commands): **CSV** \> **標題（凍結窗格）** \> **列標題 3**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ROW\_HEADING\_3 (4094)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4094);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4094
```
