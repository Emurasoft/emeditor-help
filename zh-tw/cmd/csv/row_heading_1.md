# 列標題 1 命令

## 摘要

把第一欄作為標題並且固定不捲動。

## 說明

把第一欄作為標題並且固定不捲動。

## 運行方法

- 預設功能表:CSV \>標題（凍結窗格） \>列標題 1
- [全部命令](../tools/all_commands):CSV \>標題（凍結窗格） \>列標題 1
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ROW\_HEADING\_1 (4092)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4092);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4092
```
