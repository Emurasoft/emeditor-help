# 置中對齊命令

## 摘要

把選取的欄或目前的欄中的文字置中對齊。

## 說明

把選取的欄或目前的欄中的文字置中對齊。

## 運行方法

- 預設功能表: **編輯** \> **CSV** \> **欄** \> **置中對齊**
- [全部命令](../tools/all_commands): **編輯** \> **CSV** \> **欄** \> **置中對齊**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_ALIGN_CENTER (23233)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23233);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23233
```
