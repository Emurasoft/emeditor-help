# 編輯儲存格 (僅限儲存格選擇模式) 命令

## 摘要

在 CSV 文檔中編輯目前的儲存格。

## 說明

在 CSV 文檔中編輯目前的儲存格。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **CSV** \> **僅限儲存格選擇模式** \> **編輯儲存格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CSV_EDIT_CELL (3960)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3960);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3960
```
