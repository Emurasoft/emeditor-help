# 擷取欄命令

## 摘要

從目前的 CSV 文檔中擷取指定欄。

## 說明

從目前的 CSV 文檔中擷取指定欄。當執行這個命令時，會顯示 [**擷取欄** 對話方塊](../../dlg/extract_columns/index)，讓您可以設置擷取的欄的參數。

## 運行方法

- 預設功能表: **CSV** \> **進階** \> **擷取欄**
- [全部命令](../tools/all_commands): **CSV** \> **進階** \> **擷取欄**
- 工具列: ![](../../images/extract_columns.png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_EXTRACT\_COLUMNS (3930)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3930);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3930
```
