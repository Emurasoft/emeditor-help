# 列標題 0 命令

## 摘要

重設列標題。

## 說明

重設列標題。

## 運行方法

- 預設功能表: **CSV** \> **標題（凍結窗格）** \> **列標題 0**
- [全部命令](../tools/all_commands): **CSV** \> **標題（凍結窗格）** \> **列標題 0**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_ROW\_HEADING\_RESET (4091)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4091);
```

### \[VBScript\]

