# IPv6 位址降序排序命令

## 摘要

把目前的欄中的 IPv6 位址按降序排序。

## 說明

把目前的欄中的 IPv6 位址按降序排序。IPv6 位址必須采用 RFC 5952 推薦的規范文字表示形式。在組態屬性 [**連結** 頁面](../../dlg/properties/link/index)的 **主動字串** 清單方塊中啟用 **IPv6 位址**，讓您能區分 IPv6 位址的文字。

## 運行方法

- 預設功能表: **編輯** \> **排序** \> **IPv6 位址降序排序**
- [全部命令](../tools/all_commands): **編輯** \> **排序** \> **IPv6 位址降序排序**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_IPV6_D (4037)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4037);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4037
```
