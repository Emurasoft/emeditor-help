# 欄排序命令

### 摘要

> 對 CSV 文檔中的欄進行排序。

### 說明

> 對 CSV 文檔中的欄進行排序。當選擇該命令時會顯示 [**欄排序** 對話方塊](../../dlg/sort_columns/index)。

### 運行方法

- 預設功能表: **排序** \> **欄排序**
- [所有命令](../tools/all_commands): **排序** \> **欄排序**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令 ID

- EEID\_SORT\_COLUMNS (23238)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(23238);

#### \[VBScript\]

> editor.ExecuteCommandByID 23238
