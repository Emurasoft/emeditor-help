# 重設書籤命令

## 摘要

清除目前的行的書籤。

## 說明

清除目前的行的書籤。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): 無
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_RESET (4354)```

## 巨集

### \[JavaScript\]

```
document.selection.ClearBookmark();
```

### \[VBScript\]

```
document.selection.ClearBookmark
```
