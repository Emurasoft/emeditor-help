# 選擇更新頻道命令

## 摘要

選擇一個更新頻道。

## 說明

選擇一個更新頻道。

## 運行方法

- 預設功能表: **說明** \> **選擇更新頻道**
- [全部命令](../tools/all_commands): **說明** >
**選擇更新頻道**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_UPDATE_CHANNEL (3950)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3950);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3950
```
