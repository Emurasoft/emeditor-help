# 打開 EmEditor 首頁命令

## 摘要

打開 EmEditor 網站首頁。

## 說明

在您預設的網絡瀏覽器中打開 EmEditor 網站。

## 運行方法

- 預設功能表: **幫助** \> **打開 EmEditor 首頁**
- [全部命令](../tools/all_commands): **幫助** >
**打開 EmEditor 首頁**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_WEB_HOME (4249)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4249);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4249
```
