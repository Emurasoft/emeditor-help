# 升級命令

## 摘要

將本產品升級為 EmEditor Professional。

## 說明

將本產品升級為 EmEditor Professional。這個命令在 EmEditor Professional 中不可用。

## 運行方法

- 預設功能表:說明 >
升級
- [全部命令](../tools/all_commands):說明 >
升級
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_HELP_UPGRADE (3845)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3845);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3845
```
