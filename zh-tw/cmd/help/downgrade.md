# 降級命令

## 摘要

將產品降級為 EmEditor Free。

## 說明

將產品從 EmEditor Professional 降級為 EmEditor Free。在降級之前，EmEditor 會顯示一條確認提示。

## 運行方法

- 預設功能表:說明 >
降級
- [全部命令](../tools/all_commands):說明 >
降級
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_HELP_DOWNGRADE (3844)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3844);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3844
```
