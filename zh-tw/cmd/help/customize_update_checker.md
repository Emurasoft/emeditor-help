# 自訂更新檢查程式命令

## 摘要

自訂更新程式。

## 說明

打開更新選項 對話方塊，您可以指定 EmEditor 檢查更新的次數以及下載和儲存路徑。

## 運行方法

- 預設功能表:說明 \>自訂更新檢查程式
- [全部命令](../tools/all_commands):說明 >
自訂更新檢查程式
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_UPDATE_CHECKER (4482)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4482);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4482
```
