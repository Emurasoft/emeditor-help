# 註冊信息命令

## 摘要

顯示註冊信息。

## 說明

顯示註冊信息。

## 運行方法

- 預設功能表: **說明** \> **註冊信息**
- [所有命令](../tools/all_commands): **說明** > **註冊信息**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_HELP_REGISTRATION_INFO (23249)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23249);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23249
```
