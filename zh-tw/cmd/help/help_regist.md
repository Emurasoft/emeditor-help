# 如何購買命令

## 摘要

顯示如何購買 EmEditor 的信息。

## 說明

在「EmEditor 說明」中打開有關「如何購買」的章節。

## 運行方法

- 預設功能表:幫助 \>如何購買
- [全部命令](../tools/all_commands):幫助 >
如何購買
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_HELP_REGIST (4248)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4248
```
