# 右邊的文檔命令

## 摘要

切換至標籤頁右邊的文檔。

## 說明

把文檔切換到標籤欄上目前的打開的文檔的右邊。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **文檔導航** \> **右邊的文檔**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL + F6

## 外掛程式命令 ID

- EEID\_RIGHT\_DOC (4547)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4547);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4547
```
