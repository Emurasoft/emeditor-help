# 向左捲動命令

## 摘要

將文檔向左捲動一個字元。

## 說明

將文檔向左捲動一個字元。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 \>捲動 \>向左
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_SCROLL\_LEFT (4474)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4474);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4474
```
