# 全部最小化命令

## 摘要

最小化所有視窗。

## 說明

最小化所有打開的 EmEditor 視窗。

## 運行方法

- 預設功能表: **視窗** \> **全部最小化**
- [全部命令](../tools/all_commands): **視窗**
\> **全部最小化**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_MINIMIZE\_ALL (4244)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4244);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4244
```
