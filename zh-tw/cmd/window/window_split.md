# 分割命令

## 摘要

分割目前的視窗到窗格中。

## 說明

這個命令會把目前的視窗分割成四個窗格。用游標來控制分割位置。點擊游標將固定分割位置 (您也可以在以后根據需要改變方向)。

## 運行方法

- 預設功能表: **視窗** \> **分割**
- [全部命令](../tools/all_commands): **視窗**
\> **分割** \> **分割**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+F12

## 外掛程式命令 ID

- EEID\_WINDOW\_SPLIT (4240)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4240);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4240
```
