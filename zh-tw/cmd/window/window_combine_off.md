# 啟用標籤頁 \- 關命令

## 摘要

禁用標籤頁并分隔所有視窗。

## 說明

禁用標籤頁并分隔所有視窗。當標籤頁被禁用，每一個打開的文檔會顯示在單獨的視窗中。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **啟用標籤頁** \> **啟用標籤頁 \- 關**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_COMBINE\_OFF (4408)

## 巨集

### \[JavaScript\]

```
editor.EnableTab = false;
```

### \[VBScript\]

```
editor.EnableTab = False
```
