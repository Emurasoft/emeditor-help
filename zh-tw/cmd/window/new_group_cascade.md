# 新增層疊群組命令

### 摘要

> 從組合視窗中分離目前的文檔為一個新的群組并排列視窗使之重疊。

### 說明

> 從組合視窗中分離目前的文檔為一個新的群組并排列視窗使之重疊。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **群組** \> **新增層疊群組**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_NEW\_GROUP\_CASCADE (4393)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4393);

#### \[VBScript\]

> editor.ExecuteCommandByID 4393
