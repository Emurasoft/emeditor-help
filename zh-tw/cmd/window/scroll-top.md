# 捲動至頂部命令

## 摘要

捲動到文檔的頂部。

## 說明

捲動到文檔的頂部。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗** \> **捲動** \> **至頂部**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_SCROLL\_TOP (3864)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3864);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3864
```
