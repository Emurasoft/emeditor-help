# 最少訪問的文檔命令

### 摘要

> 切換至最少訪問的文檔。

### 說明

> 切換至最少訪問的文檔。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **文檔導航** \> **最少訪問的文檔**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_PREV\_ZORDER\_DOC (4545)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4545);

#### \[VBScript\]

> editor.ExecuteCommandByID 4545
