# 移動到下一個命令

### 摘要

> 移動目前的文檔到下一個標籤頁位置。

### 說明

> 移動目前的文檔到下一個標籤頁位置。您可以用這個命令來變更文檔在標籤欄上的順序。另外，您也可以把標籤頁拖放到標籤欄上所需的位置。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **文檔導航** \> **移動到下一個**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_MOVE\_NEXT (4382)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4382);

#### \[VBScript\]

> editor.ExecuteCommandByID 4382
