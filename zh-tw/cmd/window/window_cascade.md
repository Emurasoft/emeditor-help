# 層疊視窗命令

## 摘要

排列視窗使之層疊。

## 說明

排列 EmEditor 視窗，讓非組合視窗層疊在一個視窗中方便快速訪問。

## 運行方法

- 預設功能表: **視窗** \> **層疊**
- [全部命令](../tools/all_commands): **視窗**
\> **層疊**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_CASCADE (4241)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4241);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4241
```
