# 總是置頂 \- 關命令

## 摘要

取消將該視窗置頂顯示。

## 說明

解除目前的視窗總是置頂的狀態。其他文檔將一直在被設置為總是置頂的文檔下方顯示。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 \>
總是置頂 \>
總是置頂 \- 關
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_ALWAYS\_TOP\_OFF (4344)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4344);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4344
```
