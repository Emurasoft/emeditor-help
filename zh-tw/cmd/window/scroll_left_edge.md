# 捲動至左邊緣命令

## 摘要

捲動到文檔的左邊緣。

## 說明

捲動到文檔的左邊緣。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 > 捲動 \>左邊緣
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_SCROLL\_LEFT\_EDGE (3870)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3870);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3870
```
