# 垂直平鋪視窗命令

## 摘要

垂直平鋪排列視窗，使其不重疊。

## 說明

垂直排列非組合的 EmEditor 視窗，使其不重疊平鋪。

## 運行方法

- 預設功能表:視窗 \>垂直平鋪
- [全部命令](../tools/all_commands):視窗
\>垂直平鋪
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_WINDOW\_TILE\_VERT (4243)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4243);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4243
```
