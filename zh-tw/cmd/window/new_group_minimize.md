# 新增群組并最小化命令

## 摘要

從組合視窗中分離目前的文檔為一個新的群組并最小化。

## 說明

從組合視窗中分離目前的文檔為一個新的群組并最小化。這個命令等同于拖放標籤頁到目前的 EmEditor 視窗外，當出現提示時，選擇新增群組 并最小化新增視窗。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 \>
群組
\>新增群組并最小化
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_NEW\_GROUP\_MINIMIZE (4392)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4392);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4392
```
