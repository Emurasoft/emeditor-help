# 新增群組命令

## 摘要

從組合視窗中把目前的文檔分離為一個新的群組。

## 說明

從組合視窗 (標籤頁群組) 中把目前的文檔分離為一個新的群組。這個與把一個標籤頁拖放到目前的 EmEditor 視窗之外并在快顯的下拉功能表中選擇 **新增群組** 的行為是一樣的。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗** \>
**群組**
\> **新增群組**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_NEW\_GROUP (4391)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4391);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4391
```
