# 捲動至底部命令

## 摘要

捲動到文檔的底部。

## 說明

捲動到文檔的底部。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 \>捲動 \>至底部
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_SCROLL\_BOTTOM (3865)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3865);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3865
```
