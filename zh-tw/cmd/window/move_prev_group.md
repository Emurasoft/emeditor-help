# 移動到上一個群組命令

## 摘要

移動目前的文檔到上一個群組中。

## 說明

移動目前的文檔到上一個群組中。 (重新合併為一個視窗)。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗
\>群組 \>移動到上一個群組
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_MOVE\_PREV\_GROUP (4396)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4396);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4396
```
