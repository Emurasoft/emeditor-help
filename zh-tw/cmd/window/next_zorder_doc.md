# 最后訪問的文檔命令

### 摘要

> 切換至最后訪問的文檔。

### 說明

> 切換至最后訪問的文檔。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **視窗**
\> **切換至最后訪問的文檔。**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_NEXT\_ZORDER\_DOC (4544)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4544);

#### \[VBScript\]

> editor.ExecuteCommandByID 4544
