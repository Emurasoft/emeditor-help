# 向左翻頁命令

## 摘要

把文檔向左捲動一頁。

## 說明

把文檔向左捲動一頁。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗 \>捲動 \>向左翻頁
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_SCROLL\_PAGE\_LEFT (3868)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3868);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3868
```
