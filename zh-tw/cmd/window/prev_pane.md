# 上一個窗格命令

## 摘要

切換到上一個窗格。

## 說明

把游標移動到上一個窗格，當視窗被分割為多個窗格時。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):視窗
\>窗格 \>上一個窗格
- 工具列: 無
- 狀態列: 無
- 預設捷徑: SHIFT+F6

## 外掛程式命令 ID

- EEID\_PREV\_PANE (4215)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4215);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4215
```
