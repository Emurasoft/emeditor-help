# 格式化文檔命令

## 摘要

用語言伺服器協議格式化整個文檔。

## 說明

用語言伺服器協議格式化整個文檔。

## 運行方法

- 預設功能表: **轉換** \> **格式化文檔**
- [所有命令](../tools/all_commands): **轉換** \> **格式化文檔**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: Ctrl+K，D

## 外掛程式命令ID

```
EEID_FORMAT_DOCUMENT (23244)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23244);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23244
```
