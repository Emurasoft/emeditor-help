# 定位點化命令

## 摘要

將空白字元轉換為 Tab 字元。

## 說明

在每一行的開頭把選取的空白字元轉換為 Tab 字元。被轉換的空白字元數必須與一個 Tab 所設置的空白字元數相同，這個設定可以在 [**Tab/縮排** 對話方塊](../../dlg/properties/general/indent/index) 中找到。

## 運行方法

- 預設功能表: **轉換** \> **定位點化**
- [全部命令](../tools/all_commands): **轉換** \> **定位點化**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_TABIFY (4356)```

## 巨集

### \[JavaScript\]

```
document.selection.Tabify();
```

### \[VBScript\]

```
document.selection.Tabify
```
