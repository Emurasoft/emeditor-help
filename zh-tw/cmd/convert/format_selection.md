# 格式化選區命令

## 摘要

用語言伺服器協議格式化選取區域。

## 說明

用語言伺服器協議格式化選取區域。

## 運行方法

- 預設功能表: **轉換** \> **格式化選區**
- [所有命令](../tools/all_commands): **轉換** \> **格式化選區**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: Ctrl+K，D

## 外掛程式命令ID

```
EEID_FORMAT_SELECTION (23245)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23245);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23245
```
