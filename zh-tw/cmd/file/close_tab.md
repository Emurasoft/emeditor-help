# 關閉標籤命令

## 摘要

關閉選取的標籤。

## 說明

這個命令會關閉選取的標籤。

## 運行方法

- 預設功能表: 無
- [所有命令](../tools/all_commands): **檔案** \> **關閉** \> **關閉標籤**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID__CLOSE_TAB (23246)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23246);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23246
```
