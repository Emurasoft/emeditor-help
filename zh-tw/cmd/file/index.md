# 檔案類別

|     |     |
| --- | --- |
| **[新增文字](file_new)** | 創建一個新的文字檔案。 |
| [**新增**(快顯功能表)](new_config_popup) | 顯示一個快顯功能表來使用指定的組態新增一個檔案。 |
| **[新增并貼上](file_new_paste)** | 創建一個新的檔案并插入剪貼簿中的內容。 |
| [**新增并貼為引文**](file_new_paste_prefix) | 創建一個新的檔案並把剪貼簿內容貼為引用文字。 |
| [**新增檔案，貼為引文并換行**](file_new_paste_prefix_return) | 創建一個新的檔案，貼上選定內容為引用文字并換行。 |
| [**顯示系統匣圖示**](new_tray_icon) | 顯示一個新的系統匣圖示。 |
| [**新增組態清單**](file_new_config) | 用一個指定組態(多個項目)創建一個新檔案。 |
| **[打開](file_open)** | 打開一個已存在的檔案。 |
| **[關閉并打開](file_close_open)** | 關閉目前的檔案并打開另一個已存在的檔案。 |
| **[最後關閉的檔案](most_recent_closed_file)** | 打開最後關閉的一個文件。 |
| **[儲存](file_save)** | 儲存目前的檔案。 |
| **[另存新檔](file_save_as)** | 使用一個新的名稱儲存目前的檔案。 |
| **[重新命名](file_save_rename)** | 重新命名目前的檔案。 |
| **[全部儲存](file_save_all)** | 儲存所有目前的打開的檔案。 |
| **[以指定編碼全部儲存](file_save_all_as)** | 除了未命名的文檔外，以指定的目標資料夾，編碼以及換行方式儲存所有打開的文檔。 |
| **[進階打開](advanced_open)** | 用進階選項打開現有檔案。 |
| **[儲存為系統預設編碼](file_save_ansi)** | 用 [系統預設編碼](../../glossary/systemdefaultencoding) 儲存目前的檔案。 |
| **[儲存為 UTF-16LE](file_save_unicode)** | 用 Unicode(UTF-16LE) 編碼儲存目前的檔案。 |
| **[儲存為帶簽名的 UTF-16LE](save_utf16le_sign)** | 用帶簽名的 Unicode(UTF-16LE) 編碼儲存目前的檔案。 |
| **[儲存為不帶簽名的 UTF-16LE](save_utf16le_nosign)** | 用不帶簽名的 Unicode(UTF-16LE) 編碼儲存目前的檔案。 |
| **[儲存為 UTF-16BE](file_save_unicode_bigendian)** | 用 Unicode(UTF-16BE) 編碼儲存目前的檔案。 |
| **[儲存為帶簽名的 UTF-16BE](save_utf16be_sign)** | 用帶簽名的 Unicode(UTF-16BE) 編碼儲存目前的檔案。 |
| **[儲存為不帶簽名的 UTF-16BE](save_utf16be_nosign)** | 用不帶簽名的 Unicode(UTF-16BE) 編碼儲存目前的檔案。 |
| **[儲存為 UTF-8](file_save_utf8)** | 用 Unicode(UTF-8) 編碼儲存目前的檔案。 |
| **[儲存為帶簽名的 UTF-8](save_utf8_sign)** | 用帶簽名的 Unicode(UTF-8) 編碼儲存目前的檔案。 |
| **[儲存為不帶簽名的 UTF-8](save_utf8_nosign)** | 用不帶簽名的 Unicode(UTF-8) 編碼儲存目前的檔案。 |
| **[儲存為 UTF-7](file_save_utf7)** | 用 Unicode(UTF-7) 編碼儲存目前的檔案。 |
| [**儲存為二進位編碼(ASCII 視圖)**](file_save_binary) | 用二進位編碼(ASCII 視圖)儲存目前的檔案。 |
| [**儲存為二進位編碼(十六進位視圖)**](file_save_hex) | 用二進位編碼(十六進位視圖)儲存目前的檔案。 |
| **[儲存為日文 Shift JIS](file_save_932)** | 用日文 Shift JIS 編碼儲存目前的檔案。 |
| **[儲存為日文 JIS](file_save_jis)** | 用日文 JIS 編碼儲存目前的檔案。 |
| **[儲存為日文 EUC](file_save_euc)** | 用日文 EUC 編碼儲存目前的檔案。 |
| [**要儲存的編碼清單**](file_save_defined) | 用一個指定的編碼(多個項目)儲存目前的檔案。 |
| **[儲存換行為 CR+LF](save_as_crlf)** | 儲存換行為 CR+LF。 |
| **[儲存換行為 CR](save_as_cr)** | 只以 CR 方式儲存換行。 |
| **[僅儲存換行為 LF](save_as_lf)** | 僅以 LF 方式儲存換行。 |
| **[複製檔案完整路徑](copy_file_path)** | 複製目前的檔案的完整路徑名到剪貼簿。 |
| **[複製檔案所在資料夾名](copy_file_dir)** | 複製目前的檔案所在資料夾的完整路徑名到剪貼簿。 |
| **[重新載入為相同編碼](file_reload)** | 用預設編碼重新載入目前的檔案。 |
| [**重新載入**(快顯功能表)](reload_popup) | 顯示快顯功能表來指定一個編碼并用該編碼重新載入檔案。 |
| **[重新載入并偵測所有編碼](file_reload_detect_all)** | 使用最適合的編碼重新載入目前的檔案。 |
| **[重新載入為系統預設編碼](file_reload_ansi)** | 用 [系統預設編碼](../../glossary/systemdefaultencoding) 重新載入目前的檔案。 |
| **[重新載入為 UTF-16LE](file_reload_unicode)** | 用 Unicode(UTF-16LE) 重新載入目前的檔案。 |
| **[重新載入為 UTF-16BE](file_reload_unicode_bigendian)** | 用 Unicode(UTF-16BE) 重新載入目前的檔案。 |
| **[重新載入為 UTF-8](file_reload_utf8)** | 用 Unicode(UTF-8) 重新載入目前的檔案。 |
| **[重新載入為 UTF-7](file_reload_utf7)** | 用 Unicode(UTF-7) 重新載入目前的檔案。 |
| [**重新載入為二進位(ASCII 視圖)**](file_reload_binary) | 用二進位(ASCII 視圖)重新載入目前的檔案。 |
| [**重新載入為二進位(十六進位視圖)**](file_reload_hex) | 用二進位(十六進位視圖)重新載入目前的檔案。 |
| **[重新載入為日文 Shift JIS](file_reload_932)** | 用日文 Shift JIS 重新載入目前的檔案。 |
| **[重新載入為日文 JIS](file_reload_jis)** | 用日文 JIS 重新載入目前的檔案。 |
| **[重新載入為日文 EUC](file_reload_euc)** | 用日文 EUC 重新載入目前的檔案。 |
| [**要重新載入的編碼清單**](file_reload_defined) | 用一個指定的編碼(多個項目)重新載入目前的檔案。 |
| **[驗證](file_verify)** | 驗證當前文件與原始文件是否相同。 |
| **[列印](file_print)** | 列印目前的檔案。 |
| **[立即列印](file_print_direct)** | 使用預設的印表機而不指定任何選項列印目前的檔案。 |
| **[列印預覽](print_preview)** | 顯示目前的文檔的列印預覽。 |
| **[還原預設工作區](load_workspace)** | 還原一個已儲存的預設工作區狀態。 |
| **[儲存預設工作區](save_workspace)** | 儲存預設工作區。 |
| [**清除預設工作區**](erase_workspace) | 清除預設工作區。 |
| [**打開工作區**](workspace_open) | 打開一個被儲存的工作區檔案。 |
| [**儲存工作區**](workspace_save_current) | 儲存工作區到目前的工作區檔案中。 |
| [**另存新工作區**](workspace_save_as) | 用一個新檔名儲存工作區。 |
| [**添加到我的最愛**](add_to_favorites) | 將目前的檔案添加到我的最愛清單中。 |
| [**要打開的我的最愛檔案**](favorite_file1) | 打開指定的我的最愛檔案(多個項目)。 |
| **[儲存并關閉](file_save_exit)** | 儲存并關閉目前的檔案。 |
| **[關閉](app_exit)** | 關閉目前的檔案。 |
| **[關閉標籤](close_tab)** | 關閉選取的標籤。 |
| **[儲存并全部關閉 (僅目前的桌面)](save_close_desktop)** | 儲存并關閉所有在目前的虛擬桌面上打開的檔案。 |
| **[全部關閉 (僅目前的桌面)](close_desktop)** | 關閉所有在目前的虛擬桌面上打開的檔案。 |
| **[儲存并全部關閉](save_exit_all)** | 儲存并關閉所有打開的檔案。 |
| **[全部關閉](exit_all)** | 關閉所有打開的檔案。 |
| **[關閉所有其他檔案](close_all_others)** | 關閉除了目前工作的檔案之外所以打開的檔案。 |
| [**直接關閉不儲存**](app_quit) | 不儲存就直接關閉目前的檔案。 |
| **[全部關閉不儲存 (僅目前的桌面)](quit_desktop)** | 在目前的虛擬桌面上，關閉所有打開的檔案并不儲存。 |
| **[全部關閉不儲存](quit_all)** | 不儲存就全部關閉目前的檔案。 |
| **[儲存工作區與所有檔案并全部關閉 (僅目前的桌面)](save_workspace_close_desktop)** | 在目前的虛擬桌面上，儲存工作區與所有檔案并關閉所有打開的檔案。 |
| **[儲存工作區并全部關閉 (僅目前的桌面)](save_workspace_quit_desktop)** | 在目前的虛擬桌面上，儲存工作區并關閉所有打開的檔案。 |
| **[儲存工作區，儲存，并全部關閉](save_workspace_exit_all)** | 儲存工作區，儲存，并關閉所有打開的檔案。 |
| **[儲存工作區并全部關閉](save_workspace_quit_all)** | 儲存工作區并關閉所有打開的檔案。 |
| [**最近使用的文檔清單**](file_mru_file1) | 打開一個指定的最近訪問的文檔(多個項目)。 |
| [**最近使用的資料夾清單**](file_mru_folder1) | 打開一個指定的最近訪問的資料夾(多個項目)。 |
| [**最近關閉的檔案**](recent_closed_file1) | 打開一個指定的最近關閉的文檔(多個項目)。 |
| [**最近使用的工作區**](workspace_recent_file1) | 打開一個指定的最近使用過的工作區 (多個項目) 。 |
| **[激活或新增](activate_or_new)** | 激活 EmEditor 或創建一個新檔案(僅用於系統匣圖示)。 |
