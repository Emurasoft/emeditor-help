# 添加到我的最愛命令

## 摘要

將目前的檔案添加到我的最愛清單中。

## 說明

將目前的檔案添加到我的最愛清單中。

## 運行方法

- 預設功能表: **檔案** \> **我的最愛**\> **添加到我的最愛**
- [所有命令](../tools/all_commands): **檔案** \> **我的最愛**\> **添加到我的最愛**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_ADD_TO_FAVORITES (23250)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23250);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23250
```
