# 驗證命令

## 摘要

驗證當前文件與原始文件是否相同。

## 說明

此命令顯示 **[檔案驗證結果](../../dlg/file_verify_result/index.md)** 對話方塊，您可以在其中檢視檔案驗證過程的結果。

## 運行方法

- 預設功能表: **檔案** \> **驗證**
- [全部命令](../tools/all_commands): **檔案** \> **驗證**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_FILE_VERIFY (23279)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(23279);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23279
```
