# 關閉所有其他檔案命令

## 摘要

關閉除了目前工作的檔案之外所以打開的檔案。

## 說明

這個命令會關閉所有已打開的檔案，除了目前的在工作中的檔案。如果存在一個已被修改的文檔，則會顯示一個即時消息來讓您選擇是否想要儲存變更。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **關閉**
\> **關閉所有其他檔案**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CLOSE_ALL_OTHERS (4384)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4384);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4384
```
