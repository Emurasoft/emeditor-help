# 全部關閉 (僅目前的桌面) 命令

### 摘要

> 關閉所有在目前的虛擬桌面上打開的檔案。

### 說明

> 關閉所有在目前的虛擬桌面上打開的檔案。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **關閉**
\> **全部關閉 (僅目前的桌面)**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_CLOSE\_DESKTOP (3935)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3935);

#### \[VBScript\]

> editor.ExecuteCommandByID 3935
