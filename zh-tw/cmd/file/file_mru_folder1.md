# 最近使用的資料夾清單命令

## 摘要

打開一個指定的最近訪問的資料夾 (多個項目) 。

## 說明

這個命令由多個功能表項目組成。有了這個命令，您能選擇一個包含有多個最近打開的檔案的資料夾。所顯示的最近訪問的資料夾數
可以在 **工具** 功能表 (**工具** \> **自訂** \> **歷史記錄**) 下的 [**自訂** 對話方塊](../../dlg/customize/index) 中 [**歷史記錄** 頁面](../../dlg/customize/history/index) 上的 **最近使用的檔案數** 文字方塊中設定。

## 運行方法

- 預設功能表: **系統系統匣圖示功能表** \> **最近使用的資料夾** \> **(選擇資料夾路徑)**
- [全部命令](../tools/all_commands): **檔案** \> **打開**
\> **最近使用的資料夾** \> **(選擇資料夾路徑)**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
從 EEID_FILE_MRU_FOLDER1 到 EEID_FILE_MRU_FOLDER1 + 63 (從 4992
```
到 4992 + 63)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4992 + i);  // i 是一個從 0 到 63 的整數
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4992 + i   ' i 是一個從 0 到 63 的整數
```
