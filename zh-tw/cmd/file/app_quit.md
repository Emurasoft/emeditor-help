# 直接關閉不儲存命令

## 摘要

不儲存就直接關閉目前的檔案。

## 說明

這個命令不儲存就關閉目前的檔案。所有的變更都會被取消。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **關閉** \> **直接關閉不儲存**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_APP_QUIT (4120)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4120);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4120
```
