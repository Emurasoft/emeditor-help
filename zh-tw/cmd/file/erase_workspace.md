# 清除預設工作區命令

## 摘要

清除預設工作區。

## 說明

這個命令會清除以前通過 [**儲存預設工作區** 命令](save_workspace) 儲存的狀態。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **工作區**
\> **清除預設工作區**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ERASE_WORKSPACE (4450)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4450);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4450
```
