# 用指定編碼全部儲存命令

### 摘要

> 除了未命名的文檔外，以指定的目標資料夾，編碼以及換行方式儲存所有打開的文檔。

### 說明

> 這個命令會顯示一個 **用指定編碼全部儲存** 對話方塊。在對話方塊中，您可以用一個特定的編碼和指定的選項來儲存所有打開的檔案 (除了無標題的文檔之外) 。這個對話方塊讓您能選擇目標資料夾，要儲存的編碼，以及換行方式。
>
> 要使用這個命令，您想要儲存的打開文檔必須已經在這個操作之前就已經被儲存過。換句話說，您不能對還未被創建的無標題文檔使用 **用指定編碼全部儲存** 命令。對于這些文檔請使用 **儲存** 或 **另存新檔** 命令。

### 運行方法

- 預設功能表: **檔案** \> **用指定編碼全部儲存**
- [全部命令](../tools/all_commands): **檔案** \> **用指定編碼全部儲存**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_FILE\_SAVE\_ALL\_AS (3843)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3843);

#### \[VBScript\]

> editor.ExecuteCommandByID 3843
