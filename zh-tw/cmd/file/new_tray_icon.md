# 顯示托盤圖示命令

## 摘要

顯示一個新的系統托盤圖示。

## 說明

這個命令會在 Windows 任務欄的通知區域中顯示 EmEditor 的托盤圖示。如果 EmEditor 的托盤圖示已經存在，這個命令不會有任何作用。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **新增** \> **顯示托盤圖示**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_NEW_TRAY_ICON (4121)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4121);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4121
```
