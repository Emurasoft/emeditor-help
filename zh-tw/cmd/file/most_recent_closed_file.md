# 最後關閉的檔案命令

## 摘要

打開最後關閉的一個文件。

## 說明

打開最後關閉的一個文件。這個命令不能恢復文件中未儲存的更改。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):檔案 \>打開
\>最後關閉的檔案
- 工具列:
無
- 狀態列: 無
- 預設捷徑: CTRL + SHFT + T

## 外掛程式命令ID

```
EEID_MOST_RECENT_CLOSED_FILE (4583)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4583);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4583
```
