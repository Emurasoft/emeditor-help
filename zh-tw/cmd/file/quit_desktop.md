# 全部關閉不儲存 (僅目前的桌面) 命令

### 摘要

> 在目前的虛擬桌面上，關閉所有打開的檔案并不儲存。

### 說明

> 在目前的虛擬桌面上，關閉所有打開的檔案并不儲存。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檔案** \> **關閉**
\> **全部關閉不儲存 (僅目前的桌面)**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_QUIT\_DESKTOP (3936)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3936);

#### \[VBScript\]

> editor.ExecuteCommandByID 3936
