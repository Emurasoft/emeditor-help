# 新增檔案，貼為引文并換行命令

## 摘要

創建一個新的檔案，貼上選定內容為引用文字并換行。

## 說明

這個命令等同于 [新增文字 命令](file_new) 加 [貼為引文并換行 命令](../edit/paste_prefix_return)。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):檔案 \>新增 \>新增檔案，貼為引文并換行
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_NEW_PASTE_PREFIX_RETURN (4272)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4272);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4272
```
