# 最近使用的工作區命令

## 摘要

打開一個指定的最近使用過的工作區 (多個項目) 。

## 說明

這個命令由多個功能表項目組成，顯示最近打開的工作區檔案清單。這個命令會打開一個指定的工作區檔案來把目前在 EmEditor 中所有打開的文檔的完整路徑名稱，游標位置，以及其他設定還原成之前通過 [**儲存工作區** 命令](workspace_save_current) 或 [**另存新工作區** 命令](workspace_save_as) 所儲存的檔案的設定。這個命令所顯示的檔案數可以到 **工具** 功能表下的 [**自訂** 對話方塊](../../dlg/customize/index) 中 [**歷史記錄** 頁面](../../dlg/customize/history/index) 上進行設定 (**工具** \> **自訂** \> **歷史記錄**)。在 **最近使用的檔案數** 文字方塊中指定清單上顯示的檔案數目。

## 運行方法

- 預設功能表: **檔案** \> **工作區** \> **(最近使用的工作區)**
- [全部命令](../tools/all_commands): **檔案 > 工作區 > 最近使用的工作區 \> (最近使用的工作區)**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
從 EEID_WORKSPACE_RECENT_FILE1 到 EEID_WORKSPACE_RECENT_FILE1 + 63 (從 22784 到 22784 + 63)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID (22784 + i);  // i 是一個從 0 到 63 的整數
```

### \[VBScript\]

```
editor.ExecuteCommandByID 22784 + i  ' i 是一個從 0 到 63 的整數
```
