# 自訂語言命令

## 摘要

打開自訂對話方塊中的語言頁面。

## 說明

顯示讓您能定義一般應用程式設定的 [**自訂** 對話方塊](../../dlg/customize/index) 中的語言頁面。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**自訂** >
**語言**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_LANG (9047)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9047);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9047
```
