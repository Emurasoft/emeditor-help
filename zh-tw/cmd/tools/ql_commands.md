# 搜尋命令指令

## 摘要

打開 **快速啟動** 視窗并選擇 **命令** 模式。

## 說明

打開 **快速啟動** 視窗并選擇 **命令** 模式。

## 運行方法

- 預設功能表: **說明** \> **搜尋命令**
- [所有命令](all_commands): **說明** \> **搜尋命令**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+SHIFT+C

## 外掛程式命令ID

```
EEID_QL_COMMANDS (3951)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3951);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3951
```
