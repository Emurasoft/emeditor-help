# 自訂優化命令

## 摘要

打開自訂對話方塊中的優化頁面。

## 說明

顯示 [**自訂** 對話方塊](../../dlg/customize/index) 中的優化頁面，讓你能定義通用應用設置。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**自訂** >
**優化**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_OPTIMIZATION (9061)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9061);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9061
```
