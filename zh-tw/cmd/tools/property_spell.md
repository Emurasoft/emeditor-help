# 拼字檢查屬性命令

## 摘要

目前的組態的拼字檢查屬性。

## 說明

顯示[組態屬性](../../dlg/properties/index) 對話方塊中的 [拼字檢查 頁面](../../dlg/properties/spell/index)。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具 >
當前組態屬性 \>拼字檢查
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_PROPERTY_SPELL (8975)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(8975);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8975
```
