# 合併檔案命令

## 摘要

將打開的文檔合併至一個檔案。

## 說明

打開 **[合併檔案精靈](../../dlg/combine_files/index)**，讓您能把打開的文檔合併成一個檔案。

## 運行方法

- 預設功能表: **工具** \> **分割/合併** \> **合併檔案**
- [所有命令](all_commands): **工具**
> **分割/合併** \> **合併檔案**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_COMBINE_FILES (4607)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4607);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4607
```
