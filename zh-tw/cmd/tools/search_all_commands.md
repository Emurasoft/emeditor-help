# 快速啟動命令

## 摘要

快速啟動一個指定的命令。

## 說明

選擇這個命令會顯示 **快速啟動** 視窗，讓您能搜尋所有命令并且運行一個被搜尋的命令。

## 運行方法

- 預設功能表: **工具** \> **快速啟動**
- [所有命令](all_commands): **工具**
\> **快速啟動**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+Q

## 外掛程式命令ID

```
EEID_SEARCH_ALL_COMMANDS (4608)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4608);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4608
```
