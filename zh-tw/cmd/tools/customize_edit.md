# 自訂編輯命令

## 摘要

打開自訂對話方塊中的編輯頁面。

## 說明

顯示讓您能定義一般應用程式設定的 [**自訂** 對話方塊](../../dlg/customize/index) 中的編輯頁面。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**自訂** >
**編輯**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_EDIT (9049)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9049);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9049
```
