# 外部工具清單命令

## 摘要

啟動一個指定的外部工具 (多個項目) 。

## 說明

這個命令由多個功能表項目組成。該命令會啟動指定的外部工具。可用的外部工具可以在 [**外部工具** 對話方塊](../../dlg/tools/index) 中被定義。

## 運行方法

- 預設功能表: **工具** \> **外部工具** \> ( **工具名稱**)
- [所有命令](all_commands): **外部工具**
\> ( **工具名稱**)
- 工具列: 外部工具工具列上的按鈕
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
From EEID_TOOL1 through EEID_TOOL1 + 255 (從 8192 到 8192 + 255)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(8192+i);  // i 是從 0 到 255 的整數
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8192+i  ' i 是從 0 到 255 的整數
```
