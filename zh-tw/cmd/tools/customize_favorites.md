# 自訂我的最愛命令

## 摘要

打開自訂對話方塊中的我的最愛頁面。

## 說明

顯示讓你能定義通用應用設定的 [**自訂** 對話方塊](../../dlg/customize/index) 中的我的最愛頁面。

## 運行方法

- 預設功能表: **檔案** > **我的最愛** > **自訂我的最愛...**
- [所有命令](all_commands): **工具** > **自訂** > **我的最愛**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_FAVORITES (20716)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(20716);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 20716
```
