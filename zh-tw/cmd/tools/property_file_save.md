# 儲存檔案詳細信息屬性命令

### 摘要

> 顯示儲存檔案詳細信息屬性。

### 說明

> 顯示 **[組態屬性](../../dlg/properties/index)** 對話方塊 **檔案** 頁面上的 [**儲存詳細信息** 子對話方塊](../../dlg/properties/file/save_details/index)。

### 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**目前的組態屬性** \> **儲存檔案詳細信息**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_PROPERTY\_FILE\_SAVE (8979)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(8979);

#### \[VBScript\]

> editor.ExecuteCommandByID 8979
