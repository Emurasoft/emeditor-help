# 自訂功能表命令

## 摘要

自訂功能表。

## 說明

顯示 [自訂功能表 對話方塊](../../dlg/menus/index)，讓您能自訂 EmEditor 的功能表。

## 運行方法

- 預設功能表:工具 \>自訂功能表
- [所有命令](all_commands):工具 >
自訂功能表
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_MENU (4266)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4266);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4266
```
