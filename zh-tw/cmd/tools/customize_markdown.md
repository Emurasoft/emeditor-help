# 自訂 Markdown 命令

## 摘要

打開自訂對話方塊中的 Markdown 頁面。

## 說明

顯示讓你能定義通用應用設定的 [**自訂** 對話方塊](../../dlg/customize/index) 中的 Markdown 頁面。

## 運行方法

- 預設功能表: 無
- [全部命令](all_commands): **工具** > **自訂** > **Markdown**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_MARKDOWN (9075)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9075);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9075
```
