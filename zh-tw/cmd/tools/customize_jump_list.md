# 自訂跳轉清單命令

### 摘要

> 自訂跳轉清單。

### 說明

> 自訂跳轉清單。當您右擊任務欄上的 EmEditor 圖示時會跳轉清單。這個功能在 Windows 7 和以后的版本中可用。

### 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**自訂跳轉**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_CUSTOMIZE\_JUMP\_LIST (4455)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(4455);

#### \[VBScript\]

> editor.ExecuteCommandByID 4455
