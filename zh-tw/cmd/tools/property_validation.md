# 語法檢查屬性命令

## 摘要

顯示語法檢查屬性。

## 說明

顯示 **[組態屬性](../../dlg/properties/index)** 對話方塊中的 [**語法檢查** 頁面](../../dlg/properties/validation/index)。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**目前的組態屬性** \> **語法檢查**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_PROPERTY_VALIDATION (8980)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(8980);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8980
```
