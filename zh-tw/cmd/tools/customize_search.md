# 自訂搜尋命令

## 摘要

打開自訂對話方塊中的搜尋頁面。

## 說明

顯示讓您能定義一般應用程式設定的 [自訂 對話方塊](../../dlg/customize/index) 中的搜尋頁面。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具 >
自訂 >
搜尋
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_SEARCH (9041)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9041);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9041
```
