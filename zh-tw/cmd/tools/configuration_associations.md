# 檔案關聯組態命令

## 摘要

將檔案類型關聯到組態項。

## 說明

顯示 [**檔案關聯組態** 對話方塊](../../dlg/configuration_associations/index)，讓您能選取與特定檔案類型相關聯的 EmEditor 組態。

## 運行方法

- 預設功能表: **工具** \> **檔案關聯組態**
- [所有命令](all_commands): **工具** \> **檔案關聯組態**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CONFIG_ASSOCIATE (3847)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3847);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3847
```
