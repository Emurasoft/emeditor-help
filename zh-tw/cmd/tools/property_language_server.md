# 語言伺服器屬性命令

### 摘要

> 目前的組態的語言伺服器屬性。

### 說明

> 顯示 **[組態屬性](../../dlg/properties/index)** 對話方塊中的 [**語言伺服器** 頁面](../../dlg/properties/char_check/index)。

### 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**目前的組態屬性** \> **語言伺服器**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_PROPERTY\_LANGUAGE\_SERVER (8982)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(8982);

#### \[VBScript\]

> editor.ExecuteCommandByID 8982
