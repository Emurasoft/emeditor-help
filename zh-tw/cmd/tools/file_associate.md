# 與 EmEditor 關聯命令

## 摘要

關聯檔案副檔名。

## 說明

顯示 [與 EmEditor 關聯 對話方塊](../../dlg/file_associate/index)，讓使用者可以指定要與 EmEditor 相關聯的檔案副檔名。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具 >
與 EmEditor 關聯
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_FILE\_ASSOCIATE (4223)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4223);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4223
```
