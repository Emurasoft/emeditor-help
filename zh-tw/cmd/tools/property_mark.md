# 標記屬性命令

## 摘要

目前的組態的標記屬性。

## 說明

顯示[組態屬性](../../dlg/properties/index) 對話方塊中的 [標記 頁面](../../dlg/properties/marks/index)。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具 >
當前組態屬性 \>標記
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_PROPERTY_MARK (8971)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(8971);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 8971
```
