# 自訂標記 \- 選項 命令

## 摘要

顯示自訂標記 對話方塊并選擇選項 頁面。

## 說明

顯示 [自訂標記 對話方塊](../../dlg/customize_markers/index)，并選擇選項 頁面。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具
\>自訂標記 \>選項
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_MARKERS_OPTIONS (3958)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3958);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3958
```
