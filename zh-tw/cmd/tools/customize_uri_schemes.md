# 自訂 URI 配置命令

## 摘要

打開自訂對話方塊中的 URI 配置頁面。

## 說明

顯示讓你能定義通用應用設定的 [自訂 對話方塊](../../dlg/customize/index) 中的 URI 配置頁面。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands):工具 >
自訂 >
URI 配置
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_URI_SCHEMES (9065)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(9065);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9065
```
