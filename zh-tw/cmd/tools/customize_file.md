# 自訂檔案命令

### 摘要

> 打開自訂對話方塊中的檔案頁面。

### 說明

> 顯示讓您能定義一般應用程式設定的 [**自訂** 對話方塊](../../dlg/customize/index) 中的檔案頁面。

### 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**自訂** >
**檔案**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令ID

- EEID\_CUSTOMIZE\_FILE (9040)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(9040);

#### \[VBScript\]

> editor.ExecuteCommandByID 9040
