# 重設所有設定命令

## 摘要

重設所有 EmEditor 設置并重啟 EmEditor。

## 說明

重設所有 EmEditor 設置并重啟EmEditor。

## 運行方法

- 預設功能表: **工具**
\> **重設所有設定**
- [所有命令](all_commands): **工具**
\> **重設所有設定**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_RESET_ALL (3860)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3860);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3860
```
