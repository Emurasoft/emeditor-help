# 鍵盤屬性命令

## 摘要

目前的組態的鍵盤屬性。

## 說明

顯示 **[組態屬性](../../dlg/properties/index)** 對話方塊中的 [**鍵盤** 頁面](../../dlg/properties/keyboard/index)。

## 運行方法

- 預設功能表: 無
- [所有命令](all_commands): **工具** >
**當前組態屬性** \> **鍵盤**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_PROPERTY_KEYBOARD (8974)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(8974);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8974
```
