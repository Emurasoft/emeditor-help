# 忽略注釋命令

## 摘要

比較文檔時忽略注釋。

## 說明

比較文檔時忽略注釋。

## 運行方法

- 預設功能表:比較 \>忽略注釋
- [全部命令](../tools/all_commands):比較 \>忽略注釋
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_IGNORE_COMMENT (4503)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4503);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4503
```
