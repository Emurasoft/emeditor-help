# 下一更改的字元命令

## 摘要

跳轉到下一更改的字元處。

## 說明

在比較模式下，跳轉到下一更改的字元處。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **比較** \> **下一更改的字元**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: CTRL+F7

## 外掛程式命令ID

```
EEID_COMPARE_NEXT_CHAR (4090)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4090);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4090
```
