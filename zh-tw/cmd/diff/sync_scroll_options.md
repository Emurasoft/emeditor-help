# 按參數同步捲動命令

## 摘要

啟動同步捲動精靈以同步捲動最近檢視的兩個文檔。

## 說明

啟動同步捲動精靈，用指定參數來同步捲動最近檢視的兩個文檔。

## 運行方法

- 預設功能表:比較 \>按參數同步捲動
- [全部命令](../tools/all_commands):比較 \>按參數同步捲動
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SYNC_SCROLL_OPTIONS (4496)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4496);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4496
```
