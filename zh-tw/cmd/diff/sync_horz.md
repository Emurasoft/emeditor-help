# 同步水平捲動命令

## 摘要

同步水平捲動。

## 說明

在兩個文檔被比較時，同步水平捲動。

## 運行方法

- 預設功能表:比較 \>同步水平捲動
- [全部命令](../tools/all_commands):比較 \>同步水平捲動
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SYNC_HORZ (4506)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4506);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4506
```
