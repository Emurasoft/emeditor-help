# 全部複製到左邊命令

## 摘要

將所有變更複製到左側。

## 說明

將所有變更複製到左側。

## 運行方法

- 預設功能表: 無
- [所有命令](../tools/all_commands): **比較** \> **全部複製到左邊**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_COPY_ALL_TO_LEFT (23247)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23247
```
