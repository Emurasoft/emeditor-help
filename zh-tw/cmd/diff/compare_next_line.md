# 下一更改行命令

## 摘要

跳轉至下一個更改的行。

## 說明

在比較模式下，跳轉至下一個更改的行。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):比較 \>下一更改行
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_COMPARE_NEXT_LINE (4557)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4557);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4557
```
