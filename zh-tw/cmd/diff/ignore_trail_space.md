# 忽略末尾空格命令

## 摘要

比較文檔時忽略每行末尾的空格。

## 說明

比較文檔時忽略每行末尾的空格。

## 運行方法

- 預設功能表: **比較** \> **忽略末尾空格**
- [全部命令](../tools/all_commands): **比較** \> **忽略末尾空格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_IGNORE_TRAIL_SPACE (4499)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4499);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4499
```
