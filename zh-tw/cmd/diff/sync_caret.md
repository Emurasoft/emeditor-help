# 同步游標位置命令

## 摘要

同步游標位置。

## 說明

在兩個文檔被比較時，同步游標位置。

## 運行方法

- 預設功能表: **比較** \> **游標位置**
- [全部命令](../tools/all_commands): **比較** \> **游標位置**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SYNC_CARET (4507)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4507);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4507
```
