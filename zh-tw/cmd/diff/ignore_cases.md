# 忽略大小寫命令

## 摘要

比較文檔時忽略大小寫。

## 說明

比較文檔時忽略大小寫 (不區分大小寫)。

## 運行方法

- 預設功能表:比較 \>忽略大小寫
- [全部命令](../tools/all_commands):比較 \>忽略大小寫
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_IGNORE_CASES (4501)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4501);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4501
```
