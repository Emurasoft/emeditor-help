# 完成后還原視窗位置命令

## 摘要

當比較或同步完成后還原視窗位置。

## 說明

當比較或同步完成或取消后還原原始視窗/標籤位置。

## 運行方法

- 預設功能表: **比較** \> **完成后還原視窗位置**
- [全部命令](../tools/all_commands): **比較** \> **完成后還原視窗位置**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SYNC_RESTORE_POS (4546)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4546);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4546
```
