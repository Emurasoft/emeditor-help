# 為有改動的行設置書籤命令

## 摘要

為所有已修改的行添加書籤。

## 說明

在比較模式下，為所有已修改的行添加書籤。

## 運行方法

- 預設功能表:比較 \>為有改動的行設置書籤
- [全部命令](../tools/all_commands):比較 \>為有改動的行設置書籤
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_COMPARE_BOOKMARK (4593)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4593);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4593
```
