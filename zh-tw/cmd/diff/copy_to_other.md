# 複製到其他命令

## 摘要

將此文檔中的這個變更複製到其他文檔。

## 說明

將此文檔中的這個變更複製到其他文檔。

## 運行方法

- 預設功能表: **比較** \> **複製到其他**
- [所有命令](../tools/all_commands): **比較** \> **複製到其他**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_COPY_TO_OTHER (23247)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23247
```
