# 全部複製到右邊命令

## 摘要

將所有變更複製到右側。

## 說明

將所有變更複製到右側。

## 運行方法

- 預設功能表: 無
- [所有命令](../tools/all_commands): **比較** \> **全部複製到右邊**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_COPY_ALL_TO_RIGHT (23248)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23248
```
