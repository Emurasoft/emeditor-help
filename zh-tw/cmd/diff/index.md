# 比較類別

|     |     |
| --- | --- |
| **[比較](compare_direct)** | 不指定參數比較最后訪問的兩個文檔。 |
| **[按參數比較](compare_options)** | 以指定參數比較最后訪問的兩個文檔。 |
| **[重新掃描](compare_rescan)** | 重新掃描并重新整理所比較的文檔。 |
| **[同步捲動](sync_scroll_direct)** | 不指定參數直接同步捲動兩個最近訪問的文檔。 |
| **[按參數同步捲動](sync_scroll_options)** | 啟動同步捲動精靈以同步捲動最近檢視的兩個文檔。 |
| **[分割檢視](split_view_direct)** | 在一個視窗中分割顯示兩個最近查看的文檔。 |
| **[重設](compare_reset)** | 重設比較或同步捲動模式并清除比較結果。 |
| **[忽略前導空格](ignore_lead_space)** | 比較文檔時忽略每行前導的空格。 |
| **[忽略末尾空格](ignore_trail_space)** | 比較文檔時忽略每行末尾的空格。 |
| **[忽略內嵌空格](ignore_embedded_space)** | 比較文檔時忽略每行內嵌的空格。 |
| **[忽略大小寫](ignore_cases)** | 比較文檔時忽略大小寫。 |
| **[忽略換行符](ignore_returns)** | 比較文檔時忽略換行符。 |
| **[忽略注釋](ignore_comment)** | 比較文檔時忽略注釋。 |
| **[忽略編碼](ignore_encoding)** | 比較文檔時忽略編碼。 |
| **[同步垂直捲動](sync_vert)** | 同步垂直捲動。 |
| **[同步水平捲動](sync_horz)** | 同步水平捲動。 |
| **[同步游標位置](sync_caret)** | 同步游標位置。 |
| **[完成后還原視窗位置](sync_restore_pos)** | 當比較或同步完成后還原視窗位置。 |
| **[儲存參數](compare_save_options)** | 儲存參數來比較或同步捲動。 |
| **[上一更改](compare_prev)** | 跳轉至上一更改。 |
| **[下一更改](compare_next)** | 跳轉至下一更改。 |
| **[上一更改行](compare_prev_line)** | 跳轉至上一個更改的行。 |
| **[下一更改行](compare_next_line)** | 跳轉至下一個更改的行。 |
| **[上一更改的字元](compare_prev_char)** | 跳轉到上一更改的字元處。 |
| **[下一更改的字元](compare_next_char)** | 跳轉到下一更改的字元處。 |
| **[為有改動的行設置書籤](compare_bookmark)** | 為所有已修改的行添加書籤。 |
| **[生成報告](compare_generate_report)** | 根據目前的比較結果生成報告檔案。 |
| **[跳轉到其他](compare_jump_to_other)** | 跳轉到正在與目前的文檔比較或捲動的另一個文檔。 |
| **[全部複製到左邊](copy_all_to_left)** | 將所有變更複製到左側。 |
| **[全部複製到右邊](copy_all_to_right)** | 將所有變更複製到右側。 |
