# 跳轉到其他命令

## 摘要

跳轉到正在與目前的文檔比較或捲動的另一個文檔。

## 說明

跳轉到正在與目前的文檔比較或捲動的另一個文檔。

## 運行方法

- 預設功能表:比較 \>跳轉到其他
- [全部命令](../tools/all_commands):比較 \>跳轉到其他
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_COMPARE_JUMP_TO_OTHER (4048)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4048);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4048
```
