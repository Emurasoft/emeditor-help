# 帶不同標記的 CR 和 LF 命令

## 摘要

顯示帶不同標記的 CR 和 LF。

## 說明

顯示帶不同標記的 CR 和 LF。在預設設定下，EmEditor 會用同樣的標記顯示 CR 和 LF。

## 運行方法

- 預設功能表: **檢視** \> **標記** \> **顯示帶不同標記的 CR 和 LF。**
- [全部命令](../tools/all_commands): **檢視** \> **標記** \> **顯示帶不同標記的 CR 和 LF。**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_VIEW_CR_LF_SEPARATE (4537)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4537);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4537
```
