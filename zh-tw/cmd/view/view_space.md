# 空格命令

## 摘要

顯示或隱藏空格標記。

## 說明

顯示或隱藏空格標記。在預設設定下，EmEditor 不顯示空格標記 (藍色下劃線) 。

## 運行方法

- 預設功能表: **檢視** \> **標記** \> **空格**
- [全部命令](../tools/all_commands): **檢視** \> **標記** \> **空格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_VIEW_SPACE (4535)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4535);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4535
```
