# 清除儲存過的行的標記命令

## 摘要

清除所有指示儲存行的標記。

## 說明

清除所有指示儲存行的標記。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **標記**
\> **清除儲存過的行的標記**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CLEAR_SAVED_MARKERS (4011)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4011);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4011
```
