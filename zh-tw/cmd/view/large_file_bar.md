# 切換大檔案控制器命令

## 摘要

顯示或隱藏大檔案控制器。

## 說明

顯示或隱藏大檔案控制器。當打開一個大檔案時，EmEditor 會顯示大檔案控制器，如果 **當打開一個大檔案時，顯示大檔案控制器** 核取方塊 (在自訂對話方塊 [檔案頁面](../../dlg/customize/file/index) 上) 被勾選了。

## 運行方法

- 預設功能表: **檢視** \> **大檔案控制器**
- [全部命令](../tools/all_commands): **檢視** >
**大檔案控制器**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_LARGE_FILE_BAR (4449)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4449);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4449
```
