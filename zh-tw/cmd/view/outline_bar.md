# 大綱顯示條命令

## 摘要

顯示或隱藏大綱顯示條。

## 說明

顯示或隱藏大綱顯示條。

## 運行方法

- 預設功能表: **檢視** \> **大綱顯示條**
- [全部命令](../tools/all_commands): **檢視** \> **大綱** \> **大綱顯示條**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_OUTLINE_BAR (3872)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3872);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3872
```
