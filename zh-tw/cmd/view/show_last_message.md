# 顯示最后一條消息命令

### 摘要

> 顯示最后收到的消息 (僅在系統匣圖示功能表中顯示)。

### 說明

> 當 EmEditor 從另一臺 PC 上收到一條消息時，通過 **發送選取的文字** 命令，EmEditor 會用一個氣球狀提示或一個 toast 顯示該消息。在氣球狀提示或 toast 消失之後，最后一條收到的消息可以通過使用這個命令被追回。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): 無
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無
