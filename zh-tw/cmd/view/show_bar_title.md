# 顯示工具列標題命令

## 摘要

顯示工具列標題。

## 說明

顯示或隱藏工具列標題。工具列標題包括 **外掛程式**， **工具** 還有 **視窗**。

## 運行方法

- 預設功能表: **檢視** >
**工具列** \> **工具列標題**
- [全部命令](../tools/all_commands): **檢視** >
**工具列** \> **工具列標題**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SHOW_BAR_TITLE (4340)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4340);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4340
```
