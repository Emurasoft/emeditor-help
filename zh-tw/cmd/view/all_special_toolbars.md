# 切換所有特殊工具列命令

## 摘要

顯示或隱藏所有特殊工具列。

## 說明

顯示或隱藏所有特殊工具列。

## 運行方法

- 預設功能表: **檢視** \> **工具列** \> **所有特殊工具列**
- [全部命令](../tools/all_commands): **檢視** >
**工具列** \> **所有特殊工具列**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_ALL_SPECIAL_TOOLBARS (4069)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4069);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4069
```
