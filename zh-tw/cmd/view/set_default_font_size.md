# 把目前的字型設為預設值命令

## 摘要

把目前的字型大小設為預設大小。

## 說明

把目前的字型大小設為預設大小。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **字型** >
**把目前的字型設為預設值**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SET_DEFAULT_FONT_SIZE (3877)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3877);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3877
```
