# 控制字元命令

## 摘要

顯示或隱藏控制字元。

## 說明

顯示或隱藏控制字元。

## 運行方法

- 預設功能表:檢視 \>標記 \>控制字元
- [全部命令](../tools/all_commands):檢視 \>標記 \>控制字元
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_VIEW_CONTROL (4538)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4538);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4538
```
