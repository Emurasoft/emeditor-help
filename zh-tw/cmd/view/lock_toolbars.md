# 鎖定工具列命令

## 摘要

鎖定所有工具列。

## 說明

鎖定工具列如果沒有被鎖定，或解除鎖定工具列如果工具列已被鎖定。當沒有被鎖定時，工具列可以被通過拖曳在每個工具列的左側邊緣上的把手來移動。被鎖定的工具列將不會移動，并且也不會在每個工具列的左側邊緣上顯示把手。

## 運行方法

- 預設功能表: **檢視** >
**工具列** \> **鎖定工具列**
- [全部命令](../tools/all_commands): **檢視** >
**工具列** \> **鎖定工具列**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_LOCK_TOOLBARS (4341)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4341);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4341
```
