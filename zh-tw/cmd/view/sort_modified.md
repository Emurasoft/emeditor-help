# 按修改時間排列命令

## 摘要

按修改時間排列標籤頁。

## 說明

按上次修改時間排列標籤頁 (最近修改的在最左邊)。如果勾選了 [**自動排列** 命令](auto_sort)，每當一個檔案被打開或儲存為新的檔案名時，標籤頁就會按這個命令自動排列。如果 [**自動排列** 命令](auto_sort) 沒有被勾選的話，這個命令只會執行一次。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **標籤排列方式**
\> **已修改**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_MODIFIED (4400)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4400);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4400
```
