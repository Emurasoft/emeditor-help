# 啟用快速鍵 (輸出列) 命令

## 摘要

啟用或停用輸出列內的快速鍵。

## 說明

啟用或停用輸出列內的快速鍵。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** >
**輸出列** \> **啟用快速鍵**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_TOGGLE_OUTPUT_SHORTCUTS (4549)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4549);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4549
```
