# 重設字型大小命令

## 摘要

重設顯示的字型大小。

## 說明

重設顯示的字型大小。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):檢視 \>字型 >
重設字型大小
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+0

## 外掛程式命令ID

```
EEID_RESET_FONT_SIZE (3876)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3876);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3876
```
