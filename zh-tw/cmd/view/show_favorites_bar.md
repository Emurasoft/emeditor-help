# 我的最愛工具列命令

## 摘要

顯示或隱藏我的最愛工具列。

## 說明

顯示或隱藏我的最愛工具列。

## 運行方法

- 預設功能表: **檢視** > **工具列** \> **我的最愛**
- [所有命令](../tools/all_commands): **檢視** > **工具列** \> **我的最愛**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_SHOW_FAVORITES_BAR (23251)
```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(23251);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23251
```
