# 通知命令

## 摘要

顯示或隱藏通知。

## 說明

顯示或隱藏通知。

## 運行方法

- 預設功能表:檢視 \>通知
- [全部命令](../tools/all_commands):檢視 \>通知
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_TOGGLE_NOTIFICATIONS (4061)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4061);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4061
```
