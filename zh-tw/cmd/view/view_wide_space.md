# 全形空格命令

## 摘要

顯示或隱藏全形空格。

## 說明

顯示或隱藏全形空格。在預設設定下，EmEditor 不顯示全形空格標記 (藍色長方形) 。

## 運行方法

- 預設功能表: **檢視** \> **標記** \> **全形空格**
- [全部命令](../tools/all_commands): **檢視** \> **標記** \> **全形空格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_VIEW_WIDE_SPACE (4536)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4536);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4536
```
