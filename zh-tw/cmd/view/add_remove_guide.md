# 添加/刪除使用者自訂參考線命令

## 摘要

在游標位置處添加或刪除使用者自訂參考線 (或在尺規上指定的位置處)。

## 說明

在游標位置處，添加/刪除使用者自訂參考線 (或在尺規所指的位置處)。

## 運行方法

- 預設功能表:檢視 \>標記 \>添加/刪除使用者自訂參考線
- [全部命令](../tools/all_commands):檢視 >
標記 \> 添加/刪除使用者自訂參考線
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ADD_REMOVE_GUIDE (3862)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3862);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3862
```
