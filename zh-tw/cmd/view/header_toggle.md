# 欄標題命令

## 摘要

顯示或隱藏欄標題。

## 說明

顯示或隱藏欄標題。在預設設定下，當你打開一個 CSV 檔案時，EmEditor 會顯示欄標題。

## 運行方法

- 預設功能表:CSV \>欄標題
- [全部命令](../tools/all_commands):CSV \>欄標題
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_HEADER_TOGGLE (3985)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3985);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3985
```
