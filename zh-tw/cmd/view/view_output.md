# 切換輸出列命令

## 摘要

顯示或隱藏輸出列。

## 說明

顯示或隱藏輸出列。輸出列會在一個外部工具啟用時顯示，當 [**使用輸出列** 核取方塊](../../dlg/tools/properties/index) 被勾選。

## 運行方法

- 預設功能表: **檢視** \> **輸出**
- [全部命令](../tools/all_commands): **檢視** >
**輸出列** >
**輸出**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: ALT+2

## 外掛程式命令ID

```
EEID_VIEW_OUTPUT (4420)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4420);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4420
```
