# 計算標記相符項目命令

## 摘要

計算目前的文檔中所有與標記相符的項目數。

## 說明

計算目前的文檔中所有與標記相符的項目數。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **標記**
\> **計算相符項目**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_MARKER_COUNT_MATCHES (4042)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4042);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4042
```
