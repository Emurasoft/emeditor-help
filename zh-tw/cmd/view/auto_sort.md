# 自動排列命令

## 摘要

自動排列標籤頁。

## 說明

如果勾選了這個命令，EmEditor 會自動排列標籤頁。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **檢視** \> **標籤排列方式**
\> **自動排列**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_AUTO_SORT (4404)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4404);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4404
```
