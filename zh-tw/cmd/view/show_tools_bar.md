# 外部工具工具列命令

## 摘要

顯示或隱藏工具列上的外部工具。

## 說明

顯示或隱藏工具列上的外部工具。外部工具工具列會顯示所有被使用者定義的外部工具。

## 運行方法

- 預設功能表: **檢視** >
**工具列** \> **外部工具工具列**
- [全部命令](../tools/all_commands): **檢視** >
**工具列** \> **外部工具工具列**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SHOW_TOOLS_BAR (4319)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4319);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4319
```
