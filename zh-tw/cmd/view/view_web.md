# 網頁瀏覽器命令

## 摘要

顯示或隱藏網頁瀏覽器。

## 說明

顯示或隱藏網頁瀏覽器。

## 運行方法

- 預設功能表: **檢視** \> **網頁瀏覽器**
- [所有命令](../tools/all_commands): **檢視** \> **網頁瀏覽器**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

```
EEID_VIEW_WEB (23243)```

## 巨集

### \[JavaScript\]

```
WebBar.Visible = !WebBar.Visible;
```

### \[VBScript\]

```
WebBar.Visible = Not WebBar.Visible
```
