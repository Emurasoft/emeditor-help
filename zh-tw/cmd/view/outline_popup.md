# 大綱顯示 (快顯功能表) 命令

## 摘要

顯示一個快顯功能表來指定大綱設定。

## 說明

顯示一個快顯功能表來指定大綱設定。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):檢視 >
大綱 >
大綱 (快顯功能表)
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_OUTLINE_POPUP (3875)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3875);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3875
```
