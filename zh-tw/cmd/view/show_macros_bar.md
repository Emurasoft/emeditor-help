# 巨集工具列命令

## 摘要

顯示或隱藏巨集工具列。

## 說明

顯示或隱藏巨集工具列。巨集工具列會顯示所有已定義的「我的巨集」。

## 運行方法

- 預設功能表:檢視 >
工具列 \>巨集工具列
- [全部命令](../tools/all_commands):檢視 >
工具列 \>巨集工具列
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SHOW_MACROS_BAR (4421)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4421);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4421
```
