# 用字母字元作為欄標題命令

## 摘要

使用字母字元作為欄標題。

## 說明

使用字母字元作為欄標題。

## 運行方法

- 預設功能表: **CSV** \> **欄標題類型** \> **A, B, C, ...**
- [全部命令](../tools/all_commands): **CSV** \> **欄標題類型** \> **A, B, C, ...**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_HEADER_ALPHA (3987)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3987);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3987
```
