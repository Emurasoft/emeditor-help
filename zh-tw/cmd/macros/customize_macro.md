# 自訂巨集命令

## 摘要

自訂巨集。

## 說明

顯示 [**自訂巨集** 對話方塊](../../dlg/macro_customize/index)， 讓您能自訂巨集的設定。

## 運行方法

- 預設功能表: **巨集** \> **自訂**
- [全部命令](../tools/all_commands): **巨集**
\> **自訂**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CUSTOMIZE_MACRO (4350)```

## 巨集

不可用。
