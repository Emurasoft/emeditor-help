# 設為目前的巨集命令

## 摘要

把已打開的檔案設定為目前的巨集。

## 說明

把目前的檔案設置為目前的巨集。在 [**運行巨集** 命令](quick_macro_run) 的功能表上，它會顯示檔案名并且會讓您知道這個檔案已被選取。

## 運行方法

- 預設功能表: **巨集** \> **設為目前的巨集**
- [全部命令](../tools/all_commands): **巨集**
\> **設為目前的巨集**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_MACRO_SELECT_THIS (4349)```

## 巨集

不可用。
