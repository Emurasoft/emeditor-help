# 向右延伸一個字元 (垂直選擇模式)命令

## 摘要

將垂直選擇向右延伸一個字元。

## 說明

將垂直選擇向右延伸一個字元。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>延伸選區
\>向右延伸一個字元 (垂直選擇模式)
- 工具列: 無
- 狀態列: 無
- 預設捷徑: ALT+CTRL+向右鍵

## 外掛程式命令ID

```
EEID_BOX_EXTEND_RIGHT (4025)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4025);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4025
```
