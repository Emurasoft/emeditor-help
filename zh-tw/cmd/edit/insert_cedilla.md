# 插入下加符號命令

## 摘要

在輸入字元 c 或 C 時在該字元上插入下加符號。

## 說明

在選擇這個命令后，在游標處插入一個下加符號 (ç 或 Ç) 通過輸入 c 或 C。

## 運行方法

- 預設功能表: **插入** \> **讀音符號** \> **下加符號**
- [全部命令](../tools/all_commands): **插入** \> **讀音符號** \> **下加符號**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL + ,

## 外掛程式命令ID

```
EEID_INSERT_CEDILLA (4310)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4310);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4310
```
