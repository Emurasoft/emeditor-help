# 目前文檔中的下一個書籤（周圍）命令

## 摘要

轉到目前的文檔下一書籤處，如果找不到，就轉到目前的的第一個書籤處。

## 說明

轉到目前的文檔下一書籤處，如果找不到，就轉到目前的的第一個書籤處。

## 運行方法

- 預設功能表:書籤 \>下一個書籤（周圍）
- [全部命令](../tools/all_commands):書籤 \>下一個書籤（周圍）
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_NEXT_AROUND (3999)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3999);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3999
```
