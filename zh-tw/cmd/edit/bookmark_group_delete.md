# 刪除目前的群組中書籤標示行命令

## 摘要

刪除目前的群組中所有書籤標示行。

## 說明

刪除目前的群組中所有書籤標示行。

## 運行方法

- 預設功能表:書籤 \>目前的群組 \>刪除書籤標示行
- [全部命令](../tools/all_commands):書籤 \>目前的群組 \>刪除書籤標示行
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_GROUP_DELETE (4596)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4596);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4596
```
