# IPv4 位址降序排序命令

## 摘要

把目前的欄中的 IPv4 位址按降序排序。

## 說明

把目前的欄中的 IPv4 位址按降序排序。

## 運行方法

- 預設功能表:編輯 \>排序 \>IPv4 位址降序排序
- [全部命令](../tools/all_commands):編輯 \>排序 \>IPv4 位址降序排序
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_IPV4_D (4035)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4035);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4035
```
