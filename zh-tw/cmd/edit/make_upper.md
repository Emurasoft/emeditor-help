# 大寫命令

## 摘要

把選定文字全部轉換為大寫字元。

## 說明

把選定文字全部轉換為大寫字元。例如，a 會變成 A, ä 會變成 Ä，還有 λ 會變成 Λ.

## 運行方法

- 預設功能表:轉換 \>大寫
- [全部命令](../tools/all_commands):轉換 \>大寫
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+SHIFT+U

## 外掛程式命令ID

```
EEID_MAKE_UPPER (4149)```

## 巨集

### \[JavaScript\]

```
document.selection.ChangeCase(eeCaseUpperCase);
```

### \[VBScript\]

```
document.selection.ChangeCase eeCaseUpperCase
```
