# 在欄中篩選命令

### 摘要

> 用所選文字或游標處的單字篩選目前的欄中的文檔。

### 說明

> 用所選文字或游標處的單字篩選目前的欄中的文檔。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **高級** \> **在欄中篩選**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_FILTER\_COLUMN (3983)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3983);

#### \[VBScript\]

> editor.ExecuteCommandByID 3983
