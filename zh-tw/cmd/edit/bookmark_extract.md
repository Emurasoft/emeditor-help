# 擷取目前的文件中的書籤標示行到新檔案中命令

## 摘要

抽出目前的文檔中所有書籤標示行并複製到一個新檔案中。

## 說明

抽出目前的文檔中所有書籤標示行并複製到一個新文檔中。這個命令不會從原來的文檔中刪除書籤標示行。

## 運行方法

- 預設功能表: **書籤** \> **擷取書籤標示行至新檔案**
- [全部命令](../tools/all_commands): **書籤** \> **擷取書籤標示行至新檔案**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_EXTRACT (4590)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4590);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4590
```
