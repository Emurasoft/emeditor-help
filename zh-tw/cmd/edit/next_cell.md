# 下一儲存格命令

## 摘要

在 CSV 模式下跳轉至下一儲存格。

## 說明

在 CSV 模式下跳轉至下一儲存格。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **CSV/排序** \> **下一儲存格**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_NEXT_CELL (4490)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4490);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4490
```
