# 欄標題唯讀命令

### 摘要

> 切換選項使欄標題唯讀。

### 說明

> 切換選項使欄標題唯讀。

### 運行方法

- 預設功能表: **CSV** \> **標題（凍結窗格）** \> **欄標題唯讀**
- [全部命令](../tools/all_commands): **CSV** \> **標題（凍結窗格）** \> **欄標題唯讀**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令 ID

- EEID\_HEADING\_POPUP (3900)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3900);

#### \[VBScript\]

> editor.ExecuteCommandByID 3900
