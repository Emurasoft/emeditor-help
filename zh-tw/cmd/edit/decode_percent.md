# 將百分號編碼轉換為 Unicode (目前的編碼)命令

### 摘要

> 用目前的編碼把選中的百分號編碼文字解碼為 Unicode。

### 說明

> 用目前的編碼把選中的文字從百分號編碼解碼為相對應的 Unicode。

### 運行方法

- 預設功能表: **轉換** \> **編碼/解碼** \> **將百分號編碼轉換為 Unicode (目前的編碼)**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼** \> **將百分號編碼轉換為 Unicode (目前的編碼)**
- 工具列:
None
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_DECODE\_PERCENT (3854)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3854);

#### \[VBScript\]

> editor.ExecuteCommandByID 3854
