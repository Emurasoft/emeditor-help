# 文檔頂部命令

## 摘要

移動游標到目前的文檔的頂部。

## 說明

把游標移到文檔的開頭或頂部。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>垂直移動游標
\>文檔頂部
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+HOME

## 外掛程式命令ID

```
EEID_TOP (4168)```

## 巨集

### \[JavaScript\]

```
document.selection.StartOfDocument(false);
```

### \[VBScript\]

```
document.selection.StartOfDocument false
```
