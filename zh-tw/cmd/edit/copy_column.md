# 復制欄命令

## 摘要

復制選取的欄或目前的欄并把內容貼上到剪貼簿中。

## 說明

復制選取的欄或目前的欄并把內容貼上到剪貼簿中。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>復制 \>復制欄
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_COPY\_COLUMN (3964)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3964);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3964
```
