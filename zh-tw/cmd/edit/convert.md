# 半形/全形轉換命令

## 摘要

轉換選定文字中的半形和全形字元。

## 說明

這個命令會把半形字元轉換為全形字元，或者把全形字元轉換為半形字元。當執行該命令時， [「半形/全形轉換」 對話方塊](../../dlg/convert/index) 會顯示，您可以設置字元類型以及範圍。

## 運行方法

- 預設功能表:轉換 \>半形/全形轉換
- [全部命令](../tools/all_commands):轉換 \>半形/全形轉換
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+T

## 外掛程式命令ID

```
EEID_CONVERT (4148)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4148);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4148
```
