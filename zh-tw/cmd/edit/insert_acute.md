# 插入尖音符號命令

## 摘要

插入一個帶有尖音符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有尖音符號的字元 (á，é，
í，ó，ú，ý，ś，ź，Á，É，Í，
Ó，Ú，Ý，Ś 或 Ź) 通過輸入 a，e，i，o，u，y，s，z，A，E，I，O，U，Y，S 或 Z；一個單引號 (’) 通過輸入 \` ；或一個雙引號 (”) 通過輸入 " 。

## 運行方法

- 預設功能表:插入 \>讀音符號 \>尖音
- [全部命令](../tools/all_commands):插入 \>讀音符號 \>尖音
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+'

## 外掛程式命令ID

```
EEID_INSERT_ACUTE (4304)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID (4304);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4304
```
