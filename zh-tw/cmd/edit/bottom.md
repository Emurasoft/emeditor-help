# 文檔底部命令

## 摘要

移動游標到目前的文檔的底部。

## 說明

移動游標到目前的文檔的末尾。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **垂直移動游標**
\> **文檔底部**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+END

## 外掛程式命令ID

```
EEID_BOTTOM (4169)```

## 巨集

### \[JavaScript\]

```
document.selection.EndOfDocument(false);
```

### \[VBScript\]

```
document.selection.EndOfDocument false
```
