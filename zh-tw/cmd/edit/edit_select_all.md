# 全選命令

## 摘要

選擇全部。

## 說明

選擇目前的文檔中的所有字元。

## 運行方法

- 預設功能表:編輯 \>全選
- [全部命令](../tools/all_commands):編輯 \>延伸選區
\>全選
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+A

## 外掛程式命令ID

```
EEID_EDIT_SELECT_ALL (4136)```

## 巨集

## \[JavaScript\]

```
document.selection.SelectAll();
```

## \[VBScript\]

```
document.selection.SelectAll
```
