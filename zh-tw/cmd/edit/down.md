# 下一行命令

## 摘要

移動游標到下一行。

## 說明

把游標向下移動一行。等同于按一次下移箭頭鍵。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>垂直移動游標
\>下一行
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: 下移箭頭

## 外掛程式命令ID

```
EEID_DOWN (4161)```

## 巨集

### \[JavaScript\]

```
document.selection.LineDown(false,1);
```

### \[VBScript\]

```
document.selection.LineDown false,1
```
