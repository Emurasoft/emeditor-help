# 固定欄寬命令

## 摘要

用目前的分隔符位置把目前的含有分隔值的文檔轉換為固定欄寬。

## 說明

用目前的分隔符位置把目前的含有分隔值的文檔轉換為固定欄寬。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):CSV \>轉換為 \>固定欄寬
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_CONVERT\_TO\_FIXED\_WIDTH (3883)

## 巨集

### \[JavaScript\]

```
>editor.ExecuteCommandByID(3883);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3883
```
