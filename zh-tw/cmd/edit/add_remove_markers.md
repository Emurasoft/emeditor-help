# 添加/刪除標記命令

## 摘要

將已選中的文字添加至標記清單或從標記清單中刪除已選中的文字。

## 說明

將已選中的文字添加至標記清單或從標記清單中刪除已選中的文字如果文字已被添加至清單。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **進階**
\> **添加/刪除標記**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ADD_REMOVE_MARKERS (4591)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4591);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4591
```
