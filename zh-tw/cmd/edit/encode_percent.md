# 將 Unicode 轉換為百分號編碼 (目前的編碼) 命令

### 摘要

> 用當前編碼對選中的文字進行百分號編碼。

### 說明

> 用當前編碼對選中的文字進行百分號編碼。

### 運行方法

- 預設功能表: **轉換** \> **編碼/解碼** \> **將 Unicode 轉換為百分號編碼 (目前的編碼)**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼** \> **將 Unicode 轉換為百分號編碼 (目前的編碼)**
- 工具列:
None
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令ID

- EEID\_ENCODE\_PERCENT (3855)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3855);

#### \[VBScript\]

> editor.ExecuteCommandByID 3855
