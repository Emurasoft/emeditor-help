# 唯讀命令

### 摘要

> 切換目前的文件的唯讀狀態。

### 說明

> 啟用或禁用文件的唯讀狀態。

### 運行方法

- 預設功能表: **編輯** \> **唯讀**
- [全部命令](../tools/all_commands): **編輯** \> **進階**
\> **唯讀**
- 工具列: 無
- 狀態列: 雙擊 **「唯讀」**
- 預設鍵盤快速鍵: 無

### 外掛程式命令ID

- EEID\_READ\_ONLY (4113)

### 巨集

#### \[JavaScript\]

> document.ReadOnly=true;

或

> document.ReadOnly=false;

#### \[VBScript\]

> document.ReadOnly=true

或

> document.ReadOnly=false
