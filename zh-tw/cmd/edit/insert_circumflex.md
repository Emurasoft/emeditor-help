# 插入抑揚符號命令

## 摘要

插入一個帶有抑揚符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有抑揚符號的字元 (â，ê，î，ô，û，Â，Ê，Î，
Ô 或 Û) 通過輸入 a，e，i，o，u，A，E，I，O 或 U。

## 運行方法

- 預設功能表:插入 \>讀音符號 \>抑揚符號
- [全部命令](../tools/all_commands):插入 \>讀音符號 \>抑揚符號
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+SHIFT+6

## 外掛程式命令ID

```
EEID_INSERT_CIRCUMFLEX (4305)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID (4305);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4305
```
