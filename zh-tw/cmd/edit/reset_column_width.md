# 重設欄寬命令

## 摘要

重設 CSV 文檔中所選欄的寬度。

## 說明

重設 CSV 文檔中所選欄的寬度。

## 運行方法

- 預設功能表:CSV \>欄 \>重設欄寬
- [全部命令](../tools/all_commands):CSV \>欄 \>重設欄寬
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_RESET\_COLUMN\_WIDTH (4052)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4052);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4052
```
