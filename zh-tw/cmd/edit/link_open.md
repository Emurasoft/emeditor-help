# 打開連結命令

## 摘要

打開一個超連結。

## 說明

在游標處打開一個超連結。

## 運行方法

- 預設功能表:編輯 \>打開連結
- [全部命令](../tools/all_commands):編輯 \>進階
\>打開連結
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: 無

## 外掛程式命令ID

```
EEID_LINK_OPEN (4141)```

## 巨集

## \[JavaScript\]

```
document.selection.OpenLink();
```

## \[VBScript\]

```
document.selection.OpenLink
```
