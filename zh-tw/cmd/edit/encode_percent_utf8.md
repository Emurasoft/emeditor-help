# 將 Unicode 轉換為百分號編碼 (UTF-8) 命令

## 摘要

用 UTF-8 編碼對選中的文字進行百分號編碼。

## 說明

用 UTF-8 編碼對選中的文字進行百分號編碼。

## 運行方法

- 預設功能表:轉換 \>編碼/解碼 \>將 Unicode 轉換為百分號編碼(UTF-8)
- [全部命令](../tools/all_commands):轉換 \>編碼/解碼 \>將 Unicode 轉換為百分號編碼(UTF-8)
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ENCODE_PERCENT_UTF8 (3857)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3857);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3857
```
