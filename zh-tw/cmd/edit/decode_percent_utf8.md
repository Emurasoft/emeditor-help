# 將百分號編碼轉換為 Unicode (UTF-8)命令

## 摘要

用UTF-8編碼把選中的百分比編碼文字解碼為 Unicode。

## 說明

把選中的文字用UTF-8編碼從百分比編碼解碼為相對應的 Unicode 文字。

## 運行方法

- 預設功能表:轉換 \>編碼/解碼 \>將百分號編碼轉換為 Unicode (UTF-8)
- [全部命令](../tools/all_commands):轉換 \>編碼/解碼 \>將百分號編碼轉換為 Unicode (UTF-8)
- 工具列: 無e
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_DECODE_PERCENT_UTF8 (3856)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3856);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3856
```
