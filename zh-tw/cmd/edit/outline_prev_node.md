# 上一個節點命令

## 摘要

當顯示大綱視圖時移動游標到上一個節點。

## 說明

當顯示大綱視圖時移動游標到上一個節點。要使用該功能，先要啟用大綱來顯示大綱視圖。

## 運行方法

- 預設功能表: **編輯** \> **大綱** \> **上一個節點**
- [全部命令](../tools/all_commands): **編輯** \> **大綱** \> **上一個節點**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: Alt+Up

## 外掛程式命令ID

```
EEID_OUTLINE_PREV_NODE (4416)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4416);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4416
```
