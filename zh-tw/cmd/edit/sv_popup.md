# CSV (快顯功能表) 命令

### 摘要

> 顯示一個能選擇 CSV 模式的快顯功能表。

### 說明

> 顯示一個能選擇 CSV 模式的快顯功能表。

### 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **CSV** \> **CSV (快顯功能表)**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

### 外掛程式命令 ID

- EEID\_SV\_POPUP (3899)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3899);

#### \[VBScript\]

> >editor.ExecuteCommandByID 3899
