# 取消多選區最下方選取命令

## 摘要

取消多選區最下方的選取。

## 說明

取消多選區最下方的選取。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>延伸選區
\>取消多選區最下方選取
- 工具列: 無
- 狀態列: 無
- 預設捷徑: ALT+CTRL+SHIFT+向下鍵

## 外掛程式命令ID

```
EEID_CANCEL_BOTTOM_SEL (4031)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4031);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4031
```
