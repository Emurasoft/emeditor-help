# 切換輸入法命令

## 摘要

為遠東語言啟用或禁用輸入法編輯器。

## 說明

為遠東語言啟用或禁用輸入法編輯器。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **進階** \> **輸入法**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+F11

## 外掛程式命令ID

```
EEID_TOGGLE_IME (4198)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4198);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4198
```
