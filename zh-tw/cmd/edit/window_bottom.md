# 視窗底部命令

## 摘要

移動游標到視窗底部。

## 說明

移動游標到視窗底部。這個命令不會移動垂直捲軸的位置。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>垂直移動游標
\>視窗底部
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: ALT+PAGE DOWN

## 外掛程式命令ID

```
EEID_WINDOW_BOTTOM (4293)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4293);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4293
```
