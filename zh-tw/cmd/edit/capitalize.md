# 首字母大寫命令

## 摘要

讓選區中每一個單字的首字母大寫。

## 說明

讓選區中每一個單字的首字母大寫，并且把單字中的其他字母轉換為小寫。

## 運行方法

- 預設功能表: **轉換** \> **首字母大寫**
- [全部命令](../tools/all_commands): **轉換** \> **首字母大寫**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CAPITALIZE (4381)```

## 巨集

### \[JavaScript\]

```
document.selection.ChangeCase(eeCaseCapitalize);
```

### \[VBScript\]

```
document.selection.ChangeCase eeCaseCapitalize
```
