# Unicode 正規化表單 D (標準分解) 命令

## 摘要

將 Unicode 正規化表單 D (標準分解) 應用於所選字串。

## 說明

將在「 [Unicode 標準第 15 號附件 Unicode Normalization Forms](http://unicode.org/reports/tr15/)」中定義的 Unicode 正規化表單 D (標準分解) 應用於所選字串。

## 運行方法

- 預設功能表:轉換 \>編碼/解碼 \>Unicode 正規化表單 D (標準分解)
- [全部命令](../tools/all_commands):轉換 \>編碼/解碼 \>Unicode 正規化表單 D (標準分解)
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_UNICODE\_NORM\_FD (4054)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4054);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4054
```
