# 右移到單字末尾命令

## 摘要

將游標向右移動到單字的末尾。

## 說明

將游標向右移動到單字的末尾。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>水平移動游標
\>右移到單字末尾
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+向右鍵

## 外掛程式命令ID

```
EEID_RIGHT_WORD_END (4030)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4030);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4030
```
