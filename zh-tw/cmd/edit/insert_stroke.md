# 插入畫線符號命令

## 摘要

在輸入字元 o、O 或 c 時在該字元上插入畫線符號。

## 說明

在選擇這個命令后，在游標處插入一個帶有畫線符號的字元 (ø 或 Ø) 通過輸入 o 或
O，或一個美分標記 (¢) 通過輸入 c。

## 運行方法

- 預設功能表:插入 \>讀音符號 \>畫線符號
- [全部命令](../tools/all_commands):插入 \>讀音符號 \>畫線符號
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+/

## 外掛程式命令ID

```
EEID_INSERT_STROKE (4311)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4311);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4311
```
