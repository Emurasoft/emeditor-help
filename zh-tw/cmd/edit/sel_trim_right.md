# 刪除尾端空格命令

## 摘要

刪除選區內尾端的空格，tab，以及全形空格。

## 說明

刪除選區內尾端的空格，tab，以及全形空格。

## 運行方法

- 預設功能表:轉換 \>刪除尾端空格
- [全部命令](../tools/all_commands):轉換 \>刪除尾端空格
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SEL_TRIM_RIGHT (4016)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4016);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4016
```
