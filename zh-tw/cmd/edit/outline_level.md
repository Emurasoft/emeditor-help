# 級別 2-16 命令

## 摘要

當顯示大綱時折疊或展開整個文檔中的第 2-16 級別。

## 說明

在整個文檔中折疊或展開特定的級別 (2-16) 當顯示大綱時。要使用該功能，先要啟用大綱來顯示大綱視圖。

## 運行方法

- 預設功能表:編輯
\>大綱
\>第 2-5 級別
- [全部命令](../tools/all_commands):編輯
\>大綱
\>級別
\>第 2-16 級別
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_OUTLINE_LEVEL_2 (8832) - ID_OUTLINE_LEVEL_16 (8846)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(8832);
\-
editor.ExecuteCommandByID(8846);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 8832
\-
editor.ExecuteCommandByID 8846
```
