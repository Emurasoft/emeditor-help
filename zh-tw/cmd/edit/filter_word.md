# 用選取內容或游標處的單字篩選

## 摘要

用選取文字或游標處的單字篩選文檔。

## 說明

用選取文字或游標處的單字篩選文檔。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **進階**
\> **篩選器**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_FILTER_WORD (3929)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3929);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3929
```
