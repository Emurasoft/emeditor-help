# 剪下目前的文件中書籤標示行命令

## 摘要

剪下當前文檔中所有書籤標示行并把它們移動到剪貼簿。

## 說明

剪下當前文檔中所有書籤標示行并把它們移動到剪貼簿。

## 運行方法

- 預設功能表:書籤 \>剪下書籤標示行
- [全部命令](../tools/all_commands):書籤 \>剪下書籤標示行
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_CUT (3852)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3852);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3852
```
