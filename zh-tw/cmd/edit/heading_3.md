# 欄標題 3 命令

## 摘要

把前三行作為標題并且固定不捲動。

## 說明

把前三行作為標題并且固定不捲動。

- 預設功能表:CSV \>標題（凍結窗格） \>欄標題 3
- [全部命令](../tools/all_commands):CSV \>標題（凍結窗格） \>欄標題 3
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無
