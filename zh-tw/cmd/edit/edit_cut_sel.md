# 剪下選區命令

## 摘要

剪下選定內容至剪貼簿。

## 說明

剪下選定內容并把它放到剪貼簿上。在這個命令后，您可以通過把游標移動到另一個地方并且運行 [貼上 指示詞](edit_paste) 來放置文字。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>剪下
\>剪下選區
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_EDIT_CUT_SEL (4444)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4444);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4444
```
