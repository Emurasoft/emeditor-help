# 后退命令

## 摘要

移動游標至上一位置。

## 說明

移動游標至上一位置。

## 運行方法

- 預設功能表: **編輯** \> **進階** \> **后退**
- [全部命令](../tools/all_commands): **編輯** \> **進階** \> **后退**
- 工具列:
![](../../images/emeditor12_back_button.png)
- 狀態列: 無
- 預設捷徑: ALT + LEFT

## 外掛程式命令ID

```
EEID_CARET_BACK (4597)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4597);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4597
```
