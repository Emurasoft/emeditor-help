# 重複上一個指令命令

## 摘要

重複上一個執行的指令。

## 說明

重複上一個選擇執行的指令。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **工具** \> **重做** \> **重複上一個指令**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+Shift+G

## 外掛程式命令ID

```
EEID_REPEAT_LAST_COMMAND (3846)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3846);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3846
```
