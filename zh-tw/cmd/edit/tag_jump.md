# 跳轉至標記命令

## 摘要

跳轉至標記。

## 說明

這個命令可以在運行 [在文件中尋找 命令](../search/grep) 之後使用，或當您要解析錯誤日志以尋找一個字串或一個錯誤時。如果一個文件的路徑和一個行號被顯示在一行的起始位置處，這個命令會打開該文件并且跳轉至有問題的行號。

## 運行方法

- 預設功能表:編輯 \>進階 \>跳轉至標記
- [全部命令](../tools/all_commands):編輯 \>進階
\>跳轉至標記
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: F10

## 外掛程式命令ID

```
EEID_TAG_JUMP (4147)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4147);
```

### \[VBScript\]

```

```
