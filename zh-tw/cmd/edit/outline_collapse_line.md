# 折疊目前的行命令

## 摘要

當顯示大綱視圖時折疊目前的行。

## 說明

當顯示大綱視圖時，這個命令會折疊目前的行。這與點擊文字行左邊的「-」按鈕效果一樣。要使用該功能，先要啟用大綱來顯示大綱視圖。

## 運行方法

- 預設功能表: **編輯** \> **大綱** \> **折疊目前的行**
- [全部命令](../tools/all_commands): **編輯** \> **大綱** \> **折疊目前的行**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_OUTLINE_COLLAPSE_LINE (4413)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4413);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4413
```
