# 方塊垂直選取命令

## 摘要

切換方塊垂直選取模式。

## 說明

切換方塊垂直選取模式。該命令讓您用鍵盤亮顯多欄的字元。用箭頭鍵移動游標來延伸或縮短選區。選擇 [複製 命令](edit_copy) 或 [剪下 命令](edit_cut) 將終止列選擇模式。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>延伸選區 \>方塊垂直選取
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+Shift+F8

## 外掛程式命令ID

```
EEID_SELECT_BOX (4155)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4155);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4155
```
