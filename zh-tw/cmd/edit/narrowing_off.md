# 停用僅編輯選定區域模式命令

## 摘要

使整個文檔可編輯。

## 說明

使整個文檔可編輯。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **僅編輯選定區域模式** \> **停用僅編輯選定區域模式**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_NARROWING_OFF (4458)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4458);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4458
```
