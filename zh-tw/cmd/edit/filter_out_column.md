# 在欄中篩選除外命令

## 摘要

篩選目前的欄中的文檔而不使用選取的文字或游標處的單字。

## 說明

篩選目前的欄中的文檔而不使用選取的文字或游標處的單字。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **高級** \> **在欄中篩選除外**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_FILTER\_OUT\_COLUMN (4012)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4012);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4012
```
