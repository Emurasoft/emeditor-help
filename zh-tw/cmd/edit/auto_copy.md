# 自動復制命令

## 摘要

切換自動復制模式。

## 說明

切換自動復制模式。

## 運行方法

- 預設功能表: **編輯** \> **CSV** \> **自動復制**
- [全部命令](../tools/all_commands): **編輯** \> **復制** \> **自動復制**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_AUTO\_COPY (3979)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3979);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3979
```
