# 對單引號內的字串進行拼字檢查命令

## 摘要

對單引號內的字串進行拼字檢查。

## 說明

對單引號內的字串進行拼字檢查。錯誤拼字的字詞會以紅色下劃線顯示。

## 運行方法

- 預設功能表:編輯 \>拼字檢查 \>檢查拼字于 \>單引號內的字串
- [全部命令](../tools/all_commands):編輯 \>拼字檢查 \>檢查拼字于 \>單引號內的字串
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SPELL_SINGLE_QUOTES (4514)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4514);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4514
```
