# 跳轉到上一個命令

## 摘要

跳轉到輸出欄中的上一個位置。

## 說明

跳轉到輸出欄中的上一個位置。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **進階**
\> **跳轉到上一個**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: SHIFT+F4

## 外掛程式命令ID

```
EEID_JUMP_PREV (4443)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4443);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4443
```
