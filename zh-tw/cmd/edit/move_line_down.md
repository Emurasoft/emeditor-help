# 向下移動行命令

## 摘要

將已選中的行或目前的行下移一行。

## 說明

將已選中的行或目前的行下移一行。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **移動**
\> **向下移動行**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL + SHIFT + DOWN

## 外掛程式命令ID

```
EEID_MOVE_LINE_DOWN (3859)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3859);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3859
```
