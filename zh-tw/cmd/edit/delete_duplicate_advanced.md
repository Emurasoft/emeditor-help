# 刪除/把重複行設為書籤 (進階) 命令

## 摘要

顯示一個對話方塊來指定設置并刪除在選區或整個文檔中的重複行。

## 說明

顯示一個對話方塊來指定設置并刪除在選區或整個文檔中的重複行。當執行該命令時，會顯示 [刪除/把重複行設為書籤 對話方塊](../../dlg/delete_duplicate_advanced/index)，讓您能設置選項。

## 運行方法

- 預設功能表:排序 \>刪除/把重複行設為書籤 (進階)
- [全部命令](../tools/all_commands):排序 \>刪除/把重複行設為書籤 (進階)
- 工具列: ![](../../images/deleteduplicatelines_advanced.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_DELETE_DUPLICATE_ADVANCED (3920)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3920);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3920
```
