# 完全刪除命令

## 摘要

刪除選取的文字以及儲存格。

## 說明

刪除選取的文字以及儲存格。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>刪除 \>完全刪除
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_DELETE_COMPLETE (3961)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3961);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3961
```
