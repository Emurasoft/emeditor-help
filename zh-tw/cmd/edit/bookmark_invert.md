# 倒置目前的文件中的書籤命令

## 摘要

倒置目前的文件中全部的書籤。

## 說明

倒置目前的文檔中的所有書籤，為沒有書籤標示的行添加書籤，并清除已添加書籤行的書籤。

## 運行方法

- 預設功能表: **書籤** \> **倒置書籤**
- [全部命令](../tools/all_commands): **書籤** \> **倒置書籤**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_INVERT (4588)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4588);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4588
```
