# 選擇行命令

## 摘要

切換行選擇模式。

## 說明

切換行選擇模式。這個命令讓您能用鍵盤亮顯多行。用箭頭鍵移動游標來延伸或縮短選區。選擇 [**複製** 命令](edit_copy) 或 [**剪下** 命令](edit_cut) 將終止該選擇模式。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **編輯** \> **延伸選區**
\> **選擇行**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+F8

## 外掛程式命令ID

```
EEID_SELECT_LINE (4154)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4154);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4154
```
