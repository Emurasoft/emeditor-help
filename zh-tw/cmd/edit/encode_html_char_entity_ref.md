# 將 Unicode 轉換為 HTML 字元實體參照命令

## 摘要

把選取的文字編碼為 HTML 字元實體參照。

## 說明

把選取的文字編碼為 HTML 字元實體參照。

## 運行方法

- 預設功能表: **轉換** \> **編碼/解碼** \> **將 Unicode 轉換為 HTML 字元實體參照**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼** \> **將 Unicode 轉換為 HTML 字元實體參照**
- 工具列:
None
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_ENCODE_HTML_CHAR_ENTITY_REF (4563)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4563);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4563
```
