# Base64 轉換為二進位檔案命令

## 摘要

把選取的 Base64 編碼文字解碼為一個二進位檔案。

## 說明

把選取的 Base64 編碼文字解碼為一個二進位檔案。

## 運行方法

- 預設功能表: **轉換** \> **編碼/解碼** \> **Base64 轉換為二進位檔案**
- [全部命令](../tools/all_commands): **轉換** \> **編碼/解碼** \> **Base64 轉換為二進位檔案**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_DECODE\_BASE64\_BINARY (3943)

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3943);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3943
```
