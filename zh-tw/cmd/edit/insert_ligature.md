# 插入連字符號命令

## 摘要

插入一個連字符號。

## 說明

在選擇這個命令后，在游標處插入一個帶有連字符號的字元 (æ，œ，Æ 或 Œ) 通過輸入 a, o, A 或 O, 或一個急轉的 s (ß) 通過輸入 s。

## 運行方法

- 預設功能表: **插入** \> **讀音符號** \> **連字符號**
- [全部命令](../tools/all_commands): **插入** \> **讀音符號** \> **連字符號**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+SHIFT+7

## 外掛程式命令ID

```
EEID_INSERT_LIGATURE (4309)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4309);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4309
```
