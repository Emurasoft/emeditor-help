# 分割欄命令

## 摘要

分割 CSV 文檔中選取的欄。

## 說明

此命令會顯示 [**分割欄** 對話方塊](../../dlg/split_column/index) 來分割 CSV 文檔中選取的欄。

## 運行方法

- 預設功能表: **CSV** \> **欄** \> **分割欄**
- [全部命令](../tools/all_commands): **CSV** \> **欄** \> **分割欄**
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令 ID

- EEID\_SPLIT\_COLUMN (4057)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4057);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4057
```
