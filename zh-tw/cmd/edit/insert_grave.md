# 插入低音符號命令

## 摘要

插入一個帶有低音符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有低音符號的字元 (à，è，ì，ò，ù，À，È，Ì，
Ò 或 Ù) 通過輸入 a，e，i，o，u，A，E，I，O
或 U；插入一個單引號 (‘) 通過輸入 \` ；或一個雙引號 (“) 通過輸入 " 。

## 運行方法

- 預設功能表:插入 \>讀音符號 \>低音符號
- [全部命令](../tools/all_commands):插入 \>讀音符號 \>低音符號
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+\`

## 外掛程式命令ID

```
EEID_INSERT_GRAVE (4303)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4303);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4303
```
