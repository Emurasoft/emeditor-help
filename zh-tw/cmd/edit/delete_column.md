# 刪除欄命令

## 摘要

刪除選取的欄或目前的欄。

## 說明

刪除選取的欄或目前的欄。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>刪除 \>刪除欄
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_DELETE\_COLUMN (3962)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3962);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3962
```
