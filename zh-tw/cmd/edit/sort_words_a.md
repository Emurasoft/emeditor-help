# 按字數從少到多排序命令

## 摘要

把目前的欄中的文字按字數以升序順序排序。

## 說明

把目前的欄中的文字按字數以升序順序排序。

## 運行方法

- 預設功能表:排序 \>按字數從少到多排序
- [全部命令](../tools/all_commands):排序 \>按字數從少到多排序
- 工具列： 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_WORDS_A (3945)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(3945);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3945
```
