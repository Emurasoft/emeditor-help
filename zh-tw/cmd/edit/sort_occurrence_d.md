# 按出現次數降序排序命令

## 摘要

把目前的欄中的出現次數按降序排序。

## 說明

把目前的欄中的出現次數按降序排序。

## 運行方法

- 預設功能表:排序 \>按出現次數降序排序
- [全部命令](../tools/all_commands):排序 \>按出現次數降序排序
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_OCCURRENCE_D (4002)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4002);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4002
```
