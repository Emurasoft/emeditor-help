# 插入長音符號命令

## 摘要

插入一個帶有長音符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有長音符號的字元 (ā，ē，ī，ō，ū，Ā，Ē，Ī，Ō 或 Ū) 通過輸入 a，e，i，o，u，A，E，I，O 或 U。

## 運行方法

- 預設功能表: **插入** \> **讀音符號** \> **長音符號**
- [全部命令](../tools/all_commands): **插入** \> **讀音符號** \> **長音符號**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: ALT + -

## 外掛程式命令ID

```
EEID_INSERT_MACRON (3861)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3861);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3861
```
