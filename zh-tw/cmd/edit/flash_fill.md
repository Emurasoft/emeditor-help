# 快速填入 (僅限儲存格選擇模式) 命令

## 摘要

自動填滿 CSV 文檔的整個列中的值。

## 說明

自動填滿 CSV 文檔的整個列中的值。

## 運行方法

- 預設功能表:CSV \>進階 \>快速填入
- [全部命令](../tools/all_commands):CSV \>進階 \>快速填入
- 工具列: ![](../../images/flash_fill.PNG)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_FLASH\_FILL (3991)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3991);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3991
```
