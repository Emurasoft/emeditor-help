# 語法檢查命令

## 摘要

檢查目前的文檔的語法並顯示結果。

## 說明

檢查目前的文檔的語法並顯示結果。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>語法檢查
- 工具列: ![](../../images/csv_validate.png)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_CSV\_VALIDATE (3990)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3990);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3990
```
