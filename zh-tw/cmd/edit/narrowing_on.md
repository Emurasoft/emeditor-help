# 啟用僅編輯選定區域模式命令

## 摘要

把選定的區域設置為可編輯的區域，其他剩余的區域則不允許訪問。

## 說明

把選定的區域 (包括選定區域的整個邏輯行) 設置為可編輯的區域，其他剩余的區域則不允許訪問。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):編輯 \>僅編輯選定區域 \>啟用僅編輯選定區域模式
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_NARROWING_ON (4457)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4457);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4457
```
