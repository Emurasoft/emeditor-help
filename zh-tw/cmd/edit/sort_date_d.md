# 按日期和時間從新到舊排序命令

## 摘要

把目前的欄中的日期和時間按降序排序。

## 說明

把目前的欄中的日期和時間按降序排序。

## 運行方法

- 預設功能表:排序 \>按日期和時間從新到舊排序
- [全部命令](../tools/all_commands):排序 \>按日期和時間從新到舊排序
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_SORT_DATE_A (3973)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3973);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3973
```
