# 重新轉換命令

## 摘要

使用可以進行重新轉換的輸入法重新轉換選定區域。

## 說明

通過使用能夠重新轉換的輸入法編輯器 (IME)，通常在日文 IME 中可以找到，重新轉換已經被轉換的文字。

## 運行方法

- 預設功能表:轉換 \>重新轉換
- [全部命令](../tools/all_commands):轉換 \>重新轉換
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: 無 (通常，IME 設置可以在日文鍵盤上使用轉換 鍵 。)

## 外掛程式命令ID

```
EEID_RECONVERT (4199)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4199);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4199
```
