# 清除內容 (僅限儲存格選擇模式) 命令

## 摘要

清除 CSV 文檔中所選區域的內容。

## 說明

清除 CSV 文檔中所選區域的內容。

## 運行方法

- 預設功能表:CSV \>清除內容
- [全部命令](../tools/all_commands):CSV \>僅限儲存格選擇模式 \>清除內容
- 工具列: 無
- 狀態列: 無
- 預設捷徑:DELETE 鍵 (僅限儲存格選擇模式)

## 外掛程式命令ID

```
EEID_CLEAR_CONTENTS (4033)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4033);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4033
```
