# 插入波狀符號命令

## 摘要

插入一個帶有波狀符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有波狀符號的字元 (ã，ñ，õ，ẽ，ũ，Ã，Ñ，Õ，Ẽ 或 Ũ) 通過輸入 a，n，o，e，u，A，N，O，E 或 U。

## 運行方法

- 預設功能表:插入
\>讀音符號 \>波狀符號
- [全部命令](../tools/all_commands):插入
\>讀音符號 \>波狀符號
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+SHIFT+\`

## 外掛程式命令ID

```
EEID_INSERT_TILDE (4306)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4306);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4306
```
