# 將純文字轉換為 Base64 (目前的編碼) 命令

## 摘要

用目前的編碼將選取的純文字編碼為 Base64 編碼。

## 說明

用目前的編碼將選取的純文字編碼為 Base64 編碼。

## 運行方法

- 預設功能表:轉換 \>編碼/解碼 \>將純文字轉換為 Base64 (目前的編碼)
- [全部命令](../tools/all_commands):轉換 \>編碼/解碼 \>將純文字轉換為 Base64 (目前的編碼)
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令 ID

- EEID\_ENCODE\_BASE64 (3940)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3940);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3940
```
