# 插入商標符號命令

## 摘要

插入一個商標符號。

## 說明

在游標處插入一個商標符號 (™)。

## 運行方法

- 預設功能表:插入
\>標記 \>商標符號
- [全部命令](../tools/all_commands):插入
\>標記 \>商標符號
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: ALT+CTRL+T

## 外掛程式命令ID

```
EEID_INSERT_TRADEMARK (4316)```

## 巨集

### \[JavaScript\]

```
document.selection.Text="™";
```

### \[VBScript\]

```
document.selection.Text="™"
```
