# 在上方插入一行命令

## 摘要

在目前的游標位置上方插入新行。

## 說明

在游標目前的所在位置上方插入新的一行。

## 運行方法

- 預設功能表: **插入** \> **在上方插入一行**
- [全部命令](../tools/all_commands): **插入** \> **在上方插入一行**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: SHIFT+ENTER

## 外掛程式命令ID

```
EEID_LINE_OPEN_ABOVE (4195)```

## 巨集

### \[JavaScript\]

```
document.selection.LineOpen(true);
```

### \[VBScript\]

```
document.selection.LineOpen true
```
