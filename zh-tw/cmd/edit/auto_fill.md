# 自動填滿/自動重複命令

### 摘要

> 將一數列或模式延續到相鄰的儲存格中。

### 說明

> 將一數列或模式延續到相鄰的儲存格中。

### 運行方法

- 預設功能表: **插入** \> **自動填滿/自動重複**
- [全部命令](../tools/all_commands): **插入** \> **自動填滿/自動重複**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

### 外掛程式命令 ID

- EEID\_AUTO\_FILL (3992)

### 巨集

#### \[JavaScript\]

> editor.ExecuteCommandByID(3992);

#### \[VBScript\]

> editor.ExecuteCommandByID 3992
