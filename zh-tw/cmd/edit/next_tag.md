# 搜尋符合的標籤命令

## 摘要

移動游標至對應的標籤。

## 說明

當游標位于一個 XML 或 XHTML 文檔中的標籤處時，這個命令會把游標移到與所選取的標籤相對應的標籤處。如果游標在標籤所包括的範圍之外，游標會被放在相對應的標籤之外。如果游標在標籤包括的範圍之內，游標會被放在相對應的標籤之內。

## 運行方法

- 預設功能表:搜尋 \>搜尋符合的標籤
- [全部命令](../tools/all_commands):編輯 \>水平移動游標
\>搜尋符合的標籤
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+.

## 外掛程式命令ID

```
EEID_NEXT_TAG (4601)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4601);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4601
```
