# 下一個節點命令

## 摘要

當顯示大綱視圖時移動游標到下一個節點。

## 說明

當顯示大綱視圖時移動游標到下一個節點。要使用該功能，先要啟用大綱來顯示大綱視圖。

## 運行方法

- 預設功能表:編輯 \>大綱 \>下一個節點
- [全部命令](../tools/all_commands):編輯 \>大綱 \>下一個節點
- 工具列: 無
- 狀態列: 無
- 預設捷徑: Alt+Down

## 外掛程式命令ID

```
EEID_OUTLINE_NEXT_NODE (4415)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4415);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4415
```
