# 按文字長度從長到短命令

## 摘要

在目前的欄中按文字長度從長到短排序文字。

## 說明

在目前的欄中按文字長度從長到短排序文字。在自訂 對話方塊中排序 頁面上的當按長度排序時，將全形字元作為 2 個字元排序 核取方塊定義了全形字元的計算方式。

## 運行方法

- 預設功能表:排序 \>按文字長度從長到短
- [全部命令](../tools/all_commands):排序 \>按文字長度從長到短
- 工具列: ![](../../images/sort_length_d.png)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_SORT_LEGNTH_D (3918)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3918);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3918
```
