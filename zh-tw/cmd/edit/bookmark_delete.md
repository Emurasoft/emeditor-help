# 刪除目前的文件中的書籤標示行命令

## 摘要

刪除目前的文檔中所有書籤標示行。

## 說明

刪除目前的文檔中所有書籤標示行。

## 運行方法

- 預設功能表: **書籤** \> **刪除書籤標示行**
- [全部命令](../tools/all_commands): **書籤** \> **刪除書籤標示行**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_BOOKMARK_DELETE (4589)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4589);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4589
```
