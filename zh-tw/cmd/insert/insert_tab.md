# 插入 Tab 鍵命令

## 摘要

在游標處插入一個 Tab 鍵。

## 說明

在目前的游標位置處插入一個 Tab 鍵。

## 運行方法

- 預設功能表: **插入** \> **Tab**
- [全部命令](../tools/all_commands): **插入** \> **Tab**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: Tab

## 外掛程式命令ID

```
EEID_INSERT_TAB (3878)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3878);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3878
```
