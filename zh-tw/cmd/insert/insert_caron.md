# 插入變音符號命令

## 摘要

插入一個帶有變音符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有變音符號的字元 (ǎ，ǐ，ǒ，ǔ，Ǎ，Ǐ，Ǒ or Ǔ) 通過輸入 a，i，o，u，A，I，O 或 U。

## 運行方法

- 預設功能表: **插入** \> **讀音符號** \> **變音符號**
- [全部命令](../tools/all_commands): **插入** \> **讀音符號** \> **變音符號**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: ALT+SHIFT+6

## 外掛程式命令ID

```
EEID_INSERT_CARON (4369)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4369);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4369
```
