# 插入注冊符號命令

## 摘要

插入一個注冊符號。

## 說明

在游標處插入一個注冊符號 (®)。

## 運行方法

- 預設功能表: **插入** \> **標記** \> **注冊符號**
- [全部命令](../tools/all_commands): **插入** \> **標記** \> **注冊符號**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: ALT+CTRL+R

## 外掛程式命令ID

```
EEID_INSERT_REGISTERED (4315)```

## 巨集

### \[JavaScript\]

```
document.selection.Text="®";
```

### \[VBScript\]

```
document.selection.Text="®"
```
