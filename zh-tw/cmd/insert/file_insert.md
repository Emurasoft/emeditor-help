# 插入命令

## 摘要

插入一個指定的檔案到目前的游標位置。

## 說明

這個命令會顯示 **插入** 對話方塊，讓您能選取一個您想要插入到目前的檔案的檔案。在選取一個檔案后，這個命令會讀取被選取的檔案并把它的內容插入到游標所在位置。

## 運行方法

- 預設功能表: **插入** \> **插入**
- [全部命令](../tools/all_commands): **插入** \> **插入**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FILE_INSERT (4108)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4108);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4108
```
