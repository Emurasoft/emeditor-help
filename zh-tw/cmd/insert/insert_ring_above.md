# 插入圓環重音符號命令

## 摘要

插入一個帶有圓環重音符號的字元。

## 說明

在選擇這個命令后，在游標處插入一個帶有圓環重音符號的字元 (å 或 Å) 通過輸入 a 或 A，或度數符號 (°) 通過輸入一個空格。

## 運行方法

- 預設功能表: **插入** \> **讀音符號** \> **圓環重音符號**
- [全部命令](../tools/all_commands): **插入** \> **讀音符號** \> **圓環重音符號**
- 工具列: 無
- 狀態列: 無
- 預設鍵盤快速鍵: CTRL+SHIFT+2

## 外掛程式命令ID

```
EEID_INSERT_RING_ABOVE (4308)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4308);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4308
```
