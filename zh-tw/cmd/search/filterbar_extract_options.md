# 抽出選項 (篩選工具列) 命令

## 摘要

指定抽出符合字串命令的選項。

## 說明

指定抽出符合字串命令的選項。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **搜索**
\> **篩選工具列** \> **抽出選項**
- 工具列: ![](../../images/extract_all.png) (篩選工具列) >  **抽出選項**
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_FILTERBAR_EXTRACT_OPTIONS (4086)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4086);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4086
```
