# 僅 LF (篩選工具列) 命令

## 摘要

切換「篩選」工具列上的「僅 LF」命令。

## 說明

切換「篩選」工具列上的「僅 LF」命令。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜索
\>篩選工具列 \>符合換行符號 \>僅 LF
- 工具列:  ![](../../images/match_newline_characters.png) \+ 僅 LF (篩選工具列)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FILTERBAR_LF_ONLY (4008)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4008);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4008
```
