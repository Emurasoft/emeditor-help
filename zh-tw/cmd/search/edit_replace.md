# 取代命令

## 摘要

取代一個字串。

## 說明

用一個指定的字串取代另一個指定的字串。這個命令會顯示 [**取代** 對話方塊](../../dlg/replace/index)。指定一個要尋找的字串，要取代的字串，以及其他參數，就可以開始取代指定字串了。

## 運行方法

- 預設功能表: **搜尋** \> **取代**
- [全部命令](../tools/all_commands): **搜尋**
\> **取代**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+H

## 外掛程式命令ID

```
EEID_EDIT_REPLACE (4201)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4201);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4201
```
