# 取代 >\> (搜尋工具列) 命令

## 摘要

切換搜尋工具列上的「取代 >>」按鈕。

## 說明

切換搜尋工具列上的「取代 >>」按鈕。這將顯示取代 對話方塊。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜尋
\>搜尋工具列 \>取代 >>
- 工具列: ![](../../images/find_replace_dlg.png) (搜尋工具列)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FINDBAR_REPLACE_DLG (4579)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4579);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4579
```
