# 僅限于選區內 (搜尋工具列) 命令

## 摘要

切換搜尋工具列上「僅限于選區內」按鈕的狀態。

## 說明

切換搜尋工具列上「僅限于選區內」按鈕的狀態。當這個命令被切換時，用尋找 功能將僅在選取的區域中搜尋。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜尋
\>搜尋工具列 \>僅限于選區內
- 工具列: ![](../../images/find_sel_only.png) (搜尋工具列)
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FINDBAR_SEL_ONLY (4580)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4580);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4580
```
