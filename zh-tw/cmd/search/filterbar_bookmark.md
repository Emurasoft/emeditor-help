# 全部設為書籤 (篩選工具列) 命令

## 摘要

為所有篩選出來的行設置書籤。

## 說明

為所有篩選出來的行設置書籤。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands): **搜尋**
\> **篩選工具列** \> **全部設為書籤**
- 工具列:  ![](../../images/bookmark_all.png) (篩選工具列)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_FILTERBAR_BOOKMARK (3927)```

## 巨集

editor.ExecuteCommandByID(3927);
