# 抽出常用字串命令

## 摘要

將常用字串抽出到新文檔中。這個命令會顯示 [**抽出常用字串** 對話方塊](../../dlg/extract_frequent/index)。

## 說明

將常用字串抽出到新文檔中。

## 運行方法

- 預設功能表: **搜索** \> **抽出常用字串**
- [全部命令](../tools/all_commands): **搜索**
\> **抽出常用字串**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_EXTRACT_FREQUENT (4041)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4041);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4041
```
