# 關閉搜尋工具列命令

## 摘要

關閉搜尋工具列。

## 說明

關閉搜尋工具列。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜尋
\>搜尋工具列 \>關閉搜尋工具列
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_CLOSE_FIND_BAR (4584)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4584);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4584
```
