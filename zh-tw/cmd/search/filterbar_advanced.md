# 進階篩選 (篩選工具列) 命令

## 摘要

顯示「進階篩選」對話方塊。

## 說明

顯示「進階篩選」對話方塊。

## 運行方法

- 預設功能表: 無
- [全部命令](../tools/all_commands):搜索
\>篩選工具列 \>進階篩選
- 工具列: ![](../../images/filterbar_advanced.png) (篩選工具列)
- 狀態列: 無
- 預設快速鍵: 無

## 外掛程式命令ID

```
EEID_FILTERBAR_ADVANCED (3919)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3919);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3919
```
