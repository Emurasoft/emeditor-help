# 將焦點設在篩選工具列命令

## 摘要

把焦點設到篩選工具列上。

## 說明

把焦點設到篩選工具列上。

## 運行方法

- 預設功能表:搜尋 \>篩選
- [全部命令](../tools/all_commands):搜尋
\>篩選工具列 \>將焦點設在篩選工具列
- 工具列: 無
- 狀態列: 無
- 預設快速鍵: CTRL+SHIFT+D

## 外掛程式命令 ID

- EEID\_FOCUS\_FILTER\_BAR (3912)

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(3912);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3912
```
