# 添加下一個符合的結果命令

## 摘要

添加與指定文字元合的下一個結果到選區。

## 說明

添加與指定文字元合的下一個結果到選區。

## 運行方法

- 預設功能表: **搜尋** \> **添加下一個符合的結果**
- [全部命令](../tools/all_commands): **搜尋**
\> **添加下一個符合的結果**
- 工具列: 無
- 狀態列: 無
- 預設捷徑: CTRL+R

## 外掛程式命令ID

```
EEID_ADD_NEXT_OCCURRENCE (4604)```

## 巨集

### \[JavaScript\]

```
editor.ExecuteCommandByID(4604);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4604
```
