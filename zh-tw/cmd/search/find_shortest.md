# 尋找非空最短行/儲存格命令

## 摘要

尋找非 CSV 文檔中的非空最短行或 CSV 文檔的目前的欄中的非空最短儲存格。

## 說明

尋找非 CSV 文檔中的非空最短行或 CSV 文檔的目前的欄中的非空最短儲存格。

## 運行方法

- 預設功能表:搜索 \>尋找非空最短行/儲存格
- [全部命令](../tools/all_commands):搜索
\>尋找非空最短行/儲存格
- 工具列: 無
- 狀態列: 無
- 預設捷徑: 無

## 外掛程式命令ID

```
EEID_FIND_SHORTEST (4050)```

## 巨集

## \[JavaScript\]

```
editor.ExecuteCommandByID(4050);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4050
```
