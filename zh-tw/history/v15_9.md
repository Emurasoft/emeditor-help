# Version 15.9

2016年4月19日

## 一般新增功能

- 添加了新的CSV 轉換器。
- CSV 轉換命令優化。
- 添加另一個分隔符到 CSV 文檔的右邊。
-剪下欄，復制欄，貼上，刪除欄，在左邊插入一欄，以及在右邊插入一欄 命令被添加到了右擊尺規顯示的快顯功能表上。
- 添加了在上方插入一行 和在下方插入一行 命令到右擊編輯器左邊緣會顯示的功能表中。
- 廢棄了復原緩衝區的設定。改用實體記憶體大小來決定復原的信息是否應該保留。實體記憶體閾值可以在自訂 對話方塊的進階 頁面上設置。
- 新版本讓你能取消目前的命令當復原數據不可用時。

## 新增選項

- 添加了預設欄寬 文字方塊到自訂 對話方塊中的CSV 頁面上。
- 添加了總是丟棄過長的復原信息來加速 核取方塊到自訂 對話方塊中的編輯 頁面上。
- 添加了最大記憶體大小 文字方塊到自訂 對話方塊中的排序 頁面上。

## 新增命令

- [CSV 轉換器](../cmd/edit/csv_converter)
- [刪除欄](../cmd/edit/delete_column)
- [剪下欄](../cmd/edit/cut_column)
- [復制欄](../cmd/edit/copy_column)
- [在右邊插入一欄](../cmd/edit/insert_column_right)

## 外掛程式新增功能

-復制 以及全選 命令被添加到字數統計 外掛程式中的內容功能表上。
- 能夠用路徑調用程式碼片段。

## 外掛程式 API 新增功能

- 延伸了 [FIND\_REGEX\_INFO\_EX 結構](../plugin/structure/find_regex_info_ex)。添加了BACK\_REF 結構。

## 巨集的新增功能

- 添加了 [Regex 對象](../macro/regex/index)， [Matches 集合](../macro/matches/index)，and [Match 對象](../macro/match/index)。
- 添加了 [regex 屬性](../macro/editor/regex) 到Editor 對象中。
- 在JSON 對象中支持JScript Version 5.8。

## 修復 Bug

- 修復了一個與 CSV 儲存格模式相關的 bug。
- 優化了用 Onigmo 進行的 Lookbehind 規則運算式搜索。
- 修復了在選取多個區域時與全部取代相關的一個 bug。
- 修復了一個與巨集的 Replace 方法相關的 bug。
