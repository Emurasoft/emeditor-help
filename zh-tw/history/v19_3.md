# Version 19.3

2019年10月8日

## 一般新增功能

- 當選擇多行或進行多個選取後執行篩選，篩選除外，在欄中篩選，或在欄中篩選除外 命令時，新版本會把多個字串設為進階篩選。
- 當選擇多行或進行多個選取後執行新增標記 或新增或刪除標記 命令，新版本會把多個字串添加到標記 中，或從標記 工具列中刪除這些字串。
- 添加了全設為書籤 命令到在標記 工具列上單擊滑鼠右鍵所顯示的內容功能表中。
- 添加了更多的選項讓你能自訂啟動 視窗。
- 在啟動 視窗中的清單控制項之外單擊滑鼠右鍵時，新版本將顯示一個內容功能表。
- 添加了管線分隔 格式到預設的 CSV 格式定義，並將一個現有格式的稱謂該成比較報告。
- 當文檔包含很長的行時，提高了各種操作的速度。

## 新增選項

- 添加了當儲存 Unicode 為 ANSI 時提示 核取方塊，編碼 下拉清單方塊以及換行符號 下拉清單方塊到合併檔案 精靈中的 [目的檔案和選項 頁面](../dlg/combine_files/combine_wiz_dest) 上。
- 添加了顯示額外的按鈕，貼齊主視窗，變更的日期，類型，和大小 核取方塊以及大圖示 和小圖示 選項按鈕到 [自訂啟動視窗 對話方塊](../dlg/customize_start/index) 中。
