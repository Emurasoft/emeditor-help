# Version 4.10

2005年1月23日

- [**狀態列** 頁面](../dlg/customize/status/index) 被添加到了自訂對話方塊中。
- 您能變更 Windows 工具列按鈕順序通過在 Windows 工具列上拖放一個按鈕。
- 添加了 [**關閉所有其他檔案** 命令](../cmd/file/close_all_others)。
- 添加了 [**移動到下一個** 命令](../cmd/window/window_move_next) 以及 [**移動到上一個** 命令](../cmd/window/window_move_prev)。
- 狀態列現在顯示在游標位置處的字元代碼值。
- 添加了 [**首字母大寫** 命令](../cmd/edit/capitalize)。
- eeCaseCapitalize 標志添加到了 [**ChangeCase** 方式](../macro/selection/selection_changecase) 中。
- 添加了 [**分割行** 命令](../cmd/edit/split_lines) 以及 [**合併行** 命令](../cmd/edit/join_lines)。
- eeFormatSplitLines 標志和 eeFormatJoinLines 標志添加到了 [**Format** 方式](../macro/selection/selection_format) 中。
- 添加了 [**導入和導出** 命令](../cmd/tools/import_export)。
- 在鍵盤布局視窗中添加了內容功能表。
- 您現在能顯示或編輯「鍵盤布局」對話方塊中選取的項目的屬性。
- 您現在能顯示所有組態的「鍵盤布局」。
- 您現在能把選取的項目作為文字複製到剪貼簿上。
- 您現在能通過在屬性對話方塊的「鍵盤」頁面上，按下 Ctrl 鍵，放開，再按下 PageUp 或 PageDown 來分配 Ctrl+PageUp 或 Ctrl+PageDown 鍵。
- 16 色的工具列現在可以顯示外部工具圖示。
