# Version 18.8

2019年4月23日

## 一般新增功能

- 用多執行緒程式碼和最近 CPU 中的 AVX-2 指令集提高了打開很大檔案，尋找/取代/篩選， **大寫**， **小寫**，以及 **首字母大寫**， **全形**，以及 **半形** 命令的速度。
- 大大提高了特別是打開包含很長行的文檔時的速度。
- 新版本允許您在對話方塊的所有文字方塊中使用 CTRL + SHIFT + V 將以前復制的文字貼上到編輯器中。
- 更新了 IPTip.jsee 巨集以支持新的 GeoLite2 CSV 格式。如果啟用了 IP v4 活動字串，則當游標移動到 IP v4 地址時，此巨集會將國家/地區名稱顯示為工具提示。
- 變更了預設鍵盤快速鍵。SHIFT + TAB 被指定為 [**減少行縮排** 命令](../cmd/edit/unindent)，CTRL + SHFT + T 被指定為 [**最近關閉的檔案** 命令](../cmd/file/most_recent_closed_file)。

### 測試結果：

|  | v18.7 | v18.8 | 比較 |
| --- | --- | --- | --- |
| 小寫命令 | 56.6 秒 | 1.47  秒 | 快 38.5 倍 |
| 首字母大寫命令 | 72.5 秒 | 2.64 秒 | 快 27.5 倍 |
| 全形命令 | 116 秒 | 4.27 秒 | 快 27.2 倍 |
| 打開一個很大的檔案 | 8.08 秒 | 7.55 秒 | 快 6.56% |
| 用 "abc" 取代全部 "@@@"（預設選項） | 1.86 秒 | 0.922 秒 | 快 2.02 倍 |
| 篩選 "abc"（預設選項） | 1.67 秒 | 0.906 秒 | 快 1.84 倍 |
| 在沒有換行的很大的文檔中移動游標\*1 | 1.36 秒 | 0.578 秒 | 快 2.35 倍 |

\*1 3.09 GB，1 行（沒有換行），XML 檔案，用巨集把游標移動到行尾，剩 20 個字元處。使用臨時檔案選項關。

其他：4.72 GB，50,000,000 行，隨機 ASCII 數據，使用臨時檔案選項關。

測試系統：Windows 10 (64-bit)，Core i9-9900K，32GB RAM，2TB SSD。

## 新增選項

- 添加了 **提示如果 Unicode (UTF-16) 檔案大小為奇數（以位元組為單位）** 核取方塊到 **自訂** 對話方塊的 [**檔案** 頁面](../dlg/customize/file/index) 上。
- 添加了 **執行「插入/刪除換行符號」命令時，忽略選擇范圍中的最后一個字元**， **在半形/全形轉換期間將半形反斜線 (U + 005C) 轉換為全形日幣 (Yen) 符號 (U + FFE5)**，和 **在半形/全形轉換期間將半形反斜線 (U + 005C) 轉換為全形韓幣 (Won) 符號 (U + FFE6)** 核取方塊到 **自訂** 對話方塊的 [**編輯** 頁面](../dlg/customize/edit/index) 上。

## 外掛程式 API 新增功能

- 添加了 **eeWidthJapaneseYen** 和 **eeWidthKoreanWon** 標志到 **Selection** 對象的 [**ChangeWidth** 方法](../macro/selection/selection_changewidth) 中。

## 新增巨集功能

- 添加了 **FLAG\_JAPANESE\_YEN** 和 **FLAG\_KOREAN\_WON** 標志到 [**EE\_CONVERT** 消息](../plugin/message/ee_convert) 和 [**Editor\_Convert** 內嵌函式](../plugin/macro/editor_convert) 中。
