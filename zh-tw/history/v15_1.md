# Version 15.1

2015年5月6日

## 一般新增功能

- 大大增強了 **尋找**, **取代全部**， **篩選**， **復原取代全部**，以及 **選擇所有符合結果** 這些命令的執行時間。當處理大檔案時(大于 10,000 行)時，這些命令會以 **多線程** 的方式運行。
- 您能自訂 EmEditor 來顯示許多任務的執行時間。
- 添加了 **擷取欄** 命令，讓您能在 CSV 文檔中提取指定列并創建一個新的文檔。
- 在搜尋文檔后，狀態列會分開顯示符合結果與符合行數 (在單一行中可能有多于一個符合結果)。
- 您可以 **多檔尋找** 中忽略二進位檔案。
- 新的版本允許 **每個使用者** 的安裝。 **每個使用者** 的安裝將不會在 Windows 7 或之後的版本上顯示 **使用者帳戶控制** ( **UAC**) 提示。 **每個使用者** 的安裝可以通過在 **安裝類型** 對話方塊中選擇 **只有我** 單選按鈕來進行。如果在命令提示中已有一個安靜安裝，請添加 "MSIINSTALLPERUSER=1" 選項。安裝類型不能在更新時更改。
- **.eeWorkspace** 檔案已與 EmEditor 相關聯。雙擊 **.eeWorkspace** 檔案會還原 EmEditor 的工作區。
- 所有快速鍵設定能在每個使用者的安裝中被更改。
- **與 EmEditor 相關聯** 對話方塊現在不需要管理權限訪問。
- 當一個使用者在篩選工具列上的下拉清單上選擇篩選字串所在的列時，EmEditor 能隨著選擇的列水平捲動檔案，讓指定列可見。
- 當點擊 **狀態列** 上的任一項目時，會出現一個內容功能表。選擇 **複製** 會把內容複製到剪貼簿上。

## 新增選項

- 添加了 [**擷取欄** 對話方塊](../dlg/extract_columns/index)。
- **顯示執行時間** 核取方塊被添加到 **自訂** 對話方塊中的 [**狀態列** 頁面上](../dlg/customize/status/index)。
- 添加了 **忽略二進位檔案** 和 **在結果中顯示被忽略的檔案** 核取方塊到通過 **多檔尋找** 或 **多檔取代** 對話方塊訪問的 [**進階** 對話方塊](../dlg/advanced/index) 中。
- 添加了 **啟動 EmEditor 時顯示系統匣圖示** 核取方塊到 **自訂** 對話方塊中的 [**捷徑** 頁面](../dlg/customize/shortcut/index) 上。
- 所有在 **自訂更多捷徑** 對話方塊上的內容被移動到 **自訂** 對話方塊中的 [**捷徑** 頁面](../dlg/customize/shortcut/index) 上。
- 在 [**分割檔案** 精靈](../dlg/split_to_files/index) 中， **頁眉和頁腳** 頁面上的 **頁眉 \[>\]** 功能表中現在包括了 **$(FirstLine)**，讓您把文檔中的第一行作為頁眉。
- **線程數** 下拉清單方塊被添加到了 **自訂** 對話方塊中的 [**進階** 頁面](../dlg/customize/advanced/index) 上。
