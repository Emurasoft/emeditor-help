# Version 18.9

2019年5月9日

## 一般新增功能

- 新版本允許您尋找/取代/篩選 指定範圍的IPv4 / IPv6 位址。IP 位址範圍可以用 [數字範圍運算式語法](../howto/search/number_range_syntax) 或IPv4 / IPv6 CIDR 標記 指定。
- 用多執行緒程式碼大大提高了尋找/取代/篩選，大寫，小寫，首字母大寫，全形，以及半形 命令的速度，即使用臨時檔案打開非常大的檔案。
- 大大提高了儲存 和另存新檔 命令的速度。

### 測試結果：

## 新增選項

- 添加了IP 位址/ CIDR 標記 核取方塊/文字方塊到 [輸入數字範圍 對話方塊](../dlg/number_range/index) 中。
- 添加了IPv4 位址升序排序，IPv4 位址降序排序，IPv6 位址升序排序，和IPv6 位址降序排序 選項按鈕到 [多欄排序 對話方塊](../dlg/sort_multi/index) 中。
- 新版本讓您能輸入一個大於 4096 MB 的數字到自訂 對話方塊 [進階 頁面](../dlg/customize/advanced/index) 上的每個檔案的最大記憶體大小 文字方塊中（僅限 64 位元版）。

## 新增命令

- [IPv4 位址升序排序](../cmd/edit/sort_ipv4_a)
- [IPv4 位址降序排序](../cmd/edit/sort_ipv4_d)
- [IPv6 位址升序排序](../cmd/edit/sort_ipv6_a)
- [IPv6 位址降序排序](../cmd/edit/sort_ipv6_d)

## 外掛程式 API 新增功能

- 添加了 SORT\_IPV4 和 SORT\_IPV6 標志到 [SORT\_INFO 結構](../plugin/structure/sort_info) 以及 [Editor\_Sort 內嵌函式](../plugin/macro/editor_sort) 中。

## 新增巨集功能

- 添加了I（排序 IPv4 位址）和P （排序 IPv6 位址）選項到Document 對象中的 [Sort 方法](../macro/document/sort) 中。
