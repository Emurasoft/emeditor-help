# Version 14.1

December 23, 2013

## 一般新增功能

- 增強的多重選取功能: 當您進行多重選取時，您可以使用 SHIFT + RIGHT 或 LEFT 鍵來延伸選區。
- 現在能指定多達15個字元的行注釋 (過去是3個字元) 。
- 您可以自訂已添加書簽的行 的顏色。
- 即使作了選取，當前行的顏色 也會被保留。
- 您現在可以為每一個組態設置一個預設的初始資料夾。當一個無標題 文檔被儲存時，另存新檔 對話方塊將使用指定的初始資料夾進行初始化。要注意的是如果一個已知的資料夾 (例如文檔，圖片 等)被指定，Windows 系統可能會重新定向資料夾位置。如果 Windows 8.1 系統上有設置Sky Drive 集成，那文檔資料夾就可能會被重新定向到Sky Drive 文檔 資料夾中。

## 新增選項

-已添加書簽的行 項目被添加到組態屬性 中顯示 頁面上的指定部分 清單方塊中。
- "不再顯示這個消息"核取方塊被添加到拒絕訪問對話方塊中，讓您可以跳過這條提示消息。這個對話方塊會在 EmEditor 試著儲存檔案到 Program Files 或 Windows 系統資料夾中時出現。這個設置不會在 EmEditor 會話中被記住，所以當您重新啟動 EmEditor 后，這條消息還是會出現。
- [新建檔案詳細信息 對話方塊中](../dlg/properties/file/new_details/index) 新增了設置初始資料夾 核取方塊和初始資料夾 文字方塊。
