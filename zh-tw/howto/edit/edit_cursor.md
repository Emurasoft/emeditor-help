# 移動游標位置

按下箭頭鍵讓游標上，下，左，右移動。

### 提示

- 要用游標移動游標位置，只要點擊目的地點。

另外，您還可以用下列快速鍵移動游標。

- **CTRL+ RIGHT** 右移一個單字。
- **CTRL+ LEFT** 左移一個單字。
- **HOME** 移至目前的行行首。
- **ALT+ HOME** 移至目前的行邏輯行行首。
- **END** 移至目前的行行末。
- **ALT+ END** 移至目前的行邏輯行行首。
- **PAGE UP** 上移一頁。
- **PAGE DOWN** 下移一頁
- **CTRL+ HOME** 移到檔案頂部。
- **CTRL+ END** 移到檔案底部。
