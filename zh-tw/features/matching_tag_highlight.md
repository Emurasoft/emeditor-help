# 標籤亮顯符合功能

新的版本能在 XHTML 和 XML 文檔中自動亮顯標籤里的內容以及游標處的符合標籤。這個功能使閱讀文檔以及找到符合的標記變得容易。這個功能只有當一個文檔是格式良好的 XML 文檔時，才能正常工作。如果在一個文檔中沒有結束標記的話，比如一個舊的 HTML 文檔，那么這個功能就可能不會正常工作。另外，在使用此功能前，在組態屬性中 **「亮顯 (2)」** 頁面上勾選 **「亮顯配對標籤」** 核取方塊。

另外，您還可以用 **「尋找符合的標籤」** 和 **「延伸選區到配對的標籤」** 命令在標記和它的符合標籤之間跳轉。
CTRL +. 和 CTRL + SHIFT + . 分別被設置為這兩個命令的預設捷徑。
