# 立即訊息 (外掛程式)

一個簡單的 Intranet 通訊功能。在選取一小段文字后，在外掛程式 工具列或功能表上選擇「傳送選取的文字」 外掛程式，您會看到「傳送至」 的對話方塊。您可以選擇要傳送訊息的電腦，也可以勾選「傳送至所有電腦」 的核取方塊把文字發送給所有聯機電腦。要使用這個功能，每臺電腦都必須有安裝 EmEditor，并且 EmEditor 的系統匣圖示一定要在任務欄運行。接收一方的屏幕上會有一個氣球狀提示。請點擊這個提示或者右擊系統匣圖示以選取「顯示最后一條消息」 的命令，EmEditor 會打開一個新視窗來顯示收到的訊息。發送的文字長度是有限制的。另外，訊息提醒在 Windows 8 系統上會以Toast 通知的形式顯示。外掛程式可以在 EmEditor 資源庫中找到。
