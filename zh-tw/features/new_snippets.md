# 新增程式碼片段外掛程式

程式碼片段外掛程式在這個版本中得到大大改進。有了這個新的外掛程式，您不需要按很多次鍵就能輸入更多的文字。新增的程式碼片段外掛程式功能包括:

- 程式碼片段樹狀目錄不僅包括程式碼片段，還包括巨集。
- 您可以指定觸發字串和快捷鍵 (預設情況下是Tab) 。
- 您可以指定工具提示的內容；當觸發字串與輸入的字串符合時，工具提示會出現。
- TAB 以及 SHIFT + TAB 能被用來在占位符之間跳轉。
- 新增的外殼程式代碼 (Shell Code) 格式讓您能把程式碼片段作為篩選器使用。外殼程式代碼 (Shell Code) 讓您能把標準輸入傳遞到外部工具中，然后接收標準輸出并把它作為一個程式碼片段插入。
