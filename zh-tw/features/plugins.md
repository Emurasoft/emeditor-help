# 外掛程式

EmEditor 通過支持用戶創建的外掛程式不斷擴展和延伸其實用性。例如,您可以用拼寫 (Spelling) 外掛程式來檢查英文文本的拼寫，還可以用解碼 Unicode (Decode Unicode) 外掛程式來對選取的一段由HTML/XML字符引用或通用字符名編碼的文本進行解碼。外掛程式是由動態連結庫檔案 (DLL) 編寫的。當您使用 EmEditor 時，您可以通過把 DLL 外掛程式檔案移動到 EmEditor 資料夾下的外掛程式 (plug-in) 資料夾中來激活這些外掛程式。請參考 **[外掛程式](../plugin/index)** 章節來獲取更多有關創建您自己的外掛程式的說明。
