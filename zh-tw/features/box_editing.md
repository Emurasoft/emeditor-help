# 垂直選取編輯

垂直選取編輯功能讓您能不用規則運算式就快速地取代相似的行。

例如，您要處理的一個檔案有如下所示的文字:

name1

name2

name3

name4

現在，您想要把「name1」到「name4」的首字母大寫。在「name1」的左邊點擊一下，然后按住 ALT 鍵并垂直選擇所有的「n」 (或者您也可以按 CTRL + SHIFT + F8)，然后再按一次向右鍵 (RIGHT)，按四次向下鍵 (DOWN)。

name1

name2

name3

name4

當所有的「n」被選取時，輸入「N」。

Name1

Name2

Name3

Name4

您會看到所有「n」都被「N」取代了。
