# 關鍵字亮顯

EmEditor為下列的編程語言預設了易分辨的色彩文本:

_Bat，C#，C++，CSS，HTML，Ini，Java，JavaScript，JSP，Pascal (Delphi)，_
_Perl，PerlScript，PHP，Python，Ruby，SQL，TeX (LaTeX)，VBScript，WindowsScript，_
_x86 Assembler，XML，還有基本文本。_ 在 HTML 配置中，ASP標簽 ( `<% ... %>`) 以及標準指令碼標簽 ( `<SCRIPT> ... </SCRIPT>`) 所包含的文本是根據其在「語言」屬性中指定的指令碼語言 (JavaScript，VBScript 或 PerlScript) 被格式的。并且，HTML-嵌入式的特殊語法可用於 JSP 和 PHP 配置中。
