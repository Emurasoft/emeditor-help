# 支持 Unicode

EmEditor 支持 Unicode，一種支持多語言環境的編碼方式。您不僅可以打開 Unicode 檔案，而且還可以用 Unicode，編輯檔案來同時顯示多種語言。

EmEditor 支持 _低位優先 Unicode，UTF-8，UTF-7，波羅的文，_
_中歐字符，簡體中文，繁體中文，西里爾文，希臘文，_
_日文 (Shift-JIS) ，日文 (JIS) ，日文 (EUC) ，韓文，泰文，土耳其文，越南文，西歐語系_ 以及所有其他可用於 Windows 的編碼。

## 註意：

EmEditor 不支援使用從右到左的腳本語言，例如阿拉伯語和希伯來語。
