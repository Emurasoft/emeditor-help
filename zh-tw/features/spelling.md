# 新增拼寫功能

新版的 EmEditor 使用 **Hunspell 資源庫**。很多應用程式，例如 **OpenOffice**，和 **Firefox**，都用 Hunspell 作為拼寫檢查器，在多語言環境中檢查拼寫。在您輸入文字時，拼寫錯誤的字詞下面會出現一段紅色曲線，游標右擊拼寫錯誤的字詞，您會看到有幾個字詞選項供您選擇，讓您能快捷地更正拼寫。 EmEditor 編輯器預先安裝了預設的美國英語詞典。您可以在 **[OpenOffice.org wiki](https://wiki.openoffice.org/wiki/Dictionaries)**– (拼寫檢查器詞典) 下載別的詞典。只要把 **\*.dic** 以及 **\*.aff** 檔案複製到 EmEditor 安裝資料夾下的 **字典** 子資料夾中就行了。 (通常路徑是 C:/Program Files/EmEditor/Dictionaries) 。
