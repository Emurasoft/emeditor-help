# 百分比編碼 (也稱 URL 編碼)

您可以用百分比編碼 (即 URL 編碼) 對選取的文字進行編碼或解碼。當游標指針懸停在百分比編碼文字上時，一個工具提示會出現，并自動顯示解碼的文字。

四個新增命令: [**將百分比編碼轉換為 Unicode (當前編碼)**](../cmd/edit/decode_percent)， [**將 Unicode 轉換為百分比編碼 (當前編碼)**](../cmd/edit/encode_percent)， [**將百分比編碼轉換為 Unicode (UTF-8)**](../cmd/edit/decode_percent_utf8)，以及 [**將 Unicode 轉換為百分比編碼 (UTF-8)**](../cmd/edit/encode_percent_utf8)。
