# 強大的標簽頁功能

由于所有窗口都在同一視窗中以單一進程運行，而不像之前版本僅僅合并窗口，被增強后的標簽頁功能現在更加穩定。

- 您能選擇標簽頁 或按鈕 樣式。
- 您可以點擊標簽右邊的關閉按鈕 (×) 來關閉活動的文檔。
- 新版的 EmEditor 支持分組窗口，并且您還可以拖放任何標簽頁到另一個分組中或創建一個新的分組。

- 您可以自訂雙擊標簽頁或用鼠標中鍵單擊標簽頁所要執行的命令。

- 您能自訂標簽頁上的上下文菜單。
- 您還可以自訂新的標簽頁插入位置，當關閉一個標簽頁時要激活哪一個標簽頁，以及用哪一個窗口來打開新文檔。
- 標簽頁能通過檔案名稱，類型，修改時間或最近激活的時間來排序。您也可以用這些條件來指定自動排列 的方式。
- 當 EmEditor 窗口有太多標簽頁時，標簽頁欄能被延伸到多行或自動調節標簽頁寬度。
