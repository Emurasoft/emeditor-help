# 多選區編輯

EmEditor讓您能在同一個文檔中選取位于不同位置的文字一起編輯。要進行多選區編輯時，您可以先選取一段文字，然后用游標加 CTRL 鍵繼續選取別的文字，就可以同時編輯這些選定的文字。另外，您也可以用鍵盤操作該功能。在選取文字的開始位置處先按一下 F8 鍵，然后使用鍵盤上的方向鍵移動游標至選取文字的結尾處，再次按 F8 鍵來完成選取；然后再次移動游標至下一個文字選取開始處，重複這個步驟直到您選取了所有要選取的區域。您可以使用 **[「添加下一個符合的結果」](../cmd/search/add_next_occurrence)** 命令 (CTRL+R) 尋找與游標處相同文字的文字。或者使用 **[「添加下下個符合的結果」](../cmd/search/add_next2_occurrence)** 命令，把下一個符合結果排除在外, 直接添加在下一個之後的符合結果。 **[「選擇所有符合的結果」](../cmd/search/select_all_occurrences)** 命令 (CTRL+SHIFT+A) 可以讓您選取所有符合的結果。

在您選取了多個文字之後，您輸入的新的文字將立即取代所有被選取的內容。您也可以按一下 BACKSPACE 鍵來刪除每個選區的最后一個字元。此外，您還可以將許多轉換指令用在選區之外的區域。

**「尋找全部」** 被添加到 **[「尋找」](../dlg/find/index)** 對話方塊中，讓您能選取所有符合您搜尋條件的結果。

與多選區編輯一起，EmEditor v13同樣提高了垂直選擇編輯的功能。例如，當您進行零寬度垂直選取時，您可以按一下 BACKSPACE 鍵來刪除每一行的最后一個字元。這讓您能更容易地刪除多行文字的末尾部分！

v13還添加了 **[「交換開始位置和結束位置」](../cmd/edit/switch_start_end_select)** 命令 (SHIFT+F8)。在這個版本之前，在進行垂直選取之後，游標一直在選取範圍的末尾處。因此，要更改選取的範圍，您要按 SHIFT 鍵加箭頭鍵來移動選區的末尾，但是您無法把游標移動到選區的起始位置，來更改選區的起始範圍。現在，您可以用這個「交換開始位置和結束位置」命令，快捷鍵是 SHIFT+F8，把游標移動到選區的起始位置，然后調整起始位置的選取範圍。
