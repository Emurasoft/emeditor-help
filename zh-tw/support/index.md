# 支持

如果您有任何有關如何使用 EmEditor 的疑問，或如何購買授權，請使用下列資源:

- [常見問題解答](https://zh-tw.emeditor.com/support/)
- [論壇](https://www.emeditor.com/forums/)
- [聯系我們](https://zh-tw.emeditor.com/support/#contact)
- 另外，您還可以到 [Super User](https://superuser.com/questions/tagged/emeditor) 上提問（錯誤報告除外），只需在問題中標記 `emeditor`，我們會儘快檢視並回復您的問題。請確保遵循[如何提問指南](https://superuser.com/help/how-to-ask)。
