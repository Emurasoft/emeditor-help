# Stripe 訂閱頁面

所有與您的客戶中心賬戶關聯的 Stripe 訂閱都列在 [Stripe 訂閱頁面](https://support.emeditor.com/en/account/subscriptions)。該頁面還包含一個[購買新的 Stripe 訂閱的連結](https://support.emeditor.com/en/stripe_payment)，用於您的賬戶。

以下部分描述了 Stripe 訂閱清單中每一列的含義。

1. **Stripe 訂閱 ID**：Stripe 訂閱的 ID
2. **創建於**：訂閱建立的日期
3. **數量**：此訂閱的許可數量
4. **狀態**：可用訂閱標記為 `active`（活躍）。如果您取消了訂閱，它將標記為 `canceled`（已取消）。[檢視其他狀態值](https://docs.stripe.com/api/subscriptions/object#subscription_object-status)。
5. **註冊的設備**：與此訂閱關聯的註冊的設備數量。您可以檢視[註冊的設備](https://support.emeditor.com/en/account/devices)以獲取有關您設備的詳細信息。

## 支持

如果訂閱清單包含不正確的信息，請[聯繫我們](https://www.emeditor.com/support/#contact)。
