# 註冊的設備頁面

[註冊的設備頁面](https://support.emeditor.com/zh-hant/account/devices)包含您已註冊過 EmEditor 的所有設備的清單。

## Columns

每個設備記錄包含以下信息。

1. **設備 ID**：這是此設備的標識符，是註冊表項、使用者 ID 和機器 ID 的唯一組合。當您把 EmEditor 註冊到新設備時，會創建一條新的設備記錄。
2. **註冊金鑰**：這是用於註冊該設備的註冊金鑰。如果您購買 Stripe 訂閱，則此欄位為空。
3. **Stripe 訂閱 ID**: 這是用於註冊該設備的 Stripe 訂閱。如果您使用註冊金鑰，則此欄位為空。
3. **機器 ID**：此 ID 標識了這是一個 Windows 安裝。該值是從 `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Cryptography\MachineGuid` 的 Windows 註冊表值複製而來的。
4. **註冊日期**：您在設備上註冊 EmEditor 的日期。
5. **最後驗證日期**：EmEditor 最後一次串連到我們的服務器以驗證註冊的日期。每次當您在設備上打開 EmEditor 時都會進行一次驗證。EmEditor 可能會在驗證期間串連到我們的服務器，但這並不是每次都會發生。因此，最後一次驗證日期可能不是您上次打開 EmEditor 的時間。
6. **安裝類型**：該值取決於您如何安裝 EmEditor，可以是「桌面版」、「可攜式版」、或「市集應用版」。
7. **標籤**：該值包含您在 **註冊產品** 對話方塊的 **標籤** 欄位中寫入的內容。該標籤可說明您了解您的哪臺計算機與此設備記錄相對應。
8. **編輯/註銷**：單擊「編輯」可以變更標籤、檢查設備是否已註冊以及註銷設備。

## 設備限制

您可以為每個授權註冊一定數量的設備。該限制在您的 EULA 中註明。如果您註冊的設備數量超過允許的數量，您將在 EmEditor 中看到一條通知。

## 註銷

要從一個設備上註銷 EmEditor，請到該設備上解除安裝 EmEditor 或單擊 **說明** 功能表中的 **降級**。您還可以單擊此表中的「編輯」來註銷此設備。
