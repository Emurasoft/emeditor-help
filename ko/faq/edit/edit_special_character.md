# 질문. 어떻게 특수 문자를 삽입할 수 있습니까?

**편집** 메뉴의 **고급** 을 선택한 후, **특수 문자 삽입** 을 선택해 주십시오.
삽입하고자 하는 유니코드 문자 코드를 삽입하기 위해서,
[라틴 확장 문자 참조](../../howto/edit/special_char_sample) 를 참고해 주십시오.
만일 사용자의 윈도우 운영 체제에 설치되어 있다면, **문자 맵** 을 이용해서 문자를 삽입할 수 있습니다.
(시작 메뉴 > 모든 프로그램 > 보조 프로그램 > 시스템 도구 \> 문자 맵)
키보드 바로가기의 목록을 위해서 특수 문자를 제대로 디스플레이 하기 위해서 글꼴을 조정해야 합니다.

키보드 바로 가기에 대한 [특수 문자 삽입하는 법](../../howto/edit/edit_special_char) 을 참고하십시오.
