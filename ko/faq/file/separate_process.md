# 질문. 엠에디터는 다른 응용프로그램으로 부터 외부 문서 편집기로 지정되어 있습니다. 엠에디터에서 바뀐 내용이 왜 응용 프로그램에 적용되지 않는것 입니까?

만일 엠에디터가 메일 클라이언트 프로그램과 같은 다른 응용 프로그램에서 외부 문서 편집기로 지정되어 있고,
응용 프로그램이 끈내기와 파일 바꾸기를 위해서 엠에디터의 과정을 모니터 한다면,
응용 프로그램은 엠에디터에서 편집되고 저장된 파일을 반영할 수 없을것입니다.
작동하게 하기 위해서는, 엠에디터를 다른 프로세스로서 시작하기 위해서 [/sp 옵션](../../howto/file/file_commandline) 을
지정해 주십시오. 그러나 이 경우, 엠에디터 창은 분리된 그룹으로서 디스플레이 됩니다.
