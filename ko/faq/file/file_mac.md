# 질문. 매킨토시 텍스트 파일을 읽을 때, 어떤 문자들은 다른 문자로 변환됩니다. 어떻게 매킨토시 텍스트 파일을 정확하게 읽을 수 있나요?

매킨토시는 윈도우와 약간 다른 코드를 사용합니다.
윈도우 2000/XP/2003/VISTA에서, 매킨토시 토드 페이지가 설치되었을 때,
당신은 메킨토시 텍스트 파일을 윈도우 텍스트 파일로 변환시킬 수 있습니다.
첫번째로, [**인코딩 정의**](../../cmd/tools/define_code_page) 대화 상자에서,
**새로 만들기** 버튼을 누르고, 매킨토시 인코딩을 선택해 주십시오. 예를 들어,
"10001 (MAC - Japanese)"가 있습니다.
대화를 닫기 위해서 **확인** 을 두번 클릭하십시오.

다음으로, **파일** 메뉴에서, [**열기** 명령](../../cmd/file/file_open) 을 선택하고, 정의괸 인코딩을
선택하십시오. 예를 들어, **코드 페이지** 콤보 박스의 "10001 (MAC - Japanese)" 그리고 나서
읽고자 하는 매킨토시 파일을 선택해 주십시오.

윈도우 98/Me에서, 매킨토시 코드 페이지가 설치되지 않았다면, 매킨토시 텍스트 파일을 정확하게 읽을 수 없습니다.
만일 특수문자가 포함되어 있다면, 윈도우 코드 페이지에서 읽을 수 없습니다.
