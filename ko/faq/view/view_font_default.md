# 질문. 초기값을 이용해서 파일을 열 때, 읽을 수 없고 만일 글꼴 범주 하위 메뉴에서 다른 문자 세트를 선택하면 이것이 잘못되었다고 뜹니다. 하지만 파일을 열 때, 올바른 인코딩을 선택하면 작동합니다. 왜 이렇게 되는 것 인가요?

엠에디터의 내부 프로세스는 유니코드 입니다. 엠에디터는 ANSI파일을 열 때, ANSI 텍스트를 유니코드 텍스트로 변환합니다. 파일을 열 때, **대화 상자 열기** 에서 인코딩을 선택해 주셔야 합니다. 만일 인코딩을 선택하지 않으셨다면, 엠에디터는 파일을
[시스템 초기 인코딩](../../glossary/systemdefaultencoding) 으로 초기화 할 것입니다. 다른 인코딩으로 파일을 열고자 하신다면,인코딩을 지정하신 후 열어주십시오. 나중에 **파일** 메뉴에서
[**다시로드할 인코딩 리스트**](../../cmd/file/file_reload_defined) 를 선택하면 인코딩을 바꾸실 수 있습니다.
