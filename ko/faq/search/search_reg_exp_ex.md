# 질문. 정규식의 예에는 무엇이 있나요?

- 큰 따옴표로 둘러싸인 줄

".\*?"
- \[ \]으로 둘러싸인 줄

\\\[\[^\\\[\]\*?\\\]
- 여러 이름

\[a-zA-Z\_\]\[a-zA-Z\_0-9\]\*
- IP 주소

(\[0-9\]{1,3})\\.(\[0-9\]{1,3})\\.(\[0-9\]{1,3})\\.(\[0-9\]{1,3})
- URL

(\\S+)://(\[^:/\]+)(:(\\d+))?(/\[^#\\s\]\*)(#(\\S+))?
- 탭으로 이어지는 줄

\\t.\*$
- 히라카나

\[\\x{3041}-\\x{309e}\]
- 전자 가타카나

\[\\x{309b}-\\x{309c}\\x{30a1}-\\x{30fe}\]
- 반자 카나

\[\\x{ff61}-\\x{ff9f}\]
- CJK 표의 문자

\[\\x{3400}-\\x{9fff}\\x{f900}-\\x{fa2d}\]
- CJK 표의 문자 표시

\[\\x{3000}-\\x{3037}\]
- 한글

\[\\x{1100}-\\x{11f9}\\x{3131}-\\x{318e}\\x{ac00}-\\x{d7a3}\]
- 삽입 // 줄의 시작에 삽입

바꾸기 with: //
- 제거 // 줄의 시작에서 제거

찾기: ^//

- 후행 공백을 제거

- (abc)를 \[abc\]로 바꾸기

찾기: <H3(.\*?)>
