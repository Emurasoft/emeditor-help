# 버전 12

2012년 9월 25일

## 새로운 일반 기능

- **대응 태그 강조**\- 새로운 엠에디터는 XHTML과 XML 문서에서 커서의 태그와 일치하는 태그를 자동으로 강조하여 보여줍니다. 이 새로운 기능으로 문서읽기와 대응태그 찾기가 훨씬 간편해졌습니다. 이 기능은 잘 짜인 XML 문서 내에서만 사용 가능하며 엔딩 태그가 문서 내에 존재하지 않을시(오래된 HTML 문서의
경우) 에는 작동되지 않을 수 있습니다. 대응태그 강조 기능이 실행되기 위해서는 구성 속성 탭의 **강조하기(2)** 에서 **대응태그 강조하기** 체크 상자가 설정되어 있어야 합니다. 또한, 보다 간편한 태그와 대응태그 간의 이동을 위해 **대응태그 찾기** 와 **대응태그 확장** 명령이 추가되었습니다. CTRL + . 와 CTRL + SHIFT + . 가 기본 단축키로 설정되어 있습니다.
- **향상된 책갈피**\- 책갈피 기능이 더욱 개선되었습니다. 이제 검색된 줄 또는 두 파일 간의 변경을 책갈피로 표시할 수 있습니다. 또한 문서 내 책갈피 반전, 책갈피된 줄 삭제 및 책갈피된 줄을 새로운 파일로 가져오기 등도 가능해졌습니다.
- **향상된 마커 기능**\- 마커 기능 또한 더욱 향상되었습니다. 새로운 **마커에서 추가/제거** 명령기능으로 마커 도구모음에서 텍스트 추가와 제거하기가 빠르고 간편해졌습니다. 마크된 텍스트를 다른 텍스트로 대체하는 **바꾸기** 명령이 마우스 오른쪽 클릭
메뉴에 추가되었습니다. 자동 마커를 영문자와 숫자에서만 사용하는 것을 옵션으로 선택할 수 있게 되었습니다. 또한 다른 여러 유용한 옵션들이 **마커 사용자 지정** 대화상자의 옵션 탭에 추가되었습니다.
- **메시지 (플러그인)**\- 간단한 인트라넷 통신 기능이 추가되었습니다. 짧은 텍스트를 선택한 후 **플러그인** 도구모음이나 메뉴의 **선택된 텍스트 보내기** 플러그인으로 **보내는곳** 대화상자가 나타나게 한 다음, 텍스트 메시지를 보낼 컴퓨터를 지정합니다. 컴퓨터를 선택하거나 **모든**
**컴퓨터로 보내기** 체크 상자를 이용하여 선택된 텍스트를 지정된 컴퓨터(들)로 보냅니다. 텍스트를 받기 위해서는 텍스트를 받고자 하는 컴퓨터에 엠에디터가 설치되어 있고, 엠에디터 트레이 아이콘이 실행되고 있어야 하며 데스크톱이 표시되어 있어야만 합니다. 텍스트를 받을 때, 말풍선이 나타납니다. 말풍선을 클릭하거나 마우스 오른쪽 버튼으로 트레이 아이콘을 클릭, **마지막 수신된 메시지** 명령을 선택하면 새로운
엠에디터 창이 받은 메시지와 함께 열립니다. 보내는 텍스트의 길이에는 제한이 있으며, 윈도우 2000에서는 트레이 아이콘에 말풍선이 나타나지 않는다는 점을 유의해 주세요. 윈도우 8의 경우, 말풍선 대신 **토스트** 가 나타납니다. 플러그인은 라이브러리에서 찾으실 수 있습니다.

- **최적화**\- 엠에디터를 시작할 때, 새로운 버전은 내 매크로 파일에서 #icon과 #title을 더 이상 스캔하지 않습니다. 이것으로 보다 빠른 엠에디터 시작 구동이 가능해졌습니다. 이 #icon 과 #title 줄들이 엠에디터 시작 이전에 변경되었을 경우에는, **내 매크로**
도구모음을 마우스 오른쪽 클릭하여 **새로고침** 을 해주셔야 합니다.
- **윈도우 8 지원** \- 새로운 버전부터 윈도우 8이 지원됩니다. 스크롤, 자르기, 복사, 붙여넣기, 삭제, 되돌리기, 다시하기, 뒤로, 앞으로, 프린트, 저장, 그리고 기타 다른 기능들이 터치 입력으로 지원됩니다. 오른손 사용자 설정이 기본값으로 된 태블렛 PC에서는 메뉴가 왼쪽에 나타납니다. 윈도우 8에서는
엠에디터가 다른 PC에서 메시지를 받을 때마다 토스트가 메시지 수신을 나타내어 메트로앱이 전경 창으로 되어 있는 경우에도 수신된 메시지를 볼 수 있습니다.
- **익스플로어 컨텍스트 메뉴** 가 이제 64비트 윈도우의 32비트 엠에디터에서 지원됩니다.
- **찾기** 대화상자가 열려있고 포커스되어 있을 때 F3 또는 Shift + F3을 누르게 되면 **아래 찾기** 또는 **위로 찾기** 버튼을 눌렀을 때와 같은 기능을 합니다.
- **모두 교체** 등과 같은 여러 단축키들이 찾기 도구모음이 키보드 포커스 설정되어 있는 경우에도 활성화됩니다.
- **일괄작업 교체** 대화상자가 JavaScript 또는 VBScript 파일로 추출될 수 있습니다.
- **onenote://** 가 이제 하이퍼링크 되었습니다.
- **문자 코드 값** 명령이 이제 유니코드 값에 더해 유니코드 대체 값 또한 보여줍니다. 예를 들어, U+10000는 아래와 같이 나타납니다:

U+10000

U+D800 U+DC00
- **뒤로** 와 **앞으로** 도구모음 버튼이 추가되었습니다.
- 새로운 **찾기 드롭다운 목록 상자** 가 키맵 창의 도구모음에 추가되었습니다.
- 구성 속성의 **키보드** 탭에 새로운 **>** 와 **<** 버튼이 추가되었습니다. 이 버튼들은 현재 할당된 명령 또는 기록 내 이전 명령으로 점프할 수 있게 합니다.
- **CR/LF 리턴을 클립보드에 보존** 이 체크되어있을 때, CSV문서에서 수직선택 붙여넣기는 이제 특정 조건에 따라 리턴을 보존합니다.
- 라인이 래핑되어있을 시에는 수직 선택 붙여넣기 동작이 바뀝니다.
- 이제 도구모음에서 토글끄기와 켜기를 할 때마다 도구모음 위치가 기억되며, 스크린 넘기기 동작이 감소합니다.
- 새로운, 닫기, 자르기, 복사, 찾기, 붙여넣기, 삭제, 취소, 다시하기, 뒤로, 앞으로, 프린트, 저장, 검색, 도움, 맞춤법, 그리고 기타 기능들과 같은 더 다양한 어플리케이션 명령들이 특정 마우스와 키보드에서 지원됩니다.

## 새로운 플러그인 기능들

### HTML Bar 플러그인

- 이제 **선택한 구성에서 자동으로 HTML Bar 보여주기** 체크 상자가 설정되지 않은 경우에도 HTML Bar가 도구모음 창에 나타내어집니다.

### 코드조각 플러그인

- 이제 CTRL 키를 누르고 코드조각 트리위에 아이템을 끌어다놓음으로써 아이템을 복사할 수 있습니다.
- 텍스트를 텍스트 영역, 혹은 다른 응용 프로그램에서 가져오게 되는 경우에는 코드조각 타이틀이 79번째 문자에서 잘려집니다.
- **자르기** (Ctrl+X), **복사** (Ctrl+C), 그리고 **붙여넣기** (Ctrl+V) 명령기능들이 **코드조각** 플러그인에 추가되었습니다. 이 명령기능들로 코드조각 트리에서 코드조각 아이템들을 보다 쉽게 복사하거나 옮길 수 있습니다.
또한 **붙여넣기** 기능의 경우, 간단한 코드조각 텍스트를 코드조각 트리에 삽입하는데에 사용될 수 있습니다.
- 코드조각 트리에 마우스 포인터를 갖다대면 코드조각 텍스트가 풍선 도움말의 형태로 나타납니다(이 기능을 활성화하기 위해서는  **컨텍스트 메뉴 \- 보기** \- **풍선 도움말** 이 체크되어 있어야 합니다),

### 프로젝트 플러그인

- **현재 파일** 이 **기호목록** 사용자 지정바의 드롭다운 목록에 추가되었습니다. 이로써 프로젝트중이 아닌 파일도 보는 것이 가능해졌습니다.
- **새로고침** 버튼이 **기호목록** 사용자 지정바에 **새로고침** 버튼이 추가되었습니다.
- 이제 프로젝트중이 아닌 파일에도 기호목록이 사용 가능합니다. **현재 파일과 모든 열린 파일** 아이템이 기호 드롭다운 목록에 추가되었습니다.
- **모두 확장** 과 **모두 축소** 명령 기능들이 컨텍스트 메뉴에 추가되었습니다.
- Ctags.exe가 최신버전으로 대체되었습니다(Ctags v5.8).

## 새로운 옵션들

- **마우스 오른쪽 클릭 \+ 휠로 탭 스위치** 체크 상자가 사용자 지정 대화상자의 마우스 탭에 추가되었습니다.
- **외부 응용 프로그램에 복사된 클립 보드를 모니터** 체크 상자가 사용자 지정 대화 상자의 기록 탭에 추가되었습니다. 기본 설정 모드의 엠에디터는 마이크로소프트 엑셀과 같은 특정 응용 프로그램과의 작동 중지를 피하기 위해 클립보드를 모니터하지 않습니다.
