# 버전 4.05

2004년 8월 4일

- 주석 처리된 행 후에 자동 들여쓰기 동작을 수정하였습니다.
- 백슬래시로 끝나는 정규식 문자열을 검색하면 EmEditor가 작동을 멈추던 오류를 수정하였습니다.
- EmEditor가 80 문자 이상의 줄을 포함하는 구문 파일을 가져오기 할 수 없던 오류를 수정하였습니다.
- EmEditor가 특정 조건 아래에서 "\\\"와 같은 표현을 파일 내 에서 바꾸기 할 수 없던 오류를 수정하였습니다.
- 플러그 인 사용자 지정 대화 상자에서 선택된 플러그 인을 제거할 수 없을 때 메시지를 추가하였습니다.
- 창이 결합되지 않았을 때 EmEditor가 파일 내 찾기 명령 후에 창 최소화를 클릭하여 저장된 창 크기를 복원하지 않는 오류를 수정하였으며, 창 위치가 저장됩니다.
- Windows XP 태블릿 PC 버전에서 파일 내 찾기를 하는 도중 EmEditor가 작동을 멈추던 오류를 수정하였습니다.
- "다른 프로그램으로 변경" 드롭 다운 리스트 박스에 "확인"을 선택하였을 때 현재 파일을 삭제하고 같은 이름으로 바꾸려는 경우에 확인 메시지가 나타나지 않던 오류를 수정하였습니다.
- "다른 프로그램으로 변경" 드롭 다운 리스트 박스에 "확인"을 선택하였을 때 확인 메시지를 표시하는 메인 창이 활성화되지 않던 오류를 수정하였습니다.
