# 버전 14

2013년 12월 9일

## 새로운 일반 기능

- 다중 선택이 강화되었습니다. 다중 선택이 만들어 질 때, 선택된 여러 위치에 커서가 동시에 나타나게 됩니다.
- 네 개의 새로운 명령: [퍼센트 인코딩을 유니코드로 (현재 인코딩)](../cmd/edit/decode_percent), [유니코드를 퍼센트 인코딩으로 (현재 인코딩)](../cmd/edit/encode_percent), [퍼센트 인코딩을 유니코드로 (UTF-8)](../cmd/edit/decode_percent_utf8), 및
[유니코드를 퍼센트 인코딩으로 (UTF-8)](../cmd/edit/encode_percent_utf8) 가 추가되었습니다. 퍼센트 인코딩으로 인코딩 된 텍스트 위에 마우스 포인터를 가져가면 툴팁이 자동으로 디코딩된 텍스트를 표시합니다.
- [줄 위로 이동](../cmd/edit/move_line_up) 과
[줄 아래로 이동](../cmd/edit/move_line_down) 명령이 추가되었습니다.
- 이제찾기 대화 상자가 열렸을 때,찾기 명령이나CTRL + F 를 누르면 키보드 포커스가찾기 드롭 다운 리스트 박스로 설정됩니다. 마찬가지로, 바로가기로 활성화되었거나 메뉴에서 선택되었을 때바꾸기,파일 내 찾기 및파일 내 바꾸기 대화 상자는찾기 드롭 다운 리스트 박스에 키보드 포커스를 설정합니다.
-매크로 사용자 지정 또는메뉴 대화 상자를 닫을 때 화면 새로 고침을 줄이기 위해 최적화하였습니다.
- _TeamViewer_ 가 동시에 실행되고 로그인되었을 때 EmEditor가 작동을 중지하였던 문제를 해결하였습니다.

- [유지 관리 키 입력 대화 상자](../dlg/enter_maintenance/index) 가 추가되었습니다.
