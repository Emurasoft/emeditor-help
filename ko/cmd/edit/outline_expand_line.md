# 선 확대 명령

### 요약

> 윤곽이 표시되는 동안 현재 선을 확대합니다.

### 설명

> 윤곽이 표시되는 동안 이 명령은 현재 선을 확대합니다. 텍스트 줄의 왼쪽에서 "+" 버튼을 누른것과 동일하게 작용됩니다.
> 윤곽을 표시하기 위해서는 플러그 인 윤곽이 필요합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **윤곽** \> **선 확대**
- [모든 명령](../tools/all_commands): **편집** \> **윤곽** \> **선 확대**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_OUTLINE\_EXPAND\_LINE (4414)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4414);

#### \[VBScript\]

> editor.ExecuteCommandByID 4414
