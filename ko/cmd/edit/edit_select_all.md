# 모두 선택 명령

## 요약

모두 선택합니다.

## 설명

현 문서의 모든 문자를 선택합니다.

## 실행하는 방법

- 기본 메뉴:편집 \>모두 선택
- [모든 명령](../tools/all_commands):편집 \>선택 확장 \>모두 선택
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+A

## 플러그인 명령 ID

```
EEID_EDIT_SELECT_ALL (4136)```

## 매크로

## \[JavaScript\]

```
document.selection.SelectAll();
```

## \[VBScript\]

```
document.selection.SelectAll
```
