# IME 명령

## 요약

입력기를 활성화 또는 비활성화합니다.

## 설명

극동 언어 입력기를 활성화 또는 비활성화합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>고급 \>IME
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+F11

## 플러그인 명령 ID

```
EEID_TOGGLE_IME (4198)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4198);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4198
```
