# 창 오른쪽 명령

## 요약

현재 창의 오른쪽 가장자리로 이동합니다.

## 설명

현재 창의 오른쪽 가장자리로 이동합니다.
이 명령은 가로 스크롤 위치를 이동시키지 않습니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **가로로 커서 이동** \> **창 오른쪽**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_WINDOW_RIGHT (4294)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4294);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4294
```
