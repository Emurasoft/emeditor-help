# 줄의 끝부터 선택 명령

### 요약

> 선택 영역 줄의 끝부터 선택합니다.

### 설명

> 세로 선택 편집을 할 선택 영역 줄의 끝 부분을 선택합니다. 이 행들을 동시에 편집할 수 있습니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **선택 모드** \> **줄의 끝부터 선택**
- [모든 명령](../tools/all_commands): **편집** \> **선택 모드** \> **줄의 끝부터 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SELECT\_END\_LINES (3849)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3849);

#### \[VBScript\]

> editor.ExecuteCommandByID 3849
