# 반자/전자 전환 명령

### 요약

> 반자 및 전자 사이의 선택을 변환합니다.

### 설명

> 이 명령은 반자 문자는 전자 문자로, 전자 문자는 반자 문자로 변환시킵니다.
> 이 명령을 실행 하면 문자와 선택의 유형을 설정 할 수 있는 [**반자/전자 전환** 대화 상자](../../dlg/convert/index) 가
> 표시됩니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **고급**
\> **반자/전자 전환**
- [모든 명령](../tools/all_commands): **편집** \> **고급**
\> **반자/전자 전환**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+T

### 플러그인 명령 ID

- EEID\_CONVERT (4148)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4148);

#### \[VBScript\]

> editor.ExecuteCommandByID 4148
