# 삭제된 문자열 복원 명령

## 요약

커서 위치에 이전에 삭제된 문자열을 복원합니다.

## 설명

커서 위치에 가장 최근에 삭제된 문자열을 복원합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>붙여넣기 \>삭제된 문자열 복원
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_RESTORE_DELETED (4419)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID (4419);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4419
```
