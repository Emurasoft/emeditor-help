# 좁히기 초기화 명령

## 요약

전체 문서에 액세스가 가능하게 합니다.

## 설명

전체 문서에 액세스가 가능하게 합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>좁히기 \>좁히기 초기화
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_NARROWING_OFF (4458)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4458);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4458
```
