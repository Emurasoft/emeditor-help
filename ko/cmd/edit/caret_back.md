# 뒤로 명령

### 요약

> 이전 위치로 커서를 이동합니다.

### 설명

> 이전 위치로 커서를 이동합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **고급** \> **뒤로**
- [모든 명령](../tools/all_commands): **편집** \> **고급** \> **뒤로**
- 도구 모음:
![](../../images/emeditor12_back_button.png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT + LEFT

### 플러그인 명령 ID

- EEID\_CARET\_BACK (4597)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4597);

#### \[VBScript\]

> editor.ExecuteCommandByID 4597
