# 복사 및 선택 취소 명령

### 요약

> 선택 영역을 복사하여 클립보드에 붙여넣고 텍스트 선택을 취소합니다.

### 설명

> 선택 영역을 복사하여 클립보드에 붙여넣고 텍스트 선택을 취소합니다.
> 이 명령은 [**복사** 명령](edit_copy) 에 따른 [**취소** 명령](escape) 과 동일하게 작용됩니다.
> 이 명령 후에 다른 위치로 커서를 움직여 [**붙여넣기** 명령](edit_paste) 을 실행 하여 선택 사항을 붙여 넣을 수 있습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **복사**
> **복사 및 선택 취소**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+SHIFT+C

### 플러그인 명령 ID

- EEID\_EDIT\_COPY\_DESELECT (4128)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4128);

#### \[VBScript\]

> editor.ExecuteCommandByID 4128
