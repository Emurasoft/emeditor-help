# 양음 부호 명령

## 요약

양음 부호로 문자를 삽입합니다.

## 설명

이 명령을 선택한 뒤에 커서 위치에 a, e, i, o, u, y, s, z, A, E, I, O, U, Y, S 또는 Z를 입력하면 양음 부호 문자 (á, é,
í, ó, ú, ý, ś, ź, Á, É, Í,
Ó, Ú, Ý, Ś 또는 Ź)를, \`를 입력하면 작은 따옴표 표시(’), 또는 "를 입력하면 큰 따옴표(”)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **양음 부호**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+'

## 플러그인 명령 ID

```
EEID_INSERT_ACUTE (4304)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID (4304);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4304
```
