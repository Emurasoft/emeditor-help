# 책갈피 초기화 명령

### 요약

> 현재 줄에 책갈피를 초기화시킵니다.

### 설명

> 현재 줄에 책갈피를 초기화시킵니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **책갈피** \> **책갈피 초기화**
- [모든 명령](../tools/all_commands): **편집** \> **책갈피** \> **책갈피 초기화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_BOOKMARK\_RESET (4354)

### 매크로

#### \[JavaScript\]

> document.selection.ClearBookmark();

#### \[VBScript\]

> document.selection.ClearBookmark
