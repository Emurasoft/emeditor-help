# 번호 매기기 명령

### 요약

> 커서의 위치 또는 수직 선택에 번호를 삽입합니다.

### 설명

> 커서의 위치 또는 수직 선택에 번호를 삽입합니다.
> 이 명령을 실행 하면 삽입할 숫자들을 지정할수 있는 [**번호 매기기** 대화 상자](../../dlg/insert_numbering/index) 가
> 표시됩니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **고급**
\> **번호 매기기**
- [모든 명령](../tools/all_commands): **편집** \> **고급**
\> **번호 매기기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+N

### 플러그인 명령 ID

- EEID\_NUMBERING (4558)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4558);

#### \[VBScript\]

> editor.ExecuteCommandByID 4558
