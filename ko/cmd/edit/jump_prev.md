# 이전 점프 명령

### 요약

> 출력 바에서 이전 위치로 점프합니다.

### 설명

> 출력 바에서 이전 위치로 점프합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **고급**
\> **이전 점프**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: SHIFT+F4

### 플러그인 명령 ID

- EEID\_JUMP\_PREV (4443)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4443);

#### \[VBScript\]

> editor.ExecuteCommandByID 4443
