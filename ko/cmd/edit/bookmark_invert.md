# 이 문서의 책갈피 반전 명령

### 요약

> 현재 문서의 모든 책갈피를 반전시킵니다.

### 설명

> 책갈피가 되어있지 않은 줄은 책갈피를 하고, 책갈피가 되어있는 줄은 책갈피를 없애어 현재 문서의 모든 책갈피를 반전시킵니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **책갈피** \> **책갈피 반전**
- [모든 명령](../tools/all_commands): **편집** \> **책갈피** \> **책갈피 반전**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_BOOKMARK\_INVERT (4588)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4588);

#### \[VBScript\]

> editor.ExecuteCommandByID 4588
