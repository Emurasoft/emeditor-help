# 유니코드를 퍼센트 인코딩으로 (현재 인코딩) 명령

### 요약

> 선택된 텍스트를 현재 인코딩을 사용하여 퍼센트 인코딩으로 인코딩합니다.

### 설명

> 선택된 텍스트를 현재 인코딩을 사용하여 퍼센트 인코딩으로 인코딩합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **인코드/디코드 선택** \> **유니코드를 퍼센트 인코딩으로 (현재 인코딩)**
- [모든 명령](../tools/all_commands): **편집** \> **인코드/디코드 선택**
\> **유니코드를 퍼센트 인코딩으로 (현재 인코딩)**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_ENCODE\_PERCENT (3855)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3855);

#### \[VBScript\]

> editor.ExecuteCommandByID 3855
