# 대응태그 찾기 명령

### 요약

> 대응하는 태그로 커서를 이동합니다.

### 설명

> XML 또는 XHTML 문서에 커서가 위치하는 경우, 이 명령은 대응하는 태그로 커서를 이동합니다.
> 만약 태그 영역의 외부에 커서가 위치하는 경우에는 대응하는 태그의 외부로 커서를 이동합니다.
> 만약 태그 역영의 내부에 커서가 위치하는 경우에는 대응하는 태그의 내부로 커서를 이동합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **고급** \> **대응태그 찾기**
- [모든 명령](../tools/all_commands): **편집** \> **가로로 커서 이동** \> **대응태그 찾기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+.

### 플러그인 명령 ID

- EEID\_NEXT\_TAG (4601)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4601);

#### \[VBScript\]

> editor.ExecuteCommandByID 4601
