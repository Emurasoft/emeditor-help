# 억음 부호 명령

### 요약

> 억음 부호로 문자를 삽입합니다.

### 설명

> 이 명령을 선택한 뒤에 커서 위치에 a, e, i, o, u, A, E, I, O 또는 U를 입력하면 억음 부호 문자 (à, è, ì, ò, ù, À, È, Ì,
> Ò 또는 Ù)를, \`를 입력하면 작은 따옴표 표시(‘), 또는 "를 입력하면 큰 따옴표(“)를 삽입합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **억음 부호**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+\`

### 플러그인 명령 ID

- EEID\_INSERT\_GRAVE (4303)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4303);

#### \[VBScript\]

> editor.ExecuteCommandByID 4303
