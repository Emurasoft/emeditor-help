# 좁히기 설정 명령

## 요약

선택된 부분 만을 편집 가능한 영역으로 설정하고 나머지는 엑세스 할수 없도록 합니다.

## 설명

선택된 부분 만 (전체 논리 줄 포함)을 편집 가능한 영역으로 설정하고 나머지는 엑세스 할수 없도록 합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>좁히기 \>좁히기 설정
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_NARROWING_ON (4457)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4457);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4457
```
