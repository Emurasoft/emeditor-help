# 줄 아래로 명령

## 요약

한 줄 아래로 커서를 이동합니다.

## 설명

한 줄 아래로 커서를 이동합니다. 아래 화살표키를 누른것과 동일시 작용됩니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **세로로 커서 이동** \> **줄 아래로**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: DOWN ARROW

## 플러그인 명령 ID

```
EEID_DOWN (4161)```

## 매크로

### \[JavaScript\]

```
document.selection.LineDown(false,1);
```

### \[VBScript\]

```
document.selection.LineDown false,1
```
