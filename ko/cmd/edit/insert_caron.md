# 반대 곡절 부호 명령

## 요약

반대 곡절 부호로 문자를 삽입합니다.

## 설명

이 명령을 선택한 후에, a, i, o, u, A, I, O 또는 U를 입력하면 커서 위치에 반대 곡정 부호 문자 (ǎ, ǐ, ǒ, ǔ, Ǎ, Ǐ, Ǒ 또는 Ǔ)를
삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **반대 곡절 부호**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+SHIFT+6

## 플러그인 명령 ID

```
EEID_INSERT_CARON (4369)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4369);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4369
```
