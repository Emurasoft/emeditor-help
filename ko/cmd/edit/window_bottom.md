# 창 아래쪽 명령

## 요약

현재 창의 아래쪽으로 커서를 이동합니다.

## 설명

현재 창의 아래쪽으로 커서를 이동합니다. 이 명령은 세로 스크롤 위치를 이동시키지 않습니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **세로로 커서 이동** \> **창 아래쪽**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+PAGE DOWN

## 플러그인 명령 ID

```
EEID_WINDOW_BOTTOM (4293)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4293);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4293
```
