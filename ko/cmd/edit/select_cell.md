# 셀 선택 명령

### 요약

> CSV, TSV, 또는 DSV 모드에서 현재 커서 위치의 셀을 선택합니다.

### 설명

> CSV, TSV, 또는 DSV 모드에서 현재 커서 위치의 셀을 선택합니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **구분된 값/정렬** \> **셀 선택**
- [모든 명령](../tools/all_commands): **편집** \> **구분된 값/정렬** \> **셀 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SELECT\_CELL (4489)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4489);

#### \[VBScript\]

> editor.ExecuteCommandByID 4489
