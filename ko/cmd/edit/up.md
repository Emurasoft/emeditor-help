# 줄 위로 명령

## 요약

한 줄 위로 커서를 이동합니다.

## 설명

한 줄 위로 커서를 이동합니다.
위 화살표키를 누르는 것과 동일시 작용합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>세로로 커서 이동 \>줄 위로
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: UP ARROW

## 플러그인 명령 ID

```
EEID_UP (4160)```

## 매크로

### \[JavaScript\]

```
document.selection.LineUp(false,1);
```

### \[VBScript\]

```
document.selection.LineUp false,1
```
