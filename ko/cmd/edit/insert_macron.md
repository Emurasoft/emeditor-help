# 장음 명령

## 요약

장음 부호로 문자를 삽입합니다.

## 설명

이 명령을 선택한 후에, a, i, o, u, A, I, O 또는 U를 입력하면 커서 위치에 장음 부호 문자 (ā, ē, ī, ō, ū, Ā, Ē, Ī, Ō 또는 Ū)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **장음**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT + -

## 플러그인 명령 ID

```
EEID_INSERT_CARON (3861)```
