# 다시 변환 명령

## 요약

재변환이 가능한 IME를 사용하여 선택 영역을 다시 변환합니다.

## 설명

이미 변환된 텍스트를 일반적으로 일본어 IME에서 볼수 있는 재변환이 가능한 입력기(IME)를 사용하여 다시 변환합니다.

## 실행하는 방법

- 기본 메뉴:편집 \>선택 변환 \>다시 변환
- [모든 명령](../tools/all_commands):편집 \>선택 변환 \>다시 변환
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음 (IME 설정은 일반적으로 일본어 키보드의변환 키 사용을 허용합니다.)

## 플러그인 명령 ID

```
EEID_RECONVERT (4199)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4199);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4199
```
