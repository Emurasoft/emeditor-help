# CSV/TSV/DSV 열 선택 명령

## 요약

CSV/TSV/DSV 문서 안에서 수직 선택 모드로 현재 열을 선택합니다.

## 설명

CSV/TSV/DSV 문서 안에서 수직 선택 모드로 현재 열을 선택합니다.

## 실행하는 방법

- 기본 메뉴: **편집** \> **구분된 값/정렬** \> **열 선택**
- [모든 명령](../tools/all_commands): **편집** \> **구분된 값/정렬** \> **열 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음
