# 링크 열기 명령

### 요약

> 링크를 엽니다.

### 설명

> 커서 위치에 하이퍼링크를 엽니다.

### 실행하는 방법

- 기본 메뉴: **편집** \> **링크 열기**
- [모든 명령](../tools/all_commands): **편집** > **고급**
\> **링크 열기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_LINK\_OPEN (4141)

### 매크로

#### \[JavaScript\]

> document.selection.OpenLink();

#### \[VBScript\]

> document.selection.OpenLink
