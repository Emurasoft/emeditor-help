# 반전된 느낌표 명령

## 요약

반전된 느낌표를 삽입합니다.

## 설명

커서 위치에 반전된 느낌표 (¡)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **반전된 느낌표**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+CTRL+SHIFT+1

## 플러그인 명령 ID

```
EEID_INSERT_INVERTED_EXCLAMATION (4313)```

## 매크로

### \[JavaScript\]

```
document.selection.Text="¡";
```

### \[VBScript\]

```
document.selection.Text="¡"
```
