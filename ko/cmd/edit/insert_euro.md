# 유로 명령

## 요약

유로 부호를 삽입합니다.

## 설명

커서 위치에 유로 부호 (€)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **유로**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+CTRL+E

## 플러그인 명령 ID

```
EEID_INSERT_EURO (4317)```

## 매크로

### \[JavaScript\]

```
document.selection.Text="€";
```

### \[VBScript\]

```
document.selection.Text="€"
```
