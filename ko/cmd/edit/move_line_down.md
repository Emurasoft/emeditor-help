# 줄 아래로 이동 명령

## 요약

선택된 줄을 이동하거나 현재 줄을 한 줄 아래로 이동합니다.

## 설명

선택된 줄을 이동하거나 현재 줄을 한 줄 아래로 이동합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>이동
\>줄 아래로 이동
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL + SHIFT + DOWN

## 플러그인 명령 ID

```
EEID_MOVE_LINE_DOWN (3859)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(3859);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3859
```
