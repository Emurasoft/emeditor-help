# 다음 노드 확장 명령

## 요약

윤곽이 표시되는 동안 다음 노드로 선택을 확장합니다.

## 설명

윤곽이 표시되는 동안 다음 노드로 선택을 확장합니다. 윤곽을 표시하기 위해서는 플러그 인 윤곽이 필요합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **윤곽** \> **다음 노드 확장**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: Alt+Shift+Down

## 플러그인 명령 ID

```
EEID_SHIFT_NEXT_NODE (4417)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4417);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4417
```
