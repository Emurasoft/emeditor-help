# 분음 부호표 명령

## 요약

분음 부호표로 문자를 삽입합니다.

## 설명

이 명령을 선택한 후에 a, e, i, o, u, y, A, E,
I, O, U 또는 Y를 입력하면 커서 위치에 움라우트라고도 불리우는 분음 부호표 (ä, ë, ï, ö, ü, ÿ, Ä,
Ë, Ï, Ö, Ü 또는 Ÿ)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **분음 부호표**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL + SHIFT + ;

## 플러그인 명령 ID

```
EEID_INSERT_DIAERESIS (4307)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4307);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4307
```
