# 반전 탭/줄 내어쓰기 명령

### 요약

> 커서를 한 탭 문자 왼쪽으로 이동하거나 줄을 내어씁니다.

### 설명

> 커서를 한 탭 문자 왼쪽으로 이동하거나 줄을 내어씁니다.
> 여러 행이 선택된 경우, 이 명령은 탭 문자나 선택된 각 행의 공백과 일치하는 수를 삭제함으로써 줄을 내어씁니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **가로로 커서 이동** \> **반전 탭/줄 내어쓰기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: SHIFT+TAB

### 플러그인 명령 ID

- EEID\_SHIFT\_TAB (4189)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4189);

#### \[VBScript\]

> editor.ExecuteCommandByID 4189
