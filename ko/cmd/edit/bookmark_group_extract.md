# 이 그룹의 책갈피 설정된 모든 줄을 새 파일로 이동 명령

## 요약

이 그룹에서 책갈피된 모든 줄을 추출하여 새 파일로 이동시킵니다.

## 설명

이 그룹에서 책갈피된 모든 줄을 추출하여 새 파일로 가져옵니다.
이 명령은 기존 문서에서 책갈피된 줄을 제거하지 않습니다.

## 실행하는 방법

- 기본 메뉴:편집 \>책갈피 \>이 그룹 \>책갈피 설정된 모든 줄을 새 파일로 이동
- [모든 명령](../tools/all_commands):편집 \>책갈피 \>이 그룹 \>책갈피 설정된 모든 줄을 새 파일로 이동
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_BOOKMARK_GROUP_EXTRACT (4592)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4592);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4592
```
