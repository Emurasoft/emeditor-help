# 탭 삽입/줄 들여쓰기 명령

## 요약

탭 문자를 삽입하거나 줄을 들여씁니다.

## 설명

커서 위치에서 탭 문자를 삽입합니다. 여러 행이 선택된 경우에, 이 명령은 선택된 각 행의 시작점에 탭 문자를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):편집 \>가로로 커서 이동 \>탭 삽입/줄 들여쓰기
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: TAB

## 플러그인 명령 ID

```
EEID_TAB (4188)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4188);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4188
```
