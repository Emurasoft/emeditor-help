# 이전 틀린 맞춤법 명령

## 요약

이 문서에서 이전 틀린 맞춤법으로 점프합니다.

## 설명

이 문서에서 이전 틀린 맞춤법으로 점프합니다.

## 실행하는 방법

- 기본 메뉴:편집 \>맞춤법 \>이전 틀린 맞춤법
- [모든 명령](../tools/all_commands):편집 \>맞춤법 \>이전 틀린 맞춤법
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+SHIFT+M

## 플러그인 명령 ID

```
EEID_SPELL_PREV (4555)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4555);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4555
```
