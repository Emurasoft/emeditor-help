# 마커에서 추가/삭제 명령

## 요약

선택된 텍스트를 마커 목록에 추가하거나 선택된 텍스트를 마커 목록에서 삭제합니다.

## 설명

선택된 텍스트를 마커 목록에 추가하거나, 마커 리스트에 이미 존재한다면 선택된 텍스트를 마커 목록에서 삭제합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **고급**
\> **마커에서 추가/삭제**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_ADD_REMOVE_MARKERS (4591)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4591);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4591
```
