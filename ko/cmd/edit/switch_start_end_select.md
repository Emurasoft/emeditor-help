# 시작점과 끝점을 전환 명령

## 요약

선택 영역의 시작 지점과 끝 지점을 변경합니다.

## 설명

선택의 반대쪽 끝으로 커서를 이동합니다. (예를 들어, 커서가 현재 선택의 시작점에 있는 경우, 이 명령으로 선택의 마지막 점으로 이동하게 합니다. 반대의 경우에도 마찬가지로 적용됩니다.)

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **선택 모드** \> **시작점과 끝점을 전환**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: Shift + F8

## 플러그인 명령 ID

```
EEID_SWITCH_START_END_SELECT (3850)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(3850);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3850
```
