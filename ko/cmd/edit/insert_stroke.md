# 획 부호 명령

## 요약

획 부호로 문자를 삽입합니다.

## 설명

이 명령을 선택한 후에 o 또는 O를 입력하면 획 부호 문자 (ø 또는 Ø)를 삽입하고, c를 입력하면
센트 부호 (¢)를 삽입합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **삽입**
\> **강조 표시/ 특수 문자 삽입** \> **획 부호**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+/

## 플러그인 명령 ID

```
EEID_INSERT_STROKE (4311)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4311);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4311
```
