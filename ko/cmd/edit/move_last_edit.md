# 마지막 편집된 위치로 이동 명령

## 요약

커서를 마지막 편집 위치로 이동합니다.

## 설명

커서를 현재 문서에서 가장 최근에 편집했던 위치로 이동합니다.

## 실행하는 방법

- 기본 메뉴: **편집** \> **고급** \> **마지막 편집된 위치로 이동**
- [모든 명령](../tools/all_commands): **편집** \> **고급**
\> **마지막 편집된 위치로 이동**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL + SHIFT + Z

## 플러그인 명령 ID

```
EEID_MOVE_LAST_EDIT (4345)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4345);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4345
```
