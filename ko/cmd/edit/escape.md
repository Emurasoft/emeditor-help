# 취소 명령

## 요약

선택을 취소합니다.

## 설명

현재 문서의 선택 내역을 모두 취소합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **편집** \> **선택 확장** \> **취소**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ESC

## 플러그인 명령 ID

```
EEID_ESCAPE (4187)```

## 매크로

### \[JavaScript\]

```
document.selection.Collapse();
```

### \[VBScript\]

```
document.selection.Collapse
```
