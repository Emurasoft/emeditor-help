# 트레이 아이콘 보이기 명령

## 요약

새로운 시스템 트레이 아이콘을 표시합니다.

## 설명

이 명령은 윈도우 작업 표시 줄에 엠에디터 트레이 아이콘을 나타냅니다. 엠에디터 트레이 아이콘이 이미 존재하는 경우에는 아무런 영향을 주지 않습니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):파일 \>새로 만들기 \>트레이 아이콘 보이기
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_NEW_TRAY_ICON (4121)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4121);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4121
```
