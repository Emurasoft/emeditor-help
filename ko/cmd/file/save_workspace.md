# 작업 영역 저장 명령

## 요약

작업 영역을 저장합니다.

## 설명

이 명령은 현재 엠에디터에 열려있는 모든 문서의 전체 경로 이름, 커서 위치, 그리고 기타 설정들을 저장합니다.
[**작업 영역 복원** 명령](load_workspace) 이 이 명령에 의해 저장된 위치와 설정을 복원합니다.

## 실행하는 방법

- 기본 메뉴: **시스템 트레이 아이콘 메뉴** \> **작업 영역 저장**
- [모든 명령](../tools/all_commands): **파일** \> **작업 영역** \> **작업 영역 저장**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+SHIFT+0

## 플러그인 명령 ID

```
EEID_SAVE_WORKSPACE (4330)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4330);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4330
```
