# 작업 영역 삭제 명령

### 요약

> 작업 영역을 삭제합니다.

### 설명

> [**작업 영역 저장** 명령](save_workspace) 에 의해 이전에 저장된 설정 상태를 삭제합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **파일** \> **작업 영역** \> **작업 영역 삭제**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_ERASE\_WORKSPACE (4450)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4450);

#### \[VBScript\]

> editor.ExecuteCommandByID 4450
