# 작업 영역 복원 명령

### 요약

> 저장된 작업 영역의 상태를 복원합니다.

### 설명

> [**작업 영역 저장** 명령](save_workspace) 을 통하여 이전에 엠에디터에 저장된 상태로 현재 열려있는 모든 문서의
> 전체 경로 이름, 커서 위치, 그리고 기타 설정들을 복원합니다.

### 실행하는 방법

- 기본 메뉴: **시스템 트레이 아이콘 메뉴** \> **작업 영역 복원**
- [모든 명령](../tools/all_commands): **파일** \> **작업 영역**
\> **작업 영역 복원**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+0

### 플러그인 명령 ID

- EEID\_LOAD\_WORKSPACE (4329)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4329);

#### \[VBScript\]

> editor.ExecuteCommandByID 4329
