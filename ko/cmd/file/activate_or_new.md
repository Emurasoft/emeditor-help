# 활성화나 새로 만들기 명령

### 요약

> 엠에디터를 활성화하거나 새 파일을 생성합니다 (트레이 아이콘에서만 사용 가능).

### 설명

> 엠에디터가 현재 실행되지 않고 있다면 이 명령으로 엠에디터를 열게 됩니다. 엠에디터가 이미 실행되고 있다면
> 이 명령으로 새로운 무제 문서를 생성하게 됩니다. 이 명령은 트레이 아이콘에서만 사용 가능합니다.
