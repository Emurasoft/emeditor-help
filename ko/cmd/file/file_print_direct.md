# 지금 프린트 명령

### 요약

> 옵션을 지정하지 않고 기본 프린터에서 현재 파일을 인쇄합니다.

### 설명

> 이 명령은 **인쇄** 대화 상자를 표시하지 않고 즉시 인쇄를 실행 합니다.
> 인쇄에 사용하는 프린터는 **장치 및 프린터 (Windows 8, 7, and Vista)** ,또는 제어판의 **프린터 및 팩스 (WinXP)** 및
> **프린터 (Win98 / ME)** 에 기본으로
> 설정되어 있는 프린터 입니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **파일** \> **인쇄** \> **지금 프린트**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FILE\_PRINT\_DIRECT (4115)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4115);

#### \[VBScript\]

> editor.ExecuteCommandByID 4115
