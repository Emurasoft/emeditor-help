# 가장 최근 닫은 문서 명령

### 요약

> 가장 최근에 닫은 문서를 엽니다.

### 설명

> 가장 최근에 닫은 문서를 엽니다. 이 명령은 문서의 저장되지 않은 변경 사항을 복구하지 않습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **파일** \> **열기**
\> **가장 최근 닫은 문서**
- 도구 모음:
없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_MOST\_RECENT\_CLOSED\_FILE (4583)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4583);

#### \[VBScript\]

> editor.ExecuteCommandByID 4583
