# 삽입 명령

## 요약

현재 커서 위치에 지정된 파일을 삽입합니다.

## 설명

이 명령은 현재 파일에 삽입하고 싶은 파일을 선택할 수 있는삽입 대화 상자를 나타냅니다.
파일이 선택되면 이 명령은 선택된 파일을 읽고 커서 위치에 파일의 내용을 삽입합니다.

## 실행하는 방법

- 기본 메뉴:파일 \>삽입 \>삽입
- [모든 명령](../tools/all_commands):파일 \>삽입
\>삽입
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_FILE_INSERT (4108)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4108);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4108
```
