# 엠에디터 홈페이지 가기 명령

### 요약

> 엠에디터 홈페이지를 엽니다.

### 설명

> 기본 인터넷 브라우저에서 엠에디터 웹 사이트를 엽니다.

### 실행하는 방법

- 기본 메뉴: **도움말** \> **엠에디터 홈페이지 가기**
- [모든 명령](../tools/all_commands): **도움말** >
**엠에디터 홈페이지 가기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WEB\_HOME (4249)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4249);

#### \[VBScript\]

> editor.ExecuteCommandByID 4249
