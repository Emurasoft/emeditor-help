# 업그레이드 명령

### 요약

> 이 제품을 엠에디터 Professional로 업그레이드 합니다.

### 설명

> 이 제품을 엠에디터 Professional로 업그레이드 합니다.
> 엠에디터 Professional에서는 이 명령이 가능하지 않습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **도움말** >
**업그레이드**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_HELP\_UPGRADE (3845)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3845);

#### \[VBScript\]

> editor.ExecuteCommandByID 3845
