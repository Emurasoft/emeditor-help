# 키보드 맵 명령

## 요약

사용 가능한 모든 키보드 단축키를 표시합니다.

## 설명

키보드 단축기와 사용 가능한 모든 명령의 리스트를 표시합니다.

## 실행하는 방법

- 기본 메뉴: **도움말** \> **키보드 맵**
- [모든 명령](../tools/all_commands): **도움말** >
**키보드 맵**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_KEYBOARD_MAP (4298)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4298);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4298
```
