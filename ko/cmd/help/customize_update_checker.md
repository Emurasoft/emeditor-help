# 업데이트 확인 사용자 지정 명령

### 요약

> 업데이트 검사기를 사용자 정의합니다.

### 설명

> 얼마나 자주 엠에디터에서 업그레이드를 확인할지 지정할 수 있고, 업데이트가 다운로드되며 저장되는 **업데이트 옵션** 대화 상자를 엽니다.

### 실행하는 방법

- 기본 메뉴: **도움말** \> **업데이트 확인 사용자 지정**
- [모든 명령](../tools/all_commands): **도움말** >
**업데이트 확인 사용자 지정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CUSTOMIZE\_UPDATE\_CHECKER (4482)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4482);

#### \[VBScript\]

> editor.ExecuteCommandByID 4482
