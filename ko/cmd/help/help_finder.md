# 도움말 항목 검색 명령

### 요약

> 엠에디터 도움말 항목을 표시합니다.

### 설명

> 엠에디터 도움말 모듈을 열고, 도움말 항목을 찾아보거나 검색할수 있습니다.

### 실행하는 방법

- 기본 메뉴: **도움말** \> **도움말 항목 검색**
- [모든 명령](../tools/all_commands): **도움말** >
**도움말 항목 검색**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+F1

### 플러그인 명령 ID

- EEID\_HELP\_FINDER (4247)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4247);

#### \[VBScript\]

> editor.ExecuteCommandByID 4247
