# 구매 방법 명령

## 요약

엠에디터 구매 방법에 대한 정보를 표시합니다.

## 설명

엠에디터 도움말에서 구매 방법 섹션을 엽니다.

## 실행하는 방법

- 기본 메뉴:도움말 \>구매 방법
- [모든 명령](../tools/all_commands):도움말 >
구매 방법
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_HELP_REGIST (4248)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4248
```
