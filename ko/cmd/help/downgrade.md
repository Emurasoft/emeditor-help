# 다운그레이드 명령

## 요약

이 제품을 엠에디터 Free로 다운그레이드 합니다.

## 설명

이 제품을 엠에디터 Professional에서 엠에디터 Free로 다운그레이드 합니다.
사용자는 다운그레이드가 실행되기 이전에 확인 메시지를 받게됩니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):도움말 >
다운그레이드
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_HELP_DOWNGRADE (3844)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(3844);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3844
```
