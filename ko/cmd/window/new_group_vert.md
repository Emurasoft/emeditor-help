# 새 세로 그룹 명령

## 요약

새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 중복되지 않도록 하여 세로로 창을 배열합니다.

## 설명

새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 중복되지 않도록 하여 세로로 창을 배열합니다.

## 실행하는 방법

- 기본 메뉴:창 \>새 세로 그룹 명령
- [모든 명령](../tools/all_commands):창 \>그룹 \>새 세로 그룹 명령
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음
