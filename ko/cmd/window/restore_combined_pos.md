# 탭 창 위치 복원 명령

### 요약

> 이전에 저장했던 위치로 탭 창을 복원시킵니다.

### 설명

> 이 명령은 이전에 저장했던 위치로 탭 창을 복원시킵니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **복원** \> **탭 창 위치 복원**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_RESTORE\_COMBINED\_POS (4543)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4543);

#### \[VBScript\]

> editor.ExecuteCommandByID 4543
