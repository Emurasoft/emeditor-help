# 분할 제거 명령

### 요약

> 분할을 제거합니다.

### 설명

> 현재 문서로부터 모든 분할을 제거합니다.

### 실행하는 방법

- 기본 메뉴: **창** \> **분할 제거**
- [모든 명령](../tools/all_commands): **창** \>
**분할** \> **분할 제거**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: SHIFT+F12

### 플러그인 명령 ID

- EEID\_DELETE\_PANES (4284)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4284);

#### \[VBScript\]

> editor.ExecuteCommandByID 4284
