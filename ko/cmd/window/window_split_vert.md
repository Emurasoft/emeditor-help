# 세로 분할 명령

### 요약

> 현재 창을 수직으로 분할합니다.

### 설명

> 이 명령은 현재 창을 두개의 수직 창으로 분할합니다.
> 마우스를 이용하여 분할되는 위치를 제어합니다.
> 클릭하여 고정된 위치에 분할을 설정합니다.
> (필요에따라 방향을 변경할 수 있습니다.)

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **분할** \> **세로 분할**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WINDOW\_SPLIT\_VERT (4300)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4300);

#### \[VBScript\]

> editor.ExecuteCommandByID 4300
