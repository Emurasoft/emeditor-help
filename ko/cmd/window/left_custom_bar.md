# 왼쪽 사용자 지정 바 명령

## 요약

왼쪽으로 사용자 지정 바를 전환합니다.

## 설명

가능한 경우 왼쪽으로 사용자 지정 바를 전환합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **사용자 지정 바** \> **왼쪽**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_FOCUS_LEFT_BAR (4425)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4425);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4425
```
