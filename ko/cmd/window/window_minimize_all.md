# 모두 최소화 명령

## 요약

모든 창을 최소화합니다.

## 설명

열려있는 모든 엠에디터 창을 최소화합니다.

## 실행하는 방법

- 기본 메뉴:창 \>모두 최소화
- [모든 명령](../tools/all_commands):창 \>모두 최소화
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_WINDOW_MINIMIZE_ALL (4244)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4244);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4244
```
