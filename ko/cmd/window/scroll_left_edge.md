# 왼쪽 끝으로 스크롤 명령

## 요약

현재 문서의 왼쪽 끝으로 스크롤 합니다.

## 설명

현재 문서의 왼쪽 끝으로 스크롤 합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):창 \>스크롤 \>왼쪽 끝으로
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SCROLL_LEFT_EDGE (3870)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(3870);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3870
```
