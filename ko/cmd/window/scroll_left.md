# 왼쪽으로 스크롤 명령

### 요약

> 한 문자 왼쪽으로 문서를 스크롤합니다.

### 설명

> 한 문자 왼쪽으로 문서를 스크롤합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **스크롤** \> **왼쪽**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WINDOW\_SCROLL\_LEFT (4474)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4474);

#### \[VBScript\]

> editor.ExecuteCommandByID 4474
