# 활성 창 명령

## 요약

활성 창으로 전환합니다.

## 설명

사용자 지정 바가 표시되었을 때 활성 창으로 전환합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):창 \>창
\>활성 창
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_ACTIVE_PANE (4409)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4409);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4409
```
