# 페이지 위로 스크롤 명령

### 요약

> 문서를 한 페이지 위로 스크롤 합니다

### 설명

> 문서를 한 페이지 위로 스크롤 합니다

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **스크롤** \> **페이지 위로**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SCROLL\_PAGE\_UP (3866)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3866);

#### \[VBScript\]

> editor.ExecuteCommandByID 3866
