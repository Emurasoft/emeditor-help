# 이전 그룹으로 이동 명령

## 요약

현재 문서를 이전 그룹으로 이동합니다.

## 설명

현재 문서를 이전 그룹으로 이동합니다 (하나의 창으로 재결합).

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):창 \>그룹 \>이전 그룹으로 이동
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MOVE_PREV_GROUP (4396)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4396);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4396
```
