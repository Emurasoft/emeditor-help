# 세로 분할 설정/해제 명령

## 요약

창 세로 분할을 설정/해제 합니다.

## 설명

현재 창이 가로로 분할되어 있거나 분할되어 있지 않은 경우, 이 명령은 현재 창을
수직 창으로 분할하고, 즉시 창의 중앙에 분할 위치를 고정합니다.
현재 창이 이미 수직으로 분할되어 있는 경우, 이 명령은 현재 창으로부터 수직 분할을 제거합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):창 \>분할 \>세로 분할 설정/해제
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_WINDOW_SPLIT_VERT_TOGGLE (4386)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4386);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4386
```
