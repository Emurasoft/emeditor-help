# 세로 바둑판식 배열 명령

### 요약

> 겹치지 않도록 창을 세로로 배열합니다.

### 설명

> 결합되지 않은 엠에디터 창들을 겹치지 않도록 세로로 배열합니다.

### 실행하는 방법

- 기본 메뉴: **창** \> **세로 바둑판식 배열**
- [모든 명령](../tools/all_commands): **창** \> **세로 바둑판식 배열**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WINDOW\_TILE\_VERT (4243)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4243);

#### \[VBScript\]

> editor.ExecuteCommandByID 4243
