# 분할 명령

## 요약

실행중인 창을 창들로 분할합니다.

## 설명

이 명령은 현재 창을 네개의 창으로 분할합니다.
마우스를 이용하여 분할되는 위치를 제어합니다.
클릭하여 고정된 위치에 분할을 설정합니다.
(필요에따라 방향을 변경할 수 있습니다.)

## 실행하는 방법

- 기본 메뉴:창 \>분할
- [모든 명령](../tools/all_commands):창 \>분할 \>분할
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+F12

## 플러그인 명령 ID

```
EEID_WINDOW_SPLIT (4240)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4240);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4240
```
