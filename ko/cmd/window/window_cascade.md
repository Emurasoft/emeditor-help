# 계단식 배열 명령

## 요약

겹치도록 창들을 배열합니다.

## 설명

결합되지 않은 창들에 쉽게 접근하도록 엠에디터 창을 겹쳐서 배열합니다.

## 실행하는 방법

- 기본 메뉴: **창** \> **계단식 배열**
- [모든 명령](../tools/all_commands): **창** \> **계단식 배열**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_WINDOW_CASCADE (4241)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4241);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4241
```
