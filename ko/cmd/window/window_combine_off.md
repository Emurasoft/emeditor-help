# 탭 활성화 \- 꺼짐 명령

### 요약

> 모든 창을 분리하기 위해 탭을 비활성화 합니다.

### 설명

> 모든 창을 분리하기 위해 탭을 비활성화 합니다.
> 탭이 비활성화 되었을 때, 열려있는 각 문서들은 별도의 창으로 표시됩니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **탭 활성화** \> **탭 활성화 \- 꺼짐**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_WINDOW\_COMBINE\_OFF (4408)

### 매크로

#### \[JavaScript\]

> editor.EnableTab = false;

#### \[VBScript\]

> editor.EnableTab = False
