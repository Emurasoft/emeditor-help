# 분할 시 세로 스크롤링 동기화 명령

## 요약

분할 된 창의 세로 스크롤을 동기화합니다.

## 설명

분할 된 창의 세로 스크롤을 동기화합니다.

## 실행하는 방법

- 기본 메뉴: **창** \> **분할 시 세로 스크롤링 동기화**
- [모든 명령](../tools/all_commands): **창** \> **분할** \> **분할 시 세로 스크롤링 동기화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SYNC_VERT_SCROLL_SPLIT (4524)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4524);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4524
```
