# 새 최소화 그룹 명령

### 요약

> 새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 최소화합니다.

### 설명

> 새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 최소화합니다.
> 이 명령은 현재 엠에디터 창 밖으로 탭을 드래그하고 확인 메시지가 나타났을 때 **새 그룹** 을 선택한 후, 새로운
> 창을 최소화하는 것과 동일합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \>
**그룹** \> **새 최소화 그룹**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_NEW\_GROUP\_MINIMIZE (4392)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4392);

#### \[VBScript\]

> editor.ExecuteCommandByID 4392
