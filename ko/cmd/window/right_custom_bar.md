# 오른쪽 사용자 지정 바 명령

### 요약

> 오른쪽으로 사용자 지정 바를 전환합니다.

### 설명

> 가능한 경우 오른쪽으로 사용자 지정 바를 전환합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **사용자 지정 바** \> **오른쪽**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FOCUS\_RIGHT\_BAR (4427)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4427);

#### \[VBScript\]

> editor.ExecuteCommandByID 4427
