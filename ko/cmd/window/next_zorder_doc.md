# 마지막으로 엑세스한 문서 명령

### 요약

> 마지막으로 엑세스한 문서로 전환합니다.

### 설명

> 마지막으로 엑세스한 문서로 전환합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **마지막으로 엑세스한 문서**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_NEXT\_ZORDER\_DOC (4544)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4544);

#### \[VBScript\]

> editor.ExecuteCommandByID 4544
