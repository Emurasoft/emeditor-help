# 다음 창 또는 사용자 지정 바 명령

### 요약

> 다음 창 또는 사용자 지정 바로 전환합니다.

### 설명

> 창이 여러 내부 창으로 분할되어 있는 경우 커서를 다음 창으로 이동하거나, 그렇지 않은 경우 사용자 지정 바로 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **창** \> **다음 창 또는 사용자 지정 바**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_NEXT\_PANE\_OR\_BAR (4553)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4553);

#### \[VBScript\]

> editor.ExecuteCommandByID 4553
