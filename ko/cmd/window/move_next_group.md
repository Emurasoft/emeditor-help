# 다음 그룹으로 이동 명령

### 요약

> 현재 문서를 다음 그룹으로 이동합니다.

### 설명

> 현재 문서를 다음 그룹으로 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **그룹** \> **다음 그룹으로 이동**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_MOVE\_NEXT\_GROUP (4397)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4397);

#### \[VBScript\]

> editor.ExecuteCommandByID 4397
