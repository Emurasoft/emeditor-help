# 왼쪽 문서 명령

### 요약

> 탭 표시 줄의 왼쪽 문서로 전환합니다.

### 설명

> 탭 표시 줄에서 현재 열려있는 문서 왼쪽 문서로 전환합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **문서 탐색** \> **왼쪽 문서**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL + SHIFT + F6

### 플러그인 명령 ID

- EEID\_LEFT\_DOC (4548)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4548);

#### \[VBScript\]

> editor.ExecuteCommandByID 4548
