# 그룹 내 모든 오른쪽 닫기 명령

## 요약

현재 그룹 내 모든 오른쪽 문서들을 닫습니다.

## 설명

현재 그룹의 실행되고 있는 문서로부터 오른쪽 탭에 있는 모든 문서들을 닫습니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** > **그룹** \> **그룹 내 모든 오른쪽 닫기 명령**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_GROUP_CLOSE_RIGHT (4390)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4390);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4390
```
