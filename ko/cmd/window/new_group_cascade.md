# 새 계단식 배열 그룹 명령

## 요약

새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 그들이 중복되도록 창을 배열합니다.

## 설명

새 그룹으로 결합된 창으로부터 현재의 문서를 분리하고, 그들이 중복되도록 창을 배열합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):창 \>그룹 \>새 계단식 배열 그룹
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_NEW_GROUP_CASCADE (4393)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4393);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4393
