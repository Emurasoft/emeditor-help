# 다음 이동 명령

## 요약

다음 탭 위치로 현재 문서를 이동합니다.

## 설명

다음 탭 위치로 현재 문서를 이동합니다.
탭 표시 줄에 나타나는 문서의 순서를 변경하는데 이 명령을 사용할수 있습니다.
또는, 탭 표시줄에서 원하는 위치로 탭을 드래그할 수 있습니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **창** \> **문서 탐색** \> **다음 이동**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

EEID_MOVE_NEXT (4382)```
