# 반환 명령

### 요약

> 반환 마크를 표시하거나 숨깁니다.

### 설명

> 반환 마크를 표시하거나 숨깁니다. 기본적으로 엠에디터는 반환 마크 (파란 아래 방향 화살표)를 표시합니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **반환**
- [모든 명령](../tools/all_commands): **보기** \> **표시** >
**반환**
- 도구 모음:
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_RETURN (4532)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4532);

#### \[VBScript\]

> editor.ExecuteCommandByID 4532
