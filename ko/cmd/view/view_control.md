# 제어 문자 명령

## 요약

제어 문자 마크를 표시하거나 숨깁니다.

## 설명

제어 문자 마크를 표시하거나 숨깁니다.

## 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **제어 문자**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \> **제어 문자**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_VIEW_CONTROL (4538)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4538);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4538
```
