# 탭 명령

### 요약

> 탭 마크를 표시하거나 숨깁니다.

### 설명

> 탭 마크를 표시하거나 숨깁니다.
> 기본적으로 엠에디터는 탭 마크 (파란 오른쪽 방향 화살표)를 표시하지 않습니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **탭**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \> **탭**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_TAB (4534)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4534);

#### \[VBScript\]

> editor.ExecuteCommandByID 4534
