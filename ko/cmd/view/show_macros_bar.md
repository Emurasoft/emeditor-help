# 매크로 도구 모음 명령

## 요약

매크로 도구 모음을 표시하거나 숨깁니다.

## 설명

매크로 도구 모음을 표시하거나 숨깁니다.
매크로 도구 모음은 지정된 내 매크로를 표시합니다.

## 실행하는 방법

- 기본 메뉴: **보기** >
**도구 모음** \> **매크로 도구 모음**
- [모든 명령](../tools/all_commands): **보기** >
**도구 모음** \> **매크로 도구 모음**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SHOW_매크로_BAR (4421)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4421);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4421
```
