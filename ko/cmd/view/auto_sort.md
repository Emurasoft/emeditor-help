# 자동 정렬 명령

## 요약

탭을 자동으로 정렬합니다.

## 설명

이 명령이 체크되어 있는 경우, 엠에디터는 탭을 자동으로 정렬합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **보기** \> **탭 정렬 방식** \> **자동 정렬**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_AUTO_SORT (4404)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4404);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4404
```
