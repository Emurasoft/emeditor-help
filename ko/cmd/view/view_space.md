# 공백 명령

### 요약

> 공백 마크를 표시하거나 숨깁니다.

### 설명

> 공백 마크를 표시하거나 숨깁니다.
> 기본적으로 엠에디터는 공백 마크 (파란 밑줄)를 표시하지 않습니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **공백**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \> **공백**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_SPACE (4535)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4535);

#### \[VBScript\]

> editor.ExecuteCommandByID 4535
