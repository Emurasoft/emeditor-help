# 모든 마커 활성화 명령

모든 마커를 활성화합니다.

## 설명

모든 마커를 활성화합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):보기 \>마커 \>모두 활성화
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MARKER_ENABLE_ALL (4585)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4585);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4585
