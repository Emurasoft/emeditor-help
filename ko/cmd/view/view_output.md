# 출력 명령

### 요약

> 출력 바를 표시하거나 숨깁니다.

### 설명

> 출력 바를 표시하거나 숨깁니다.
> [**출력 모음 사용** 체크 박스](../../dlg/tools/properties/index) 가 체크되어 있는
> 상태에서 외부 도구가 시작될 때 출력바가 나타납니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **출력**
- [모든 명령](../tools/all_commands): **보기** >
**출력 모음** >
**출력**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: ALT+2

### 플러그인 명령 ID

- EEID\_VIEW\_OUTPUT (4420)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4420);

#### \[VBScript\]

> editor.ExecuteCommandByID 4420
