# 새로고침 명령

### 요약

> 모든 도구 모음을 새로고침 합니다.

### 설명

> 모든 도구 모음을 새로고침 합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **보기** >
**도구 모음** \> **새로고침**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_REFRESH\_도구 모음S (4422)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4422);

#### \[VBScript\]

> editor.ExecuteCommandByID 4422
