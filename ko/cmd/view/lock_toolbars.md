# 도구 모음 잠금 명령

## 요약

모든 도구 모음들을 잠금니다.

## 설명

도구 모음이 아직 잠겨있지 않다면 잠그거나 이미 잠겨있는 도구 모음의 잠금을 해제합니다.
잠금이 해제 되었을 때는 각 도구 모음의 왼쪽 가장자리의 그립을 드래그하여 주위로 이동시킬 수 있습니다.
잠긴 도구 모음은 왼쪽 가장자리의 그립을 표시하지 않습니다.

## 실행하는 방법

- 기본 메뉴:보기 >
도구 모음 \>도구 모음 잠금
- [모든 명령](../tools/all_commands):보기 >
도구 모음 \>도구 모음 잠금
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_LOCK_도구 모음S (4341)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4341);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4341
```
