# 찾기 도구 모음 명령

## 요약

찾기 도구 모음을 표시하거나 숨깁니다.

## 설명

찾기 도구 모음을 표시하거나 숨깁니다.
찾기 도구 모음은 문서 내에서 문자열을 찾거나 대체하도록 합니다.

## 실행하는 방법

- 기본 메뉴:보기 >
도구 모음 \>찾기 도구 모음
- [모든 명령](../tools/all_commands):보기 >
도구 모음 \>찾기 도구 모음
- 도구 모음:
![](../../images/emeditor12_toggle_find_bar_button.png)
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SHOW_FIND_BAR (4559)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4559);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4559
```
