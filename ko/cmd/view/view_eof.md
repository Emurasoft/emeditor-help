# 파일의 끝 명령

### 요약

> 파일의 끝 마크를 표시하거나 숨깁니다.

### 설명

> 파일의 끝 마크를 표시하거나 숨깁니다.
> 기본적으로 엠에디터는 파일의 끝 마크 (파란 왼쪽 방향 화살표)를 표시합니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **파일의 끝**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \> **파일의 끝**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_EOF (4533)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4533);

#### \[VBScript\]

> editor.ExecuteCommandByID 4533
