# 마커 도구 모음 명령

### 요약

> 마커 도구 모음을 표시하거나 숨깁니다.

### 설명

> 마커 도구 모음을 표시하거나 숨깁니다.
> 마커 도구 모음은 모든 지정된 마커를 나타냅니다.

### 실행하는 방법

- 기본 메뉴: **보기** >
**도구 모음** \> **마커 도구 모음**
- [모든 명령](../tools/all_commands): **보기** >
**도구 모음** \> **마커 도구 모음**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SHOW\_MARKERS\_BAR (4561)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4561);

#### \[VBScript\]

> editor.ExecuteCommandByID 4561
