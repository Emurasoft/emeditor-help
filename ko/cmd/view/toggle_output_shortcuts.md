# 바로가기 키 활성화 명령

## 요약

출력 표시 줄의 바로가기 키를 활성화 또는 비활성화 합니다.

## 설명

출력 표시 줄의 바로가기 키를 활성화 또는 비활성화 합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **보기** >
**출력 모음** \> **바로가기 키 활성화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_TOGGLE_OUTPUT_SHORTCUTS (4549)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4549);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4549
```
