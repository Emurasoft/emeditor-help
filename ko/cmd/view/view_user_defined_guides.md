# 사용자 지정 가이드 명령

### 요약

> 사용자 지정 가이드를 보이거나 숨깁니다.

### 설명

> 사용자 지정 가이드를 보이거나 숨깁니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **들여쓰기 가이드**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \>
**사용자 지정 가이드**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_VIEW\_USER\_DEFINED\_GUIDES (3863)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3863);

#### \[VBScript\]

> editor.ExecuteCommandByID 3863
