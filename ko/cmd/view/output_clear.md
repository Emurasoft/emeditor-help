# 출력 지우기 명령

### 요약

> 출력 바의 내용을 지웁니다.

### 설명

> 출력 바의 내용을 지웁니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **보기** >
**출력 모음** \> **출력 지우기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_OUTPUT\_CLEAR (4452)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4452);

#### \[VBScript\]

> editor.ExecuteCommandByID 4452
