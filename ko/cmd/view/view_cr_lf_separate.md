# 다른 표시 CR 및 LF 명령

## 요약

CR 및 LF를 다른 마크로 표시합니다.

## 설명

CR 및 LF를 다른 마크로 표시합니다.
기본적으로 엠에디터는 CR 과 LF를 같은 마크로 표시합니다.

## 실행하는 방법

- 기본 메뉴:보기 \>표시 \>다른 표시 CR 및 LF
- [모든 명령](../tools/all_commands):보기 \>표시 \>다른 표시 CR 및 LF
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_VIEW_CR_LF_SEPARATE (4537)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4537);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4537
```
