# 플러그 인 도구 모음 명령

### 요약

> 플러그 인 도구 모음을 표시하거나 숨깁니다.

### 설명

> 플러그 인 도구 모음을 표시하거나 숨깁니다.
> 플러그 인 도구 모음을 설치된 모든 플러그인 아이콘을 나타냅니다.

### 실행하는 방법

- 기본 메뉴: **보기** >
**도구 모음** \> **플러그 인 도구 모음**
- [모든 명령](../tools/all_commands): **보기** >
**도구 모음** \> **플러그 인 도구 모음**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SHOW\_PLUGINS\_BAR (4285)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4285);

#### \[VBScript\]

> editor.ExecuteCommandByID 4285
