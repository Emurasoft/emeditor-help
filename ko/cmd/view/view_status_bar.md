# 상태 표시줄 명령

## 요약

상태 표시줄을 표시하거나 숨깁니다.

## 설명

상태 표시줄을 표시하거나 숨깁니다.
상태 표시줄은 엠에디터 창의 아래쪽에 표시됩니다.
상태 표시줄은 현재 구성, 현재 줄, 현재 열, 현재 인코딩, 덮어쓰기 모드 ("OVR") 및
읽기 전용 모드 ("읽기")를 표시합니다.

## 실행하는 방법

- 기본 메뉴:보기 \>상태 표시줄
- [모든 명령](../tools/all_commands):보기 >
상태 표시줄
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_VIEW_STATUS_BAR (4212)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4212);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4212
```
