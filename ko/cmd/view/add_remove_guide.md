# 사용자 지정 가이드 추가/삭제 명령

### 요약

> 커서 위치 (또는 줄자에 표시된 위치)에 사용자 지정 가이드를 추가하거나 삭제합니다.

### 설명

> 커서 위치 (또는 줄자에 표시된 위치)에 사용자 지정 가이드를 추가하거나 삭제합니다.

### 실행하는 방법

- 기본 메뉴: **보기** \> **표시** \> **사용자 지정 가이드 추가/삭제**
- [모든 명령](../tools/all_commands): **보기** \> **표시** \>
**사용자 지정 가이드 추가/삭제**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_ADD\_REMOVE\_GUIDE (3862)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3862);

#### \[VBScript\]

> editor.ExecuteCommandByID 3862
