# 도구 모음 제목 명령

## 요약

도구 모음 제목을 표시합니다.

## 설명

도구 모음 제목을 표시합니다.
도구 모음 제목은 **플러그인**, **도구**, 및 **창** 을 포함합니다.

## 실행하는 방법

- 기본 메뉴: **보기** >
**도구 모음** \> **도구 모음 제목**
- [모든 명령](../tools/all_commands): **보기** >
**도구 모음** \> **도구 모음 제목**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_SHOW_BAR_TITLE (4340)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4340);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4340
```
