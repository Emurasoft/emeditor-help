# 인코딩 정의 명령

## 요약

파일을 열거나 저장할때 사용하는 인코딩을 정의합니다.

## 설명

파일을 열고 저장하는 인코딩을 정의할 수 있는 [**인코딩 정의** \
대화 상자](../../dlg/encodings/index) 를 나타냅니다.

## 실행하는 방법

- 기본 메뉴: **도구** \> **인코딩 정의**
- [모든 명령](all_commands): **도구** >
**인코딩 정의**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_DEFINE_CODE_PAGE (4264)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4264);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4264
```
