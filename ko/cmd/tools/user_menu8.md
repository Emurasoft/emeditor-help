# 사용자 메뉴 (8) 명령

### 요약

> 두 키 입력 바로가기의 최초 단축키로 사용할 사용자 메뉴 (8)을 나타냅니다.

### 설명

> 두 키 입력 바로가기의 최초 단축키로 사용할 사용자 메뉴 (8)을 나타냅니다.
> 사용자 메뉴는 [**메뉴 사용자 지정** 명령](customize_menu) 으로 지정할 수 있습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**사용자 메뉴** \> **사용자 메뉴 (8)**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_USER\_MENU1 (4436)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4436);

#### \[VBScript\]

> editor.ExecuteCommandByID 4436
