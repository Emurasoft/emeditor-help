# 상황에 맞는 메뉴 명령

## 요약

상황에 맞는 메뉴를 나타냅니다.

## 설명

엠에디터 창 안에서 상황에 맞는 메뉴를 표시합니다.
상황에 맞는 메뉴에 보여지는 메뉴 명령들은 [**메뉴 사용자 지정** 명령](customize_menu) 을 통해 사용자 지정할 수 있습니다.
이 메뉴는 오른쪽 마우스 버튼은 클릭하였을 때도 나타납니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**상황에 맞는 메뉴**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CONTEXT_MENU (4301)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4301);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4301
```
