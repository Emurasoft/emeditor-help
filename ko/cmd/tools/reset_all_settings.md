# 모든 설정 초기화 명령

### 요약

> EmEditor의 모든 설정을 초기화하고 EmEditor를 다시 시작합니다.

### 설명

> EmEditor의 모든 설정을 초기화하고 EmEditor를 다시 시작합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**모든 설정 초기화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CUSTOMIZE\_TOOLS (3860)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3860);

#### \[VBScript\]

> editor.ExecuteCommandByID 3860
