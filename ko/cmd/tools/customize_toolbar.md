# 도구 모음 사용자 지정 명령

## 요약

도구 모음 설정 사용자 정의를 합니다.

## 설명

도구 모음 사용자 정의가 가능한 [도구 모음 대화 상자](../../dlg/toolbars/index) 를
표시합니다.

## 실행하는 방법

- 기본 메뉴:도구 \>도구 모음 사용자 지정
- [모든 명령](all_commands):도구 >
도구 모음 사용자 지정
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CUSTOMIZE_도구 모음 (4224)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4224);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4224
```
