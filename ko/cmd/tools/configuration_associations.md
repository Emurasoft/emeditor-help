# 연결 구성 명령

### 요약

> 파일 유형의 구성을 연결합니다.

### 설명

> 어느 엠에디터의 구성이 특정한 파일 유형과 연결되는지 선택할 수 있는
> [**연결 구성** 대화 상자](../../dlg/configuration_associations/index) 를 표시합니다.

### 실행하는 방법

- 기본 메뉴: **도구** \> **연결 구성**
- [모든 명령](all_commands): **도구** \> **연결 구성**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CONFIG\_ASSOCIATE (3847)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(3847);

#### \[VBScript\]

> editor.ExecuteCommandByID 3847
