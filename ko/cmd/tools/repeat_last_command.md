# 지난 명령 다시 실행 명령

## 요약

지난 명령을 다시 실행합니다.

## 설명

마지막으로 선택된 명령을 다시 실행합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** > **지난 명령 다시 실행**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+Shift+G

## 플러그인 명령 ID

```
EEID_REPEAT_LAST_COMMAND (3846)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(3846);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3846
```
