# 트레이 아이콘 사용자 지정 명령

## 요약

트레이 아이콘을 사용자 지정합니다.

## 설명

시스템 트레이 아이콘 설정을 사용자 지정할 수 있는
[트레이 아이콘 사용자 지정 대화 상자](../../dlg/tray/index) 를 표시합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands):도구 >
트레이 아이콘 사용자 지정
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CUSTOMIZE_TRAY (4273)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4273);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4273
```
