# 플러그 인 사용자 지정 명령

### 요약

> 플러그 인을 사용자 정의합니다．

### 설명

> 플러그 인 설정을 사용자 지정 가능한
> [**플러그 인 사용자 지정** \
> 대화 상자](../../dlg/plugins/index) 를 나타냅니다.

### 실행하는 방법

- 기본 메뉴: **도구** \> **플러그 인 사용자 지정**
- [모든 명령](all_commands): **도구** >
**플러그 인 사용자 지정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CUSTOMIZE\_PLUG\_INS (4238)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4238);

#### \[VBScript\]

> editor.ExecuteCommandByID 4238
