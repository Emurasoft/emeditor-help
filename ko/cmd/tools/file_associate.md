# 엠에디터와 연결 명령

## 요약

엠에디터와 파일 확장자를 연결합니다.

## 설명

사용자가 어느 파일 확상자를 엠에디터와 연결할 지 지정할 수 있는
[엠에디터와 연결 대화 상자](../../dlg/file_associate/index) 를 표시합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands):도구 >
엠에디터와 연결
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_FILE_ASSOCIATE (4223)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4223);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4223
```
