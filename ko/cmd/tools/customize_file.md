# 사용자 지정 파일 명령

## 요약

사용자 지정 대화 상자의 파일 탭을 엽니다.

## 설명

응용 프로그램 설정을 정의할 수 있는
[**사용자 지정 대화 상자**](../../dlg/customize/index) 의 파일 탭을 표시합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](all_commands): **도구** >
**사용자 지정** >

**파일**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CUSTOMIZE_FILE (9040)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(9040);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9040
```
