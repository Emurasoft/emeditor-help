# 가져오기 및 보내기 명령

## 요약

파일로부터 설정을 가져오거나 파일로 현재 설정을 내보낼 마법사를 나타냅니다.

## 설명

이 명령은 파일로부터 설정을 가져오거나 파일로 설정을 내보내기를 선택할 수 있는
[가져오기 및 보내기 마법사](../../dlg/import_export/index) 를 표시합니다.

## 실행하는 방법

- 기본 메뉴:도구 >
가져오기 및 보내기
- [모든 명령](all_commands):도구 >
가져오기 및 보내기
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_IMPORT_EXPORT (4380)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4380);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4380
```
