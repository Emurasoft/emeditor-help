# 빠른 실행 명령

## 요약

모든 명령을 검색합니다.

## 설명

이 명령을 선택하면 모든 명령을 검색하고 검색된 명령을 실행할 수 있는
빠른 실행 창을 나타냅니다.

## 실행하는 방법

- 기본 메뉴:도구 \>빠른 실행
- [모든 명령](all_commands):도구 >빠른 실행
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+Q

## 플러그인 명령 ID

```
EEID_SEARCH_ALL_COMMANDS (4608)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4608);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4608
```
