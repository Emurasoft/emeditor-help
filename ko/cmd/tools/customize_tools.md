# 도구 사용자 지정 명령

### 요약

> 외부 도구를 사용자 지정합니다.

### 설명

> 외부 도구를 사용자 지정할 수 있는 [**외부 도구** 대화 상자](../../dlg/tools/index) 를
> 표시합니다.

### 실행하는 방법

- 기본 메뉴: **도구** \> **외부 도구** \> **도구 사용자 지정**
- [모든 명령](all_commands): **도구** >
**도구 사용자 지정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CUSTOMIZE\_TOOLS (4324)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4324);

#### \[VBScript\]

> editor.ExecuteCommandByID 4324
