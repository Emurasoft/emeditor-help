# 메뉴 사용자 지정 명령

## 요약

메뉴를 사용자 정의합니다．

## 설명

엠에디터의 메뉴를 사용자 지정할수 있는
[메뉴 사용자 지정 \
대화 상자](../../dlg/menus/index) 를 나타냅니다.

## 실행하는 방법

- 기본 메뉴:도구 \>메뉴 사용자 지정
- [모든 명령](all_commands):도구 >
메뉴 사용자 지정
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CUSTOMIZE_MENU (4266)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4266);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4266
```
