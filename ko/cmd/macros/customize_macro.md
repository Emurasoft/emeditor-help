# 매크로 사용자 지정 명령

## 요약

매크로 사용자 지정을 합니다.

## 설명

매크로 세팅을 사용자 지정할 수 있는
[**매크로 사용자 지정** 대화 상자](../../dlg/macro_customize/index) 를 표시합니다.

## 실행하는 방법

- 기본 메뉴: **매크로** \> **사용자 지정**
- [모든 명령](../tools/all_commands): **매크로**
\> **사용자 지정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_CUSTOMIZE_MACRO (4350)```
