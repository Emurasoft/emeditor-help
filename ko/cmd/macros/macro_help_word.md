# 매크로 키워드 찾기 명령

## 요약

선택된 매크로 키워드에 대한 도움말 항목을 찾습니다.

## 설명

엠에디터 도움말에 선택된 매크로 키워드에 대한 항목을 찾습니다.

## 실행하는 방법

- 기본 메뉴: **매크로** \> **키워드 찾기**
- [모든 명령](../tools/all_commands): **매크로**
\> **키워드 찾기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: F1

## 플러그인 명령 ID

```
EEID_MACRO_HELP_WORD (4361)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4361);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4361
```
