# 임시 옵션으로 매크로 실행 명령

## 요약

임시 옵션으로 매크로를 실행합니다.

## 설명

이 명령은 매크로를 실행 하기 전에 매크로의 반복 횟수와 검색이 실패하였을 시 중지 여부를 설정할 수 있는
[매크로 임시 옵션 대화 상자](../../dlg/macro_temp_options/index) 를 표시합니다.
이 명령은 [매크로 실행 명령](quick_macro_run) 과 동일하게 작용합니다.

## 실행하는 방법

- 기본 메뉴:매크로 \>임시 옵션으로 매크로 실행
- [모든 명령](../tools/all_commands):매크로
\>임시 옵션으로 매크로 실행
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_MACRO_RUN_옵션 (4364)```

## 매크로

Not available.
