# 이 매크로 선택 명령

### 요약

> 열려져 있는 파일을 현재 매크로로 설정합니다.

### 설명

> 열려져 있는 파일을 현재 매크로로 설정합니다.
> [**매크로 실행** 명령](quick_macro_run) 메뉴에
> 파일 이름과 이 파일이 선택된 것을 나타낼 것입니다.

### 실행하는 방법

- 기본 메뉴: **매크로** \> **이 파일 선택**
- [모든 명령](../tools/all_commands): **매크로**
\> **이 파일 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_MACRO\_SELECT\_THIS (4349)

### 매크로

> Not available.
