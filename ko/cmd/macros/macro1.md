# 매크로 리스트 명령

## 요약

내 매크로 목록에서 지정된 매크로를 실행합니다 (여러 항목).

## 설명

이 명령은 여러 항목으로 구성되어 있습니다.
이 명령은 내 매크로 디렉토리에 등록된 특정 매크로를 실행합니다.
[매크로 사용자 지정 명령](customize_macro) 의 내 매크로에서 사용할 매크로 파일을
설정할 수 있습니다.

## 실행하는 방법

- 기본 메뉴:매크로 \> (매크로 이름 선택)
- [모든 명령](../tools/all_commands):매크로
\> (매크로 이름 선택)
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
From EEID_MACRO1 through EEID_MACRO1 + 1023 (From 9216 through 9216 + 1023)```

## 매크로

Not available.
