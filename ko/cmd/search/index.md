# 검색 범주

|     |     |
| --- | --- |
| **[찾기 명령](edit_find)** | 문자열을 찾습니다. |
| **[바꾸기 명령](edit_replace)** | 문자열을 대체합니다. |
| **[다음 명령](edit_repeat)** | 다음 항목을 찾습니다. |
| **[이전 명령](edit_repeat_back)** | 이전 항목을 찾습니다. |
| **[다음 단어 찾기 명령](find_next_word)** | 현재 단어의 다음 항목을 찾습니다. |
| **[이전 단어 찾기 명령](find_prev_word)** | 현재 단어의 이전 항목을 찾습니다. |
| **[다음 항목 추가 명령](add_next_occurrence)** | 선택에 지정된 텍스트의 다음 항목을 추가합니다. |
| **[다음으로 다음인 항목 추가 명령](add_next2_occurrence)** | 선택에 지정된 텍스트의 다음으로 다음인 항목을 추가합니다. |
| **[모든 항목 선택 명령](select_all_occurrences)** | 지정된 텍스트의 모든 항목을 선택합니다. |
| [**다음 바꾸기 명령**](replace_next) | 이전에 검색된 항목의 다음 항목을 찾아 대체합니다. |
| **[찾을 단어 설정 명령](retrieve_find_text)** | 쿼리 문자열로 현재 단어를 설정합니다. |
| [**바꿀 단어 설정 명령**](retrieve_replace_text) | 현재 단어를 바꿀 문자열로 설정합니다. |
| **[찾은 강조 표시 삭제 명령](erase_find_hilite)** | 검색어의 강조 표시를 삭제합니다. |
| **[다음 유니코드 찾기 명령](find_next_unicode)** | 저장할 인코딩으로 변환 할 수 없는 다음 유니코드 문자를 찾습니다. |
| **[이전 유니코드 찾기 명령](find_prev_unicode)** | 저장할 인코딩으로 변환 할 수 없는 이전 유니코드 문자를 찾습니다. |
| **[유니코드 강조 표시 삭제 명령](erase_unicode_hilite)** | 저장할 인코딩으로 변환 할 수 없는 유니코드 문자의 강조 표시를 삭제합니다. |
| **[파일 내 찾기 명령](grep)** | 일치하는 문자열을 여러 파일내에서 찾습니다. |
| **[파일 내 바꾸기 명령](replace_in_files)** | 여러 파일 내에서 텍스트를 대체합니다. |
| **[찾기 도구 모음으로 포커스 명령](focus_find_bar)** | 찾기 도구 모음으로 포커스를 설정합니다. |
| **[찾기 도구 모음 닫기 명령](close_find_bar)** | 찾기 도구 모음을 닫습니다. |
| **[찾기 정규식 검색（찾기 도구 모음）명령](findbar_browse_exp_f)** | 찾기 도구 모음에서 검색에 사용된 문자열을 정규 표현식 또는 이스케이프 시퀀스로 탐색합니다. |
| **[바꾸기 정규식 검색（찾기 도구 모음）명령](findbar_browse_exp_r)** | 대체된 문자열을 정규 표현식 또는 이스케이프 시퀀스로 탐색합니다. |
| **[이전 찾기（찾기 도구 모음）명령](findbar_find_prev)** | 찾기 도구 모음을 이용하여 이전 항목을 찾습니다. |
| **[다음 찾기（찾기 도구 모음）명령](findbar_find_next)** | 찾기 도구 모음을 이용하여 다음 항목을 찾습니다. |
| **[증분 검색（찾기 도구 모음）명령](findbar_incremental)** | 찾기 도구 모음에서 증분 검색 버튼을 설정/해제 합니다. |
| **[열린 모든 문서 검색（찾기 도구 모음）명령](findbar_open_doc)** | 찾기 도구 모음에서 열린 모든 문서 검색 버튼을 설정/해제 합니다. |
| **[대／소문자 구분（찾기 도구 모음）명령](findbar_case)** | 찾기 도구 모음에서 대/소문자 구분 버튼을 설정/해제 합니다. |
| **[이스케이프 시퀀스 사용（찾기 도구 모음）명령](findbar_escape)** | 찾기 도구 모음에서 이스케이프 시퀀스 사용 버튼을 설정/해제 합니다. |
| **[정규식 사용（찾기 도구 모음）명령](findbar_reg_exp)** | 찾기 도구 모음에서 정규식 사용 버튼을 설정/해제 합니다. |
| **[전체 단어 일치（찾기 도구 모음）명령](findbar_only_word)** | 찾기 도구 모음에서 전체 단어 일치 버튼을 설정/해제 합니다. |
| **[문서의 처음／끝으로 이동（찾기 도구 모음）명령](findbar_around)** | 찾기 도구 모음에서 문서의 처음/끝으로 이동 버튼을 설정/해제 합니다. |
| **[개수 일치（찾기 도구 모음）명령](findbar_count)** | 찾기 도구 모음에서 개수 일치 버튼을 설정/해제 합니다. |
| **[바꾸기 >\> （찾기 도구 모음）명령](findbar_replace_dlg)** | 찾기 도구 모음에서 바꾸기 >\> 버튼을 설정/해제 합니다. |
| **[다음 바꾸기（찾기 도구 모음）명령](findbar_replace_next)** | 찾기 도구 모음을 이용하여 다음 일치하는 문자열을 대체합니다. |
| **[모두 바꾸기（찾기 도구 모음）명령](findbar_replace_all)** | 찾기 도구 모음을 이용하여 일치하는 모든 문자열을 대체합니다. |
| **[선택 내에서만 (찾기 도구 모음）명령](findbar_sel_only)** | 찾기 도구 모음에서 선택 내에서만 버튼을 설정/해제 합니다. |
