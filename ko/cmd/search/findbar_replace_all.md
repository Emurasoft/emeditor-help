# 모두 바꾸기（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음을 이용하여 일치하는 모든 문자열을 대체합니다.

### 설명

> 찾기 도구 모음을 이용하여 일치하는 모든 문자열을 대체합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **모두 바꾸기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_REPLACE\_ALL (4582)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4582);

#### \[VBScript\]

> editor.ExecuteCommandByID 4582
