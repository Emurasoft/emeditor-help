# 찾기 도구 모음으로 포커스 명령

## 요약

찾기 도구 모음으로 포커스를 설정합니다.

## 설명

찾기 도구 모음으로 포커스를 설정합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음으로 포커스**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+D

## 플러그인 명령 ID

```
EEID_FOCUS_FIND_BAR (4565)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4565);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4565
```
