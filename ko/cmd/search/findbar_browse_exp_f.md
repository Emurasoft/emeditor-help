# 찾기 정규식 검색（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음에서 검색에 사용된 문자열을 정규 표현식 또는 이스케이프 시퀀스로 탐색합니다.

### 설명

> 찾기 도구 모음에서 검색에 사용된 문자열을 정규 표현식 또는 이스케이프 시퀀스로 탐색합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **찾기 정규식 검색**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_BROWSE\_EXP\_F (4567)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4567);

#### \[VBScript\]

> editor.ExecuteCommandByID 4567
