# 다음으로 다음인 항목 추가 명령

### 요약

> 선택에 지정된 텍스트의 다음으로 다음인 항목을 추가합니다.

### 설명

> 선택에 지정된 텍스트의 다음으로 다음인 항목을 추가합니다.

### 실행하는 방법

- 기본 메뉴: **검색** \> **다음으로 다음인 항목 추가**
- [모든 명령](../tools/all_commands): **검색** > **다음으로 다음인 항목 추가**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_ADD\_NEXT2\_OCCURRENCE (4605)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4605);

#### \[VBScript\]

> editor.ExecuteCommandByID 4605
