# 바꾸기 명령

### 요약

> 문자열을 대체합니다.

### 설명

> 지정된 문자열을 또다른 지정된 문자열로 대체합니다. 이 명령은 [**바꾸기** 대화 상자](../../dlg/replace/index)
> 를 표시합니다. 찾을 문자열, 대체할 문자열, 그리고 다른 옵션들을 지정하면 지정된 문자열로 바꾸기를 시작합니다.

### 실행하는 방법

- 기본 메뉴: **검색** \> **바꾸기**
- [모든 명령](../tools/all_commands): **검색** \> **바꾸기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+H

### 플러그인 명령 ID

- EEID\_EDIT\_REPLACE (4201)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4201);

#### \[VBScript\]

> editor.ExecuteCommandByID 4201
