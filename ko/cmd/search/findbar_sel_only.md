# 선택 내에서만 (찾기 도구 모음）명령

### 요약

> 찾기 도구 모음에서 선택 내에서만 버튼을 설정/해제 합니다.

### 설명

> 찾기 도구 모음에서 선택 내에서만 버튼을 설정/해제 합니다.
> 이 명령이 실행될 때, **찾기** 기능을 사용하면 선택된 영역 안에서만 검색합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **선택 내에서만**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_SEL\_ONLY (4580)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4580);

#### \[VBScript\]

> editor.ExecuteCommandByID 4580
