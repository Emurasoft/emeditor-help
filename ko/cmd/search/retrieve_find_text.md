# 찾을 단어 설정 명령

## 요약

쿼리 문자열로 현재 단어를 설정합니다.

## 설명

[**다음** 명령](edit_repeat) 을 위해 현재 커서 위치의 단어를 쿼리 문자열로 설정합니다.
이 명령을 실행 한 후에 [**다음** 명령](edit_repeat) 을 선택하면, 즉시 이 명령으로 지정된 단어의 다음
항목을 찾을 수 있습니다. [**찾기** 대화 상자](../../dlg/find/index) 에 **\> 버튼** 을 클릭하였을 때 보이는 메뉴에 커서에 단어가 체크 되지 않았다면
이 명령을 통해 지정된 단어를 기본값으로 나타냅니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾을 단어 설정**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_RETRIEVE_FIND_TEXT (4325)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4325);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4325
```
