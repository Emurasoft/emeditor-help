# 문서의 처음／끝으로 이동（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음에서 문서의 처음/끝으로 이동 버튼을 설정/해제 합니다.

### 설명

> 찾기 도구 모음에서 문서의 처음/끝으로 이동 버튼을 설정/해제 합니다.
> 이 명령이 설정되었을 때, 엠에디터는 **다음** 또는 **이전** 명령 중에 문서의 마지막이나 처음에 도달 할 때
> 검색을 계속하기 위해 처음이나 끝으로 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **문서의 처음／끝으로 이동**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_AROUND (4577)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4577);

#### \[VBScript\]

> editor.ExecuteCommandByID 4577
