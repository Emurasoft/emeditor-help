# 증분 검색（찾기 도구 모음）명령

## 요약

찾기 도구 모음에서 증분 검색 버튼을 설정/해제 합니다.

## 설명

찾기 도구 모음에서 증분 검색 버튼을 설정/해제 합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **증분 검색**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_FINDBAR_INCREMENTAL (4571)```

## 매크로

## \[JavaScript\]

```
editor.ExecuteCommandByID(4571);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4571
```
