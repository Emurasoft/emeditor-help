# 모든 항목 선택 명령

### 요약

> 지정된 텍스트의 모든 항목을 선택합니다.

### 설명

> 현 문서에 지정된 텍스트의 모든 항목을 선택합니다.

### 실행하는 방법

- 기본 메뉴: **검색** \> **모든 항목 선택**
- [모든 명령](../tools/all_commands): **검색** > **모든 항목 선택**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+Shift+A

### 플러그인 명령 ID

- EEID\_SELECT\_ALL\_OCCURRENCES (4603)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4603);

#### \[VBScript\]

> editor.ExecuteCommandByID 4603
