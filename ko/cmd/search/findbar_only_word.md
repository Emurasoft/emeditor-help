# 전체 단어 일치（찾기 도구 모음）명령

## 요약

찾기 도구 모음에서 전체 단어 일치 버튼을 설정/해제 합니다.

## 설명

찾기 도구 모음에서 전체 단어 일치 버튼을 설정/해제 합니다.
이 명령이 설정되었을 때, 일치하는 위치의 전체 단어가 검색 용어와 일치하는 경우에만 검색 결과를 반환합니다.
(예를 들어, "searches"은 "search"에 대하여 일치된 것으로 반환되지 않습니다.)

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):검색 \>찾기 도구 모음 \>전체 단어 일치
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_FINDBAR_ONLY_WORD (4576)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4576);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4576
```
