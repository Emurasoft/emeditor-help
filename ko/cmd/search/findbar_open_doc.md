# 열린 모든 문서 검색（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음에서 열린 모든 문서 검색 버튼을 설정/해제 합니다.

### 설명

> 찾기 도구 모음에서 열린 모든 문서 검색 버튼을 설정/해제 합니다.
> **찾기** 명령을 이용하여 현재 창의 모든 열린 문서를 검색합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **열린 모든 문서 검색**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_OPEN\_DOC (4572)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4572);

#### \[VBScript\]

> editor.ExecuteCommandByID 4572
