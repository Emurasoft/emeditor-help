# 다음 바꾸기 명령

### 요약

> 다음 항목을 찾아 대체합니다.

### 설명

> 같은 옵션으로 이전에 검색된 항목의 다음 항목을 찾아 대체합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **다음 바꾸기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_REPLACE\_NEXT (4447)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4447);

#### \[VBScript\]

> editor.ExecuteCommandByID 4447
