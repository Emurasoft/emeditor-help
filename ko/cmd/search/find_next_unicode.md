# 다음 유니코드 찾기 명령

## 요약

저장할 인코딩으로 변환 할 수 없는 다음 유니코드 문자를 찾습니다.

## 설명

문자를 선택하여 저장할 인코딩으로 변환 할 수 없는 다음 유니코드 문자를 찾습니다.
변환 할 수 없는 유니코드 문자가 강조 표시되어있지 않은 경우, 이 명령은 그 문자를 강조 표시합니다.
강조 표시의 색상은 현재 구성 속성 대화 상자 [디스플레이 탭](../../dlg/properties/display/index) 의
부분 지정 리스트 박스 아래강조 표시 (8) 을 통해 지정할 수 있습니다.

파일을 저장하는 데 사용되는 인코딩은 다음과 같이 결정됩니다:

"이 문서는 선택된 인코딩으로 문서를 저장할 경우 손실될 수 있는 문자를 포함합니다."는 확인 메시지가 나타난 경우에,
유니코드 정보를 유지하려면 아래의 취소를 클릭 하고 다른 이름으로 저장을 선택 한 후 인코딩 드롭 다운 리스트 박스로 부터
유니코드 옵션을 선택합니다.
"계속하시겠습니까?"라는 메시지가 나타나고, 파일은 파일을 저장하려고 할 때 지정한 인코딩을 사용하여 저장됩니다.

[유니코드 강조 표시 삭제 명령](erase_unicode_hilite) 이 선택된 후에 파일을 저장하는데
사용된 인코딩은 상태 표시줄에 표시된 인코딩에 해당됩니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands):검색 \>다음 유니코드 찾기
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: F9

## 플러그인 명령 ID

```
EEID_FIND_NEXT_UNICODE (4375)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4375);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4375
```
