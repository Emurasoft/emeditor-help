# 이전 찾기（찾기 도구 모음）명령

### 요약

> 찾기 도구 모음을 이용하여 이전 항목을 찾습니다.

### 설명

> 찾기 도구 모음을 이용하여 이전 항목을 찾습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **이전 찾기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_FINDBAR\_FIND\_PREV (4569)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4569);

#### \[VBScript\]

> editor.ExecuteCommandByID 4569
