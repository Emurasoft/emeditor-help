# 찾기 도구 모음 닫기 명령

### 요약

> 찾기 도구 모음을 닫습니다.

### 설명

> 찾기 도구 모음을 닫습니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **검색** \> **찾기 도구 모음** \> **찾기 도구 모음 닫기**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_CLOSE\_FIND\_BAR (4584)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4584);

#### \[VBScript\]

> editor.ExecuteCommandByID 4584
