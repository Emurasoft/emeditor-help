# 다음 변경된 줄 명령

### 요약

> 다음으로 변경된 줄로 이동합니다.

### 설명

> 비교 모드일 때 다음으로 변경된 줄로 이동합니다.

### 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **비교** \> **다음 변경된 줄**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+F7

### 플러그인 명령 ID

- EEID\_COMPARE\_NEXT\_LINE (4557)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4557);

#### \[VBScript\]

> editor.ExecuteCommandByID 4557
