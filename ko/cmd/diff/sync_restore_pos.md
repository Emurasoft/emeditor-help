# 완료 후 창 위치 복원 명령

### 요약

> 비교 또는 동기화가 완료되면 창 위치를 복원시킵니다.

### 설명

> 비교 또는 동기화가 취소되거나 완료되었을 때 창/탭의 위치를 원래대로 복원시킵니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **완료 후 창 위치 복원**
- [모든 명령](../tools/all_commands): **비교** \> **완료 후 창 위치 복원**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SYNC\_RESTORE\_POS (4546)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4546);

#### \[VBScript\]

> editor.ExecuteCommandByID 4546
