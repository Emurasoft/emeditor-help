# 인코딩 무시 명령

## 요약

문서를 비교할 때 인코딩을 무시합니다.

## 설명

문서를 비교할 때 인코딩을 무시합니다.

## 실행하는 방법

- 기본 메뉴:비교 \>인코딩 무시
- [모든 명령](../tools/all_commands):비교 \>인코딩 무시
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_IGNORE_ENCODING (4504)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4504);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4504
```
