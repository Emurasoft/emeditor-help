# 이전 변경 내용 명령

### 요약

> 이전 변경 사항으로 이동합니다.

### 설명

> 비교 모드일 때 이전의 다른 사항으로 이동합니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **이전 변경 내용**
- [모든 명령](../tools/all_commands): **비교** \> **이전 변경 내용**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: SHIFT+F7

### 플러그인 명령 ID

- EEID\_COMPARE\_PREV (4509)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4509);

#### \[VBScript\]

> editor.ExecuteCommandByID 4509
