# 반환 무시 명령

### 요약

> 문서를 비교할 때 각 줄의 반환을 무시합니다.

### 설명

> 문서를 비교할 때 각 줄의 반환을 무시합니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **반환 무시**
- [모든 명령](../tools/all_commands): **비교** \> **반환 무시**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_IGNORE\_RETURNS (4502)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4502);

#### \[VBScript\]

> editor.ExecuteCommandByID 4502
