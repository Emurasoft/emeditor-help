# 세로 스크롤링 동기화 명령

### 요약

> 세로 스크롤링을 동기화 합니다.

### 설명

> 비교되는 두 문서의 세로 스크롤링을 동기화 합니다.

### 실행하는 방법

- 기본 메뉴: **비교** \> **세로 스크롤링 동기화**
- [모든 명령](../tools/all_commands): **비교** \> **세로 스크롤링 동기화**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

### 플러그인 명령 ID

- EEID\_SYNC\_VERT (4505)

### 매크로

#### \[JavaScript\]

> editor.ExecuteCommandByID(4505);

#### \[VBScript\]

> editor.ExecuteCommandByID 4505
