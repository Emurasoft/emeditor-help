# 이전 변경된 줄 명령

## 요약

이전에 변경된 줄로 이동합니다.

## 설명

비교 모드일 때 이전에 변경된 줄로 이동합니다.

## 실행하는 방법

- 기본 메뉴: 없음
- [모든 명령](../tools/all_commands): **비교** \> **이전 변경된 줄**
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: CTRL+SHIFT+F7

## 플러그인 명령 ID

```
EEID_COMPARE_PREV_LINE (4556)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4556);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4556
```
