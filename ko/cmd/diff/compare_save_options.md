# 옵션 저장 명령

## 요약

비교 또는 스크롤을 동기화하는 옵션을 저장합니다.

## 설명

현재 비교하거나 동기화하는 세션에 지정한 옵션을 저장합니다.

## 실행하는 방법

- 기본 메뉴:비교 \>옵션 저장
- [모든 명령](../tools/all_commands):비교 \>옵션 저장
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_COMPARE_SAVE_옵션 (4508)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4508);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4508
```
