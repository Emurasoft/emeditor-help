# 옵션과 비교 명령

## 요약

파일 비교 마법사에서 지정된 옵션으로 최근에 본 두 문서를 비교합니다.

## 설명

파일 비교 마법사에서 지정된 옵션으로 최근에 본 두 문서를 비교합니다.

## 실행하는 방법

- 기본 메뉴:비교 \>옵션과 비교
- [모든 명령](../tools/all_commands):비교 \>옵션과 비교
- 도구 모음: 없음
- 상태 표시줄: 없음
- 기본 바로 가기 키: 없음

## 플러그인 명령 ID

```
EEID_COMPARE_옵션 (4493)```

## 매크로

### \[JavaScript\]

```
editor.ExecuteCommandByID(4493);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4493
```
