# 새로운 플러그인

다음의 플러그인은 버전 7에서 새로 소개되었습니다. 프로젝트 플러그인과 워드 완성 플러그인은사용자 지정 도구 모음 기능을 이용할 수 있습니다.

-
찾기 모음 플러그인 \- 도구 모음에서 부터 증분 찾기를 포함한 검색을 할 수 있습니다.
- [HTML바 플러그인](../howto/plugin/plugin_htmlbar) \- 자주 사용된 HTML 태그와 다른 요소들을 삽입할 수 있도록 도와주는 버튼으로 구성된 도구 모음을 디스플레이 합니다..
- [프로젝트 플러그인](../howto/plugin/plugin_projects) \- 폴더 트리를 디스플레이 하며 폴더트리로부터의 파일을 열 수 있습니다.
- [단어 완성 플러그인](../howto/plugin/plugin_wordcomplete) \- 엠에디터에서 이전에 사용된 단어와 강조표시된 단어로 구성된 목록을 디스플레이 합니다.
