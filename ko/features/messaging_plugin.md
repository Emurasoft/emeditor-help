# 메시지

간단한 인트라넷 통신 기능이 추가되었습니다. 짧은 텍스트를 선택한 후 **플러그인** 도구모음이나 메뉴의
**선택된 텍스트 보내기** 플러그인으로 **보내는곳** 대화상자가 나타나게 한 다음, 텍스트
메시지를 보낼 컴퓨터를 지정합니다. 컴퓨터를 선택하거나 **모든 컴퓨터로 보내기** 체크 상자를 이용하여 선택된 텍스트를
지정된 컴퓨터(들)로 보냅니다.

텍스트를 받기 위해서는 텍스트를 받고자 하는 컴퓨터에 엠에디터가 설치되어 있고, 엠에디터 트레이 아이콘이 실행되고 있어야 하며 데스크톱이
표시되어 있어야만 합니다. 텍스트를 받을 때, 말풍선이 나타납니다. 말풍선을 클릭하거나 마우스 오른쪽 버튼으로 트레이 아이콘을 클릭,
**마지막 수신된 메시지** 명령을 선택하면 새로운 엠에디터 창이 받은 메시지와 함께 열립니다.

보내는 텍스트의 길이에는 제한이 있으며, 윈도우 8의
경우, 말풍선 대신 **토스트** 가 나타납니다. 플러그인은 라이브러리에서 찾으실 수 있습니다.
