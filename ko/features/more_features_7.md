# 더 많은 기능 추가

버전 7의 더 많은 기능을 소개합니다:

- 가상 공간 모드.
- 탭 상의 아이콘만 읽기.
- 각 탭 혹은 확성탭에 버튼 닫기.
- 두번 키 입력이 사용가능한 사용자 정의된 메뉴.
- **부드러운 스크롤링**
- 세번 클릭으로 전체 문서줄 선택하기.
- 점프 대화 상자의 **열** 텍스트 상자.
- **바꾼 내용 저장 ...** 메시지 박스에 있는 **모두에 적용** 체크박스.
- 문자 공간 조정 능력.
- 윤곽 바를 위한 인코딩 선택 능력.
- 자동 저장/작업 공간으로 로드 옵션.
- 오른쪽 혹은 모든 오른쪽 강조표시 옵션.
- 대화상자 열기로 여러 파일을 동시에 열기.
- 색깔 설정을 위한 **테마** 기능, 그리고 색 미리보기 기능.
- **투명** 옵션은 텍스트 그리고 배경 색에 추가되었습니다.
- 자동 업데이트로 인해서 엠에디터가 작동을 중지하거나 윈도우가 재시작을 강요할 때, 엠에디터는 대화상자 표시 없이
파일 이름이 "AutoSave-" 로 시작하는 수정된 파일을 자동으로 저장합니다.
또한, 엠에디터는 재시작 시 자동적으로 복구된 파일을 표시하는 메시지 박스를 확인합니다.
- 필요시 파일이 KB, MB, 혹은 GB로 표시되어 파일이 열릴때, 파일 크기는 상태표시줄에 표시됩니다.
- **찾기** 드롭다운 목록의 검색 기록에서 항목을 지우기 위해서 Alt + Delete 를 입력합니다.
- **코드 조각** 을 이용해서 이제 여러 문자열을 확인할 수 있습니다.
- **#포함** 플러그인은 이제 여러 문자열을 허용합니다.
- 이벤트 구동 매크로는 이제 정의될 수 있습니다.
- 찾기/바꾸기 대화의 오른쪽 하단 코너에 있는 테두리를 끌고 크기조정을 함으로써 다중선이 입력되어질 수 있습니다.
- **줄 들여쓰기** 는 지정될 수 있습니다.
- 윈도우 비스타에서 **열기/다른 이름으로 저장** 대화 상자는 윈도우 비스타 디자인과
쉽게 파일의 이전 버전으로 액세스 할 수 있습니다. 이것은 윈도우 비스타의 기능 중 하나입니다.
