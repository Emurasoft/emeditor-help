# 수직 선택 편집

수직 선택 편집 기능으로 비슷한 문자열을 정규식을 사용하지 않고 바꾸실 수 있습니다.

예를 들어, 다음의 문자가 파일에 포함하고 있을 때:

name1

name2

name3

name4

name1 부터 name4 까지의 소문자 n을 대문자로 바꾸고 싶다면, name1의 왼쪽을 클릭한 후, 모든 n들을 ALT키를 누른 채 수직으로 선택해 줍니다.(혹은 CTRL + SHIFT + F8 를 누르고 오른쪽 화살표시를 한번 누른 후 내림키를 네번 누르십시오.)

name1

name2

name3

name4

모든 n들이 선택된 동안 N 을 입력해 주십시오.

Name1

Name2

Name3

Name4

모든 n 이 N 으로 바뀐것을 볼 수 있습니다.
