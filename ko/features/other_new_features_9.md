# 다른 새로운 기능

버전 9에 포함된 다른 기능 소개:

- #포함하거나 라벨 뒤에서 들여쓰기: 줄은 이제 무시되어질 수 있습니다.
- 가로/세로 줄은 이제 텍스트 밑에 그어질 수 있으며, 깜박거림은 이제 완전히 사라졌습니다.
- 경고 대화창이 나타난 후에 열리는 동안 무효한 문자가 검색되어 졌을 때, 커서는 이제 무효한 문자 위치로 점프합니다.
- 자동 검색 인코딩 대화 상자에 UTF-16 입력이 추가되었습니다.
- 바꾸기 형식 "(?n:true\_expression:false\_expression)" 이 정규식 바꾸기 형식에 추가되었습니다.
- 파일 대화 상자에 있는 파일 내 찾기는 이제 모덜리스 입니다.
- 증분 검색은 찾기와 바꾸기 대화 상자에서 지원됩니다.
- 찾기와 바꾸기 대화 상자의 단일 그리고 복수 텍스트 상자 사이에 = 버튼은 입력/취소에 추가되었습니다.
- IME 맥락과 관련괸 다시 변환은 이제 지원됩니다.
- 커서 크기와 색은 이제 사용자 지정됩니다.
- 사용자 지정 바의 텍스트 색과 배경 색은 지정될 수 있습니다.
