# 플러그인

엠에디터의 기능은 사용자에 의해 생성된 플러그인 지원을 통합해서 확장되어왔습니다.
예를 들면, 영어를 사용할 때 맞춤법 플러그인을 사용할 수 있고, HTML/XML 문자 참조 혹은 유니버설 문자 이름에 의해 인코딩된
선택된 단어를 디코딩 하기 위해 유니코드 디코딩 플러그 인을 사용할 수 있습니다.
플러그인은 동적 연결 라이브러리 (DLL) 파일로 쓰여졌으며, '플러그인'폴더에 저장되어 있습니다.
이는 엠에디터가 시작될 때 자동적으로 생성됩니다.

개인 플러그인의 생성을 위한 더 자세한 정보를 위해서는 여기를 참조해 주시기 바랍니다:
