# 유지 관리 키 입력 대화 상자

이 대화 상자는 [**EmEditor 소개** 대화 상자](../about/index) 에서
**유지 관리 키** 입력 버튼을 클릭하면 나타납니다.

### 유지 관리 키 텍스트 박스

EmEditor Professional의 현재 버전에 대한 유지 관리 키를 클립보드에 복사하고,
이 텍스트 박스를 선택한 후, **CTRL + V** 를 누르거나 상황에 맞는 메뉴를 표시하기 위해 오른쪽 마우스를 클릭하여
**붙여넣기** 를 선택해주십시오. EmEditor Professional 유지 관리 플랜을 구매하시면 유지 관리 키가 이메일을 통해
보내지거나 구매 페이지에 표시됩니다. EmEditor Professional 라이센스가 필요하며, 유지 관리 플랜을 계속하기 위해
유지 관리 키를 추가하기 이전에 EmEditor Professional의 등록 키가 이미 EmEditor에 입력되어 있어야 하는것을
기억하시기 바랍니다.
