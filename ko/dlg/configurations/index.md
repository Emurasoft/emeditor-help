# 구성 정의 대화 상자

이 대화 상자는 [**구성 정의** 명령](../../cmd/tools/config) 이 선택되었을 때 나타납니다.
이 곳에서 구성을 정의할 수 있습니다.

## 구성 리스트 박스

현재 정의된 구성 목록입니다.

## 선택 버튼

선택된 항목을 형재 구성으로 설정하고 대화 상자를 닫습니다.

## 닫기 버튼

대화 상자를 닫으려면 이 버튼을 클릭합니다.

## 새로 만들기 버튼

목록에 새로운 항목을 추가하려면 이 버튼을 클릭합니다.
이 버튼을 클릭할 때 [**새로 만들기** 대화 상자](new_config/index) 가 나타납니다.

## 삭제 버튼

목록으로부터 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

## 복사 버튼

선택된 항목의 복사본을 만듭니다.

## 모두 선택 버튼

전체 목록을 선택합니다.

## 속성 버튼

목록에서 선택된 구성 속성을 나타내거나 편집합니다.

## 기본값으로 설정 버튼

선택된 항목을 기본 구성으로 설정합니다. 엠에디터가 시작될 때 또는
[**새 텍스트** 명령](../../cmd/file/file_new) 이 선택될 때 기본 구성으로 선택됩니다.

## 연결 버튼

[**구성 연결** 대화 상자](../configuration_associations/index) 로 가려면
이 버튼을 선택합니다.

## 모두 초기화 버튼

모든 항목을 기본값으로 초기화하기 위해서 이 버튼을 클릭합니다.
