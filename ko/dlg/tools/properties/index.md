# 외부 도구 속성 대화 상자

이 대화 상자는 **외부 도구** 대화 상자에서 **속성** 버튼이 선택되면 나타납니다.

## 

### 제목 텍스트 박스

**도구** 메뉴의 **외부 도구** 하위 메뉴에 나타날 도구 또는 명령의 이름입니다.

### 명령 텍스트 박스

실행하려는 .exe, .com, .bat, .cmd, 또는 기타 파일로 경로를 지정합니다.

### 인수 텍스트 박스

도구가 실행될 때 도구에 전달되는 인수를 지정합니다.
미리 정의된 인수 목록에서 선택하려면 **화살표** ( **>**) 버튼을 사용합니다.
%WinDir%와 같은 환경 변수 또한 지정할 수 있습니다.

### 초기 디렉토리 텍스트 박스

도구의 작업 디렉토리를 지정합니다.
디렉토리를 선택하려면 **화살표** ( **>**) 버튼을 사용합니다.
%WinDir%와 같은 환경 변수 또한 지정할 수 있습니다.

### 아이콘 경로 텍스트 박스

아이콘이 샘플된 파일 이름을 입력합니다.

### 현재 아이콘 리스트 박스

사용 가능한 아이콘과 현재 아이콘이 선택되어있는 목록입니다.

### 파일 저장 체크 박스

이 항목이 선택된 경우, 엠에디터는 도구를 실행하기 이전에 현재 열려있는 문서를 저장합니다.

### 출력 모음 사용 체크 박스

이 항목이 선택된 경우, 엠에디터는 출력 모음에 표준 출력과 표준 오류 출력을 재지정합니다.

### 끝낼 때 닫기 체크 박스

이 항목이 선택된 경우, 도구 프로세스가 종료된 후 출력 모음을 닫습니다.

### 입력 드롭다운 리스트 박스

외부 도구를 위한 표준 입력을 선택합니다.

|     |     |
| --- | --- |
| 없음 | 표준 입력이 사용되지 않습니다. |
| 선택 영역 | 선택된 텍스트가 표준 입력으로 사용됩니다. |
| 문서 | 전체 문서가 표준 입력으로 사용됩니다. |
| 사용자 지정 | 사용자 지정 형식에서 표준 입력을 지정합니다.<br> 이 항목이 선택된 경우 **사용자 지정** 텍스트 박스와 **EOF 추가** 체크 박스 또한 반드시 지정되어야 합니다. |

### 사용자 지정 텍스트 박스

사용자 지정된 표준 입력을 지정합니다. 이 텍스트 박스는 **입력** 드롭다운 리스트 박스에서
**사용자 지정** 이 선택된 경우에만 사용 가능합니다.

### EOF 추가 체크 박스

이 항목이 선택된 경우, 사용자 지정 표준 입력의 끝에 **파일의 끝** 문자가 추가됩니다.
이 체크 박스는 **입력** 드롭다운 리스트 박스에서 **사용자 지정** 이 선택된 경우에만 사용 가능합니다.

### 출력 드롭다운 리스트 박스

외부 도구를 위한 표준 출력을 선택합니다.

|     |     |
| --- | --- |
| 버리기 | 표준 출력이 삭제됩니다. |
| 선택 영역 바꾸기 | 표준 출력이 선택된 텍스트를 대체합니다. |
| 문서 바꾸기 | 표준 출력이 전체 문서를 대체합니다. |
| 선택 전 삽입 | 표준 출력이 선택된 텍스트 이전에 삽입됩니다. |
| 선택 후 삽입 | 표준 출력이 선택된 텍스트 뒤에 삽입됩니다. |
| 도구 설명으로 보이기 | 표준 출력이 도구 설명으로 표시됩니다. |
| 새 문서 생성 | 표준 출력이 새로운 문서의 내용이 됩니다. |

### 인코딩 드롭다운 리스트 박스

출력 모음의 인코딩을 선택합니다.

### 표준 오류 드롭다운 리스트 박스

외부 도구의 표준 오류를 표시할 방법을 선택합니다.

|     |     |
| --- | --- |
| 버리기 | 표준 오류가 삭제됩니다. |
| 선택 영역 바꾸기 | 표준 오류가 선택된 텍스트를 대체합니다. |
| 문서 바꾸기 | 표준 오류가 전체 문서를 대체합니다. |
| 선택 전 삽입 | 표준 오류가 선택된 텍스트 이전에 삽입됩니다. |
| 선택 후 삽입 | 표준 오류가 선택된 텍스트 뒤에 삽입됩니다. |
| 도구 설명으로 보이기 | 표준 오류가 도구 설명으로 표시됩니다. |
| 새 문서 생성 | 표준 오류가 새로운 문서의 내용이 됩니다. |
| 출력바로 디스플레이 | 표준 오류가 출력바에 표시됩니다. |

### ... 버튼

지정한 파일을 찾으려면 이 버튼을 선택합니다.

### \> 버튼

미리 정의된 인수 목록으로부터 선택하려면 이 버튼을 사용합니다.
