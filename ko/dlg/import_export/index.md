# 가져오기 및 보내기 마법사

이 마법사는 [**가져오기 및 보내기** 명령](../../cmd/tools/import_export) 이
선택되면 나타납니다. 그 후에 파일로부터 가져오는 설정이나 현재 설정을 파일로 내보내기를 선택할 수 있습니다.

마법사 내에 체크 박스 또는 텍스트 박스와 같은 각 컨트롤에 대한 더 자세한 정보를 찾으려면
아래의 컨트롤 목록에서 선택하거나 컨트롤이 선택되었을 때 **F1** 키를 누릅니다.

### 실행하려는 동작을 선택하십시오 리스트 박스

목록에서 수행할 작업을 선택합니다.

|     |     |
| --- | --- |
| **모든 설정을 등록 파일로 보냅니다.** | 구성, 메뉴, 도구 모음, 외부 도구 및 최근 사용한 파일 목록을 포함한 모든 개인 설정을<br> 등록 파일로 내보냅니다. 차후에 현재의 설정을 복원하려면 이 파일을 사용해 가져오게 하거나<br> 다른 컴퓨터로 이 설정을 복사할수 있습니다. |
| **모든 설정을 INI 파일로 보냅니다.** | 모든 개인 및 전역 설정을 INI 파일로 보냅니다.<br> 이 옵션은 INI 파일이 이미 엠에디터 설치 폴더에 존재하는 경우에는 사용할 수 없습니다. |
| **모든 설정을 등록 파일로부터 가져옵니다.** | **모든 설정을 등록 파일로 보냅니다** 를 선택하여 이전에 저장된 파일을 읽습니다.<br> 이전에 저장된 설정을 복원하려면 이 선택을 사용할 수 있습니다. |
| **USB 드라이브와 같은 제거 가능 드라이브를 기본 설정으로 설정합니다.** | 엠에디터 파일들을 복사하고 이동식 드라이브에 기본 설정으로 INI 파일을 생성합니다.<br> 이 옵션은 INI 파일이 이미 엠에디터 설치 폴더에 존재하는 경우에는 사용할 수 없습니다. |
| **USB 드라이브와 같은 제거 가능 드라이브를 설정하고 INI 파일로 모든 설정을 보냅니다.** | 엠에디터 파일들을 복사하고 현재 설정으로 이동식 드라이브에 INI 파일을 생성합니다.<br> 이 옵션은 INI 파일이 이미 엠에디터 설치 폴더에 존재하는 경우에는 사용할 수 없습니다. |
| **레지스트리에 INI 파일을 가져옵니다.** | 레지스트리에 INI 파일을 가져옵니다. |

**참고**

INI 파일을 생성하려고 옵션중 하나를 선택하였을 때 다음의 INI 파일이 생성됩니다:

- eeCommon.ini
- eeConfig.ini
- eeLM.ini

엠에디터가 시작하고 설치 폴더에서 **eeUseIni.ini** 을 발견하면,
엠에디터는 레지스트리 대신에 이 INI 파일을 사용하고, 레지스트리에 어느 설정도 저장하지 않습니다.
