# 메뉴 속성 대화 상자

이 대화 상자는 [**사용자 지정 메뉴** 대화 상자](../index) 에서 **속성** 버튼이 선택되면 나타납니다.

## 명령 라디오 버튼

명령 메뉴를 지정합니다.

## 팝업 라디오 버튼

메뉴 폴더로 팝업 메뉴를 지정합니다.

## 구분 기호 라디오 버튼

구분 명령을 위한 구분 기호를 지정합니다.

## 범주 리스트 박스

명령에 가능한 범주 입니다.

## 명령 리스트 박스

선택된 범주에 가능한 명령입니다.

## 설명 텍스트

선택된 명령의 설명을 표시합니다.

## 이름 텍스트 박스

메뉴 이름을 지정합니다. 바로가기 키를 지정하려면 & 마크를 사용합니다.
