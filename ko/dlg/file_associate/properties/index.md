# (확장자) 속성 대화 상자

이 대화 상자는 [엠에디터와 연결 대화 상자](../index) 의속성 버튼이 선택되었을 때 나타납니다.
지정된 확장자의 속성을 설정할 수 있습니다.

## 확장자 텍스트 박스

연결하기 원하는 파일 확장자를 입력합니다.

## 파일 유형 텍스트 박스

파일 유형을 입력합니다.

## 이 파일의 아이콘 샘플 텍스트 박스

아이콘 샘플을 가져온 파일 이름을 입력합니다.

## ... 버튼

지정한 파일을 찾으려면 이 버튼을 선택합니다.

## 현재 아이콘 리스트 박스

사용 가능한 아이콘과 현재 선택된 아이콘 목록입니다.
