# 반자/전자 변환 대화 상자

이 대화 상자는 [**반자/전자 전환** 명령](../../cmd/edit/convert) 이 선택되었을 때 나타납니다.
변환할 문자 유형과 변환 방법을 지정할 수 있습니다.

## 가타카나 체크 박스

가타카나로 변환을 적용합니다.

## 가나 표시 체크 박스

가나 표시로 변환을 적용합니다.

## 알파벳 및 숫자 체크 박스

알파벳과 숫자로 변환을 적용합니다.

## 사전순 표시 체크 박스

사전순 표시로 변환을 적용합니다.

## 공백 체크 박스

공백으로 변환을 적용합니다.

## 전자 문자 라디오 버튼

반자 문자를 전자 문자로 변환을 적용합니다.

## 반자 문자 라디오 버튼

전자 문자를 반자 문자로 변환을 적용합니다.

## 모두 라디오 버튼

모든 텍스트를 변환합니다.

## 선택 라디오 버튼

선택된 부분만 변환합니다.
