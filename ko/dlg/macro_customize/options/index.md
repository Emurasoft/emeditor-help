# 옵션 탭

**옵션** 탭은 **매크로** 메뉴와 관련된 옵션 설정을 하도록 합니다.

### 새 매크로가 선택 또는 저장 될 시 내 매크로에 추가 체크 박스

이 항목이 선택된 경우,
[**매크로 저장** 명령](../../../cmd/macros/macro_save) 또는
[**매크로 선택** 명령](../../../cmd/macros/macro_select) 으로 새로운 매크로가 선택되었을 때
**내 매크로** 에 추가됩니다.

### 새 기록된 매크로 편집 시 매크로 언어 묻기 체크 박스

임시 매크로가 선택되어 있는 동안 **매크로 편집** 을 선택하면
JavaScript 또는 VBScript에서 매크로 언어를 선택하기 위한 대화 상자를 표시합니다.

### 마우스 활성 및 기타 응용 프로그램의 키보드 출력 기록 체크 박스

이 항목이 선택된 경우, 엠에디터는 엠에디터 내의 일반 키 입력 뿐만 아니라
기타 응용프로그램에 마우스 활동 및 키보드 입력을 기록합니다.

### 모든 마우스 움직임 기록 체크 박스

이 항목이 선택된 경우, 엠에디터는 마우스 버튼을 위 아래로 하는 것을 포함한 기타 마우스 활동뿐만 아니라
모든 마우스의 움직임을 기록합니다.

### 취침 모드 삽입 전 유휴 시간 텍스트 박스

취침 모드를 삽입하기 전 대기하는 밀리 초단위의 최소 시간을 지정합니다.

### 폴더 텍스트 박스

**내 매크로** 의 기본 폴더를 지정합니다.
[**매크로 저장** 명령](../../../cmd/macros/macro_save) 또는
[**매크로 선택** 명령](../../../cmd/macros/macro_select) 을 사용하여 매크로를 선택하면
이 폴더가 기본 폴더가 됩니다.

### ... 버튼

지정한 파일을 찾으려면 이 버튼을 선택합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
