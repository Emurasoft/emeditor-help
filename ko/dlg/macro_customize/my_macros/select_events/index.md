# 이벤트 선택 대화 상자

이 대화 상자는 [**매크로 사용자 지정 대화 상자**](../../index) 의
[**내 매크로** 탭](../index) 에서 **이벤트** 버튼이 선택되면 나타납니다.

이 대화 상자는 관리자 사용 권한으로 로그온 되어있는 경우에만 사용 가능합니다.

### 이벤트 선택 리스트 박스

매크로가 자동으로 실행을 시작하는 트리거로 선택될 수 있는 이벤트의 목록을 표시합니다.

### 지연 시간 슬라이더

특정한 이벤트의 경우, 이 슬라이더는 매크로가 실행되기 전 대기하는 지연 시간을 조정하는데 사용될 수 있습니다.
