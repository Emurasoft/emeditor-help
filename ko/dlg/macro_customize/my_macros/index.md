# 내 매크로 탭

**내 매크로** 탭은 **내 매크로** 에 관련된 옵션을 설정하도록 합니다.

## 내 매크로 리스트 박스

**내 매크로** 목록을 표시합니다.

## 새로 만들기 버튼

목록에 새로운 항목을 추가하려면 이 버튼을 클릭합니다.

## 삭제 버튼

목록에서 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

## 모두 삭제 버튼

목록에서 모든 항목을 삭제하려면 이 버튼을 클릭합니다.

## 위로 버튼

목록에서 선택된 항목을 위로 이동합니다.

## 아래로 버튼

목록에서 선택된 항목을 아래로 이동합니다.

## 편집 버튼

매크로를 편집할 수 있도록 선택된 매크로를 엠에디터로 엽니다.

## 이벤트 버튼

선택된 매크로와 연관될 수 있는 이벤트를 선택하도록
[**이벤트 선택** 대화 상자](select_events/index) 를 표시합니다.

## 이벤트 시 실행 체크 박스

지정된 이벤트에서 선택한 매크로를 실행합니다.

## 이벤트 텍스트 박스

매크로를 실행할 이벤트를 표시합니다.

## 경로 텍스트 박스

선택된 매크로의 전체 경로를 표시합니다.
