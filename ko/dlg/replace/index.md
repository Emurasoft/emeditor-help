# 바꾸기 대화 상자

이 대화상자는 [**바꾸기** 명령](../../cmd/search/edit_replace) 이
선택되었을 때 나타납니다.
찾을 문자열과 대체할 문자열 및 다른 옵션들 또한 지정할 수 있습니다.

## 찾기 드롭다운 리스트 박스

검색할 문자열을 입력합니다. 다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다.
드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

## \> 버튼

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

|     |     |
| --- | --- |
| 선택된 텍스트 | 이 항목이 선택된 경우, 대화 상자는 선택된 텍스트로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 커서에 단어 | 이 항목이 선택된 경우, 대화 상자는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 사용자 지정 | 이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 기본값으로 저장 | (선택된 텍스트, 커서에 단어 및 사용자 지정)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다. |
| 다중 선 | 드롭다운 리스트 박스의 한 선과 다중선 사이에서 선택/취소 합니다.<br> (이 명령은 찾기 대화 상자와 바꾸기 대화 상자에서만 나타납니다.) |

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다.
리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

## 바꾸기 드롭다운 리스트 박스

대체하기 원하는 문자열을 입력합니다.
다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다.
드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

## 대/소문자 구분 체크 박스

문자열을 검색할 때 대/소문자를 일치시킵니다.

## 정규식 사용 체크 박스

정규식 표현을 가능하게 합니다. 정규식 표현식 구문에 대한 더 자세한 내용은
[정규식 사용하기](../../howto/search/search_regexp) 를 참조하십시기 바랍니다.

## 이스케이프 시퀀스 사용 체크 박스

이스케이프 시퀀스를 가능하게 합니다. 이스케이프 시퀀스로 다음의 문자들을 사용할 수 있습니다.

|     |     |
| --- | --- |
| \\a | 경고 (벨) |
| \\b | 백 스페이스 |
| \\f | 용지 공급 |
| \\n | 새로운 줄 |
| \\t | 수평 탭 |
| \\v | 수직 탭 |
| \\\ | 백 슬래시 |
| \\oooooo | 8진수 유니코드 문자 |
| \\xhhhh | 16진수 유니코드 문자 |

Null 문자 (\\0)는 사용되지 않을 수 있습니다. \\r대신 \\n이 사용되어야만 합니다.

## 전체 단어 일치 체크 박스

단어만을 검색합니다. 단어는 이와 같은 문자( A – Z, a – z, 0 – 9, 또는 밑줄)로 시작하고
끝나는 문자열로 정의됩니다. 전자 문자로 둘러쌓인 문자열을 단어로 간주합니다. 정규식 표현이 사용되는 경우, 이 체크 박스는 제대로 작동하지
않을 수도 있습니다. 정규식 표현을 사용하는 경우, 대신 단어의 경계식 (\\<, \\\> 및 \\b)을 사용하시기 바랍니다.

## 증분 검색 체크 박스

이 항목이 선택된 경우, **찾기** 드롭다운 리스트 박스에 입력하면 즉시 검색을 시작합니다.

## 모든 열린 문서 검색 체크 박스

동일한 프레임 창에 열려져 있는 모든 문서를 검색합니다.

## 선택 내에서만 체크 박스

선택된 텍스트 내에서만 대체합니다.

## 완료 시 닫기 체크 박스

검색이 완료되면 이 대화 상자를 닫습니다.

## 다음 찾기 버튼

현재 위치로부터 다음 항목을 검색합니다.

## 바꾸기 버튼

대체한 후 그 문자열의 다음 항목을 검색합니다.

## 모두 바꾸기 버튼

문서내의 모든 문자열을 대체합니다.

## << 찾기 버튼

선택된 옵션으로 지정된 문자열을 위해 [**찾기** \
대화 상자](../find/index) 를 표시하기 위해 이 버튼을 클릭합니다.

## 닫기 버튼

이 대화 상자를 닫기 위해 이 버튼을 클릭합니다.

## 사용자 지정 버튼

**사용자 지정 대화 상자** 를 표시하기 위해 이 버튼을 클릭합니다.

## 연속 바꾸기 >\> 버튼

**바꾸기** 대화 상자와 **연속 바꾸기** 대화 상자 사이에서 전환합니다.

다음의 제어 장치들은 이 대화 상자가 **연속 바꾸기** 대화 상자로 확장되었을 때에만 나타납니다.

## 항목에 추가 버튼

연속 바꾸기 리스트에 현재의 설정을 추가합니다.

## 수정 버튼

연속 바꾸기 이스트의 선택된 항목을 현재 설정으로 수정합니다.

## 리스트 박스

연속 바꾸기 처리에서 사용될 찾기 및 바꾸기 결합의 목록을 표시합니다.

## 모두 활성/비활성 체크 박스

리스트의 모든 항목을 활성/비활성화 합니다.

## 위로 버튼

리스트의 선택된 항목을 위로 이동합니다.

## 아래로 버튼

리스트의 선택된 항목을 아래로 이동합니다.

## 삭제 버튼

리스트에서 선택된 항목을 삭제하기 위해 이 버튼을 클릭합니다.

## 모두 선택 버튼

전체 리스트를 선택합니다.

## 가져오기 버튼

연속 바꾸기의 목록을 포함한 파일을 가져옵니다.

## 보내기 버튼

연속 바꾸기의 목록을 포함한 파일을 내보냅니다.

## 모두 연속 바꾸기 버튼

연속 바꾸기의 리스트를 사용하여 문서 안의 모든 문자열을 대체합니다.

이 대화 상자는 대화 상자의 오른쪽 하단 모서리를 드래그하여 크기를 조정할 수 있습니다.
대화 상자가 커질 때, 여러줄의 문자열이 검색어로 입력될 수 있습니다.
다중선 텍스트 박스가 활성화 되는 동안, CTRL + ENTER키를 사용하여 새로운 줄을 삽입할 수 있습니다.
