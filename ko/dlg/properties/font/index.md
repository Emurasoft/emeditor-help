# 글꼴 사용자 지정 대화 상자

이 대화 상자는 [**글꼴 사용자 지정** 명령](../../../cmd/view/font) 이 선택되거나
구성 속성 [**디스플레이** 탭](../display/index) 또는 [**인쇄** 탭](../print/index) 의
**Font** 버튼이 선택되었을 때 나타납니다.

### 글꼴 범주 리스트 박스

리스트로부터 변경하기 원하는 글꼴의 문자 범주를 선택합니다.

### 아래 목록에 기본 프린터 글꼴 포함 체크 박스

기본 프린터가 글꼴을 제공하는 경우, 글꼴 선택을 확장하도록
**글꼴 표시** 와 **글꼴 인쇄** 목록에 그 글꼴들을 추가할 수 있습니다.

### 글꼴 표시를 프린터 글꼴로 사용 체크 박스

프린터 글꼴과 글꼴 표시와 같아서 글꼴 표시를 프린터 글꼴로 사용합니다.

### 글꼴 표시 그룹 박스 \- 변경 버튼

글꼴 표시에 사용될 글꼴을 지정합니다.

### 글꼴 인쇄 그룹 박스 \- 변경 버튼

인쇄에 사용될 글꼴을 지정합니다.

### 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.

### 모두 선택 버튼

전체 리스트를 선택합니다.
