# 강조 표시 (2) 탭

**강조 표시 (2)** 탭은 강조 표시 (2)와 관련된 속성을 설정하도록 합니다.

## 설명 그룹 박스 \- 시작 텍스트 박스

다중 선 설명을 위한 시작 문자열을 선택합니다.

## 설명 그룹 박스 \- 종료 텍스트 박스

다중 선 설명을 위한 종료 문자열을 선택합니다.

## 설명 그룹 박스 \- 줄 설명 (1) 텍스트 박스

라인 설명을 지정하는 문자열을 선택합니다.

## 설명 그룹 박스 \- 줄 설명 (2) 텍스트 박스

라인 설명을 지정하는 문자열을 선택합니다.

## 큰 따옴표로 문자열을 묶습니다 그룹 박스 \- 작은 따옴표 \['...'\] 체크 박스

이 항목이 선택된 경우, 작은 따옴표 표시로 닫힌 문자열은 강조 표시됩니다.

## 큰 따옴표로 문자열을 묶습니다 그룹 박스 \- 큰 따옴표 \["..."\] 체크 박스

이 항목이 선택된 경우, 큰 따옴표 표시로 닫힌 문자열은 강조 표시됩니다.

## 큰 따옴표로 문자열을 묶습니다 그룹 박스 \- 다음 줄로 계속하기 체크 박스

이 항목이 선택된 경우, 인용 문자열이 다음 줄로 확장될수 있습니다.

## 큰 따옴표로 문자열을 묶습니다 그룹 박스 \- 이스케이프 문자 텍스트 박스

인용 부호에 대한 이스케이프 문자를 선택합니다.

## 스크립트 그룹 박스 \- 시작 텍스트 박스

스크립트의 시작 문자열을 선택합니다.

## 스크립트 그룹 박스 \- 종료 텍스트 박스

스크립트의 종료 문자열을 선택합니다.

## 특수 구문 드롭다운 리스트 박스

다음의 항목에서 선택합니다.

|     |     |
| --- | --- |
| **(없음)** | HTML와 관련이 없는 파일을 사용합니다. |
| **HTML** | HTML 또는 ASP 파일을 사용합니다. HTML, CSS, JavaScript, VBScript, 및 Script configurations을 사용합니다. |
| **HTML-포함** | JSP 또는 PHP, HTML 포함 언어와 같은 스크립트 언어를 사용합니다.<br> HTML, CSS, JavaScript, VBScript, 및 Script configurations을 사용합니다. |

## 대응 태그 강조하기 체크 박스

XHTML과 XML 문서 내 커서 위치에 내부 태그와 대응 태그를 강결합니다.
이 기능은 문서가 잘 구성된 XML 문서일때만 작용합니다. 이 기능은 종료 태그가 오래된 HTML 문서와 같은 문서로 부터
누락된 경우, 작동하지 않을 수 있습니다.

## HTML/XML 문자 참조와 UCN에 강조 표시 체크 박스

문서 내에서 HTML/XML 문자 참조와 유니버설 문자 이름 (UCN)에 강조 표시를 합니다.
이러한 강조된 문자열에 마우스 커서를 가져 가면 디코딩된 문자열을 나타내는 도구 설명을 표시합니다.

## 표시 일치 괄호 강조하기 체크 박스

일치하는 중괄호 (괄호나 대괄호)를 강조 표시합니다.

## 자동 완성 괄호/따옴표 체크 박스

괄호와 따옴표 마크를 자동 완성합니다.

## 대괄호 쌍 리스트 박스

자동 완성 가능한 대괄호 쌍의 목록입니다.
체크되어 있지 않은 항목들이 자동 완성되지 않는 동안 체크되어 있는 항목들은 자동으로 완성됩니다.
새로운 대괄호의 쌍을 추가할수 없습니다.

## 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
