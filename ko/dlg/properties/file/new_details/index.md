# 새 파일 상세 정보 대화 상자

이 대화 상자는 구성 속성 [파일 탭](../index) 의 [새 파일 버튼](../index) 을
선택하면 나타납니다.

## 인코딩 드롭다운 리스트 박스

새 파일을 저장할 인코딩을 선택합니다.

## 유니코드 서명 추가 (BOM) 체크 박스

유니코드 또는 UTF-8로 새 파일을 저장 할 때 서명을 추가합니다.

## 반환 방법 드롭다운 리스트 박스

새로운 파일을 생성할 때 반환 방법을 선택합니다.

## 서식 파일 사용 체크 박스

새로운 파일로 시작하는 서식 파일을 사용합니다.

## 서식 파일 텍스트 박스

새로운 파일로 로드할 서식 파일 이름을 지정합니다.

## ... 버튼

지정한 파일을 찾으려면 이 버튼을 선택합니다.

## 초기 폴더 설정 체크 박스

기본 초기 폴더를 설정합니다.제목 없음 문서가 저장된 경우
다른 이름으로 저장 대화 상자가 지정된 초기 폴더로 시작됩니다.
(문서,사진, 등등과 같은)알려진 폴더 가 지정된 경우,
윈도우 시스템이 폴더 위치를 재지정하는 것을 참고하십시오.
Sky Drive 통합이 윈도우8.1에 설치되어 있는 경우,문서 폴더는
Sky Drive Documents (스카이 드라이브 문서) 폴더로 재지정 됩니다.

## 초기 폴더 텍스트 박스

기본 초기 폴더를 위한 폴더 이름을 지정합니다.

## ... button

지정된 초기 폴더를 찾기 위해 이 버튼을 클릭합니다.

## 초기화 버튼

기본값으로 초기화합니다.
[초기화 대화 상자](../../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
