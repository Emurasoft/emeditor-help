# 세부 정보 저장 대화 상자

이 대화 상자는 구성 속성 [파일 탭](../index) 의저장 중 버튼이 선택되면 나타납니다.

## 유니코드를 ANSI로 저장할 때 확인 체크 박스

파일을 유니코드, 유니코드(big endian), 또는 UTF-8가 아닌 인코딩으로 저장할 때,
문서가 지정된 인코딩으로 변활될 수 없는 유니코드 문자를 포함하고 있는 경우 확인 메시지를 나타냅니다.

## 유니코드를 HTML/XML 문자 집합 참조로 저장 체크 박스

이 항목이 선택된 경우, 대상 코드 페이지로 변환될수 없는 유니코드 문자를
&#10070;와 같은 숫자형 엔터티 코드로 인코딩됩니다.

## 이름이 정해진 엔터티 참조 사용 체크 박스

이 항목이 선택된 경우, 대상 코드 페이지로 변환될수 없는 유니코드 문자를
&copy;와 같은 이름이 정해진 엔터티 코드로 인코딩됩니다.
이 옵션은 Internet Explorer 5.0 이상이 설치되어있는 경우에만 사용할 수 있습니다.

## 탭을 공백으로 저장 체크 박스

문서를 저장할 때 탭을 공백으로 변환합니다.

## 저장 시 반환 삽입 체크 박스

문서를 저장할 때 줄이 바뀌는 지점에서 반환을 삽입합니다.

## 저장 시 비어있는 파일 삭제 체크 박스

문서가 비어있는 경우 그 파일을 삭제합니다.

## 항상 저장 활성화 체크 박스

문서가 수정되지 않은 경우라도 [저장 명령](../../../../cmd/file/file_save) 을
활성화 합니다.

## 줄의 마지막에서 공백 삭제 체크 박스

문서를 저장할 때 줄 마지막의 공백을 삭제합니다.

## 제목없음 문서 자동 이름 지정 체크 박스

이 항목이 선택된 경우, 제목이 지정되지 않은 문서가 저장 될 때 문서의 첫번째 줄이 파일의 이름으로 사용될 것입니다.

## 이름을 변경할시 오래된 파일을 지우기 전에 확인 체크 박스

이 항목이 선택된 경우, 엠에디터는 이름을 변경할 시 오래된 파일을 지우기 전에 확인하는 대화 상자를 나타낼 것입니다.

## 인코딩 드롭다운 리스트 박스

파일을 저장할 때 인코딩 방법을 선택합니다.

## 유니코드 서명 추가 (BOM) 체크 박스

유니코드, 유니코드(big endian) 또는 UTF-8로 파일을 저장할 때, 바이트 순서 표시(BOM) 서명을 파일에 추가합니다.

## 반환 방법 드롭다운 리스트 박스

반환을 저장하는 방법을 지정합니다.
변경 없음 은 반환 스타일을 변경하지 않습니다.
CR+LF (Windows) 은 모든 반환을 CR+LF으로 변경합니다.
CR 만 (Macintosh) 은 모든 반환을 CR로 변경합니다.
LF 만 (Unix) 은 모든 반환을 LF로 변경합니다.
기본적으로, 윈도우는 모든 반환에 CR+LF을 사용하고, Macintosh는 CR만을, Unix는 LF만을 사용합니다.

## 초기화 버튼

기본값으로 초기화합니다.
[초기화 대화 상자](../../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
