# 파일 탭

**파일** 탭은 파일 작업에 관련된 속성을 설정하도록 합니다.

## Null 문자 발견시 확인 체크 박스

파일을 열었을 때 그 파일이 [무효 문자](../../../glossary/nullcharacter) 를 포함하고
있는 경우 경고 메시지를 나타냅니다. 파일이 열렸을 때 무효 문자는 공백으로 전환됩니다.
무효 문자를 포함한 파일은 원본 파일을 손상 할 수 있기 때문에 주의해야 합니다.

## 잘못된 문자 발견 시 확인 체크 박스

지정된 인코딩으로 전환할수 없는 유니코드 문자를 포함한 파일을 열은 경우 경고 메시지를 나타냅니다:
"일부 문자들은 지정된 인코딩을 사용하여 전환이 불가능합니다. 계속하여 편집하거나 저장하는 경우 파일이 손상될 수 있습니다."
예를 들어, Japanese EUC 파일을 Japanese Shift-JIS 인코딩으로 열려고 시도하는 경우와
Japanese Shift JIS 인코딩 범위 안에 있지 않은 문자 코드 값을 포함한 파일인 경우 경고 메시지가 나타납니다.

**알림**

이 체크 박스가 선택되어 있더라도, 일부 인코딩은 경고 메시지를 표시하지 않습니다.
예를 들어, Japanese Shift-JIS 파일을 Japanese EUC 인코딩으로 열려고 시도하면 경고 메시지를 표시하지 않습니다.

## 불일치 반환 시 확인 체크 박스

파일을 열었을 때 파일의 반환 유형이 일치하지 않은 경우 경고 메시지를 표시합니다.

## 전체 경로와 함께 파일 이름 보이기 체크 박스

엠에디터 창 제목에 전제 경로와 함께 파일 이름을 표시합니다.

## 활성화 중이 아닐 경우 전체 경로 제외 체크 박스

창이 활성화 되지 않는 한, 전체 경로와 함께 파일 이름을 표시하지 않습니다.

## 메모장 호환 가능 다이어리 체크 박스

메모장 다이어리 기능을 활성화합니다. 첫번째 줄에 .LOG로 저장하는 경우,
자동으로 파일을 열 때마다 문서 마지막에 시간과 날짜를 삽입합니다.

## HTML/XML 문자 집합 검색 체크 박스

HTML 파일로부터 "charset=...", XML 파일로부터 "encoding=...",
또는 Python과 Ruby 파일로부터 "#coding=..."을 찾고, 해당하는 인코딩을 사용합니다.
HTML, XML, Python, 또는 Ruby 파일을 연 경우,이 옵션만을 선택하고 다른 옵션들의 선택은 제거합니다.

## 모두 검색 체크 박스

사용 가능한 모든 코드 페이지로부터 통계적으로 코드 페이지를 검색합니다.
특히 파일이 매우 작은 경우, 검출 실수를 할수 있습니다. 이 옵션은 HTML/XML 문자 집합 검색을 포함하지 않습니다.

## 서명 검색 (BOM) 체크 박스

파일이 바이트 순서 표시 (BOM) 서명을 포함하는지 여부를 감지하고, BOM 서명을 감지한 경우,
엠에디터는 이 파일을 UTF-16LE, UTF-16BE, 또는 UTF-8 파일로 엽니다.

## UTF-8 검색 체크 박스

통계적으로 UTF-8을 검색합니다.

## CSV 검색 (쉼표 구분) 체크 박스

엠에디터가 파일을 열 때 CSV(쉼표로 구분) 파일을 검색합니다.
검색되기 위해서, CSV 파일은 적어도 두 줄 이상이어야 하며 동일한 숫자의 쉼표가 파일의 각 줄에 사용되어야만 합니다.

## TSV 검색 (탭 구분) 체크 박스

엠에디터가 파일을 열 때 TSV(탭으로 구분) 파일을 검색합니다.
검색되기 위해서, TSV 파일은 적어도 두 줄 이상이어야 하며 동일한 숫자의 탭이 파일의 각 줄에 사용되어야만 합니다.

## DSV 검색 (사용자 정의된 구분 기호 구분) 체크 박스

엠에디터가 파일을 열 때 DSV(사용자 정의된 구분 기호로 구분) 파일을 검색합니다.
검색되기 위해서, DSV 파일은 적어도 두 줄 이상이어야 하며 동일한 숫자의 구분 기호가 파일의 각 줄에 사용되어야만 합니다.

## 구분 기호 텍스트 박스

DSV(사용자 정의된 구분 기호로 구분)에 사용할 구분 기호를 지정합니다.

## 여는 인코딩 드롭다운 리스트 박스

오픈할 인코딩을 선택합니다.

## 기타 프로그램에 의하여 변경됨 드롭다운 리스트 박스

편집중에 있던 파일이 다른 프로그램에 의해 변경된 경우 수행할 작업을 선택합니다.

|     |     |
| --- | --- |
| 무시 | 파일이 다른 프로그램에 의해 변경된 경우더라도 파일을 다시 로드하지 않습니다.<br> 이 옵션은 파일의 읽기 전용 특성이 변경되더라도 엠에디터의 읽기 전용 상태를 변경하지 않습니다. |
| 확인 | 파일을 다시 로드할지 확인 여부를 묻는 메시지를 표시합니다.<br> 이 옵션은 파일의 읽기 전용 특성이 변경되었을 때 엠에디터의 읽기 전용 상태를 변경하도록 합니다. |
| 자동 다시 로드 | 수정되지 않는 한 항상 파일을 다시 로드 합니다.<br> 이 옵션은 파일의 읽기 전용 특성이 변경되었을 때 엠에디터의 읽기 전용 상태를 변경하도록 합니다. |
| 잠금 유지 | 엠에디터를 단독으로 사용하여 이 파일을 열고,<br> 이 파일이 열려 있는 동안 그 파일을 작성하려는 다른 프로그램을 금지합니다.<br> 이 옵션은 파일의 읽기 전용 특성이 변경되었을 때 엠에디터의 읽기 전용 상태를 변경하도록 합니다. |

## 기본 확장자 텍스트 박스

파일 확장자를 지정하지 않고 **파일 유형** 콤보 박스의 **모든 파일 (\*.\*)** 을 사용해 다른 이름으로
파일을 저장하는 경우,
이 텍스트 박스에 선택한 것이 파일 확장자가 됩니다.

## 모니터 간격 텍스트 박스

파일 변경을 모니터하는 시간 간격을 초단위로 지정합니다.
이 텍스트 박스는 **기타 프로그램에 의하여 변경됨** 드롭다운 리스트 박스의 **무시** 를 제외한 항목이 선택되었을 때 사용 가능합니다.

## 새 파일 버튼

새로운 파일을 생성할 때 인코딩과 다른 설정을 구성하는
[**새 파일 상세 정보** 대화 상자](new_details/index) 를 표시하려면 이 버튼을 클릭합니다.

## 저장 중 버튼

파일을 저장할 때 인코딩과 다른 설정을 구성하는
[**세부 정보 저장** 대화 상자](save_details/index) 를 표시하려면 이 버튼을 클릭합니다.
