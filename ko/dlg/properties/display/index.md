# 디스플레이 탭

**디스플레이** 탭은 디스플레이와 관련된 속성을 설정하도록 합니다.

## 테마 드롭다운 리스트 박스

색상의 결합을 정의하는 테마를 선택합니다.

## \> 버튼

이 버튼이 선택되었을 때 테마와 관련된 메뉴가 나타납니다. 다음의 옵션 중 하나를 선택합니다.

|     |     |
| --- | --- |
| **저장** | 선택된 테마의 설정을 저장합니다. |
| **복사** | 선택된 테마를 다른 이름으로 복사합니다. |
| **이름 변경** | 테마의 이름을 변경합니다. |
| **삭제** | 선택된 테마를 삭제합니다. |
| **가져오기** | 테마 파일로부터 테마를 가져옵니다. |
| **보내기** | 선택된 테마를 테마 파일로 내보냅니다. |
| **모두 보내기** | 모든 테마를 내보냅니다. |
| **모두 초기화** | 모든 테마를 초기화합니다. |

## 부분 지정 리스트 박스

색상을 적용하고 속성을 밑줄로 강조할 항목을 선택합니다.

## 텍스트 색 드롭다운 리스트 박스

텍스트의 색상을 선택합니다.

## ... 버튼

지정된 텍스트 색상을 찾으려면 이 버튼을 클릭합니다.

## 배경화면 색 드롭다운 리스트 박스

배경화면 색상을 선택합니다.

## ... 버튼

지정된 바탕화면 색상을 찾으려면 이 버튼을 클릭합니다.

## 스타일 드롭다운 리스트 박스

선택된 항목에 밑줄, 진하게, 기울이기, 물결모양 줄 또는 점선 유형을 적용합니다.

## 색 검색 텍스트 박스

강조 표시하기 위해 이전에 검색된 문자열의 숫자를 지정합니다.

## 모두 선택 버튼

모든 목록을 선택합니다.

## 따옴표 텍스트 박스

**따옴표 줄** 을 인식할 때 사용되는 따옴표 표시의 목록입니다.

## 줄 공백 텍스트 박스

줄 사이 공백의 간격을 지정합니다. 이 값은 한 줄 높이의 1/10에 해당합니다.

## 문자 공백 텍스트 박스

픽셀 문자 사이 공백의 간격을 지정합니다.

## (디스플레이 예문) 문자

현재 선택되어 있는 글꼴로 텍스트의 예문을 나타냅니다.

## 글꼴 버튼

텍스트를 나타내는데 사용될 글꼴을 지정합니다.

## 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
