# 링크 탭

링크 탭은 링크와 관련된 속성을 설정하도록 합니다.

## URL 링크 체크 박스

URL을 클릭하여 외부 응용 프로그램을 실행합니다. 외부 응용 프로그램은 익스플로러(보기- 옵션- 파일 유형)로 설정됩니다.

## 메일을 전송하려면 메일 주소를 클릭 체크 박스

메일 주소(@가 포함된 문자열)를 클릭하여 메일을 전송합니다.

## 따옴표 안의 파일 이름을 클릭하여 파일 열기 체크 박스

따옴표 안의 파일 이름을 클릭하여 파일을 엽니다.

## 인용문을 클릭하여 전체 문자열을 선택 체크 박스

[강조 표시 (2) 탭](../highlight2/index) 에서 정의된 따옴표 마크로 닫힌
문자열을 클릭하면 마크 사이의 전체 문자열이 선택됩니다.

## 더블 클릭 만을 활성화 체크 박스

더블 클릭하는 경우에만 URL과 메일 주소를 엽니다.

## URL을 클릭하여 전체 문자열을 선택 체크 박스

이 항목이 선택 되었을 때 URL을 클릭하면 전체 문자열을 선택합니다.

## 더블 바이트 문자를 URL로 인식 체크 박스

더블 바이트 문자를 URL로 인식합니다.

## 싱글 바이트 가나 및 가나 표시를 URL로 인식 체크 박스

싱글 바이트 가나 및 가나 표시를 URL로 인식합니다.

## 이 문자들을 인식 텍스트 박스

URL로 인식된 문자들을 지정합니다. 이 문자들을 128 이하의 ASCII 코드 값으로 구성됩니다.

## 끝에서 문자 허용 안됨 텍스트 박스

이 문자들을 인식 텍스트 박스에서 지정된 문자 중에서, URL의 끝에 허용되지 않는 문자를 지정합니다.

## 초기화 버튼

기본값으로 초기화 합니다.
[초기화 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
