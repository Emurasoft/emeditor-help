# 줄 바꾸기 없음 탭

**줄 바꾸기 없음** 탭은 기본 작업에 관련된 속성을 설정하도록 합니다.

### 줄 바꾸기 없는 단어 활성화 체크 박스

이 문자들은 줄의 끝에서 줄 바꿈이 허용되지 않습니다.

### 단어 줄 바꾸기 없음 리스트 박스

아래의 리스트에 있는 줄 바꾸기 없는 단어를 활성화하여 줄 바꿈으로 부터 방지합니다.

### 추가 버튼

리스트에 새로운 항목을 추가하려면 이 버튼을 선택합니다.

### 삭제 버튼

리스트의 선택된 항목을 삭제하려면 이 버튼을 선택합니다.

### 가져오기 버튼

CSV 우형 파일로부터 줄 바꾸기 없는 단어를 가져옵니다.
파일의 내용은 문자열로 구성되고 사용할 수 있는지 여부를 지정합니다.

### 보내기 버튼

CSV 우형 파일로부터 줄 바꾸기 없는 단어를 내보냅니다.
파일의 내용은 문자열로 구성되고 사용할 수 있는지 여부를 지정합니다.

### 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
