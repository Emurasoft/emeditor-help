# 탭/들여쓰기 대화 상자

이 대화 상자는 구성 속성 [일반 탭](../index) 의탭/들여쓰기 버튼이 선택되었을 때 나타납니다.

## 자동 들여쓰기 활성화 체크 박스

새 줄을 입력할 때 이전 줄과 같은 방식으로 공백을 삽입합니다.

## 탭을 위한 공백 삽입체크 박스

들여쓰기를 위해 탭 대신 공백을 사용합니다.

## 줄 바꿔서 들여쓰기 체크 박스

논리 행을 줄바꿈 하였을 때, 처음 표시 줄과 같은 방식으로 남아있는 표시 줄에
가상 공백을 삽입합니다.

## 정규식 사용 체크 박스

이 항목이 선택된 경우, 정규식 표현으로들여쓰기 시작 텍스트 박스,
들여쓰기 끝 텍스트 박스, 및설명문의 끝 텍스트 박스를 지정할 수 있습니다.

## 새로운 줄에서만 들여쓰기 체크 박스

이 항목이 선택된 경우, 새로운 줄에서만 들여쓰기가 삽입됩니다.

## 탭 열 텍스트 박스

문자 내의 탭 열입니다.

## 들여쓰기 열 텍스트 박스

문자 내의 들여쓰기 열입니다.

## 들여쓰기 시작 텍스트 박스

들여쓰기를 시작 하기위한 문자를 지정합니다.

## 들여쓰기 끝 텍스트 박스

들여쓰기를 마치는 문자를 지정합니다.

## 설명문의 끝 텍스트 박스

설명문의 끝을 나타내는 문자를 지정합니다.

## (정규식) 다음 무시 텍스트 박스

이 텍스트 박스 안에 지정된 정규식 표현과 일치하는 줄 이후에는 자동 들여쓰기가 무시됩니다.

## 초기화 버튼

기본값으로 초기화합니다.
[초기화 대화 상자](../../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
