# 구성 속성

이 대화 상자는 [**구성 정의** 대화 상자](../configurations/index) 에서
**속성** 버튼이 선택될 때, [**현재 구성 속성** 명령](../../cmd/tools/customize) 이
선택될 때, 또는 [**모든 구성 속성** 명령](../../cmd/tools/all_prop) 이 선택되었을 때 나타납니다.
