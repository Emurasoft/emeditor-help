# 키보드 탭

**키보드** 탭의 기본 작업과 관련된 속성을 설정하도록 합니다.

## 범주 드롭다운 리스트 박스

명령에 대해 사용 가능한 범주입니다.

## 명령 드롭다운 리스트 박스

선택된 범주에 대해 사용 가능한 명령입니다.

## < 버튼

이전에 선택된 명령으로 이동합니다.

## 새 바로가기 키 입력 텍스트 박스

선택된 명령에 대한 바로가기 키를 입력합니다.

## 현재 할당된 곳

방금 전 누른 바로가기 키가 현재 할당된 명령입니다.

## \> 버튼

현재 할당된 키와 연결 되어있는 명령으로 이동합니다.

## 현재 키 리스트 박스

선택된 명령에 할당된 현재 키 입니다.

## 설명 텍스트 박스

선택된 명령에 대한 설명입니다.

## 할당 버튼

현재 선택된 명령을 위한 바로가기 키를 할당합니다.

## 삭제 버튼

현재 선택된 바로가기 키를 삭제합니다.

## 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
