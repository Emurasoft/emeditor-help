# 표시 탭

**표시** 탭은 표시와 관련된 속성을 설정하도록 합니다.

## 반환 보이기 체크 박스

문서에서 반환 마크를 표시하려면 선택합니다.

## 파일의 끝 보이기 체크 박스

문서 내 파일 끝 마크를 표시하려면 선택합니다.

## 탭 보이기 체크 박스

문서 내 탭 마크를 표시하려면 선택합니다.

## 공백 보이기 체크 박스

텍스트 내 싱글 바이트 문자 공백 마크를 표시합니다.

## 넓은 공백 보이기 체크 박스

문서 내 더블 바이트 문자 공백 마크를 표시하려면 선택합니다.

## CR과 LF를 다른 표시로 보이기 체크 박스

CR과 LF를 구분하기 위해 다른 마크를 사용하려면 선택합니다.

## 컨트롤 문자 보이기 체크 박스

텍스트 내 컨트롤 문자를 표시합니다.

## 들여쓰기 가이드 보이기 체크 박스

들여쓰기 가이드를 표시합니다.

## **사용자 지정 가이드 보이기 체크 박스**

사용자 지정 가이드를 나타냅니다.

## **사용자 지정 가이드 위치 텍스트 박스**

사용자 지정 가이드의 열 위치를 지정합니다. 다중 위치는 콤마(,)로 구분될 수 있습니다.

## **변경/저장된 줄 강조 표시 체크 박스**

변경/저장된 줄은 엠에디터 창의 왼쪽 가장자리에 강조 표시됩니다.

## 초기화 버튼

기본값으로 초기화합니다.
[**초기화** 대화 상자](../reset/index) 가 나타나고 기타 구성에서 복사할 수 있도록 합니다.
