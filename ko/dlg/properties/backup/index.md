# 백업 탭

**백업** 탭은 백업 작업과 관련된 속성을 설정하도록 합니다.

### 백업 폴더에 백업 저장 체크 박스

기존 파일을 저장할 때 **백업 폴더** 텍스트 박스에 지정된 폴더에 원본 파일을 저장합니다.

### 같은 폴더에 백업 저장 체크 박스

기존 파일을 저장할 때 같은 폴더에 원본 파일을 저장합니다.

### 백업을 위하여 휴지통 사용 체크 박스

기존 파일을 저장할 때 휴지통에 원본 파일을 저장합니다.
하지만, 네트워크 드라이브 및 이동식 디스크 드라이브는 휴지통을 가지고 있지 않으므로, 이 항목은 해당 드라이브에 대해서
무시됩니다. 또한, 휴지통의 용량보다 큰 크기의 파일들을 저장되지 않습니다.

### 휴지통 이용이 불가능 할 경우 백업 폴더에 저장 체크 박스

**백업을 위하여 휴지통 사용** 체크 박스가 선택되면, 네트워크 드라이브나 이동식 디스크 드라이브에
저장하는 경우 백업 폴더에 원본 파일을 저장합니다. 하지만, 휴지통의 용량보다 큰 크기의 파일들은 백업 폴더에 저장되지 않습니다.

### 같은 파일 이름이 존재할 경우 이름 바꾸기 체크 박스

백업 폴더 또는 같은 폴더에 같은 파일 이름이 이미 존재하는 경우 원본 파일의 이름을 바꿉니다.

### 백업 폴더 텍스트 박스

**백업 폴더에 백업 저장** 체크 박스 또는 **휴지통 이용이 불가능 할 경우 백업 폴더에 저장** 체크 박스가
선택된 경우 백업 폴더를 설정합니다. 폴더 이름에 경로 분리 표시 "\\"를 포함하지 않은 경우, 하위 폴더 이름이 지정됩니다.
예를 들어, "백업"이 지정되고 여는 파일이 "C:\\work\\test.txt"인 경우,
백업은 "C:\\work\\backup\\test.txt"에 생성될 것입니다.
폴더 이름에 경로 분리 표시 "\\"를 포함한 경우, 전체 경로가 지정됩니다.
