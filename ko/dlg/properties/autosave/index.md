# 자동 저장 탭

자동 저장 탭은 자동 저장 작업과 관련된 속성을 설정하도록 합니다.

## 자동 저장 체크 박스

지정된 시간 간격으로 문서를 자동으로 저장합니다.
자동 저장 폴더에 저장 체크 박스가 선택되어 있는 경우, 지정된 폴더에 파일을 저장합니다.
자동 저장 기능은 매우 큰 파일이 열려있을 때 작동되지 않습니다.

## 자동 저장 폴더에 저장 체크 박스

이 항목이 선택된 경우,자동 저장 폴더에 저장 텍스트 박스에 지정된 폴더에 저장합니다.
이 항목이 선택되어 있지 않은 경우, 파일은 동일한 폴더에 저장됩니다.
제목이 정해지지 않은 파일은 체크된 경우에만 저장되고 파일 이름은 nnn이 정수인 "txtnnn.tmp"이 됩니다.
폴더의 전체 경로가 지정되어야만 합니다.

## 같은 파일 이름이 존재할 경우 이름 변경 체크 박스

동일한 파일 이름이 이미 존재하는 경우, 다른 이름으로 저장합니다.

## 자동 저장 폴더 텍스트 박스

자동 저장 폴더에 저장 체크 박스가 선택된 경우 자동 저장 폴더를 설정합니다.
