# 도구 모음 대화 상자

이 대화 상자는 [**도구 모음 사용자 지정** 명령](../../cmd/tools/customize_toolbar) 이 선택되면 나타납니다.

## 도구 모음 리스트 박스

이미 정의된 도구 모음 리스트 입니다.

## 플러그 인 도구 모음 디스플레이 체크 박스

플러그 인 도구 모음을 표시하거나 숨깁니다.

## 도구 도구 모음 디스플레이 체크 박스

외부 도구 모음을 표시하거나 숨깁니다.

## 매크로 도구 모음 디스플레이 체크 박스

내 매크로에 접근 가능한 도구 모음을 표시합니다.

## 도구 모음 제목 디스플레이 체크 박스

버튼 스타일이 선택되면 **플러그 인 도구 모음**, **도구 도구 모음** 과 **탭 모음** 의
도구 모음 제목을 표시합니다.

## 도구 모음 잠금 체크 박스

도구 모음 위치를 잠급니다.

## 큰 아이콘 사용 체크 박스

큰 아이콘으로 도구 모음을 표시합니다.

## 기본 도구 모음에 매크로 제목 디스플레이 체크 박스

이 항목이 선택된 경우, 매크로 버튼들은 기본 도구 모음에서 제목 텍스트를 포함합니다.
선택되지 않은 경우, 기본 도구 모음의 버튼들은 텍스트를 포함하지 않고 각 버튼은 작게 유지됩니다.

## 16 색 라디오 버튼

16 색상 아이콘으로 도구 모음을 표시합니다.

## 트루 컬러 라디오 버튼

트루 컬러 (24 비트) 아이콘으로 도구 모음을 표시합니다.
(트루 컬러 아이콘은 Windows XP/2003/Vista에서만 가능합니다.)

## 닫기 버튼

대화 상자를 닫으려면 이 버튼을 클릭합니다.

## 새로 만들기 버튼

새로운 항목을 목록에 추가하려면 이 버튼을 클릭합니다.

## 삭제 버튼

목록에서 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

## 사용자 지정 버튼

선택된 도구 모음을 사용자 지정합니다.

## 모두 초기화 버튼

모든 항목을 기본 설정으로 초기화하려면 이 버튼을 클릭합니다.
