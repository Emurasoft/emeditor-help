# 등록 키 입력 대화 상자

이 대화 상자는 올바른 등록 키가 입력되지 않은 상태에서 엠에디터를 시작할 때 나타납니다.

## 귀하의 이름 텍스트 박스

이름을 입력합니다

## 등록 키 텍스트 박스

현재 버전의 엠에디터 Professional에 해당하는 등록키를 클립보드에 복사하여
텍스트 박스를 선택하고,CTRL + V 를 누르거나 오른쪽 클릭으로 상황에 맞는 메뉴를 표시하여
붙여넣기 를 선택합니다. 엠에디터 Professional를 구매하였다면 키가 나타나거나 이메일을 통해 보내집니다.
이전 버전의 엠에디터에서 업그레이드 한 경우, Emurasoft 고객 센터에서 현 버전의 키를 요청할 수 있습니다.
(https://support.emeditor.com/).
