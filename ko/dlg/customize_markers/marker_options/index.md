# 옵션 탭

**옵션** 탭은 마커 옵션을 사용자 지정하도록 합니다.

## 기본 색상 드롭다운 리스트 박스

마커 목록에 추가 할 때 다음 마커에 사용될 색상을 지정합니다.

## 마커 제거시 알림 메시지 표시 체크 박스

이 체크 박스가 선택되어있으면
사용자가 마커 도구모음의 오른쪽 클릭 메뉴로부터 **삭제** 명령을 선택하였을 때
엠에디터가 확인 메시지를 나타냅니다.

## 자동적으로 마커 도구 모음 보이거나 숨기기 체크 박스

이 체크 박스가 선택되어있으면
엠에디터는 도구 모음에 버튼이 없는 경우 자동으로 마커 도구 모음을 숨기고,
사용자가 마커를 정의 할때 도구모음을 표시합니다.

## **마커 도구 모음이 숨겨졌을 때 자동적으로 마커 비활성 체크 박스**

체크 박스가 체크되면, EmEditor는 마커 도구 모음이 숨겨졌을 때 자동으로 모든 마커를 비활성화 합니다.
