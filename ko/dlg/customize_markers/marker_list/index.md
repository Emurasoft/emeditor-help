# 마커 목록 탭

**마커 목록** 탭은 마커 목록을 사용자 지정하도록 합니다.

## 마커 목록 리스트 박스

강조 표시될 마커의 목록 입니다. 항목을 활성화 또는 비활성화 하려면 체크박스를 클릭합니다.

## 제목 체크 박스

지정된 제목을 사용합니다.

## 제목 텍스트 박스

새로운 마커의 제목을 지정하도록 합니다.

## 선택 항목 활성화 체크 박스

목록에서 선택된 항목을 활성화 또는 비활성화 합니다.

## 추가 버튼

목록에 새로운 항목을 추가하려면 이 버튼을 클릭합니다.

## 삭제 버튼

목록에서 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

## 위로 버튼

목록에서 선택된 항목을 위로 이동합니다.

## 아래로 버튼

목록에서 선택된 항목을 아래로 이동합니다.

## 모두 선택 버튼

모든 리스트를 선택합니다.

## 색상 드롭다운 리스트 박스

목록에서 선택된 항목의 색상을 지정합니다.

## 단어 단위로 검색 체크 박스

전체 단어가 문자열인 경우에만 선택된 문자열을 강조 표시합니다.

## 대/소문자 구분 체크 박스

대/소문자가 구분된 경우에만 선택된 단어를 강조 표시합니다.

## 정규식 사용 체크 박스

정규식 표현을 사용한 선택 문자열을 강조 표시합니다.
