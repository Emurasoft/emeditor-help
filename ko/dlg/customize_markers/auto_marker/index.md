# 자동 마커 탭

**자동 마커** 탭은 마커 목록을 사용자 지정하도록 합니다.

### 자동 마커 활성화 체크 박스

자동 마커를 활성화 합니다.

### 커서에 단어 재개 체크 박스

커서 위치에 단어를 다시 찾습니다.

### 단어 단위로 검색 체크 박스

전체 단어가 문자열인 경우에만 선택된 문자열을 강조 표시합니다.

### 대/소문자 구분 체크 박스

대/소문자를 구분한 경우에만 선택된 단어를 강조 표시합니다.

### 영문자와 숫자만 체크 박스

단어가 알파벳과 숫자로만 구성되어 있는 경우에만 선택된 단어를 강조 표시합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
