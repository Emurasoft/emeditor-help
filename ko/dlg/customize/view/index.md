# 보기 탭

보기 탭 은 보기에 관련된 설정을 사용자 지정하도록 합니다.

## 사용자 지정 모음 색 \- 시스템 색 사용 체크 박스

이 항목이 선택된 경우, 각 사용자 지정 모음의 텍스트와 배경색이 시스템 색상을 사용합니다.

## 사용자 지정 모음 색 \- 텍스트 버튼

사용자 지정 모음의 텍스트 색상을 변경하려면 이 버튼을 클릭합니다.

## 사용자 지정 모음 색 \- 배경 버튼

사용자 지정 모음의 배경 색을 변경하려면 이 버튼을 클릭합니다.

## 사용자 지정 모음 색 \- (샘플) 텍스트

샘플 텍스트를 현재 선택된 색상으로 보여줍니다.

## 커서 크기와 색 \- 너비 드롭다운 리스트 박스

커서의 너비를 지정합니다.

## 커서 크기와 색 \- 높이 드롭다운 리스트 박스

커서의 높이를 지정합니다.

## 커서 크기와 색 \- 시스템 색 사용 체크 박스

이 항목이 선택된 경우, 커서 색상이 배경 색상에 반전됩니다.

## 커서 크기와 색 \- 색 버튼

커서의 색상을 변경하려면 이 버튼을 클릭합니다.

## 이중 버퍼링 사용 체크 박스

이중 버퍼링을 사용한 스크린에 텍스트를 그릴 이중 버퍼링을 사용합니다.
이중 버퍼링은 깜박 거림을 사라지게 하지만 그리기의 속도를 느리게 합니다.
느린 컴퓨터를 사용하고 그리기의 속도가 느려지는 경우 이 체크 박스의 선택을 취소하시기 바랍니다.

## 초기화 버튼

기본 설정으로 초기화 합니다.
