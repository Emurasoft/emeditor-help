# 표시 탭

**표시 탭** 은 표시 작업에 관련된 설정을 사용자 지정하도록 합니다.

## 사용자 지정 CR+LF 체크 박스/텍스트 박스

지정된 표시로 CR+LF 표시를 지정합니다. 이 옵션은 [**반환 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 사용자 지정 CR 체크 박스/텍스트 박스

지정된 표시로 CR 표시를 지정합니다. 이 옵션은 [**반환 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 사용자 지정 LF 체크 박스/텍스트 박스

지정된 표시로 LF 표시를 지정합니다. 이 옵션은 [**반환 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 사용자 지정 EOF 체크 박스/텍스트 박스

지정된 표시로 EOF 표시를 지정합니다. 이 옵션은 [**파일의 끝 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 탭 사용자 지정 체크 박스/텍스트 박스

지정된 표시로 탭 표시를 지정합니다. 이 옵션은 [**탭 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 공백 사용자 지정 체크 박스/텍스트 박스

지정된 표시로 공백 표시를 지정합니다. 이 옵션은 [**공백 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

## 넓은 공백 사용자 지정 체크 박스/텍스트 박스

지정된 표시로 넓은 공백 표시를 지정합니다. 이 옵션은 [**넓은 공백 보이기** 체크 박스](../../properties/marks/index) 가 선택된 경우에만 적용됩니다.

기본 설정으로 초기화 합니다.
