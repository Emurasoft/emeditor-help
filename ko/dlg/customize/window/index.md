# 창 탭

**창 탭** 은 엠에디터 창에 관련된 설정을 사용자 지정하도록 합니다.

### 창 위치 복원 체크 박스

엠에디터를 실행할 때 창 위치를 복원합니다. 창 위치는 **현재 창 위치 저장** 버튼을
누르면 저장됩니다.

### 창 크기 복원 체크 박스

엠에디터를 실행할 때 창 크기를 복원합니다. 창 크기는 **현재 창 위치 저장** 버튼을
누르면 저장됩니다.

### 창 탭 위치 복원

엠에디터 탭 창이 닫히거나 창이 결합되지 않을 때 창 탭 위치가 자동으로 저장됩니다.
엠에디터가 탭 창으로 시작되거나 엠에디터의 창이 결합될 때 창의 탭 위치를 복원합니다.

### 소리로 알림 체크 박스

검색된 문자열을 찾을수 없는 상황과 같은 경우에 알림음 소리를 냅니다.

### 다음 문서 명령을 위하여 마지막 사용된 문서로 변경 체크 박스

이 항목이 선택된 경우, [**다음 문서** 명령](../../../cmd/window/next_window) 은
마지막으로 사용된 문서로 전환됩니다. 선택되지 않은 경우, [**다음 문서** 명령](../../../cmd/window/next_window) 은
탭 모음에 나타난 다음 문서로 전환됩니다.

### 탭 활성화 시 창을 종료하여 그룹 닫기 체크 박스

탭 활성화 시 엠에디터 오른쪽 상단 끝에 있는 **닫기** 버튼을 클릭하여
같은 그룹의 모든 엠에디터 문서를 닫습니다.

### 닫기 명령을 선택하여 마지막 문서를 닫지 않음 체크 박스

이 항목이 선택된 경우,
한 문서만 열려져 있는 동안 탭을 활성화하고 사용자가 **닫기** 명령을 선택하게 되면
엠에디터는 종료되지만 새로운 문서를 위한 준비가 되게 합니다.

### 삽입/덮어쓰기 모드 저장 체크 박스

이 항목이 선택된 경우 **삽입/덮어쓰기** 모드가 저장되고, 다음 번에 새로운 엠에디터를
시작했을 때 **삽입/덮어쓰기** 모드가 복원됩니다. 이 항목이 선택되지 않은 경우, 엠에디터는 항상 **삽입** 모드로
시작합니다.

### 자동 작업 영역 드롭다운 리스트 박스

작업영역을 자동으로 저장하거나 로드하고 싶은 방법을 지정합니다. 아래의 옵션중에 하나를 선택합니다:

|     |     |
| --- | --- |
| **없음** | 작업영역을 저장 또는 복원하지 않습니다. |
| **저장만** | 마지막 그룹을 닫으려고 할 때 작업영역이 자동으로 저장되지만, 이 옵션은 첫번째 엠에디터 창이 시작될 때<br> 작업영역을 자동으로 복원되지 않습니다. |
| **실행만** | 첫번째 엠에디터 창이 시작될 때 작업영역이 자동으로 복원되지만, 이 옵션은 마지막 그룹을 닫으려고 할 때<br> 작업영역을 자동으로 저장하지 않습니다. |
| **저장 및 복원** | 마지막 그룹을 닫으려고 할 때 작업영역을 자동으로 저장하고, 첫번째 엠에디터 창을 시작할 때 <br> 자동으로 복원됩니다. |
| **실행 취소 정보를 저장, 복원 및 보관합니다** | 마지막 그룹을 닫으려고 할 때 작업영역을 자동으로 저장하고, 첫번째 엠에디터 창을 시작할 때 <br> 자동으로 복원됩니다. 실행 취소와 다시 실행 정보 또한 저장되고 복원됩니다. |
| **실행 취소된 정보를 저장, 복원과 보관, 및 저장 파일을 확인하지 않음** | 마지막 그룹을 닫으려고 할 때 작업영역을 자동으로 저장하고, 첫번째 엠에디터 창을 시작할 때 <br> 자동으로 복원됩니다. 실행 취소와 다시 실행 정보 또한 저장되고 복원되고, 마지막 그룹을 닫을 때 파일 저장을 알리는 확인<br> 메시지가 표지되지 않습니다. |

### 나가기 전 묻지 않음 체크 박스

이 항목인 선택된 경우, 작업 영역 옵션을 선택하는 대화 상자가 나타나지 않습니다.

### 현재 창 위치 저장 버튼

이 버튼을 클릭하고 **창 위치 복원** 체크 박스 또는 **창 크기 복원** 체크 박스를
선택하면 창 위치 또는 크기가 복원됩니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
