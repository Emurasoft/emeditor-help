# 편집 탭

**편집 탭** 은 편집 작업에 관련된 설정을 사용자 지정하도록 합니다.

## 정규식을 사용하여 태그 서식 지정 체크 박스

정규 표현식을 사용하여 태그 서식을 지정하도록 합니다.

## 찾기 텍스트 박스

태그와 일치하는 정규 표현식을 지정합니다.

## 파일 이름 텍스트 박스

파일 이름과 일치하는 정규 표현식을 지정합니다.

## 줄 번호 텍스트 박스

행 번호와 일치하는 정규 표현식을 지정합니다.

## 한 글자씩 실행 취소 (엠에디터 재 실행 필요) 체크 박스

이 항목이 선택된 경우, [**실행 취소** 명령](../../../cmd/edit/edit_undo) 은
단어를 문자 하나 하나씩 실행 취소 합니다. 이 작업은 또한 문자 한 글자씩 삽입 명령을 기록하는 매크로에도 영향을
미치게 됩니다. 새로운 설정을 활성화하려면, 엠에디터를 재실행해야 합니다.

## 다중 선택 가능 체크 박스

다중 선택을 사용할지 여부를 지정합니다.
다중 선택을 하기 위해서는 한 부분을 선택한 후에 CTRL 키를 누른 상태에서 원하는 다른 부분을 선택하시면 됩니다.
또 다른 방법으로는, F8키를 누른후 커서를 확장하고 싶은 영역을 모두 선택 할 때까지 이동하고 F8키를 눌러 선택을 완료합니다.
그 후, 다시 한번 F8키를 눌러서 다음 선택하고싶은 영역으로 이동하여 위의 과정을 반복하면 원하는 모든 영역을 추가로 선택하실 수 있습니다.

## Internet Explorer에서 임시 폴더가 원본 보기로 열릴 경우 읽기 전용 체크 박스

Internet Explorer에서 소스 보기로 임시 파일을 열었을 때 읽기 전용으로 전환합니다.

## 더이상 가능한 실행 취소가 없을 때 확인 체크 박스

더이상 가능한 실행 취소 정보가 없을 때 확인하는 대화 상자를 표시합니다.

## EmEmditor가 클립보드를 열수 없을 때 확인 체크 박스

엠에디터가 클립보드를 열수 없을 때 확인하는 대화 상자를 표시합니다.

## 다음 문자를 사전순으로 처리 텍스트 박스

편집할 때 다음의 문자들을 사전순으로 처리합니다.
예를 들어, 단어를 두 번 클릭할 때 이 문자들은 단어의 한 부분으로 처리됩니다.
하지만, 이 설정은 정규 표현식 \\w 연산자 또는 찾기 대화 상자의 **전체 단어 일치** 옵션에는 적용되지 않습니다.

## 초기화 버튼

기본 설정으로 초기화 합니다.
