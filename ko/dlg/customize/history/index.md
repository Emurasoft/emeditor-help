# 기록 탭

**기록 탭** 은 기록에 관련된 설정을 사용자 지정하도록 합니다.

### 최근 파일 수 텍스트 박스

**파일** 메뉴와 트레이 아이콘 메뉴에 나타내는 최근에 사용한 파일의 수입니다.
최대 수는 64입니다.

### 최근 글꼴 수 텍스트 박스

**보기** 메뉴에 나타내는 최근에 사용한 글꼴의 수 입니다. 최대 수는 64입니다.

### 현재 폴더 내 파일의 스트립 경로 체크 박스

이 항목이 선택된 경우,
이 항목이 선택되면, 활성 파일과 같은 폴더에 있는 파일의 경로가 표시되지 않습니다.
하지만, 새로운 파일이 생성되거나 열린 다른 파일이 없는 경우 전체 경로가 나타납니다.
트레이 아이콘 메뉴는 항상 전체 경로를 표시합니다.

### 기타 키보드 바로가기 드롭다운 리스트 박스

메뉴 항목의 수가 10을 초과할 경우 키보드 바로가기를 지정합니다.
다음 옵션 중 하나를 선택합니다.

|     |     |
| --- | --- |
| **바로 가기 없음** | 키보드 바로가기를 추가하지 않습니다. |
| **모두 영(0)** | 모두 영(0)을 추가합니다. |
| **알파벳 추가** | 알파벳(A-Z)을 추가합니다. 이 항목이 선택된 경우, 메뉴 구성에 따라서<br> 알파벳과 기존의 다른 메뉴 항목의 키보드 바로가기가 대립 될수도 있습니다. |

### 외부 응용 프로그램에서 복사된 클립보드 내용 모니터 체크 박스

이 항목이 선택된 경우, 엠에디터는 외부 응용 프로그램에서 복사된 클립보드 내용을 모니터하지 않습니다.

### 기록 지우기 버튼

최근에 사용된 파일과 최근에 찾기, 바꾸기에 사용된 문자열을 모두 지우려면 이 버튼을 클릭합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
