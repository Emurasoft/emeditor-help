# 파일 탭

**파일 탭** 은 파일 작업과 관련된 설정을 사용자 지정하도록 합니다.

### 마지막 폴더 기억 체크 박스

**열기** 와 **다른 이름으로 저장** 대화 상자에서 가장 최근에 사용된 폴더를 나타냅니다.

### 새 파일 열때 확인 체크 박스

존재하지 않는 파일을 열려고 하는 경우 경고 메시지를 나타냅니다.

### 항상 최근 문서 폴더에 추가 체크 박스

파일을 열거나 저장할 때 윈도우 **내 최근 문서** 폴더에 바로가기를 저장합니다.
이 체크 박스는 최근에 사용한 항목을 포함하기 위해 윈도우 7 점프 목록의 최근 범주에 대한 순서에 있어야 합니다.

### 최근 파일 목록에서 인터넷 임시 저장파일을 제외 체크 박스

이 항목이 선택된 경우, 엠에디터는 인터넷 캐시 폴더에 있는 파일 경로를 추가하지 않습니다.

### 커서 위치와 책갈피 저장 체크 박스

최근에 사용된 파일의 커서 위치와 책갈피를 저장합니다.

### 항상 검색된 모든 결과 보이기 체크 박스

파일을 열때 **모두 검색** 체크 박스가 선택되었거나
**[모든 인코딩 검색하여 다시로드](../../../cmd/file/file_reload_detect_all)** 명령이
선택되어있는 경우, [**모든 결과 검색** 대화 상자](../../detect_result/index)
가 항상 표시됩니다. 하지만, UTF-16 서명 있음, UTF-16 서명 없음, 또는 UTF-8 서명 있음이 검색된 경우
그 파일을 이 대화 상자를 표시하지 않고 즉시 로드됩니다.

### 다중 인코딩이 검색될 경우에만 보이기 체크 박스

파일을 열 때 **모두 검색** 체크 박스가 선택되어 있거나 **[모든 인코딩 검색하여 다시로드](../../../cmd/file/file_reload_detect_all)** 명령이
선택되어 있는 경우에서 여러 인코딩이 검색된 경우에만 [**모든 결과 검색** 대화 상자](../../detect_result/index) 를
표시합니다.

### 파일 검색 중에도 검색된 모든 결과 보이기 체크 박스

**모두 검색** 체크 박스가 선택되어 있을 때는 파일에서 찾기 도중에
[**모든 결과 검색** 대화 상자](../../detect_result/index) 를 표시합니다.

### 프린터 엑세스 피하기 체크 박스

프린터에 엑세스하는 것을 방지합니다. 이 항목이 선택된 경우, 엠에디터가 실행될 때
**페이지 번호 보이기** 체크 박스가 선택된 경우라도 [**인쇄** 명령](../../../cmd/file/file_print) 또는
[**인쇄 미리보기** 명령](../../../cmd/file/print_preview) 이 선택되기까지 페이지 번호가 표시되지 않습니다.
프린터 엑세스 속도가 느린 경우 이 옵션 사용을 권장합니다.

### 대용량 파일을 열때 대용량 파일 컨트롤러 나타내기 체크 박스

이 항목이 선택된 경우,
(사용자 지정 대화 상자 [고급 탭](../advanced/index) 의 **비동기적으로 열기를 위하여 파일 사이즈를**
**최소화하십시오** 텍스트 박스에 지정된 크기보다 큰) 대용량 파일을 열 때 엠에디터가 자동적으로
대용량 파일 컨트롤러를 나타냅니다.

### 파일 필터 사용자 지정 텍스트 박스

**열기** 대화 상자와 **다른 이름으로 저장** 대화 상자의
**파일 유형** 콤보 박스에 나타나는 파일 필터 사용자 지정을 설정합니다.

### 엠에디터와 연결 버튼

어느 파일 확장자를 엠에디터와 연결하려고 하는지 지정할 수 있는
[**엠에디터와 연결** 대화 상자](../../file_associate/index) 를 표시합니다.
