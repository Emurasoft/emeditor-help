# 검색 탭

검색 탭 은 검색 작업에 관련된 설정을 사용자 지정하도록 합니다.

## 찾기/바꾸기 드롭다운 목록의 글꼴 변경 체크 박스

현재 열려있는 창의 글꼴과 동일한 글꼴을찾기 드롭다운 리스트 박스에 사용합니다.

## 선택된 글꼴의 문자 설정이 시스템 기본값일 경우에만 글꼴 변경 체크 박스

찾기/바꾸기 드롭다운 목록의 글꼴 변경 체크 박스가 선택되었을 때,
현재 열린 창 글꼴의 문자 설정이 시스템 기본값일 경우에만
현재 열려있는 창의 글꼴과 동일한 글꼴을찾기 드롭다운 리스트 박스에 사용합니다.

## 파일 내에서 바꾸기로 대체한 문자열 강조 표시 체크 박스

이 항목이 선택된 경우, 엠에디터가 [파일 내 바꾸기\
명령](../../../cmd/search/replace_in_files) 으로 대체한 문자열을 강조 표시 합니다. 선택되지 않은 경우, 엠에디터는 검색 문자열을 강조 표시합니다.

## 정규식은 줄 바꾸기 문자와 일치 체크 박스

정규 표현식을 사용한 문자열을 검색할 때 기간이 새 라인 문자와 일치 할수 있는지 여부를 지정합니다.
이 항목이 선택되지 않은 경우라도정규식 검색을 위한 추가 줄 텍스트 박스에 0보다 큰 숫자가 지정되면
정규식 표현 \\s은 새 라인 문자와 일치할 수 있습니다.
[파일 내 찾기 명령](../../../cmd/search/grep) 을 제외하고,
일치 가능한 줄의 실제 수는정규식 검색을 위한 추가 줄 텍스트 박스에 지정된 수에 따라 달라집니다.

## 찾기/바꾸기 대화 상자에 바로가기 비활성 대화 상자 체크 박스

이 항목이 선택된 경우, 찾기/바꾸기 대화상자의 키보드 바로가기가 비활성화 됩니다.

## 문서의 끝에 도달하였을 시 확인 체크 박스

이 항목이 선택된 경우, 엠에디터는 검색이 문서의 끝에 도달하였을 때 확인하는 대화 상자를 나타냅니다.

## 정규식 검색을 위한 추가 줄 텍스트 박스

정규식 표현을 사용할 때 문자열 검색을 위한 추가 줄의 수를 지정합니다. 엠에디터는 정규 표현식 기능의 매개
변수로 문자열을 라인별로 전달합니다. 이 방법이긴 하지만, 정규식 표현 <td>.\*?</td>은 문자열이 새로운 라인을 포함하고 있는 경우
<td> 과 </td> 사이의 문자열과 일치할수 없습니다.
이 텍스트 박스의 숫자를 0보다 크게 지정하고정규식은 줄 바꾸기 문자와 일치할 수 있습니다 체크 박스를
선택하면 새로운 라인의 지정된 숫자까지 문자열을 검색할 수 있습니다.
반면에, [파일 내 찾기 명령](../../../cmd/search/grep) 은 이 옵션과 관계없이
항상 전체 파일로부터 검색합니다.

## 열린 상태로 유지될 경우 교체할 수 있는 최대 파일 수 텍스트 박스

파일 내 바꾸기 명령의 [수정된 파일 열린 상태 유지 체크 박스](../../replace_in_files/index) 가
선택되었을 때 최대로 대체할 파일의 수를 지정합니다. 더 큰 숫자가 지정되는 경우, 많은 창이 동시에 실행되고 엠에디터는 더 많은 자원을 소모하게 됩니다.

## 검색 후에 편집기로의 포커스 변환 체크 박스

이 항목이 선택된 경우, 찾기 도구 모임을 사용하여 검색하기를 마친 후에 편집기로 포커스를 변환합니다.

## ESC 입력 시 닫기 체크 박스

이 항목이 선택된 경우, 찾기 도구 모음을 사용하여 검색하기를 마친 후에 찾기 도구 모음을 닫습니다.

## 대/소문자 구분

이 항목이 선택된 경우, 다음 항목 추가 또는 모든 항목을 선택하는 동안
엠에디터는 선택된 단어와 대/소문자가 일치하는 단어만을 검색합니다.

## 전체 단어 일치

이 항목이 선택된 경우, 다음 항목 추가 또는 모든 항목을 선택하는 동안 엠에디터는
전체 단어를 일치시킵니다.
