# 마우스 탭

**마우스 탭** 은 마우스에 관련된 설정을 사용자 지정하도록 합니다.

### 탭 \- 마우스 중간 버튼 클릭 드롭다운 리스트 박스

탭에서 마우스 중간 버튼을 눌렀을 때의 동작을 지정합니다.

### 탭 \- 마우스 왼쪽 버튼 두 번 클릭 드롭다운 리스트 박스

탭에서 마우스 왼쪽 버튼을 두 번 클릭했을 때의 동작을 지정합니다.

### 마우스 휠을 사용하여 선택 체크 박스

마우스 휠을 사용하여 탭을 선택합니다.

### 바깥쪽 탭 \- 마우스 중간 버튼 클릭 드롭다운 리스트 박스

탭 모음의 바깥쪽 탭을 마우스 중간 버튼으로 클릭했을 때의 동작을 지정합니다.

### 바깥쪽 탭 \- 마우스 왼쪽 버튼 두 번 클릭 드롭다운 리스트 박스

탭 모음의 바깥쪽 탭을 마우스 왼쪽 버튼으로 두 번 클릭했을 때의 동작을 지정합니다.

### 텍스트 끌어서 놓기 활성화 체크 박스

이 항목이 선택된 경우, 복사하거나 다른 곳으로 이동하려는 선택 텍스트를 끌어서 놓을수 있습니다.
CTRL 키를 누른 상태에서 끌어서 놓는 경우 텍스트가 복사됩니다.
SHIFT 키를 누른 상태에서 끌어서 놓는 경우 텍스트가 이동됩니다.
끌어서 놓는 동안 아무 키도 누르지 않는 경우, 같은 창에 놓으면 텍스트가 이동되고
다른 창에 놓으면 복사됩니다.

### 줄 선택을 하기 위한 세번 클릭 활성화 체크 박스

현재 커서 위치에 줄을 선택하기위해 마우스를 세 번 클릭하도록 합니다.

### 글꼴 크기 변경을 위하여 CTRL + 휠 체크 박스

이 항목이 선택된 경우, CTRL 키를 누른 상태에서 마우스 휠을 돌려서 글꼴 크기를 변경할 수 있습니다.

### 마우스로 활성화될 시 커서 위치 설정 체크 박스

이 항목이 선택된 경우, 활성화 시키기위해 엠에디터 창을 클릭하면 커서 위치가 이동됩니다.

### 마우스 오른쪽 버튼 클릭 \+ 마우스 휠을 사용하여 선택 체크 박스

이 항목이 선택된 경우, 마우스 오른쪽 버튼을 누른 상태에서 마우스 휠을 돌리면 엠에디터가 탭 사이를 순환합니다.

### CTRL + 클릭 사용하여 단어 선택 체크 박스

이 항목이 선택된 경우, CTRL 키를 누른 상태에서 단어를 클릭하면
(시작과 끝이 A-Z, a-z, 0-9, 또는 밑줄인 문자열로 정의되는) 단어를 선택합니다.

### 초기화 버튼

기본 설정으로 초기화 합니다.
