# 언어 탭

**언어 탭** 은 메뉴와 대화 상자에 사용되는 언어의 사용자 지정을 하도록 합니다.

## 시스템 언어 사용 라디오 버튼

하나 이상의 언어 구성 요소가 엠에디터에 설치되어 있으면 이 버튼이 선택된 경우
엠에디터는 윈도우 시스템 언어와 같은 언어를 사용하고자 합니다.
같은 언어의 사용이 불가능한 경우, 엠에디터는 설치된 언어를 사용합니다. 선택된 언어는 엠에디터 메뉴, 대화 상자, 도움말,
및 기타 사용자 인터페이스의 요소에 사용됩니다.
