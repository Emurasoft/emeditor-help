# 고급 대화 상자

이 대화 상자는 [**파일 내 찾기** 대화 상자](../find_in_files/index)
또는 [**파일 내 바꾸기** 대화 상자](../replace_in_files/index) 의 **고급** 버튼이
선택되었을 때 나타납니다.

## 다음 파일 또는 폴더 무시 체크 박스

이 항목이 선택된 경우, 엠에디터는 **무시할 파일 또는 폴더**
텍스트 박스에서 지정된 파일 또는 폴더 이름을 검색하지 않습니다.

## 무시할 파일 또는 폴더 텍스트 박스

무시할 파일 또는 폴더의 이름을 지정합니다. 여러 이름을 지정하려면, 세미콜론( **;**)으로
구분합니다.
