# 파일 내 찾기 대화 상자

이 대화 상자는 [**파일 내 찾기** 명령](../../cmd/search/grep) 이 선택되었을 때 나타납니다.
찾을 문자열과 파일 내에서 찾기 위한 다른 옵션들 또한 지정할 수 있습니다.

## 찾기 드롭다운 리스트 박스

검색할 문자열을 입력합니다. 다중선 텍스트 박스인 경우, CTRL + ENTER을 눌러서 새로운 줄을 입력할 수 있습니다.
드롭다운 리스트가 열려있는 동안, ALT + DELETE를 눌러 선택된 기록 사항을 삭제합니다.

## \> 버튼

이 버튼을 클릭해 가능한 명령의 목록을 나타냅니다.

|     |     |
| --- | --- |
| 선택된 텍스트 또는 커서에 단어 | 이 항목이 선택된 경우, 대화 상자는 선택된 텍스트 또는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 선택된 텍스트 | 이 항목이 선택된 경우, 대화 상자는 선택된 텍스트로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 커서에 단어 | 이 항목이 선택된 경우, 대화 상자는 커서에 단어로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 사용자 지정 | 이 항목이 선택된 경우, 대화 상자는 마지막으로 사용된 문자열로 찾기 드롭다운 리스트 박스를 초기화 합니다. |
| 기본값으로 저장 | (선택된 텍스트, 커서에 단어 및 사용자 지정)옵션을 다음에 이 대화 상자가 표시될 때 기본값으로 나타나도록 저장합니다. |

이 명령은 사용 가능한 이스케이프 시퀀스 또는 정규 표현식의 목록을 포함하고 있습니다.
리스트에서 한 항목을 선택하면 버튼 옆의 텍스트 박스에 선택된 항목을 삽입합니다.

## 파일 유형 드롭다운 리스트 박스

검색하려는 파일 유형입니다. 와일드카드(\*와 ?)가 사용 가능합니다.
또한, 여러 파일 형식은 세미콜론(;)으로 구분 가능합니다.

## 대/소문자 구분 체크 박스

문자열을 검색할 때 대/소문자를 일치시킵니다.

## 하위 폴더 보기 체크 박스

**폴더 내** 드롭다운 리스트 박스의 하위 폴더를 찾아봅니다.

## 정규식 사용 체크 박스

정규식 표현을 가능하게 합니다. 정규식 표현식 구문에 대한 더 자세한 내용은
[정규식 사용하기](../../howto/search/search_regexp) 를 참조하십시기 바랍니다.

## 이스케이프 시퀀스 사용 체크 박스

이스케이프 시퀀스를 가능하게 합니다. 이스케이프 시퀀스로 다음의 문자들을 사용할 수 있습니다.

|     |     |
| --- | --- |
| \\a | 경고 (벨) |
| \\b | 백 스페이스 |
| \\f | 용지 공급 |
| \\n | 새로운 줄 |
| \\t | 수평 탭 |
| \\v | 수직 탭 |
| \\\ | 백 슬래시 |
| \\oooooo | 8진수 유니코드 문자 |
| \\xhhhh | 16진수 유니코드 문자 |

Null 문자 (\\0)는 사용되지 않을 수 있습니다. \\r대신 \\n이 사용되어야만 합니다.

## 전체 단어 일치 체크 박스

단어만을 검색합니다. 단어는 이와 같은 문자( A – Z, a – z, 0 – 9, 또는 밑줄)로 시작하고
끝나는 문자열로 정의됩니다. 전자 문자로 둘러쌓인 문자열을 단어로 간주합니다. 정규식 표현이 사용되는 경우, 이 체크 박스는 제대로 작동하지
않을 수도 있습니다. 정규식 표현을 사용하는 경우, 대신 단어의 경계식 (\\<, \\\> 및 \\b)을 사용하시기 바랍니다.

## 완료 시 닫기 체크 박스

검색이 완료되면 대화 상자를 닫습니다.

## 인코딩 드롭다운 리스트 박스

인코딩을 선택합니다. **구성된 인코딩** 을 선택한 경우, 열린 파일 확장명과 연계된 인코딩이 선택됩니다.

## 모두 검색 체크 박스

사용 가능한 모든 코드 페이지로부터 통계적으로 코드 페이지를 검색합니다.
특히 파일이 매우 작은 경우, 검출 실수를 할수 있습니다. 이 옵션은 HTML/XML 문자 집합 검색을 포함하지 않습니다.

## HTML/XML 문자 집합 검색 체크 박스

HTML 파일로부터 "charset=...", XML 파일로부터 "encoding=...",
또는 Python과 Ruby 파일로부터 "#coding=..."을 찾고, 해당하는 인코딩을 사용합니다.
HTML, XML, Python, 또는 Ruby 파일을 연 경우,이 옵션만을 선택하고 다른 옵션들의 선택은 제거합니다.

## 유니코드 서명 검색 체크 박스

파일이 바이트 순서 표시 (BOM) 서명을 포함하는지 여부를 감지하고, BOM 서명을 감지한 경우,
엠에디터는 이 파일을 UTF-16LE, UTF-16BE, 또는 UTF-8 파일로 엽니다.

## UTF-8 검색 체크 박스

통계적으로 UTF-8을 검색합니다.

## 출력 옵션 드롭 다운 리스트 박스

결과를 어떻게 표시할지 선택합니다.

|     |     |
| --- | --- |
| 파일 이름과 줄 표시 | 파일 이름과 검색한 문자열을 포함한 전체 줄을 모두 결과로 표시합니다. |
| 파일 이름만 표시 | 파일 이름만이 표시되면 검색한 문자열을 포함한 전체 줄을 결과로 표시되지 않습니다. |
| 일치하는 줄만 표시 | 검색한 문자열을 포함한 전체 줄만을 결과로 표시합니다. |
| 일치하는 문자열만 표시 | 오직 일치하는 문자열만 결과로 표시합니다. |

## 출력 모음 사용 체크 박스

이 항목이 선택된 경우, 파일 내 찾기의 결과들이 출력 모음을 사용하여 표시됩니다.

## 찾기 버튼

지정된 조건으로 검색을 시작합니다.

## 바꾸기 >\> 버튼

검색할 현 문자열과 옵션을 저장하고, 현 대화 상자를 닫아 [**파일 내 바꾸기** 대화 상자](../replace_in_files/index) 를 나타내기 위해 이 버튼을 클릭합니다.

## 고급 버튼

고급 옵션을 설정할 [**고급** 대화 상자](../advanced/index) 를
나타내기 위해 이 버튼을 클릭합니다.

이 대화 상자를 통해 다음의 대화 상자 또한 사용 가능합니다.
