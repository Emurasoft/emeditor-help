# 구성 연결 대화 상자

## 파일 유형 리스트 박스

선택된 구성과 연관된 모든 파일 유형 목록입니다.
연결을 설정하여, 파일을 열거나 연결 확장자로 파일을 저장할 때 연결 구성이 자동으로 선택됩니다.

## 와일드카드 사용 라디오 버튼

선택된 파일 유형이 \\* 과 ? 같은 와이드카드를 사용하도록 지정합니다.

## 정규식 사용 라디오 버튼

선택된 파일 유형이 정규 표현식을 사용도록 지정합니다.

## 구성 드롭 박스

어느 구성이 선택된 파일 유형과 연결되는지 지정합니다.

## 새로 만들기 버튼

목록에 새로운 항목을 추가하려면 이 버튼을 클립합니다.

## 삭제 버튼

목록으로부터 선택된 항목을 삭제하려면 이 버튼을 클릭합니다.

## 위로 버튼

목록에서 선택된 항목을 위로 이동합니다.

## 아래로 버튼

목록에서 선택된 항목을 아래로 이동합니다.

## 상위 버튼

목록에서 선택된 항목을 가장 위로 이동합니다.

## 하위 버튼

목록에서 선택된 항목을 가장 아래로 이동합니다.

## 즉시 적용 버튼

설정을 지금 즉시 적용합니다.

## 엠에디터 속성과 연결

어느 파일 확장자를 엠에디터에 연결할지 지정 가능한
[엠에디터와 연결 대화 상자](../../dlg/file_associate/index) 를 나타냅니다.

이 대화 상자를 통해 다음의 대화 상자들을 사용 가능합니다.

 (엠에디터 속성과 연결 버튼 선택)
