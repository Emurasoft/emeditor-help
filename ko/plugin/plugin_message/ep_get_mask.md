# EP\_GET\_MASK

도구 모음에 플러그 인 버튼을 위한 마스크 색상을 검색합니다.

EP\_GET\_MASK

hwnd = hwndParent;

wParam = flags;

lParam = 0;

## 매개 변수

_hwndParent_

EmEditor 프레임 창의 창 핸들입니다.

_flags_

검색하려는 비트맵의 비트맵 색상 농도를 지정합니다.

|     |     |
| --- | --- |
| **값** | **의미** |
| BITMAP\_24BIT\_COLOR | 24비트 색상 (트루 컬러) 비트맵 |
| BITMAP\_256\_COLOR | 256 색상 비트맵 |

## 반환 값

도구 모음에 플러그 인 버튼을 위한 마스크 색상을 RGB(r, g, b)값으로 반환해야 합니다.
비트맵의 마스크 색상은 도구 모음의 배경 색상과 교체됩니다. 라지 비트맵의 마스크 색상은 스몰 비트맵의 마스크
색상과 동일하여야 합니다. 현재, 16 색상 비트맵의 마스크 색상은 지정할 수 없습니다. 반환 값이 0인 경우,
EmEditor는 기본 마스크 색상으로 RGB(192,192,192)를 사용합니다.
