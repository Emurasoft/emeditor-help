# EE\_DO\_IDLE

도구 모음, 창 제목, 탭 및 다른 항목들을 새로 고침합니다. 이 메시지를 명시적으로 보내거나
[Editor\_DoIdle](../macro/editor_doidle) 인라인 함수를 사용할 수 있습니다.

EE\_DO\_IDLE

wParam = (WPARAM) (BOOL) bResetTab;

lParam = (LPARAM) 0;

## 매개 변수

_bResetTab_

> 탭을 초기화 합니다.

## 반환 값

> 반환 값이 사용되지 않습니다.

## 버전

> 엠에디터 프로페셔널 버전 6.00 이상에서만 지원됩니다.
