이러한 상수는 [헤더 파일 (plugin.h)](https://download.emeditor.info/include/plugin.h) 에서 정의됩니다.

다음 Windows API 메시지 또한 지원됩니다. 그 지원 중 일부는 완벽하지 않을 수 있습니다.
메시지를 위한 창 핸드 매개 변수로 EmEditor 프레임이 아닌 보기의 창 핸들을 지정합니다.
이러한 메시지에 대한 더 많은 정보를 위해 Microsoft MSDN 라이브러리를 참고하십시오.

|     |     |
| --- | --- |
| EM\_GETSEL | 현재 선택 영역의 시작과 끝 문자 위치를 검색합니다. |
| EM\_SCROLLCARET | 보기로 커서를 스크롤합니다. |
| EM\_SETSEL | 문자의 범위를 선택합니다. |
| EM\_REPLACESEL | 지정된 텍스트의 현재 선택 영역을 교체합니다. |
| WM\_CLEAR | 현재 선택 영역을 삭제합니다. |
| WM\_COPY | 클립보드에 현재 선택 영역을 복사합니다. |
| WM\_CUT | 현재 선택 영역을 삭제하고 클립보드에 삭제된 텍스트를 복사합니다. |
| WM\_GETTEXT | 버퍼에 문서 전체를 복사합니다. |
| WM\_GETTEXTLENGTH | 종료된 null 문자를 제외한 문서 전체를 검색하기 위해 필요한 버퍼의 크기를 검색합니다. |
| WM\_PASTE | 현재 커서 위치에 클립보드의 현재 내용을 복사합니다. |
| WM\_SETTEXT | 문서 전체를 설정합니다. |
| WM\_UNDO | 마지막 작업을 실행 취소합니다. |
