# Command IDs

Command ID는 [EE\_EXEC\_COMMAND 메시지](../message/ee_exec_command), [Editor\_ExecCommand 인라인 함수](../macro/editor_execcommand), [EE\_QUERY\_STATUS 메시지](../message/ee_query_status), 및 [Editor\_QueryStatus 인라인 함수](../macro/editor_querystatus) 에 의해 사용됩니다.

Command ID는 [명령 참조](../../cmd/index) 에 설명되어 있습니다.

이러한 상수들은 [헤더 파일 (plugin.h)](https://download.emeditor.info/include/plugin.h) 에 정의되어 있습니다.
