# Highlight 속성 (SpellProp 개체)

구성 속성의 [**맞춤법** 탭](../../dlg/properties/spell/index) 의 **강조 표시된 텍스트** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.Highlight;
object.Highlight = b;
```

### \[VBScript\]

```
b = object.Highlight
object.Highlight = b
```

## 버전

엠에디터 프로페셔널 버전 10.00 이상에서만 지원됩니다.
