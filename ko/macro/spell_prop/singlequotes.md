# SingleQuotes 속성 (SpellProp 개체)

구성 속성의 [**맞춤법** 탭](../../dlg/properties/spell/index) 의 **작은 따옴표로 닫힌 문자열** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.SingleQuotes;
object.SingleQuotes = b;
```

### \[VBScript\]

```
b = object.SingleQuotes
object.SingleQuotes = b
```

## 버전

엠에디터 프로페셔널 버전 10.00 이상에서만 지원됩니다.
