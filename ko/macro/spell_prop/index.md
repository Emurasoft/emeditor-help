# SpellProp 개체

## 속성

|     |     |
| --- | --- |
| [Enabled](enabled) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>맞춤법 확인 체크 박스에 해당합니다. |
| [CamelCase](camelcase) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>CamelCase 지원 체크 박스에 해당합니다. |
| [Quote](quote) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>인용줄 체크 박스에 해당합니다. |
| [SingleQuotes](singlequotes) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>작은 따옴표로 닫힌 문자열 체크 박스에 해당합니다. |
| [DoubleQuotes](doublequotes) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>큰 따옴표로 닫힌 문자열 체크 박스에 해당합니다. |
| [Comment](comment) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>주석 체크 박스에 해당합니다. |
| [Script](script) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>스크립트 체크 박스에 해당합니다. |
| [InTag](intag) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>태그 체크 박스에 해당합니다. |
| [Highlight](highlight) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>강조 표시된 텍스트 체크 박스에 해당합니다. |
| [Hyperlink](hyperlink) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>하이퍼링크 체크 박스에 해당합니다. |
| [Rest](rest) | 구성 속성의 [맞춤법 탭](../../dlg/properties/spell/index) 의<br>지정되지 않은 텍스트 체크 박스에 해당합니다. |
