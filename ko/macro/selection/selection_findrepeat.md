# FindRepeat 메서드

같은 문자열로 이전 검색을 반복합니다.

#### \[JavaScript\]

document.selection. **FindRepeat**( _nFlags_ );

#### \[VBScript\]

document.selection. **FindRepeat** _nFlags_

## 매개 변수

_nFlags_

다음의 값들의 결합을 지정합니다:

|     |     |
| --- | --- |
| eeFindRepeatNext | 커서 위치에서 아래로 검색합니다. |
| eeFindRepeatPrevious | 커서 위치에서 위로 검색합니다. |
| eeFindRepeatWord | 선택 영역이 비어있는 경우 커서 위치에 있는 단어를 검색합니다. |

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
