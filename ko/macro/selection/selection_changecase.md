# ChangeCase 메서드 (Selection 개체)

선택된 텍스트의 대,소문자를 변경합니다.

## 

### \[JavaScript\]

```
document.selection.ChangeCase( nFlags );
```

### \[VBScript\]

```
document.selection.ChangeCase nFlags
```

## 매개 변수

_nFlags_

다음의 값들 중 하나를 지정합니다:

|     |     |
| --- | --- |
| eeCaseLowerCase | 대문자에서 소문자로 변환합니다. |
| eeCaseUpperCase | 소문자에서 대문자로 변환합니다. |
| eeCaseCapitalize | 각 단어의 첫번째 글자를 대문자로 바꿉니다. (엠에디터 프로페셔널 버전 4.10 이상에서만 지원됩니다.) |

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
