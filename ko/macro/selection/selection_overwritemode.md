# OverwriteMode 속성 (Selection 개체)

덮어쓰기 또는 삽입 모드를 나타내는 플래그를 설정하거나 검색합니다.

## 

### \[JavaScript\]

```
bOverwrite = document.selection.OverwriteMode;
document.selection.OverwriteMode = bOverwrite;
```

### \[VBScript\]

```
bOverwrite = document.selection.OverwriteMode
document.selection.OverwriteMode = bOverwrite
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
