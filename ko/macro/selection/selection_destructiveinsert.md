# DestructiveInsert 메서드 (Selection 개체)

기존의 텍스트에 덮어쓰기하여 텍스트를 삽입합니다.

## 

### \[JavaScript\]

```
document.selection.DestructiveInsert( str );
```

### \[VBScript\]

```
document.selection.DestructiveInsert str
```

## 매개 변수

_str_

기존의 텍스트에 덮어쓰기하여 삽입할 문자열을 지정합니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
