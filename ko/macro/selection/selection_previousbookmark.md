# PreviousBookmark 메서드 (Selection 개체)

문서 내에서 이전 책갈피로 이동합니다.

## 

### \[JavaScript\]

```
bFound = document.selection.PreviousBookmark();
```

### \[VBScript\]

```
bFound = document.selection.PreviousBookmark
```

## 반환 값

이전 책갈피를 발견하면 true를 반환하고, 또는 발견하지 못하면 false를 반환합니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
