# PreserveReturnsClipboard 속성 (GeneralProp 개체)

구성 속성 [**일반** 탭](../../dlg/properties/general/index) 의 **(CR/LF) 클립보드 반환 유지** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.PreserveReturnsClipboard;
object.PreserveReturnsClipboard = b;
```

### \[VBScript\]

```
b = object.PreserveReturnsClipboard
object.PreserveReturnsClipboard = b
```
