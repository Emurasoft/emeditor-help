# PasteAnsi 속성 (GeneralProp 개체)

구성 속성 [**일반** 탭](../../dlg/properties/general/index) 의 **항상 ANSI로 붙여 넣기** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.PasteAnsi;
object.PasteAnsi = b;
```

### \[VBScript\]

```
b = object.PasteAnsi
object.PasteAnsi = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
