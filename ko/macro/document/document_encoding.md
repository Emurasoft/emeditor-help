# Encoding 속성

열린 파일의 현재 인코딩을 검색하거나 설정합니다.

#### \[JavaScript\]

_nEncoding_ = document. **Encoding**;

document. **Encoding** = _nEncoding_;

#### \[VBScript\]

_nEncoding_ = document. **Encoding**

document. **Encoding** = _nEncoding_

## 매개 변수

_nEncoding_

**[인코딩 상수](../const/const_encoding)** 로부터
선택하거나 Windows 운영 체제에서 사용되는 코드 페이지를 지정합니다.

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
