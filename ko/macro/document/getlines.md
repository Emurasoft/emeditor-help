# GetLines 메서드 (Document 개체)

문서의 줄 수를 검색합니다.

## 

### \[JavaScript\]

```
nLines = document.GetLines( [ nFlags ] );
```

### \[VBScript\]

```
nLines = document.GetLines( [ nFlags ] )
```

## 매개 변수

_nFlags_

선택 사항입니다. 다음의 값을 지정할 수 있습니다.
아무런 값이 지정되지 않은 경우, 논리적 좌표가 지정됩니다.

|     |     |
| --- | --- |
| eeGetLineView | 좌표가 보이게 지정됩니다. |

## 버전
