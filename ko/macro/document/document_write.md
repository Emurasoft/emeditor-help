# write 메서드 (Document 개체)

현재 커서 위치에 문자열을 삽입하거나 덮어쓰기합니다.

## 

### \[JavaScript\]

```
document.write( str );
```

### \[VBScript\]

```
document.write str
```

## 매개 변수

_str_

삽입하거나 덮어쓰기하려는 문자열을 지정합니다.

## 예시

### \[JavaScript\]

```
document.write( "Hello" );
```

### \[VBScript\]

```
document.write "Hello"
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
