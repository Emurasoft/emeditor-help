# Name 속성 (Document 개체)

현재 문서의 파일 이름만을 검색합니다.

## 

### \[JavaScript\]

```
strName = document.Name;
```

### \[VBScript\]

```
strName = document.Name
```

## 예시

### \[JavaScript\]

```
alert( document.Name );
```

### \[VBScript\]

```
alert document.Name
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
