# HighlightTag 속성

태그를 강조 표시 할자의 여부를 결정합니다.

#### \[JavaScript\]

_bHighlightTag_ = document. **HighlightTag**;

document. **HighlightTag** = _bHighlightTag_;

#### \[VBScript\]

_bHighlightTag_ = document. **HighlightTag**

document. **HighlightTag** = _bHighlightTag_

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
