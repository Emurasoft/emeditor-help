# FontCategory 속성

현재 글꼴 범주를 검색하거나 설정합니다.

#### \[JavaScript\]

_nCategory_ = document. **FontCategory**;

document. **FontCategory** = _nCategory_;

#### \[VBScript\]

_nCategory_ = document. **FontCategory**

document. **FontCategory** = _nCategory_

## 매개 변수

_nCategory_

다음의 값에서 선택합니다.

|     |     |
| --- | --- |
| eeFontCategoryDefault | 표준 |
| eeFontCategoryArabic | 아랍어 |
| eeFontCategoryBaltic | 발트어 |
| eeFontCategoryCentralEuropean | 중앙 유럽어 |
| eeFontCategoryChineseSimplified | 중국어 간체 |
| eeFontCategoryChineseTraditional | 중국어 번체 |
| eeFontCategoryCyrillic | 키릴어 |
| eeFontCategoryGreek | 그리스어 |
| eeFontCategoryHebrew | 히브리어 |
| eeFontCategoryJapanese | 일본어 |
| eeFontCategoryKorean | 한국어 |
| eeFontCategoryThai | 태국어 |
| eeFontCategoryTurkish | 터키어 |
| eeFontCategoryVietnamese | 베트남어 |
| eeFontCategoryWesternEuropean | 서유럽어 |
| eeFontCategoryOEM | OEM/DOS |
