# writeln 메서드 (Document 개체)

현재 커서 위치에 문자열과 새 줄을 삽입하거나 덮어쓰기합니다.

## 

### \[JavaScript\]

```
document.writeln( str );
```

### \[VBScript\]

```
document.writeln str
```

## 매개 변수

_str_

삽입 또는 덮어 쓰기 할 문자열을 지정합니다.

## 예시

### \[JavaScript\]

```
document.writeln( "Hello" );
```

### \[VBScript\]

```
document.writeln "Hello"
```

## 버전

엠에디터 프로페셔널 버전 4.00 이상에서만 지원됩니다.
