# 키 입력과 마우스 작업 녹음과 재생 (기능)

엠에디터는 다른 응용 프로그램을 대상으로 키 입력과 마우스 작업을
기록하고 실행할 수 있습니다.
이 기능은 특정한 작업을 데스크탑, 브라우저, 또는 사용자가 원하는 응용 프로그램에 자동화 할 수 있습니다.
개발자의 경우, 이 기능을 응용 프로그램을 테스트 하는데 사용할 수 있습니다.
[Window 개체](../window/index) 는 창에 대한 대부분에 대해 알수있는
모든 종류의 속성을 포함하고, [Windows 컬렉션](../windows/index) 은
특정 창에 대한 최상위 창 또는 하위 창을 열거하는데 사용될 수 있습니다.
[Shell 개체](../shell/index) 는 키 입력과 마우스 활동을 보내고
최상위 창을 열거하거나 찾습니다.
