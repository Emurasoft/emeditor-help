# Windows 스크립트 호스트

엠에디터 매크로는 Windows 스크립트 호스트 (WSH) 엔진을 기반으로 하므로,
다양한 Windows 구성 요소 개체 모델 (COM) 개체를 사용할 수 있습니다.
예를 들어, RegExp 개체를 사용하여 정규 표현식 검색, FileSystemObject 개체를 사용하여
파일 조작, 바로가기 생성, WshShell 개체를 사용하여 Windows 레지스트리 조작, 또는
WshNetwork 개체를 사용하여 네트워킹 기능 작업을 수행할 수 있습니다.
더 나아가, 엠에디터에서 생성된 문서를 복사한 후에 외부 응용 프로그램에 붙여넣고 인쇄하기 위해
(Word와 Excel과 같은) 자동화를 지원하는 외부 응용 프로그램을 활용하는 매크로를 생성하고 실행할 수 있습니다.
