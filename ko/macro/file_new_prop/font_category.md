# FontCategory 속성 (FileNewProp 개체)

[**새 파일 상세 정보** 대화 상자](../../dlg/properties/file/new_details/index) 의
**글꼴 범주 드롭 다운** 리스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.FontCategory;
object.FontCategory = n;
```

### \[VBScript\]

```
n = object.FontCategory
object.FontCategory = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
