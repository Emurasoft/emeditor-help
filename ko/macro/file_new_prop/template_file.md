# TemplateFile 속성 (FileNewProp 개체)

[**새 파일 상세 정보** 대화 상자](../../dlg/properties/file/new_details/index) 의
**서식 파일** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
s = object.TemplateFile;
object.TemplateFile = s;
```

### \[VBScript\]

```
s = object.TemplateFile
object.TemplateFile = s
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
