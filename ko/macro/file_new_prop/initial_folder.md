# InitialFolder 속성 (FileNewProp 개체)

[새 파일 상세 정보 대화 상자](../../dlg/properties/file/new_details/index) 의
초기 폴더 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
s = object.InitialFolder;
object.InitialFolder = s;
```

### \[VBScript\]

```
s = object.InitialFolder
object.InitialFolder = s
```

## 버전

EmEditor Professional 버전 14.1 이상에서만 지원됩니다.
