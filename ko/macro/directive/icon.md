# \#icon 지침서

도구 모음 버튼을 사용할 아이콘을 지정합니다. 이 지침서는 메인 기본 코드 위 스크립트의 첫 줄에 지정되어야 합니다.

#icon = " _filename_", _index_

## 매개 변수

_filename_

> 현재 매크로 파일 또는 전체 경로로부터 상대 경로와 아이콘을 추출하는 파일을 지정합니다. 경로가 생략된 경우,
> 현재 매크로 파일이 존재하는 파일의 제목을 지정합니다.

_index_

> 지정된 파일에 여러 개의 아이콘이 존재하는 경우에는 사용할 인덱스를 지정합니다.
> 지정이 생략된 경우엔 0이 선택됩니다.

## 예시

계산기(calc.exe)의 아이콘을 사용합니다.

#icon = "C:\\WINDOWS\\system32\\calc.exe",0

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
