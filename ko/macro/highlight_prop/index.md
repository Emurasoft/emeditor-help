# HighlightProp 개체

## 속성

|     |     |
| --- | --- |
| [CommentBegin](comment_begin) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>설명 그룹 박스 \- 시작 텍스트 박스에 해당합니다. |
| [CommentEnd](comment_end) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>설명 그룹 박스 \- 종료 텍스트 박스에 해당합니다. |
| [Enabled](enabled) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>이 단어들을 강조 표시 체크 박스에 해당합니다. |
| [EscapeCharacter](escape_character) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>따옴표로 닫힌 문자열 그룹 박스 \- 이스케이프 문자 텍스트 박스에 해당합니다. |
| [HighlightCharRef](highlight_char_ref) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>HTML/XML 문자 참조와 UCN에 강조 표시 체크 박스에 해당합니다. |
| [HighlightMatchingBrackets](highlight_matching_brackets) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>표시 일치 괄호 강조하기 체크 박스에 해당합니다. |
| [LineComment1](line_comment1) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>설명 그룹 박스 \- 줄 설명 텍스트 박스에 해당합니다. |
| [LineComment2](line_comment2) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>설명 그룹 박스 \- 줄 설명 체크 박스에 해당합니다. |
| [List](list) | [HighlightList 컬렉션](../highlight_list/index) 을 검색합니다. |
| [QuoteContinue](quote_continue) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>따옴표로 닫힌 문자열 그룹 박스 \- 다음 줄로 계속하기 체크 박스에 해당합니다. |
| [QuoteDouble](quote_double) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>따옴표로 닫힌 문자열 그룹 박스 \- 큰 따옴표 "..." 체크 박스에 해당합니다. |
| [QuoteSingle](quote_single) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>따옴표로 닫힌 문자열 그룹 박스 \- 작은 따옴표 '...' 체크 박스에 해당합니다. |
| [ScriptBegin](script_begin) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>스크립트 그룹 박스 \- 시작 텍스트 박스에 해당합니다. |
| [ScriptEnd](script_end) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>스크립트 그룹 박스 \- 종료 텍스트 박스에 해당합니다. |
| [SpecialSyntax](special_syntax) | 구성 속성 [강조 표시 (2) 탭](../../dlg/properties/highlight2/index) 의<br>특수 구문 드롭 다운 리스트 박스에 해당합니다. |
| [TagBegin](tag_begin) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>태그 시작 텍스트 박스에 해당합니다. |
| [TagEnd](tag_end) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>태그 종료 텍스트 박스에 해당합니다. |
