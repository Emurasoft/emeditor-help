# QuoteContinue 속성

구성 속성 [**강조 표시 (2)** 탭](../../dlg/properties/highlight2/index) 의 **따옴표로 닫힌 문자열 그룹 박스 \- 다음 줄로 계속하기** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **QuoteContinue**;

object. **QuoteContinue** = _b_;

#### \[VBScript\]

_b_ =
object. **QuoteContinue**

object. **QuoteContinue** = _b_

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
