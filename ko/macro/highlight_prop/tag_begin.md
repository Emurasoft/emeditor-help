# TagBegin 속성 (HighlightProp 개체)

구성 속성 [**강조 표시 (1)** 탭](../../dlg/properties/highlight1/index) 의 **태그 시작** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
str = object.TagBegin;
object.TagBegin = str;
```

### \[VBScript\]

```
str = object.TagBegin
object.TagBegin = str
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
