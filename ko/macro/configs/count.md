# Count 속성 (Configs 컬렉션)

구성의 수를 검색합니다.

## 

### \[JavaScript\]

```
nCount = editor.Configs.Count;
```

### \[VBScript\]

```
nCount = editor.Configs.Count
```

## 예시

### \[JavaScript\]

```
alert( "Number of Configurations: " + editor.Configs.Count );
```

### \[VBScript\]

```
alert "Number of Configurations: " & editor.Configs.Count
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
