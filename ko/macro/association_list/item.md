# Item 속성 (AssociationList 컬렉션)

지정된 인덱스의 [AssociationItem 개체](../association_item/index) 를 검사합니다.

## 

### \[JavaScript\]

```
item = list.Item( Index );
```

### \[VBScript\]

```
item = list.Item( Index )
```

## 매개 변수

_Index_

1부터 시작하는 정수로된 항목의 인덱스를 지정합니다.

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
