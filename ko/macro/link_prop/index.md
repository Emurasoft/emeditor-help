# LinkProp 개체

## 속성

|     |     |
| --- | --- |
| [DBCharURL](dbc_har_url) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **더블 바이트 문자를 URL로 인식** 체크 박스에 해당합니다. |
| [DoubleClickOnly](double_click_only) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **더블 클릭 만을 활성화** 체크 박스에 해당합니다. |
| [KanaURL](kana_url) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **싱글 바이트 가나 및 가나 표시를 URL로 인식** 체크 박스에 해당합니다. |
| [MailTo](mail_to) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **메일을 전송하려면 메일 주소를 클릭** 체크 박스에 해당합니다. |
| [OpenFileInQuotes](open_file_in_quotes) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **따옴표 안의 파일 이름을 클릭하여 파일 열기** 체크 박스에 해당합니다. |
| [SelectInQuotes](select_in_quotes) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **인용문을 클릭하여 전체 문자열을 선택** 체크 박스에 해당합니다. |
| [URL](url) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **URL 링크** 체크 박스에 해당합니다. |
| [URLCharacters](url_characters) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **이 문자들을 인식** 텍스트 박스에 해당합니다. |
| [URLCharactersNotEnd](url_characters_not_end) | 구성 속성 [**링크** 탭](../../dlg/properties/link/index) 의<br> **끝에서 문자 허용 안됨** 텍스트 박스에 해당합니다. |
