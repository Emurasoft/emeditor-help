# DoubleClickOnly 속성 (LinkProp 개체)

구성 속성 [링크 탭](../../dlg/properties/link/index) 의더블 클릭 만을 활성화 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DoubleClickOnly;
object.DoubleClickOnly = b;
```

### \[VBScript\]

```
b = object.DoubleClickOnly
object.DoubleClickOnly = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
