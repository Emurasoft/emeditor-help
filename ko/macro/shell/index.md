# Shell 개체

## 속성

|     |     |
| --- | --- |
| **[ForegroundWindow](foreground_window)** | 현재 전경 창을 검색합니다. |
| **[windows](windows)** | 최상위 Windows 컬렉션을 반환합니다. |

## 메서드

|     |     |
| --- | --- |
| **[FindWindow](find_window)** | 클래스 이름 및/또는 창 제목으로 최상위 Window 개체를 검색합니다. |
| **[SendKeys](send_keys)** | 하나 이상의 키 입력 (또는 마우스 활동)을 활성 창에 전달합니다. |

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
