# SmoothSpeed 속성 (ScrollProp 개체)

구성 속성 [스크롤 탭](../../dlg/properties/scroll/index) 의부드럽게 스크롤 슬라이더 에 해당합니다.

## 

### \[JavaScript\]

```
n = object.SmoothSpeed;
object.SmoothSpeed = n;
```

### \[VBScript\]

```
n = object.SmoothSpeed
object.SmoothSpeed = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
