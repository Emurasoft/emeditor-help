# ScrollProp 개체

## 속성

|     |     |
| --- | --- |
| [DoubleLineScroll](double_line_scroll) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **이중선 스크롤** 체크 박스에 해당합니다. |
| [FastCursorMove](fast_cursor_move) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **빠른 커서 움직임** 체크 박스에 해당합니다. |
| [HalfPage](half_page) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **1/2 페이지 스크롤** 체크 박스에 해당합니다. |
| [HorizontalBar](horizontal_bar) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **가로 스크롤 바 드롭 다운** 리스트 박스에 해당합니다. |
| [HorizontalGrid](horizontal_grid) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **가로 눈금** 체크 박스에 해당합니다. |
| [HorizontalLine](horizontal_line) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **가로 줄** 체크 박스에 해당합니다. |
| [HorizontalPage](horizontal_page) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **항상 1 페이지 가로 스크롤 활성화** 체크 박스에 해당합니다. |
| [MoveCursorScroll](move_cursor_scroll) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **스크롤링으로 커서 이동** 체크 박스에 해당합니다. |
| [ShowOnlyActive](show_only_active) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **활성 창에서만 스크롤 바 보이기** 체크 박스에 해당합니다. |
| [Smooth](smooth) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **부드럽게 스크롤** 체크 박스에 해당합니다. |
| [SmoothSpeed](smooth_speed) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **부드럽게 스크롤 슬라이더** 에 해당합니다. |
| [VerticalBar](vertical_bar) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **세로 스크롤 바 드롭 다운** 리스트 박스에 해당합니다. |
| [VerticalLine](vertical_line) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **세로 줄** 체크 박스에 해당합니다. |
| [VerticalPage](vertical_page) | 구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의<br> **항상 1 페이지 세로 스크롤 활성화** 체크 박스에 해당합니다. |
