# HorizontalGrid 속성

구성 속성 [**스크롤** 탭](../../dlg/properties/scroll/index) 의 **가로 눈금** 체크 박스에 해당합니다.

#### \[JavaScript\]

_b_ =
object. **HorizontalGrid**;

object. **HorizontalGrid** = _b_;

#### \[VBScript\]

_b_ =
object. **HorizontalGrid**

object. **HorizontalGrid** = _b_

## 버전

엠에디터 프로페셔널 버전 10.00 이상에서만 지원됩니다.
