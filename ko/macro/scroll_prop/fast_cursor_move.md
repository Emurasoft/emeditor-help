# FastCursorMove 속성 (ScrollProp 개체)

구성 속성 [스크롤 탭](../../dlg/properties/scroll/index) 의빠른 커서 움직임 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.FastCursorMove;
object.FastCursorMove = b;
```

### \[VBScript\]

```
b = object.FastCursorMove
object.FastCursorMove = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
