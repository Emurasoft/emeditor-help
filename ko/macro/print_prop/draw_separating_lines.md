# DrawSeparatingLines 속성 (PrintProp 개체)

구성 속성 [인쇄 탭](../../dlg/properties/print/index) 의구분 선 그리기 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DrawSeparatingLines;
object.DrawSeparatingLines = b;
```

### \[VBScript\]

```
b = object.DrawSeparatingLines
object.DrawSeparatingLines = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
