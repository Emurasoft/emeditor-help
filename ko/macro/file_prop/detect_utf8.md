# DetectUTF8 속성 (FileProp 개체)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **UTF-8 검색** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DetectUTF8;
object.DetectUTF8 = b;
```

### \[VBScript\]

```
b = object.DetectUTF8
object.DetectUTF8 = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
