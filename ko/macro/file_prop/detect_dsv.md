# DetectDSV 속성 (FileProp 개체)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **DSV 검색 (사용자 정의된 구분 기호 구분)** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DetectDSV;
object.DetectDSV = b;
```

### \[VBScript\]

```
b = object.DetectDSV
object.DetectDSV = b
```
