# PromptNull 속성 (FileProp 개체)

구성 속성 [**파일** 탭](../../dlg/properties/file/index) 의 **NULL 문자 발견시 확인** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.PromptNull;
object.PromptNull = b;
```

### \[VBScript\]

```
b = object.PromptNull
object.PromptNull = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
