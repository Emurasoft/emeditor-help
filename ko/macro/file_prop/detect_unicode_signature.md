# DetectUnicodeSignature 속성 (FileProp 개체)

구성 속성 [파일 탭](../../dlg/properties/file/index) 의유니코드 서명 검색 (BOM) 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DetectUnicodeSignature;
object.DetectUnicodeSignature = b;
```

### \[VBScript\]

```
b = object.DetectUnicodeSignature
object.DetectUnicodeSignature = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
