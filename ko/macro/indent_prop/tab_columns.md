# TabColumns 속성 (IndentProp 개체)

[**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의 **탭 열** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.TabColumns;
object.TabColumns = n;
```

### \[VBScript\]

```
n = object.TabColumns
object.TabColumns = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
