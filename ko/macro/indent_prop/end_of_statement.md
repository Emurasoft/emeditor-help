# EndOfStatement 속성

[**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의 **설명문의 끝** 텍스트 박스에 해당합니다.

#### \[JavaScript\]

_s_ = object. **EndOfStatement**;

object. **EndOfStatement** = _s_;

#### \[VBScript\]

_s_ = object. **EndOfStatement**

object. **EndOfStatement** = _s_
