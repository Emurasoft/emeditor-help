# IndentProp 개체

## 속성

|     |     |
| --- | --- |
| [AutoIndent](auto_indent) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **자동 들여쓰기 활성화** 체크 박스에 해당합니다. |
| [Begin](begin) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **들여쓰기 시작** 텍스트 박스에 해당합니다. |
| [End](end) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **들여쓰기 끝** 텍스트 박스에 해당합니다. |
| [EndOfStatement](end_of_statement) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **설명문의 끝** 텍스트 박스에 해당합니다. |
| [IgnoreAfter](ignore_after) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **(정규식) 다음 무시** 텍스트 박스에 해당합니다. |
| [IndentColumns](indent_columns) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **들여쓰기 열** 텍스트 박스에 해당합니다. |
| [InsertSpaces](insert_spaces) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **탭을 위한 공백 삽입** 체크 박스에 해당합니다. |
| [RegExp](reg_exp) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **정규식 사용** 체크 박스에 해당합니다. |
| [TabColumns](tab_columns) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **탭 열** 텍스트 박스에 해당합니다. |
| [WrapIndent](wrap_indent) | [**탭/들여쓰기** 대화 상자](../../dlg/properties/general/indent/index) 의<br> **줄 바꿔서 들여쓰기** 체크 박스에 해당합니다. |
