# ShowReturns 속성 (MarkProp 개체)

구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의 **반환 보이기** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.ShowReturns;
object.ShowReturns = b;
```

### \[VBScript\]

```
b = object.ShowReturns
object.ShowReturns = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
