# MarkProp 개체

## 속성

|     |     |
| --- | --- |
| [CrLfDifferent](cr_lf_different) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **CR과 LF를 다른 표시로 보이기** 체크 박스에 해당합니다. |
| [ShowControlCharacters](show_control_characters) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **컨트롤 문자 보이기** 체크 박스에 해당합니다. |
| [ShowDBSpaces](show_db_spaces) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **넓은 공백 보이기** 체크 박스에 해당합니다. |
| [ShowEOF](show_eof) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **파일의 끝 보이기** 체크 박스에 해당합니다. |
| [ShowIndentGuides](show_indent_guides) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **들여쓰기 가이드 보이기** 체크 박스에 해당합니다. |
| [ShowReturns](show_returns) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **반환 보이기** 체크 박스에 해당합니다. |
| [ShowSpaces](show_spaces) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **공백 보이기** 체크 박스에 해당합니다. |
| [ShowTabs](show_tabs) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의<br> **탭 보이기** 체크 박스에 해당합니다. |
| [ShowUserDefinedGuides](show_user_defined_guides) | 구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의 **사용자 지정 가이드 보이기** 체크 박스에 해당합니다. |
