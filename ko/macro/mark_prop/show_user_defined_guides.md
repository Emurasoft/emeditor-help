# ShowUserDefinedGuides 속성 (MarkProp 개체)

구성 속성 [**표시** 탭](../../dlg/properties/marks/index) 의
**사용자 지정 가이드 보이기** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.ShowUserDefinedGuides;
object.ShowUserDefinedGuides = b;
```

### \[VBScript\]

```
b = object.ShowUserDefinedGuides
object.ShowUserDefinedGuides = b
```

## 버전

EmEditor Professional 버전 14.2.0 이상에서만 지원됩니다.
