# FontItem 개체

## 속성

|     |     |
| --- | --- |
| [Bold](bold) | 글꼴이 진하게 포맷된 경우 트루입니다. |
| [Italic](italic) | 글꼴이 기울게 포맷된 경우 트루입니다. |
| [Name](name) | 개체의 이름을 지정합니다. |
| [Script](script) | 글꼴의 스크립트를 지정합니다. |
| [Size](size) | 글꼴의 크기를 포인트로 지정합니다. |

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.


```{toctree}
:maxdepth: 1
bold
italic
name
script
size
```
