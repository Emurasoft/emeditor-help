# Reset 메서드 (Config 개체)

현재 구성을 초기화합니다.

## 

### \[JavaScript\]

```
object.Reset( [ strName ] );
```

### \[VBScript\]

```
object.Reset [ strName ]
```

## 매개 변수

_strName_

선택 사항입니다. 초기화할 기본 구성의 이름을 지정합니다. 생략된 경우, 기본 구성의 현재 이름이 사용됩니다. 이름이 기본 구성 중 하나가
아닌 경우 오류가 발생합니다.

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
