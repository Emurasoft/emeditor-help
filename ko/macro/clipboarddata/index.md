# clipboardData 개체

## 메서드

|     |     |
| --- | --- |
| **[clearData](cleardata)** | 클립보드로부터 하나 이상의 데이터 형식을 삭제합니다. |
| **[getData](getdata)** | 클립보드로부터 지정된 형식 내에서 데이터를 검색합니다. |
| **[setData](setdata)** | 클립보드에 지정된 형식 내 데이터를 할당합니다. |

## 버전

엠에디터 프로페셔널 버전 5.00 이상에서만 지원됩니다.
