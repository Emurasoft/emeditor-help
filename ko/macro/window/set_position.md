# SetPosition 메서드 (Window 개체)

창의 위치와 크기를 설정합니다.

## 

### \[JavaScript\]

```
wnd.SetPosition( x, y, width, height );
```

### \[VBScript\]

```
wnd.SetPosition x, y, width, height
```

## 매개 변수

_x_

창의 수평 위치를 픽셀로 지정합니다.

_y_

창의 수직 위치를 픽셀로 지정합니다.

_width_

창의 너비를 픽셀로 지정합니다.

_height_

창의 높이를 픽셀로 지정합니다.
