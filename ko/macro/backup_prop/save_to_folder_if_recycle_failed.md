# SaveToFolderIfRecycleFailed 속성 (BackupProp 개체)

구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **휴지통 이용이 불가능 할 경우 백업 폴더에 저장** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.SaveToFolderIfRecycleFailed;
object.SaveToFolderIfRecycleFailed = b;
```

### \[VBScript\]

```
b = object.SaveToFolderIfRecycleFailed
object.SaveToFolderIfRecycleFailed = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
