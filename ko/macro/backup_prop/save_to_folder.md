# SaveToFolder 속성 (BackupProp 개체)

구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **백업 폴더에 백업 저장** 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.SaveToFolder;
object.SaveToFolder = b;
```

### \[VBScript\]

```
b = object.SaveToFolder
object.SaveToFolder = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
