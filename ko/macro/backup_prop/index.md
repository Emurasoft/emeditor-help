# BackupProp 개체

## 속성

|     |     |
| --- | --- |
| [Folder](folder) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **백업 폴더** 텍스트 박스에 해당합니다. |
| [RenameIfSameNameExists](rename_if_same_name_exists) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **같은 파일 이름이 존재할 경우 이름 바꾸기** 체크 박스에 해당합니다. |
| [SaveToFolder](save_to_folder) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **백업 폴더에 백업 저장** 체크 박스에 해당합니다. |
| [SaveToFolderIfRecycleFailed](save_to_folder_if_recycle_failed) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **휴지통 이용이 불가능 할 경우 백업 폴더에 저장** 체크 박스에 해당합니다. |
| [SaveToSameFolder](save_to_same_folder) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **같은 폴더에 백업 저장** 체크 박스에 해당합니다. |
| [UseRecycleBin](use_recycle_bin) | 구성 속성 [**백업** 탭](../../dlg/properties/backup/index) 의 **백업을 위하여 휴지통 사용** 체크 박스에 해당합니다. |
