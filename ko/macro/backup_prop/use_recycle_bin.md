# UseRecycleBin 속성 (BackupProp 개체)

구성 속성 [백업 탭](../../dlg/properties/backup/index) 의백업을 위하여 휴지통 사용 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.UseRecycleBin;
object.UseRecycleBin = b;
```

### \[VBScript\]

```
b = object.UseRecycleBin
object.UseRecycleBin = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
