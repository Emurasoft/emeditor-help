# Track 메서드 (PopupMenu 개체)

팝업 메뉴를 표시합니다.

## 

### \[JavaScript\]

```
popupmenu.Track( flags );
```

### \[VBScript\]

```
popupmenu.Track flags
```

## 매개 변수

_flags_

메뉴가 어느 곳에 표시될지 지정합니다. 다음의 값이 지정될 수 있습니다.
지정되지 않는 경우, 현재 커서 위치에 팝업 메뉴가 표시됩니다.

|     |     |
| --- | --- |
| eePosMouse | 현재 마우스 포인터 위치에 팝업 메뉴를 표시합니다. |

## 버전

엠에디터 프로페셔널 버전 5.00 이상에서만 지원됩니다.
