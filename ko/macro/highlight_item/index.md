# HighlightItem 개체

## 속성

|     |     |
| --- | --- |
| [Case](case) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>대/소문자 구분 체크 박스에 해당합니다. |
| [Color](color) | 개체의 색상을 지정합니다. |
| [InsideTag](inside_tag) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>내부 태그 만 체크 박스에 해당합니다. |
| [Name](name) | 개체의 이름을 지정합니다. |
| [RegExp](reg_exp) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>정규식 체크 박스에 해당합니다. |
| [RightAll](right_all) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>오른쪽 모두 강조 표시 체크 박스에 해당합니다. |
| [RightSide](right_side) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>오른쪽 강조 표시 체크 박스에 해당합니다. |
| [Word](word) | 구성 속성 [강조 표시 (1) 탭](../../dlg/properties/highlight1/index) 의<br>전체 단어 만 체크 박스에 해당합니다. |
