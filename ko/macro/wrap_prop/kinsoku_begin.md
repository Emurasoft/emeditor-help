# KinsokuBegin 속성 (WrapProp 개체)

구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의 **줄 시작 점에서 불가능** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
str = object.KinsokuBegin;
object.KinsokuBegin = str;
```

### \[VBScript\]

```
str = object.KinsokuBegin
object.KinsokuBegin = str
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
