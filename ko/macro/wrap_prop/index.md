# WrapProp 개체

## 속성

|     |     |
| --- | --- |
| [Kinsoku](kinsoku) | 구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의<br> **이 문자들 배치** 체크 박스에 해당합니다. |
| [KinsokuBegin](kinsoku_begin) | 구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의<br> **줄 시작 점에서 불가능** 텍스트 박스에 해당합니다. |
| [KinsokuEnd](kinsoku_end) | 구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의<br> **줄 끝 점에서 불가능** 텍스트 박스에 해당합니다. |
| [WordWrap](word_wrap) | 구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의<br> **단어 배치** 체크 박스에 해당합니다. |
| [WordWrapAfterMarks](word_wrap_after_marks) | 구성 속성의 [**배치** 탭](../../dlg/properties/wrap/index) 의<br> **표시 후 단어 배치 활성화** 체크 박스에 해당합니다. |
