# FontProp 개체

## 속성

|     |     |
| --- | --- |
| [DisplayList](display_list) | FontList 개체를 검색합니다. |
| [PrintList](print_list) | PrintList 개체를 검색합니다. |
| [SameFontPrint](same_font_print) | 글꼴 사용자 지정 대화 상자의 **글꼴 표시를 프린터 글꼴로 사용** 체크 박스에 해당합니다. |
| [UsePrinterFont](use_printer_font) | 글꼴 사용자 지정 대화 상자의 **아래 목록에 기본 프린터 글꼴 포함** 체크 박스에 해당합니다. |

## 버전
