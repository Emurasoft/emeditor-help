# UsePrinterFont 속성 (FontProp 개체)

[**글꼴 사용자 지정** 대화 상자의](../../dlg/properties/font/index) 아래 목록에 기본 프린터 글꼴 포함 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.UsePrinterFont;
object.UsePrinterFont = b;
```

### \[VBScript\]

```
b = object.UsePrinterFont
object.UsePrinterFont = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
