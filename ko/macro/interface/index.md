# Interface 개체

## 속성

|     |     |
| --- | --- |
| **[x](x)** | 지원되는 플러그 인에 의해 전달된 커서의 수평 위치를 반환합니다. |
| **[y](y)** | 지원되는 플러그 인에 의해 전달된 커서의 수직 위치를 반환합니다. |

## 메서드

|     |     |
| --- | --- |
| **[write](write)** | 지원되는 플러그 인에 문자열을 전달합니다. |
| **[writeln](writeln)** | 지원되는 플러그 인에 문자열과 새로운 줄을 전달합니다. |

## 버전

엠에디터 프로페셔널 버전 9.00 이상에서만 지원됩니다.
