# DeleteEmpty 속성 (FileSaveProp 개체)

[세부 정보 저장 대화 상자](../../dlg/properties/file/save_details/index) 의
저장 시 비어있는 파일 삭제 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.DeleteEmpty;
object.DeleteEmpty = b;
```

### \[VBScript\]

```
b = object.DeleteEmpty
object.DeleteEmpty = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
