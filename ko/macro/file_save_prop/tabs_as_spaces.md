# TabsAsSpaces 속성 (FileSaveProp 개체)

[세부 정보 저장 대화 상자](../../dlg/properties/file/save_details/index) 의
탭을 공백으로 저장 체크 박스에 해당합니다.

## 

### \[JavaScript\]

```
b = object.TabsAsSpaces;
object.TabsAsSpaces = b;
```

### \[VBScript\]

```
b = object.TabsAsSpaces
object.TabsAsSpaces = b
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
