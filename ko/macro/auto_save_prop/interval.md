# Interval 속성 (AutoSaveProp 개체)

구성 속성 [**자동 저장** 탭](../../dlg/properties/autosave/index) 의 **시간(분)** 텍스트 박스에 해당합니다.

## 

### \[JavaScript\]

```
n = object.Interval;
object.Interval = n;
```

### \[VBScript\]

```
n = object.Interval
object.Interval = n
```

## 버전

엠에디터 프로페셔널 버전 7.00 이상에서만 지원됩니다.
