# GetUnicodeName 方法 (Editor 对象)

检索指定字符或字符串的 Unicode 名。

## 

### \[JavaScript\]

```
strName = editor.GetUnicodeName( strChar );
```

### \[VBScript\]

```
strName = editor.GetUnicodeName( strChar )
```

## 参数

_strChar_

指定源字符或字符串。

## 返回值

返回值时指定的字符或字符串的 Unicode 名。

## 版本

支持 EmEditor Professional v19.1 或之后的版本。
