# EnableTab 属性

设置或检索是否标签页被启用。所有的文档可能无法立即使用在标签页被启用或禁用后。因此，在标签页被启用或禁用后，可能有必要先退出宏再运行该宏。

#### \[JavaScript\]

_bEnableTab_ = editor. **EnableTab**;

editor. **EnableTab** = _bEnableTab_;

#### \[VBScript\]

_bEnableTab_ = editor. **EnableTab**

editor. **EnableTab** = _bEnableTab_

## 版本

支持 EmEditor 5.00 或之后的版本。
