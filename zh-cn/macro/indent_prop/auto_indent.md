# AutoIndent 属性 (IndentProp 对象)

与 [**制表符/缩进** 对话框](../../dlg/properties/general/indent/index) 上的 **启用自动缩进** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.AutoIndent;
object.AutoIndent = b;
```

### \[VBScript\]

```
b = object.AutoIndent
object.AutoIndent = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
