# IndentProp 对象

## 属性

|     |     |
| --- | --- |
| [AutoIndent](auto_indent) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的启用自动缩进 复选框相对应。 |
| [Begin](begin) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的开始缩进 文本框相对应。 |
| [End](end) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的结束缩进 文本框相对应。 |
| [EndOfStatement](end_of_statement) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的语句结束 文本框相对应。 |
| [IgnoreAfter](ignore_after) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的忽略正则表达式之后的内容 文本框相对应。 |
| [IndentColumns](indent_columns) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的缩进列 文本框相对应。 |
| [InsertSpaces](insert_spaces) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的将制表符转换为空格 复选框相对应。 |
| [RegExp](reg_exp) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的使用正则表达式 复选框相对应。 |
| [TabColumns](tab_columns) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的Tab 列 文本框相对应。 |
| [WrapIndent](wrap_indent) | 与 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 上的换行缩进 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。


```{toctree}
:maxdepth: 1
auto_indent
begin
end
end_of_statement
ignore_after
indent_columns
insert_spaces

wrap_indent
```
