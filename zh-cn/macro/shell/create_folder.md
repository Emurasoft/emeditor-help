# CreateFolder 方法 (Shell 对象)

创建一个文件夹。

## 

### \[JavaScript\]

```
shell.CreateFolder( strFolder );
```

### \[VBScript\]

```
shell.CreateFolder strFolder
```

## 参数

_strFolder_

指定要创建的文件夹。

## 示例

### \[JavaScript\]

```
shell.CreateFolder( "C:\\\\Test" );
```

### \[VBScript\]

```
shell.CreateFolder "C:\\Test"
```

## 版本

支持 EmEditor Professional 22.1 或之后的版本。
