# GetEnv 方法 (Shell 对象)

检索环境变量。

## 

### \[JavaScript\]

```
str = shell.GetEnv( strName );
```

### \[VBScript\]

```
str = shell.GetEnv( strName )
```

## 参数

_strFile_

环境变量的名称。

## 示例

### \[JavaScript\]

```
str = shell.GetEnv( "OPENAI_API_KEY" );
```

### \[VBScript\]

```
str = shell.GetEnv( "OPENAI_API_KEY" )
```

## 返回值

返回指定的环境变量的内容。

## 版本

支持 EmEditor Professional 24.0 或之后的版本。
