# Fuzzy 属性

设置或检索模糊匹配标志。此特殊标志使用模糊匹配，并禁用正则表达式。你不能将模糊匹配与正则表达式结合使用。如果设置了此属性，会忽略 **[Engine](engine)** 和 **[SeparateCrLf](separate_cr_lf)** 属性。

#### \[JavaScript\]

_b_ = reg. **Fuzzy**;

reg. **Fuzzy** = _b_;

#### \[VBScript\]

_b_ = reg. **Fuzzy**;

reg. **Fuzzy** = _b_;

## 版本

支持 EmEditor Professional Version 22.0 或之后的版本。
