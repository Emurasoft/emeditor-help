# Regex 对象

## 属性

|     |     |
| --- | --- |
| [Engine](engine) | 设置会检索正则表达式引擎类型。 |
| [Fuzzy](fuzzy) | 设置或检索模糊匹配标志。 |
| [Global](global) | 设置或检索全局标志。 |
| [IgnoreCase](ignore_case) | 设置或检索是否正则表达式忽略大小写。 |
| [OnlyWord](only_word) | 设置或检索是否正则表达式仅匹配整个单词。 |
| [Pattern](pattern) | 设置或检索正则表达式模式。 |
| [SeparateCrLf](separate_cr_lf) | 设置或检索正则表达式是否分开处理 CR 以及 LF。 |

## 方法

|     |     |
| --- | --- |
| [Find](find) | 用正则表达式搜索指定的字符串并返回一个 Match 对象如果发现匹配。 |
| [Replace](replace) | 用正则表达式搜索指定的字符串，并用指定的字符串替换。 |
| [Test](test) | 测试正则表达式是否与指定字符串成功匹配。 |
