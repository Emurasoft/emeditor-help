# ShowCompletionList 属性 (LanguageServerProp 对象)

与**显示完成列表**复选框相对应，该复选框在配置属性对话框中的[**语言服务器**页面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.ShowCompletionList;
object.ShowCompletionList = b;
```

### \[VBScript\]
```
b = object.ShowCompletionList
object.ShowCompletionList = b
```

## 版本

支持 EmEditor Professional 版 23.1 或之后的版本。
