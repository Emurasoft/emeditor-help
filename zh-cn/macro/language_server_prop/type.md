# Type 属性 (LanguageServerProp 对象)

与**文档类型**下拉列表框相对应，该列表框在在配置属性对话框中的[**语言服务器**页面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.Type;
object.Type = b;
```

### \[VBScript\]

```
b = object.Type
object.Type = b
```

## 版本

支持 EmEditor Professional 版 23.00 或之后的版本。
