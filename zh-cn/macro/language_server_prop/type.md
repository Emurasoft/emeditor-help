# Type 属性 (LanguageServerProp 对象)

与**文档类型**下拉列表框相对应，该列表框在在配置属性对话框中的[**语言服务器**页面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
type = object.Type;
object.Type = type;
```

### \[VBScript\]

```
type = object.Type
object.Type = type
```

## 备注

Type 属性应该是以下值之一。

|     |     |
| --- | --- |
| eeTypeCSS | CSS |
| eeTypeCpp | C++ |
| eeTypeHTML | HTML |
| eeTypeJS | JavaScript |
| eeTypeJSON | JSON |
| eeTypePerl | Perl |
| eeTypePython | Python |


## 版本

支持 EmEditor Professional 版 23.0 或之后的版本。
