# Enabled 属性 (LanguageServerProp 对象)

与**启用语言服务器协议**复选框相对应，该复选框在配置属性对话框中的[**语言服务器**页面](../../dlg/properties/language_server/index)上。

## 

### \[JavaScript\]

```
b = object.Enabled;
object.Enabled = b;
```

### \[VBScript\]

```
b = object.Enabled
object.Enabled = b
```

## 版本

支持 EmEditor Professional 版 23.00 或之后的版本。
