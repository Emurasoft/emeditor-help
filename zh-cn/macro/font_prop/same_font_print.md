# SameFontPrint 属性 (FontProp 对象)

与自定义字体对话框中的 [**将显示字体作为打印机字体** 复选框](../../dlg/properties/font/index) 相对应。

## 

### \[JavaScript\]

```
b = object.SameFontPrint;
object.SameFontPrint = b;
```

### \[VBScript\]

```
b = object.SameFontPrint
object.SameFontPrint = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
