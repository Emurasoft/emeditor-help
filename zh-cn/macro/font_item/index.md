# FontItem 对象

## 属性

|     |     |
| --- | --- |
| [Bold](bold) | True 如果字体的格式被设置为粗体。 |
| [Italic](italic) | True 如果字体的格式被设置为斜体。 |
| [Name](name) | 指定对象的名称。 |
| [Script](script) | 指定字体的脚本。 |
| [Size](size) | 用点数(pt)指定字体大小。 |

## 版本

支持 EmEditor 7.00 或之后的版本。

```{toctree}
:maxdepth: 1
bold
italic
name
script
size
```
