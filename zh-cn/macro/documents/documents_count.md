# Count 属性 (Documents 集合)

检索文档的数目。

## 

### \[JavaScript\]

```
nCount = editor.Documents.Count;
```

### \[VBScript\]

```
nCount = editor.Documents.Count
```

## 示例

### \[JavaScript\]

```
alert( "Number of Documents: " + editor.Documents.Count );
```

### \[VBScript\]

```
alert "Number of Documents: " & editor.Documents.Count
```

## 版本

支持 EmEditor 5.00 或之后的版本。
