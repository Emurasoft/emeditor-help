# UseTemplate 属性 (FileNewProp 对象)

与 [新建文件详细信息 对话框](../../dlg/properties/file/new_details/index) 中的使用模板 文本框相对应。

## 

### \[JavaScript\]

```
b = object.UseTemplate;
object.UseTemplate = b;
```

### \[VBScript\]

```
b = object.UseTemplate
object.UseTemplate = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
