# ReturnMethod 属性 (FileNewProp 对象)

与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **换行方式** 下拉列表框相对应。

## 

### \[JavaScript\]

```
n = object.ReturnMethod;
object.ReturnMethod = n;
```

### \[VBScript\]

```
n = object.ReturnMethod
object.ReturnMethod = n
```

## 版本

支持 EmEditor 7.00 或之后的版本。
