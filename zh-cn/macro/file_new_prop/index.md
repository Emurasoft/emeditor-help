# FileNewProp 对象

## 属性

|     |     |
| --- | --- |
| [Encoding](encoding) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **编码** 下拉列表框相对应。 |
| [InitialFolder](initial_folder) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **初始文件夹** 文本框相对应。 |
| [ReturnMethod](return_method) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **换行方式** 下拉列表框相对应。 |
| [SetInitialFolder](set_initial_folder) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **设置初始文件夹** 复选框相对应。 |
| [TemplateFile](template_file) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **模板文件** 文本框相对应。 |
| [UnicodeSignature](unicode_signature) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **添加一个 Unicode 签名 (BOM)** 复选框相对应。 |
| [UseTemplate](use_template) | 与 [**新建文件详细信息** 对话框](../../dlg/properties/file/new_details/index) 中的 **使用模板** 文本框相对应。 |
