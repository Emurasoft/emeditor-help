# SpellProp 对象

## 属性

|     |     |
| --- | --- |
| [Enabled](enabled) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的拼写检查 复选框相对应。 |
| [CamelCase](camelcase) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的支持驼峰式大小写 复选框相对应。 |
| [Quote](quote) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的引用行 复选框相对应。 |
| [SingleQuotes](singlequotes) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的单引号内的字符串 复选框相对应。 |
| [DoubleQuotes](doublequotes) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的双引号内的字符串 复选框相对应。 |
| [Comment](comment) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的注释 复选框相对应。 |
| [Script](script) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的脚本 复选框相对应。 |
| [InTag](intag) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的标记 复选框相对应。 |
| [Highlight](highlight) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的已高亮的文本 复选框相对应。 |
| [Hyperlink](hyperlink) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的超链接 复选框相对应。 |
| [Rest](rest) | 与配置属性中 [拼写检查 页面](../../dlg/properties/spell/index) 上的未指定的文本 复选框相对应。 |

## 版本

支持 EmEditor 10.00 或之后的版本。
