# KinsokuBegin 属性 (WrapProp 对象)

与配置属性中 [**换行** 页面](../../dlg/properties/wrap/index) 上的 **行首不允许换行字符** 文本框相对应。

## 

### \[JavaScript\]

```
str = object.KinsokuBegin;
object.KinsokuBegin = str;
```

### \[VBScript\]

```
str = object.KinsokuBegin
object.KinsokuBegin = str
```

## 版本

支持 EmEditor 7.00 或之后的版本。
