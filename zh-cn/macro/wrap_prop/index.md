# WrapProp 对象

## 属性

|     |     |
| --- | --- |
| [Kinsoku](kinsoku) | 与配置属性中 [换行 页面](../../dlg/properties/wrap/index) 上的将指定字符换行 复选框相对应。 |
| [KinsokuBegin](kinsoku_begin) | 与配置属性中 [换行 页面](../../dlg/properties/wrap/index) 上的行首不允许换行字符 文本框相对应。 |
| [KinsokuEnd](kinsoku_end) | 与配置属性中 [换行 页面](../../dlg/properties/wrap/index) 上的行末不允许换行字符 文本框相对应。 |
| [WordWrap](word_wrap) | 与配置属性中 [换行 页面](../../dlg/properties/wrap/index) 上的自动换行 复选框相对应。 |
| [WordWrapAfterMarks](word_wrap_after_marks) | 与配置属性中 [换行 页面](../../dlg/properties/wrap/index) 上的启用标记后自动换行 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
