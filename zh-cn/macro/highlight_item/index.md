# HighlightItem 对象

## 属性

|     |     |
| --- | --- |
| [Case](case) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的区分大小写 复选框相对应。 |
| [Color](color) | 指定对象的颜色。 |
| [InsideTag](inside_tag) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的仅在标记内 复选框相对应。 |
| [Name](name) | 指定对象的名称。 |
| [RegExp](reg_exp) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的正则表达式 复选框相对应。 |
| [RightAll](right_all) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的高亮单词与其右侧区域 复选框相对应。 |
| [RightSide](right_side) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的高亮到选定单词右侧 复选框相对应。 |
| [Word](word) | 与配置属性中 [高亮 (1) 页面](../../dlg/properties/highlight1/index) 上的全词匹配时高亮 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
