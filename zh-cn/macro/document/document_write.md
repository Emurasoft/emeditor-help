# write 方法 (Document 对象)

在当前光标位置处插入或改写一个字符串。

## 

### \[JavaScript\]

```
document.write( str );
```

### \[VBScript\]

```
document.write str
```

## 参数

_str_

指定要插入或改写为的字符串。

## 示例

### \[JavaScript\]

```
document.write( "Hello" );
```

### \[VBScript\]

```
document.write "Hello"
```

## 版本

支持 EmEditor 4.00 或之后的版本。
