# GetLines 方法 (Document 对象)

检索文档中的行数。

## 

### \[JavaScript\]

```
nLines = document.GetLines( [ nFlags ] );
```

### \[VBScript\]

```
nLines = document.GetLines( [ nFlags ] )
```

## 参数

_nFlags_

可选项。你能指定下列值。如果你没有指定任何值，EmEditor 会默认指定逻辑坐标。

|     |     |
| --- | --- |
| eeGetLineView | 在视图中指定坐标。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
