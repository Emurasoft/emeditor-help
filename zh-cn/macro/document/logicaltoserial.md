# LogicalToSerial 方法

将指定位置的逻辑坐标转换为基于单个的串行位置。

#### \[JavaScript\]

_nSerialPos_ = document. **LogicalToSerial**( _x_, _y_ );

#### \[VBScript\]

_nSerialPos_ = document. **LogicalToSerial**( _x_, _y_ )

## 参数

_x_

指定基于单个的水平（字符）位置。

_y_

指定基于单个的垂直（行）位置。

## 示例

#### \[JavaScript\]

n = document.LogicalToSerial( 5, 1 );

#### \[VBScript\]

n = document.LogicalToSerial( 5, 1 )

## 版本

支持 EmEditor Professional 17.0 或之后的版本。
