# HighlightFind 属性 (Document 对象)

决定是否要高亮搜索字符串。

## 

### \[JavaScript\]

```
bHighlightFind = document.HighlightFind;
document.HighlightFind = bHighlightFind;
```

### \[VBScript\]

```
bHighlightFind = document.HighlightFind
document.HighlightFind = bHighlightFind
```

## 版本

支持 EmEditor 4.00 或之后的版本。
