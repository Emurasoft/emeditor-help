# writeln 方法 (Document 对象)

在当前光标位置处插入或改写一个字符串并换行。

## 

### \[JavaScript\]

```
document.writeln( str );
```

### \[VBScript\]

```
document.writeln str
```

## 参数

_str_

指定要插入或改写为的字符串。

## 示例

### \[JavaScript\]

```
document.writeln( "Hello" );
```

### \[VBScript\]

```
document.writeln "Hello"
```

## 版本

支持 EmEditor 4.00 或之后的版本。
