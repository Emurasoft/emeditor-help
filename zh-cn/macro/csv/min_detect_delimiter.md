# MinDetectDelimiter 属性

与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **要检测的最少分隔符数** 文本框对应。

#### \[JavaScript\]

_n_ = object. **MinDetectDelimiter**;

object. **MinDetectDelimiter** = _n_;

#### \[VBScript\]

_n_ = object. **MinDetectDelimiter**

object. **MinDetectDelimiter** = _n_

## 版本

支持 EmEditor 19.4 或之后的版本。
