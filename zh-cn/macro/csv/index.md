# Csv 对象

## 属性

|     |     |
| --- | --- |
| [**AllowDelimiter**](allow_delimiter) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **允许分隔符在引号内** 复选框对应。 |
| [**AllowNL**](allow_nl) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **允许在引号内换行** 复选框对应。 |
| [**Delimiter**](delimiter) | 检索或设置当前 Csv 对象的分隔符。 |
| [**EnableTitle**](enable_title) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **工具栏按钮标题** 复选框对应。 |
| [**Escape**](escape) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **使用转义字符** 复选框对应。 |
| [**FirstDetectLine**](first_detect_line) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **要检测的首行** 文本框对应。 |
| [**Heading**](heading) | 与在自定义对话框中 [**CSV 格式** 页面](../../dlg/customize/csv/index) 上的 **列标题** 文本框对应。 |
| [**LinesCell**](lines_cell) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **一个单元格中的最大换行数** 文本框对应。 |
| [**MaxDetectLines**](max_detect_lines) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **要检测的最大行数** 文本框对应。 |
| [**MinDetectDelimiter**](min_detect_delimiter) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **要检测的最少分隔符数** 文本框对应。 |
| [**Name**](name) | 检索或设置当前 Csv 对象的名称。 |
| [**QuotationMark**](quotation_mark) | 检索或设置当前 Csv 对象的引号标记。 |
| [**RowHeading**](row_heading) | 与在自定义对话框中 [**CSV 格式** 页面](../../dlg/customize/csv/index) 上的 **行标题** 文本框对应。 |
| [**Title**](title) | 与在自定义对话框中 [**CSV** 页面](../../dlg/customize/csv/index) 上的 **工具栏按钮标题** 文本框对应。 |
