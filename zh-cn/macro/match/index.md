# Match 对象

## 属性

|     |     |
| --- | --- |
| [FirstIndex](first_index) | 一个只读值，它包含在匹配发生时的原始字符串中以零为基的位置。 |
| [Length](length) | 一个包含匹配字符串总长度的只读值。 |
| [Value](value) | 一个包含匹配值或文本的只读值。 |

## 版本

支持 EmEditor Professional Version 15.9 或之后的版本。
