# RenameIfSameNameExists 属性

与配置属性中 [**备份** 页面](../../dlg/properties/backup/index) 上的 **重命名文件，文件名已存在** 复选框对应。

#### \[JavaScript\]

_b_ = object. **RenameIfSameNameExists**;

object. **RenameIfSameNameExists** = _b_;

#### \[VBScript\]

_b_ = object. **RenameIfSameNameExists**

object. **RenameIfSameNameExists** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
