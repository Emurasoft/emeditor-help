# BackupProp 对象

## 属性

|     |     |
| --- | --- |
| [Folder](folder) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的备份文件夹 文本框对应。 |
| [RenameIfSameNameExists](rename_if_same_name_exists) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的重命名文件，文件名已存在 复选框对应。 |
| [SaveToFolder](save_to_folder) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的保存备份文件到备份文件夹 复选框对应。 |
| [SaveToFolderIfRecycleFailed](save_to_folder_if_recycle_failed) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的无法使用回收站，保存到备份文件夹 复选框对应。 |
| [SaveToSameFolder](save_to_same_folder) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的保存备份文件到源文件所在文件夹 复选框对应。 |
| [UseRecycleBin](use_recycle_bin) | 与配置属性中 [备份 页面](../../dlg/properties/backup/index) 上的使用回收站进行备份 复选框对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
