# DetectUnicodeSignature 属性

与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 Unicode 签名 (BOM)** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **DetectUnicodeSignature**;

object. **DetectUnicodeSignature** = _b_;

#### \[VBScript\]

_b_ =
object. **DetectUnicodeSignature**

object. **DetectUnicodeSignature** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
