# FileProp 对象

## 属性

|     |     |
| --- | --- |
| [CheckFileChanged](check_file_changed) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **被其他程序更改时** 下拉列表框相对应。 |
| [DetectAll](detect_all) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **全部检测** 复选框相对应。 |
| [DetectCharset](detect_charset) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 HTML/XML 字符集** 复选框相对应。 |
| [DetectCSV](detect_csv) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 CSV（逗号分隔）** 复选框相对应。 |
| [DetectDSV](detect_dsv) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 DSV（用户定义的分隔符分隔）** 复选框相对应。 |
| [DetectTSV](detect_tsv) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 TSV（Tab 键分隔）** 复选框相对应。 |
| [DetectUnicodeSignature](detect_unicode_signature) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测 Unicode 签名 (BOM)** 复选框相对应。 |
| [DetectUTF8](detect_utf8) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **检测**<br>**UTF-8** 复选框相对应。 |
| [Encoding](encoding) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **文件打开时的编码** 下拉列表框相对应。 |
| [MonitorInterval](monitor_interval) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **监控时间间隔** 文本框相对应。 |
| [NoFullPathIfNotActive](no_full_path_if_not_active) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **失去焦点时不显示完整路径** 复选框相对应。 |
| [NotepadDiary](notepad_diary) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **创建与记事本兼容的日志文件** 复选框相对应。 |
| [PromptInconsistentReturns](prompt_inconsistent_returns) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **换行方式不一致时进行提示** 复选框相对应。 |
| [PromptInvalid](prompt_invalid) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **如果发现无效字符时进行提示** 复选框相对应。 |
| [PromptNull](prompt_null) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **如果发现空字符时进行提示** 复选框相对应。 |
| [ShowFullPath](show_full_path) | 与配置属性中 [**文件** 页面](../../dlg/properties/file/index) 上的 **显示包含完整路径的文件名** 复选框相对应。 |
