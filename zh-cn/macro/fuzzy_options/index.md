# FuzzyOptions 对象

## 属性

|     |     |
| --- | --- |
| [CharacterRanges](character_ranges) | 检索或设置 [**CharacterRanges** 集合](../character_ranges/index)。 |
| [Flags](flags) | 设置或检索标志。 |
| [MaxEditDistance](max_edit_distance) | 检索或设置最大编辑距离。 |
| [Percent](percent) | 设置或检索相似度的百分比值。 |

## 方法

|     |     |
| --- | --- |
| [Load](load) | 从应用数据文件夹加载模糊选项。 |
| [Reset](reset) | 将模糊选项重置为默认值。 |
| [Save](save) | 将模糊选项保存到应用数据文件夹。 |

## 版本

支持 EmEditor Professional Version 22.0 或之后的版本。
