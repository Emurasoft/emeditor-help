# CrLfDifferent 属性 (MarkProp 对象)

与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **使用不同的标记来显示 CR 与 LF** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.CrLfDifferent;
object.CrLfDifferent = b;
```

### \[VBScript\]

```
b = object.CrLfDifferent
object.CrLfDifferent = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
