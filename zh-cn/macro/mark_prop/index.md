# MarkProp 对象

## 属性

|     |     |
| --- | --- |
| [CrLfDifferent](cr_lf_different) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **使用不同的标记来显示 CR 与 LF** 复选框相对应。 |
| [ShowControlCharacters](show_control_characters) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示控制字符** 复选框相对应。 |
| [ShowEOF](show_eof) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示文件结束标记** 复选框相对应。 |
| [ShowIndentGuides](show_indent_guides) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示缩进线** 复选框相对应。 |
| [ShowReturns](show_returns) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示换行标记** 复选框相对应。 |
| [ShowSpaces](show_spaces) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示空格** 复选框相对应。 |
| [ShowTabs](show_tabs) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示制表符** 复选框相对应。 |
| [ShowUserDefinedGuides](show_user_defined_guides) | 与配置属性中 [**标记** 页面](../../dlg/properties/marks/index) 上的 **显示用户自定义参考线** 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。

```{toctree}
:maxdepth: 1
cr_lf_different
show_control_characters
show_db_spaces
show_eof
show_indent_guides
show_returns
show_spaces
show_tabs
show_user_defined_guides
```
