# TextMode 属性 (DisplayItem 对象)

指定文本颜色模式。

## 

### \[JavaScript\]

```
nMode = item.TextMode;
item.TextMode = nMode;
```

### \[VBScript\]

```
nMode = item.TextMode
item.TextMode = nMode
```

## 备注

nMode 参数会是下列值之一。

|     |     |
| --- | --- |
| 0 | 自定义 |
| 1 | 默认 |
| 2 | 透明 |

## 版本

支持 EmEditor 7.00 或之后的版本。
