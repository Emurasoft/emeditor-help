# DisplayItem 对象

## 属性

|     |     |
| --- | --- |
| [BackColor](back_color) | 指定对象的背景颜色。 |
| [BackMode](back_mode) | 指定背景颜色模式。 |
| [LineColor](line_color) | 指定对象的下划线颜色。 |
| [Style](style) | 指定对象的样式。 |
| [TextColor](text_color) | 指定对象的文本颜色。 |
| [TextMode](text_mode) | 指定文本颜色模式。 |

## 版本

支持 EmEditor 19.1 或之后的版本。
