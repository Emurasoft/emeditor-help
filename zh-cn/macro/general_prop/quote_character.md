# QuoteCharacter 属性 (GeneralProp 对象)

与配置属性中 [常规 页面](../../dlg/properties/general/index) 上的引用字符 文本框相对应。

## 

### \[JavaScript\]

```
str = object.QuoteCharacter;
object.QuoteCharacter = str;
```

### \[VBScript\]

```
str = object.QuoteCharacter
object.QuoteCharacter = str
```

## 版本

支持 EmEditor 7.00 或之后的版本。
