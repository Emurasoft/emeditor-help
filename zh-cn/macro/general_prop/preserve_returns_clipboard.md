# PreserveReturnsClipboard 属性 (GeneralProp 对象)

与配置属性中 [常规 页面](../../dlg/properties/general/index) 上的在剪贴板保留换行方式 (CR/LF) 复选框相对应。

## 

### \[JavaScript\]

```
b = object.PreserveReturnsClipboard;
object.PreserveReturnsClipboard = b;
```

### \[VBScript\]

```
b = object.PreserveReturnsClipboard
object.PreserveReturnsClipboard = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
