# ShowLineNumbers 属性 (GeneralProp 对象)

与配置属性中 [**常规** 页面](../../dlg/properties/general/index) 上的 **显示行号** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.ShowLineNumbers;
object.ShowLineNumbers = b;
```

### \[VBScript\]

```
b = object.ShowLineNumbers
object.ShowLineNumbers = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
