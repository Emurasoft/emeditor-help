# AllowControlCharacter 属性 (GeneralProp 对象)

与配置属性中 [**常规** 页面](../../dlg/properties/general/index) 上的 **允许插入控制字符** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.AllowControlCharacter;
object.AllowControlCharacter = b;
```

### \[VBScript\]

```
b = object.AllowControlCharacter
object.AllowControlCharacter = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
