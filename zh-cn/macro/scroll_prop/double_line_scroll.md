# DoubleLineScroll 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **每次滚动两行** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.DoubleLineScroll;
object.DoubleLineScroll = b;
```

### \[VBScript\]

```
b = object.DoubleLineScroll
object.DoubleLineScroll = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
