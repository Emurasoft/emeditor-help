# VerticalPage 属性 (ScrollProp 对象)

与配置属性中 [滚动 页面](../../dlg/properties/scroll/index) 上的总是启用 1 页垂直滚动 复选框相对应。

## 

### \[JavaScript\]

```
b = object.VerticalPage ;
object.VerticalPage  = b;
```

### \[VBScript\]

```
b = object.VerticalPage
object.VerticalPage  = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
