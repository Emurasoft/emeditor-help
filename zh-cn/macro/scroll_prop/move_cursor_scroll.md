# MoveCursorScroll 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **通过滚动位置移动光标** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.MoveCursorScroll;
object.MoveCursorScroll = b;
```

### \[VBScript\]

```
b = object.MoveCursorScroll
object.MoveCursorScroll = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
