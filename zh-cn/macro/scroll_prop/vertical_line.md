# VerticalLine 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **垂直线** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.VerticalLine ;
object.VerticalLine  = b;
```

### \[VBScript\]

```
b = object.VerticalLine
object.VerticalLine  = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
