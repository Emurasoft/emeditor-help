# HorizontalGrid 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **水平格线** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.HorizontalGrid;
object.HorizontalGrid = b;
```

### \[VBScript\]

```
b = object.HorizontalGrid
object.HorizontalGrid = b
```

## 版本

支持 EmEditor 10.00 或之后的版本。
