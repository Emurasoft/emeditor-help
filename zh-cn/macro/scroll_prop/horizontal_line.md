# HorizontalLine 属性 (ScrollProp 对象)

与配置属性中 [**滚动** 页面](../../dlg/properties/scroll/index) 上的 **水平线** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.HorizontalLine;
object.HorizontalLine = b;
```

### \[VBScript\]

```
b = object.HorizontalLine
object.HorizontalLine = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
