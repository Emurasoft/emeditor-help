# FileSaveProp 对象

## 属性

|     |     |
| --- | --- |
| [AlwaysEnabled](always_enabled) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的总是启用保存 复选框相对应。 |
| [DefaultExtension](default_extension) | 与配置属性中 [文件 页面](../../dlg/properties/file/index) 上的默认扩展名 文本框相对应。 |
| [DeleteEmpty](delete_empty) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的保存时删除空文件 复选框相对应。 |
| [DeleteSpacesAtEnd](delete_spaces_at_end) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的删除行末空格 复选框相对应。 |
| [Encoding](encoding) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的编码 下拉列表框相对应。 |
| [InsertReturns](insert_returns) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的保存时自动换行 复选框相对应。 |
| [NamedEntity](named_entity) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的使用命名实体引用 复选框相对应。 |
| [NameUntitled](name_untitled) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的自动为无标题文档命名 复选框相对应。 |
| [NumericEntity](numeric_entity) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的把 Unicode 保存为 HTML/XML 字符引用 复选框相对应。 |
| [PromptRename](prompt_rename) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的更名时在删除旧文件前提示 复选框相对应。 |
| [PromptUnicode](prompt_unicode) | 与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的当保存 Unicode 为 ANSI 时进行提示 复选框相对应。 |
| [ReturnMethod](return_method) | 与在 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的换行方式 下拉列表框相对应。 |
| [TabsAsSpaces](tabs_as_spaces) | 与在 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的把 Tab 保存为空格 复选框相对应。 |
| [UnicodeSignature](unicode_signature) | 与在 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的添加一个 Unicode 签名 (BOM) 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
