# UnicodeSignature 属性 (FileSaveProp 对象)

与 [添加一个 Unicode 签名 (BOM) 复选框](../../dlg/properties/file/save_details/index) 相对应。

## 

### \[JavaScript\]

```
b = object.UnicodeSignature;
object.UnicodeSignature = b;
```

### \[VBScript\]

```
b = object.UnicodeSignature
object.UnicodeSignature = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
