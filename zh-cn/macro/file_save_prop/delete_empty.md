# DeleteEmpty 属性 (FileSaveProp 对象)

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **保存时删除空文件** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.DeleteEmpty;
object.DeleteEmpty = b;
```

### \[VBScript\]

```
b = object.DeleteEmpty
object.DeleteEmpty = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
