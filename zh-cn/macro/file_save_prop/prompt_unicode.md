# PromptUnicode 属性 (FileSaveProp 对象)

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **当保存 Unicode 为 ANSI 时进行提示** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.PromptUnicode;
object.PromptUnicode = b;
```

### \[VBScript\]

```
b = object.PromptUnicode
object.PromptUnicode = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
