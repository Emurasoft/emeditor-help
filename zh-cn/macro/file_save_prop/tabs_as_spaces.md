# TabsAsSpaces 属性 (FileSaveProp 对象)

与 [**把 Tab 保存为空格** 复选框](../../dlg/properties/file/save_details/index) 相对应。

## 

### \[JavaScript\]

```
b = object.TabsAsSpaces;
object.TabsAsSpaces = b;
```

### \[VBScript\]

```
b = object.TabsAsSpaces
object.TabsAsSpaces = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
