# NameUntitled 属性 (FileSaveProp 对象)

与 [保存详细信息 对话框](../../dlg/properties/file/save_details/index) 中的自动为无标题文档命名 复选框相对应。

## 

### \[JavaScript\]

```
b = object.NameUntitled;
object.NameUntitled = b;
```

### \[VBScript\]

```
b = object.NameUntitled
object.NameUntitled = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
