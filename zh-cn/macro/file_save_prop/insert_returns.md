# InsertReturns 属性 (FileSaveProp 对象)

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **保存时自动换行** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.InsertReturns;
object.InsertReturns = b ;
```

### \[VBScript\]

```
b = object.InsertReturns
object.InsertReturns = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
