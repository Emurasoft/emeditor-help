# DeleteSpacesAtEnd 属性

与 [**保存详细信息** 对话框](../../dlg/properties/file/save_details/index) 中的 **删除行末空格** 复选框相对应。

#### \[JavaScript\]

_b_ =
object. **DeleteSpacesAtEnd**;

object. **DeleteSpacesAtEnd** = _b_;

#### \[VBScript\]

_b_ =
object. **DeleteSpacesAtEnd**

object. **DeleteSpacesAtEnd** = _b_

## 版本

支持 EmEditor 7.00 或之后的版本。
