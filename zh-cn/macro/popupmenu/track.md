# Track 方法 (PopupMenu 对象)

显示弹出菜单，检索所选菜单项的标识符。

## 

### \[JavaScript\]

```
id = popupmenu.Track( flags );
```

### \[VBScript\]

```
id = popupmenu.Track( flags )
```

## 参数

> _flags_

指定菜单要显示的位置。能指定下列值。如果不指定，弹出菜单会在当前光标位置处显示。

|     |     |
| --- | --- |
| eePosMouse | 在当前鼠标指针位置处显示弹出菜单。 |

## 版本

支持 EmEditor 5.00 或之后的版本。
