# LineComment2 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“注释”群组框 \- “行注释”** 文本框相对应。

## 

### \[JavaScript\]

```
str = object.LineComment2;
object.LineComment2 = str;
```

### \[VBScript\]

```
str = object.LineComment2
object.LineComment2 = str
```

## 版本

支持 EmEditor 7.00 或之后的版本。
