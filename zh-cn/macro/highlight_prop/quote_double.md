# QuoteDouble 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“引号内的字符串”群组框 \- 双引号 \["..."\]** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.QuoteDouble;
object.QuoteDouble = b;
```

### \[VBScript\]

```
b = object.QuoteDouble
object.QuoteDouble = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
