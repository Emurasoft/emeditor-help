# QuoteSingle 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“引号内的字符串”群组框 \- 单引号 \['...'\]** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.QuoteSingle;
object.QuoteSingle = b;
```

### \[VBScript\]

```
b = object.QuoteSingle
object.QuoteSingle = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
