# CommentEnd 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **“注释”群组框 \- “结束”** 文本框相对应。

## 

### \[JavaScript\]

```
b = object.CommentEnd;
object.CommentEnd = b;
```

### \[VBScript\]

```
b = object.CommentEnd
object.CommentEnd = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
