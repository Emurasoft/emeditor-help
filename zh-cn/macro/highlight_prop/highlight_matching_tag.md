# HighlightMatchingTag 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **高亮配对标签** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.HighlightMatchingTag;
object.HighlightMatchingTag = b;
```

### \[VBScript\]

```
b = object.HighlightMatchingTag
object.HighlightMatchingTag = b
```

## 版本

支持 EmEditor 13.0.4 或之后的版本。
