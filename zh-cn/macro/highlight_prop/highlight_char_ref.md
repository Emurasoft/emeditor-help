# HighlightCharRef 属性 (HighlightProp 对象)

与配置属性中 [**高亮（2）** 页面](../../dlg/properties/highlight2/index) 上的 **高亮 HTML/XML 中的字符引用及通用字符名称** 复选框相对应。

## 

### \[JavaScript\]

```
b = object.HighlightCharRef;
object.HighlightCharRef = b;
```

### \[VBScript\]

```
b = object.HighlightCharRef
object.HighlightCharRef = b
```

## 版本

支持 EmEditor 11.00 或之后的版本。
