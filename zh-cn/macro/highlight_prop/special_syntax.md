# SpecialSyntax 属性 (HighlightProp 对象)

与配置属性中 [高亮（2） 页面](../../dlg/properties/highlight2/index) 上的特殊语法 下拉列表框相对应。

## 

### \[JavaScript\]

```
n = object.SpecialSyntax;
object.SpecialSyntax = n;
```

### \[VBScript\]

```
n = object.SpecialSyntax
object.SpecialSyntax = n
```

## 版本

支持 EmEditor 7.00 或之后的版本。
