# ChangeCase 方法 (Selection 对象)

变更所选取的文本的大小写。

## 

### \[JavaScript\]

```
document.selection.ChangeCase( nFlags );
```

### \[VBScript\]

```
document.selection.ChangeCase nFlags
```

## 参数

_nFlags_

指定下列值之一:

|     |     |
| --- | --- |
| eeCaseLowerCase | 把大写字母转换成小写字母。 |
| eeCaseUpperCase | 把小写字母转换成大写字母。 |
| eeCaseCapitalize | 大写每一个单词的第一个字母（EmEditor 4.10 或之后的版本）。 |

## 版本

支持 EmEditor 4.00 或之后的版本。
