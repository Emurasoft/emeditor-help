# OverwriteMode 属性 (Selection 对象)

设置或检索表示改写或插入模式的标志。

## 

### \[JavaScript\]

```
bOverwrite = document.selection.OverwriteMode;
document.selection.OverwriteMode = bOverwrite;
```

### \[VBScript\]

```
bOverwrite = document.selection.OverwriteMode
document.selection.OverwriteMode = bOverwrite
```

## 版本

支持 EmEditor 4.00 或之后的版本。
