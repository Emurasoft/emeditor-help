# Copy 方法

复制选定内容到剪贴板上。

#### \[JavaScript\]

document.selection. **Copy**( _nFlags_ );

#### \[VBScript\]

document.selection. **Copy** _nFlags_

## 参数

_nFlags_

指定一个下列值的组合:

|     |     |
| --- | --- |
| eeCopyUnicode | 复制为 Unicode（默认）。 |
| eeCopyQuotes | 复制为引用文本。 |

## 版本

支持 EmEditor 4.00 或之后的版本。
