# DoubleClickOnly 属性 (LinkProp 对象)

与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的仅允许双击 复选框相对应。

## 

### \[JavaScript\]

```
b = object.DoubleClickOnly;
object.DoubleClickOnly = b;
```

### \[VBScript\]

```
b = object.DoubleClickOnly
object.DoubleClickOnly = b
```

## 版本

支持 EmEditor 7.00 或之后的版本。
