# LinkProp 对象

## 属性

|     |     |
| --- | --- |
| [DBCharURL](dbc_har_url) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的识别双字节字符为 URL 复选框相对应。 |
| [DoubleClickOnly](double_click_only) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的仅允许双击 复选框相对应。 |
| [KanaURL](kana_url) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的识别单字节假名与假名符合为 URL 复选框相对应。 |
| [MailTo](mail_to) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的点击邮件地址时发送邮件 复选框相对应。 |
| [OpenFileInQuotes](open_file_in_quotes) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的点击引号内的文件名打开文件 复选框相对应。 |
| [SelectInQuotes](select_in_quotes) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的点击引号内的文本选择整个字符串 复选框相对应。 |
| [URL](url) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的链接到 URL 复选框相对应。 |
| [URLCharacters](url_characters) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的识别这些字符 复选框相对应。 |
| [URLCharactersNotEnd](url_characters_not_end) | 与配置属性中 [链接 页面](../../dlg/properties/link/index) 上的行末不允许的字符 复选框相对应。 |

## 版本

支持 EmEditor 7.00 或之后的版本。
