# URLCharactersNotEnd 属性 (LinkProp 对象)

与配置属性中 [**链接** 页面](../../dlg/properties/link/index) 上的 **行末不允许的字符** 复选框相对应。

## 

### \[JavaScript\]

```
s = object.URLCharactersNotEnd;
object.URLCharactersNotEnd = s;
```

### \[VBScript\]

```
s = object.URLCharactersNotEnd
object.URLCharactersNotEnd = s
```

## 版本

支持 EmEditor 7.00 或之后的版本。
