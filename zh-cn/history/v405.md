# Version 4.05

2004年8月4日

- 修复了在一个命令行之后的自动缩进行为。
- 当一个正则表达式搜索字符串的末尾是反斜线时，EmEditor 会突然出故障的 bug 被修复。
- EmEditor 不能导入包含大于 80 个字符长度的行的语法文件的 bug 被修复。
- 在特定条件下，EmEditor 不能用例如像 "\\\" 的表达式在文件中替换。
- 添加了一条消息当被选取的插件无法在“自定义插件”对话框中被删除时。
- 有关在窗口没有合并但窗口位置被保存后，运行“在文件中替换”命令后，EmEditor 无法在点击一个被最小化的窗口时还原保存的窗口大小的 bug 已经被修复了。
- 在 Windows XP Tablet PC 上运行 EmEditor 的“在文件中查找”命令，EmEditor 会出故障的 bug。
- 当当前文件被删除并用同样的名称被替换时，并且在“被其他程序更改时”的选项选择的是“提示”，EmEditor 还是没有显示提示信息的 bug 被修复。
- 当“被其他程序更改时”的选项选择的是“提示”，但 EmEditor 还是没有激活应该显示提示信息的主窗口的 bug 被修复。
