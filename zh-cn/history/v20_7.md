# Version 20.7

2021年5月11日

## 一般新增功能

- 新版本让 [合并文件 向导](../dlg/combine_files/index) 能使用现有文件作为源文件，而无需在编辑器中打开。
- 改进了 [剪贴板记录 功能](https://zh-cn.emeditor.com/text-editor-features/more-features/clipboard-history/)。右键单击剪贴板记录 时，在显示的菜单中选择向上 或向下，让您可以重新排序历史记录，或同时选择多个项目以粘贴合并的多个项目。
- 新版本使用单独的线程来运行宏。这样，您可以在运行宏时随时停止宏。要停止宏，您可以选择新的停止宏 命令，或者按两次ESC，或按CTRL \+BREAK。
- 在选择多个单元格作为目标之后，在单元格选择模式下添加了粘贴单元格的新方式。
- 添加了$(Path)，$(Dir)，$(Filename)，$(FilenameEx)，和$(Ext) 到 [替换表达式语法](../howto/search/replacement_expression_syntax) 中。
- 更新了Onigmo 正则表达式引擎到 Version 6.2.0。
- 更新了Hunspell 拼写引擎到 Version 1.7.0。

## 新增选项

- [合并文件 向导](../dlg/combine_files/index) 的首页被分成了包括“现有文件”选项的 [欢迎使用文件合并器 页面](../dlg/combine_files/combine_wiz_src)，和 [目标文件和选项 页面](../dlg/combine_files/combine_wiz_dest)。
- 添加了 [剪贴板 页面](../dlg/customize/clipboard/index) 到自定义 对话框中。
- 添加了4对括号对 (“/”，‘/’，《/》，和 〖/〗) 到配置属性 [高亮 (2) 页面](../dlg/properties/highlight2/index) 上的括号对 列表框中。
- 添加了使用编辑器字体 选项到下列对话框中，您可以在点击> 按钮或右键单击文本框或列表框所显示的菜单中找到该选项： [查找](../dlg/find/index)，[替换](../dlg/replace/index)，[在文件中查找](../dlg/find_in_files/index)，[在文件中替换](../dlg/replace_in_files/index)，[高级筛选](../dlg/advanced_filter/index)，以及剪贴板记录 对话框。

## 新增命令

- [自定义剪贴板](../cmd/tools/customize_clipboard)
- [停止宏](../cmd/macros/macro_stop)
- [临时隐藏引号](../cmd/view/csv_hide_quotes)

## 插件新增功能

- 添加了EI\_GET\_VIEW\_FONT，EI\_GET\_HIDE\_QUOTES，EI\_SET\_HIDE\_QUOTES 命令到 [EE\_INFO 消息](../plugin/message/ee_info) 中。

## 宏的新增功能

- 改进了 [SendKeys 方法](../macro/shell/send_keys)，以使该方法可用于 EmEditor 视图。
- 添加了 {CONVERT}，{NONCONVERT}，以及 {VKnnn} 键到 [SendKeys 方法](../macro/shell/send_keys) 中的 _str_ 参数中。
- 添加了 [HideQuotes 属性](../macro/document/hide_quotes) 到Document 对象中。
