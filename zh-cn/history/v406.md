# Version 4.06

2004年8月26日

- 修复了 v4.05 版本无法安装模板文件的 bug (template.\*)。
- 如果编码是 UTF-8 或 Unicode，状态栏现在回显示是否带签名。
- 当一个文件被外部程序修改时，会显示一条即时消息仅当焦点回复到 EmEditor 中时。
- 在安装文件夹被修改后，修复了即使勾选了"添加 EmEditor 到 Internet Explorer v5 或之后的 HTML 编辑器列表"，EmEditor 也没有显示在 Internet Explorer 上的 HTML 编辑器列表中的问题。
- 对于“层叠/平铺窗口”命令不能移动 Windows XP SP2 上活动窗口的问题进行了改进。
- 优化了“全部替换”命令。
