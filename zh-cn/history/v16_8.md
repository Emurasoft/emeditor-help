# Version 16.8

2017年5月17日

## 一般新增功能

- 大大提高了高级筛选的速度，当定义了个多个筛选条目时。

### 测试结果:

|  | v16.7 | v16.8 | 比较 |
| --- | --- | --- | --- |
| 高级筛选，不勾选“区分大小写” | 32.6 秒 | 2.57 秒 | 快 12.7 倍 |
| 高级筛选，不勾选“区分大小写” | 30.3 秒 | 2.50 秒 | 快 12.1 倍 |

\\* 10,000 行 (2.7MB) CSV 文件 (UTF-8)；条件：使用高级筛选，50,000 次搜索第一列条件，选择“对前一个条件用逻辑或运算”。测试系统：Windows 10 (64-bit) Core i7-4790 16GB RAM

- 工具栏中的「撤消」和「重做」按钮现在是下拉式按钮，单击按钮上的箭头将显示上下文菜单，允许您访问新的“撤消最近”，“重新启动”，“全部撤消”，“全部重做”，“撤消/重做历史”命令。
- 新的“撤消/重做历史”对话框允许您导出和导入撤消/重做历史记录。
- 现在可以给以下命令分配键盘快捷键：从当前文件夹打开，打开 Windows 资源管理器，打开命令提示符 以及复制文件路径名称。 在默认情况下，打开 Windows 资源管理器 命令所分配的键盘快捷键是 ALT + SHIFT + E。
