# Version 15.4

2015年10月13日

## 一般新增功能

- 大大提高了执行删除已添加书签的行 命令，以及在不同窗口中比较 两个文档的速度。
- 添加了与base64 编码 相关的命令 (参考新增命令如下)。

## 新增选项

-按字数从少到多排序 以及按字数从多到少排序 单选按钮被添加到 [多列排序 对话框](../dlg/sort_multi/index) 中。
-总是显示光标上/下1行 复选框被添加到配置属性的 [滚动 页面](../dlg/properties/scroll/index) 上。
-自定义 对话框上的标记 页面被重命名为[标记/格式](../dlg/customize/marks/index)，让您能设置日期/时间格式 当需要插入日期和时间时。
-全部应用 复选框被添加到了当试图打开含有无效字符的文件时会显示的对话框中。
-直接合并 (无关键值) 复选框被添加到了 [合并 CSV 对话框](../dlg/join_csv/index) 中。
-标记访问过的链接 复选框被添加到了配置属性中的 [链接 页面](../dlg/properties/link/index) 上。
-URL (访问过的)，邮件地址 (访问过的)，以及“在文件中查找”结果中的超链接 (访问过的) 被添加到配置属性中的 [显示 页面](../dlg/properties/display/index) 上的指定部分 列表框中。

## 新增命令

- [将 Base64 转换为纯文本 (当前编码)](../cmd/edit/decode_base64)
- [将纯文本转换为 Base64 (当前编码)](../cmd/edit/encode_base64)
- [将 Base64 转换为纯文本 (UTF-8)](../cmd/edit/decode_base64_utf8)
- [将纯文本转换为 Base64 (UTF-8)](../cmd/edit/encode_base64_utf8)
- [Base64 转换为二进制文件](../cmd/edit/decode_base64_binary)
- [二进制文件转换为 Base64](../cmd/edit/encode_base64_binary)
- [按字数从少到多排序](../cmd/edit/sort_words_a)
- [按字数从多到少排序](../cmd/edit/sort_words_d)
- [清除访问过的链接记录](../cmd/edit/clear_visit_history)

## 插件新增功能

-显示行 与页数 被添加到 [字数统计 插件上](../howto/plugin/plugin_wordcount) 的查询类型 中。

## 插件 API 新增功能

- EI\_GET\_PRINT\_PAGES 参数被添加到 [EE\_INFO 消息](../plugin/message/ee_info) ( [Editor\_Info 内联函数](../plugin/macro/editor_info)) 中。
- 添加了 EEID\_DECODE\_BASE64，EEID\_ENCODE\_BASE64，EEID\_DECODE\_BASE64\_UTF8，EEID\_ENCODE\_BASE64\_UTF8，EEID\_DECODE\_BASE64\_BINARY，EEID\_ENCODE\_BASE64\_BINARY，EEID\_SORT\_WORDS\_A，EEID\_SORT\_WORDS\_D，以及 EEID\_CLEAR\_VISIT\_HISTORY 命令。

## 宏的新增功能

-eeJoinSimpleMerge 标志被添加到 [Join 方法](../macro/editor/join) 中。
