# Version 4.10

2005年1月23日

- [**状态栏** 页面](../dlg/customize/status/index) 被添加到了自定义对话框中。
- 你能变更 Windows 工具栏按钮顺序通过在 Windows 工具栏上拖放一个按钮。
- 添加了 [**关闭所有其他文件** 命令](../cmd/file/close_all_others)。
- 添加了 [**移动到下一个** 命令](../cmd/window/window_move_next) 以及 [**移动到上一个** 命令](../cmd/window/window_move_prev)。
- 状态栏现在显示在光标位置处的字符代码值。
- 添加了 [**首字母大写** 命令](../cmd/convert/capitalize)。
- eeCaseCapitalize 标志添加到了 [**ChangeCase** 方式](../macro/selection/selection_changecase) 中。
- 添加了 [**分割行** 命令](../cmd/convert/split_lines) 以及 [**合并行** 命令](../cmd/convert/join_lines)。
- eeFormatSplitLines 标志和 eeFormatJoinLines 标志添加到了 [**Format** 方式](../macro/selection/selection_format) 中。
- 添加了 [**导入和导出** 命令](../cmd/tools/import_export)。
- 在键盘布局窗口中添加了上下文菜单。
- 你现在能显示或编辑“键盘布局”对话框中选取的项目的属性。
- 你现在能显示所有配置的“键盘布局”。
- 你现在能把选取的项目作为文本复制到剪贴板上。
- 你现在能通过在属性对话框的“键盘”页面上，按下 Ctrl 键，放开，再按下 PageUp 或 PageDown 来分配 Ctrl+PageUp 或 Ctrl+PageDown 键。
- 16 色的工具栏现在可以显示外部工具图标。
