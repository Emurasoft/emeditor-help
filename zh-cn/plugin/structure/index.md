# 结构

|     |     |
| --- | --- |
| [ACTIVE\_STRING\_INFO](active_string_info) | 用于 [EE\_GET\_ACTIVE\_STRING](../message/ee_get_active_string) 消息。 |
| [ATTR\_INFO](attr_info) | 用于 [EE\_GET\_ATTR](../message/ee_get_attr) 消息。 |
| [AUTOFILL\_INFO](autofill_info) | 用于 [EE\_AUTOFILL](../message/ee_autofill) 消息。 |
| [BATCH\_GREP\_INFO](batch_grep_info) | 用于 [Editor\_BatchFindInFiles](../macro/editor_batchfindinfiles) 和<br>[Editor\_BatchReplaceInFiles](../macro/editor_batchreplaceinfiles) 内联函数 ( [EE\_FIND\_IN\_FILESW](../message/ee_find_in_filesw) 和 [EE\_REPLACE\_IN\_FILESW消息](../message/ee_replace_in_filesw)) 。 |
| [BATCH\_INFO](batch_info) | 用于<br>[EE\_FIND\_REPLACE](../message/ee_find_replace) 消息。 |
| [CLIP\_INFO](../structure/clip_info) | 用于 [EE\_CLIP\_HISTORY](../message/ee_clip_history) 消息。 |
| [COLUMN\_INFO](column_info) | 用于 [SORT\_INFO 结构](sort_info)。 |
| [COLUMN\_STRUCT](column_struct) | 用于 [Editor\_SetColumn](../macro/editor_setcolumn) <br>内联函数 ( [EE\_SET\_COLUMN](../message/ee_set_column) 消息) 中。 |
| [COMPARE\_INFO](compare_info) | 用于 [EE\_COMPARE](../message/ee_compare) 消息。 |
| [CONVERT\_INFO](convert_info) | 用于 [EE\_CONVERT\_EX](../message/ee_convert_ex) 消息。 |
| [CONVERT\_CSV\_INFO](convert_csv_info) | 用于 [EE\_CONVERT\_CSV](../message/ee_convert_csv) 消息。 |
| [CUSTOM\_BAR\_INFO](custom_bar_info) | 用于 [Editor\_CustomBarOpen](../macro/editor_custombaropen) 内联函数 ( [EE\_CUSTOM\_BAR\_OPEN](../message/ee_custom_bar_open) 消息) 中。 |
| [CUSTOM\_BAR\_CLOSE\_INFO](custom_bar_close_info) | 用于 [EVENT\_CUSTOM\_BAR\_CLOSED 事件](../event/index) 中。 |
| [EDIT\_COLUMN\_INFO](edit_column_info) | 用于 [EE\_EDIT\_COLUMN](../message/ee_edit_column) 消息。 |
| [EXEC\_PLUGIN\_INFO](exec_plugin_info) | 用于 [EE\_EXEC\_PLUGIN](../message/ee_exec_plugin) 消息。 |
| [EXTRACT\_FREQUENT\_INFO](extract_frequent_info) | 用于 [EE\_EXTRACT\_FREQUENT](../message/ee_extract_frequent) 消息。 |
| [FILTER\_INFO](filter_info) | 用于 [EE\_FILTER](../message/ee_filter) 消息。 |
| [FILTER\_INFO\_EX](filter_info_ex) | 用于 [EE\_FILTER](../message/ee_filter) 和 [EE\_GET\_FILTER](../message/ee_get_filter) 消息。 |
| [FIND\_REGEX\_INFO](find_regex_info) | 用于 [Editor\_FindRegex 内联函数](../macro/editor_findregex) ( [EE\_FIND\_REGEX 消息](../message/ee_find_regex)) 中。 |
| [FIND\_REGEX\_INFO\_EX](find_regex_info_ex) | 用于 [Editor\_FindRegex](../macro/editor_findregex) 内联函数 ( [EE\_FIND\_REGEX](../message/ee_find_regex) 消息) 中。 |
| [FIND\_REPLACE\_INFO](find_replace_info) | 用于 [EE\_FIND\_REPLACE 消息](../message/ee_find_replace) 中。 |
| [GET\_CELL\_INFO](get_cell_info) | 用于 [Editor\_GetCell](../macro/editor_getcell) <br>内联函数 ( [EE\_GET\_CELL](../message/ee_get_cell) 消息) 中。 |
| [GET\_COLOR\_INFO](get_color_info) | 用于 [EE\_GET\_COLOR](../message/ee_get_color) 消息。 |
| [GET\_LINE\_INFO](get_line_info) | 用于 Editor\_GetLineA 和 Editor\_GetLineW 宏 (EE\_GET\_LINEA，EE\_GET\_LINEW 消息) 中。 |
| [GREP\_INFOA](grep_infoa) | 用于 [Editor\_FindInFilesA 内联函数](../macro/editor_findinfilesa)， [Editor\_ReplaceInFilesA 内联函数](../macro/editor_replaceinfilesa) ( [EE\_FIND\_IN\_FILESA 消息](../message/ee_find_in_filesa)， [EE\_REPLACE\_IN\_FILESA 消息](../message/ee_replace_in_filesa)) 中。 |
| [GREP\_INFOW](grep_infow) | 用于 [Editor\_FindInFilesW 内联函数](../macro/editor_findinfilesw)， [Editor\_ReplaceInFilesW 内联函数](../macro/editor_replaceinfilesw) ( [EE\_FIND\_IN\_FILESW 消息](../message/ee_find_in_filesw)， [EE\_REPLACE\_IN\_FILESW 消息](../message/ee_replace_in_filesw)) 中。 |
| [GREP\_INFO\_EX](grep_info_ex) | 用于 [Editor\_FindInFiles](../macro/editor_findinfiles) 内联函数，还有 [Editor\_ReplaceInFiles](../macro/editor_replaceinfiles) 内联函数 ( [EE\_FIND\_IN\_FILESW](../message/ee_find_in_filesw) 消息和 [EE\_REPLACE\_IN\_FILESW](../message/ee_replace_in_filesw) 消息) 中。 |
| [HISTORY\_INFO](history_info) | 用于 [EVENT\_HISTORY 事件](../event/index) 中。 |
| [INFO\_EX\_DATA](info_ex_data) | 用于 [EE\_INFO\_EX 消息](../message/ee_info_ex)。 |
| [JOIN\_INFO](join_info) | 用于 [EE\_JOIN](../message/ee_join) 消息。 |
| [LOAD\_FILE\_INFO\_EX](load_file_info) | 用于 [Editor\_LoadFileA](../macro/editor_loadfilea) <br>以及 [Editor\_LoadFileW](../macro/editor_loadfilew) <br>内联函数 ( [EE\_LOAD\_FILEA](../message/ee_load_filea) 和<br>[EE\_LOAD\_FILEW](../message/ee_load_filew) 消息) 中。 |
| [MANAGE\_DUPLICATES\_INFO](manage_duplicates_info) | 用于 [EE\_MANAGE\_DUPLICATES](../message/ee_manage_duplicates) 消息。 |
| [MATCH\_REGEX\_INFO](match_regex_info) | 用于 [Editor\_MatchRegex 内联函数](../macro/editor_matchregex) ( [EE\_MATCH\_REGEX 消息](../message/ee_match_regex)) 中。 |
| [MATCH\_REGEX\_INFO\_EX](match_regex_info_ex) | 用于 [Editor\_MatchRegex](../macro/editor_matchregex) 内联函数 ( [EE\_MATCH\_REGEX](../message/ee_match_regex) 消息) 中。 |
| [NUMBERING\_INFO](numbering_info) | 用于 [EE\_NUMBERING 消息](../message/ee_numbering)。 |
| [OUTLINE\_ARRAY\_INFO](outline_array_info) | 用于 <br>[Editor\_SetOutlineArray](../macro/editor_setoutlinearray) <br>内联函数 ( [EE\_SET\_OUTLINE\_ARRAY](../message/ee_set_outline_array) 消息) 中。 |
| [POINT\_PTR](point_ptr) | 用来指定一个点的位置。在 32 位的插件中，POINT\_PTR 与 POINT 结构相同。在 64 位插件中，每一个字段从 32 位整数被延伸到 64 位整数。 |
| [PIVOT\_TABLE\_INFO](pivot_table_info) | 用于 [EE\_PIVOT\_TABLE](../message/ee_pivot_table) 消息中。 |
| [QUERY\_STRING\_INFO](query_string_info) | 用于 [EE\_QUERY\_STRING\_EX 消息](../message/ee_query_string_ex) 中。 |
| [REARRANGE\_COLUMNS\_INFO](rearrange_columns_info) | 用于 [EE\_REARRANGE\_COLUMNS](../message/ee_rearrange_columns) 消息中。 |
| [REG\_QUERY\_VALUE\_INFO](reg_query_value_info) | 用于 [EE\_REG\_QUERY\_VALUE 消息](../message/ee_reg_query_value) 中。 |
| [REG\_SET\_VALUE\_INFO](reg_set_value_info) | 用于 [EE\_REG\_SET\_VALUE 消息](../message/ee_reg_set_value) 中。 |
| [RUN\_MACRO\_INFO](run_macro_info) | 用于 [EE\_RUN\_MACRO](../message/ee_run_macro) 消息。 |
| [SEL\_INFO](../../plugin/structure/sel_info) | 用于 Editor\_GetMultiSel 内联函数 ( [EE\_GET\_MULTI\_SEL](../../plugin/message/ee_get_multi_sel) 消息) 中。 |
| [SIZE\_PTR](size_ptr) | 用来指定一个大小。在 32 位插件中，SIZE\_PTR 与 SIZE 结构相同。在 64 位插件中，每一个字段都从 32 位整数被延伸到 64 位整数。 |
| [SORT\_INFO](sort_info) | 用于 [EE\_SORT](../message/ee_sort) 消息。 |
| [STRING\_BUF](string_buf) | 用于 [EE\_INFO](../message/ee_info) 消息。 |
| [SPLIT\_COLUMN\_INFO](split_column_info) | 用于 [EE\_SPLIT\_COLUMN](../message/ee_split_column) 消息。 |
| [TEMP\_INFO](temp_info) | 用于 [EE\_EDIT\_TEMP](../message/ee_edit_temp) 消息以及 [EVENT\_TEMP\_SAVING 事件](../event/index) 中。 |
| [TIP\_INFO](tip_info) | 用于 [EE\_SHOW\_TIP](../message/ee_show_tip) 消息。 |
| [TOOLBAR\_INFO](toolbar_info) | 用于 [Editor\_ToolbarOpen 内联函数](../macro/editor_toolbaropen) ( [EE\_TOOLBAR\_OPEN 消息](../message/ee_toolbar_open)) 以及与自定义工具栏相关的事件中。 |
| [UNICODE\_NAME\_INFO](unicode_name_info) | 用于 [EE\_GET\_UNICODE\_NAME 消息](../message/ee_get_unicode_name)。 |
| [UNPIVOT\_INFO](unpivot_info) | 用于 [EE\_UNPIVOT](../message/ee_unpivot) 消息中。 |
