# 强大的标签页功能

由于所有窗口都在同一视窗中以单一进程运行，而不像之前版本仅仅合并窗口，被增强后的标签页功能现在更加稳定。

- 您能选择标签页 或按钮 样式。
- 您可以点击标签右边的关闭按钮（×）来关闭活动的文档。
- 新版的EmEditor支持群组窗口，并且您还可以拖放任何标签页到另一个群组中或创建一个新的群组。

- 您可以自定义双击标签页或用鼠标中键单击标签页所要执行的命令。

- 您能自定义标签页上的上下文菜单。
- 您还可以自定义新的标签页插入位置，当关闭一个标签页时要激活哪一个标签页，以及用哪一个窗口来打开新文档。
- 标签页能通过文件名称，类型，修改时间或最近激活的时间来排序。您也可以用这些条件来指定自动排列 的方式。
- 当EmEditor窗口有太多标签页时，标签页栏能被延伸到多行或自动调节标签页宽度。
