# 垂直选取编辑

垂直选取编辑功能让您能不用正则表达式就快速地取代相似的行。

例如，你要处理的一个文件有如下所示的文本:

name1

name2

name3

name4

现在，你想要把“name1”到“name4”的首字母大写。在“name1”的左边点击一下，然后按住ALT键并垂直选择所有的“n”（或者你也可以按 CTRL + SHIFT + F8），然后再按一次向右键（RIGHT），按四次向下键（DOWN）。

name1

name2

name3

name4

当所有的“n”被选取时，输入“N”。

Name1

Name2

Name3

Name4

你会看到所有“n”都被“N”替换了。
