# 注册的设备页面

[注册的设备页面](https://support.emeditor.com/zh-hans/account/devices)包含您已注册过 EmEditor 的所有设备的列表。

## Columns

每个设备记录包含以下信息。

1. **设备 ID**：这是此设备的标识符，是注册表项、用户 ID 和机器 ID 的唯一组合。当您把 EmEditor 注册到新设备时，会创建一条新的设备记录。
2. **注册密钥**：这是用于注册该设备的注册密钥。
3. **机器 ID**：此 ID 标识了这是一个 Windows 安装。该值是从 `HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Cryptography\MachineGuid` 的 Windows 注册表值复制而来的。
4. **注册日期**：您在设备上注册 EmEditor 的日期。
5. **最后验证日期**：EmEditor 最后一次连接到我们的服务器以验证注册的日期。每次当您在设备上打开 EmEditor 时都会进行一次验证。EmEditor 可能会在验证期间连接到我们的服务器，但这并不是每次都会发生。因此，最后一次验证日期可能不是您上次打开 EmEditor 的时间。
6. **安装类型**：该值取决于您如何安装 EmEditor，可以是“桌面版”、“便携版”、或“商店应用版”。
7. **标签**：该值包含您在 **注册产品** 对话框的 **标签** 字段中写入的内容。该标签可帮助您了解您的哪台计算机与此设备记录相对应。
8. **编辑/注销**：单击“编辑”可以更改标签、检查设备是否已注册以及注销设备。

## 设备限制

您可以为每个授权注册一定数量的设备。该限制在您的 EULA 中注明。如果您注册的设备数量超过允许的数量，您将在 EmEditor 中看到一条通知。

## 注销

要从一个设备上注销 EmEditor，请到该设备上卸载 EmEditor 或单击 **帮助** 菜单中的 **降级**。您还可以单击此表中的“编辑”来注销此设备。
