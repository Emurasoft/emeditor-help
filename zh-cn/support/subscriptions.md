# Stripe 订阅页面

所有与您的客户中心账户关联的 Stripe 订阅都列在 [Stripe 订阅页面](https://support.emeditor.com/en/account/subscriptions)。该页面还包含一个[购买新的 Stripe 订阅的链接](https://support.emeditor.com/en/stripe_payment)，用于您的账户。

以下部分描述了 Stripe 订阅列表中每一列的含义。

1. **Stripe 订阅 ID**：Stripe 订阅的 ID
2. **创建于**：订阅创建的日期
3. **数量**：此订阅的许可数量
4. **状态**：可用订阅标记为 `active`（活跃）。如果您取消了订阅，它将标记为 `canceled`（已取消）。[查看其他状态值](https://docs.stripe.com/api/subscriptions/object#subscription_object-status)。
5. **注册的设备**：与此订阅关联的注册的设备数量。您可以查看[注册的设备](https://support.emeditor.com/en/account/devices)以获取有关您设备的详细信息。

## 支持

如果订阅列表包含不正确的信息，请[联系我们](https://www.emeditor.com/support/#contact)。
