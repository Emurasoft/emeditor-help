# Q. 在“文件”菜单下“重新载入”的子菜单中，除了相同编码，系统默认，UTF-16LE, UTF-16BE, UTF-8, 和 UTF-7 之外，没有其他编码可用。我怎么添加更多的编码页？

如果您没有适当地安装 EmEditor，只是把emeditor.exe文件从一台电脑复制到另一台电脑并且尝试运行程序，那么一些系统内设置的编码可能不会在 EmEditor 复制版中显示。

如果您使用的是 Windows NT/2000/XP，要安装另外的编码，您可以作为高级用户 (Power User) 或管理员身份登入，并在 **“工具”** 菜单下，选择
[**“定义编码”** 命令](../../cmd/tools/define_code_page) 然后点击 **「重置」** 按钮。

编码的可用性取决于安装在 Windows 中的语言系统文件。如果您无法在 EmEditor 中找到您要的那种编码，您可以到“控制面板”中的“区域和语言选项”安装额外的语言文件。
当语言系统文件安装到 Windows 系统中后，再次到 EmEditor 中到 **“工具”** 菜单下选择 [**“定义编码”** 命令](../../cmd/tools/define_code_page)，然后点击 **「重置」** 按钮。
