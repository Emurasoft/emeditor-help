# Q. 当我尝试打开一个非常大的文件时，会出现类似于“页面文件太小，无法完成此操作”的错误消息。如何解决此错误？

您将需要增加物理或虚拟内存才能打开一个非常大的文件。我们建议先增加 PC 中的物理内存（RAM）。如果错误仍然存在，或者您无法增加物理内存大小，则可以增加虚拟内存。要增加虚拟内存，请按照以下步骤操作：

1. 按 **Win**(⊞)+ **R** 键来打开 **运行** 对话框，输入 **SystemPropertiesPerformance.exe** 来显示 **性能选项** 对话框。
2. 选择 **性能选项** 对话框的 **高级** 选项卡。
3. 单击 **虚拟内存** 框中的 **更改** 按钮。
4. 清除 **自动管理所有驱动器的分页文件大小** 复选框。
5. 选择 **自定义大小**，然后在 **初始大小（MB）** 和 **最大大小（MB）** 框中输入建议的数字。我们建议在 **初始大小（MB）** 和 **最大大小（MB）** 框中输入相同的大数字。例如，在两个框中都输入 80000。
6. 单击确定，然后重新启动 Windows。
7. 尝试再次使用 EmEditor 打开大文件。如果错误仍然存在，请重复上述步骤以增加虚拟内存大小，直到错误消失。
