# Q. 怎么用 Western European 而不是 UTF-8 来新建 Java 文件或 XML 文件?

在默认情况下，Java 文件和 XML 文件会作为 UTF-8 打开当您从文件菜单中选择
[**新建 Java 或 XML 文件** 命令](../../cmd/file/file_new_config) 时。您可以更改这一默认设定。首先，打开 Java (或 XML)文件的 **配置属性**，选择 **文件** 页面，点击 **「新建文件…」** 按钮，从
[**编码** 下拉列表框](../../dlg/properties/file/new_details/index) 中选择 **“系统默认”**。要注意的是新建文件所设的编码可能会与正在打开的或正在保存的文件不同。
