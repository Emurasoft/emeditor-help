# Q. 怎么能设定一个默认文件夹，让“打开”对话框一直从这个文件夹中打开文件？

选择工具菜单下的“ **自定义**”，然后在“ **文件**”页面上勾选“ **记住最后打开的文件夹**”。

另外，您也可以右击 EmEditor 快捷方式，打开 **属性**。在“ **目标**”(Target)文本框中的路径后加/cd。例如，如果 EmEditor 的安装文件夹是“C:\\Program Files\\EmEditor”，那么请输入"C:\\Program Files\\EmEditor\\emeditor.exe" /cd，并且在“ **起始位置**”(Start in)文本框中输入您要打开的文件夹路径。
