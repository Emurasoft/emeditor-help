# Q. 能不能在切换标签时，让 EmEditor 从左到右移动文档标签？

可以。选择 **“工具”** 菜单下的 **“自定义”**，到
[**“窗口”** 页面](../../dlg/customize/window/index) 上，清除对 **“点击‘下一个文档’命令时切换到上次使用的文档窗口”** 复选框的勾选。当这个复选框没有被勾选时，您可以在标签文档之间从左到右( **CTRL** + **TAB**)或从右到左( **CTRL** + **SHIFT** + **TAB** )切换文档。当这个复选框被勾选时，按下 **CTRL** + **TAB** 和 **CTRL** +SHIFT+ **TAB** ，会在活动的文档和上一次查看的文档之间切换。
