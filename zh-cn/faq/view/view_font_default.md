# Q. 当我用默认设置打开一个文件时，文件变为不可读；如果我从“查看”下的“字体分类”选项中选择另一个字符集，文件看起来更糟糕。但是，如果我在打开文件时选择一个合适的编码，文件则能正常显示。这是怎么一回事？

EmEditor的内部处理编码是Unicode。当您打开一个ANSI文件时，EmEditor可以把ANSI文本转换为Unicode文本。当您要打开一个文件时，应该在“打开” 对话框中选择一个编码。如果您不选择编码，EmEditor会用 [系统默认编码](../../glossary/systemdefaultencoding) 来打开文件。
所以，要用一个不同的编码来打开文件，您需要指定打开该文件的编码。您可以在打开文件之后更改编码，到文件 菜单下选择 [重新载入 命令](../../cmd/file/file_reload_defined)，然后指定一个不同的编码。
