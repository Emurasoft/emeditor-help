# Q. 怎么检查拼写？

在 EmEditor Professional v10 后的版本中，您可以不使用任何插件就能检查拼写。在 EmEditor 的上下文菜单中(右键菜单)，选择进行“ **拼写检查**”。如果无法看到这一选项，请到“工具”下的“ **自定义菜单**”中选择“ **上下文菜单**”，然后按「 **重置**」按钮。EmEditor 默认安装了英语(美国)字典。其他语言的字典请到 extensions.OpenOffice.org ( [http://extensions.openoffice.org/](http://extensions.openoffice.org/)) 中下载。在下载一个字典文件后，请把扩展名从. oxt 改为 .zip，然后解压缩这个zip文件，并且把 \*.dic 和 \*.aff 文件复制到 EmEditor 安装文件夹中的 Dictionaries (字典) 子文件夹中(通常在 %LOCALAPPDATA%\\Programs\\EmEditor\\Dictionaries 或 C:\\Program Files\\EmEditor\\Dictionaries)。
