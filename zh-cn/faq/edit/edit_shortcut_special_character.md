# Q. 怎么给特殊字符设置快捷键？

许多特殊字符已经有默认的快捷键方式了。请点击下面的链接查看已设置的一些特殊字符的快捷键:

[插入特殊字符](../../howto/edit/edit_special_char)

除此之外，您可以用宏给任何您常用到的字符设置快捷键。例如，如果想要用一个快捷键插入ä，您可以编写一个宏(这个例子中，我们用JavaScript编写):

document.selection.Text="ä";

然后把这个文件保存为 **InsertA.jsee**。在 **“宏”** 下拉单中点击 **“选择...”**，并打开InsertA.jsee。当运行这个宏时，EmEditor便会自动插入这个字符。您只需要打开“工具”下的“配置属性”，到“键盘”页面上，在“类别”下拉列表中选择“我的宏”，并自定义一个快捷键给这个宏，设定完成后，你就能用你自定义的快捷键来插入这个特殊字符了。
