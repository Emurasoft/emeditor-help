# 安装插件

1. 从网上下载你想要安装的插件。Emurasoft 网站有很多可用的插件。
2. 按照附带的下载插件的说明正确地安装它们。通常，把一个带 **.DLL** 扩展名的文件复制到 EmEditor 目录的 **插件** 文件夹中 (一般而言是 C:\\Program Files\\EmEditor3\\PlugIns)， 就完成了安装。
3. 当你运行 EmEditor 时，你安装的插件菜单项目会被添加到 **工具** 菜单下，工具栏上也会出现一个按钮。

### 提示

- 用最新版的 EmEditor 以及插件可确保它们一起正常工作。
- 您可以自定义工具栏按钮，菜单和键盘快捷键。
