# 更改换行方式

在工具栏上点击
![No Wrap](../../images/wrapnone.gif)
不换行，
![Wrap by Characters](../../images/wrapbychar.gif) 按指定字符数换行，
![Wrap by Window](../../images/wrapbywindow.gif) 按窗口大小换行，或
![Wrap by Page](../../images/wrapbypage.gif)
按页面大小换行。

## 提示

- 要更改按字符换行的字符数，点击
![Properties for Current Configuration](../../images/properties.gif)
（如果你只想更改当前配置）或
![Properties for All Configuration](../../images/allproperties.gif)
（如果你想要更改所有配置）并到 **常规** 页面上更改 **标准行边距** 以及 **引用行边距。**
