# 下一更改命令

## 摘要

跳转至下一更改。

## 说明

在比较模式下，跳转至下一更改。

## 运行方法

- 默认菜单: **比较** \> **下一更改**
- [所有命令](../tools/all_commands): **比较** \> **下一更改**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: F7

## 插件命令ID

```
EEID_COMPARE_NEXT (4510)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4510);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4510
```
