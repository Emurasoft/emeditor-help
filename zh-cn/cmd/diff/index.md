# 比较类别

|     |     |
| --- | --- |
| **[比较](compare_direct)** | 不指定参数比较最后访问的两个文档。 |
| **[按参数比较](compare_options)** | 以指定参数比较最后访问的两个文档。 |
| **[重新扫描](compare_rescan)** | 重新扫描并刷新所比较的文档。 |
| **[同步滚动](sync_scroll_direct)** | 不指定参数直接同步滚动两个最近访问的文档。 |
| **[按参数同步滚动](sync_scroll_options)** | 启动同步滚动向导以同步滚动最近查看的两个文档。 |
| [拆分视图](split_view_direct) | 在一个窗口中拆分显示两个最近查看的文档。 |
| **[重置](compare_reset)** | 重置比较或同步滚动模式并清除比较结果。 |
| **[忽略前导空白](ignore_lead_space)** | 比较文档时忽略每行前导的空白。 |
| **[忽略末尾空白](ignore_trail_space)** | 比较文档时忽略每行末尾的空白。 |
| **[忽略内嵌空白](ignore_embedded_space)** | 比较文档时忽略每行内嵌的空白。 |
| **[忽略大小写](ignore_cases)** | 比较文档时忽略大小写。 |
| **[忽略换行符](ignore_returns)** | 比较文档时忽略换行符。 |
| **[忽略注释](ignore_comment)** | 比较文档时忽略注释。 |
| **[忽略编码](ignore_encoding)** | 比较文档时忽略编码。 |
| **[同步垂直滚动](sync_vert)** | 同步垂直滚动。 |
| **[同步水平滚动](sync_horz)** | 同步水平滚动。 |
| **[同步光标位置](sync_caret)** | 同步光标位置。 |
| **[完成后还原窗口位置](sync_restore_pos)** | 当比较或同步完成后还原窗口位置。 |
| **[保存参数](compare_save_options)** | 保存参数来比较或同步滚动。 |
| **[上一更改](compare_prev)** | 跳转至上一更改。 |
| **[下一更改](compare_next)** | 跳转至下一更改。 |
| **[上一更改行](compare_prev_line)** | 跳转至上一个更改的行。 |
| **[下一更改行](compare_next_line)** | 跳转至下一个更改的行。 |
| **[上一更改的字符](compare_prev_char)** | 跳转到上一更改的字符处。 |
| **[下一更改的字符](compare_next_char)** | 跳转到下一更改的字符处。 |
| **[为有改动的行设置书签](compare_bookmark)** | 为所有已修改的行添加书签。 |
| **[生成报告](compare_generate_report)** | 根据当前比较结果生成报告文件。 |
| **[跳转到其他](compare_jump_to_other)** | 跳转到正在与当前文档比较或滚动的另一个文档。 |
