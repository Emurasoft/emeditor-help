# 忽略大小写命令

## 摘要

比较文档时忽略大小写。

## 说明

比较文档时忽略大小写（不区分大小写）。

## 运行方法

- 默认菜单:比较 \>忽略大小写
- [所有命令](../tools/all_commands):比较 \>忽略大小写
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_IGNORE_CASES (4501)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4501);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4501
```
