# 复制到另一窗格命令

## 摘要

将此文档中的这个更改复制到另一窗格的文档中。

## 说明

将此文档中的这个更改复制到另一窗格的文档中。

## 运行方法

- 默认菜单: **比较** \> **复制到另一窗格**
- [所有命令](../tools/all_commands): **比较** \> **复制到另一窗格**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COPY_TO_OTHER (23247)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23247
```
