# 拆分视图命令

## 摘要

在一个窗口中拆分显示两个最近查看的文档。

## 说明

在一个窗口中拆分显示两个最近查看的文档。

## 运行方法

- 默认菜单: **比较** \> **拆分视图**
- [所有命令](../tools/all_commands): **比较** \> **拆分视图**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SPLIT_VIEW_DIRECT (4085)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4085);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4085
```
