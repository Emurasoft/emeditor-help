# 完成后还原窗口位置命令

## 摘要

当比较或同步完成后还原窗口位置。

## 说明

当比较或同步完成或取消后还原原始窗口/标签位置。

## 运行方法

- 默认菜单: **比较** \> **完成后还原窗口位置**
- [所有命令](../tools/all_commands): **比较** \> **完成后还原窗口位置**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SYNC_RESTORE_POS (4546)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4546);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4546
```
