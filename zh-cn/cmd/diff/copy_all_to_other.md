# 全部复制到其他命令

## 摘要

将此文档中的全部更改复制到其他文档。

## 说明

将此文档中的全部更改复制到其他文档。

## 运行方法

- 默认菜单: **比较** \> **全部复制到其他**
- [所有命令](../tools/all_commands): **比较** \> **全部复制到其他**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COPY_ALL_TO_OTHER (23248)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23248
```
