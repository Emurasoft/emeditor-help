# 按参数同步滚动命令

## 摘要

启动同步滚动向导以同步滚动最近查看的两个文档。

## 说明

启动同步滚动向导，用指定参数来同步滚动最近查看的两个文档。

## 运行方法

- 默认菜单:比较 \>按参数同步滚动
- [所有命令](../tools/all_commands):比较 \>按参数同步滚动
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SYNC_SCROLL_OPTIONS (4496)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4496);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4496
```
