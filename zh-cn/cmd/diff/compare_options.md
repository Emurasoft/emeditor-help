# 按参数比较命令

## 摘要

以指定参数比较最后访问的两个文档。

## 说明

选择指定参数来比较两个最近查看的文档。

## 运行方法

- 默认菜单:比较 \>按参数比较
- [所有命令](../tools/all_commands):比较 \>按参数比较
- 工具栏:  无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COMPARE_OPTIONS (4493)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4493);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4493
```
