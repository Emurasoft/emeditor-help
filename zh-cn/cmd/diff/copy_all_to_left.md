# 全部复制到左边命令

## 摘要

将所有更改复制到左侧。

## 说明

将所有更改复制到左侧。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **比较** \> **全部复制到左边**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COPY_ALL_TO_LEFT (23247)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23247
```
