# 跳转到其他命令

## 摘要

跳转到正在与当前文档比较或滚动的另一个文档。

## 说明

跳转到正在与当前文档比较或滚动的另一个文档。

## 运行方法

- 默认菜单:比较 \>跳转到其他
- [所有命令](../tools/all_commands):比较 \>跳转到其他
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COMPARE_JUMP_TO_OTHER (4048)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4048);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4048
```
