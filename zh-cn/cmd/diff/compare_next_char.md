# 下一更改的字符命令

## 摘要

跳转到下一更改的字符处。

## 说明

在比较模式下，跳转到下一更改的字符处。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **比较** \> **下一更改的字符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+F7

## 插件命令ID

```
EEID_COMPARE_NEXT_CHAR (4090)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4090);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4090
```
