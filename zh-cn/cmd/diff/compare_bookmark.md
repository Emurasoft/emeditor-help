# 为有改动的行设置书签命令

## 摘要

为所有已修改的行添加书签。

## 说明

在比较模式下，为所有已修改的行添加书签。

## 运行方法

- 默认菜单: **比较** \> **为有改动的行设置书签**
- [所有命令](../tools/all_commands): **比较** \> **为有改动的行设置书签**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COMPARE_BOOKMARK (4593)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4593);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4593
```
