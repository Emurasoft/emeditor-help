# 全部复制到右边命令

## 摘要

将所有更改复制到右侧。

## 说明

将所有更改复制到右侧。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **比较** \> **全部复制到右边**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_COPY_ALL_TO_RIGHT (23248)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23248
```
