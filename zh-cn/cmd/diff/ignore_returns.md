# 忽略换行符命令

## 摘要

比较文档时忽略换行符。

## 说明

比较文档时忽略换行符。

## 运行方法

- 默认菜单:比较 \>忽略换行符
- [所有命令](../tools/all_commands):比较 \>忽略换行符
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_IGNORE_RETURNS (4502)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4502);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4502
```
