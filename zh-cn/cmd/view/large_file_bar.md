# 切换大文件控制器命令

### 摘要

> 显示或隐藏大文件控制器。

### 说明

> 显示或隐藏大文件控制器。当打开一个大文件时，EmEditor 会显示大文件控制器，如果 **当打开一个大文件时，显示大文件控制器** 复选框（在自定义对话框 [文件页面](../../dlg/customize/file/index) 上）被勾选了。

### 运行方法

- 默认菜单: **查看** \> **大文件控制器**
- [所有命令](../tools/all_commands): **查看** >
**大文件控制器**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_LARGE\_FILE\_BAR (4449)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4449);

#### \[VBScript\]

> editor.ExecuteCommandByID 4449
