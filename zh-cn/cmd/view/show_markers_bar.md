# 标记工具栏命令

## 摘要

显示或隐藏标记工具栏。

## 说明

显示或隐藏标记工具栏。标记工具栏会显示所有已定义的标记。

## 运行方法

- 默认菜单: **查看** > **工具栏** \> **标记**
- [所有命令](../tools/all_commands): **查看** > **工具栏** \> **标记**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_MARKERS_BAR (4561)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4561);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4561
```
