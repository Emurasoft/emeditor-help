# 清除标记匹配计数命令

## 摘要

清除当前文档中所有标记的匹配计数。

## 说明

清除当前文档中所有标记的匹配计数。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** \> **标记**
\> **清除匹配计数**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MARKER_CLEAR_COUNT (4043)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4043);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4043
```
