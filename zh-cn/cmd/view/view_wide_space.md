# 全角空格命令

## 摘要

显示或隐藏全角空格。

## 说明

显示或隐藏全角空格。在默认设定下，EmEditor不显示全角空格标记（蓝色长方形）。

## 运行方法

- 默认菜单:查看 \>标记 \>全角空格
- [所有命令](../tools/all_commands):查看 \>标记 \>全角空格
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_WIDE_SPACE (4536)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4536);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4536
```
