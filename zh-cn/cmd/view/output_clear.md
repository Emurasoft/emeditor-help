# 清空输出栏窗口命令

## 摘要

清除输出栏窗口中的内容。

## 说明

清除输出栏窗口中的内容。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** >
**输出栏** >
**清空输出栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_OUTPUT_CLEAR (4452)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4452);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4452
```
