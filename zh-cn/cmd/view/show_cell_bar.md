# 单元格工具栏命令

## 摘要

显示或隐藏单元格工具栏。

## 说明

显示或隐藏单元格工具栏。

## 运行方法

- 默认菜单: **查看** >
**工具栏** \> **单元格工具栏**
- [所有命令](../tools/all_commands): **查看** >
**工具栏** \> **单元格工具栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_CELL_BAR (3967)```

## 宏

editor.ExecuteCommandByID(3967);

## \[VBScript\]

```
editor.ExecuteCommandByID 3967
```
