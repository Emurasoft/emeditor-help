# 按类型排列命令

## 摘要

按类型排列标签页。

## 说明

按类型排列标签页。如果勾选了 [**自动排列** 命令](auto_sort)，每当一个文件被打开或保存为新的文件名时，标签页就会按这个命令自动排列。如果 [**自动排列** 命令](auto_sort) 没有被勾选的话，这个命令只会执行一次。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** \> **标签排列方式**
\> **类型**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_TYPE (4399)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4399);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4399
```
