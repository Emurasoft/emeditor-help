# 缩进参考线命令

## 摘要

显示或隐藏缩进参考线。

## 说明

显示或隐藏缩进参考线。

## 运行方法

- 默认菜单: **查看** \> **标记** \> **缩进参考线**
- [所有命令](../tools/all_commands): **查看** \> **标记** \> **缩进参考线**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_INDENT_GUIDES (4539)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4539);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4539
```
