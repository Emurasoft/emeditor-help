# 升序命令

## 摘要

按升序排列。

## 说明

按升序排列标签页。如果勾选了 [自动排列 命令](auto_sort)，标签页会按特定顺序重新排列。如果 [自动排列 命令](auto_sort) 没有被勾选的话，标签页不会重新排列直到再次指定排列顺序。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):查看 \>标签排列方式
\>升序
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_ASCENDING (4402)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4402);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4402
```
