# 网页浏览器命令

## 摘要

显示或隐藏网页浏览器。

## 说明

显示或隐藏网页浏览器。

## 运行方法

- 默认菜单: **查看** \> **网页浏览器**
- [所有命令](../tools/all_commands): **查看** \> **网页浏览器**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

```
EEID_VIEW_WEB (23243)```

## 宏

### \[JavaScript\]

```
WebBar.Visible = !WebBar.Visible;
```

### \[VBScript\]

```
WebBar.Visible = Not WebBar.Visible
```
