# 重设字体大小命令

## 摘要

重设显示的字体大小。

## 说明

重设显示的字体大小。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):查看 \>字体 >
重设字体大小
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+0

## 插件命令ID

```
EEID_RESET_FONT_SIZE (3876)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3876);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3876
```
