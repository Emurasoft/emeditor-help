# 用户自定义参考线命令

## 摘要

显示或隐藏用户自定义参考线。

## 说明

显示或隐藏用户自定义参考线。

## 运行方法

- 默认菜单:查看 \>标记 \>用户自定义参考线
- [所有命令](../tools/all_commands):查看 \>标记 \>用户自定义参考线
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_USER_DEFINED_GUIDES (3863)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3863);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3863
```
