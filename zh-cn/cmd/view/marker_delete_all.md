# 删除全部标记命令

## 摘要

删除全部标记。

## 说明

删除全部标记。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **查看** \> **标记**
\> **删除全部**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MARKER_DELETE_ALL (4587)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4587);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4587
```
