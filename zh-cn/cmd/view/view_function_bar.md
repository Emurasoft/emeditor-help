# 切换快捷功能栏命令

## 摘要

显示或隐藏快捷功能栏。

## 说明

在EmEditor窗口低端，显示或隐藏快捷功能栏。

## 运行方法

- 默认菜单: **查看** \> **快捷功能栏**
- [所有命令](../tools/all_commands): **查看** >
**快捷功能栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_FUNCTION_BAR (4448)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4448);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4448
```
