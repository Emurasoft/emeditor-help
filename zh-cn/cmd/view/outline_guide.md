# 大纲导航命令

## 摘要

显示或隐藏大纲导航。

## 说明

显示或隐藏大纲导航。

## 运行方法

- 默认菜单: **查看** \> **大纲导航**
- [所有命令](../tools/all_commands): **查看** \> **大纲** \> **大纲导航**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_OUTLINE_GUIDE (3873)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3873);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3873
```
