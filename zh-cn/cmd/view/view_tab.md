# 制表符命令

## 摘要

显示或隐藏制表符标记。

## 说明

显示或隐藏制表符标记。在默认设定下，EmEditor 不显示制表符标记（蓝色，向右的箭头）。

## 运行方法

- 默认菜单: **查看** \> **标记** \> **制表符**
- [所有命令](../tools/all_commands): **查看** \> **标记** \> **制表符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_TAB (4534)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4534);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4534
```
