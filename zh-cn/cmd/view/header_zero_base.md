# 用从0开始的数字字符作为列标题命令

## 摘要

使用从0开始的数字字符作为列标题。

## 说明

使用从0开始的数字字符作为列标题。

## 运行方法

- 默认菜单:CSV \>列标题类型 \>0, 1, 2, ...
- [所有命令](../tools/all_commands):CSV \>列标题类型 \>0, 1, 2, ...
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HEADER_ZERO_BASE (3988)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3988);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3988
```
