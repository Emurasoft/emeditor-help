# CSV 工具栏命令

## 摘要

显示或隐藏 CSV 工具栏。

## 说明

显示或隐藏 CSV 工具栏。

## 运行方法

- 默认菜单: **查看** > **工具栏** \> **CSV**
- [所有命令](../tools/all_commands): **查看** > **工具栏** \> **CSV**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_SV_BAR (3890)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3890);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3890
```
