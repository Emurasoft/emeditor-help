# 查找工具栏命令

## 摘要

显示或隐藏查找工具栏。

## 说明

显示或隐藏查找工具栏。查找工具栏让你能查找或取代文档中的字符串。

## 运行方法

- 默认菜单: **查看** > **工具栏** \> **查找**
- [所有命令](../tools/all_commands): **查看** > **工具栏** \> **查找**
- 工具栏:
![](../../images/emeditor12_toggle_find_bar_button.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_FIND_BAR (4559)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4559);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4559
```
