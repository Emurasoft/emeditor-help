# 启用快捷键 (输出栏) 命令

## 摘要

启用或停用输出栏内的快捷键。

## 说明

启用或停用输出栏内的快捷键。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):查看 >
输出栏 \>启用快捷键
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_TOGGLE_OUTPUT_SHORTCUTS (4549)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4549);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4549
```
