# 外部工具工具栏命令

## 摘要

显示或隐藏工具栏上的外部工具。

## 说明

显示或隐藏工具栏上的外部工具。外部工具工具栏会显示所有被用户定义的外部工具。

## 运行方法

- 默认菜单:查看 >
工具栏 \>外部工具工具栏
- [所有命令](../tools/all_commands):查看 >
工具栏 \>外部工具工具栏
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SHOW_TOOLS_BAR (4319)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4319);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4319
```
