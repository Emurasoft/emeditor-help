# 带不同标记的 CR 和 LF 命令

## 摘要

显示带不同标记的 CR 和 LF。

## 说明

显示带不同标记的 CR 和 LF。在默认设定下，EmEditor 会用同样的标记显示 CR 和 LF。

## 运行方法

- 默认菜单:查看 \>标记 \>显示带不同标记的 CR 和 LF。
- [所有命令](../tools/all_commands):查看 \>标记 \>显示带不同标记的 CR 和 LF。
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_CR_LF_SEPARATE (4537)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4537);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4537
```
