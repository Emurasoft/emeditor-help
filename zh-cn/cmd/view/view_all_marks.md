# 所有标记命令

## 摘要

显示或隐藏所有标记。

## 说明

显示或隐藏所有标记。

## 运行方法

- 默认菜单:查看 \>标记 \>所有标记
- [所有命令](../tools/all_commands):查看 >
标记 \>所有标记
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_VIEW_ALL_MARKS (4040)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4040);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4040
```
