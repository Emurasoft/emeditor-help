# 插入分音符命令

## 摘要

插入一个带有分音符(元音变音符)的字符。

## 说明

在选择这个命令后，在光标处插入一个带有分音符的字符，也被称为元音变音符(ä，ë，ï，ö，ü，ÿ，Ä，Ë，Ï，Ö，Ü 或 Ÿ)通过输入 a，e，i，o，u，y，A，E，
I，O，U 或 Y。

## 运行方法

- 默认菜单: **插入** \> **音调符号** \> **分音符**
- [所有命令](../tools/all_commands): **插入** \> **音调符号** \> **分音符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL + SHIFT + ;

## 插件命令ID

```
EEID_INSERT_DIAERESIS (4307)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4307);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4307
```
