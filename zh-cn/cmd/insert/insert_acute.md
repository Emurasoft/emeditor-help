# 插入锐音符命令

## 摘要

插入一个带有锐音符的字符。

## 说明

在选择这个命令后，在光标处插入一个带有锐音符的字符(á，é，
í，ó，ú，ý，ś，ź，Á，É，Í，
Ó，Ú，Ý，Ś 或 Ź)通过输入 a，e，i，o，u，y，s，z，A，E，I，O，U，Y，S 或 Z；一个单引号(’)通过输入 \` ；或一个双引号(”)通过输入 " 。

## 运行方法

- 默认菜单: **插入** \> **音调符号** \> **锐音符**
- [所有命令](../tools/all_commands): **插入** \> **音调符号** \> **锐音符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+'

## 插件命令ID

```
EEID_INSERT_ACUTE (4304)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID (4304);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4304
```
