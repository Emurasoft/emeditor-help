# 插入版权符命令

## 摘要

插入版权符号。

## 说明

在光标处插入一个版权符号(©)。

## 运行方法

- 默认菜单: **插入** \> **标记** \> **版权符**
- [所有命令](../tools/all_commands): **插入** \> **标记** \> **版权符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT+CTRL+C

## 插件命令ID

```
EEID_INSERT_COPYRIGHT (4314)```

## 宏

### \[JavaScript\]

```
document.selection.Text="©";
```

### \[VBScript\]

```
document.selection.Text="©"
```
