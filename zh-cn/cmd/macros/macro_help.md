# 宏参考手册命令

## 摘要

显示 EmEditor 宏参考说明。

## 说明

显示 EmEditor 帮助中的宏参考说明。

## 运行方法

- 默认菜单: **宏** \> **参考手册**
- [所有命令](../tools/all_commands): **宏**
\> **参考手册**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MACRO_HELP (4360)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4360);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4360
```
