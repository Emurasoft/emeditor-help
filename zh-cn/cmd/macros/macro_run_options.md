# 使用临时选项运行宏命令

## 摘要

使用临时选项运行宏。

## 说明

这个命令会显示 [宏临时选项 对话框](../../dlg/macro_temp_options/index)，让你可以在运行宏之前，指定重复宏的次数以及是否要停止运行如果搜索失败的话。这个命令等同于 [运行宏 命令](quick_macro_run)。

## 运行方法

- 默认菜单:宏 \>使用临时选项运行
- [所有命令](../tools/all_commands):宏
\>使用临时选项运行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MACRO_RUN_OPTIONS (4364)```

## 宏

不可用。
