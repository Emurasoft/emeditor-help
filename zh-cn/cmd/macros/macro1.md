# 宏列表命令

## 摘要

在我的宏列表中运行一个指定的宏（多个项目）。

## 说明

这个命令由多个菜单项目组成。这个命令运行一个指定的在“我的宏”目录中注册过的宏。你可以用 [自定义宏命令](customize_macro) 来设置你要使用的宏文件。

## 运行方法

- 默认菜单:宏 \> (选择宏名称)
- [所有命令](../tools/all_commands):宏
\> (选择宏名称)
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
从 EEID_MACRO1 到 EEID_MACRO1 + 1023 （从 9216 到 9216 + 1023）```

## 宏

不可用。
