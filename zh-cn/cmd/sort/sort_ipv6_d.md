# IPv6 地址降序排序命令

## 摘要

把当前列中的 IPv6 地址按降序排序。

## 说明

把当前列中的 IPv6 地址按降序排序。IPv6 地址必须采用 RFC 5952 推荐的规范文本表示形式。在配置属性 [**链接** 页面](../../dlg/properties/link/index)的 **活动字符串** 列表框中启用 **IPv6 地址**，让您能区分 IPv6 地址的文本。

## 运行方法

- 默认菜单: **编辑** \> **排序** \> **IPv6 地址降序排序**
- [所有命令](../tools/all_commands): **编辑** \> **排序** \> **IPv6 地址降序排序**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_IPV6_D (4037)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4037);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4037
```
