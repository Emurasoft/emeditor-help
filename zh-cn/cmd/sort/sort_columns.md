# 列排序命令

### 摘要

> 对 CSV 文档中的列进行排序。

### 说明

> 对 CSV 文档中的列进行排序。当选择该命令时会显示 [**列排序** 对话框](../../dlg/sort_columns/index)。

### 运行方法

- 默认菜单: **排序** \> **列排序**
- [所有命令](../tools/all_commands): **排序** \> **列排序**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_SORT\_COLUMNS (23238)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(23238);

#### \[VBScript\]

> editor.ExecuteCommandByID 23238
