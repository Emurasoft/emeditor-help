# 按出现次数升序排序命令

## 摘要

把当前列中的出现次数按升序排序。

## 说明

把当前列中的出现次数按升序排序。

## 运行方法

- 默认菜单: **排序** \> **按出现次数升序排序**
- [所有命令](../tools/all_commands): **排序** \> **按出现次数升序排序**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_OCCURRENCE_A (4001)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4001);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4001
```
