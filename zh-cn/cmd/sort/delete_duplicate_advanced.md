# 删除/把重复行设为书签 (高级) 命令

## 摘要

在删除选区或整个文档中的重复行或把重复行设为书签时，显示一个对话框来编辑设定。

## 说明

在删除选区或整个文档中的重复行或把重复行设为书签时，显示一个对话框来编辑设定。当执行该命令时，会显示 [**删除/把重复行设为书签** 对话框](../../dlg/delete_duplicate_advanced/index)，让你能设置选项。

## 运行方法

- 默认菜单: **排序** \> **删除/把重复行设为书签 (高级)**
- [所有命令](../tools/all_commands): **排序** \> **删除/把重复行设为书签 (高级)**
- 工具栏: ![](../../images/deleteduplicatelines_advanced.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_DELETE_DUPLICATE_ADVANCED (3920)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3920);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3920
```
