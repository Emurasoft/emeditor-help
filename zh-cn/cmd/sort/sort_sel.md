# 排序/删除选区内重复的拆分字符串命令

## 摘要

排序或删除选区中的重复的拆分字符串。

## 说明

排序或删除选区中的重复的拆分字符串。当选择该命令时会显示 [选区排序 对话框](../../dlg/sort_sel/index)。

## 运行方法

- 默认菜单:排序 \>排序/删除选区内重复的拆分字符串
- [所有命令](../tools/all_commands):排序 \>排序/删除选区内重复的拆分字符串
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SORT\_SEL (23237)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23237);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23237
```
