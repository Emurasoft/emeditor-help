# 重置书签命令

## 摘要

清除当前行的书签。

## 说明

清除当前行的书签。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): 无
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_BOOKMARK_RESET (4354)```

## 宏

### \[JavaScript\]

```
document.selection.ClearBookmark();
```

### \[VBScript\]

```
document.selection.ClearBookmark
```
