# 打印属性命令

## 摘要

当前配置的打印属性。

## 说明

显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**打印** 页面](../../dlg/properties/print/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **打印**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_PRINT (8972)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8972);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8972
```
