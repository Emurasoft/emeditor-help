# 删除旧的设定命令

## 摘要

删除所有不再使用的旧设定。

## 说明

删除所有不再使用的旧设定。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具
\>删除旧的设定
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_DELETE_OLD_SETTINGS (3984)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3984);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3984
```
