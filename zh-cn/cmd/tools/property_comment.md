# 高亮(2)属性命令

## 摘要

当前配置的高亮(2)属性。

## 说明

显示[配置属性](../../dlg/properties/index) 对话框中的 [高亮(2) 页面](../../dlg/properties/highlight2/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具 >
当前配置属性 \>高亮(2)
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_COMMENT (8969)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8969);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8969
```
