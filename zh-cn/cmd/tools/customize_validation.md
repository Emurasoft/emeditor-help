# 自定义语法检查命令

## 摘要

打开自定义对话框中的语法检查页面。

## 说明

显示让你能定义通用应用设置的 [**自定义** 对话框](../../dlg/customize/index) 中的语法检查页面。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**自定义** >
**语法检查**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_VALIDATION (9062)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(9062);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9062
```
