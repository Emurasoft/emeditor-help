# 启动窗口命令

## 摘要

显示启动窗口。

## 说明

显示启动窗口。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具**
> **启动窗口**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_START_WINDOW (3948)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3948);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3948
```
