# 重置所有设定命令

## 摘要

重置所有 EmEditor 设置并重启 EmEditor。

## 说明

重置所有 EmEditor 设置并重启EmEditor。

## 运行方法

- 默认菜单: **工具**
\> **重置所有设定**
- [所有命令](all_commands): **工具**
\> **重置所有设定**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_RESET_ALL (3860)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3860);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3860
```
