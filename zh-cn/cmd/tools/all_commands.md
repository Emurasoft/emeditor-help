# 所有命令命令

## 摘要

显示所有命令。

## 说明

显示一个有所有可用的 EmEditor 命令的弹出菜单。在默认设定下，这个命令位于 **工具** 菜单下

## 运行方法

- 默认菜单: **工具** \> **所有命令**
- [所有命令](all_commands): **工具** >
**所有命令**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ALL_COMMANDS (4267)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4267);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4267
```
