# 保存文件详细信息属性命令

## 摘要

显示保存文件详细信息属性。

## 说明

显示[配置属性](../../dlg/properties/index) 对话框文件 页面上的 [保存详细信息 子对话框](../../dlg/properties/file/save_details/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具 >
当前配置属性 \>保存文件详细信息
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_FILE_SAVE (8979)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(8979);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 8979
```
