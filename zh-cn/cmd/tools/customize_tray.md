# 自定义托盘图标命令

## 摘要

自定义托盘图标。

## 说明

显示 [**自定义托盘图标** 对话框](../../dlg/tray/index)，让你能自定义系统托盘图标设定。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**自定义托盘图标**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_TRAY (4273)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4273);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4273
```
