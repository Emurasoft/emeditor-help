# 外部工具列表命令

### 摘要

> 启动一个指定的外部工具（多个项目）。

### 说明

> 这个命令由多个菜单项目组成。该命令会启动指定的外部工具。可用的外部工具可以在 [**外部工具** 对话框](../../dlg/tools/index) 中被定义。

### 运行方法

- 默认菜单: **工具** \> **外部工具** \> （ **工具名称**）
- [所有命令](all_commands): **外部工具** \> （ **工具名称**）
- 工具栏: 外部工具工具栏上的按钮
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- From EEID\_TOOL1 through EEID\_TOOL1 + 255 （从 8192 到 8192 + 255）

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(8192+i);  // i 是从 0 到 255 的整数

#### \[VBScript\]

> editor.ExecuteCommandByID 8192+i  ' i 是从 0 到 255 的整数
