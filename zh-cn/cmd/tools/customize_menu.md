# 自定义菜单命令

### 摘要

> 自定义菜单。

### 说明

> 显示 [**自定义菜单** 对话框](../../dlg/menus/index)，让你能自定义 EmEditor 的菜单。

### 运行方法

- 默认菜单: **工具** \> **自定义菜单**
- [所有命令](all_commands): **工具** >
**自定义菜单**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CUSTOMIZE\_MENU (4266)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4266);

#### \[VBScript\]

> editor.ExecuteCommandByID 4266
