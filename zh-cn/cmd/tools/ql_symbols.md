# 搜索符号指令

## 摘要

打开快速启动 窗口并选择符号 模式。

## 说明

打开快速启动 窗口并选择符号 模式。

## 运行方法

- 默认菜单:搜索 \>搜索符号
- [所有命令](all_commands):搜索 \>搜索符号
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+S
