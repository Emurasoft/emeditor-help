# 自定义收藏夹命令

## 摘要

打开自定义对话框中的收藏夹页面。

## 说明

显示让你能定义通用应用设置的 [**自定义** 对话框](../../dlg/customize/index) 中的收藏夹页面。

## 运行方法

- 默认菜单: **文件** > **收藏夹** > **自定义收藏夹...**
- [所有命令](all_commands): **工具** > **自定义** > **收藏夹**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_FAVORITES (20716)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(20716);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 20716
```
