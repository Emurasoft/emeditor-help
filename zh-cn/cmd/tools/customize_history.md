# 自定义历史记录命令

## 摘要

打开自定义对话框中的历史记录页面。

## 说明

显示让你能定义通用应用设置的 [自定义 对话框](../../dlg/customize/index) 中的历史记录页面。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具 >
自定义 >
历史
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_HISTORY (9042)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(9042);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9042
```
