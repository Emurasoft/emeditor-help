# 自定义跳转列表命令

### 摘要

> 自定义跳转列表。

### 说明

> 自定义跳转列表。当你右击任务栏上的 EmEditor 图标时会跳转列表。这个功能在 Windows 7 和以后的版本中可用。

### 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**自定义跳转**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CUSTOMIZE\_JUMP\_LIST (4455)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4455);

#### \[VBScript\]

> editor.ExecuteCommandByID 4455
