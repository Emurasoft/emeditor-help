# 制表符/缩进属性命令

## 摘要

显示制表符/缩进属性。

## 说明

显示 **[配置属性](../../dlg/properties/index)** 对话框 **常规** 页面上的 [**制表符/缩进** 子对话框](../../dlg/properties/general/indent/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **制表符/缩进**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_INDENT (8977)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8977);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8977
```
