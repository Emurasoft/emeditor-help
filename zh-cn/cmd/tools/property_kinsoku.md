# 换行属性命令

### 摘要

> 当前配置的换行属性。

### 说明

> 显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**换行** 页面](../../dlg/properties/wrap/index)。

### 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **换行**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_PROPERTY\_KINSOKU (8966)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(8966);

#### \[VBScript\]

> editor.ExecuteCommandByID 8966
