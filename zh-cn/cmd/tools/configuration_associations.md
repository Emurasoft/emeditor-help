# 文件关联配置命令

## 摘要

将文件类型关联到配置项。

## 说明

显示 [**文件关联配置** 对话框](../../dlg/configuration_associations/index)，让你能选取与特定文件类型相关联的 EmEditor 配置。

## 运行方法

- 默认菜单: **工具** \> **文件关联配置**
- [所有命令](all_commands): **工具** \> **文件关联配置**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CONFIG_ASSOCIATE (3847)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3847);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3847
```
