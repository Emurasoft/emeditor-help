# 语法检查属性命令

### 摘要

> 显示语法检查属性。

### 说明

> 显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**语法检查** 页面](../../dlg/properties/validation/index)。

### 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **语法检查**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_PROPERTY\_VALIDATION (8980)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(8980);

#### \[VBScript\]

> editor.ExecuteCommandByID 8980
