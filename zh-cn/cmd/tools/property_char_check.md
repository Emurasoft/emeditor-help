# 字符检查属性命令

### 摘要

> 当前配置的字符检查属性。

### 说明

> 显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**字符检查** 页面](../../dlg/properties/char_check/index)。

### 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **字符检查**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_PROPERTY\_CHAR\_CHECK (8981)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(8981);

#### \[VBScript\]

> editor.ExecuteCommandByID 8981
