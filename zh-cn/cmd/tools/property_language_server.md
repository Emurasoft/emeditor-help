# 语言服务器属性命令

## 摘要

当前配置的语言服务器属性。

## 说明

显示[配置属性](../../dlg/properties/index) 对话框中的 [语言服务器 页面](../../dlg/properties/language_server/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具 >
当前配置属性 \>语言服务器
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_LANGUAGE_SERVER (8982)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8982);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8982
```
