# 与 EmEditor 关联命令

## 摘要

关联文件扩展名。

## 说明

显示 [**与 EmEditor 关联** 对话框](../../dlg/file_associate/index)，让用户可以指定要与 EmEditor 相关联的文件扩展名。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**与 EmEditor 关联**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILE\_ASSOCIATE (4223)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4223);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4223
```
