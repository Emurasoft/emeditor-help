# 标记属性命令

## 摘要

当前配置的标记属性。

## 说明

显示 **[配置属性](../../dlg/properties/index)** 对话框中的 [**标记** 页面](../../dlg/properties/marks/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具** >
**当前配置属性** \> **标记**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_MARK (8971)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(8971);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 8971
```
