# 自定义标记 \- 标记列表 命令

### 摘要

> 显示 **自定义标记** 对话框并选择 **标记列表** 页面。

### 说明

> 显示 [**自定义标记** 对话框](../../dlg/customize_markers/index)，并选择 **标记列表** 页面。

### 运行方法

- 默认菜单: 无
- [所有命令](all_commands): **工具**
\> **自定义标记** \> **标记列表**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CUSTOMIZE\_MARKERS\_LIST (3956)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3956);

#### \[VBScript\]

> editor.ExecuteCommandByID 3956
