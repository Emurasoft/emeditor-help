# 文件属性命令

## 摘要

当前配置的文件属性。

## 说明

显示[配置属性](../../dlg/properties/index) 对话框中的 [文件 页面](../../dlg/properties/file/index)。

## 运行方法

- 默认菜单: 无
- [所有命令](all_commands):工具 >
当前配置属性 \>文件
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_PROPERTY_FILE (8962)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(8962);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8962
```
