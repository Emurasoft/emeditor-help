# 要打开的收藏的文件命令

## 摘要

打开指定的收藏夹中的文件(多个项目)。

## 说明

打开指定的收藏夹中的文件(多个项目)。

## 运行方法

- 默认菜单: **文件** \> **收藏夹**\> **收藏的文件**
- [所有命令](../tools/all_commands): **文件** \> **收藏夹**\> **收藏的文件**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FAVORITE_FILE1 (28928)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(28928);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 28928
```
