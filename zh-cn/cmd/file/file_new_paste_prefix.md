# 新建并粘贴为引用文本命令

### 摘要

> 创建一个新的文件并粘贴剪贴版内容为引用文本。

### 说明

> 这个命令等同于 [**新建文本** 命令](file_new) 加 [**粘贴为引用文本** 命令](../edit/paste_prefix)。在默认设置下，新建的文件会使用文本(Text)配置。你可以到 [**定义配置** 对话框](../../dlg/configurations/index) 中更改这默认配置。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **新建** \> **新建并粘贴为引用文本**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_NEW\_PASTE\_PREFIX (4271)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4271);

#### \[VBScript\]

> editor.ExecuteCommandByID 4271
