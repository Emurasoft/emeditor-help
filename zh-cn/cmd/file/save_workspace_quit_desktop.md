# 保存工作区并全部关闭 (仅当前桌面) 命令

## 摘要

在当前虚拟桌面上，保存工作区并关闭所有打开的文件。

## 说明

在当前虚拟桌面上，保存工作区并关闭所有打开的文件。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **保存工作区并全部关闭 (仅当前桌面)**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SAVE_WORKSPACE_QUIT_DESKTOP (3938)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3938);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3938
```
