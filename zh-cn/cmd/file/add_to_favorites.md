# 添加到收藏夹命令

## 摘要

将当前文件添加到收藏夹列表中。

## 说明

将当前文件添加到收藏夹列表中。

## 运行方法

- 默认菜单: **文件** \> **收藏夹**\> **添加到收藏夹**
- [所有命令](../tools/all_commands): **文件** \> **收藏夹**\> **添加到收藏夹**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ADD_TO_FAVORITES (23250)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23250);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23250
```
