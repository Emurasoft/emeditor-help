# 直接关闭不保存命令

## 摘要

不保存就直接关闭当前文件。

## 说明

这个命令不保存就关闭当前文件。所有的变更都会被取消。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭** \> **直接关闭不保存**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_APP_QUIT (4120)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4120);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4120
```
