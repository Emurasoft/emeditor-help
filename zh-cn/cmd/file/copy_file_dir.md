# 复制文件所在文件夹名命令

### 摘要

> 复制当前文件所在文件夹的完整路径名到剪贴板。

### 说明

> 复制当前文件所在文件夹的完整路径名到剪贴板。如果文档没有标题，这个命令则不能被选取。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **复制**
\> **文件目录名**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_COPY\_FILE\_DIR (4327)

### 宏

#### \[JavaScript\]

> document.CopyPath();

#### \[VBScript\]

> document.CopyPath
