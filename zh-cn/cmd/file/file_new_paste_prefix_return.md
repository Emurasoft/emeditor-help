# 新建文件，粘贴为引用文本并换行命令

## 摘要

创建一个新的文件，粘贴选定内容为引用文本并换行。

## 说明

这个命令等同于 [新建文本 命令](file_new) 加 [粘贴为引用文本并换行 命令](../edit/paste_prefix_return)。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):文件 \>新建 \>新建文件，粘贴为引用文本并换行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_NEW_PASTE_PREFIX_RETURN (4272)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4272);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4272
```
