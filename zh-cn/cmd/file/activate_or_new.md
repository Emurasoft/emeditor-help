# 激活或新建命令

## 摘要

激活EmEditor或创建一个新文件（仅用于托盘图标）。

## 说明

如果EmEditor当前没有运行，这个命令会打开EmEditor。如果EmEditor当前已经运行了，这个命令会创建一个新的未命名文档。命令仅显示在托盘图标菜单中。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): 无
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无
