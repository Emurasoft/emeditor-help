# 清除默认工作区命令

## 摘要

清除默认工作区。

## 说明

这个命令会清除之前通过 [保存默认工作区 命令](save_workspace) 保存的状态。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):文件 \>工作区
\>清除默认工作区
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ERASE_WORKSPACE (4450)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4450);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4450
```
