# 显示托盘图标命令

## 摘要

显示一个新的系统托盘图标。

## 说明

这个命令会在Windows任务栏的通知区域中显示EmEditor的托盘图标。如果EmEditor的托盘图标已经存在，这个命令不会有任何作用。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **新建** \> **显示托盘图标**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_NEW_TRAY_ICON (4121)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4121);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4121
```
