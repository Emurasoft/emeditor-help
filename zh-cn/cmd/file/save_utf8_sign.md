# 保存为带签名的 UTF-8 命令

### 摘要

> 用带签名的 Unicode (UTF-8) 编码保存当前文件。

### 说明

> 这个命令会带签名的 Unicode (UTF-8) 编码保存当前文件，除非文档还未被命名。如果文档无标题，会出现一个 **另存为** 对话框，让你输入一个文件名来保存文件。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **保存**
\> **以指定编码保存** \> **保存为带签名的 UTF-8**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_SAVE\_UTF8\_SIGN (4487)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4487);

#### \[VBScript\]

> editor.ExecuteCommandByID 4487
