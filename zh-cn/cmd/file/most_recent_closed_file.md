# 最后关闭的文件命令

## 摘要

打开最后关闭的一个文档。

## 说明

打开最后关闭的一个文档。这个命令不能恢复文档中未保存的更改。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):文件 \>打开
\>最后关闭的文件
- 工具栏:
无
- 状态栏: 无
- 默认快捷键: CTRL + SHFT + T

## 插件命令ID

```
EEID_MOST_RECENT_CLOSED_FILE (4583)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4583);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4583
```
