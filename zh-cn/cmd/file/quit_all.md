# 全部关闭不保存命令

## 摘要

不保存就全部关闭当前文件。

## 说明

这个命令直接关闭所有打开的窗口。该操作将放弃所有更改。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **直接关闭 (不保存)**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_QUIT_ALL (4363)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4363);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4363
```
