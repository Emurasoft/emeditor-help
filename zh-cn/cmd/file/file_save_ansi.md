# 保存为系统默认编码命令

## 摘要

用 [系统默认编码](../../glossary/systemdefaultencoding) 保存当前文件。

## 说明

这个命令会用当前文件名称保存把文档保存为 [系统默认编码](../../glossary/systemdefaultencoding) 除非该文档未命名。如果文档没有命名，会出现一个另存为 对话框，让你能输入另存为的文件名称。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):文件 \>保存
\>以指定编码保存 \>保存为系统默认编码
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILE_SAVE_ANSI (4102)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4102);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4102
```
