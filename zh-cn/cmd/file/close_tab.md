# 关闭标签命令

## 摘要

关闭选取的标签。

## 说明

这个命令会关闭选取的标签。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭** \> **关闭标签**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID__CLOSE_TAB (23246)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23246);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23246
```
