# 验证命令

## 摘要

验证当前文档与原始文件是否相同。

## 说明

此命令显示 **[文件验证结果](../../dlg/file_verify_result/index.md)** 对话框，您可以在其中查看文件验证过程的结果。

## 运行方法

- 默认菜单: **文件** \> **验证**
- [所有命令](../tools/all_commands): **文件** \> **验证**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILE_VERIFY (23279)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(23279);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 23279
```
