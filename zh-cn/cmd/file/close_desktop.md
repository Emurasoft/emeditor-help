# 全部关闭 (仅当前桌面) 命令

### 摘要

> 关闭所有在当前虚拟桌面上打开的文件。

### 说明

> 关闭所有在当前虚拟桌面上打开的文件。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **全部关闭 (仅当前桌面)**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CLOSE\_DESKTOP (3935)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3935);

#### \[VBScript\]

> editor.ExecuteCommandByID 3935
