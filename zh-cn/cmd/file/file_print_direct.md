# 立即打印命令

### 摘要

> 使用默认的打印机而不指定任何选项打印当前文件。

### 说明

> 这个命令会立即执行打印文档文档命令，而不再显示 **打印** 对话框。打印机会使用默认设置的。这个默认设置可以在控制面板中的 **设备和打印机（Windows 8，7，还有 Vista），打印机和传真（WinXP）** 或者 **打印机（Win98 / ME）** 中设定或更改。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **打印** \> **立即打印**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_FILE\_PRINT\_DIRECT (4115)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4115);

#### \[VBScript\]

> editor.ExecuteCommandByID 4115
