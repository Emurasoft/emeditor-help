# 关闭所有其他文件命令

### 摘要

> 关闭除了目前工作的文件之外所以打开的文件。

### 说明

> 这个命令会关闭所有已打开的文件，除了当前在工作中的文件。如果存在一个已被修改的文档，则会显示一个即时消息来让你选择是否想要保存变更。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **关闭**
\> **关闭所有其他文件**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_CLOSE\_ALL\_OTHERS (4384)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4384);

#### \[VBScript\]

> editor.ExecuteCommandByID 4384
