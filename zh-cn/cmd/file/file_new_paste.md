# 新建并粘贴命令

## 摘要

创建一个新的文件并插入剪贴板中的内容。

## 说明

这个命令等同于 [**新建文本** 命令](file_new) 加 [**粘贴** 命令](../edit/edit_paste)。在默认设置下，新建的文件会使用文本(Text)配置。你可以到 [**定义配置** 对话框](../../dlg/configurations/index) 中更改这默认配置。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **文件** \> **新建** \> **新建并粘贴**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILE_NEW_PASTE (4123)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4123);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4123
```
