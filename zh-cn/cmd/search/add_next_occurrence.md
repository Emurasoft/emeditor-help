# 添加下一个匹配结果命令

## 摘要

添加指定文本的下一个匹配结果到选区。

## 说明

添加指定文本的下一个匹配结果到选区。

## 运行方法

- 默认菜单: **搜索** \> **添加下一个匹配结果**
- [所有命令](../tools/all_commands): **搜索**
\> **添加下一个匹配结果**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+R

## 插件命令ID

```
EEID_ADD_NEXT_OCCURRENCE (4604)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4604);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4604
```
