# 浏览替换表达式 (查找工具栏) 命令

## 摘要

浏览用于查找工具栏中“替换”字符串的正则表达式或转义序列。

## 说明

浏览用于查找工具栏中“替换”字符串的正则表达式或转义序列。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>查找工具栏 \>浏览替换表达式
- 工具栏: ![](../../images/find_browse_exp.png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_BROWSE_EXP_R (4568)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4568);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4568
```
