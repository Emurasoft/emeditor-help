# 将焦点移至查找工具栏命令

## 摘要

将焦点移至查找工具栏。

## 说明

将焦点移至查找工具栏。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **将焦点移至查找工具栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+D

## 插件命令ID

```
EEID_FOCUS_FIND_BAR (4565)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4565);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4565
```
