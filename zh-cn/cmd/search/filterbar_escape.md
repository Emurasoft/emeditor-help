# 使用转义序列 (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的「使用转义序列」按钮。

## 说明

切换“筛选”工具栏上的「使用转义序列」按钮。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **使用转义序列**
- 工具栏: ![](../../images/find_escape.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILTERBAR\_REG\_EXP(3909)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3909);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3909
```
