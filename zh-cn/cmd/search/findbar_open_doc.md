# 搜索所有打开的文档 (查找工具栏) 命令

## 摘要

切换查找工具栏上「搜索所有打开的文档」按钮的状态。

## 说明

切换查找工具栏上「搜索所有打开的文档」按钮的状态。当按下「搜索所有打开的文档」按钮后，用 **查找** 命令就可以在当前窗口中搜索所有打开的文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **查找工具栏** \> **搜索所有打开的文档**
- 工具栏: ![](../../images/find_open_doc.png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_OPEN_DOC (4572)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4572);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4572
```
