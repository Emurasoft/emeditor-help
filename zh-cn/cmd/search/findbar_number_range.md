# 使用数字范围 (查找工具栏) 命令

## 摘要

切换查找工具栏上「使用数字范围」按钮的状态。

## 说明

切换查找工具栏上「使用数字范围」按钮的状态。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>查找工具栏 \>使用数字范围
- 工具栏: ![](../../images/use_number_range.png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_NUMBER_RANGE (4014)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4014);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4014
```
