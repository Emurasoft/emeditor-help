# 大小写需符合 (筛选工具栏) 命令

### 摘要

> 切换“筛选”工具栏上的「大小写需符合」按钮。

### 说明

> 切换“筛选”工具栏上的「大小写需符合」按钮。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **大小写需符合**
- 工具栏: ![](../../images/find_case.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_FILTERBAR\_CASE (3907)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3907);

#### \[VBScript\]

> editor.ExecuteCommandByID 3907
