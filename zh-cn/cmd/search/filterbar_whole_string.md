# 匹配整个字符串 (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的「匹配整个字符串」按钮。

## 说明

切换“筛选”工具栏上的「匹配整个字符串」按钮。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>筛选工具栏 \>匹配整个字符串
- 工具栏:  ![](../../images/filterbar_whole_string.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILTERBAR_WHOLE_STRING (4032)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4032);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4032
```
