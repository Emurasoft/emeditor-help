# 关闭筛选工具栏命令

### 摘要

> 关闭筛选工具栏。

### 说明

> 关闭筛选工具栏。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **关闭筛选工具栏**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_CLOSE\_FILTER\_BAR (3913)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3913);

#### \[VBScript\]

> editor.ExecuteCommandByID 3913
