# 使用数字范围 (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的「使用数字范围」按钮。

## 说明

切换“筛选”工具栏上的「使用数字范围」按钮。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **筛选工具栏** \> **使用数字范围**
- 工具栏:  ![](../../images/use_number_range.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILTERBAR_NUMBER_RANGE (4013)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4013);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4013
```
