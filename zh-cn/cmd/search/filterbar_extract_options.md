# 提取选项 (筛选工具栏) 命令

## 摘要

指定提取匹配字符串命令的选项。

## 说明

指定提取匹配字符串命令的选项。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>筛选工具栏 \>提取选项
- 工具栏: ![](../../images/extract_all.png) (筛选工具栏) > 提取选项
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILTERBAR_EXTRACT_OPTIONS (4086)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4086);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4086
```
