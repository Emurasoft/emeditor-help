# 增量搜索 (查找工具栏) 命令

## 摘要

切换查找工具栏上「增量搜索」按钮的状态。

## 说明

切换查找工具栏上「增量搜索」按钮的状态。当按下「增量搜索」后，EmEditor将在用户输入查找字符串的同时搜索并高亮文档中的符合结果。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **搜索**
\> **查找工具栏** \> **增量搜索**
- 工具栏: ![](../../images/find_incremental.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_INCREMENTAL (4571)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4571);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4571
```
