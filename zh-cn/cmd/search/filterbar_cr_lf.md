# CR+LF (筛选工具栏) 命令

## 摘要

切换“筛选”工具栏上的“CR+LF”命令。

## 说明

切换“筛选”工具栏上的“CR+LF”命令。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>筛选工具栏 \>匹配换行符 \>CR+LF
- 工具栏:  ![](../../images/match_newline_characters.png) \+ CR+LF (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FILTERBAR_CR_LF (4009)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4009);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4009
```
