# 查找最短行/单元格命令

## 摘要

查找非 CSV 文档中的最短行或 CSV 文档当前列中的最短单元格。

## 说明

查找非 CSV 文档中的最短行或 CSV 文档当前列中的最短单元格。

## 运行方法

- 默认菜单:搜索 \>查找最短行/单元格
- [所有命令](../tools/all_commands):搜索
\>查找最短行/单元格
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FIND_EMPTY_OR_SHORTEST (4064)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4064);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4064
```
