# 浏览筛选表达式 (筛选工具栏) 命令

## 摘要

为在筛选工具栏中使用的筛选字符串浏览正则表达式或转义序列。

## 说明

为在筛选工具栏中使用的筛选字符串浏览正则表达式或转义序列。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>筛选工具栏 \>浏览筛选表达式
- 工具栏: ![](../../images/find_browse_exp.png) (筛选工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILTERBAR\_BROWSE\_EXP\_F (3904)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3904);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3904
```
