# 使用正则表达式 (查找工具栏) 命令

## 摘要

切换工具栏上使用「正则表达式」按钮的状态。

## 说明

切换工具栏上使用「正则表达式」按钮的状态。当这个命令被激活时， [正则表达式](../../howto/search/search_regexp_syntax) 可以被用来进一步指定搜索字符串。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):搜索
\>查找工具栏 \>使用正则表达式
- 工具栏: ![](../../images/find_reg_exp.png) (查找工具栏)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_FINDBAR_REG_EXP (4574)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4574);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4574
```
