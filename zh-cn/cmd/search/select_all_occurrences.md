# 选择所有匹配结果命令

## 摘要

选择指定文本的所有匹配结果。

## 说明

选择当前文档中指定文本的所有匹配结果。

## 运行方法

- 默认菜单:搜索 \>选择所有匹配结果
- [所有命令](../tools/all_commands):搜索
\>选择所有匹配结果
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+Shift+A

## 插件命令ID

```
EEID_SELECT_ALL_OCCURRENCES (4603)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4603);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4603
```
