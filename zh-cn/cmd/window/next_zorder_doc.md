# 最后访问的文档命令

## 摘要

切换至最后访问的文档。

## 说明

切换至最后访问的文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>切换至最后访问的文档。
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_NEXT\_ZORDER\_DOC (4544)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4544);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4544
```
