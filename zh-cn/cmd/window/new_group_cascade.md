# 新建层叠群组命令

### 摘要

> 从组合窗口中分离当前文档为一个新的群组并排列窗口使之重叠。

### 说明

> 从组合窗口中分离当前文档为一个新的群组并排列窗口使之重叠。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **群组** \> **新建层叠群组**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_NEW\_GROUP\_CASCADE (4393)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4393);

#### \[VBScript\]

> editor.ExecuteCommandByID 4393
