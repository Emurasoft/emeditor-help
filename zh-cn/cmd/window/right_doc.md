# 右边的文档命令

### 摘要

> 切换至标签页右边的文档。

### 说明

> 把文档切换到标签栏上当前打开的文档的右边。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **文档导航** \> **右边的文档**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL + F6

### 插件命令 ID

- EEID\_RIGHT\_DOC (4547)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4547);

#### \[VBScript\]

> editor.ExecuteCommandByID 4547
