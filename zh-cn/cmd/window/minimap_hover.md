# 悬停时显示迷你地图命令

## 摘要

当鼠标悬停在垂直滚动条上时显示迷你地图。

## 说明

当鼠标悬停在垂直滚动条上时显示迷你地图。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>滚动
\>悬停时显示迷你地图
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_MINIMAP\_HOVER (4044)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4044);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4044
```
