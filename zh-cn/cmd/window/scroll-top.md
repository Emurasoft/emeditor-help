# 滚动至顶部命令

## 摘要

滚动到文档的顶部。

## 说明

滚动到文档的顶部。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口 \>滚动 \>至顶部
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SCROLL\_TOP (3864)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3864);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3864
```
