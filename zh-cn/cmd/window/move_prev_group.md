# 移动到上一个群组命令

## 摘要

移动当前文档到上一个群组中。

## 说明

移动当前文档到上一个群组中。（重新合并为一个窗口）。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **群组** \> **移动到上一个群组**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_MOVE\_PREV\_GROUP (4396)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4396);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4396
```
