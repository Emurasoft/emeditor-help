# 关闭当前群组中所有右侧的文档命令

### 摘要

> 关闭当前群组中当前文档右侧的所有文档窗口。

### 说明

> 关闭当前群组中当前文档右侧的所有文档窗口。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **群组** \> **关闭当前群组中所有右侧的文档**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_GROUP\_CLOSE\_RIGHT (4390)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4390);

#### \[VBScript\]

> editor.ExecuteCommandByID 4390
