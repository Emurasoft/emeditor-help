# 关闭当前群组中的所有其他文档命令

## 摘要

关闭当前群组中除当前文档外的所有其他文档。

## 说明

关闭当前群组中除当前文档外的所有其他文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>群组 \>关闭当前群组中的所有其他文档
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_GROUP\_CLOSE\_OTHERS (4388)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4388);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4388
```
