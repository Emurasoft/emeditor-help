# 移除分割命令

## 摘要

移除分割。

## 说明

从当前文档中移除所有窗口分割。

## 运行方法

- 默认菜单:窗口 \>移除分割
- [所有命令](../tools/all_commands):窗口  \>
分割 \>移除分割
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: SHIFT+F12

## 插件命令 ID

- EEID\_DELETE\_PANES (4284)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4284);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4284
```
