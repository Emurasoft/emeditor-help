# 放大迷你地图命令

## 摘要

放大迷你地图。

## 说明

放大迷你地图。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **滚动**
\> **放大**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_MINIMAP\_ZOOM\_IN (3976)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3976);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3976
```
