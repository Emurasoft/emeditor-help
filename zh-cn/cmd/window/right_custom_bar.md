# 右侧自定义分栏命令

## 摘要

自定义分栏居于右侧。

## 说明

把自定义分栏切换到右侧如果可行的话。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>自定义分栏 \>右
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FOCUS\_RIGHT\_BAR (4427)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4427);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4427
```
