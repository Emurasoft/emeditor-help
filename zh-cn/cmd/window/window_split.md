# 分割命令

### 摘要

> 分割当前窗口到窗格中。

### 说明

> 这个命令会把当前窗口分割成四个窗格。用鼠标来控制分割位置。点击鼠标将固定分割位置（你也可以在以后根据需要改变方向）。

### 运行方法

- 默认菜单: **窗口** \> **分割**
- [所有命令](../tools/all_commands): **窗口**
\> **分割** \> **分割**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+F12

### 插件命令 ID

- EEID\_WINDOW\_SPLIT (4240)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4240);

#### \[VBScript\]

> editor.ExecuteCommandByID 4240
