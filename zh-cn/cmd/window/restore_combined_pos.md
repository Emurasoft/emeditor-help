# 还原标签页窗口位置命令

## 摘要

还原标签页窗口至上次保存的位置。

## 说明

这个命令会还原标签页窗口至上次保存的位置。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **还原** \> **还原标签页窗口位置**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_RESTORE\_COMBINED\_POS (4543)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4543);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4543
```
