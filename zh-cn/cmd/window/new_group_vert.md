# 新建垂直群组命令

## 摘要

从组合窗口中分离当前文档为一个新的群组并垂直平铺排列窗口。

## 说明

从组合窗口中分离当前文档为一个新的群组并垂直平铺排列窗口。

## 运行方法

- 默认菜单: **窗口** \> **新建垂直群组**
- [所有命令](../tools/all_commands): **窗口**
\> **群组**
\> **新建垂直群组**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_NEW\_GROUP\_VERT (4395)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4395);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4395
```
