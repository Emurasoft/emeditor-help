# 顶部自定义分栏命令

## 摘要

自定义分栏居于置顶。

## 说明

把自定义分栏切换到顶部如果可行的话。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>自定义分栏 \>上
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FOCUS\_TOP\_BAR (4426)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4426);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4426
```
