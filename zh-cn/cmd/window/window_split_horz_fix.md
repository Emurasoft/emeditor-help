# 水平分割并固定位置命令

### 摘要

> 水平分割当前窗口到固定位置。

### 说明

> 这个命令会把当前窗口分割成两个水平窗格，并立即在窗口中心固定分割位置。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **分割** \> **水平分割并固定位置**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_WINDOW\_SPLIT\_HORZ\_FIX (4335)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4335);

#### \[VBScript\]

> editor.ExecuteCommandByID 4335
