# 当分割时同步垂直滚动命令

## 摘要

同步分割窗口的垂直滚动。

## 说明

同步分割窗口的垂直滚动。

## 运行方法

- 默认菜单: **窗口** \> **当分割时同步垂直滚动**
- [所有命令](../tools/all_commands): **窗口** \> **分割** \> **当分割时同步垂直滚动**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SYNC\_VERT\_SCROLL\_SPLIT (4524)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4524);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4524
```
