# 下一个窗格命令

## 摘要

切换到下一个窗格。

## 说明

把光标移动到下一个窗格，当窗口被分割为多个窗格时。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **窗格**
\> **下一个窗格**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: F6

## 插件命令 ID

- EEID\_NEXT\_PANE (4214)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4214);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4214
```
