# 切换垂直滚动条上的标记命令

### 摘要

> 切换垂直滚动条上的标记。

### 说明

> 切换垂直滚动条上的标记。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口**
\> **滚动**
\> **切换垂直滚动条上的标记**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_SCROLL\_MARKS\_TOGGLE (3989)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3989);

#### \[VBScript\]

> editor.ExecuteCommandByID 3989
