# 切换迷你地图命令

## 摘要

切换迷你地图。

## 说明

切换迷你地图。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>滚动
\>切换迷你地图
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_MINIMAP\_TOGGLE (3975)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3975);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3975
```
