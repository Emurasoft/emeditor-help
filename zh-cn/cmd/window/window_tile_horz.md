# 水平平铺窗口命令

## 摘要

水平平铺排列窗口，使其不重叠。

## 说明

水平排列非组合的 EmEditor 窗口，使其不重叠平铺。

## 运行方法

- 默认菜单:窗口 \>水平平铺
- [所有命令](../tools/all_commands):窗口
\>水平平铺
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_WINDOW\_TILE\_HORZ (4242)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4242);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4242
```
