# 最少访问的文档命令

## 摘要

切换至最少访问的文档。

## 说明

切换至最少访问的文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>文档导航 \>最少访问的文档
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_PREV\_ZORDER\_DOC (4545)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4545);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4545
```
