# 启用标签页 \- 开命令

## 摘要

启用标签页并合并所有窗口。

## 说明

启用标签页并合并所有窗口。如果不同的群组中已用标签页显示不同的文档，那么这个命令会合并所有群组。当标签页被启用，所有当前打开的文档会被显示在窗口上方的的标签菜单上。只有一个 EmEditor 图标会被显示在 Windows 任务栏中。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>启用标签页 \>启用标签页 \- 开
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_WINDOW\_COMBINE\_ON (4407)

## 宏

## \[JavaScript\]

```
editor.EnableTab = true;
```

## \[VBScript\]

```
editor.EnableTab = True
```
