# 垂直平铺窗口命令

## 摘要

垂直平铺排列窗口，使其不重叠。

## 说明

垂直排列非组合的 EmEditor 窗口，使其不重叠平铺。

## 运行方法

- 默认菜单:窗口 \>垂直平铺
- [所有命令](../tools/all_commands):窗口
\>垂直平铺
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_WINDOW\_TILE\_VERT (4243)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4243);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4243
```
