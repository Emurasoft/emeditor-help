# 层叠窗口命令

## 摘要

排列窗口使之层叠。

## 说明

排列 EmEditor 窗口，让非组合窗口层叠在一个窗口中方便快速访问。

## 运行方法

- 默认菜单:窗口 \>层叠
- [所有命令](../tools/all_commands):窗口
\>层叠
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_WINDOW\_CASCADE (4241)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4241);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4241
```
