# 滚动至底部命令

## 摘要

滚动到文档的底部。

## 说明

滚动到文档的底部。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口** \> **滚动** \> **至底部**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SCROLL\_BOTTOM (3865)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3865);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3865
```
