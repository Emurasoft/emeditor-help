# 滚动至右边缘命令

## 摘要

滚动到文档的右边缘。

## 说明

滚动到文档的右边缘。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **窗口** \> **滚动** \> **右边缘**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SCROLL\_RIGHT\_EDGE (3871)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3871);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3871
```
