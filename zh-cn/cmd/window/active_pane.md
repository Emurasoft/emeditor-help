# 活动窗格命令

## 摘要

切换到活动窗格。

## 说明

切换到活动窗格当显示自定义分栏时。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):窗口
\>窗格
\>活动窗格
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_ACTIVE\_PANE (4409)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4409);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4409
```
