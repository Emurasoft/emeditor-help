# 全部最小化命令

## 摘要

最小化所有窗口。

## 说明

最小化所有打开的 EmEditor 窗口。

## 运行方法

- 默认菜单:窗口 \>全部最小化
- [所有命令](../tools/all_commands):窗口
\>全部最小化
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_WINDOW\_MINIMIZE\_ALL (4244)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4244);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4244
```
