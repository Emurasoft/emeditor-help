# 当分割时同步水平滚动命令

## 摘要

同步分割窗口的水平滚动。

## 说明

同步分割窗口的水平滚动。

## 运行方法

- 默认菜单:窗口 \>当分割时同步水平滚动
- [所有命令](../tools/all_commands):窗口 \>分割 \>当分割时同步水平滚动
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SYNC\_HORZ\_SCROLL\_SPLIT (4523)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4523);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4523
```
