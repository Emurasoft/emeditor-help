# 使列数相同命令

## 摘要

使 CSV 文档中的列数相同。

## 说明

使 CSV 文档中的列数相同。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **使列数相同**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **使列数相同**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_MAKE\_COLUMNS\_SAME (3993)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3993);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3993
```
