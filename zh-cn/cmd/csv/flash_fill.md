# 快速填充 (仅限单元格选择模式) 命令

## 摘要

自动填充 CSV 文档的整个列中的值。

## 说明

自动填充 CSV 文档的整个列中的值。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **快速填充**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **快速填充**
- 工具栏: ![](../../images/flash_fill.PNG)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FLASH\_FILL (3991)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3991);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3991
```
