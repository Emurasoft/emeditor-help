# 列标题只读命令

## 摘要

切换选项使列标题只读。

## 说明

切换选项使列标题只读。

## 运行方法

- 默认菜单: **CSV** \> **标题（冻结窗格）** \> **列标题只读**
- [所有命令](../tools/all_commands): **CSV** \> **标题（冻结窗格）** \> **列标题只读**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_HEADING\_POPUP (3900)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3900);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3900
```
