# 下一单元格 (仅限单元格选择模式) 命令

## 摘要

在 CSV 文档中跳转到下一个单元格。

## 说明

在 CSV 文档中跳转到下一个单元格。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **仅限单元格选择模式** \> **下一单元格**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_CSV\_NEXT\_CELL (4077)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4077);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4077
```
