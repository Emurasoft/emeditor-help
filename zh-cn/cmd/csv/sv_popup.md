# CSV (弹出菜单) 命令

## 摘要

显示一个能选择 CSV 模式的弹出菜单。

## 说明

显示一个能选择 CSV 模式的弹出菜单。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **CSV** \> **CSV (弹出菜单)**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_SV\_POPUP (3899)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3899);
```

### \[VBScript\]

```
>editor.ExecuteCommandByID 3899
```
