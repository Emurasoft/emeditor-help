# 列标题命令

## 摘要

显示或隐藏列标题。

## 说明

显示或隐藏列标题。在默认设定下，当你打开一个 CSV 文件时，EmEditor 会显示列标题。

## 运行方法

- 默认菜单: **CSV** \> **列标题**
- [所有命令](../tools/all_commands): **CSV** \> **列标题**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HEADER_TOGGLE (3985)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3985);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3985
```
