# 隐藏列命令

## 摘要

隐藏选取的列或当前列。

## 说明

隐藏选取的列或当前列。

## 运行方法

- 默认菜单:CSV \>列 \>隐藏列
- [所有命令](../tools/all_commands):CSV \>列 \>隐藏列
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HIDE_COLUMN (4087)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4087);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4087
```
