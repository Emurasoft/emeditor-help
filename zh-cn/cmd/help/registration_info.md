# 注册信息命令

## 摘要

显示注册信息。

## 说明

显示注册信息。

## 运行方法

- 默认菜单: **帮助** \> **注册信息**
- [所有命令](../tools/all_commands): **帮助** > **注册信息**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HELP_REGISTRATION_INFO (23249)
```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23249);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23249
```
