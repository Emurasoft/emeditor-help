# 如何购买命令

## 摘要

显示如何购买 EmEditor 的信息。

## 说明

在“EmEditor 帮助”中打开有关“如何购买”的章节。

## 运行方法

- 默认菜单: **帮助** \> **如何购买**
- [所有命令](../tools/all_commands): **帮助** >
**如何购买**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HELP_REGIST (4248)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4248);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4248
```
