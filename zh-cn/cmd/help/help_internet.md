# Web 帮助命令

## 摘要

显示 Web 帮助。

## 说明

显示 Web 帮助。

## 运行方法

- 默认菜单: **帮助** \> **Web 帮助**
- [所有命令](../tools/all_commands): **帮助** >
**Web 帮助**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

EEID_HELP_REGIST (3932)```

editor.ExecuteCommandByID(3932);
