# 发送反馈命令

## 摘要

将反馈发送给 EmEditor 开发团队。

## 说明

将反馈发送给 EmEditor 开发团队。

## 运行方法

- 默认菜单: **帮助** \> **发送反馈**
- [所有命令](../tools/all_commands): **帮助** >
**发送反馈**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SEND_FEEDBACK (4058)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4058);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4058
```
