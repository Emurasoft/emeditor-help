# 关于 EmEditor 命令

## 摘要

显示版本信息。

## 说明

显示 [关于 EmEditor 对话框](../../dlg/about/index)，让你能确认 EmEditor 的版本信息。

## 运行方法

- 默认菜单:帮助 \>关于 EmEditor
- [所有命令](../tools/all_commands):帮助 >
关于 EmEditor
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_APP_ABOUT (4250)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4250);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4250
```
