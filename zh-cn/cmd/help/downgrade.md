# 降级命令

## 摘要

将产品降级为 EmEditor Free。

## 说明

将产品从 EmEditor Professional 降级为 EmEditor Free。在降级之前，EmEditor 会显示一条确认提示。

## 运行方法

- 默认菜单: **帮助** >
**降级**
- [所有命令](../tools/all_commands): **帮助** >
**降级**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HELP_DOWNGRADE (3844)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3844);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3844
```
