# 搜索帮助主题命令

## 摘要

显示 EmEditor 帮助主题

## 说明

打开 EmEditor 帮助模块，并让你能浏览或搜索帮助主题。

## 运行方法

- 默认菜单: **帮助** \> **搜索帮助主题**
- [所有命令](../tools/all_commands): **帮助** >
**搜索帮助主题**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+F1

## 插件命令ID

```
EEID_HELP_FINDER (4247)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4247);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4247
```
