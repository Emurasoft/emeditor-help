# 本地帮助命令

## 摘要

显示本地帮助。

## 说明

显示本地帮助。

## 运行方法

- 默认菜单: **帮助** \> **本地帮助**
- [所有命令](../tools/all_commands): **帮助** >
**本地帮助**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HELP_LOCAL (3933)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3933);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3933
```
