# 打开 EmEditor 主页命令

## 摘要

打开 EmEditor 网站主页。

## 说明

在你默认的网络浏览器中打开 EmEditor 网站。

## 运行方法

- 默认菜单:帮助 \>打开 EmEditor 主页
- [所有命令](../tools/all_commands):帮助 >
打开 EmEditor 主页
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_WEB_HOME (4249)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4249);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4249
```
