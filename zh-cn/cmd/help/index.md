# 帮助类别

|     |     |
| --- | --- |
| **[搜索帮助主题](help_finder)** | 显示 EmEditor 帮助主题 |
| **[搜索命令](../tools/ql_commands)** | 打开快速启动窗口并选择命令。 |
| **[搜索选项](../tools/ql_options)** | 打开快速启动窗口并选择选项。 |
| **[键盘映射图](keyboard_map)** | 显显示可用命令的键盘快捷键列表。 |

| **[检查更新](check_updates)** | 连接到 Emurasoft 服务器检查可用的新版本的EmEditor。 |
| **[自定义更新检查程式](customize_update_checker)** | 自定义更新程式。 |
| **[选择更新频道](update_channel)** | 选择一个更新频道。 |
| **[发送反馈](send_feedback)** | 将反馈发送给 EmEditor 开发团队。 |
| **[撰写评价](write_review)** | 撰写有关 EmEditor 的评价。 |
| **[升级](upgrade)** | 将本产品升级为 EmEditor Professional。 |
| **[降级](downgrade)** | 将产品降级为 EmEditor Free。 |
| **[关于 EmEditor](app_about)** | 显示版本信息。 |
