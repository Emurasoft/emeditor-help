# 自定义更新检查程式命令

## 摘要

自定义更新程式。

## 说明

打开 **更新选项** 对话框，你可以指定 EmEditor 检查更新的次数以及下载和保存路径。

## 运行方法

- 默认菜单: **帮助** \> **自定义更新检查程式**
- [所有命令](../tools/all_commands): **帮助** >
**自定义更新检查程式**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_CUSTOMIZE_UPDATE_CHECKER (4482)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4482);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4482
```
