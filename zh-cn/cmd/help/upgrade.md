# 升级命令

## 摘要

将本产品升级为 EmEditor Professional。

## 说明

将本产品升级为 EmEditor Professional。这个命令在 EmEditor Professional 中不可用。

## 运行方法

- 默认菜单: **帮助** >
**升级**
- [所有命令](../tools/all_commands): **帮助** >
**升级**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_HELP_UPGRADE (3845)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3845);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3845
```
