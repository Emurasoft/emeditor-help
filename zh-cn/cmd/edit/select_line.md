# 选择行命令

### 摘要

> 切换行选择模式。

### 说明

> 切换行选择模式。这个命令让你能用键盘高亮多行。用箭头键移动光标来扩展或缩短选区。选择 [**复制** 命令](edit_copy) 或 [**剪切** 命令](edit_cut) 将终止该选择模式。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **扩展选区**
\> **选择行**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+F8

### 插件命令ID

- EEID\_SELECT\_LINE (4154)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4154);

#### \[VBScript\]

> editor.ExecuteCommandByID 4154
