# 向右扩展一个字符 (垂直选择模式)命令

## 摘要

将垂直选择向右扩展一个字符。

## 说明

将垂直选择向右扩展一个字符。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>扩展选区
\>向右扩展一个字符 (垂直选择模式)
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT+CTRL+向右键

## 插件命令ID

```
EEID_BOX_EXTEND_RIGHT (4025)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4025);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4025
```
