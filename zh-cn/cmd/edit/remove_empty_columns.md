# 删除空列命令

### 摘要

> 删除 CSV 文档中的空列。

### 说明

> 删除 CSV 文档中的空列。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): 无
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_REMOVE\_EMPTY\_COLUMNS (4062)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4062);

#### \[VBScript\]

> editor.ExecuteCommandByID 4062
