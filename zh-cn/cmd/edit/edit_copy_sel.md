# 复制选区命令

### 摘要

> 复制选定内容到剪贴板。

### 说明

> 复制选定内容并把它粘贴到剪贴板。在这个指令之后，你可以通过把光标移动到另一个地方并运行 [**粘贴** 命令](edit_paste) 来放置选取内容。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **复制** \> **复制选区**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_EDIT\_COPY\_SEL (4445)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4445);

#### \[VBScript\]

> editor.ExecuteCommandByID 4445
