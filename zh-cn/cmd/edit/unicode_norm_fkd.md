# Unicode 标准化表单 KD（兼容性分解）命令

## 摘要

将 Unicode 标准化表单 KD（兼容性分解）应用于所选字符串。

## 说明

将在“ [Unicode 标准附录 #15 Unicode Normalization Forms](http://unicode.org/reports/tr15/)”中定义的 Unicode 标准化表单 KD（兼容性分解）应用于所选字符串。

## 运行方法

- 默认菜单:转换 \>编码/解码所选内容 \>Unicode 标准化表单 KD（兼容性分解）
- [所有命令](../tools/all_commands):转换 \>编码/解码所选内容 \>Unicode 标准化表单 KD（兼容性分解）
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_UNICODE\_NORM\_FKD (4056)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4056);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4056
```
