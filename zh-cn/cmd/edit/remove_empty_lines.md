# 删除空行命令

### 摘要

> 删除选区内或整个文档中的空行。

### 说明

> 删除选区内或整个文档中的空行。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): 无
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_REMOVE\_EMPTY\_LINES (4029)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4029);

#### \[VBScript\]

> editor.ExecuteCommandByID 4029
