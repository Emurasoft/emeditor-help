# 选择当前文档中书签行命令

## 摘要

选择当前文档中所有已添加书签的行。

## 说明

选择当前文档中所有已添加书签的行。

## 运行方法

- 默认菜单: **书签** \> **选择书签行**
- [所有命令](../tools/all_commands): **书签** \> **选择书签行**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_BOOKMARK_SELECT (3851)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3851);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3851
```
