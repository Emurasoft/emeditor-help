# 自动复制命令

### 摘要

> 切换自动复制模式。

### 说明

> 切换自动复制模式。

### 运行方法

- 默认菜单: **编辑** \> **CSV** \> **自动复制**
- [所有命令](../tools/all_commands): **编辑** \> **复制** \> **自动复制**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_AUTO\_COPY (3979)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3979);

#### \[VBScript\]

> editor.ExecuteCommandByID 3979
