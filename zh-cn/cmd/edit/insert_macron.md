# 插入长音符命令

## 摘要

插入一个带有长音符的字符。

## 说明

在选择这个命令后，在光标处插入一个带有长音符的字符(ā，ē，ī，ō，ū，Ā，Ē，Ī，Ō 或 Ū)通过输入 a，e，i，o，u，A，E，I，O 或 U。

## 运行方法

- 默认菜单: **插入** \> **音调符号** \> **长音符**
- [所有命令](../tools/all_commands): **插入** \> **音调符号** \> **长音符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT + -

## 插件命令ID

```
EEID_INSERT_MACRON (3861)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3861);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3861
```
