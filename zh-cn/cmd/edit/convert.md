# 半角/全角转换命令

## 摘要

转换选定文本中的半角和全角字符。

## 说明

这个命令会把半角字符转换为全角字符，或者把全角字符转换为半角字符。当执行该命令时， [半角/全角转换 对话框](../../dlg/convert/index) 会显示，你可以设置字符类型以及范围。

## 运行方法

- 默认菜单:转换 \>半角/全角转换
- [所有命令](../tools/all_commands):转换 \>半角/全角转换
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+T

## 插件命令ID

```
EEID_CONVERT (4148)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4148);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4148
```
