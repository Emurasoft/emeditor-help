# 启用仅编辑选定区域模式命令

## 摘要

把选定的区域设置为可编辑的区域，其他剩余的区域则不允许访问。

## 说明

把选定的区域（包括选定区域的整个逻辑行）设置为可编辑的区域，其他剩余的区域则不允许访问。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>仅编辑选定区域 \>启用仅编辑选定区域模式
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_NARROWING_ON (4457)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4457);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4457
```
