# 剪切列命令

### 摘要

> 剪切选取的列或当前列并把剪切的内容移到剪贴板中。

### 说明

> 剪切选取的列或当前列并把剪切的内容移到剪贴板中。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **剪切** \> **剪切列**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_CUT\_COLUMN (3963)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3963);

#### \[VBScript\]

> editor.ExecuteCommandByID 3963
