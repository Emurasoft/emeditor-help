# 复制列命令

## 摘要

复制选取的列或当前列并把内容粘贴到剪贴板中。

## 说明

复制选取的列或当前列并把内容粘贴到剪贴板中。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **复制** \> **复制列**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_COPY\_COLUMN (3964)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3964);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3964
```
