# 下一个节点命令

## 摘要

当显示大纲视图时移动光标到下一个节点。

## 说明

当显示大纲视图时移动光标到下一个节点。要使用该功能，先要启用大纲来显示大纲视图。

## 运行方法

- 默认菜单:编辑 \>大纲 \>下一个节点
- [所有命令](../tools/all_commands):编辑 \>大纲 \>下一个节点
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: Alt+Down

## 插件命令ID

```
EEID_OUTLINE_NEXT_NODE (4415)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4415);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4415
```
