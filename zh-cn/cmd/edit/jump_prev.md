# 跳转到上一个命令

## 摘要

跳转到输出栏中的上一个位置。

## 说明

跳转到输出栏中的上一个位置。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>高级
\>跳转到上一个
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: SHIFT+F4

## 插件命令ID

```
EEID_JUMP_PREV (4443)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4443);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4443
```
