# 自动填充/自动重复命令

### 摘要

> 将一序列或模式延续到相邻的单元格中。

### 说明

> 将一序列或模式延续到相邻的单元格中。

### 运行方法

- 默认菜单: **插入** \> **自动填充/自动重复**
- [所有命令](../tools/all_commands): **插入** \> **自动填充/自动重复**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_AUTO\_FILL (3992)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3992);

#### \[VBScript\]

> editor.ExecuteCommandByID 3992
