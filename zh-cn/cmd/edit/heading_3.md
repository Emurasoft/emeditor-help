# 列标题 3 命令

## 摘要

把前三行作为标题并且固定不滚动。

## 说明

把前三行作为标题并且固定不滚动。

## 运行方法

- 默认菜单:CSV \>标题（冻结窗格） \>列标题 3
- [所有命令](../tools/all_commands):CSV \>标题（冻结窗格） \>列标题 3
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_HEADING\_3 (3896)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3896);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3896
```
