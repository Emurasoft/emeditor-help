# 在列中筛选命令

### 摘要

> 用所选文本或光标处的单词筛选当前列中的文档。

### 说明

> 用所选文本或光标处的单词筛选当前列中的文档。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **高级** \> **在列中筛选**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_FILTER\_COLUMN (3983)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3983);

#### \[VBScript\]

> editor.ExecuteCommandByID 3983
