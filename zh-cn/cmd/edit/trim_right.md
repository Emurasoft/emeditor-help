# 删除行末空格命令

## 摘要

删除当前文档中所有行末的多余空格。

## 说明

删除当前文档中在一个文本字符和一个回车符之间所有的多余空格。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **删除**
\> **>删除行末空格删除行末空格**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_TRIM_RIGHT (4278)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4278);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4278
```
