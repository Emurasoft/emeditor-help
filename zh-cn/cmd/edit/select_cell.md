# 选择单元格命令

## 摘要

在 CSV 模式下选择当前光标所在位置的单元格。

## 说明

在 CSV 模式下选择当前光标所在位置的单元格。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):CSV \>选择单元格
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SELECT_CELL (4489)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4489);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4489
```
