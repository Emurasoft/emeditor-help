# 后退命令

## 摘要

移动光标至上一位置。

## 说明

移动光标至上一位置。

## 运行方法

- 默认菜单:编辑 \>高级 \>后退
- [所有命令](../tools/all_commands):编辑 \>高级 \>后退
- 工具栏:
![](../../images/emeditor12_back_button.png)
- 状态栏: 无
- 默认快捷键: ALT + LEFT

## 插件命令ID

```
EEID_CARET_BACK (4597)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4597);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4597
```
