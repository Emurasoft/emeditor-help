# 添加至标记命令

## 摘要

将已选中的文本添加至标记列表。

## 说明

将已选中的文本添加至标记列表。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **高级**
\> **添加至标记**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ADD_TO_MARKERS (4560)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4560);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4560
```
