# 筛选除外命令

## 摘要

筛选除选取文本或光标处文字外的文档。

## 说明

筛选除选取文本或光标处文字外的文档。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>高级 \>筛选除外
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_FILTER\_OUT\_WORD (3974)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3974);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3974
```
