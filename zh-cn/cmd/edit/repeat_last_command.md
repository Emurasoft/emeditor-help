# 重复上一命令命令

## 摘要

重复上一个执行的命令。

## 说明

重复上一个选择执行的命令。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **重做** \> **重复上一命令**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+Shift+G

## 插件命令ID

```
EEID_REPEAT_LAST_COMMAND (3846)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3846);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3846
```
