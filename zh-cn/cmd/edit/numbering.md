# 编号命令

## 摘要

在当前光标或垂直选区中插入编号。

## 说明

在光标位置或垂直选取处插入编号。当执行该命令时， [**“插入编号”** 对话框](../../dlg/insert_numbering/index) 会被显示，你可以指定要插入的数字。

## 运行方法

- 默认菜单: **插入** \> **编号**
- [所有命令](../tools/all_commands): **插入** \> **编号**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT+N

## 插件命令ID

```
EEID_NUMBERING (4558)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4558);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4558
```
