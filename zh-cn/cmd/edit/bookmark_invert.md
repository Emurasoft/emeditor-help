# 反转当前文档中的书签命令

## 摘要

反转当前文档中的所有书签。

## 说明

反转当前文档中的所有书签，为没有书签标示的行添加书签，并清除已添加书签行的书签。

## 运行方法

- 默认菜单:书签 \>反转书签
- [所有命令](../tools/all_commands):书签 \>反转书签
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_BOOKMARK_INVERT (4588)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4588);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4588
```
