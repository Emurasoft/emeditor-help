# 窗口顶部命令

### 摘要

> 移动光标到窗口顶部。

### 说明

> 移动光标到窗口顶部。这个命令不会移动垂直滚动条的位置。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **垂直移动光标**
\> **窗口顶部**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT+PAGE UP

### 插件命令ID

- EEID\_WINDOW\_TOP (4292)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4292);

#### \[VBScript\]

> editor.ExecuteCommandByID 4292
