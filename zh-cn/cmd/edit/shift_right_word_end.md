# 向右扩展到单词末尾命令

## 摘要

向右扩展选区到单词的末尾。

## 说明

向右扩展选区到单词的末尾。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>扩展选区
\>向右扩展到单词末尾
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+向右键

## 插件命令ID

```
EEID_SHIFT_RIGHT_WORD_END (4031)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4031);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4031
```
