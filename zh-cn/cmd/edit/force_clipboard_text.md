# 强制转换剪贴板格式为文本格式命令

### 摘要

> 把剪贴板格式强制转换为仅文本格式 (仅在托盘图标菜单中显示)。

### 说明

> 监视剪贴板内容并转换为仅文本格式，如果内容格式不是文本或文本框的话。这个命令可用于当你要在不同的文本处理器之间复制并粘贴时。该命令与 EmEditor 中的复制或粘贴命令没有关联，它只存在在托盘图标中，不存在在 EmEditor 的主程序中。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): 无
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无
