# 插入扬抑符命令

## 摘要

插入一个带有扬抑符的字符

## 说明

在选择这个命令后，在光标处插入一个带有扬抑符的字符(â，ê，î，ô，û，Â，Ê，Î，
Ô 或 Û)通过输入 a，e，i，o，u，A，E，I，O 或 U。

## 运行方法

- 默认菜单:插入 \>音调符号 \>扬抑符
- [所有命令](../tools/all_commands):插入 \>音调符号 \>扬抑符
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+6

## 插件命令ID

```
EEID_INSERT_CIRCUMFLEX (4305)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID (4305);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4305
```
