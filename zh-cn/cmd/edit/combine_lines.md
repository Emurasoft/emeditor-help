# 合并行命令

## 摘要

合并 CSV 文档中垂直相邻的重复单元格。

## 说明

此命令会显示 [合并行 对话框](../../dlg/combine_lines/index) 来合并 CSV 文档中垂直相邻的重复单元格。

## 运行方法

- 默认菜单:CSV \>高级 \>合并行
- [所有命令](../tools/all_commands):CSV \>高级 \>合并行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_COMBINE\_LINES (4060)

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4060);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4060
```
