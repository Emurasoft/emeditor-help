# 扩展选区到文档顶部命令

## 摘要

把选区扩展到文档顶部。

## 说明

把选区扩展到文档顶部。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>扩展选区
\>扩展选区到文档顶部
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+HOME

## 插件命令ID

```
EEID_SHIFT_TOP (4184)```

## 宏

### \[JavaScript\]

```
document.selection.StartOfDocument(true);
```

### \[VBScript\]

```
document.selection.StartOfDocument true
```
