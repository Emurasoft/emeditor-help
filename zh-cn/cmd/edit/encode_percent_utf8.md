# 将 Unicode 转换为百分比编码 (UTF-8) 命令

## 摘要

用 UTF-8 编码对选中的文本进行百分比编码。

## 说明

用 UTF-8 编码对选中的文本进行百分比编码。

## 运行方法

- 默认菜单:转换 \>编码/解码所选内容 \>将 Unicode 转换为百分比编码 (UTF-8)
- [所有命令](../tools/all_commands):转换 \>编码/解码所选内容 \>将 Unicode 转换为百分比编码 (UTF-8)
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_ENCODE_PERCENT_UTF8 (3857)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3857);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3857
```
