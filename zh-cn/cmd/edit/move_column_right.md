# 右移列命令

## 摘要

将选定的列或 CSV 文档中的当前列往右移一列。

## 说明

将选定的列或 CSV 文档中的当前列往右移一列。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>移动
\>右移列
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MOVE_COLUMN_RIGHT (3994)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3994);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3994
```
