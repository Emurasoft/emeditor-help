# 还原已删除的字符串命令

## 摘要

在光标位置处还原先前删除的字符串。

## 说明

在光标位置处还原最近删除的字符串。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **粘贴** \> **还原已删除的字符串**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_RESTORE_DELETED (4419)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID (4419);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4419
```
