# 提取列命令

### 摘要

> 从当前 CSV 文档中提取指定列。

### 说明

> 从当前 CSV 文档中提取指定列。当执行这个命令时，会显示 [**提取列** 对话框](../../dlg/extract_columns/index)，让你可以设置提取的列的参数。

### 运行方法

- 默认菜单: **CSV** \> **高级** \> **提取列**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **提取列**
- 工具栏: ![](../../images/extract_columns.png)
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_EXTRACT\_COLUMNS (3930)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(3930);

#### \[VBScript\]

> editor.ExecuteCommandByID 3930
