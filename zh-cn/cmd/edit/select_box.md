# 垂直框选命令

## 摘要

切换垂直框选模式。

## 说明

切换垂直框选模式。这个命令让你能用键盘高亮多列的字符。用箭头键移动光标来扩展或缩短选区。选择 [复制 命令](edit_copy) 或 [剪切 命令](edit_cut) 将终止垂直框选模式。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>扩展选区 \>垂直框选
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+Shift+F8

## 插件命令ID

```
EEID_SELECT_BOX (4155)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4155);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4155
```
