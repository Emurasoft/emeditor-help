# 添加/删除标记命令

### 摘要

> 将已选中的文本添加至标记列表或从标记列表中删除已选中的文本。

### 说明

> 将已选中的文本添加至标记列表或从标记列表中删除已选中的文本如果文本已被添加至列表。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **高级**
\> **添加/删除标记**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_ADD\_REMOVE\_MARKERS (4591)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4591);

#### \[VBScript\]

> editor.ExecuteCommandByID 4591
