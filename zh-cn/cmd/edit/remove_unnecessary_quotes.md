# 删除不需要的双引号命令

## 摘要

删除 CSV 文档中不需要的双引号。

## 说明

删除 CSV 文档中不需要的双引号。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **删除不需要的双引号**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **删除不需要的双引号**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_REMOVE\_UNNECESSARY\_QUOTES (3902)

## 宏

### \[JavaScript\]

```
>editor.ExecuteCommandByID(3902);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3902
```
