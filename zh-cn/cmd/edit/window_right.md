# 移至窗口右端命令

## 摘要

移动到当前窗口右端。

## 说明

把光标移动到当前窗口右边缘。这个命令不会移动水平滚动条位置，只是把光标移动到所显示的视窗的右端。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>水平移动光标
\>移至窗口右端
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_WINDOW_RIGHT (4294)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4294);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4294
```
