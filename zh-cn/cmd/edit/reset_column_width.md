# 重置列宽命令

### 摘要

> 重置 CSV 文档中所选列的宽度。

### 说明

> 重置 CSV 文档中所选列的宽度。

### 运行方法

- 默认菜单: **CSV** \> **列** \> **重置列宽**
- [所有命令](../tools/all_commands): **CSV** \> **列** \> **重置列宽**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_RESET\_COLUMN\_WIDTH (4052)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4052);

#### \[VBScript\]

> editor.ExecuteCommandByID 4052
