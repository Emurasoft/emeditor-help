# 对整个文档进行拼写检查命令

## 摘要

对整个文档进行拼写检查。

## 说明

对整个文档进行拼写检查。错误拼写的字词会以红色下划线显示。

## 运行方法

- 默认菜单: **编辑** \> **拼写检查** \> **检查拼写于** \> **全部**
- [所有命令](../tools/all_commands): **编辑** \> **拼写检查** \> **检查拼写于** \> **全部**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SPELL_ALL (4522)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4522);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4522
```
