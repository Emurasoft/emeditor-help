# 插入下加符命令

### 摘要

> 在输入字符 c 或 C 时在该字符上插入下加符。

### 说明

> 在选择这个命令后，在光标处插入一个下加符 (ç 或 Ç)通过输入 c 或 C。

### 运行方法

- 默认菜单: **插入** \> **音调符号** \> **下加符**
- [所有命令](../tools/all_commands): **插入** \> **音调符号** \> **下加符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL + ,

### 插件命令ID

- EEID\_INSERT\_CEDILLA (4310)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4310);

#### \[VBScript\]

> editor.ExecuteCommandByID 4310
