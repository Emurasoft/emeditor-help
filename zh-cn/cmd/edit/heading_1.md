# 列标题 1 命令

## 摘要

把第一行作为标题并且固定不滚动。

## 说明

把第一行作为标题并且固定不滚动。

## 运行方法

- 默认菜单:CSV \>标题（冻结窗格） \>列标题 1
- [所有命令](../tools/all_commands):CSV \>标题（冻结窗格） \>列标题 1
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_HEADING\_1 (3894)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3894);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3894
```
