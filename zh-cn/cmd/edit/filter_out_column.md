# 在列中筛选除外命令

### 摘要

> 筛选当前列中的文档而不使用选取的文本或光标处的单词。

### 说明

> 筛选当前列中的文档而不使用选取的文本或光标处的单词。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **高级** \> **在列中筛选除外**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_FILTER\_OUT\_COLUMN (4012)

### 宏

#### \[JavaScript\]

> editor.ExecuteCommandByID(4012);

#### \[VBScript\]

> editor.ExecuteCommandByID 4012
