# 随机排序命令

## 摘要

随机排序当前列。

## 说明

随机排序当前列。

## 运行方法

- 默认菜单: **排序** \> **随机排序**
- [所有命令](../tools/all_commands): **排序** \> **随机排序**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_RANDOM (4003)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4003);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4003
```
