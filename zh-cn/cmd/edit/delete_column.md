# 删除列命令

## 摘要

删除选取的列或当前列。

## 说明

删除选取的列或当前列。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **删除** \> **删除列**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_DELETE\_COLUMN (3962)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3962);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3962
```
