# 交换开始位置和结束位置命令

## 摘要

交换选区中的开始位置和结束位置。

## 说明

把光标移动到选区的开头或末尾（例如，如果光标目前在选区的开头，这个命令会把它移到选区的末尾，反之亦然）。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>选择模式
\>交换选区中的开始位置和结束位置。
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: Shift + F8

## 插件命令ID

```
EEID_SWITCH_START_END_SELECT (3850)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3850);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3850
```
