# 非制表位化命令

## 摘要

将制表符（tab）转换为空格。

## 说明

在每一行的开头把选取的制表符（tab）转换为空格。一个制表符所含的空格数可以在 [制表符/缩进 对话框](../../dlg/properties/general/indent/index) 中设定。

## 运行方法

- 默认菜单:转换 \>将制表符转换为空格
- [所有命令](../tools/all_commands):转换 \>将制表符转换为空格
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_UNTABIFY (4357)```

## 宏

### \[JavaScript\]

```
document.selection.Untabify();
```

### \[VBScript\]

```
document.selection.Untabify
```
