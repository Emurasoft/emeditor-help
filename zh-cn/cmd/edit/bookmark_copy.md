# 复制当前文档中书签行命令

## 摘要

复制当前文档中所有已添加书签的行并把它们移动到剪贴板。

## 说明

复制当前文档中所有已添加书签的行并把它们粘贴到剪贴板。

## 运行方法

- 默认菜单:书签 \>复制书签行
- [所有命令](../tools/all_commands):书签 \>复制书签行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_BOOKMARK_COPY (3853)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3853);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3853
```
