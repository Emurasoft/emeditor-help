# 反向选择命令

## 摘要

反向选择。

## 说明

反向选择。

## 运行方法

- 默认菜单:编辑 \>高级 \>反向选择
- [所有命令](../tools/all_commands):编辑 \>扩展选区
\>反向选择
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_INVERT_SELECTION (4065)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4065);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4065
