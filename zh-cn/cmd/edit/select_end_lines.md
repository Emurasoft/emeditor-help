# 选择行尾命令

## 摘要

选择选区中的行尾。

## 说明

选择选区中的行尾来进行垂直选取编辑。这些行能够被同时编辑。

## 运行方法

- 默认菜单:编辑 \>选择模式
\>选择行尾
- [所有命令](../tools/all_commands):编辑 \>选择模式
\>选择行尾
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SELECT_END_LINES (3849)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3849);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3849
```
