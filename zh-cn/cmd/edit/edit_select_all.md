# 全选命令

### 摘要

> 选择全部。

### 说明

> 选择当前文档中的所有字符。

### 运行方法

- 默认菜单: **编辑** \> **全选**
- [所有命令](../tools/all_commands): **编辑** \> **扩展选区**
\> **全选**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+A

### 插件命令ID

- EEID\_EDIT\_SELECT\_ALL (4136)

### 宏

#### \[JavaScript\]

> document.selection.SelectAll();

#### \[VBScript\]

> document.selection.SelectAll
