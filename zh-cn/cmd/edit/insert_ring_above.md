# 插入上圆圈符命令

## 摘要

插入一个带有上圆圈符的字符。

## 说明

在选择这个命令后，在光标处插入一个带有上圆圈符的字符(å 或 Å)通过输入 a 或 A，或度数符号(°)通过输入一个空格。

## 运行方法

- 默认菜单:插入 \>音调符号 \>上圆圈符
- [所有命令](../tools/all_commands):插入 \>音调符号 \>上圆圈符
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+2

## 插件命令ID

```
EEID_INSERT_RING_ABOVE (4308)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4308);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4308
```
