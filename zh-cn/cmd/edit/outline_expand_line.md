# 展开当前行命令

## 摘要

当显示大纲视图时展开当前行。

## 说明

当显示大纲视图时，这个命令会展开当前行。这与点击文本行左边的「+」按钮效果一样。要使用该功能，先要启用大纲来显示大纲视图。

## 运行方法

- 默认菜单:编辑 \>大纲 \>展开当前行
- [所有命令](../tools/all_commands):编辑 \>大纲 \>展开当前行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_OUTLINE_EXPAND_LINE (4414)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4414);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4414
```
