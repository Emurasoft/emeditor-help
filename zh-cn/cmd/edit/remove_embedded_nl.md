# 在 CSV 中删除嵌入式换行命令

## 摘要

在 CSV 文档中删除嵌入式换行。

## 说明

在 CSV 文档中删除嵌入式换行。

## 运行方法

- 默认菜单: **CSV** \> **高级** \> **删除嵌入式换行**
- [所有命令](../tools/all_commands): **CSV** \> **高级** \> **删除嵌入式换行**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_REMOVE\_EMBEDDED\_NL (3922)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3922);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3922
```
