# 按文本长度从短到长排序命令

## 摘要

在当前列中按文本长度从短到长排序文本。

## 说明

在当前列中按文本长度从短到长排序文本。在自定义 对话框中排序 页面上的当按长度排序时，将全角字符作为 2 个字符排序 复选框定义了全角字符的计算方式。

## 运行方法

- 默认菜单:排序 \>按文本长度从短到长排序
- [所有命令](../tools/all_commands):排序 \>按文本长度从短到长排序
- 工具栏：![](../../images/sort_length_a.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_SORT_LENGTH_A (3917)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(3917);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3917
```
