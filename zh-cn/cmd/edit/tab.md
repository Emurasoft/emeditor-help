# 制表符或增加行缩进命令

## 摘要

插入制表符或增加当前行缩进。

## 说明

在光标处插入一个制表符（tab字符）。如果多行被选取了，这个命令将在所有选取行的行首插入一个制表符。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>水平移动光标
\>插入制表符或增加行缩
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: TAB

## 插件命令ID

```
EEID_TAB (4188)```

## 宏

## \[JavaScript\]

```
editor.ExecuteCommandByID(4188);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 4188
```
