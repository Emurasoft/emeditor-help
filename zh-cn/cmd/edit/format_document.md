# 格式化文档命令

## 摘要

用语言服务器协议格式化整个文档。

## 说明

用语言服务器协议格式化整个文档。

## 运行方法

- 默认菜单: **转换** \> **格式化文档**
- [所有命令](../tools/all_commands): **转换** \> **格式化文档**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: Ctrl+K，D

## 插件命令ID

```
EEID_FORMAT_DOCUMENT (23244)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23244);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23244
```
