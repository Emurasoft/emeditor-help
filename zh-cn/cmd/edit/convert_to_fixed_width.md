# 固定列宽命令

### 摘要

> 用当前分隔符位置把当前含有分隔值的文档转换为固定列宽。

### 说明

> 用当前分隔符位置把当前含有分隔值的文档转换为固定列宽。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **CSV** \> **转换为** \> **固定列宽**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令 ID

- EEID\_CONVERT\_TO\_FIXED\_WIDTH (3883)

### 宏

#### \[JavaScript\]

> >editor.ExecuteCommandByID(3883);

#### \[VBScript\]

> editor.ExecuteCommandByID 3883
