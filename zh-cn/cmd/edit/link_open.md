# 打开链接命令

## 摘要

打开一个超链接。

## 说明

在光标处打开一个超链接。

## 运行方法

- 默认菜单:编辑 \>打开链接
- [所有命令](../tools/all_commands):编辑 \>高级
\>打开链接
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_LINK_OPEN (4141)```

## 宏

## \[JavaScript\]

```
document.selection.OpenLink();
```

## \[VBScript\]

```
document.selection.OpenLink
```
