# 取消多选区最下方选取命令

## 摘要

取消多选区最下方的选取。

## 说明

取消多选区最下方的选取。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **扩展选区**
\> **取消多选区最下方选取**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: ALT+CTRL+SHIFT+向下键

## 插件命令ID

```
EEID_CANCEL_BOTTOM_SEL (4031)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4031);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4031
```
