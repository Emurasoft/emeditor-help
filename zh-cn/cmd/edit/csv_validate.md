# 语法检查命令

## 摘要

检查当前文档的语法并显示结果。

## 说明

检查当前文档的语法并显示结果。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **语法检查**
- 工具栏: ![](../../images/csv_validate.png)
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_CSV\_VALIDATE (3990)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3990);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3990
```
