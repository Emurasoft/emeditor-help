# 展开/折叠当前行命令

## 摘要

当显示大纲视图时展开或折叠当前行。

## 说明

当显示大纲视图时，如果当前行被折叠，这个命令会展开当前行；如果当前行是展开的，那么这个命令会折叠当前行。这与点击文本行左边的「+」按钮或「-」按钮是一样的。要使用该功能，先要启用大纲来显示大纲视图。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>大纲 \>展开/折叠行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_OUTLINE_TOGGLE_LINE (4412)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4412);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4412
```
