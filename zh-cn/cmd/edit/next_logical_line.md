# 下一逻辑行命令

## 摘要

移动光标到下一个逻辑行行首。

## 说明

移动光标到下一个逻辑行行首。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): Edit > **垂直移动光标**
\> **下一逻辑行**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+PAGE DOWN

## 插件命令ID

```
EEID_NEXT_LOGICAL_LINE (4268)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4268);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4268
```
