# 删除当前群组中书签行命令

## 摘要

删除当前群组中所有已添加书签的行。

## 说明

删除当前群组中所有已添加书签的行。

## 运行方法

- 默认菜单:书签 \>当前群组 \>删除书签行
- [所有命令](../tools/all_commands):书签 \>当前群组 \>删除书签行
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_BOOKMARK_GROUP_DELETE (4596)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4596);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4596
```
