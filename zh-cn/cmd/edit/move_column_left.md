# 左移列命令

## 摘要

将选定的列或 CSV 文档中的当前列往左移一列。

## 说明

将选定的列或 CSV 文档中的当前列往左移一列。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands):编辑 \>移动
\>左移列
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_MOVE_COLUMN_LEFT (3995)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3995);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3995
```
