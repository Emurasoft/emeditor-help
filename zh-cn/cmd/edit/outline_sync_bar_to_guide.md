# 同步大纲栏和大纲导航命令

## 摘要

同步大纲栏和大纲导航的展开或折叠状态。

## 说明

当显示大纲时，这个命令会同步大纲栏和大纲导航中的展开或折叠状态。

## 运行方法

- 默认菜单:编辑 \>大纲 \>同步大纲栏和大纲导航
- [所有命令](../tools/all_commands):编辑 \>大纲 \>同步大纲栏和大纲导航
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令ID

```
EEID_OUTLINE_SYNC_BAR_TO_GUIDE (3879)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3879);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3879
```
