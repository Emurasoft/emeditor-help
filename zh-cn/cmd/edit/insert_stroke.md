# 插入划线符命令

## 摘要

在输入字符 o、O 或 c 时在该字符上插入划线符。

## 说明

在选择这个命令后，在光标处插入一个带有划线符的字符 (ø 或 Ø) 通过输入 o 或
O，或一个美分标记 (¢) 通过输入 c。

## 运行方法

- 默认菜单:插入 \>音调符号 \>划线符
- [所有命令](../tools/all_commands):插入 \>音调符号 \>划线符
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+/

## 插件命令ID

```
EEID_INSERT_STROKE (4311)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4311);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4311
```
