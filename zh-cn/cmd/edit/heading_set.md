# 设置标题（冻结窗格）命令

## 摘要

将当前单元格上方的行和左侧的列设置为不可滚动。

## 说明

将当前单元格上方的行和左侧的列设置为不可滚动。

## 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **CSV** \> **标题（冻结窗格）** \> **设置标题（冻结窗格）**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_HEADING\_SET (3881)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3881);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3881
```
