# 二进制文件转换为 Base64 命令

## 摘要

对一个二进制文件进行 Base64 编码。

## 说明

对一个二进制文件进行 Base64 编码。

## 运行方法

- 默认菜单:转换 \>编码/解码所选内容 \>二进制文件转换为 Base64
- [所有命令](../tools/all_commands):转换 \>编码/解码所选内容 \>二进制文件转换为 Base64
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

## 插件命令 ID

- EEID\_ENCODE\_BASE64\_BINARY (3944)

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(3944);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3944
```
