# 插入波形符命令

## 摘要

插入一个带有波形符的字符。

## 说明

在选择这个命令后，在光标处插入一个带有波形符的字符(ã，ñ，õ，ẽ，ũ，Ã，Ñ，Õ，Ẽ 或 Ũ)通过输入 a，n，o，e，u，A，N，O，E 或 U。

## 运行方法

- 默认菜单: **插入** \> **音调符号** \> **波形符**
- [所有命令](../tools/all_commands): **插入** \> **音调符号** \> **波形符**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: CTRL+SHIFT+\`

## 插件命令ID

```
EEID_INSERT_TILDE (4306)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(4306);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4306
```
