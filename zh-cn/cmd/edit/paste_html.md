# 粘贴为 HTML 命令

### 摘要

> 将剪贴板内容粘贴为 HTML 格式。

### 说明

> 用 HTML 格式粘贴剪贴板中的内容。

### 运行方法

- 默认菜单: 无
- [所有命令](../tools/all_commands): **编辑** \> **粘贴**
\> **粘贴为 HTML**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: 无

### 插件命令ID

- EEID\_EDIT\_PASTE\_HTML (4071)

### 宏

#### \[JavaScript\]

> document.selection.Paste(eeCopyHTML);

#### \[VBScript\]

> document.selection.Paste eeCopyHTML
