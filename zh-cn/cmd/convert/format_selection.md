# 格式化选区命令

## 摘要

用语言服务器协议格式化选取区域。

## 说明

用语言服务器协议格式化选取区域。

## 运行方法

- 默认菜单: **转换** \> **格式化选区**
- [所有命令](../tools/all_commands): **转换** \> **格式化选区**
- 工具栏: 无
- 状态栏: 无
- 默认快捷键: Ctrl+K，D

## 插件命令ID

```
EEID_FORMAT_SELECTION (23245)```

## 宏

### \[JavaScript\]

```
editor.ExecuteCommandByID(23245);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 23245
```
