# “收藏夹”页面

**收藏夹** 页面让您能自定义与收藏夹相关的设置。

## “图标”下拉列表框

指定每个文件的图标的显示方式。选择以下选项之一。

|     |     |
| --- | --- |
| **无** | 不显示任何图标。 |
| **文档类型图标** | 显示与文档类型关联的图标。 |
| **EmEditor 图标** | 显示 EmEditor 图标。 |

## “宽度”下拉列表框

指定如何调整每个按钮的宽度。选择以下选项之一。

|     |     |
| --- | --- |
| **按文件名称调整** | 每个按钮都根据其文件名长度进行调整。 |
| **只在超出指定宽度时截短** | 每个按钮都会根据其文件名长度进行调整，但仅当其长度大于 **指定宽度** 文本框中指定的宽度时才会被截短。 |
| **固定为指定的宽度** | 所有按钮的长度与 **指定宽度** 文本框中指定的长度相同。 |

## “当不适合时”下拉列表框

指定当按钮不适合窗口宽度时按钮的显示方式。指定以下选项之一。

|     |     |
| --- | --- |
| **无** | 什么也不做。 |
| **调整宽度** | 调整每个按钮的宽度以适合窗口宽度。 |
| **添加行** | 添加行以显示所有按钮。 |

## “指定宽度”文本框

在“宽度”下拉列表框中选择“只在超出指定宽度时截短”或“固定为指定宽度”时，指定每个按钮的宽度。

## 自动显示或隐藏收藏夹工具栏

勾选此复选框后，当工具栏上没有按钮时，EmEditor 会自动隐藏收藏夹工具栏；当用户向工具栏添加按钮时，EmEditor 会自动显示收藏夹工具栏。

## 「添加」按钮

单击此按钮可将新项目添加到列表中。

## 「删除」按钮

单击此按钮可从列表中删除所选项目。

## “行”文本框

指定打开该文件时初始光标位置的行号。

## “列”文本框

指定打开该文件时初始光标位置的列号。

## 「重置」按钮

重置为默认设定。
