# Markdown 页面

**Markdown** 页面让你能自定义与 Markdown 设置相关的选项。

## “自动显示 Markdown 工具栏”复选框

如果勾选了该复选框，EmEditor 会在打开 Markdown 文件时自动显示 Markdown 工具栏。

## “自动切换到 Markdown 设计视图”复选框

如果勾选了该复选框，EmEditor 会在打开 Markdown 文件时自动切换到 Markdown 设计视图。

## “超链接可以省略 .md 文件扩展名”复选框

如果勾选了该复选框，你可以在超链接中省略 .md 文件扩展名。

## “为 AI 响应创建新文档时切换到 Markdown 视图”复选框

如果勾选了该复选框，EmEditor 会在为 AI 响应创建新文档时切换到 Markdown 视图。

## “转义这些字符”复选框

指定是否在 Markdown 文件中转义指定的字符。如果你选择转义某些字符，可以在复选框下方指定字符。

## 「重置」按钮

重置为默认设定。
