# “剪贴板”页面

**剪贴板** 页面让你能自定义与剪贴板操作相关的设定。

### “粘贴时显示剪贴板历史记录”复选框

如果这个复选框被勾选了，在执行 **[粘贴](../../../cmd/edit/edit_paste)** 命令时，EmEditor 会出现 **显示剪贴板历史记录** 按钮。点击该按钮或执行 **显示剪贴板记录** 命令（默认快捷方式：CTRL + SHIFT + V）会显示 **剪贴板历史记录** 列表框，让您能快速选择以前使用的剪贴板上的文本。

### “监控外部应用程序中的剪贴板复制内容”复选框

如果这个复选框被勾选了，EmEditor 将监控在外部应用程序上复制的剪贴板内容。

### “在启动时检索 Windows 剪贴板历史记录”复选框

如果这个复选框被勾选了，EmEditor 将在启动时检索 Windows 剪贴板历史记录，以便让 EmEditor 的剪贴板历史记录包括启动 EmEditor 之前的所有 Windows 上的剪贴板历史记录。此选项仅在操作系统是 Windows 10 Build 18362 或更高版本时才可用，并且必须在 Windows 设置 - 剪贴板上启用剪贴板历史记录。

### “将最近使用的字符串移动到剪贴板历史记录的顶部”复选框

如果勾选了该复选框，EmEditor 会将最近使用的字符串移动到剪贴板历史记录的顶部。

### 「重置」按钮

重置为默认设置。
