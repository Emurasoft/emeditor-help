# “语法检查”页面

**语法检查** 页面让你能自定义语法检查器。

### “语法检查器引擎”下拉列表框

指定要给 HTML/CSS/JSON 文档使用的语法检查器引擎。

### “禁用语法检查如果行数超过”文本框

如果文档中的行数超过此数量，将禁用 HTML/CSS/JSON 文档的语法检查。

### “禁用语法检查如果行的长度超过”文本框

如果文档中每行的任何行长度超过此数字，将禁用 HTML/CSS/JSON 文档的语法检查。

### “翻译 URL”文本框

指定用于 HTML/CSS/JSON 语法检查结果的“在线翻译”命令的 URL。"%1" 是给目标语言标识符的一个占位符（例如，"ja" 代表日文），还有 "%2" 是给 UTF-8 百分比编码源文本的一个占位符。请注意，只有在“自定义”对话框的 **语言** 页面中选择了非英语时，才能使用翻译功能。

### “验证 XML 文档（XML）”复选框

如果勾选该复选框，EmEditor 将验证 XML 文档。如果不勾选，EmEditor将仅检查XML文档是否格式正确。

### “解析外部定义（XML）”复选框

如果勾选该复选框，则 EmEditor 将尝试解析外部定义，可解析的命名空间，文档类型定义（DTD）外部子集和外部实体引用。

### “自动刷新”复选框

如果勾选该复选框，EmEditor 会自动刷新语法检查高亮。

### “禁用自动刷新如果行数超过”文本框

如果文档中的行数超过此数量，将禁用自动刷新。

### “搜索 URL”文本框

指定用于语法检查结果的“在线搜索”命令的 URL。"%1" 将替换为源文本（UTF-8 百分比编码）。

### 「重置」按钮

重置为默认设定。
