# “工作区”页面

**工作区** 页面让你能自定义与 EmEditor 工作区相关的设定。

## “自动工作区”下拉列表框

指定保存和加载行为。你可以从下列选项中选择:

|     |     |
| --- | --- |
| **无** | 不保存或还原工作区。 |
| **仅保存** | 当即将关闭最后一个群组时，工作区将自动保存，但在启动 EmEditor 时不会自动恢复工作区。 |
| **仅加载** | 当启动 EmEditor 时，将自动恢复工作区，但当要关闭最后一个群组时不会自动保存工作区。 |
| **保存并还原** | 当即将关闭最后一个群组时，将自动保存工作区，并在 EmEditor 启动时自动恢复。 |
| **保存还原并保留撤消信息** | 当即将关闭最后一个群组时，将自动保存工作区，并在 EmEditor 启动时自动恢复。同时还会保存和恢复撤销和重做信息。 |
| **保存还原并保留撤消信息，且不提示保存文件** | 当即将关闭最后一个群组时，将自动保存工作区，并在 EmEditor 启动时自动恢复。同时还会保存和恢复撤销和重做信息，并在关闭最后一个群组时不显示保存文件的提示。 |

## “退出前不再询问”复选框

如果勾选了该复选框，则不会在退出时显示选择工作区选项的对话框。

## “不包含使用临时文件夹的文件”复选框

指定工作区不应包含使用临时文件夹的文件。

## “不包含超过指定行数的文件”复选框

指定工作区不应包含行数超过指定数量的文件。

## “崩溃时保存工作区”复选框

如果勾选了该复选框，EmEditor 会在崩溃时保存工作区，并允许你在下次启动 EmEditor 时恢复工作区。

## “定期保存工作区:”复选框

如果勾选了该复选框，EmEditor 将按指定的时间间隔定期保存工作区，并允许你在启动 EmEditor 时恢复工作区。

## “不要在‘保存更改？’对话框中显示‘保存工作区’按钮”复选框

如果勾选了该复选框，EmEditor 将不在“保存更改？”对话框中显示“保存工作区”按钮。

## 「重置」按钮

重置为默认设置。
