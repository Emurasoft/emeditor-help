# “文本呈现”页面

**文本呈现** 页面让你能自定义在编辑器视图中呈现文本的方式。EmEditor 会即时反应变更的设置，以便你在编辑器中预览它们。

### “使用 DirectWrite”复选框

如果勾选了该复选框，则使用 DirectWrite 在编辑器视图中绘制文本。如果未选中，则使用 GDI 绘制文本。DirectWrite 是 GPU 加速的 DirectX API，与 GDI 相比，它改善了文本的渲染和可读性。 DirectWrite 不适用于打印，打印预览或迷你地图。

### “自动检测设置”复选框

检测显示器的最佳设置。该选项可以为不同的显示器设置不同的设置。如果启用了自动检测设置，则下面的 DirectWrite 选项将被忽略。

### “抗锯齿模式”下拉列表框

设置文本显示的消除锯齿模式。 **ClearType** 是一种消除锯齿方法，可将子像素细节添加到文本中。 **灰度** 是一种更简单的抗锯齿方法。选择 **无** 将关闭消除锯齿功能。

### “呈现模式”下拉列表框

设置文本显示的呈现模式。由于抗锯齿的呈现方式，不同的呈现模式会为文本外观添加细微变化。选择 **默认** 会自动选择呈现模式。如果抗锯齿模式为“无”，则呈现模式将被忽略。ClearType 和呈现模式“轮廓”不能同时使用。请参阅 [MSDN 上的“DWRITE\_RENDERING\_MODE 枚举”](https://docs.microsoft.com/en-us/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode) 来了解每种呈现模式的详细信息。

### “Gamma”滑块

Gamma 值用于校正 Gamma。

### “对比”滑块

调整边缘清晰度。0.0是最低对比度，3.0是最高对比度。

### “ClearType 级别”滑块

ClearType 抗锯齿的比率。值为 0.0 表示不应用 ClearType，1.0 表示 ClearType在发挥全效。

### “使用彩色字体”复选框

使用嵌入的颜色信息绘制颜色到字体中。适用于 Windows 8.1 和更高版本的平台。

### “使用双缓冲”复选框

使用双缓冲在屏幕上来绘制文本。双缓冲可减少闪烁，但会使绘图变慢。如果使用较慢的计算机并且绘图很慢，请取消勾选此复选框。如果使用 DirectWrite，则此复选框禁用。

### “后备字体”列表框

定义不支持当前字体的后备字体。目前显示的字体可以在 [**“自定义字体”** 对话框](../../properties/font/index) 中指定。“后备字体”列表中最上面的字体在选择后备字体时具有最高的优先级。在列表上单击右键将显示其他命令，拖放一个项目将改变顺序。

### “显示组合成的韩文拼音字母”复选框

如果勾选了此复选框，EmEditor 将把一系列正确的韩文拼音字母显示为组合成的字符。例如，"ᄒ ᅡ ᆫ ᄀ ᅳ ᆯ"（不含空格）会被显示为 "한글"。这个选项对于在无法使用合成字符的情况下正确显示旧韩文也有效。

### 「添加」按钮

将新项目添加到列表中。

### 「删除」按钮

从列表中删除所选项目。

### 「重置」按钮

将所有文本呈现选项重设为其默认设置。
