# “自定义启动窗口”对话框

这个对话框会出现当你在 **启动** 窗口中按 **「自定义」** 按钮时。

### “总是显示启动窗口而不打开文件在启动 EmEditor 时”复选框

如果勾选了该复选框，每当 EmEditor 打开一个新文档时，都会显示启动窗口。

### “显示启动窗口当在新文档中按向下键时”复选框

如果勾选了该复选框，当在新文档中按了向下键时启动窗口就会显示。

### “如果没有设置自动工作区，在执行还原之前工作区命令时提示”复选框

如果勾选了该复选框，当你点击 **「还原之前的工作区」** 按钮，EmEditor 会显示提示消息如果没有设置自动工作区的话。

### “显示最近的工作区”复选框

如果勾选了该复选框，启动窗口会显示最近的工作区。

### “显示新增选项”复选框

如果勾选了该复选框，启动窗口会显示新增选项列表。如果你清除此选项以隐藏新增选项列表，每当 EmEditor 更新到带有新增选项的主版本时，还是会显示该列表。

### “最旧的版本”复选框

指定在“新增选项”列表框中显示的最旧的版本。如果指定了将来版本（比当前版本新），则仅显示将来版本中可用的新增选项。

### “显示额外的按钮”复选框

如果勾选了该复选框，启动窗口会显示额外的按钮。

### “对齐主窗口”复选框

如果勾选了该复选框，启动窗口会与主窗口对齐。

### “更改的日期”复选框

如果勾选了该复选框，启动窗口会包括更改的日期。

### “类型”复选框

如果勾选了该复选框，启动窗口会包括文件类型。

### “大小”复选框

如果勾选了该复选框，启动窗口会包括文件大小。

### “大图标”单选按钮

如果选择了该单选按钮，启动窗口会用大图标显示。

### “小图标”单选按钮

如果选择了该单选按钮，启动窗口会用小图标显示。

### “显示更新检查器”复选框

如果勾选了该复选框，则启动窗口将包含“更新检查器”。此选项在商店应用版本或便携式版本的 EmEditor 中不可用，或者在安装 EmEditor 时禁用了更新检查器功能，此选项也不可用。

### “仅当新版本可用时”复选框

如果勾选了该复选框，则即使勾选了 **显示更新检查器** 复选框，仅当新版本可用时，启动窗口才包括更新检查器。
