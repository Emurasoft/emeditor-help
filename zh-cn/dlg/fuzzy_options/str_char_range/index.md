# “字符串/字符范围”对话框

这个对话框会出现，当您选择 [**模糊匹配选项** 对话框](../index) 上的 [**添加** 按钮](../../properties/char_check/index) 时。

## “字符串” 单选按钮/文本框

选择字符串，并指定要匹配的字符串。

## “字符范围” 单选按钮

选择字符范围。

## “Unicode 脚本”列表框

选择 Unicode 脚本。

## “Unicode 的一般类别”列表框

Unicode 的一般类别。

## “最小字符”文本框

输入范围的最小字符。您可以输入至少 4 位十六进制值的字符代码值，带或不带 "U+" 前缀，或实际的一个字符。例如，您可以输入："U+0061"，"0061"，或 "a"。

## “最大字符”文本框

输入与 **最小字符** 文本框相同格式的范围的最大字符。

## “视为 \- 忽略”单选按钮

忽略匹配的字符串或字符。

## “视为 \- 一个字符串”文本框

输入要替换的字符或字符串。如果为空，则忽略指定的字符串或字符范围。字符串或字符可以包括以下转义序列。

|     |     |
| --- | --- |
| \\a | 警告（响铃） |
| \\b | 退格 |
| \\t | 水平制表符 |
| \\v | 垂直制表符 |
| \\\ | 反斜杠 |
| \\xhhhh | Unicode 字符在十六进制表达式中 |
