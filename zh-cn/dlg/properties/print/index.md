# “打印”页面

**打印** 页面让你能设置与打印操作相关的属性。

## “打印行号”复选框

在页面的左边打印行号。

## “忽略颜色与下划线”复选框

当打印时，忽略颜色与下划线。

## “插入分隔线”复选框

在页眉与文本之间以及文本与页脚之间插入水平线。

## “打印时按页面换行”复选框

打印时按页面宽度换行。

## “行间距”文本框

指定打印时行与行之间的距离。这个值对应于一个单一的行高度的 1/10。

## “页眉”复选框

启用指定的页眉。

## “页眉”文本框

用下列标记在页面顶部打印文档信息。

## 「>」按钮

用该按钮从预设的标记列表中选取。

|     |     |
| --- | --- |
| &f | 文件名称 |
| &n | 路径名称 |
| &p | 页号 |
| &a | 总页数 |
| &d | 今天的日期 |
| &t | 当前时间 |
| &c | 居中对齐 |
| &r | 右对齐 |
| &e | 上次修改日期 |
| &u | 上次修改时间 |

## “页脚”复选框

启用指定的页脚。

## “页脚”文本框

用下列标记在页面底部打印文档信息。

## \> button

Use this button to select from a list of predefined marks.

|     |     |
| --- | --- |
| &f | 文件名称 |
| &n | 路径名称 |
| &p | 页号 |
| &a | 总页数 |
| &d | 日期 |
| &t | 时间 |
| &c | 居中对齐 |
| &r | 右对齐 |

## 「页面设置」按钮

用该按钮来指定页面设定。

## 「字体」按钮

指定用来打印的字体。

## 「重置」按钮

重置为默认设定。会显示 [**重置** 对话框](../reset/index) 并让你能从另一个配置复制设定。
