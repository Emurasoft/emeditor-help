# “高亮（2）”页面

**高亮（2）** 页面让你能设置与高亮（2）相关的属性。

## “注释”群组框 \- “开始”文本框

为多行注释选择开头的字符串。

## “注释”群组框 \- “结束”文本框

为多行注释选择结尾的字符串。

## “注释”群组框 \- “行注释”(1) 文本框

指定一个用于行注释的字符串。

## “注释”群组框 \- “行注释”(2) 文本框

指定一个用于行注释的字符串。

## “引号内的字符串”群组框 \- 单引号 \['...'\] 复选框

如果勾选了该复选框，单引号内的字符串将被高亮。

## “引号内的字符串”群组框 \- 双引号 \["..."\] 复选框

如果勾选了该复选框，双引号内的字符串将被高亮。

## “引号内的字符串”群组框 \- “转到下一行”复选框

如果勾选了该复选框，引号内的字符串能横跨多行。

## “引号内的字符串”群组框 \- “转义字符”文本框

为引号标记指定一个转义字符。

## “脚本”群组框 \- “开始”文本框

为脚本指定开始字符串。

## “脚本”群组框 \- “结束”文本框

为脚本指定结束字符串。

## “特殊语法”下拉列表框

从下列选项中选择。

|     |     |
| --- | --- |
| **（无）** | 使用一个与 HTML 无关的文件。 |
| **HTML** | 使用一个 HTML 或 ASP 文件。这个语法使用 HTML，CSS，JavaScript，VBScript，还有 Script 配置。 |
| **HTML-内嵌** | 使用如同 JSP 或 PHP，HTML 内嵌语言的脚本语言。这个语法使用 HTML，CSS，JavaScript，VBScript，还有 Script 配置。 |

## “高亮配对标签”复选框

在 XHTML 和 XML 文档中，EmEditor 能高亮光标附近被标签限定的区域。这个功能只有当一个文档是格式良好的 XML 文档时，才能正常运作。如果在一个文档中没有结束标记的话，比如一个旧的 HTML 文档，那么这个功能就可能不会正常运作。

## “高亮配对括号”复选框

高亮匹配的括号（圆括号或方括号）。

## “自动完成括号/引号标记”复选框

自动完成括号/引号标记。

## “括号对”列表框

自动完成的括号对列表。被勾选的条目会被自动完成，不勾选的条目则不会被自动完成。你不能添加一对新的括号对。

## 「重置」按钮

重置为默认设定。会显示 [**重置** 对话框](../reset/index) 并让你能从另一个配置复制设定。
