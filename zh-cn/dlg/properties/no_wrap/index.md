# “不换行”页面

**不换行** 页面让你能设置与基本操作相关的属性。

### “启用不换行单词”复选框

不允许这些字符在行末换行。

### “不换行单词”列表框

启用列表框中的不换行单词，阻止它们换行。

### 「添加」按钮

点击该按钮添加一个新的条目到列表中。

### 「删除」按钮

点击该按钮从列表中删除被选取的条目。

### 「导入」按钮

从 CSV 类型文件中导入不换行单词。文件内容会按字符串以及是否启用被整理。

### 「导出」按钮

从 CSV 类型文件中导出不换行单词。文件内容会按字符串以及是否启用被整理。

### 「重置」按钮

重置为默认设定。会显示 [**重置** 对话框](../reset/index) 并让你能从另一个配置复制设定。
