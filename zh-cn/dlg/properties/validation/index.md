# “语法检查”页面

语法检查 页面让你能设置与语法检查相关联的属性。

## “启用语法检查”复选框

如果勾选了该复选框，语法检查错误会以红色波浪下划线（可自定义）显示，并且鼠标指针悬停在错误部分会显示有关错误信息的小提示。

## “文档类型”下拉列表框

指定要检查语法的文档类型。请注意，如果选择了CSV，则仅在 CSV 模式下时才能启用语法检查。如果选择了语言服务器协议，将会使用在 [语言服务器 页面](../language_server/index) 上指定的文档类型。

## “显示列表”下拉列表框

|     |     |
| --- | --- |
| （无） | 语法检查结果不会显示为列表。即使结果不显示为列表，如果勾选了启用语法检查 复选框，语法错误也会以红色波浪下划线表示。 |
| 在打开一个文档时 | 打开此配置的文档时，语法检查结果将显示为列表。 |
| 仅在检测到错误时 | 仅当打开此配置的文档时检测到错误时，语法检查结果才会显示为列表。 |

## 「重置」按钮

重置为默认设定。会显示 [重置 对话框](../reset/index) 并让你能从另一个配置复制设定。
