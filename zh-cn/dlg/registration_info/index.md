# “注册信息”对话框

**注册信息**对话框会显示有关您的设备以及订阅或注册密钥的信息。您还可以检查您的订阅或注册密钥是否有效。

您可以通过 **帮助 > 注册信息** 访问此对话框。

## 信息文本

文本框中可能显示以下信息。

- 第一行包含 “EmEditor Professional” 或 “EmEditor Free” 以及版本字符串。
- 如果您已登录，您的电子邮件地址将被显示。
- 如果您尚未注册，将显示“未注册”。
- 设备 ID
- 如果您使用 Stripe 订阅，则显示您的订阅 ID
- 列印您的设备验证结果。如果成功，将显示“验证成功”。如果不成功，将显示诊断信息以帮助解决问题。
    - 如果您需要支持，请[联系我们](https://zh-cn.emeditor.com/support/#contact)。

## “登录/输入密钥”按钮

此按钮会打开[注册产品对话框](../regist/index)。

## 编辑设备信息

此按钮会打开 Emurasoft 客户中心的编辑设备页面，您可以在那里查看有关您的设备的更多信息。
