# Markdown page

The **Markdown** page allows you to customize settings related to Markdown settings.

## Automatically show the Markdown toolbar check box

If this is checked, EmEditor will automatically display the Markdown toolbar when a Markdown file is opened.

## Automatically switch to the Markdown Design View check box

If this is checked, EmEditor will automatically switch to the Markdown Design View when a Markdown file is opened.

## Hyperlinks may omit the .md file extension check box

If this is checked, you many omit the .md file extension for hyperlinks.

## Switch to Markdown View when a new document is created for an AI response check box

If this is checked, EmEditor will switch to Markdown View when a new document is created for an AI response.

## Escape these characters check box

Specify whether to escape specified characters in Markdown files. If you choose to escape some characters, you can specify the characters below the check box.

## Reset button

Resets to default settings.
