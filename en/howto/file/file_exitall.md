# To Exit All

Click in the top, right-hand corner of the window while pressing the SHIFT
key.

## Tips

Alternatively, you can exit all in the following ways.

- Press ALT + SHIFT + F4.
- Press ALT + SHIFT + X.
- SelectClose All from theFile menu.
- SelectClose from theFile menu while pressing the
SHIFT key.
- Add theClose All button on the Toolbar, and click that
button.
- Click the tray icon and selectClose All from the menu.
