# SelectInQuotes Property (LinkProp Object)

Corresponds to the **Clicking**
**in Quotes Selects the Whole String** check box in the [**Link** page](../../dlg/properties/link/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.SelectInQuotes;
object.SelectInQuotes = b;
```

### \[VBScript\]

```
b = object.SelectInQuotes
object.SelectInQuotes = b
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
