# URL Property

Corresponds to the **Link**
**to URLs** check box in the [**Link** page](../../dlg/properties/link/index) of Configuration Properties.

#### \[JavaScript\]

_b_ = object. **URL**;

object. **URL** = _b_;

#### \[VBScript\]

_b_ = object. **URL**

object. **URL** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
