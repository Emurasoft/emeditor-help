# Delimiter Property (Csv Object)

Retrieves or sets the delimiter for the current Csv object.

## 

### \[JavaScript\]

```
s = object.Delimiter;
object.Delimiter = s;
```

### \[VBScript\]

```
s = object.Delimiter
object.Delimiter = s
```

## Version

Supported on EmEditor Professional Version 15.2 or later.
