# Heading Property (Csv Object)

Corresponds to the **Lines of headings** text box in the [**CSV** page](../../dlg/customize/csv/index) of the Customize dialog box.

#### \[JavaScript\]

_n_ = object. **Heading**;

object. **Heading** = _n_;

#### \[VBScript\]

_n_ = object. **Heading**

object. **Heading** = _n_

## Version

Supported on EmEditor Professional Version 19.4 or later.
