# Heading Property (Csv Object)

Corresponds to theLines of headings text box in the [CSV page](../../dlg/customize/csv/index) of the Customize dialog box.

## 

### \[JavaScript\]

```
n = object.Heading;
object.Heading = n;
```

### \[VBScript\]

```
n = object.Heading
object.Heading = n
```

## Version

Supported on EmEditor Professional Version 19.4 or later.
