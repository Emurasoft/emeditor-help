# Escape Property (Csv Object)

Corresponds to the **Use Escape Character** check box in the [**CSV** page](../../dlg/customize/csv/index) of the Customize dialog box.

#### \[JavaScript\]

_b_ = object. **Escape**;

object. **Escape** = _b_;

#### \[VBScript\]

_b_ = object. **Escape**

object. **Escape** = _b_

## Version

Supported on EmEditor Professional Version 19.4 or later.
