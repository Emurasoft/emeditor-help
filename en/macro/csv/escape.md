# Escape Property (Csv Object)

Corresponds to theUse Escape Character check box in the [CSV page](../../dlg/customize/csv/index) of the Customize dialog box.

## 

### \[JavaScript\]

```
b = object.Escape;
object.Escape = b;
```

### \[VBScript\]

```
b = object.Escape
object.Escape = b
```

## Version

Supported on EmEditor Professional Version 19.4 or later.
