# RowHeading Property (Csv Object)

Corresponds to theRow headings text box in the [CSV page](../../dlg/customize/csv/index) of the Customize dialog box.

## 

### \[JavaScript\]

```
n = object.RowHeading;
object.RowHeading = n;
```

### \[VBScript\]

```
n = object.RowHeading
object.RowHeading = n
```

## Version

Supported on EmEditor Professional Version 22.0 or later.
