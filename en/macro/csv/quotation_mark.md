# QuotationMark Property (Csv Object)

Retrieves or sets the quotation mark for the current Csv object.

## 

### \[JavaScript\]

```
ch = object.QuotationMark;
object.QuotationMark = ch;
```

### \[VBScript\]

```
ch = object.QuotationMark
object.QuotationMark = ch
```

## Version

Supported on EmEditor Professional Version 20.1 or later.
