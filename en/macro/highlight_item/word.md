# Word Property

Corresponds to the **Whole**
**Word Only** check box in the
[**Highlight (1)** page](../../dlg/properties/highlight1/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
item. **Word**;

item. **Word** = _b_;

#### \[VBScript\]

_b_ =
item. **Word**

item. **Word** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
