# Percent Property (FuzzyOptions Object)

Sets or retrieves the percent value of similarity.

#### \[JavaScript\]

_n_ = obj. **Percent**;

obj. **Percent** = _n_;

#### \[VBScript\]

_n_ = obj. **Percent**;

obj. **Percent** = _n_;

## Version

Supported on EmEditor Professional Version 22.0 or later.
