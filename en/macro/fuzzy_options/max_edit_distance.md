# MaxEditDistance Property

Sets or retrieves the maximum edit distance.

#### \[JavaScript\]

_n_ = obj. **MaxEditDistance**;

obj. **MaxEditDistance** = _n_;

#### \[VBScript\]

_n_ = obj. **MaxEditDistance**;

obj. **MaxEditDistance** = _n_;

## Version

Supported on EmEditor Professional Version 22.0 or later.
