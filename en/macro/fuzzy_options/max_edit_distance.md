# MaxEditDistance Property (FuzzyOptions Object)

Sets or retrieves the maximum edit distance.

## 

### \[JavaScript\]

```
n = obj.MaxEditDistance;
obj.MaxEditDistance = n;
```

### \[VBScript\]

```
n = obj.MaxEditDistance;
obj.MaxEditDistance = n;
```

## Version

Supported on EmEditor Professional Version 22.0 or later.
