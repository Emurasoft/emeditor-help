# DetectAll Property (FileProp Object)

Corresponds to theDetect
All check box in the [File page](../../dlg/properties/file/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.DetectAll;
object.DetectAll = b;
```

### \[VBScript\]

```
b = object.DetectAll
object.DetectAll = b
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
