# DetectUTF8 Property

Corresponds to the **Detect**
**UTF-8** check box in the
[**File** page](../../dlg/properties/file/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
object. **DetectUTF8**;

object. **DetectUTF8** = _b_;

#### \[VBScript\]

_b_ =
object. **DetectUTF8**

object. **DetectUTF8** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
