# End Property

Corresponds to the **End**
**Indent** text box in the
[**Tab/Index** dialog box](../../dlg/properties/general/indent/index).

#### \[JavaScript\]

_s_ = object. **End**;

object. **End** = _s_;

#### \[VBScript\]

_s_ = object. **End**

object. **End** = _s_

## Version

Supported on EmEditor Professional Version 7.00 or later.
