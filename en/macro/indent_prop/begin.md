# Begin Property (IndentProp Object)

Corresponds to theBegin Indent text box in the [Tab/Index dialog box](../../dlg/properties/general/indent/index).

## 

### \[JavaScript\]

```
s = object.Begin;
object.Begin = s;
```

### \[VBScript\]

```
s = object.Begin
object.Begin = s
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
