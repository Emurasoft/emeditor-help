# Begin Property (IndentProp Object)

Corresponds to the **Begin Indent** text box in the [**Tab/Index** dialog box](../../dlg/properties/general/indent/index).

#### \[JavaScript\]

_s_ = object. **Begin**;

object. **Begin** = _s_;

#### \[VBScript\]

_s_ = object. **Begin**

object. **Begin** = _s_

## Version

Supported on EmEditor Professional Version 7.00 or later.
