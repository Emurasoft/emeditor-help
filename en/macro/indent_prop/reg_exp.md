# RegExp Property

Corresponds to the **Use Regular Expressions** check box in the
[**Tab/Index** dialog box](../../dlg/properties/general/indent/index).

#### \[JavaScript\]

_b_ = object. **RegExp**;

object. **RegExp** = _b_;

#### \[VBScript\]

_b_ = object. **RegExp**

object. **RegExp** = _b_

## Version

Supported on EmEditor Professional Version 9.00 or later.
