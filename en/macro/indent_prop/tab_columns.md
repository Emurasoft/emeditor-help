# TabColumns Property (IndentProp Object)

Corresponds to the **Tab**
**Columns** text box in the
[**Tab/Index** dialog box](../../dlg/properties/general/indent/index).

## 

### \[JavaScript\]

```
n = object.TabColumns;
object.TabColumns = n;
```

### \[VBScript\]

```
n = object.TabColumns
object.TabColumns = n
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
