# FileSaveProp Object

## Properties

|     |     |
| --- | --- |
| [AlwaysEnabled](always_enabled) | Corresponds to the **Always Enable Saving** check box in the [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [DefaultExtension](default_extension) | Corresponds to the **Default**<br>**Extension** text box in the [**File** page](../../dlg/properties/file/index) of Configuration Properties. |
| [DeleteEmpty](delete_empty) | Corresponds to the **Delete**<br>**Empty Files when Saving** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [DeleteSpacesAtEnd](delete_spaces_at_end) | Corresponds to the **Delete**<br>**Spaces at End of Lines** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [Encoding](encoding) | Corresponds to the **Encoding drop-down** list box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [InsertReturns](insert_returns) | Corresponds to the **Insert**<br>**Returns when Saving** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [NamedEntity](named_entity) | Corresponds to the **Use**<br>**Named Entity Reference** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [NameUntitled](name_untitled) | Corresponds to the **Automatically Name Untitled Document** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [NumericEntity](numeric_entity) | Corresponds to the **Save**<br>**Unicode as HTML/XML Character Reference** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [PromptRename](prompt_rename) | Corresponds to the **Prompt before Deleting an Old File when Renaming** check box in the [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [PromptUnicode](prompt_unicode) | Corresponds to the **Prompt**<br>**when Saving Unicode as ANSI** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [ReturnMethod](return_method) | Corresponds to the **Return**<br>**Method drop-down** list box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [TabsAsSpaces](tabs_as_spaces) | Corresponds to the **Save**<br>**Tabs as Spaces** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |
| [UnicodeSignature](unicode_signature) | Corresponds to the **Add a Unicode Signature (BOM)** check box in the <br> [**Save Details** dialog box](../../dlg/properties/file/save_details/index). |

## Version

Supported on EmEditor Professional Version 7.00 or later.
