# FileSaveProp Object

## Properties

|     |     |
| --- | --- |
| [AlwaysEnabled](always_enabled) | Corresponds to theAlways Enable Saving check box in the [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [DefaultExtension](default_extension) | Corresponds to theDefault<br>Extension text box in the [File page](../../dlg/properties/file/index) of Configuration Properties. |
| [DeleteEmpty](delete_empty) | Corresponds to theDelete<br>Empty Files when Saving check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [DeleteSpacesAtEnd](delete_spaces_at_end) | Corresponds to theDelete<br>Spaces at End of Lines check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [Encoding](encoding) | Corresponds to theEncoding drop-down list box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [InsertReturns](insert_returns) | Corresponds to theInsert<br>Returns when Saving check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [NamedEntity](named_entity) | Corresponds to theUse<br>Named Entity Reference check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [NameUntitled](name_untitled) | Corresponds to theAutomatically Name Untitled Document check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [NumericEntity](numeric_entity) | Corresponds to theSave<br>Unicode as HTML/XML Character Reference check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [PromptRename](prompt_rename) | Corresponds to thePrompt before Deleting an Old File when Renaming check box in the [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [PromptUnicode](prompt_unicode) | Corresponds to thePrompt<br>when Saving Unicode as ANSI check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [ReturnMethod](return_method) | Corresponds to theReturn<br>Method drop-down list box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [TabsAsSpaces](tabs_as_spaces) | Corresponds to theSave<br>Tabs as Spaces check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |
| [UnicodeSignature](unicode_signature) | Corresponds to theAdd a Unicode Signature (BOM) check box in the <br> [Save Details dialog box](../../dlg/properties/file/save_details/index). |

## Version

Supported on EmEditor Professional Version 7.00 or later.
