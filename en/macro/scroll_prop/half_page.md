# HalfPage Property

Corresponds to the **Scroll Half Page** check box in the
[**Scroll** page](../../dlg/properties/scroll/index) of Configuration Properties.

#### \[JavaScript\]

_b_ =
object. **HalfPage**;

object. **HalfPage** = _b_;

#### \[VBScript\]

_b_ =
object. **HalfPage**

object. **HalfPage** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
