# WordWrap Property (WrapProp Object)

Corresponds to theWord
Wrap check box in the [Wrap page](../../dlg/properties/wrap/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.WordWrap;
object.WordWrap = b;
```

### \[VBScript\]

```
str = object.WordWrap
object.WordWrap = str
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
