# Enabled Property (CharacterRange Object)

Specifies the flag indicating whether the object is enabled.

## 

### \[JavaScript\]

```
b = item.Enabled;
item.Enabled = b;
```

### \[VBScript\]

```
b = item.Enabled
item.Enabled = b
```

## Version

Supported on EmEditor Professional Version 22.0 or later.
