# Enabled Property

Corresponds to the **Enable**
**Association** check box in the
[**Association** page](../../dlg/properties/associate/index) of Configuration Properties.

#### \[JavaScript\]

_b_ = object. **Enabled**;

object. **Enabled** = _b_;

#### \[VBScript\]

_b_ = object. **Enabled**

object. **Enabled** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
