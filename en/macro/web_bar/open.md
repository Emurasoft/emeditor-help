# Open Method (WebBar Object)

Opens a web site of the specified URL.

## 

### \[JavaScript\]

```
WebBar.Open( sURL );
```

### \[VBScript\]

```
WebBar.Open sURL
```

## Parameters

_sURL_

Specifies the URL string to be displayed.

## Version

Supported on EmEditor Professional Version 23.0 or later.
