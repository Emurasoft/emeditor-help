# Visible Property (WebBar Object)

Shows or hides the Web Browser.

## 

### \[JavaScript\]

```
bVisible = WebBar.Visible;
WebBar.Visible = bVisible;
```

### \[VBScript\]

```
bVisible = WebBar.Visible;
WebBar.Visible = bVisible;
```

## Version

Supported on EmEditor Professional Version 23.0 or later.
