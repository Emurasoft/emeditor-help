# SeparateCrLf Property (Regex Object)

Sets or retrieves whether the regular expression treats CR and LF separately.

## 

### \[JavaScript\]

```
b = reg.SeparateCrLf;
reg.SeparateCrLf = b;
```

### \[VBScript\]

```
b = reg.SeparateCrLf;
reg.SeparateCrLf = b;
```

## Version

Supported on EmEditor Professional Version 15.9 or later.
