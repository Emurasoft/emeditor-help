# OnlyWord Property (Regex Object)

Sets or retrieves whether the regular expression matches only whole words.

## 

### \[JavaScript\]

```
b = reg.OnlyWord;
reg.OnlyWord = b;
```

### \[VBScript\]

```
b = reg.OnlyWord;
reg.OnlyWord = b;
```

## Version

Supported on EmEditor Professional Version 15.9 or later.
