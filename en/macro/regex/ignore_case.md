# IgnoreCase Property (Regex Object)

Sets or retrieves whether the regular expression ignores cases.

## 

### \[JavaScript\]

```
b = reg.IgnoreCase;
reg.IgnoreCase = b;
```

### \[VBScript\]

```
b = reg.IgnoreCase;
reg.IgnoreCase = b;
```

## Version

Supported on EmEditor Professional Version 15.9 or later.
