# Visible Property (OutputBar Object)

Shows or hides the output bar.

#### \[JavaScript\]

_bVisible_ = OutputBar. **Visible**;

OutputBar. **Visible** = _bVisible_;

#### \[VBScript\]

_bVisible_ = OutputBar. **Visible**;

OutputBar. **Visible** = _bVisible_;

## Version

Supported on EmEditor Professional Version 7.00 or later.
