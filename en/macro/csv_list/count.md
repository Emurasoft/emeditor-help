# Count Property (CsvList Collection)

Retrieves the number of Csv objects.

## 

### \[JavaScript\]

```
nCount = editor.CsvList.Count;
```

### \[VBScript\]

```
nCount = editor.CsvList.Count
```

## Examples

### \[JavaScript\]

```
alert( "Number of Csv objects: " + editor.CsvList.Count );
```

### \[VBScript\]

```
alert "Number of Csv objects: " & editor.CsvList.Count
```

## Version

Supported on EmEditor Professional Version 19.4 or later.
