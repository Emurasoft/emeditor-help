# Add Method (CsvList Collection)

Adds an item.

## 

### \[JavaScript\]

```
list.Add( strName, strDelimiter );
```

### \[VBScript\]

```
list.Add strName, strDelimiter
```

## Parameters

_strName_

Specifies a name.

_strDelimiter_

Specifies a delimiter.

## Version

Supported on EmEditor Professional Version 19.4 or later.
