# Insert Method

Inserts an item.

#### \[JavaScript\]

list. **Insert**( _Index_, _obj_ );

#### \[VBScript\]

list. **Insert** _Index_, _obj_

## Parameters

_Index_

Specifies the index of the item as a one-based integer.

_obj_

Specifies an item.

## Version

Supported on EmEditor Professional Version 19.4 or later.
