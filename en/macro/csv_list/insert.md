# Insert Method (CsvList Collection)

Inserts an item.

## 

### \[JavaScript\]

```
list.Insert( Index, obj );
```

### \[VBScript\]

```
list.Insert Index, obj
```

## Parameters

_Index_

Specifies the index of the item as a one-based integer.

_obj_

Specifies an item.

## Version

Supported on EmEditor Professional Version 19.4 or later.
