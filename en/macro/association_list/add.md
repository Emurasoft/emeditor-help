# Add Method (AssociationList Collection)

Adds an item.

## 

### \[JavaScript\]

```
list.Add( sName, bEnabled );
```

### \[VBScript\]

```
list.Add sName, bEnabled
```

## Parameters

_sName_

Specifies the name of the item to add.

_bEnabled_

Specifies whether the item is enabled.

## Version

Supported on EmEditor Professional Version 7.00 or later.
