# Interval Property (AutoSaveProp Object)

Corresponds to the Minutes text box in the Auto Save page of Configuration Properties.

## 

### \[JavaScript\]

```
n = object.Interval;
object.Interval = n;
```

### \[VBScript\]

```
n = object.Interval
object.Interval = n
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
