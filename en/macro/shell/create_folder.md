# CreateFolder Method (Shell Object)

Creates a folder.

## 

### \[JavaScript\]

```
shell.CreateFolder( strFolder );
```

### \[VBScript\]

```
shell.CreateFolder strFolder
```

## Parameters

_strFolder_

Specifies a folder to create.

## Examples

### \[JavaScript\]

```
shell.CreateFolder( "C:\\\Test" );
```

### \[VBScript\]

```
shell.CreateFolder "C:\\Test"
```

## Version

Supported on EmEditor Professional Version 22.1 or later.
