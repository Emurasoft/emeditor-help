# CreateFolder Method

Creates a folder.

#### \[JavaScript\]

shell. **CreateFolder**( _strFolder_ );

#### \[VBScript\]

shell. **CreateFolder** _strFolder_

## Parameters

_strFolder_

Specifies a folder to create.

## Examples

#### \[JavaScript\]

shell.CreateFolder( "C:\\\Test" );

#### \[VBScript\]

shell.CreateFolder "C:\\Test"

## Version

Supported on EmEditor Professional Version 22.1 or later.
