# KeepRunning Property (Shell Object)

Retrieves or sets the flag to keep running the macro while using V8.

## 

### \[JavaScript\]

```
b = shell.KeepRunning;
shell.KeepRunning = b;
```

### \[VBScript\]

```
b = shell.KeepRunning
shell.KeepRunning = b
```

## Version

Supported on EmEditor Professional Version 23.1 or later.
