# FullName Property

Retrieves the complete path and the file name of the document.

#### \[JavaScript\]

_strName_ = document. **FullName;**

#### \[VBScript\]

_strName_ = document. **FullName**

## Examples

#### \[JavaScript\]

alert( document.FullName );

#### \[VBScript\]

alert document.FullName

## Version

Supported on EmEditor Professional Version 4.00 or later.
