# filters Property (Document Object)

Retrieves or sets the [**Filters** Collection](../filters/index).

## 

### \[JavaScript\]

```
fs = document.filters;
document.filters = fs;
```

### \[VBScript\]

```
Set fs = document.filters
document.filters = fs
```

## Version

Supported on EmEditor Professional Version 16.0 or later.
