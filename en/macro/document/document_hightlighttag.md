# HighlightTag Property (Document Object)

Determines whether to highlight tags.

## 

### \[JavaScript\]

```
bHighlightTag = document.HighlightTag;
document.HighlightTag = bHighlightTag;
```

### \[VBScript\]

```
bHighlightTag = document.HighlightTag
document.HighlightTag = bHighlightTag
```

## Version

Supported on EmEditor Professional Version 4.00 or later.
