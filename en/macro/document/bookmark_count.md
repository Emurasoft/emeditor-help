# BookmarkCount Property

Retrieves the number of bookmarks in the document.

## 

### \[JavaScript\]

```
n = document.BookmarkCount;
```

### \[VBScript\]

```
n = document.BookmarkCount
```

## Version

Supported on EmEditor Professional Version 17.9 or later.
