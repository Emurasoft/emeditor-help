# Path Property (Document Object)

Retrieves the path of the document without the file name.

#### \[JavaScript\]

_strPath_ = document. **Path**;

#### \[VBScript\]

_strPath_ = document. **Path**

## Examples

#### \[JavaScript\]

alert( document.Path );

#### \[VBScript\]

alert document.Path

## Version

Supported on EmEditor Professional Version 4.00 or later.
