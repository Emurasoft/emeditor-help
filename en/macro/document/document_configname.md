# ConfigName Property (Document Object)

Retrieves or sets the current configuration name.

## 

### \[JavaScript\]

```
strName = document.ConfigName;
document.ConfigName = strName;
```

### \[VBScript\]

```
strName = document.ConfigName
document.ConfigName = strName
```

## Version

Supported on EmEditor Professional Version 4.00 or later.
