# writeln Method

Passes a string and a newline character to a supported plug-in.

#### \[JavaScript\]

Interface. **writeln**( _str_ );

#### \[VBScript\]

Interface. **writeln** _str_

## Parameters

_str_

Specifies the string to be passed.

## Examples

#### \[JavaScript\]

Interface. **writeln**( "Hello" );

#### \[VBScript\]

Interface. **writeln** "Hello"

## Version

Supported on EmEditor Professional Version 9.00 or later.
