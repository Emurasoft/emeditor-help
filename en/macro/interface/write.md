# write Method

Passes a string to a supported plug-in.

#### \[JavaScript\]

Interface. **write**( _str_ );

#### \[VBScript\]

Interface. **write** _str_

## Parameters

_str_

Specifies the string to be passed.

## Examples

#### \[JavaScript\]

Interface. **write**( "Hello" );

#### \[VBScript\]

Interface. **write** "Hello"

## Version

Supported on EmEditor Professional Version 9.00 or later.
