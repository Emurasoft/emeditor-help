# TagEnd Property (HighlightProp Object)

Corresponds to the **End**
**Tag** text box in the
[**Highlight (1)** page](../../dlg/properties/highlight1/index) of Configuration Properties.

#### \[JavaScript\]

_str_ = object. **TagEnd**;

object. **TagEnd** = _str_;

#### \[VBScript\]

_str_ = object. **TagEnd**

object. **TagEnd** = _str_

## Version

Supported on EmEditor Professional Version 7.00 or later.
