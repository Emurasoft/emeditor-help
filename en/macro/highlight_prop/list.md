# List Property (HighlightProp Object)

Retrieves the [HighlightList Collection](../highlight_list/index).

## 

### \[JavaScript\]

```
b = object.List;
object.List = b;
```

### \[VBScript\]

```
b = object.List
object.List = b
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
