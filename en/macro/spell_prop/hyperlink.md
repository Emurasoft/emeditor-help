# Hyperlink Property (SpellProp Object)

Corresponds to theHyperlinks check box in the [Spelling page](../../dlg/properties/spell/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.Hyperlink;
object.Hyperlink = b;
```

### \[VBScript\]

```
b = object.Hyperlink
object.Hyperlink = b
```

## Version

Supported on EmEditor Professional Version 10.00 or later.
