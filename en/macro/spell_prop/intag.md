# InTag Property (SpellProp Object)

Corresponds to theTags check box in the
[Spelling page](../../dlg/properties/spell/index) of Configuration Properties.

## 

### \[JavaScript\]

```
b = object.InTag;
object.InTag = b;
```

### \[VBScript\]

```
b = object.InTag
object.InTag = b
```

## Version

Supported on EmEditor Professional Version 10.00 or later.
