# ShowIndentGuides Property

Corresponds to the Show Indent Guides check box in the Mark tab of Configuration Properties.

#### \[JavaScript\]

_b_ =
object. **ShowIndentGuides**;

object. **ShowIndentGuides**
= _b_;

#### \[VBScript\]

_b_ =
object. **ShowIndentGuides**

object. **ShowIndentGuides**
= _b_

## Version

Supported on EmEditor Professional Version 8.00 or later.
