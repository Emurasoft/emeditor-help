# Enabled Property (AssociationItem Object)

Specifies the flag indicating whether the object is enabled.

#### \[JavaScript\]

_b_ =
item. **Enabled**;

item. **Enabled** = _b_;

#### \[VBScript\]

_b_ =
item. **Enabled**

item. **Enabled** = _b_

## Version

Supported on EmEditor Professional Version 7.00 or later.
