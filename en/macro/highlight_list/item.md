# Item Property

Retrieves the [HighlightItem object](../highlight_item/index) for the specified index.

#### \[JavaScript\]

_item_ =
list. **Item**( _Index_ );

#### \[VBScript\]

_item_ =
list. **Item**( _Index_ )

## Parameters

_Index_

Specifies the index of the item as a one-based integer.

## Version

Supported on EmEditor Professional Version 7.00 or later.
