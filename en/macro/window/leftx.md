# LeftX Property (Window Object)

Returns or sets the horizontal position of the specified window, in pixels.

#### \[JavaScript\]

_x_ = wnd. **LeftX**;

wnd. **LeftX** = _x_;

#### \[VBScript\]

_x_ = wnd. **LeftX**

wnd. **LeftX** = x

## Version

Supported on EmEditor Professional Version 7.00 or later.
