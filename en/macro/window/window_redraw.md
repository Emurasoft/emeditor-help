# Redraw Property (Window Object)

Allows changes in EmEditor to be redrawn or prevents changes in EmEditor from being redrawn.

## 

### \[JavaScript\]

```
b =Redraw;
Redraw = b;
```

### \[VBScript\]

```
b =Redraw
Redraw = b;
```

## Version

Supported on EmEditor Professional Version 5.00 or later.
