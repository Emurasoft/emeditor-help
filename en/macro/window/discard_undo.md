# DiscardUndo Property

Returns the flag to indicate whether EmEditor discards undo information to improve the speed of replace, insert or delete.

#### \[JavaScript\]

_b_ = **DiscardUndo**;

**DiscardUndo** = _b_;

#### \[VBScript\]

_b_ = **DiscardUndo**

**DiscardUndo** = _b_

## Version

Supported on EmEditor Professional Version 14.4 or later.
