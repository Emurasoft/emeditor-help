# Sleep Method (Window Object)

Pauses the macro execution for the specified time in milliseconds.

## 

### \[JavaScript\]

```
Sleep( nMilliseconds);
```

### \[VBScript\]

```
Sleep nMilliseconds
```

## Parameters

_nMilliseconds_

Specifies a time to pause in milliseconds.

## Version

Supported on EmEditor Professional Version 4.02 or later.
