# Style Property (DisplayItem Object)

Specifies the style of the object.

#### \[JavaScript\]

_n_ = item. **Style**;

item. **Style** = _n_;

#### \[VBScript\]

_n_ = item. **Style**

item. **Style** = _n_

## Notes

The nStyle parameter should be one of the following values.

|     |     |
| --- | --- |
| 0 | Normal |
| 1 | Underline |
| 2 | Bold |
| 3 | Italic |
| 4 | Wiggly line |
| 5 | Dotted line |
| 6 | Thick line |

## Version

Supported on EmEditor Professional Version 7.00 or later.
