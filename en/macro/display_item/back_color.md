# BackColor Property (DisplayItem Object)

Specifies the background color of the object.

## 

### \[JavaScript\]

```
rgb = item.BackColor;
item.BackColor = rgb;
```

### \[VBScript\]

```
rgb = item.BackColor;
item.BackColor = rgb;
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
