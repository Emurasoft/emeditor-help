# LineColor Property (DisplayItem Object)

Specifies the underline color of the object.

## 

### \[JavaScript\]

```
rgb = item.LineColor;
item.LineColor = rgb;
```

### \[VBScript\]

```
rgb = item.LineColor;
item.LineColor = rgb;
```

## Version

Supported on EmEditor Professional Version 19.1 or later.
