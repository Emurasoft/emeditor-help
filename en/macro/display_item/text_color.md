# TextColor Property (DisplayItem Object)

Specifies the background color of the object.

## 

### \[JavaScript\]

```
rgb = item.TextColor;
item.TextColor = rgb;
```

### \[VBScript\]

```
rgb = item.TextColor;
item.TextColor = rgb;
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
