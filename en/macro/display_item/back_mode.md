# BackMode Property (DisplayItem Object)

Specifies the background color mode.

## 

### \[JavaScript\]

```
nMode = item.BackMode;
item.BackMode = nMode;
```

### \[VBScript\]

```
nMode = item.BackMode
item.BackMode = nMode
```

## Notes

The nMode parameter should be one of the following values.

|     |     |
| --- | --- |
| 0 | Custom |
| 1 | Default |
| 2 | Transparent |

## Version

Supported on EmEditor Professional Version 7.00 or later.
