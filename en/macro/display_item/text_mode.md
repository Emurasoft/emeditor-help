# TextMode Property (DisplayItem Object)

Specifies the text color mode.

## 

### \[JavaScript\]

```
nMode = item.TextMode;
item.TextMode = nMode;
```

### \[VBScript\]

```
nMode = item.TextMode
item.TextMode = nMode
```

## Notes

The nMode parameter should be one of the following values.

|     |     |
| --- | --- |
| 0 | Custom |
| 1 | Default |
| 2 | Transparent |

## Version

Supported on EmEditor Professional Version 7.00 or later.
