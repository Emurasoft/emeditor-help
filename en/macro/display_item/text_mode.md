# TextMode Property

Specifies the text color mode.

#### \[JavaScript\]

_nMode_ = item. **TextMode**;

item. **TextMode** = _nMode_;

#### \[VBScript\]

_nMode_ = item. **TextMode**

item. **TextMode** = _nMode_

## Notes

The nMode parameter should be one of the following values.

|     |     |
| --- | --- |
| 0 | Custom |
| 1 | Default |
| 2 | Transparent |

## Version

Supported on EmEditor Professional Version 7.00 or later.
