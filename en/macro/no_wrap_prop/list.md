# List Property (NoWrapProp Object)

Retrieves the [NoWrapList collection](../no_wrap_list/index).

## 

### \[JavaScript\]

```
list = object.List;
object.List = list;
```

### \[VBScript\]

```
list = object.List
object.List = list
```

## Version

Supported on EmEditor Professional Version 7.00 or later.
