# EE\_EMPTY\_UNDO\_BUFFER

Empties the buffer for the Undo and Redo commands. You can send this message
explicitly or use the
[Editor\_EmptyUndoBuffer](../macro/editor_emptyundobuffer)
inline function.

EE\_EMPTY\_UNDO\_BUFFER

wParam = 0;

lParam = 0;

## Parameters

None.

## Return Values

The return value is not used.
