# Jump Previous command

## Summary

Jumps to the previous location in the Output Bar.

## Description

Jumps to the previous location in the Output Bar.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Advanced**
\> **Jump Previous**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: SHIFT+F4

## Plug-in Command ID

```
EEID_JUMP_PREV (4443)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4443);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4443
```
