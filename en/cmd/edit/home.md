# Home command

## Summary

Moves the cursor to the start of the current line.

## Description

Moves the cursor to the start of the current line.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands):Edit \>Move Cursor Horizontally
\>Home
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: CTRL + ALT + HOME

## Plug-in Command ID

```
EEID_HOME (4164)```

## Macros

### \[JavaScript\]

```
document.selection.StartOfLine(false,eeLineView);
```

### \[VBScript\]

```
document.selection.StartOfLine false,eeLineView
```
