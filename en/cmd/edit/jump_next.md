# Jump Next command

### Summary

> Jumps to the next location in the Output Bar.

### Description

> Jumps to the next location in the Output Bar.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Advanced**
\> **Jump Next**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: F4

### Plug-in Command ID

- EEID\_JUMP\_NEXT (4442)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(4442);

#### \[VBScript\]

> editor.ExecuteCommandByID 4442
