# Next Cell command

### Summary

> Jumps to the next cell in CSV mode.

### Description

> Jumps to the next cell in CSV mode.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **CSV** \> **Next Cell**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

### Plug-in Command ID

- EEID\_NEXT\_CELL (4490)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(4490);

#### \[VBScript\]

> editor.ExecuteCommandByID 4490
