# Paste as HTML command

## Summary

Pastes Clipboard contents as the HTML format.

## Description

Pastes Clipboard contents as the HTML format.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Paste**
\> **Paste as HTML**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

## Plug-in Command ID

```
EEID_EDIT_PASTE_HTML (4071)```

## Macros

### \[JavaScript\]

```
document.selection.Paste(eeCopyHTML);
```

### \[VBScript\]

```
document.selection.Paste eeCopyHTML
```
