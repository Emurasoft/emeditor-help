# Cancel command

## Summary

Cancels the current selection.

## Description

Unselects any selection on the current document.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Extend Selection**
\> **Collapse**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: ESC

## Plug-in Command ID

```
EEID_ESCAPE (4187)```

## Macros

### \[JavaScript\]

```
document.selection.Collapse();
```

### \[VBScript\]

```
document.selection.Collapse
```
