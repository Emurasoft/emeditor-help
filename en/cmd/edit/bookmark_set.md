# Set Bookmark command

### Summary

> Sets a bookmark on the current line.

### Description

> Sets a bookmark on the current line.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Bookmarks** \> **Set Bookmark**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

### Plug-in Command ID

- EEID\_BOOKMARK\_SET (4353)

### Macros

#### \[JavaScript\]

> document.selection.SetBookmark();

#### \[VBScript\]

> document.selection.SetBookmark
