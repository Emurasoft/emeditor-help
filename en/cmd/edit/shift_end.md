# End Extend command

## Summary

Extends the selection to the end of the current line.

## Description

Extends the selection to the end of the current line.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Extend Selection**
\> **End Extend**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: SHIFT+END

## Plug-in Command ID

```
EEID_SHIFT_END (4182)```

## Macros

### \[JavaScript\]

```
document.selection.EndOfLine(true,eeLineView);
```

### \[VBScript\]

```
document.selection.EndOfLine(true,eeLineView);
```
