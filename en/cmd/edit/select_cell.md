# Select Cell command

## Summary

Selects the cell of the current cursor position in CSV mode.

## Description

Selects the cell of the current cursor position in CSV mode.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **CSV** \> **Select Cell**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

## Plug-in Command ID

```
EEID_SELECT_CELL (4489)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4489);
```

### \[VBScript\]

```
