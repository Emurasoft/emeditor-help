# Line Down command

### Summary

> Moves the cursor down one line.

### Description

> Moves the cursor down one line.Â  Equivalent to pushing the DOWN ARROW
> key once.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Move Cursor Vertically**
\> **Line Down**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: DOWN ARROW

### Plug-in Command ID

- EEID\_DOWN (4161)

### Macros

#### \[JavaScript\]

> document.selection.LineDown(false,1);

#### \[VBScript\]

> document.selection.LineDown false,1
