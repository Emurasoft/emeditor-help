# Bottom command

## Summary

Moves the cursor to the end of the document.

## Description

Moves the cursor to the end of the document.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Edit** \> **Move Cursor Vertically**
\> **Bottom**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: CTRL+END

## Plug-in Command ID

```
EEID_BOTTOM (4169)```

## Macros

## \[JavaScript\]

```
document.selection.EndOfDocument(false);
```

## \[VBScript\]

```
document.selection.EndOfDocument false
```
