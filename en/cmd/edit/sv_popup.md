# CSV (popup menu) command

### Summary

> Shows a popup menu to select a CSV mode.

### Description

> Shows a popup menu to select a CSV mode.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **CSV** \> **CSV (Popup Menu)**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None
