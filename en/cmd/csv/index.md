| **[Use Alphabetical Characters for Column Header](../view/header_alpha)** | Uses alphabetical characters for the column header. |
| **[Use Zero-Based Numerical Characters for Column Header](../view/header_zero_base)** | Uses zero-based numerical characters for the column header. |
| **[Toggle Headings (Freeze or Unfreeze Panes)](../edit/heading_toggle)** | Sets or resets lines above and columns left of the current cell non-scrollable. |

| **[Row Heading 1](../csv/row_heading_1)** | Sets the first one column heading and non-scrollable. |
| **[Row Heading 2](../csv/row_heading_2)** | Sets the first two columns headings and non-scrollable. |
| **[Row Heading 3](../csv/row_heading_3)** | Sets the first three columns headings and non-scrollable. |
| **[Row Heading 4](../csv/row_heading_4)** | Sets the first four columns headings and non-scrollable. |
| **[Read Only in Column Headings](../edit/read_only_headings)** | Toggle the option to make the column headings as read only. |
| **[Heading (Popup)](../edit/heading_popup)** | Shows a popup menu to set headings. |
| **[Convert to CSV (multiple items)](../edit/convert_to_sv)** | Converts the current separated value document or fixed width column document to the specified separated value document (multiple items). |
| **[Fixed Width Columns](../edit/convert_to_fixed_width)** | Converts the current separated value document to the fixed width columns using the current separator positions. |
| **[CSV Converter](../edit/csv_converter)** | Shows or hides the CSV Converter. |
| **[Adjust Separator Positions (Whole Document)](../edit/adjust_column)** | Adjusts all separator positions in the CSV document. |
| **[Adjust Separator Positions (Visible Lines Only)](../edit/adjust_column_visible)** | Adjusts separator positions in the visible lines only of the CSV document. |
| **[Hide Columns](../csv/hide_column)** | Hides the selected columns or current column. |
| **[Unhide Columns](../csv/unhide_column)** | Unhides the selected columns or current column. |
| **[Reset Column Width](../edit/reset_column_width)** | Resets the width of the selected columns of the CSV document. |
| **[Align Left](../csv/align_left)** | Align text in the selected columns or current column left. |
| **[Align Center](../csv/align_center)** | Align text in the selected columns or current column center. |
| **[Align Right](../csv/align_right)** | Align text in the selected columns or current column right. |
| **[Select CSV Column without Headings](../edit/select_column)** | Selects the current column without headings as vertical selection mode in the CSV document. |
| **[Select CSV Column with Headings](../edit/select_column_with_headings)** | Selects the current column with headings as vertical selection mode in the CSV document. |
| **[Insert Column Left](../edit/insert_column)** | Inserts a new column at the left side of the current column in the separated value document. |
| **[Insert Column Right](../edit/insert_column_right)** | Inserts a new column at the right side of the current column in the separated value document. |
| **[Add Column](../edit/add_column)** | Adds a new column at the right side of the last column in the separated value document. |
| **[Duplicate Columns](../edit/duplicate_column)** | Duplicates selected columns of the CSV document. |
| **[Move/Copy Columns](../edit/move_column)** | Moves or copies selected columns or the current column of the CSV document to the specified position. |
| **[Manage Columns](../csv/manage_columns)** | Moves, copies, deletes, or sorts columns of the CSV document. |
| **[Combine Columns](../edit/combine_columns)** | Combines selected columns of the CSV document. |
| **[Split Columns](../edit/split_column)** | Splits the selected columns of the CSV document. |
| **[Make Number of Columns Same](../edit/make_columns_same)** | Makes the number of columns same in a CSV document. |
| **[Remove Unnecessary Quotes](../edit/remove_unnecessary_quotes)** | Removes unnecessary double-quotes in the CSV documents. |
| **[Remove Embedded Newlines in CSV](../edit/remove_embedded_nl)** | Removes embedded newlines in CSV documents. |
| **[Quote All Cells](../edit/add_quotes)** | Quotes all cells in the CSV document with double-quotes. |
| **[Combine Lines](../edit/combine_lines)** | Combines vertical adjacent duplicate cells of the CSV document. |
| **[Flash Fill (Cell Selection Mode Only)](../edit/flash_fill)** | Automatically fills values in the entire column of the CSV document. |
| **[Transpose](../edit/transpose)** | Switches the rows and columns of the CSV document. |
| **[Pivot Table](../edit/pivot_table)** | Creates a pivot table in the CSV document. |
| **[Unpivot](../edit/unpivot)** | Converts columns into rows by flattening the CSV data. |
| [**Join CSV**](../edit/join_csv) | Joins two CSV documents. |
| [**Extract Columns**](../edit/extract_columns) | Extracts specified columns from the current CSV document. |

| **[Add/Remove Separator](../edit/add_remove_separator)** | Adds or removes a separator at the cursor position (or at the pointed position on the ruler) while in the normal mode to prepare for conversion into the fixed width columns. |
| **[Remove All Separators](../edit/remove_all_separators)** | Removes all the separators while in the normal mode. |
| **[Columns/Separators (popup menu)](../edit/sv_sep_popup)** | Shows a popup menu to work with columns or separators. |
| **[Edit Cell (Cell Selection Mode Only)](../edit/csv_edit_cell)** | Edits the current cell in the CSV document. |
| **[Clear Contents (Cell Selection Mode Only)](../edit/clear_contents)** | Clear the contents of the selection in the CSV document. |
| **[Next Cell (Cell Selection Mode Only)](../edit/csv_next_cell)** | Jumps to the next cell in the CSV document. |
| **[Previous Cell (Cell Selection Mode Only)](../edit/csv_prev_cell)** | Jumps to the previous cell in the CSV document. |

| **[Cell Below (Cell Selection Mode Only)](../edit/csv_down)** | Jumps to the cell below in the CSV document. |

| **[Left Edge (Cell Selection Mode Only)](../edit/left_edge)** | Moves to the left edge of the current data region. |
| **[Top Edge (Cell Selection Mode Only)](../edit/top_edge)** | Moves to the top edge of the current data region. |
| **[Bottom Edge (Cell Selection Mode Only)](../edit/bottom_edge)** | Moves to the bottom edge of the current data region. |

| **[Bottom Edge Extend (Cell Selection Mode Only)](../edit/shift_bottom_edge)** | Extends the selection to the bottom edge of the current data region. |

| **[Next Cell](../edit/next_cell)** | Jumps to the next cell in CSV mode. |
