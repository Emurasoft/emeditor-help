# Column Heading 0 command

## Summary

Resets column headings.

## Description

Resets column headings.

- [All Commands](../tools/all_commands): **CSV** \> **Headings (Freeze Panes)** \> **Column Heading 0**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None
