# Customize Web Browser command

## Summary

Opens the Web Browser page of the Customize dialog box.

## Description

Displays the Web Browser page of the [**Customize** dialog box](../../dlg/customize/index),
where you can define common application settings.

## How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Web Browser**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_CUSTOMIZE_OPTIMIZATION (9069)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(9069);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9069
```
