# File Properties command

## Summary

Displays File properties.

## Description

Displays the [**File** page](../../dlg/properties/file/index)
of the **[Configuration Properties](../../dlg/properties/index)**
dialog box.

## How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Properties for Current Configuration** \> **File**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_PROPERTY_FILE (8962)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(8962);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8962
```
