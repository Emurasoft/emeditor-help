# Customize Update command

### Summary

> Opens the Update page of the Customize dialog box.

### Description

> Displays the [**Update** page](../../dlg/customize/update/index) of the [**Customize** dialog box](../../dlg/customize/index),
> where you can define common application settings.

### How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Update**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_CUSTOMIZE\_UPDATE (9064)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(9064);

#### \[VBScript\]

> editor.ExecuteCommandByID 9064
