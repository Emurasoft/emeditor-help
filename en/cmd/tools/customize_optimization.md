# Customize Optimization command

## Summary

Opens the Optimization page of the Customize dialog box.

## Description

Displays the Optimization page of the [**Customize** dialog box](../../dlg/customize/index),
where you can define common application settings.

## How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Optimization**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_CUSTOMIZE_OPTIMIZATION (9061)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(9061);
```

### \[VBScript\]

```
