# Highlight (2) Properties command

## Summary

Displays Highlight (2) properties.

## Description

Displays the [Highlight (2) page](../../dlg/properties/highlight2/index)
of the[Configuration Properties](../../dlg/properties/index)
dialog box.

## How to Run

- Default Menu: None
- [All Commands](all_commands):Tools >
Properties for Current Configuration \>Highlight (2)
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_PROPERTY_COMMENT (8969)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(8969);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 8969
```
