# Customize Format command

### Summary

> Opens the Format page of the Customize dialog box.

### Description

> Displays the Format page of the [**Customize** dialog box](../../dlg/customize/index),
> where you can define common application settings.

### How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Format**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_CUSTOMIZE\_FORMAT (9056)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(9056);

#### \[VBScript\]

> editor.ExecuteCommandByID 9056
