# Customize Marks command

### Summary

> Opens the Marks page of the Customize dialog box.

### Description

> Displays the Marks page of the [**Customize** dialog box](../../dlg/customize/index),
> where you can define common application settings.

### How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** >
**Customize** >
**Marks**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_CUSTOMIZE\_MARKS (9050)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(9050);

#### \[VBScript\]

> editor.ExecuteCommandByID 9050
