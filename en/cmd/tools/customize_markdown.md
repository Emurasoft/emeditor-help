# Customize Markdown command

## Summary

Opens the Markdown page of the Customize dialog box.

## Description

Displays the **Markdown** page of the [**Customize** dialog box](../../dlg/customize/index),
where you can define common application settings.

## How to Run

- Default Menu: None
- [All Commands](all_commands): **Tools** > **Customize** > **Markdown**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_CUSTOMIZE_MARKDOWN (9075)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(9075);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 9075
```
