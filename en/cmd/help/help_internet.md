# Web Help command

### Summary

> Displays Web Help.

### Description

> Displays Web Help.

### How to Run

- Default Menu: **Help** \> **Web Help**
- [All Commands](../tools/all_commands): **Help** >
**Web Help**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_HELP\_REGIST (3932)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(3932);

#### \[VBScript\]

> editor.ExecuteCommandByID 3932
