# Format Selection command

## Summary

## Description

Formats the selection using the Language Server Protocol.

## How to Run

- Default Menu: **Convert** \> **Format**
- [All Commands](../tools/all_commands): **Convert** \> **Format Selection**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: Ctrl+K, F
