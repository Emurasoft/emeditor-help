# Insert Tab command

## Summary

Inserts a tab at the current cursor position.

## Description

Inserts a tab at the current cursor position.

## How to Run

- Default Menu: **Insert** \> **Tab**
- [All Commands](../tools/all_commands): **Insert** \> **Tab**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

EEID_INSERT_TAB (3878)```
