# CSV Toolbar command

## Summary

Shows or hides the CSV toolbar.

## Description

Shows or hides the CSV toolbar.

## How to Run

- Default Menu: **View** >
**Toolbars** \> **CSV Toolbar**
- [All Commands](../tools/all_commands): **View** >
**Toolbars** \> **CSV Toolbar**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_SHOW_SV_BAR (3890)```

## Macros

## \[JavaScript\]

```
editor.ExecuteCommandByID(3890);
```

## \[VBScript\]

```
editor.ExecuteCommandByID 3890
```
