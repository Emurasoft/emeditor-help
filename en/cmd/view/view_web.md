# Web Browser command

## Summary

Shows or hides the Web Browser.

## Description

Shows or hides the Web Browser.

## How to Run

- Default Menu: **View** \> **Web Browser**
- [All Commands](../tools/all_commands): **View** \> **Web Browser**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_VIEW_WEB (23243)```

## Macros

### \[JavaScript\]

```
WebBar.Visible = !WebBar.Visible;
```

### \[VBScript\]

```
WebBar.Visible = Not WebBar.Visible
```
