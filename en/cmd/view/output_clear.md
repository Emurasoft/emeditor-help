# Clear Output Bar command

## Summary

Clears the contents of the Output Bar.

## Description

Clears the contents of the Output Bar.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **View** >
**Output Bar** >
**Clear Output**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_OUTPUT_CLEAR (4452)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4452);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4452
```
