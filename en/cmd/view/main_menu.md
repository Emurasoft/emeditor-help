# Main Menu command

## Summary

Shows or hides the main menu.

## Description

Shows or hides the main menu.

## How to Run

- Default Menu: **View** \> **Main Menu**
- [All Commands](../tools/all_commands): **View** >
**Main Menu**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_MAIN_MENU (4072)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4072);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4072
```
