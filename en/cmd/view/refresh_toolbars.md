# Refresh Toolbars command

## Summary

Refreshes all toolbars.

## Description

Refreshes all toolbars.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **View** >
**Toolbars** \> **Refresh**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

## Plug-in Command ID

```
EEID_REFRESH_TOOLBARS (4422)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4422);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4422
```
