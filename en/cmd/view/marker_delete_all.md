# Delete All Markers command

### Summary

> Deletes all markers.

### Description

> Deletes all markers.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **View** \> **Markers**
\> **Delete All**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_MARKER\_DELETE\_ALL (4587)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(4587);

#### \[VBScript\]

> editor.ExecuteCommandByID 4587
