# Scroll Right command

## Summary

Scrolls the document right by one character.

## Description

Scrolls the document right by one character.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Window** \>
**Scroll** \> **Right**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

## Plug-in Command ID

```
EEID_WINDOW_SCROLL_RIGHT (4475)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(4475);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 4475
```
