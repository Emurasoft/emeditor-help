# Zoom In Minimap command

## Summary

Zooms in the minimap.

## Description

Zooms in the minimap.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands):Window \>Scroll \>Zoom In
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

## Plug-in Command ID

```
EEID_MINIMAP_ZOOM_IN (3976)```

## Macros

### \[JavaScript\]

```
editor.ExecuteCommandByID(3976);
```

### \[VBScript\]

```
editor.ExecuteCommandByID 3976
```
