# Scroll Top command

### Summary

> Scrolls the document to the top of the document.

### Description

> Scrolls the document to the top of the document.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Window** \> **Scroll** \> **Top**
- Toolbar: None
- Status Bar: None
- Default Keyboard Shortcut: None

### Plug-in Command ID

- EEID\_SCROLL\_TOP (3864)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(3864);

#### \[VBScript\]

> editor.ExecuteCommandByID 3864
