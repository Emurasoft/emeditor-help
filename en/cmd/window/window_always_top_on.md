# Always on Top - On command

### Summary

> Enables always on top.

### Description

> Enables always on top.

### How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Window** \>
**Always on Top** \>
**Always on Top - On**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None

### Plug-in Command ID

- EEID\_WINDOW\_ALWAYS\_TOP\_ON (4343)

### Macros

#### \[JavaScript\]

> editor.ExecuteCommandByID(4343);

#### \[VBScript\]

> editor.ExecuteCommandByID 4343
