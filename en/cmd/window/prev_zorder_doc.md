# Least Accessed Document command

## Summary

Switches to the least accessed document.

## Description

Switches to the least accessed document.

## How to Run

- Default Menu: None
- [All Commands](../tools/all_commands): **Window**
\> **Document Navigation** \> **Least Accessed Document**
- Toolbar: None
- Status Bar: None
- Default Shortcut Key: None
